## 赛事说明

### AgiBot World Challenge @ ICRA 2026

**赛道一：Manipulation（操作任务）**
- 考察机器人在真实场景中的操作能力
- VLA模型控制

**赛道二：World Model（世界模型）**
- 考察视频预测能力
- EVAC模型：动作条件视频生成
- 根据首帧图像和动作信号，预测未来的视频帧

### 评测指标

| 指标 | 含义 | EVAC 基线 |
|------|------|-----------|
| PSNR | 峰值信噪比，越高越好 | 20.98 |
| Scene Consistency | 场景一致性，值越接近1越好 | 0.90 |
| nDTW | 轨迹动态时间规整，越高越好 | 0.91 |
