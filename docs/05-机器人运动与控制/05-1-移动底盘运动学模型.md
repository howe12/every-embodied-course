## 1. 差分驱动模型概述

**差分驱动（Differential Drive）** 是移动机器人最常用的底盘驱动方式之一。其核心思想是通过两个独立驱动的轮子之间的速度差来实现机器人的转向。

### 1.1 差分驱动系统组成

```
        ┌─────────────────────────┐
        │        机器人本体        │
        │   ┌─────────────────┐   │
        │   │                 │   │
        │   │    ← 左右轮 →   │   │
        │   │   ●         ●  │   │
        │   │   左轮    右轮  │   │
        │   └─────────────────┘   │
        │         ▲               │
        │         │ 前方向        │
        └─────────────────────────┘
```

**关键参数**：
- **L**：轮距（两个轮子中心之间的距离）
- **r**：轮子半径
- **v_L**：左轮线速度
- **v_R**：右轮线速度

### 1.2 运动学模型假设

在推导运动学模型之前，我们做出以下假设：
1. 机器人运动在二维平面内
2. 轮子与地面之间无滑动（纯滚动）
3. 机器人质心位于两轮轴线的几何中心

## 2. 正解：轮速 → 机器人位姿

**正向运动学（Forward Kinematics）**：已知左右轮速，计算机器人的位姿变化。

### 2.1 运动学公式

```
机器人中心速度 v = (v_R + v_L) / 2        （平均速度）
机器人角速度 ω = (v_R - v_L) / L          （速度差除以轮距）
```

**公式推导**：
- 当左右轮速相等时，机器人直线前进：v = v_R = v_L，ω = 0
- 当右轮快于左轮时，机器人左转：v_R > v_L → ω > 0
- 当左轮快于右轮时，机器人右转：v_L > v_R → ω < 0

### 2.2 位姿更新方程

```
Δx = v × cos(θ) × Δt
Δy = v × sin(θ) × Δt
Δθ = ω × Δt
```

其中 (x, y, θ) 是机器人的位姿，θ 为航向角。

### 2.3 Python实现

```python
import numpy as np

class DifferentialDriveKinematics:
    """差分驱动运动学模型"""
    
    def __init__(self, wheel_radius: float, wheelbase: float):
        """
        初始化运动学模型
        
        Args:
            wheel_radius: 轮子半径 (m)
            wheelbase: 轮距，两个轮子中心之间的距离 (m)
        """
        self.r = wheel_radius
        self.L = wheelbase
    
    def forward_kinematics(self, v_left: float, v_right: float) -> tuple:
        """
        正解：轮速 -> 机器人线速度和角速度
        
        Args:
            v_left: 左轮线速度 (m/s)
            v_right: 右轮线速度 (m/s)
            
        Returns:
            (v, omega): 线速度 (m/s), 角速度 (rad/s)
        """
        v = (v_right + v_left) / 2.0
        omega = (v_right - v_left) / self.L
        return v, omega
    
    def compute_odometry(self, x: float, y: float, theta: float,
                          v_left: float, v_right: float, dt: float) -> tuple:
        """
        计算里程计：根据轮速更新机器人位姿
        
        Args:
            x, y: 当前位姿 (m)
            theta: 当前航向角 (rad)
            v_left: 左轮速度 (m/s)
            v_right: 右轮速度 (m/s)
            dt: 时间间隔 (s)
            
        Returns:
            (new_x, new_y, new_theta): 更新后的位姿
        """
        # 计算机器人的线速度和角速度
        v, omega = self.forward_kinematics(v_left, v_right)
        
        # 更新位姿
        if abs(omega) < 1e-6:
            # 近似直线运动
            new_x = x + v * np.cos(theta) * dt
            new_y = y + v * np.sin(theta) * dt
            new_theta = theta
        else:
            # 曲线运动
            new_x = x + (v / omega) * (np.sin(theta + omega * dt) - np.sin(theta))
            new_y = y + (v / omega) * (np.cos(theta) - np.cos(theta + omega * dt))
            new_theta = theta + omega * dt
        
        # 归一化角度到 [-π, π]
        new_theta = np.arctan2(np.sin(new_theta), np.cos(new_theta))
        
        return new_x, new_y, new_theta


# 示例使用
if __name__ == "__main__":
    # TurtleBot3参数
    r = 0.033  # 轮子半径 33mm
    L = 0.287  # 轮距 287mm
    
    kin = DifferentialDriveKinematics(r, L)
    
    # 假设左右轮速 (m/s)
    v_left = 0.1
    v_right = 0.15
    
    # 计算线速度和角速度
    v, omega = kin.forward_kinematics(v_left, v_right)
    print(f"线速度: {v:.3f} m/s")
    print(f"角速度: {omega:.3f} rad/s")
    
    # 模拟1秒后的位姿
    x, y, theta = 0.0, 0.0, 0.0
    dt = 0.1
    for _ in range(10):
        x, y, theta = kin.compute_odometry(x, y, theta, v_left, v_right, dt)
    print(f"1秒后位姿: x={x:.3f}, y={y:.3f}, θ={np.degrees(theta):.1f}°")
```

## 3. 逆解：目标速度 → 轮速分配

**逆向运动学（Inverse Kinematics）**：已知期望的机器人线速度 v 和角速度 ω，计算左右轮速。

### 3.1 运动学公式

由正解公式反推：

```
v_R = v + ω × L / 2
v_L = v - ω × L / 2
```

### 3.2 Python实现

```python
class DifferentialDriveKinematics:
    # ... (前面的代码保持不变)
    
    def inverse_kinematics(self, v: float, omega: float) -> tuple:
        """
        逆解：机器人目标速度 -> 轮速分配
        
        Args:
            v: 机器人线速度 (m/s)
            omega: 机器人角速度 (rad/s)
            
        Returns:
            (v_left, v_right): 左轮速度, 右轮速度 (m/s)
        """
        v_right = v + omega * self.L / 2.0
        v_left = v - omega * self.L / 2.0
        return v_left, v_right
    
    def check_velocity_limits(self, v_left: float, v_right: float,
                               max_v: float = 1.0) -> bool:
        """
        检查轮速是否超过限制
        
        Args:
            v_left: 左轮速度
            v_right: 右轮速度
            max_v: 最大允许速度 (m/s)
            
        Returns:
            是否在限制内
        """
        return abs(v_left) <= max_v and abs(v_right) <= max_v


# 示例：逆解使用
if __name__ == "__main__":
    kin = DifferentialDriveKinematics(0.033, 0.287)
    
    # 期望机器人以 0.2 m/s 直线前进
    v, omega = 0.2, 0.0
    v_left, v_right = kin.inverse_kinematics(v, omega)
    print(f"直线前进: v_L={v_left:.3f}, v_R={v_right:.3f}")
    
    # 期望机器人以 0.2 m/s 前进，同时左转 (omega = 0.5 rad/s)
    v, omega = 0.2, 0.5
    v_left, v_right = kin.inverse_kinematics(v, omega)
    print(f"左转: v_L={v_left:.3f}, v_R={v_right:.3f}")
```

## 4. 运动约束

在实际应用中，机器人运动受到多种约束限制，必须在控制器中加以考虑。

### 4.1 最大线速度/角速度约束

机器人机械结构限制了最大速度和角速度：

```python
class ConstrainedKinematics:
    """带约束的差分驱动运动学"""
    
    def __init__(self, wheel_radius: float, wheelbase: float,
                 max_linear_vel: float = 1.0,
                 max_angular_vel: float = 2.0):
        self.kin = DifferentialDriveKinematics(wheel_radius, wheelbase)
        self.max_v = max_linear_vel
        self.max_omega = max_angular_vel
    
    def constrain_velocity(self, v: float, omega: float) -> tuple:
        """
        限制速度在可行范围内
        
        Args:
            v: 期望线速度
            omega: 期望角速度
            
        Returns:
            (v, omega): 限制后的速度
        """
        # 限制线速度
        v = np.clip(v, -self.max_v, self.max_v)
        
        # 限制角速度
        omega = np.clip(omega, -self.max_omega, self.max_omega)
        
        # 还要考虑组合限制：v 和 omega 不能同时达到最大
        # 这是一个简化处理
        return v, omega
    
    def safe_inverse_kinematics(self, v: float, omega: float) -> tuple:
        """
        安全的逆解：先约束速度，再计算轮速
        
        Returns:
            (v_left, v_right, clipped): 轮速和是否被限制
        """
        v, omega = self.constrain_velocity(v, omega)
        v_left, v_right = self.kin.inverse_kinematics(v, omega)
        
        # 检查轮速是否超限
        max_wheel_v = self.max_v  # 假设轮速限制等于最大线速度
        clipped = False
        
        if abs(v_left) > max_wheel_v or abs(v_right) > max_wheel_v:
            clipped = True
            # 缩放到最大允许值
            max_abs = max(abs(v_left), abs(v_right))
            scale = max_wheel_v / max_abs
            v_left *= scale
            v_right *= scale
        
        return v_left, v_right, clipped
```

### 4.2 加速度限制

除了速度限制，加速度也不可忽略，否则会导致机器人抖动或打滑：

```python
class AccelerationLimitedController:
    """加速度限制的速度控制器"""
    
    def __init__(self, max_linear_accel: float = 1.0,
                 max_angular_accel: float = 2.0):
        self.max_linear_accel = max_linear_accel
        self.max_angular_accel = max_angular_accel
        self.prev_v = 0.0
        self.prev_omega = 0.0
    
    def smooth_velocity(self, target_v: float, target_omega: float,
                        dt: float) -> tuple:
        """
        平滑速度，应用加速度限制
        
        Args:
            target_v: 目标线速度
            target_omega: 目标角速度
            dt: 控制周期
            
        Returns:
            (v, omega): 平滑后的速度
        """
        # 计算速度变化
        dv = target_v - self.prev_v
        domega = target_omega - self.prev_omega
        
        # 限制加速度
        max_dv = self.max_linear_accel * dt
        max_domega = self.max_angular_accel * dt
        
        dv = np.clip(dv, -max_dv, max_dv)
        domega = np.clip(domega, -max_domega, max_domega)
        
        # 更新速度
        new_v = self.prev_v + dv
        new_omega = self.prev_omega + domega
        
        # 保存为上一帧速度
        self.prev_v = new_v
        self.prev_omega = new_omega
        
        return new_v, new_omega
```

### 4.3 避免打滑

打滑发生在轮子与地面之间的摩擦力不足以支持期望的运动时：

```python
class AntiSlipController:
    """防打滑控制器"""
    
    def __init__(self, wheel_radius: float, wheelbase: float,
                 max_friction_accel: float = 3.0):
        """
        Args:
            max_friction_accel: 最大摩擦加速度 (m/s²)
        """
        self.kin = DifferentialDriveKinematics(wheel_radius, wheelbase)
        self.max_friction_accel = max_friction_accel
    
    def check_slip(self, v_left: float, v_right: float,
                   current_v: float, dt: float) -> tuple:
        """
        检查是否可能打滑，返回修正后的轮速
        
        Returns:
            (v_left, v_right, is_slipping): 修正后的轮速和打滑标志
        """
        # 计算目标加速度
        a = (v_left + v_right) / 2.0 / dt
        
        if abs(a) > self.max_friction_accel:
            # 限制加速度在摩擦极限内
            scale = self.max_friction_accel / abs(a)
            v_left *= scale
            v_right *= scale
            return v_left, v_right, True
        
        return v_left, v_right, False
```

## 5. 运动学仿真

### 5.1 完整仿真程序

```python
import numpy as np
import matplotlib.pyplot as plt

class DifferentialDriveSimulator:
    """差分驱动机器人仿真器"""
    
    def __init__(self, wheel_radius: float = 0.033, wheelbase: float = 0.287):
        self.kin = DifferentialDriveKinematics(wheel_radius, wheelbase)
        self.reset()
    
    def reset(self):
        """重置机器人状态"""
        self.x = 0.0
        self.y = 0.0
        self.theta = 0.0
    
    def step(self, v_left: float, v_right: float, dt: float = 0.1):
        """仿真一步"""
        self.x, self.y, self.theta = self.kin.compute_odometry(
            self.x, self.y, self.theta, v_left, v_right, dt
        )
        return self.x, self.y, self.theta
    
    def drive_circle(self, radius: float, linear_vel: float,
                     duration: float, dt: float = 0.1):
        """
        让机器人走圆轨迹
        
        Args:
            radius: 圆的半径 (m)
            linear_vel: 线速度 (m/s)
            duration: 持续时间 (s)
        """
        # 圆周运动: omega = v / radius
        omega = linear_vel / radius
        
        # 逆解得到轮速
        v_left, v_right = self.kin.inverse_kinematics(linear_vel, omega)
        
        # 仿真
        steps = int(duration / dt)
        trajectory = []
        
        for _ in range(steps):
            self.step(v_left, v_right, dt)
            trajectory.append((self.x, self.y, self.theta))
        
        return trajectory


def run_simulation():
    """运行仿真并可视化"""
    sim = DifferentialDriveSimulator()
    
    # 仿真场景1: 走直线
    print("=== 场景1: 直线前进 ===")
    sim.reset()
    for _ in range(100):
        sim.step(0.1, 0.1, 0.1)
    print(f"终点: x={sim.x:.3f}, y={sim.y:.3f}, θ={np.degrees(sim.theta):.1f}°")
    
    # 仿真场景2: 原地旋转
    print("\n=== 场景2: 原地旋转 ===")
    sim.reset()
    for _ in range(100):
        sim.step(-0.1, 0.1, 0.1)
    print(f"终点: x={sim.x:.3f}, y={sim.y:.3f}, θ={np.degrees(sim.theta):.1f}°")
    
    # 仿真场景3: 走圆
    print("\n=== 场景3: 走半径1m的圆 ===")
    sim.reset()
    traj = sim.drive_circle(radius=1.0, linear_vel=0.2, duration=6.28)
    
    # 验证：走完一圈应该回到原点附近
    final = traj[-1]
    print(f"终点: x={final[0]:.3f}, y={final[1]:.3f}, θ={np.degrees(final[2]):.1f}°")
    print(f"预期: x≈1.0, y≈0.0 (起点)"
          )
    
    # 可视化
    fig, ax = plt.subplots(1, 1, figsize=(8, 8))
    xs = [p[0] for p in traj]
    ys = [p[1] for p in traj]
    ax.plot(xs, ys, 'b-', linewidth=2)
    ax.scatter([0], [0], c='green', s=100, marker='o', label='起点')
    ax.scatter([xs[-1]], [ys[-1]], c='red', s=100, marker='x', label='终点')
    ax.set_xlabel('X (m)')
    ax.set_ylabel('Y (m)')
    ax.set_title('差分驱动机器人圆周运动轨迹')
    ax.legend()
    ax.grid(True)
    ax.set_aspect('equal')
    plt.tight_layout()
    plt.savefig('trajectory.png', dpi=150)
    print("\n轨迹已保存到 trajectory.png")


if __name__ == "__main__":
    run_simulation()
```

## 6. 公式-代码对照表

| 概念 | 数学公式 | Python代码 |
|------|----------|------------|
| 机器人线速度 | $v = \frac{v_R + v_L}{2}$ | `v = (v_right + v_left) / 2.0` |
| 机器人角速度 | $\omega = \frac{v_R - v_L}{L}$ | `omega = (v_right - v_left) / L` |
| 右轮速度 | $v_R = v + \omega \cdot \frac{L}{2}$ | `v_right = v + omega * L / 2.0` |
| 左轮速度 | $v_L = v - \omega \cdot \frac{L}{2}$ | `v_left = v - omega * L / 2.0` |
| 位姿更新(直线) | $\Delta x = v \cos\theta \cdot \Delta t$ | `x + v * cos(theta) * dt` |
| 位姿更新(曲线) | $\theta_{new} = \theta + \omega \cdot \Delta t$ | `theta + omega * dt` |

```python
# 参数
L = 0.3  # 轮距 m
r = 0.05  # 轮半径 m

# 轮速
v_left = 0.1  # m/s
v_right = 0.2  # m/s
dt = 2.0  # s

# 正解计算
v = (v_right + v_left) / 2  # = 0.15 m/s
omega = (v_right - v_left) / L  # = 0.333 rad/s

# 位姿更新（近似直线，因为 omega 较小）
x = v * np.cos(0) * dt  # = 0.3 m
y = v * np.sin(0) * dt  # = 0 m
theta = omega * dt  # = 0.667 rad = 38.2°

print(f"2秒后位姿: x={x:.3f}m, y={y:.3f}m, θ={np.degrees(theta):.1f}°")
```

**答案**：约 (0.30m, 0m, 38.2°)
</details>

### 练习2：目标速度计算轮速

已知机器人参数 L = 0.287m（TurtleBot3）。

**问题**：如果要使机器人以 0.3 m/s 的线速度前进，同时以 0.5 rad/s 的角速度左转，左右轮速应设置为多少？

<details>
<summary>点击查看答案</summary>

```python
L = 0.287  # m
v = 0.3  # m/s
omega = 0.5  # rad/s

# 逆解
v_right = v + omega * L / 2  # = 0.37175 m/s
v_left = v - omega * L / 2  # = 0.22825 m/s

print(f"左轮: {v_left:.4f} m/s, 右轮: {v_right:.4f} m/s")
```

**答案**：左轮约 0.228 m/s，右轮约 0.372 m/s
</details>

## 8. 总结

本节课我们学习了差分驱动机器人的运动学模型：

1. **正向运动学**：已知轮速求机器人速度
   - $v = (v_R + v_L) / 2$
   - $\omega = (v_R - v_L) / L$

2. **逆向运动学**：已知机器人目标速度求轮速
   - $v_R = v + \omega \cdot L / 2$
   - $v_L = v - \omega \cdot L / 2$

3. **运动约束**：
   - 速度限制
   - 加速度限制
   - 防打滑处理

4. **仿真验证**：通过Python代码可以仿真和验证运动学模型
