## 1. 协同控制概述

移动机器人-机械臂协同系统（Mobile Manipulation）是指将移动底盘与机械臂整合在一起，使机器人具备在更大范围内移动并执行精细操作的能力。这种系统在工业物流、家庭服务、仓储管理等场景中有着广泛应用。

### 1.1 应用场景

**工业物流**：在工厂和仓库中，移动机械臂可以完成货物的拣选、搬运、分拣等任务。例如，亚马逊的Kiva机器人配合机械臂进行订单拣选，极大提高了物流效率。

**家庭服务**：服务机器人需要在家中自主移动并执行家务任务，如取放物品、整理房间等。这要求机器人具备精确的导航和操作能力。

**医疗辅助**：移动机械臂可以用于医院中的药品配送、医疗器械操作等，减轻医护人员的工作负担。

**危险环境作业**：在核电站、火灾现场等危险环境中，移动机械臂可以代替人类执行探测、取样、救援等任务。

### 1.2 系统架构

**任务规划层**负责理解高层任务指令，将其分解为可执行的子动作序列，并估计当前系统状态。

**协调控制层**是系统的核心，它协调移动底盘和机械臂的运动，确保两者之间的动作能够正确同步。

**执行层**直接与硬件交互，输出控制指令到驱动器和传感器。

### 1.3 协同控制模式

移动机械臂协同控制主要有三种模式：

**顺序执行模式**：先完成移动，再执行机械臂操作，或者反之。这是最简单的模式，但效率较低。

**同时协调模式**：移动底盘和机械臂同时运动，但彼此独立。这种模式提高了效率，但可能产生冲突。

**紧耦合模式**：移动底盘和机械臂作为一个整体进行规划和控制，考虑两者之间的运动约束。这是最高效但也最复杂的模式。

## 2. 协同运动规划

协同运动规划是移动机械臂系统的核心问题，需要同时考虑移动平台的路径规划和机械臂的运动规划。

### 2.1 运动学模型

移动机械臂的运动学模型需要同时描述移动底盘和机械臂的状态。

**移动底盘运动学**：假设使用差速驱动底盘，其运动学模型为：

```
x_dot = v * cos(θ)
y_dot = v * sin(θ)
θ_dot = ω
```

其中 v 是线速度，ω 是角速度，θ 是航向角。

**机械臂运动学**：使用标准的DH参数或URDF模型描述机械臂的正逆运动学。

**复合运动学**：移动机械臂的末端执行器位置由底盘位置和机械臂末端位置共同决定：

```
P_ee = P_base + R_base * P_arm
```

其中 P_ee 是末端执行器位置，P_base 是底盘位置，R_base 是底盘姿态旋转矩阵，P_arm 是机械臂末端相对于底盘的位置。

### 2.2 协同路径规划

协同路径规划需要在以下空间中进行搜索：

**底盘位置空间（SE(2)）**：移动底盘在平面上的位姿，包含位置(x, y)和方向θ。

**机械臂关节空间（JR）**：机械臂各关节的角度组合。

**复合配置空间（C = SE(2) × JR）**：底盘和机械臂状态的组合。

```
┌─────────────────────────────────────────────────────────────────┐
│                      配置空间可视化                               │
│                                                                  │
│        机械臂关节角度                                            │
│        ▲                                                        │
│        │    ┌───┐                                               │
│        │    │   │  ← 可行配置区域                               │
│        │    │   │                                               │
│        │    └───┘                                               │
│        │         \                                             │
│        │          \  ← 障碍物区域                               │
│        │           \                                            │
│        └───────────────────────────────────────────────────────▶│
│                        底盘位置x                                  │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 移动基座避障

移动基座的避障需要考虑机械臂的存在。当机械臂展开时，会占用更大的空间，可能影响移动路径。

**扩展障碍物方法**：将机械臂的当前位置作为"虚拟障碍物"加入到环境地图中，确保移动路径不会与机械臂发生碰撞。

**可穿越区域计算**：根据机械臂的伸展程度，动态计算可穿越区域的大小。当机械臂收缩时，移动机器人可以通过更狭窄的空间。

**分层规划**：先在简化的配置空间中进行粗略规划，然后在完整配置空间中进行精细化。

### 4.2 状态机实现

使用状态机管理复杂的任务序列：

```python
# state_machine.py
from enum import Enum

class ManipulationState(Enum):
    IDLE = "idle"
    NAVIGATING = "navigating"
    APPROACHING = "approaching"
    GRASPING = "grasping"
    LIFTING = "lifting"
    TRANSPORTING = "transporting"
    PLACING = "placing"
    RETRACTING = "retracting"
    COMPLETED = "completed"
    FAILED = "failed"

class ManipulationStateMachine:
    def __init__(self):
        self.state = ManipulationState.IDLE
        self.error_message = None
        
    def transition(self, new_state):
        """状态转换"""
        self.state = new_state
        print(f"State transition: {new_state.value}")
        
    def can_transition(self, from_state, to_state):
        """检查状态转换是否合法"""
        valid_transitions = {
            ManipulationState.IDLE: [ManipulationState.NAVIGATING],
            ManipulationState.NAVIGATING: [ManipulationState.APPROACHING, ManipulationState.FAILED],
            ManipulationState.APPROACHING: [ManipulationState.GRASPING, ManipulationState.FAILED],
            ManipulationState.GRASPING: [ManipulationState.LIFTING, ManipulationState.FAILED],
            ManipulationState.LIFTING: [ManipulationState.TRANSPORTING, ManipulationState.FAILED],
            ManipulationState.TRANSPORTING: [ManipulationState.PLACING, ManipulationState.FAILED],
            ManipulationState.PLACING: [ManipulationState.RETRACTING, ManipulationState.FAILED],
            ManipulationState.RETRACTING: [ManipulationState.COMPLETED, ManipulationState.FAILED],
            ManipulationState.COMPLETED: [ManipulationState.IDLE],
            ManipulationState.FAILED: [ManipulationState.IDLE],
        }
        
        return to_state in valid_transitions.get(from_state, [])
```

```python
    def __lt__(self, other):
```

```python
# mobile_manipulation/mobile_manipulation/mobile_coordinator.py
import rclpy
from rclpy.node import Node
from rclpy.action import ActionClient
from geometry_msgs.msg import Pose, PoseStamped, TransformStamped
from sensor_msgs.msg import JointState
from moveit_msgs.action import MoveGroup
from nav2_msgs.action import NavigateToPose
from moveit_msgs.msg import MoveitErrorCodes
import math
import tf_transformations

class MobileManipulatorCoordinator(Node):
    def __init__(self):
        super().__init__('mobile_manipulator_coordinator')
        
        # 参数
        self.declare_parameter('pick_height_offset', 0.05)
        self.declare_parameter('place_height_offset', 0.05)
        self.declare_parameter('approach_distance', 0.2)
        
        self.pick_height_offset = self.get_parameter('pick_height_offset').value
        self.place_height_offset = self.get_parameter('place_height_offset').value
        self.approach_distance = self.get_parameter('approach_distance').value
        
        # 动作客户端
        self.moveit_client = ActionClient(self, MoveGroup, '/move_group')
        self.nav_client = ActionClient(self, NavigateToPose, '/navigate_to_pose')
        
        # 状态
        self.is_navigating = False
        self.is_manipulating = False
        
        self.get_logger().info('Mobile Manipulator Coordinator initialized')
        
    def execute_pick_place(self, pick_pos, place_pos):
        """执行抓取-搬运-放置任务"""
        self.get_logger().info('Starting pick and place task')
        
        # 计算抓取位置
        pick_pose = self.create_pose(
            pick_pos[0], pick_pos[1], pick_pos[2] + self.pick_height_offset,
            0, math.pi/2, 0
        )
        
        # 计算放置位置
        place_pose = self.create_pose(
            place_pos[0], place_pos[1], place_pos[2] + self.place_height_offset,
            0, math.pi/2, 0
        )
        
        # 1. 导航到抓取位置
        self.get_logger().info('Step 1: Navigating to pick location')
        pick_approach = self.compute_approach(pick_pose, self.approach_distance)
        
        if not self.navigate_to(pick_approach):
            self.get_logger().error('Navigation to pick failed')
            return False
            
        # 2. 移动机械臂到抓取位置
        self.get_logger().info('Step 2: Moving arm to pick pose')
        if not self.move_arm_to(pick_pose, 'tool0'):
            self.get_logger().error('Move to pick pose failed')
            return False
            
        # 3. 抓取（模拟）
        self.get_logger().info('Step 3: Grasping object')
        self.simulate_grasp()
        
        # 4. 抬起
        self.get_logger().info('Step 4: Lifting object')
        lift_pose = self.compute_lift_pose(pick_pose)
        self.move_arm_to(lift_pose, 'tool0')
        
        # 5. 导航到放置位置
        self.get_logger().info('Step 5: Navigating to place location')
        place_approach = self.compute_approach(place_pose, self.approach_distance)
        
        if not self.navigate_to(place_approach):
            self.get_logger().error('Navigation to place failed')
            return False
            
        # 6. 移动机械臂到放置位置
        self.get_logger().info('Step 6: Moving arm to place pose')
        if not self.move_arm_to(place_pose, 'tool0'):
            self.get_logger().error('Move to place pose failed')
            return False
            
        # 7. 释放
        self.get_logger().info('Step 7: Releasing object')
        self.simulate_release()
        
        # 8. 撤回
        self.get_logger().info('Step 8: Retracting arm')
        retract_pose = self.compute_retract_pose(place_pose)
        self.move_arm_to(retract_pose, 'tool0')
        
        self.get_logger().info('Pick and place task completed successfully!')
        return True
        
    def create_pose(self, x, y, z, roll, pitch, yaw):
        """创建Pose消息"""
        pose = Pose()
        pose.position.x = x
        pose.position.y = y
        pose.position.z = z
        
        quat = tf_transformations.quaternion_from_euler(roll, pitch, yaw)
        pose.orientation.x = quat[0]
        pose.orientation.y = quat[1]
        pose.orientation.z = quat[2]
        pose.orientation.w = quat[3]
        
        return pose
        
    def compute_approach(self, target_pose, distance):
        """计算接近位置"""
        approach = Pose()
        direction = math.atan2(
            target_pose.position.y,
            target_pose.position.x
        )
        
        approach.position.x = target_pose.position.x - distance * math.cos(direction)
        approach.position.y = target_pose.position.y - distance * math.sin(direction)
        approach.position.z = target_pose.position.z + 0.15
        approach.orientation = target_pose.orientation
        
        return approach
        
    def compute_lift_pose(self, pick_pose):
        """计算抬起位置"""
        lift = Pose()
        lift.position.x = pick_pose.position.x
        lift.position.y = pick_pose.position.y
        lift.position.z = pick_pose.position.z + 0.15
        lift.orientation = pick_pose.orientation
        return lift
        
    def compute_retract_pose(self, place_pose):
        """计算撤回位置"""
        return self.compute_lift_pose(place_pose)
        
    def navigate_to(self, pose):
        """导航到目标位置"""
        goal_msg = NavigateToPose.Goal()
        goal_msg.pose.header.frame_id = 'map'
        goal_msg.pose.header.stamp = self.get_clock().now().to_msg()
        goal_msg.pose.pose = pose
        
        self.nav_client.wait_for_server()
        
        send_goal_future = self.nav_client.send_goal_async(goal_msg)
        rclpy.spin_until_future_complete(self, send_goal_future, timeout_sec=10.0)
        
        goal_handle = send_goal_future.result()
        
        if not goal_handle.accepted:
            self.get_logger().warn('Navigation goal rejected')
            return False
            
        self.get_logger().info('Navigation goal accepted')
        
        result_future = goal_handle.get_result_async()
        rclpy.spin_until_future_complete(self, result_future, timeout_sec=60.0)
        
        result = result_future.result()
        if result and result.result == 1:
            self.get_logger().info('Navigation succeeded')
            return True
        else:
            self.get_logger().error('Navigation failed')
            return False
            
    def move_arm_to(self, pose, end_effector_link):
        """使用MoveIt2移动机械臂"""
        goal_msg = MoveGroup.Goal()
        
        # 设置目标姿态
        goal_msg.request.group_name = "manipulator"
        goal_msg.request.end_effector_link = end_effector_link
        
        # 使用姿态目标
        pose_goal = PoseStamped()
        pose_goal.header.frame_id = "base_link"
        pose_goal.header.stamp = self.get_clock().now().to_msg()
        pose_goal.pose = pose
        
        goal_msg.request.goal_pose = pose_goal
        goal_msg.request.planning_pipeline = "ompl"
        goal_msg.request.num_planning_attempts = 5
        goal_msg.request.allowed_planning_time = 2.0
        goal_msg.request.max_velocity_scaling_factor = 0.5
        goal_msg.request.max_acceleration_scaling_factor =
0.5
        
        self.moveit_client.wait_for_server()
        
        send_goal_future = self.moveit_client.send_goal_async(goal_msg)
        rclpy.spin_until_future_complete(self, send_goal_future, timeout_sec=5.0)
        
        goal_handle = send_goal_future.result()
        
        if not goal_handle.accepted:
            self.get_logger().warn('MoveIt goal rejected')
            return False
            
        result_future = goal_handle.get_result_async()
        rclpy.spin_until_future_complete(self, result_future, timeout_sec=10.0)
        
        result = result_future.result()
        
        if result and result.error_code.val == MoveitErrorCodes.SUCCESS:
            self.get_logger().info('MoveIt motion succeeded')
            return True
        else:
            self.get_logger().error(f'MoveIt motion failed: {result.error_code.val}')
            return False
            
    def simulate_grasp(self):
        """模拟抓取动作"""
        self.get_logger().info('Grasping...')
        # 这里可以添加实际的抓取逻辑
        
    def simulate_release(self):
        """模拟释放动作"""
        self.get_logger().info('Releasing...')
        # 这里可以添加实际的释放逻辑

def main(args=None):
    rclpy.init(args=args)
    
    coordinator = MobileManipulatorCoordinator()
    
    # 执行示例任务：抓取并放置
    # 抓取位置：x=1.0, y=0.0, z=0.0（相对于map坐标系）
    # 放置位置：x=2.0, y=1.0, z=0.0
    coordinator.execute_pick_place(
```

```python
# visual_servoing.py
import rclpy
from rclpy.node import Node
from geometry_msgs.msg import Twist
from sensor_msgs.msg import Image
from cv_bridge import CvBridge
import cv2
import numpy as np

class VisualServoingController(Node):
    def __init__(self):
        super().__init__('visual_servoing_controller')
        
        # 图像订阅
        self.bridge = CvBridge()
        self.image_sub = self.create_subscription(
            Image,
            '/camera/image_raw',
            self.image_callback,
            10
        )
        
        # 控制指令发布
        self.cmd_pub = self.create_publisher(Twist, '/cmd_vel', 10)
        
        # 目标位置（图像中心）
        self.target_x = 320  # 假设图像宽度640
        self.target_y = 240   # 假设图像高度480
        
        # PID参数
        self.kp = 0.001
        self.ki = 0.0001
        self.kd = 0.001
        
        # 误差积分
        self.error_x_integral = 0
        self.error_y_integral = 0
        
        # 上一次误差
        self.last_error_x = 0
        self.last_error_y = 0
        
    def image_callback(self, msg):
        """处理图像并计算控制指令"""
        try:
            cv_image = self.bridge.imgmsg_to_cv2(msg, desired_encoding='bgr8')
        except Exception as e:
            self.get_logger().error(f'CV Bridge error: {e}')
            return
        
        # 检测目标（简化：使用颜色分割检测红色物体）
        hsv = cv2.cvtColor(cv_image, cv2.COLOR_BGR2HSV)
        lower_red = np.array([0, 100, 100])
        upper_red = np.array([10, 255, 255])
        mask = cv2.inRange(hsv, lower_red, upper_red)
        
        # 找轮廓
        contours, _ = cv2.findContours(mask, cv2.RETR_TREE, cv2.CHAIN_APPROX_SIMPLE)
        
        if contours:
            # 取最大轮廓
            largest_contour = max(contours, key=cv2.contourArea)
            M = cv2.moments(largest_contour)
            
            if M['m00'] > 0:
                cx = int(M['m10'] / M['m00'])
                cy = int(M['m01'] / M['m00'])
                
                # 计算误差
                error_x = self.target_x - cx
                error_y = self.target_y - cy
                
                # PID控制
                self.error_x_integral += error_x
                self.error_y_integral += error_y
                
                derivative_x = error_x - self.last_error_x
                derivative_y = error_y - self.last_error_y
                
                vx = self.kp * error_x + self.ki * self.error_x_integral + self.kd * derivative_x
                vy = self.kp * error_y + self.ki * self.error_y_integral + self.kd * derivative_y
                
                # 发布控制指令
                cmd = Twist()
                cmd.linear.x = -vy * 0.001  # 前进/后退
                cmd.linear.y = -vx * 0.001  # 左右移动
                cmd.angular.z = -vx * 0.002  # 旋转
                
                self.cmd_pub.publish(cmd)
                
                self.last_error_x = error_x
                self.last_error_y = error_y
                
                # 可视化
                cv2.drawContours(cv_image, [largest_contour], -1, (0, 255, 0), 2)
                cv2.circle(cv_image, (cx, cy), 5, (0, 0, 255), -1)
                
        cv2.imshow('Visual Servoing', cv_image)
        cv2.waitKey(1)

def main(args=None):
    rclpy.init(args=args)
    node = VisualServoingController()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```
