## 📖 学习路径说明

### 第一阶段：基础入门

| 序号 | 课程 | 描述 |
| :--- | :--- | :--- |
| 01-1 | 机器人发展历史 | 了解机器人发展历程 |
| 01-2 | 机器人系统组成 | 硬件系统构成 |
| 01-3 | 具身智能概述 | 核心概念、大小脑架构、世界模型 |

### 第二阶段：机器人学基础

| 序号 | 课程 | 描述 |
| :--- | :--- | :--- |
| 02-1 | 空间描述与变换 | 坐标系、齐次矩阵 |
| 02-2 | 机器人运动学 | DH参数、正逆运动学 |
| 02-3 | 速度运动学 | 雅可比矩阵 |
| 02-4 | 机器人动力学 | 拉格朗日/牛顿欧拉法 |
| 02-5 | 控制算法 | PID/LQR/MPC |
| 02-6 | 综合实战 | 仿真验证 |

### 第三阶段：ROS开发

| 序号 | 课程 | 描述 |
| :--- | :--- | :--- |
| 04-1 | ROS发展历程 | ROS1 vs ROS2 |
| 04-2 | ROS2系统架构 | 节点、话题、服务 |
| 04-3 | ROS2安装 | 环境配置 |
| 04-4 | Linux基础 | 命令行操作 |
| 04-5 | ROS2工作空间 | 功能包创建 |
| 04-6 | Colcon构建 | 编译工具 |
| 04-7 | 开发工具 | GDB、VS Code |
| 04-8 | 常用工具 | RViz、Gazebo |
| 04-9 | Git版本控制 | 团队协作 |

### 第四阶段：感知与控制

| 序号 | 课程 | 描述 |
| :--- | :--- | :--- |
| 05 | 机器人运动与控制 | 电机控制、舵机 |
| 06 | 传感器数据获取 | 摄像头、激光雷达 |
| 07 | 机器人仿真技术 | Gazebo环境部署 |

### 第五阶段：智能决策

| 序号 | 课程 | 描述 |
| :--- | :--- | :--- |

## 🎯 推荐学习顺序

```
入门 → 机器人学 → ROS → 仿真 → 感知 → 规划 → 大模型 → 实践
```

1. **先学 01 和 02** - 打好理论基础
2. **再学 04** - 掌握开发工具
3. **然后 05-07** - 动手实践
4. **接着 08-12** - 深入智能
5. **最后 14-16** - 前沿与综合
