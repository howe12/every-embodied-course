## 🚀 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/howe12/every-embodied-course.git
cd every-embodied-course

# 2. 安装ROS2 (以Humble为例)
sudo apt update
sudo apt install ros-humble-desktop

# 3. 运行第一个示例
ros2 run turtlesim turtlesim_node
```
