## 🚀 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/howe12/every-embodied-course.git
cd every-embodied-course

# 2. 安装ROS2 (以Humble为例)
sudo apt update
sudo apt install ros-humble-desktop

# 3. 运行第一个示例
ros2 run turtlesim turtlesim_node
```

## 📖 学习路径

本教程分为多个阶段，带你逐层深入具身智能：

| 阶段 | 核心技能 | 产出 |
| :--- | :--- | :--- |
| **第一阶段** | 机器人基础、ROS2入门、Linux操作 | 搭建ROS2开发环境 |
| **第二阶段** | 工作空间、构建工具、常用工具 | 独立开发ROS2功能包 |
| **第三阶段** | 硬件控制、传感器选型 | 机器人硬件实战 |
| **第四阶段** | 计算机视觉、深度学习 | 视觉感知能力 |
| **第五阶段** | 强化学习、策略优化 | 智能决策系统 |
| **第六阶段** | VLA大模型、前沿研究 | 具身智能进阶 |
