## 1. 语音合成概述

### 1.1 什么是语音合成（TTS）

**语音合成**（Text-to-Speech，TTS）是将文本转换为自然语音的技术。它是语音交互系统的"嘴巴"，让机器人能够"说话"。与语音识别（ASR）相反，TTS接收文本输入，输出音频波形。

```
文本输入 → [TTS系统] → 音频波形输出
```

TTS系统的典型应用场景：

| 应用场景 | 示例 | 情绪需求 |
|----------|------|----------|
| 智能助手 | "今天的天气是晴天" | 友好、中性 |
| 导航播报 | "前方500米左转" | 清晰、正式 |
| 有声读物 | 朗读小说章节 | 多样化、有感情 |
| 机器人交互 | "我听到了，请稍等" | 活泼、亲切 |
| 无障碍辅助 | 屏幕阅读器 | 清晰、连贯 |

### 1.2 TTS 发展历史

语音合成技术经历了三个主要阶段：

| 时代 | 技术路线 | 合成质量 | 典型方法 |
|------|----------|----------|----------|
| **拼接合成** | 录制真人语音片段，按音素/音节拼接 | ★★★☆☆ | 单元选择、双音素HMM |
| **参数合成** | 统计模型预测声学参数，再生成语音 | ★★☆☆☆ | HMM参数合成、CLARET |
| **神经网络合成** | 深度学习端到端生成高质量语音 | ★★★★★ | WaveNet、Tacotron、VITS、FastSpeech |

**发展脉络详解：**

1. **拼接合成时代（1980s-2000s）**
   - 预先录制大规模语音库
   - 根据文本选择最匹配的语音片段拼接
   - 优点：音质高、真实人声感
   - 缺点：需要大量存储、无法灵活变调、拼接处可能不自然

2. **参数合成时代（2000s-2010s）**
   - 使用统计模型（如HMM）预测声学参数
   - 通过声码器将参数转换为波形
   - 优点：体积小、灵活可控
   - 缺点：音质较差、机械感强

3. **神经网络合成时代（2017至今）**
   - 2017年：WaveNet 提出使用扩散模型生成原始音频
   - 2018年：Tacotron2 实现端到端梅尔频谱生成
   - 2019年：FastSpeech 实现并行生成
   - 2020年：VITS 结合VAE和Flow实现高质量端到端合成
   - 2023年：GPT-SoVITS 实现零样本语音克隆
   - 优点：音质极高、自然度高、支持情绪控制
   - 缺点：计算量大、实时性挑战

### 1.3 端到端 TTS 原理

端到端TTS模型直接从文本映射到音频波形，核心架构分为三部分：

```
文本 → [文本分析模块] → [声学模型/梅尔解码器] → [声码器] → 音频波形
```

**文本分析模块（Text Analysis）**
- 文本正则化：将数字、缩写、符号转为可读词
- 分词：中文分词、英文分词
- 韵律预测：音素时长、重音、停顿

**声学模型（Acoustic Model）**
- 输入：文本/音素序列
- 输出：梅尔频谱（Mel Spectrogram）
- 代表模型：FastSpeech2、Transformer-TTS

**声码器（Vocoder）**
- 输入：梅尔频谱
- 输出：原始音频波形
- 代表模型：HiFi-GAN、WaveNet、Parallel WaveGAN

### 1.4 TTS 评估指标

| 指标 | 全称 | 描述 | 理想值 |
|------|------|------|--------|
| **MOS** | Mean Opinion Score | 平均主观意见分，1-5分人工评分 | 4.0+ |
| **实时率** | RTF (Real-Time Factor) | 生成1秒音频所需时间，越小越好 | < 0.5 |
| **首包延迟** | TTFT (Time to First Token) | 开始播放前的等待时间 | < 500ms |
| **CMOS** | Comparative MOS | 相对参考系统的质量差异 | > 0 |
| **PESQ** | Perceptual Evaluation of Speech Quality | 感知语音质量评估 | > 3.0 |

### 2.1 文本分析（Text Analysis）

文本分析是TTS流水线第一步，负责将原始文本转换为模型可处理的表示。

**2.1.1 文本正则化（Text Normalization, TN）**

将非标准词转换为标准文字形式：

```python
# 文本正则化示例
def text_normalize(text):
    """将文本中的数字、符号转换为可读形式"""
    import re
    
    # 电话号码正则化
    text = re.sub(r'(\d{3})-(\d{4})-(\d{4})', r'\1杠\2杠\3', text)
    
    # 数字正则化（简化示例）
    text = re.sub(r'(\d+)', lambda m: num2words(int(m.group(1))), text)
    
    # 货币符号
    text = text.replace('$', '美元')
    text = text.replace('¥', '人民币')
    
    # 缩写展开
    abbreviations = {'U.S.': '美国', 'Mr.': '先生', 'Dr.': '博士'}
    for abbr, full in abbreviations.items():
        text = text.replace(abbr, full)
    
    return text

# 示例
text = "我在U.S.花了$100买Dr.Smith的书"
normalized = text_normalize(text)
print(normalized)  # 输出: 我在美国花了美元100买博士Smith的书
```

### 2.2 声学模型（Acoustic Model）

声学模型将文本/音素序列转换为梅尔频谱（Mel Spectrogram），这是视觉化声音的核心表示。

**2.2.1 梅尔频谱基础**

梅尔频谱是符合人类听觉特性的频谱表示：

```python
import numpy as np
import librosa
import matplotlib.pyplot as plt

def compute_mel_spectrogram(audio_path=None, sr=22050, n_mels=80):
    """
    计算梅尔频谱
    参数:
        audio_path: 音频文件路径（可选，直接用示例信号演示）
        sr: 采样率
        n_mels: 梅尔滤波器组数量
    返回:
        mel_spectrogram: 梅尔频谱矩阵 [n_mels, time_steps]
    """
    # 如果有音频文件则加载，否则生成示例信号
    if audio_path:
        audio, sr = librosa.load(audio_path, sr=sr)
    else:
        # 生成440Hz正弦波（标准音A）作为示例
        t = np.linspace(0, 2, sr * 2)
        audio = 0.5 * np.sin(2 * np.pi * 440 * t)
    
    # 计算短时傅里叶变换（STFT）
    D = np.abs(librosa.stft(audio, n_fft=2048, hop_length=512))
    
    # 转换为梅尔频谱
    mel_spec = librosa.feature.melspectrogram(
        S=D, sr=sr, n_mels=n_mels,
        fmin=0, fmax=8000,  # 人类语音频率范围
        hop_length=512, n_fft=2048
    )
    
    # 转换为对数刻度（dB单位）
    mel_spec_db = librosa.power_to_db(mel_spec, ref=np.max)
    
    print(f"梅尔频谱形状: {mel_spec_db.shape}")
    print(f"时间步数: {mel_spec_db.shape[1]}, 频率 bins: {mel_spec_db.shape[0]}")
    
    return mel_spec_db

# 演示
mel = compute_mel_spectrogram()
```

**2.2.2 FastSpeech2 模型原理**

FastSpeech2 是微软提出的并行TTS模型，核心创新是引入可变性能量（Variance Adapter）：

```
音素序列
    ↓
[嵌入层] → [音素编码器 (Transformer)] → [解码器] → 梅尔频谱
    ↓
[可变性能量适配器]
    ├── 持续时间预测器（Duration Predictor）
    ├── 音高预测器（Pitch Predictor）
    └── 能量预测器（Energy Predictor）
```

```python
# FastSpeech2 核心原理伪代码
class FastSpeech2Model(nn.Module):
    """FastSpeech2 简化实现原理"""
    
    def __init__(self, vocab_size, d_model=256, n_heads=4, n_layers=4):
        super().__init__()
        # 音素嵌入
        self.phoneme_embedding = nn.Embedding(vocab_size, d_model)
        self.positional_encoding = PositionalEncoding(d_model)
        
        # 音素编码器
        self.encoder = TransformerEncoder(n_layers, d_model, n_heads)
        
        # 可变性能量适配器
        self.duration_predictor = DurationPredictor(d_model)
        self.pitch_predictor = PitchPredictor(d_model)
        self.energy_predictor = EnergyPredictor(d_model)
        
        # 梅尔频谱解码器
        self.decoder = TransformerDecoder(n_layers, d_model, n_heads)
        
        # 线性投影到梅尔频谱
        self.mel_projection = nn.Linear(d_model, 80)  # 80个梅尔频段
    
    def forward(self, phoneme_ids, target_pitch=None, target_energy=None, 
                target_duration=None, target_mel=None, use_predicted=True):
        """
        前向传播
        参数:
            phoneme_ids: 音素ID序列 [batch, seq_len]
            use_predicted: 是否使用预测值（训练时用真实值，推理时用预测值）
        """
        # 1. 音素编码
        x = self.phoneme_embedding(phoneme_ids)
        x = self.positional_encoding(x)
        x = self.encoder(x)
        
        # 2. 预测可变性能量
        predicted_duration = self.duration_predictor(x)  # [batch, seq_len]
        predicted_pitch = self.pitch_predictor(x)        # [batch, seq_len]
        predicted_energy = self.energy_predictor(x)      # [batch, seq_len]
        
        # 3. 长度调节（通过重复音素来匹配目标长度）
        if use_predicted:
            duration = predicted_duration
        else:
            duration = target_duration
        
        # 长度调节：重复每个音素 duration 次
        x = self.length_regulator(x, duration)
        
        # 4. 注入韵律信息
        if use_predicted:
            pitch = predicted_pitch
            energy = predicted_energy
        else:
            pitch = target_pitch
            energy = target_energy
        
        x = x + pitch + energy  # 注入韵律特征
        
        # 5. 解码生成梅尔频谱
        x = self.decoder(x)
        mel_output = self.mel_projection(x)
        
        return {
            'mel_spectrogram': mel_output,
            'predicted_duration': predicted_duration,
            'predicted_pitch': predicted_pitch,
            'predicted_energy': predicted_energy
        }
    
    def length_regulator(self, x, duration):
        """长度调节器：将音素序列根据持续时间展开"""
        # duration: [batch, seq_len]，表示每个音素重复次数
        batch_size, max_len = duration.shape
        
        # 展开操作
        output = []
        for b in range(batch_size):
            expanded = []
            for i, d in enumerate(duration[b]):
                # 将第i个音素重复d次
                expanded.append(x[b, i].repeat(int(d.item()), 1))
            output.append(torch.cat(expanded, dim=0))
        
        # 填充到同一长度
        max_out_len = max(o.shape[0] for o in output)
        padded = []
        for o in output:
            if o.shape[0] < max_out_len:
                pad = torch.zeros(max_out_len - o.shape[0], o.shape[1], device=o.device)
                o = torch.cat([o, pad], dim=0)
            padded.append(o)
        
        return torch.stack(padded)
```

### 2.3 声码器（Vocoder）

声码器将梅尔频谱转换为原始音频波形，是TTS的最后一步。

**2.3.1 声码器对比**

| 声码器 | 类型 | 优点 | 缺点 | 质量评分 |
|--------|------|------|------|----------|
| **WaveNet** | 自回归 | 音质极高 | 速度慢 | ★★★★★ |
| **HiFi-GAN** | GAN | 高速、高质量 | 偶有伪影 | ★★★★☆ |
| **Parallel WaveGAN** | Flow | 并行生成 | 质量一般 | ★★★☆☆ |
| **Griffin-Lim** | 传统 | 速度快 | 音质差 | ★★☆☆☆ |

**2.3.2 HiFi-GAN 原理**

HiFi-GAN 是目前最流行的开源声码器，使用GAN结构实现高保真音频生成：

```python
# HiFi-GAN 核心原理伪代码
class HiFiGANGenerator(nn.Module):
    """HiFi-GAN 生成器：将梅尔频谱转换为波形"""
    
    def __init__(self, mel_channels=80, resblock_kernel_sizes=[3, 7, 11],
                 upsample_rates=[8, 8, 2, 2]):
        super().__init__()
        
        self.mel_channels = mel_channels
        
        # 初始卷积：将梅尔频谱映射到隐藏维度
        self.conv_pre = nn.Conv1d(mel_channels, 512, kernel_size=7, padding=3)
        
        # 上采样层（逐步将时序长度扩大）
        self.upsamples = nn.ModuleList()
        self.resblocks = nn.ModuleList()
        
        for i, (rate, kernel_size) in enumerate(zip(upsample_rates, [16, 16, 4, 4])):
            self.upsamples.append(
                nn.ConvTranspose1d(512 // (2**i), 512 // (2**(i+1)),
                                  kernel_size=rate * 2, stride=rate, padding=rate // 2)
            )
            # 添加多分辨率残差块
            for k in resblock_kernel_sizes:
                self.resblocks.append(ResBlock(k, 512 // (2**(i+1))))
        
        # 最终卷积
        self.conv_post = nn.Conv1d(256, 1, kernel_size=7, padding=3)
        
        # 激活函数
        self.leaky_relu = nn.LeakyReLU(0.2)
    
    def forward(self, mel):
        """
        前向传播
        参数:
            mel: 梅尔频谱 [batch, 80, time_steps]
        返回:
            audio: 生成的波形 [batch, 1, time_steps * upsample_rate]
        """
        x = self.conv_pre(mel)  # [B, 512, T]
        x = self.leaky_relu(x)
        
        for i, upsample in enumerate(self.upsamples):
            x = upsample(x)  # 上采样
            x = self.leaky_relu(x)
            
            # 应用多个残差块
            xs = None
            for j in range(len(self.resblocks)):
                if xs is None:
                    xs = self.resblocks[j * len(self.upsamples) + i](x)
                else:
                    xs += self.resblocks[j * len(self.upsamples) + i](x)
            x = xs / len(self.resblocks)
        
        x = self.conv_post(x)
        x = torch.tanh(x)  # 输出范围 [-1, 1]
        
        return x


class ResBlock(nn.Module):
    """残差块，用于HiFi-GAN的多分辨率特征提取"""
    
    def __init__(self, kernel_size, channels):
        super().__init__()
        self.convs = nn.ModuleList([
            nn.Conv1d(channels, channels, kernel_size, padding=kernel_size//2)
            for _ in range(2)
        ])
    
    def forward(self, x):
        for conv in self.convs:
            x = self.leaky_relu(conv(x))
        return x + x  # 残差连接
```

**2.3.3 音频播放**

```python
import sounddevice as sd
import numpy as np
import librosa

def play_audio(audio_data, sr=22050):
    """
    播放音频数据
    参数:
        audio_data: 音频数组（numpy数组）
        sr: 采样率

def save_audio(audio_data, sr, filename):
    """
    保存音频到文件
    参数:
        audio_data: 音频数组
        sr: 采样率
        filename: 保存路径（支持 .wav, .mp3 等格式）
    """
    import scipy.io.wavfile as wavfile
    
    # 转换为 int16 格式
    audio_int16 = (audio_data * 32767).astype(np.int16)
    
    wavfile.write(filename, sr, audio_int16)
    print(f"音频已保存到: {filename}")
```

### 4.1 情绪语音合成概述

情绪语音合成（Emotional TTS）是让合成语音带有情感色彩的技术，对于具身智能机器人的交互体验至关重要。

**为什么情绪很重要：**

| 情绪 | 应用场景 | 用户体验提升 |
|------|----------|-------------|
| **开心/喜悦** | 欢迎语、表扬 | 友好、亲切 |
| **悲伤** | 安慰、遗憾表达 | 共情、真实 |
| **愤怒** | 警告、紧急提醒 | 紧迫、严肃 |
| **惊讶** | 意外情况提示 | 自然、真实 |
| **中性** | 日常对话、信息播报 | 专业、清晰 |

### 4.2 情绪控制参数

情绪语音合成的核心是通过调节声学参数来表达不同情绪：

```python
import numpy as np
import librosa

class EmotionTTSController:
    """情绪TTS控制器 - 通过参数调节实现情绪表达"""
    
    # 不同情绪对应的声学参数范围
    EMOTION_PARAMS = {
        'happy': {
            'pitch_shift': 2.0,      # 音高上移（半音）
            'speed': 1.15,           # 语速加快
            'energy': 1.2,           # 能量/音量提升
            'pause_insert': 0.3,     # 句间停顿
            'pitch_range': 1.3,      # 音高变化范围扩大
        },
        'sad': {
            'pitch_shift': -2.0,     # 音高下移
            'speed': 0.85,           # 语速减慢
            'energy': 0.8,           # 能量降低
            'pause_insert': 0.5,     # 停顿增多
            'pitch_range': 0.7,      # 音高变化范围缩小
        },
        'angry': {
            'pitch_shift': 1.5,      # 音高略微上移
            'speed': 1.3,            # 语速加快
            'energy': 1.4,           # 能量显著提升
            'pause_insert': 0.1,     # 停顿减少
            'pitch_range': 1.5,      # 音高变化剧烈
        },
        'surprised': {
            'pitch_shift': 3.0,      # 音高大幅上移
            'speed': 1.1,            # 语速略快
            'energy': 1.3,           # 能量提升
            'pause_insert': 0.2,     # 短停顿
            'pitch_range': 1.8,     # 音高变化大
        },
        'neutral': {
            'pitch_shift': 0.0,     # 无变化
            'speed': 1.0,            # 正常语速
            'energy': 1.0,           # 正常能量
            'pause_insert': 0.25,   # 正常停顿
            'pitch_range': 1.0,     # 正常音高变化
        }
    }
    
    def __init__(self, tts_model):
        """
        初始化情绪控制器
        参数:
            tts_model: 底层TTS模型（Coqui TTS等）
        """
        self.tts = tts_model
    
    def apply_emotion(self, audio, emotion='neutral'):
        """
        对音频应用情绪参数
        参数:
            audio: 原始音频 (numpy array)
            emotion: 情绪类型
        返回:
            emotional_audio: 处理后的音频
        """
        params = self.EMOTION_PARAMS.get(emotion, self.EMOTION_PARAMS['neutral'])
        sr = 22050  # 假设采样率
        
        # 1. 音高调整（使用 librosa）
        if params['pitch_shift'] != 0:
            # 将半音转换为音程比例
            pitch_factor = 2 ** (params['pitch_shift'] / 12)
            audio = self._pitch_shift(audio, pitch_factor, sr)
        
        # 2. 语速调整
        if params['speed'] != 1.0:
            audio = librosa.effects.time_stretch(audio, rate=params['speed'])
        
        # 3. 能量/音量调整
        if params['energy'] != 1.0:
            audio = audio * params['energy']
            # 防止削波
            audio = np.clip(audio, -1.0, 1.0)
        
        # 4. 平滑处理（避免突兀变化）
        audio = self._smooth_audio(audio)
        
        return audio
    
    def _pitch_shift(self, audio, factor, sr):
        """音高调整"""
        # 使用 librosa 进行音高调整
        audio = librosa.effects.pitch_shift(audio, sr=sr, n_steps=factor)
        return audio
    
    def _smooth_audio(self, audio, window_size=5):
        """平滑音频，避免突兀变化"""
        from scipy.ndimage import uniform_filter1d
        smoothed = uniform_filter1d(audio, size=window_size)
        return smoothed.astype(audio.dtype)
    
    def synthesize_emotional(self, text, emotion='neutral', output_path=None):
        """
        合成带情绪的语音
        参数:
            text: 输入文本
            emotion: 情绪类型
            output_path: 输出文件路径
        返回:
            audio: 音频数据
        """
        # 1. 使用底层TTS生成基础音频
        base_audio = self.tts.tts(text)
        
        # 2. 应用情绪参数
        emotional_audio = self.apply_emotion(base_audio, emotion)
        
        # 3. 保存（如果指定路径）
        if output_path:
            import soundfile as sf
            sf.write(output_path, emotional_audio, 22050)
        
        return emotional_audio


def demo_emotion_tts():
    """情绪TTS演示"""
    from TTS.api import TTS
    
    # 初始化TTS
    tts = TTS(model_name="tts_models/zh-CN/baker/tacotron2-DDC-GST",
```

```python
def adaptive_emotion_transfer():
    """自适应情绪迁移 - 根据文本内容自动判断情绪"""
    
    # 简单的情绪词匹配规则
    EMOTION_KEYWORDS = {

    }

                    emotion_scores[emotion] += 1

        max_emotion = max(emotion_scores, key=emotion_scores.get)
        return max_emotion if emotion_scores[max_emotion] > 0 else 'neutral'
```

```python
        self.default_pitch = self.get_parameter('default_pitch').value
        self.sample_rate = self.get_parameter('sample_rate').value
        
        # 初始化TTS模型
        self.get_logger().info(f'正在初始化TTS模型: {model_name}')
        try:
            self.tts = TTS(model_name=model_name, progress_bar=False)
            self.get_logger().info('TTS模型初始化成功!')
        except Exception as e:
            self.get_logger().error(f'TTS模型初始化失败: {e}')
            self.tts = None
        
        # QoS设置
        qos = QoSProfile(depth=10, reliability=ReliabilityPolicy.RELIABLE)
        
        # 订阅者：接收待合成的文本
        self.sub_text = self.create_subscription(
            String,
            '/tts/text',
            self.text_callback,
            qos
        )
        
        # 订阅者：接收带情绪参数的文本
        self.sub_text_emotion = self.create_subscription(
            String,
            '/tts/text_emotion',
            self.text_emotion_callback,
            qos
        )
        
        # 订阅者：接收控制命令
        self.sub_cmd = self.create_subscription(
            String,
            '/tts/command',
            self.command_callback,
            qos
        )
        
        # 发布者：发布合成的音频数据
        self.pub_audio = self.create_publisher(
            AudioData,
            '/tts/audio',
            qos
        )
        
        # 发布者：发布合成状态
        self.pub_status = self.create_publisher(
            String,
            '/tts/status',
            qos
        )
        
        # 发布者：发布情绪状态
        self.pub_emotion = self.create_publisher(
            String,
            '/tts/emotion_state',
            qos
        )
        
        # 音频播放锁（防止多线程冲突）
        self.play_lock = threading.Lock()
        
        # 当前情绪状态
        self.current_emotion = self.default_emotion
        
        self.get_logger().info('TTS节点已就绪')
    
    def text_callback(self, msg):
        """处理文本合成请求"""
        text = msg.data
        self.get_logger().info(f'收到合成请求: {text[:50]}...' if len(text) > 50 else f'收到合成请求: {text}')
        
        # 使用默认参数合成
        self.synthesize_and_publish(text, emotion=self.current_emotion)
    
    def text_emotion_callback(self, msg):
        """处理带情绪参数的文本请求（JSON格式）"""
        import json
        try:
            data = json.loads(msg.data)
            text = data.get('text', '')
            emotion = data.get('emotion', self.current_emotion)
            speed = data.get('speed', self.default_speed)
            pitch = data.get('pitch', self.default_pitch)
            
            self.get_logger().info(f'收到情绪合成请求: {text[:30]}... [情绪:{emotion}]')
            
            self.synthesize_and_publish(text, emotion=emotion, speed=speed, pitch=pitch)
        except json.JSONDecodeError as e:
            self.get_logger().error(f'JSON解析失败: {e}')
    
    def command_callback(self, msg):
        """处理控制命令"""
        cmd = msg.data.lower().strip()
        
        if cmd == 'stop':
            self.get_logger().info('停止当前播放')
            # 发送停止命令（由播放节点处理）
            stop_msg = String()
            stop_msg.data = 'stop'
            # 通过发布空音频触发停止
            
        elif cmd.startswith('emotion:'):
            emotion = cmd.split(':')[1]
            self.set_emotion(emotion)
        
        elif cmd == 'list_emotions':
            emotions = ['neutral', 'happy', 'sad', 'angry', 'surprised']
            self.get_logger().info(f'可用情绪: {emotions}')
    
    def synthesize_and_publish(self, text, emotion='neutral', speed=1.0, pitch=0.0):
        """
        合成语音并发布
        参数:
            text: 输入文本
            emotion: 情绪类型
            speed: 语速
            pitch: 音高
        """
        if self.tts is None:
            self.get_logger().error('TTS模型未初始化')
            return
        
        try:
            # 合成音频
            audio_data = self.tts.tts(text)
            
            # 应用情绪参数
            audio_data = self.apply_audio_modifications(
                audio_data, emotion, speed, pitch
            )
            
            # 转换为字节数据
            audio_bytes = self.audio_to_bytes(audio_data)
            
            # 发布音频数据
            audio_msg = AudioData()
            audio_msg.data = audio_bytes
            self.pub_audio.publish(audio_msg)
            
            # 发布情绪状态
            emotion_msg = String()
            emotion_msg.data = emotion
            self.pub_emotion.publish(emotion_msg)
            
            # 发布完成状态
            status_msg = String()
            status_msg.data = 'done'
            self.pub_status.publish(status_msg)
            
            self.get_logger().info(f'合成完成 [情绪:{emotion}, 时长:{len(audio_data)/self.sample_rate:.2f}s]')
            
        except Exception as e:
            self.get_logger().error(f'合成失败: {e}')
            status_msg = String()
            status_msg.data = f'error: {str(e)}'
            self.pub_status.publish(status_msg)
    
    def apply_audio_modifications(self, audio, emotion, speed, pitch):
        """应用音频修改（情绪、语速、音高）"""
        import librosa
        
        # 1. 音高调整
        if pitch != 0:
            n_steps = pitch / 12 * 2  # 将参数转换为半音
            audio = librosa.effects.pitch_shift(audio, sr=self.sample_rate, n_steps=n_steps)
        
        # 2. 语速调整
        if speed != 1.0:
            audio = librosa.effects.time_stretch(audio, rate=speed)
        
        # 3. 情绪能量调整（简化版）
        emotion_energy = {
            'happy': 1.2, 'sad': 0.8, 'angry': 1.3,
            'surprised': 1.1, 'neutral': 1.0
        }
        energy_factor = emotion_energy.get(emotion, 1.0)
        audio = np.clip(audio * energy_factor, -1.0, 1.0)
        
        return audio
    
    def audio_to_bytes(self, audio_data):
        """将numpy数组转换为字节数据"""
        # 转换为float32
        if audio_data.dtype != np.float32:
            audio_data = audio_data.astype(np.float32)
        
        # 转换为字节
        return audio_data.tobytes()
    
    def set_emotion(self, emotion):
        """设置当前情绪"""
        valid_emotions = ['neutral', 'happy', 'sad', 'angry', 'surprised']
        if emotion in valid_emotions:
            self.current_emotion = emotion
            self.get_logger().info(f'情绪已设置为: {emotion}')
            
            # 发布情绪变化
            msg = String()
            msg.data = emotion
            self.pub_emotion.publish(msg)
        else:
            self.get_logger().warn(f'无效的情绪类型: {emotion}')


def main(args=None):
    """主函数"""
    rclpy.init(args=args)
    
    node = TTSNode()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        node.get_logger().info('TTS节点关闭')
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
# ros2_audio_player_node.py
# ROS2 音频播放节点

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import AudioData
from std_msgs.msg import String, Bool
from rclpy.qos import QoSProfile, ReliabilityPolicy
import numpy as np
import sounddevice as sd
import threading

class AudioPlayerNode(Node):
    """ROS2 音频播放节点"""
    
    def __init__(self):
        super().__init__('audio_player_node')
        
        # 音频播放参数
        self.declare_parameter('sample_rate', 22050)
        self.declare_parameter('channels', 1)
        self.declare_parameter('dtype', 'float32')
        self.declare_parameter('block_size', 1024)
        
        self.sample_rate = self.get_parameter('sample_rate').value
        self.channels = self.get_parameter('channels').value
        
        # 播放状态
        self.is_playing = False
        self.audio_buffer = []
        self.play_lock = threading.Lock()
        self.current_stream = None
        
        # QoS设置
        qos = QoSProfile(depth=10, reliability=ReliabilityPolicy.RELIABLE)
        
        # 订阅者：接收音频数据
        self.sub_audio = self.create_subscription(
            AudioData,
            '/tts/audio',
            self.audio_callback,
            qos
        )
        
        # 订阅者：接收控制命令
        self.sub_cmd = self.create_subscription(
            String,
            '/tts/command',
            self.cmd_callback,
            qos
        )
        
        # 发布者：发布播放状态
        self.pub_status = self.create_publisher(
            String,
            '/audio_player/status',
            qos
        )
        
        # 初始化音频流
        self.init_audio_stream()
        
        self.get_logger().info(f'音频播放节点已就绪 (采样率:{self.sample_rate}Hz, 通道:{self.channels})')
    
    def init_audio_stream(self):
        """初始化音频流"""
        try:
            # 查询可用设备
            devices = sd.query_devices()
            self.get_logger().info(f'可用音频设备: {len(devices)}个')
            
            # 打印默认输出设备
            default_device = sd.query_devices(kind='output')
            self.get_logger().info(f'默认输出设备: {default_device["name"]}')
            
        except Exception as e:
            self.get_logger().error(f'音频设备初始化失败: {e}')
    
    def audio_callback(self, msg):
        """处理接收到的音频数据"""
        with self.play_lock:
            # 将字节数据转换为numpy数组
            audio_data = np.frombuffer(msg.data, dtype=np.float32)
            
            if len(audio_data) > 0:
                self.audio_buffer.extend(audio_data.tolist())
                
                # 如果当前没有在播放，开始播放
                if not self.is_playing:
                    self.start_playback()
    
    def start_playback(self):
        """开始播放音频"""
        if not self.audio_buffer or self.is_playing:
            return
        
        self.is_playing = True
        
        # 在单独线程中播放（避免阻塞ROS）
        playback_thread = threading.Thread(target=self._playback_thread)
        playback_thread.daemon = True
        playback_thread.start()
    
    def _playback_thread(self):
        """播放线程"""
        with self.play_lock:
            audio_array = np.array(self.audio_buffer, dtype=np.float32)
            self.audio_buffer.clear()
        
        try:
            # 发布开始播放状态
            status_msg = String()
            status_msg.data = 'playing'
            self.pub_status.publish(status_msg)
            
            # 使用sounddevice播放
            sd.play(audio_array, self.sample_rate, channels=self.channels)
            sd.wait()  # 等待播放完成
            
            # 发布播放完成状态
            status_msg.data = 'done'
            self.pub_status.publish(status_msg)
            
        except Exception as e:
            self.get_logger().error(f'播放失败: {e}')
        finally:
            with self.play_lock:
                self.is_playing = False
    
    def cmd_callback(self, msg):
        """处理控制命令"""
        cmd = msg.data.lower().strip()
        
        if cmd == 'stop':
            self.stop_playback()
        elif cmd == 'pause':
            self.pause_playback()
        elif cmd == 'resume':
            self.resume_playback()
    
    def stop_playback(self):
        """停止播放"""
        with self.play_lock:
            self.audio_buffer.clear()
            if self.is_playing:
                sd.stop()
                self.is_playing = False
                self.get_logger().info('播放已停止')
    
    def pause_playback(self):
        """暂停播放"""
        if self.is_playing:
            sd.stop()
            self.get_logger().info('播放已暂停')
    
    def resume_playback(self):
        """恢复播放"""
        if not self.is_playing and self.audio_buffer:
            self.start_playback()


def main(args=None):
    rclpy.init(args=args)
    
    player = AudioPlayerNode()
    
    try:
        rclpy.spin(player)
    except KeyboardInterrupt:
        player.get_logger().info('音频播放节点关闭')
    finally:
        player.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
# tts_complete_example.py
# Coqui TTS 完整使用示例

import os
import time
import numpy as np
import sounddevice as sd
import soundfile as sf
from TTS.api import TTS

class CoquiTTSDemo:
    """Coqui TTS 演示类"""
    
    def __init__(self):
        """初始化TTS"""
        print("=" * 60)
        print("Coqui TTS 演示程序")
        print("=" * 60)
        
        # 初始化TTS（使用中文模型）
        self.tts = TTS(
            model_name="tts_models/zh-CN/baker/tacotron2-DDC-GST",
            progress_bar=False,
            verbose=False
        )
        print("✓ TTS模型加载成功")
    
    def synthesize_and_play(self, text, emotion='neutral'):
        """合成并播放"""
        print(f"\n[合成] 文本: {text}")
        print(f"[参数] 情绪: {emotion}")
        
        start_time = time.time()
        
        # 合成
        audio = self.tts.tts(text)
        
        # 应用情绪
        audio = self.apply_emotion_simple(audio, emotion)
        
        elapsed = time.time() - start_time
        print(f"[完成] 耗时: {elapsed:.3f}s, 时长: {len(audio)/22050:.2f}s")
        
        # 播放
        print("[播放] ", end='', flush=True)
        sd.play(audio, 22050)
        
        # 实时反馈
        for _ in range(int(len(audio) / 22050)):
            print("▮", end='', flush=True)
            time.sleep(1)
        print(" ▯")
        
        sd.wait()
        print("[停止] 播放完成")
        
        return audio
    
    def apply_emotion_simple(self, audio, emotion):
        """简单的情绪应用（避免导入librosa时的复杂依赖）"""
        import numpy as np
        
        emotion_map = {
            'happy': {'speed': 1.15, 'gain': 1.1},
            'sad': {'speed': 0.85, 'gain': 0.8},
            'angry': {'speed': 1.3, 'gain': 1.3},
            'surprised': {'speed': 1.1, 'gain': 1.2},
            'neutral': {'speed': 1.0, 'gain': 1.0},
        }
        
        params = emotion_map.get(emotion, emotion_map['neutral'])
        
        # 简单的速度调整（重采样）
        if params['speed'] != 1.0:
            # 使用简单的线性插值进行速度调整
            n_samples = int(len(audio) / params['speed'])
            indices = np.linspace(0, len(audio) - 1, n_samples)
            audio = np.interp(indices, np.arange(len(audio)), audio).astype(np.float32)
        
        # 音量调整
        audio = np.clip(audio * params['gain'], -1.0, 1.0)
        
        return audio
    
    def save_to_file(self, audio, filename):
        """保存音频到文件"""
        sf.write(filename, audio, 22050)
        print(f"✓ 已保存: {filename}")
    
    def batch_synthesis(self, texts, output_dir="outputs/"):
        """批量合成"""
        os.makedirs(output_dir, exist_ok=True)
        
        print("\n" + "=" * 60)
        print(f"批量合成: {len(texts)} 条文本")
        print("=" * 60)
        
        for i, text in enumerate(texts):
            audio = self.tts.tts(text)
            filename = os.path.join(output_dir, f"output_{i:03d}.wav")
            self.save_to_file(audio, filename)
            
            # 每5条播报一次进度
            if (i + 1) % 5 == 0:
                print(f"进度: {i+1}/{len(texts)}")
        
        print("\n✓ 批量合成完成!")


def main():

    demo = CoquiTTSDemo()
    
    # 示例1: 基础合成
    demo.synthesize_and_play("你好，我是具身智能机器人，很高兴认识你！")
    
    # 示例2: 不同情绪
    emotions = ['happy', 'sad', 'angry', 'surprised', 'neutral']
    for emotion in emotions:
        demo.synthesize_and_play("今天我们有很多任务要完成！", emotion=emotion)
    
    # 示例3: 批量合成
    batch_texts = [
        "第一段语音合成测试文本。",
        "第二段用于测试批量合成功能。",
        "第三段展示机器人的语音交互能力。",
        "第四段说明语音合成在具身智能中的应用。",
        "第五段结束本次演示。",
    ]
    demo.batch_synthesis(batch_texts)
    
    print("\n" + "=" * 60)
```

```python
# audio_utils.py
# 音频保存与播放工具

import numpy as np
import soundfile as sf
import sounddevice as sd
import io

class AudioManager:
    """音频管理器"""
    
    def __init__(self, sample_rate=22050, channels=1):
        self.sample_rate = sample_rate
        self.channels = channels
    
    def save_wav(self, audio, filename):
        """保存为WAV文件"""
        # 确保音频格式正确
        if audio.dtype != np.float32:
            audio = audio.astype(np.float32)
        
        # 确保范围在[-1, 1]
        audio = np.clip(audio, -1.0, 1.0)
        
        sf.write(filename, audio, self.sample_rate)
        print(f"✓ 已保存: {filename} ({len(audio)/self.sample_rate:.2f}s)")
    
    def save_mp3(self, audio, filename, bitrate='128k'):
        """保存为MP3文件（需要ffmpeg）"""
        import subprocess
        
        # 先保存为临时WAV
        temp_wav = '__temp__.wav'
        self.save_wav(audio, temp_wav)
        
        # 转换为MP3
        subprocess.run([
            'ffmpeg', '-i', temp_wav,
            '-b:a', bitrate,
            '-y', filename
        ], capture_output=True)
        
        # 删除临时文件
        import os
        os.remove(temp_wav)
        
        print(f"✓ 已保存MP3: {filename}")
    
    def play(self, audio, blocking=True):
        """播放音频"""
        if audio.dtype != np.float32:
            audio = audio.astype(np.float32)
        
        audio = np.clip(audio, -1.0, 1.0)
        
        sd.play(audio, self.sample_rate, channels=self.channels)
        
        if blocking:
            sd.wait()
            print("✓ 播放完成")
    
    def play_with_progress(self, audio, progress_bar=True):
        """带进度显示的播放"""
        if audio.dtype != np.float32:
            audio = audio.astype(np.float32)
        
        audio = np.clip(audio, -1.0, 1.0)
        
        duration = len(audio) / self.sample_rate
        sd.play(audio, self.sample_rate)
        
        if progress_bar:
            import time
            start_time = time.time()
            while sd.get_stream().active:
                elapsed = time.time() - start_time
                progress = min(elapsed / duration, 1.0)
                bar_len = 30
                filled = int(bar_len * progress)
                bar = '[' + '█' * filled + '░' * (bar_len - filled) + ']'
                print(f"\r{bar} {progress*100:.1f}% ({elapsed:.1f}s/{duration:.1f}s)", end='', flush=True)
                time.sleep(0.1)
            print()
        
        print("✓ 播放完成")
    
    def bytes_to_audio(self, audio_bytes):
        """将字节数据转换为音频数组"""
        audio = np.frombuffer(audio_bytes, dtype=np.float32)
        return audio
    
    def audio_to_bytes(self, audio):
        """将音频数组转换为字节数据"""
        if audio.dtype != np.float32:
            audio = audio.astype(np.float32)
        return audio.tobytes()
    
    def resample(self, audio, target_sr):
        """重采样到目标采样率"""
        import librosa
        resampled = librosa.resample(audio, orig_sr=self.sample_rate, target_sr=target_sr)
        self.sample_rate = target_sr
        return resampled
```

```python
# ros2_voice_interaction_node.py
# 完整的语音交互ROS2节点

import rclpy
from rclpy.node import Node
from std_msgs.msg import String, Bool, Float32
from sensor_msgs.msg import AudioData
from geometry_msgs.msg import Twist
from rclpy.qos import QoSProfile, ReliabilityPolicy
import numpy as np
import threading
import time

class VoiceInteractionNode(Node):
    """ROS2 语音交互主节点"""
    
    def __init__(self):
        super().__init__('voice_interaction_node')
        
        # 状态管理
        self.is_listening = True
        self.is_speaking = False
        self.conversation_history = []
        self.current_intent = None
        
        # 初始化各模块
        self.init_asr()
        self.init_nlu()
        self.init_dialogue()
        self.init_tts()
        
        # QoS设置

        # 发布者

        self.pub_intent = self.create_publisher(String, '/voice/intent', qos)
        self.pub_response = self.create_publisher(String, '/voice/response', qos)
        self.pub_tts_text = self.create_publisher(String, '/tts/text', qos)
        self.pub_motion = self.create_publisher(Twist, '/cmd_vel', qos)
        self.pub_listen_state = self.create_publisher(Bool, '/voice/listening', qos)

            self.asr_model = whisper.load_model("base")
            self.get_logger().info('ASR模型加载成功')
        except Exception as e:

            self.asr_model = None
    
    def init_nlu(self):

        # 简单的意图识别规则
        self.intent_patterns = {
            'greeting': ['你好', '嗨', '早上好', '晚上好', 'hi', 'hello'],
            'move_forward': ['前进', '往前走', '向前走'],
            'move_backward': ['后退', '往后走', '后退'],
            'turn_left': ['左转', '向左转', '左拐'],
            'turn_right': ['右转', '向右转', '右拐'],

            'status': ['状态', '怎么样', '你好吗'],

        }
        self.get_logger().info('NLU模块初始化成功')
    
    def init_dialogue(self):
        """初始化对话管理"""
        self.dialogue_responses = {
            'greeting': ['你好！今天过得怎么样？', '嗨！有什么我可以帮你的吗？'],
            'move_forward': ['好的，我正在前进。', '好的，前进中...'],
            'move_backward': ['好的，正在后退。', '明白，后退中...'],
            'turn_left': ['好的，向左转。', '好的，左转...'],
            'turn_right': ['好的，向右转。', '好的，右转...'],
            'stop': ['好的，已停止。', '停止完成。'],
            'status': ['我的状态很好，谢谢关心！', '一切正常，随时待命！'],
            'bye': ['再见！有需要随时叫我。', '拜拜！'],
        }
        self.get_logger().info('对话模块初始化成功')
    
    def init_tts(self):
        """初始化TTS模块"""
        try:
            from TTS.api import TTS
            self.tts = TTS(model_name="tts_models/zh-CN/baker/tacotron2-DDC-GST",

        except Exception as e:
            self.get_logger().warn(f'TTS模块初始化失败: {e}')
            self.tts = None
    
    def text_input_callback(self, msg):
        """处理文本输入（ASR结果）"""
        text = msg.data.strip()
        if not text:
            return
        
        self.get_logger().info(f'识别文本: {text}')
        
        # 发布ASR文本
        asr_msg = String()
        asr_msg.data = text
        self.pub_asr_text.publish(asr_msg)
        
        # 处理对话
        self.process_input(text)
    
    def command_callback(self, msg):
        """处理外部命令"""
        cmd = msg.data.lower().strip()
        
        if cmd == 'start_listening':
            self.is_listening = True
            self.get_logger().info('开始监听')
        elif cmd == 'stop_listening':
            self.is_listening = False
            self.get_logger().info('停止监听')
        elif cmd == 'speak_test':
            self.speak('这是一段测试语音。')
    
    def listen_state_callback(self, msg):
        """处理监听状态变化"""
        self.is_listening = msg.data
    
    def process_input(self, text):
        """处理用户输入"""
        # 1. 意图识别
        intent = self.recognize_intent(text)
        self.current_intent = intent
        
        # 发布意图
        intent_msg = String()
        intent_msg.data = intent
        self.pub_intent.publish(intent_msg)
        
        # 2. 生成响应
        response = self.generate_response(intent, text)
        
        # 3. 执行动作（如果是动作类意图）
        self.execute_action(intent)
        
        # 4. 语音响应
        self.speak(response)
        
        # 5. 发布响应文本
        resp_msg = String()
        resp_msg.data = response
        self.pub_response.publish(resp_msg)
        
        # 6. 更新对话历史
        self.conversation_history.append({'user': text, 'bot': response, 'intent': intent})
    
    def recognize_intent(self, text):
        """识别用户意图"""
        text_lower = text.lower()
        
        max_match = 0
        best_intent = 'unknown'
        
        for intent, keywords in self.intent_patterns.items():
            for keyword in keywords:
                if keyword in text:
                    match_count = len(keyword)
                    if match_count > max_match:
                        max_match = match_count
                        best_intent = intent
        
        self.get_logger().info(f'识别意图: {best_intent}')
        return best_intent
    
    def generate_response(self, intent, text):
        """生成对话响应"""
        import random
        
        if intent in self.dialogue_responses:
            responses = self.dialogue_responses[intent]
            response = random.choice(responses)
        else:
            response = '抱歉，我不太理解你的意思。能再说一遍吗？'
        
        return response
    
    def execute_action(self, intent):
        """执行机器人动作"""
        twist = Twist()
        
        action_mapping = {
            'move_forward': (0.3, 0.0),   # (linear, angular)
            'move_backward': (-0.3, 0.0),
            'turn_left': (0.0, 0.5),
            'turn_right': (0.0, -0.5),
            'stop': (0.0, 0.0),
        }
        
        if intent in action_mapping:
            linear, angular = action_mapping[intent]
            twist.linear.x = linear
            twist.angular.z = angular
            self.pub_motion.publish(twist)
            
            # 3秒后停止
            if intent != 'stop':
                threading.Timer(3.0, lambda: self.pub_motion.publish(Twist())).start()
    
    def speak(self, text):
        """TTS语音播报"""
        if self.tts is None:
            self.get_logger().warn('TTS未初始化，跳过语音播报')
            return
        
        self.is_speaking = True
        
        try:
            # 生成音频
            audio = self.tts.tts(text)
            
            # 发布到TTS节点
            from sensor_msgs.msg import AudioData
            audio_msg = AudioData()
            audio_msg.data = audio.tobytes()
            
            # 注意：这里简化处理，实际应通过service调用
            tts_msg = String()
            tts_msg.data = text
            self.pub_tts_text.publish(tts_msg)
            
            self.get_logger().info(f'播报: {text}')
            
        except Exception as e:
            self.get_logger().error(f'TTS失败: {e}')
        finally:
            self.is_speaking = False
    
    def publish_status(self):
        """发布状态心跳"""
        state_msg = Bool()
        state_msg.data = self.is_listening
        self.pub_listen_state.publish(state_msg)


def main(args=None):
    rclpy.init(args=args)
    
    node = VoiceInteractionNode()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        node.get_logger().info('语音交互节点关闭')
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
# latency_optimizer.py
# 延迟优化工具

import time
import numpy as np
from functools import lru_cache
import hashlib

class TTSLatencyOptimizer:
    """TTS延迟优化器"""
    
    def __init__(self, tts_model):
        self.tts = tts_model
        self.cache = {}  # 文本 -> 音频缓存
        self.cache_size = 100  # 最大缓存数量
    
    def get_cache_key(self, text):
        """生成缓存键"""
        return hashlib.md5(text.encode()).hexdigest()
    
    def synthesize_cached(self, text):
        """带缓存的TTS合成"""
        cache_key = self.get_cache_key(text)
        
        if cache_key in self.cache:
            return self.cache[cache_key]
        
        # 合成
        audio = self.tts.tts(text)
        
        # 缓存
        if len(self.cache) < self.cache_size:
            self.cache[cache_key] = audio
        
        return audio
    
    def preload_phrases(self, phrases):
        """预加载常用短语"""
        print("预加载常用短语到缓存...")
        for phrase in phrases:
            self.synthesize_cached(phrase)
        print(f"✓ 已预加载 {len(phrases)} 个短语")
    
    def clear_cache(self):
        """清空缓存"""
        self.cache.clear()
        print("缓存已清空")


class StreamingTTS:
    """流式TTS（分段合成和播放）"""
    
    def __init__(self, tts_model, chunk_duration=0.5, sample_rate=22050):
        self.tts = tts_model
        self.chunk_duration = chunk_duration  # 每段时长（秒）
        self.sample_rate = sample_rate
        self.chunk_samples = int(chunk_duration * sample_rate)
    
    def synthesize_streaming(self, text, audio_callback):
        """
        流式合成文本
        参数:
            text: 输入文本
            audio_callback: 每生成一个片段就调用的回调函数
        """
        import sounddevice as sd
        
        # 将文本分成多个句子
        sentences = self.split_sentences(text)
        
        for sentence in sentences:
            if not sentence.strip():
                continue
            
            # 合成这个句子
            audio = self.tts.tts(sentence)
            
            # 分段发送
            for i in range(0, len(audio), self.chunk_samples):
                chunk = audio[i:i + self.chunk_samples]
                if len(chunk) > 0:
                    audio_callback(chunk)
    
    def split_sentences(self, text):
        """将文本分成句子"""
        import re
        sentences = re.split(r'[。！？\n]+', text)
        return [s.strip() for s in sentences if s.strip()]
    
    def play_streaming(self, text):
        """流式播放"""
        import sounddevice as sd
        
        def callback(chunk):
            sd.play(chunk, self.sample_rate)
        
        self.synthesize_streaming(text, callback)
        sd.wait()
```

```python
# test_tts_performance.py
# TTS性能测试脚本

import time
import numpy as np
from TTS.api import TTS
import soundfile as sf

class TTSPerformanceTester:
    """TTS性能测试"""
    
    def __init__(self, model_name="tts_models/zh-CN/baker/tacotron2-DDC-GST"):
        print(f"加载模型: {model_name}")
        self.tts = TTS(model_name=model_name, progress_bar=False)
        print("模型加载完成")
    
    def measure_latency(self, text, num_runs=10):
        """测量合成延迟"""
        print(f"\n延迟测试: 运行 {num_runs} 次")
        
        latencies = []
        for i in range(num_runs):
            start = time.time()
            audio = self.tts.tts(text)
            elapsed = time.time() - start
            
            rtf = elapsed / (len(audio) / 22050)  # 实时因子
            latencies.append(elapsed)
            
            print(f"  Run {i+1}: {elapsed:.3f}s (RTF: {rtf:.3f})")
        
        print(f"\n统计结果:")
        print(f"  平均延迟: {np.mean(latencies):.3f}s")
        print(f"  最小延迟: {np.min(latencies):.3f}s")
        print(f"  最大延迟: {np.max(latencies):.3f}s")
        print(f"  标准差:   {np.std(latencies):.3f}s")
        
        return latencies
    
    def measure_quality(self, text):
        """测量音频质量（生成音频的基本指标）"""
        print(f"\n质量测试: {text[:30]}...")
        
        start = time.time()
        audio = self.tts.tts(text)
        elapsed = time.time() - start
        
        # 基本指标
        duration = len(audio) / 22050
        rtf = elapsed / duration
        
        # 峰值电平
        peak = np.max(np.abs(audio))
        
        # RMS能量
        rms = np.sqrt(np.mean(audio**2))
        
        # DC偏移
        dc_offset = np.mean(audio)
        
        print(f"  时长: {duration:.2f}s")
        print(f"  实时率(RTF): {rtf:.3f} (越小越好)")
        print(f"  峰值电平: {20*np.log10(peak):.1f} dB")
        print(f"  RMS能量: {20*np.log10(rms+1e-10):.1f} dB")
        print(f"  DC偏移: {dc_offset:.6f}")
        
        # 保存测试音频
        sf.write("quality_test.wav", audio, 22050)
        print(f"  音频已保存: quality_test.wav")
        
        return {
            'duration': duration,
            'rtf': rtf,
            'peak_db': 20*np.log10(peak),
            'rms_db': 20*np.log10(rms+1e-10),
        }
    
    def stress_test(self, phrases, num_iterations=100):
        """压力测试：重复合成"""
        print(f"\n压力测试: {num_iterations} 次迭代")
        
        total_time = 0
        failures = 0
        
        for i in range(num_iterations):
            text = phrases[i % len(phrases)]
            
            try:
                start = time.time()
                audio = self.tts.tts(text)
                elapsed = time.time() - start
                total_time += elapsed
                
                if (i + 1) % 10 == 0:
                    print(f"  Progress: {i+1}/{num_iterations}")
                    
            except Exception as e:
                failures += 1
                print(f"  失败: {e}")
        
        print(f"\n压力测试结果:")
        print(f"  总耗时: {total_time:.2f}s")
        print(f"  平均耗时: {total_time/num_iterations:.3f}s")
        print(f"  失败次数: {failures}/{num_iterations}")
        print(f"  成功率: {(num_iterations-failures)/num_iterations*100:.1f}%")


def main():
    tester = TTSPerformanceTester()
    
    # 测试文本
    test_text = "你好，我是具身智能机器人，欢迎使用语音合成系统。"
```

### 8.3 声音质量评估

```python
# audio_quality_assessment.py
# 音频质量评估工具

import numpy as np
import librosa
from scipy.signal import correlate
import soundfile as sf

class AudioQualityAssessor:
    """音频质量评估"""
    
    def calculate_pesq(self, audio1, audio2, sr=22050):
        """
        计算PESQ分数（感知语音质量评估）
        注意：这是简化版，实际PESQ计算更复杂
        """
        # 确保长度一致
        min_len = min(len(audio1), len(audio2))
        audio1 = audio1[:min_len]
        audio2 = audio2[:min_len]
        
        # 计算频谱相关性
        spec1 = np.abs(librosa.stft(audio1))
        spec2 = np.abs(librosa.stft(audio2))
        
        # 简化PESQ计算（频谱相关性）
        correlation = np.corrcoef(spec1.flatten(), spec2.flatten())[0, 1]
        
        # 转换到PESQ-like分数（-0.5到5）
        pesq_score = 0.5 + 4.5 * (correlation + 0.5) / 1.5
        
        return max(0, min(5, pesq_score))
    
    def calculate_stoi(self, audio1, audio2, sr=22050):
        """
        计算STOI分数（短时客观 intelligibility）
        简化版实现
        """
        min_len = min(len(audio1), len(audio2))
        audio1 = audio1[:min_len]
        audio2 = audio2[:min_len]
        
        # 分帧
        frame_length = int(0.02 * sr)  # 20ms帧
        hop_length = int(0.01 * sr)    # 10ms跳跃
        
        frames1 = librosa.util.frame(audio1, frame_length, hop_length)
        frames2 = librosa.util.frame(audio2, frame_length, hop_length)
        
        # 计算帧间相关性
        correlations = []
        for f1, f2 in zip(frames1.T, frames2.T):
            corr = np.corrcoef(f1, f2)[0, 1]
            correlations.append(corr)
        
        stoi_score = np.mean(correlations)
        
        return max(0, min(1, stoi_score))
    
    def detect_artifacts(self, audio, sr=22050):
        """
        检测音频伪影
        """
        artifacts = []
        
        # 1. 检测削波
        if np.max(np.abs(audio)) > 0.99:
            artifacts.append('clipping')
        
        # 2. 检测直流偏移
        dc_offset = np.abs(np.mean(audio))
        if dc_offset > 0.01:
            artifacts.append(f'dc_offset ({dc_offset:.4f})')
        
        # 3. 检测低能量（可能为静音或无效音频）
        rms = np.sqrt(np.mean(audio**2))
        if rms < 0.01:
            artifacts.append('low_energy')
        
        # 4. 检测高频噪声（通过高频能量占比）
        spec = np.abs(librosa.stft(audio))
        low_freq_energy = np.sum(spec[:len(spec)//8, :])
        high_freq_energy = np.sum(spec[len(spec)//2:, :])
        if high_freq_energy > low_freq_energy * 0.5:
            artifacts.append('high_frequency_noise')
        
        return artifacts
    
    def full_assessment(self, audio_path):
        """完整质量评估"""
        audio, sr = librosa.load(audio_path, sr=22050)
        
        print(f"\n{'='*50}")
        print(f"音频质量评估: {audio_path}")
        print(f"{'='*50}")
        
        # 基本信息
        duration = len(audio) / sr
        print(f"时长: {duration:.2f}s")
        print(f"采样率: {sr}Hz")
        
        # 能量指标
        peak = np.max(np.abs(audio))
        rms = np.sqrt(np.mean(audio**2))
        print(f"峰值电平: {20*np.log10(peak+1e-10):.1f} dB")
        print(f"RMS能量: {20*np.log10(rms+1e-10):.1f} dB")
        
        # 伪影检测
        artifacts = self.detect_artifacts(audio, sr)
        if artifacts:
            print(f"检测到伪影: {', '.join(artifacts)}")
        else:
            print("未检测到明显伪影")
        
        print(f"{'='*50}\n")
        
        return {
            'duration': duration,
            'peak_db': 20*np.log10(peak+1e-10),
            'rms_db': 20*np.log10(rms+1e-10),
            'artifacts': artifacts
        }


# 使用示例
if __name__ == "__main__":
    assessor = AudioQualityAssessor()
    result = assessor.full_assessment("test_audio.wav")
```

### 8.4 常见问题排查

| 问题现象 | 可能原因 | 解决方案 |
|----------|----------|----------|
| TTS合成很慢 | 模型太大、CPU性能不足 | 使用量化模型、流式合成、GPU加速 |
| 合成音频有杂音 | 音频格式问题、声码器问题 | 检查采样率、尝试其他声码器 |
| 中文发音错误 | 分词问题、发音词典缺失 | 检查文本正则化、添加自定义词典 |
| 情绪调节无效 | 模型不支持情绪参数 | 使用支持情绪的模型、检查参数范围 |
| 实时率(RTF)过高 | 计算量大 | 模型量化、减少计算量、使用轻量模型 |
| 音频削波/失真 | 音量过大 | 降低增益、添加归一化 |

**15. 参考实现：**
```python
from TTS.api import TTS
import numpy as np
import soundfile as sf

class EmotionalTTS:
    def __init__(self):
        self.tts = TTS(model_name="tts_models/zh-CN/baker/tacotron2-DDC-GST")
        self.emotion_params = {
            'happy': {'pitch': 2, 'speed': 1.15, 'energy': 1.2},
            'sad': {'pitch': -2, 'speed': 0.85, 'energy': 0.8},
            'angry': {'pitch': 1.5, 'speed': 1.3, 'energy': 1.4},
            'neutral': {'pitch': 0, 'speed': 1.0, 'energy': 1.0},
        }
    
    def synthesize(self, text, emotion='neutral', output_path=None):
        import librosa
        
        # 基本合成
        audio = self.tts.tts(text)
        
        # 获取情绪参数
        params = self.emotion_params.get(emotion, self.emotion_params['neutral'])
        
        # 音高调整
        if params['pitch'] != 0:
            audio = librosa.effects.pitch_shift(audio, sr=22050, n_steps=params['pitch'])
        
        # 语速调整
        if params['speed'] != 1.0:
            audio = librosa.effects.time_stretch(audio, rate=params['speed'])
        
        # 音量调整
        audio = np.clip(audio * params['energy'], -1.0, 1.0)
        
        # 保存
        if output_path:
            sf.write(output_path, audio, 22050)
        
        return audio

# 使用
tts = EmotionalTTS()
tts.synthesize("你好世界！", emotion="happy", output_path="happy.wav")
```
