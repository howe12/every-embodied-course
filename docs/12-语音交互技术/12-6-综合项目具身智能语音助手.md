## 2. 项目概述

### 2.1 具身智能语音助手的定义

**具身智能语音助手**（Embodied Intelligent Voice Assistant）是指具备以下特征的智能系统：

1. **具身性（Embodied）**：系统部署在真实的物理机器人上，能够感知环境、执行物理动作
2. **交互性（Interactive）**：通过语音与用户进行自然语言对话交互
3. **自主性（Autonomous）**：能够理解高层指令，自主规划执行路径
4. **多模态（Multimodal）**：融合语音、视觉、文本等多种感知信息进行决策

> **与传统语音助手的区别**
>
> | 特性 | 传统语音助手 | 具身智能语音助手 |
> |------|-------------|----------------|
> | 执行能力 | 仅信息查询/播放 | 可操控机器人执行物理动作 |
> | 环境感知 | 无 | 依赖视觉/雷达等传感器 |
> | 反馈方式 | 语音/屏幕 | 语音 + 物理动作 |
> | 自主程度 | 低（指令驱动） | 高（意图驱动+自主规划） |

### 2.2 核心功能需求

**F1：语音唤醒与输入**
- 支持唤醒词（如"你好小智"）唤醒机器人
- 支持实时语音流输入
- 支持打断和打断后重新输入

**F2：语音识别（ASR）**
- 实时将语音转为文本
- 支持中文普通话识别
- 在机器人本地运行（离线优先）
- 识别延迟 < 500ms

**F3：意图理解（NLU）**
- 准确识别用户意图（navigation、manipulation、information、control）
- 提取关键槽位（目标位置、物体名称、动作类型等）
- 支持多轮对话澄清

**F4：对话管理**
- 管理对话状态机（IDLE → LISTENING → PROCESSING → EXECUTING → SPEAKING → IDLE）
- 支持多轮上下文记忆
- 处理对话冲突和模糊指令

**F5：视觉感知**
- 目标检测：识别场景中的常见物体
- 空间定位：估算物体相对机器人的位置
- 场景理解：理解场景布局和可通行区域

**F6：任务规划与执行**
- 将高层指令分解为可执行的动作序列
- 调用运动控制模块执行
- 处理执行失败和异常情况

**F7：语音反馈（TTS）**
- 将系统响应转为自然语音
- 支持情绪化语音合成
- 实时合成延迟 < 300ms

### 2.3 非功能需求

| 指标 | 要求 | 说明 |
|------|------|------|
| 端到端响应延迟 | < 3s | 从用户说完到开始执行 |
| ASR 实时率 | < 0.5s | 语音输入到文本输出 |
| TTS 合成延迟 | < 0.3s | 文本到语音开始播放 |
| 系统 CPU 占用 | < 70% | 峰值时不超过 70% |
| 系统内存占用 | < 8GB | 全部模块加载后 |

### 2.4 技术选型依据

| 模块 | 选型 | 依据 |
|------|------|------|
| **ASR** | Whisper (OpenAI) | 开源、效果优秀、支持多语言、有 ONNX 加速版本 |
| **NLU** | Hugging Face Transformers + 规则引擎 | 灵活切换模型、支持本地部署 |
| **TTS** | Fish Speech / Coqui TTS | 开源、中文支持好、可微调 |
| **LLM** | Qwen / ChatGLM | 开源中文大模型、本地部署能力强 |
| **视觉检测** | YOLOv8 (Ultralytics) | 速度快、精度高、接口简洁 |
| **机械臂控制** | MoveIt2 | ROS2 标准、生态完善 |
| **导航** | Nav2 | ROS2 标准、室外室内均支持 |

## 4. 模块详细设计

### 4.1 语音感知模块（ASR）

**设计目标：** 将麦克风采集的音频流实时转换为文本，具备噪声抑制、断句检测能力。

```python
# asr_node.py - 语音识别节点
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import numpy as np
import whisper
import torch

class ASRNode(Node):
    """语音识别节点：接收音频流，输出识别文本"""
    
    def __init__(self):
        super().__init__('asr_node')
        
        # ========== 参数声明 ==========
        self.declare_parameter('model_name', 'base')          # Whisper 模型大小
        self.declare_parameter('language', 'zh')              # 识别语言
        self.declare_parameter('use_vad', True)               # 是否使用 VAD
        self.declare_parameter('audio_sample_rate', 16000)    # 采样率
        
        self.model_name = self.get_parameter('model_name').value
        self.language = self.get_parameter('language').value
        self.use_vad = self.get_parameter('use_vad').value
        self.sample_rate = self.get_parameter('audio_sample_rate').value
        
        # ========== 模型加载 ==========
        self.get_logger().info(f'加载 Whisper 模型: {self.model_name}')
        self.model = whisper.load_model(self.model_name)
        self.device = 'cuda' if torch.cuda.is_available() else 'cpu'
        self.get_logger().info(f'ASR 运行设备: {self.device}')
        
        # VAD 模型（用于语音活动检测）
        if self.use_vad:
            from funasr import AutoVAD
            self.vad_model = AutoVAD(model='fsmn-vad')
            self.get_logger().info('VAD 模型加载完成')
        
        # 音频缓冲区
        self.audio_buffer = np.array([], dtype=np.float32)
        self.is_speaking = False
        self.silence_count = 0
        self.speech_threshold = 30  # 连续静音帧数阈值
        
        # ========== ROS2 通信 ==========
        self.audio_sub = self.create_subscription(
            String,  # 简化，实际应为 AudioData
            '/audio/audio_raw',
            self.audio_callback,
            10
        )
        self.text_pub = self.create_publisher(String, '/asr/text', 10)
        self.confidence_pub = self.create_publisher(String, '/asr/confidence', 10)
        
        self.get_logger().info('ASR 节点初始化完成')
    
    def audio_callback(self, msg: String):
        """音频数据回调函数"""
        # 简化实现：msg.data 为 base64 编码的音频
        # 实际使用时替换为真实的 AudioData 消息处理
        import base64
        try:
            audio_bytes = base64.b64decode(msg.data)
            audio_data = np.frombuffer(audio_bytes, dtype=np.int16)
            audio_float = audio_data.astype(np.float32) / 32768.0
        except Exception:
            # 模拟音频数据（用于测试）
            audio_float = np.random.randn(self.sample_rate).astype(np.float32) * 0.01
        
        if self.use_vad:
            self._process_with_vad(audio_float)
        else:
            self._process_direct(audio_float)
    
    def _process_with_vad(self, audio_float: np.ndarray):
        """使用 VAD 进行语音活动检测"""
        self.audio_buffer = np.concatenate([self.audio_buffer, audio_float])
        
        if len(self.audio_buffer) >= self.sample_rate * 0.5:
            try:
                # VAD 检测
                speech_probs = self.vad_model.detect(self.audio_buffer)
                is_speech = np.max(speech_probs) > 0.5
            except Exception:
                is_speech = np.max(np.abs(audio_float)) > 0.02
            
            if is_speech:
                self.is_speaking = True
                self.silence_count = 0
            else:
                self.silence_count += 1
                
                if self.is_speaking and self.silence_count > self.speech_threshold:
                    self.is_speaking = False
                    self._recognize_buffer()
    
    def _process_direct(self, audio_float: np.ndarray):
        """直接处理（无 VAD，按固定长度识别）"""
        self.audio_buffer = np.concatenate([self.audio_buffer, audio_float])
        
        if len(self.audio_buffer) >= self.sample_rate * 3:
            self._recognize_buffer()
    
    def _recognize_buffer(self):
        """对缓冲区中的音频执行识别"""
        if len(self.audio_buffer) < self.sample_rate * 0.5:
            self.audio_buffer = np.array([], dtype=np.float32)
            return
        
        try:
            # 执行 Whisper 识别
            result = self.model.transcribe(
                self.audio_buffer,
                language=self.language,
                fp16=(self.device == 'cuda')
            )
            
            text = result['text'].strip()
            
            if text:
                text_msg = String()
                text_msg.data = text
                self.text_pub.publish(text_msg)
                
                confidence_msg = String()
                confidence_msg.data = str(result.get('avg_logprob', 0))
                self.confidence_pub.publish(confidence_msg)
                
                self.get_logger().info(f'ASR 识别结果: {text}')
        
        except Exception as e:
            self.get_logger().error(f'ASR 识别失败: {e}')
        
        finally:
            self.audio_buffer = np.array([], dtype=np.float32)
            self.silence_count = 0

def main(args=None):
    rclpy.init(args=args)
    node = ASRNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

### 4.2 意图理解模块（NLU）

**设计目标：** 将用户文本解析为结构化的意图和槽位信息。

```python
# nlu_node.py - 自然语言理解节点
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import json
import re
from dataclasses import dataclass
from typing import Optional, Dict, List

@dataclass
class IntentResult:
    """意图识别结果数据结构"""
    intent: str
    confidence: float
    slots: Dict[str, str]
    raw_text: str

class NLUNode(Node):
    """自然语言理解节点：文本 → 意图 + 槽位"""
    
    def __init__(self):
        super().__init__('nlu_node')
        
        # ========== 参数 ==========
        self.declare_parameter('use_llm', False)       # 是否使用 LLM 进行理解
        self.declare_parameter('llm_model', 'qwen')    # LLM 模型名称
        
        self.use_llm = self.get_parameter('use_llm').value
        self.llm_model = self.get_parameter('llm_model').value
        
        # ========== 意图定义 ==========
        self.intent_definitions = {
            'navigation': {
                'keywords': ['去', '走到', '导航到', '移动到', '过来', '到'],
                'slots': ['target_place'],
                'examples': ['去厨房', '走到桌子旁边', '导航到卧室']
            },
            'manipulation': {
                'keywords': ['拿', '抓', '拿起', '递', '放', '放置'],
                'slots': ['object_name', 'target_place', 'source_place'],
                'examples': ['把苹果拿给我', '把杯子放在桌上', '从厨房拿水']
            },
            'information': {
                'keywords': ['什么', '几个', '哪里', '有没有', '告诉'],
                'slots': ['query_topic'],
                'examples': ['这是什么', '桌上有几个东西', '厨房在哪里']
            },
            'greeting': {
                'keywords': ['你好', '嗨', '早上好', 'hi', 'hello'],
                'slots': [],
                'examples': ['你好', '嗨', '早上好']
            },
            'control': {
                'keywords': ['停止', '暂停', '继续', '等待'],
                'slots': ['action'],
                'examples': ['停下来', '暂停', '等一下']
            },
            'unknown': {
                'keywords': [],
                'slots': [],
                'examples': []
            }
        }
        
        # ========== 槽位提取规则 ==========
        self.slot_patterns = {
            'target_place': re.compile(r'(?:去|到|在)\s*([\u4e00-\u9fa5]+)', re.I),
            'object_name': re.compile(r'(?:把|拿|抓|放)\s*([\u4e00-\u9fa5]+?)(?:给|在|到|\s|$)', re.I),
            'source_place': re.compile(r'从([\u4e00-\u9fa5]+)', re.I),
            'action': re.compile(r'(停止|暂停|继续|等待)', re.I)
        }
        
        # ========== LLM 模型（可选） ==========
        if self.use_llm:
            self._init_llm()
        
        # ========== ROS2 通信 ==========
        self.text_sub = self.create_subscription(
            String, '/asr/text', self.text_callback, 10
        )
        self.intent_pub = self.create_publisher(String, '/nlu/intent', 10)
        self.slots_pub = self.create_publisher(String, '/nlu/slots', 10)
        
        self.get_logger().info('NLU 节点初始化完成')
    
    def _init_llm(self):
        """初始化 LLM 模型"""
        try:
            from transformers import AutoModelForCausalLM, AutoTokenizer
            
            model_map = {
                'qwen': 'Qwen/Qwen-7B-Chat',
                'chatglm': 'THUDM/chatglm-6b'
            }
            model_name = model_map.get(self.llm_model, 'gpt2')
            
            self.get_logger().info(f'加载 LLM 模型: {model_name}')
            self.tokenizer = AutoTokenizer.from_pretrained(
                model_name, trust_remote_code=True
            )
            self.model = AutoModelForCausalLM.from_pretrained(
                model_name, trust_remote_code=True, device_map='auto'
            )
            self.get_logger().info('LLM 模型加载完成')
            
        except Exception as e:
            self.get_logger().warn(f'LLM 加载失败，使用规则引擎: {e}')
            self.use_llm = False
    
    def text_callback(self, msg: String):
        """文本输入回调"""
        text = msg.data.strip()
        if not text:
            return
        
        self.get_logger().info(f'NLU 输入文本: {text}')
        
        # 意图识别
        intent_result = self._recognize_intent(text)
        
        # 槽位填充
        slots = self._extract_slots(text, intent_result.intent)
        intent_result.slots = slots
        
        # 发布结果
        intent_msg = String()
        intent_msg.data = json.dumps({
            'intent': intent_result.intent,
            'confidence': intent_result.confidence,
            'text': text
        }, ensure_ascii=False)
        self.intent_pub.publish(intent_msg)
        
        slots_msg = String()
        slots_msg.data = json.dumps(slots, ensure_ascii=False)
        self.slots_pub.publish(slots_msg)
        
        self.get_logger().info(f'NLU 意图: {intent_result.intent}, 槽位: {slots}')
    
    def _recognize_intent(self, text: str) -> IntentResult:
        """识别用户意图"""
        # 首先尝试关键词匹配
        for intent_name, intent_info in self.intent_definitions.items():
            if intent_name == 'unknown':
                continue
            for keyword in intent_info['keywords']:
                if keyword in text:
                    return IntentResult(
                        intent=intent_name,
                        confidence=0.95,
                        slots={},
                        raw_text=text
                    )
        
        # 如果启用了 LLM，使用 LLM 进行意图识别
        if self.use_llm:
            return self._llm_recognize_intent(text)
        
        return IntentResult(intent='unknown', confidence=0.5, slots={}, raw_text=text)
    
    def _llm_recognize_intent(self, text: str) -> IntentResult:
        """使用 LLM 识别意图"""
        prompt = f"""用户说：「{text}」
请判断用户的意图，只能返回以下意图之一：navigation, manipulation, information, greeting, control, unknown
直接返回意图名称，不要解释。"""
        
        inputs = self.tokenizer(prompt, return_tensors='pt').to(self.model.device)
        outputs = self.model.generate(**inputs, max_new_tokens=10)
        response = self.tokenizer.decode(outputs[0], skip_special_tokens=True)
        
        for intent_name in self.intent_definitions.keys():
            if intent_name in response.lower():
                return IntentResult(
                    intent=intent_name, confidence=0.85, slots={}, raw_text=text
                )
        
        return IntentResult(intent='unknown', confidence=0.5, slots={}, raw_text=text)
    
    def _extract_slots(self, text: str, intent: str) -> dict:
        """提取槽位信息"""
        slots = {}
        
        if intent == 'navigation':
            match = self.slot_patterns['target_place'].search(text)
            if match:
                slots['target_place'] = match.group(1)
        
        elif intent == 'manipulation':
            # 提取物体名
            for match in self.slot_patterns['object_name'].finditer(text):
                obj = match.group(1)
                if obj and len(obj) > 1:
                    slots['object_name'] = obj
                    break
            
            target_match = self.slot_patterns['target_place'].search(text)
            if target_match:
                slots['target_place'] = target_match.group(1)
            
            source_match = self.slot_patterns['source_place'].search(text)
            if source_match:
                slots['source_place'] = source_match.group(1)
        
        elif intent == 'information':
            slots['query_topic'] = text
        
        elif intent == 'control':
            action_match = self.slot_patterns['action'].search(text)
            if action_match:
                slots['action'] = action_match.group(1)
        
        return slots

def main(args=None):
    rclpy.init(args=args)
    node = NLUNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
```

### 4.3 对话管理模块

**设计目标：** 管理对话状态机，处理多轮对话，决定系统响应策略。

```python
# dialog_manager_node.py - 对话管理节点
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import json
from enum import Enum
from dataclasses import dataclass, field
from typing import List, Optional

class DialogStateMachine(Node):
    """对话状态机：管理对话状态和策略"""
    
    class State(Enum):
        IDLE = 'idle'              # 空闲，等待用户输入
        LISTENING = 'listening'    # 正在听
        UNDERSTANDING = 'understanding'  # 理解中
        EXECUTING = 'executing'    # 执行中
        SPEAKING = 'speaking'      # 语音反馈中
        WAITING_CONFIRM = 'waiting_confirm'  # 等待确认
        ERROR = 'error'            # 错误状态
    
    @dataclass
    class DialogContext:
        """对话上下文"""
        history: List[dict] = field(default_factory=list)
        current_intent: Optional[str] = None
        current_slots: dict = field(default_factory=dict)
        pending_actions: List[str] = field(default_factory=list)
        confirm_needed: bool = False
        error_message: Optional[str] = None
    
    def __init__(self):
        super().__init__('dialog_manager_node')
        
        self.declare_parameter('max_history', 10)
        self.max_history = self.get_parameter('max_history').value
        
        # 状态机
        self.state = self.State.IDLE
        self.context = self.DialogContext()
        
        # ========== ROS2 通信 ==========
        self.intent_sub = self.create_subscription(
            String, '/nlu/intent', self.intent_callback, 10
        )
        self.slots_sub = self.create_subscription(
            String, '/nlu/slots', self.slots_callback, 10
        )
        self.execute_result_sub = self.create_subscription(
            String, '/execute/result', self.execute_result_callback, 10
        )
        
        self.state_pub = self.create_publisher(String, '/dialog/state', 10)
        self.action_pub = self.create_publisher(String, '/dialog/action', 10)
        self.response_pub = self.create_publisher(String, '/tts/text', 10)
        
        self.state_timer = self.create_timer(1.0, self.publish_state)
        
        self.get_logger().info('对话管理节点初始化完成')
    
    def intent_callback(self, msg: String):
        """接收意图识别结果"""
        try:
            data = json.loads(msg.data)
            intent = data.get('intent', 'unknown')
            text = data.get('text', '')
            
            self.get_logger().info(f'收到意图: {intent}')
            self.context.current_intent = intent
            self.context.error_message = None
            self.state = self.State.UNDERSTANDING
            self._add_to_history('user', text, intent)
            
        except json.JSONDecodeError:
            self.get_logger().error('意图消息 JSON 解析失败')
    
    def slots_callback(self, msg: String):
        """接收槽位填充结果"""
        try:
            slots = json.loads(msg.data)
            self.context.current_slots.update(slots)
            self.get_logger().info(f'更新槽位: {slots}')
            self._decide_next_action()
        except json.JSONDecodeError:
            self.get_logger().error('槽位消息 JSON 解析失败')
    
    def execute_result_callback(self, msg: String):
        """接收执行结果"""
        try:
            result = json.loads(msg.data)
            self.get_logger().info(f'执行结果: {result}')
            self._handle_execute_result(result)
        except json.JSONDecodeError:
            self.get_logger().error('执行结果 JSON 解析失败')
    
    def _decide_next_action(self):
        """根据当前意图和槽位决定下一步动作"""
        intent = self.context.current_intent
        slots = self.context.current_slots
        
        # 检查槽位是否完整
        missing_slots = self._check_missing_slots(intent, slots)
        
        if missing_slots:
            # 需要澄清
            self.state = self.State.WAITING_CONFIRM
            response = f'请问您说的是{missing_slots[0]}吗？'
            self._speak(response)
        else:
            # 槽位完整，执行动作
            self.state = self.State.EXECUTING
            action = {
                'type': intent,
                'slots': slots,
                'action_type': 'execute'
            }
            action_msg = String()
            action_msg.data = json.dumps(action, ensure_ascii=False)
            self.action_pub.publish(action_msg)
    
    def _check_missing_slots(self, intent: str, slots: dict) -> List[str]:
        """检查缺失的槽位"""
        required_slots = {
            'navigation': ['target_place'],
            'manipulation': ['object_name'],
            'information': [],
            'greeting': [],
            'control': ['action'],
        }
        
        required = required_slots.get(intent, [])
        missing = [s for s in required if s not in slots or not slots[s]]
        return missing
    
    def _handle_execute_result(self, result: dict):
        """处理执行结果，生成语音反馈"""
        status = result.get('status', 'unknown')
        
        if status == 'success':
            response = result.get('message', '任务完成')
        elif status == 'failed':
            response = f'执行失败：{result.get("reason", "未知原因")}'
        else:
            response = '任务执行中'
        
        self._speak(response)
        self.state = self.State.IDLE
    
    def _speak(self, text: str):
        """生成语音反馈"""
        self.state = self.State.SPEAKING
        msg = String()
        msg.data = text
        self.response_pub.publish(msg)
        self._add_to_history('system', text, '')
        
        # 定时器回调：说话结束后回到 IDLE
        def on_speak_done(_):
            if self.state == self.State.SPEAKING:
                self.state = self.State.IDLE
        
        self.create_timer(3.0, on_speak_done)
    
    def publish_state(self):
        """定时发布状态"""
        state_msg = String()
        state_msg.data = json.dumps({
            'state': self.state.value,
            'intent': self.context.current_intent,
            'slots': self.context.current_slots
        }, ensure_ascii=False)
        self.state_pub.publish(state_msg)
    
    def _add_to_history(self, role: str, text: str, intent: str):
        """添加对话历史"""
        self.context.history.append({
            'role': role,
            'text': text,
            'intent': intent,
            'timestamp': self.get_clock().now().to_msg().sec
        })
        # 限制历史长度
        if len(self.context.history) > self.max_history:
            self.context.history = self.context.history[-self.max_history:]

def main(args=None):
    rclpy.init(args=args)
    node = DialogStateMachine()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
### 4.4 视觉感知模块

# vision_node.py - 视觉感知节点
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
from std_msgs.msg import String
import numpy as np
import cv2
from dataclasses import dataclass
from typing import List, Optional, Tuple

@dataclass
class Detection:
    """检测结果"""
    class_name: str
    confidence: float
    bbox: Tuple[int, int, int, int]  # x1, y1, x2, y2
    center: Tuple[int, int]

class VisionNode(Node):
    """视觉感知节点：目标检测 + 空间定位"""
    
    def __init__(self):
        super().__init__('vision_node')
        
        # ========== 参数 ==========
        self.declare_parameter('model_name', 'yolov8n.pt')

        self.conf_threshold = self.get_parameter('conf_threshold').value
        self.camera_topic = self.get_parameter('camera_topic').value
        self.enable_depth = self.get_parameter('enable_depth').value

        from ultralytics import YOLO
        self.model = YOLO(self.model_name)
        self.get_logger().info('视觉模型加载完成')

            'person': '人', 'cup': '杯子', 'bottle': '瓶子',
            'apple': '苹果', 'banana': '香蕉', 'orange': '橘子',
            'chair': '椅子', 'table': '桌子', 'book': '书',
            'phone': '手机', 'keyboard': '键盘', 'mouse': '鼠标'
        }
        
        # ========== ROS2 通信 ==========
        self.image_sub = self.create_subscription(
            Image, self.camera_topic, self.image_callback, 10
        )
        self.detection_pub = self.create_publisher(
            String, '/vision/detections', 10
        )
        self.query_sub = self.create_subscription(
            String, '/vision/query', self.query_callback, 10
        )
        
        self.get_logger().info('视觉感知节点初始化完成')
    
    def image_callback(self, msg: Image):
        """图像处理回调"""
        try:
            # 将 ROS Image 消息转换为 numpy 数组
            np_arr = np.frombuffer(msg.data, dtype=np.uint8)
            image = np_arr.reshape(msg.height, msg.width, 3)
            
            # 执行目标检测
            results = self.model(image, conf=self.conf_threshold, verbose=False)
            
            # 解析检测结果
            detections = self._parse_results(results)
            
            # 发布检测结果
            import json
            det_msg = String()
            det_msg.data = json.dumps([
                {
                    'class': d.class_name,
                    'class_cn': self.object_name_mapping.get(d.class_name, d.class_name),
                    'confidence': float(d.confidence),
                    'bbox': d.bbox,
                    'center': d.center
                }
                for d in detections
            ], ensure_ascii=False)
            self.detection_pub.publish(det_msg)
            
        except Exception as e:
            self.get_logger().error(f'图像处理失败: {e}')
    
    def _parse_results(self, results) -> List[Detection]:
        """解析 YOLO 检测结果"""
        detections = []
        
        for result in results:
            boxes = result.boxes
            if boxes is None:
                continue
            
            for box in boxes:
                x1, y1, x2, y2 = box.xyxy[0].cpu().numpy()
                conf = box.conf[0].cpu().numpy()
                cls_id = int(box.cls[0].cpu().numpy())
                cls_name = self.model.names[cls_id]
                
                cx, cy = int((x1 + x2) / 2), int((y1 + y2) / 2)
                
                detections.append(Detection(
                    class_name=cls_name,
                    confidence=conf,
                    bbox=(int(x1), int(y1), int(x2), int(y2)),
                    center=(cx, cy)
                ))
        
        return detections
    
    def query_callback(self, msg: String):
        """查询回调：根据物体名称查找位置"""
        import json
        
        try:
            query = json.loads(msg.data)
            target_name = query.get('object_name', '')
            
            # 发布检测结果后查询
            # 这里简化处理，实际应等待检测结果返回
            detections = self._get_cached_detections()
            
            matched = [d for d in detections 
                      if target_name in d.class_name 
                      or self.object_name_mapping.get(d.class_name, '') == target_name]
            
            response = {
                'found': len(matched) > 0,
                'count': len(matched),
                'positions': [{'center': d.center, 'bbox': d.bbox} for d in matched]
            }
            
            # 发布查询结果
            response_pub = self.create_publisher(String, '/vision/query_result', 10)
            response_msg = String()
            response_msg.data = json.dumps(response, ensure_ascii=False)
            response_pub.publish(response_msg)
            
        except Exception as e:
            self.get_logger().error(f'查询处理失败: {e}')
    
    def _get_cached_detections(self) -> List[Detection]:
        """获取缓存的检测结果（简化实现）"""
        return []

def main(args=None):
    rclpy.init(args=args)
    node = VisionNode()
    rclpy.spin(node)
    node.destroy_node()

            return
        
        try:
            action = json.loads(msg.data)
            action_type = action.get('type', 'unknown')
            slots = action.get('slots', {})
            
            self.get_logger().info(f'收到动作指令: {action_type}, 槽位: {slots}')
            self._execute_action(action_type, slots)
            
        except json.JSONDecodeError:
            self.get_logger().error('动作指令 JSON 解析失败')
    
    def _execute_action(self, action_type: str, slots: dict):
        """执行具体动作"""
        self.is_executing = True
        
        if action_type == 'navigation':
            self._do_navigate(slots)
        elif action_type == 'manipulation':
            self._do_manipulation(slots)
        elif action_type == 'control':
            self._do_control(slots)
        else:
            self._publish_result('failed', '未知动作类型')
            self.is_executing = False
    
    def _do_navigate(self, slots: dict):
        """执行导航动作"""
        target_place = slots.get('target_place', '')
        
        self.get_logger().info(f'开始导航到: {target_place}')
        
        # 构造导航目标
        nav_goal = {
            'place_name': target_place,
            'pose': {
                'position': {'x': 1.0, 'y': 2.0, 'z': 0.0},
                'orientation': {'x': 0, 'y': 0, 'z': 0, 'w': 1}
            }
        }
        
        nav_msg = String()
        nav_msg.data = json.dumps(nav_goal, ensure_ascii=False)
        self.nav_goal_pub.publish(nav_msg)
        
        # 模拟导航完成（实际使用 action 回调）
        self.create_timer(3.0, lambda _: self._on_navigate_complete(target_place))
    
    def _on_navigate_complete(self, target_place: str):
        """导航完成回调"""
        self.get_logger().info(f'导航到 {target_place} 完成')
        self._publish_result('success', f'已到达{target_place}')
        self.is_executing = False
    
    def _do_manipulation(self, slots: dict):
        """执行操作动作"""
        object_name = slots.get('object_name', '')
        target_place = slots.get('target_place', '')
        
        self.get_logger().info(f'开始操作: {object_name} -> {target_place}')
        
        if not self.has_arm:
            self._publish_result('failed', '机器人没有机械臂')
            self.is_executing = False
            return
        
        # 构造抓取指令
        grasp_goal = {
            'object_name': object_name,
            'target_place': target_place,
            'action': 'grasp_and_place'
        }
        
        arm_msg = String()
        arm_msg.data = json.dumps(grasp_goal, ensure_ascii=False)
        self.arm_goal_pub.publish(arm_msg)
        
        # 模拟操作完成
        self.create_timer(5.0, lambda _: self._on_manipulation_complete(object_name))
    
    def _on_manipulation_complete(self, object_name: str):
        """操作完成回调"""
        self.get_logger().info(f'操作 {object_name} 完成')
        self._publish_result('success', f'已经把{object_name}拿过来了')
        self.is_executing = False
    
    def _do_control(self, slots: dict):
        """执行控制动作"""
        action = slots.get('action', 'stop')
        
        if action == '停止' or action == '暂停':
            self.get_logger().info('执行停止命令')
            self._publish_result('success', '已停止')
        elif action == '继续':
            self._publish_result('success', '继续执行')
        
        self.is_executing = False
    
    def _publish_result(self, status: str, message: str, reason: str = ''):
        """发布执行结果"""
        result = {
            'status': status,
            'message': message,
            'reason': reason
        }
        
        result_msg = String()
        result_msg.data = json.dumps(result, ensure_ascii=False)
        self.result_pub.publish(result_msg)

def main(args=None):
    rclpy.init(args=args)
    node = MotionControlNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

### 4.6 语音生成模块（TTS）

**设计目标：** 将文本转换为自然语音，输出到扬声器。

```python
# tts_node.py - 语音合成节点
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import numpy as np
import sounddevice as sd
import threading
import queue
from dataclasses import dataclass
from typing import Optional

@dataclass
class TTSConfig:
    """TTS 配置"""
    sample_rate: int = 22050
    engine: str = 'coqui'  # coqui / fish_speech / espeak
    voice: str = 'default'
    speed: float = 1.0

class TTSNode(Node):
    """语音合成节点：文本 → 语音 → 扬声器"""
    
    def __init__(self):
        super().__init__('tts_node')
        
        # ========== 参数 ==========
        self.declare_parameter('sample_rate', 22050)
        self.declare_parameter('engine', 'coqui')
        self.declare_parameter('voice', 'default')
        self.declare_parameter('speed', 1.0)
        
        self.config = TTSConfig(
            sample_rate=self.get_parameter('sample_rate').value,
            engine=self.get_parameter('engine').value,
            voice=self.get_parameter('voice').value,
            speed=self.get_parameter('speed').value
        )
        
        # ========== 初始化 TTS 引擎 ==========
        self._init_tts_engine()
        
        # ========== 音频播放队列 ==========
        self.audio_queue = queue.Queue()
        self.is_playing = False
        self.stop_flag = False
        
        # 播放线程
        self.play_thread = threading.Thread(target=self._play_loop, daemon=True)
        self.play_thread.start()
        
        # ========== ROS2 通信 ==========
        self.text_sub = self.create_subscription(
            String, '/tts/text', self.text_callback, 10
        )
        self.playback_pub = self.create_publisher(
            String, '/tts/playback', 10
        )
        
        self.get_logger().info('TTS 节点初始化完成')
    
    def _init_tts_engine(self):
        """初始化 TTS 引擎"""
        engine = self.config.engine
        
        if engine == 'coqui':
            try:
                from TTS.api import TTS
                self.get_logger().info('初始化 Coqui TTS')
                # 使用 Coqui TTS（支持中文）
                self.tts = TTS(model_name='tts_models/zh-CN/baker/tacotron2-DDC-GST')
                self.get_logger().info('Coqui TTS 初始化完成')
            except Exception as e:
                self.get_logger().warn(f'Coqui TTS 加载失败: {e}，使用模拟模式')
                self.tts = None
        
        elif engine == 'fish_speech':
            try:
                # Fish Speech 接口
                self.get_logger().info('Fish Speech 待集成')
                self.tts = None
            except Exception as e:
                self.get_logger().warn(f'Fish Speech 加载失败: {e}')
                self.tts = None
        
        else:
            self.get_logger().warn(f'未知 TTS 引擎: {engine}，使用模拟模式')
            self.tts = None
    
    def text_callback(self, msg: String):
        """接收待合成文本"""
        text = msg.data.strip()
        if not text:
            return
        
        self.get_logger().info(f'TTS 收到文本: {text}')
        self.audio_queue.put(text)
    
    def _synthesize(self, text: str) -> Optional[np.ndarray]:
        """合成语音"""
        if self.tts is None:
            # 模拟模式：生成静音或简单正弦波
            self.get_logger().warn('使用 TTS 模拟模式')
            duration = len(text) * 0.1  # 估算时长
            samples = int(self.config.sample_rate * duration)
            return np.zeros(samples, dtype=np.float32)
        
        try:
            # Coqui TTS 合成
            wav = self.tts.tts(text)
            return wav
        except Exception as e:
            self.get_logger().error(f'TTS 合成失败: {e}')
            return None
    
    def _play_loop(self):
        """音频播放循环（在单独线程中运行）"""
        while not self.stop_flag:
            try:
                # 阻塞等待文本
                text = self.audio_queue.get(timeout=1.0)
                
                # 合成音频
                audio = self._synthesize(text)
                
                if audio is not None:
                    self.is_playing = True
                    
                    # 播放音频
                    sd.play(audio, self.config.sample_rate)
                    sd.wait()  # 等待播放完成
                    
                    self.is_playing = False
                    
                    # 发布播放完成消息
                    playback_msg = String()
                    playback_msg.data = 'done'
                    self.playback_pub.publish(playback_msg)
                
            except queue.Empty:
                continue
            except Exception as e:
                self.get_logger().error(f'播放异常: {e}')
    
    def destroy_node(self):
        """清理节点"""
        self.stop_flag = True
        sd.stop()
        super().destroy_node()

def main(args=None):
    rclpy.init(args=args)
    node = TTSNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
```

### 5.3 多模态融合中枢

```python
# fusion/fusion_center.py - 多模态融合中枢
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import json
from dataclasses import dataclass, field
from typing import Dict, List, Optional, Any
from threading import Lock

@dataclass
class FusedContext:
    """融合后的上下文"""
    timestamp: float = 0.0
    intent: Optional[str] = None
    slots: Dict[str, str] = field(default_factory=dict)
    vision_objects: List[Dict] = field(default_factory=list)
    robot_pose: Optional[Dict] = None
    confidence: float = 0.0
    reasoning: str = ''

class FusionCenter(Node):
    """多模态融合中枢：整合语音、视觉、状态信息"""
    
    def __init__(self):
        super().__init__('fusion_center')
        
        self.declare_parameter('fusion_strategy', 'late')  # late / early / attention
        self.declare_parameter('context_ttl', 30.0)      # 上下文有效期（秒）
        
        self.fusion_strategy = self.get_parameter('fusion_strategy').value
        self.context_ttl = self.get_parameter('context_ttl').value
        
        # 锁，保证线程安全
        self.lock = Lock()
        
        # 各模态数据缓存
        self.nlu_data: Optional[dict] = None
        self.vision_data: List[dict] = []
        self.state_data: Optional[dict] = None
        self.last_update: float = 0.0
        
        # ========== ROS2 通信 ==========
        # NLU 数据输入
        self.nlu_sub = self.create_subscription(
            String, '/nlu/intent', self.nlu_callback, 10
        )
        
        # 视觉数据输入
        self.vision_sub = self.create_subscription(
            String, '/vision/detections', self.vision_callback, 10
        )
        
        # 状态数据输入
        self.state_sub = self.create_subscription(
            String, '/robot/state', self.state_callback, 10
        )
        
        # 融合结果输出
        self.fusion_pub = self.create_publisher(
            String, '/fusion/context', 10
        )
        
        # 查询请求
        self.query_sub = self.create_subscription(
            String, '/fusion/query', self.query_callback, 10
        )
        
        self.get_logger().info('多模态融合中枢初始化完成')
    
    def nlu_callback(self, msg: String):
        """接收 NLU 数据"""
        with self.lock:
            try:
                self.nlu_data = json.loads(msg.data)
                self._trigger_fusion()
            except json.JSONDecodeError:
                self.get_logger().error('NLU 数据解析失败')
    
    def vision_callback(self, msg: String):
        """接收视觉数据"""
        with self.lock:
            try:
                self.vision_data = json.loads(msg.data)
            except json.JSONDecodeError:
                self.vision_data = []
    
    def state_callback(self, msg: String):
        """接收机器人状态"""
        with self.lock:
            try:
                self.state_data = json.loads(msg.data)
            except json.JSONDecodeError:
                pass
    
    def query_callback(self, msg: String):
        """处理融合数据查询"""
        try:
            query = json.loads(msg.data)
            query_type = query.get('type', '')
            
            with self.lock:
                if query_type == 'get_objects':
                    # 查询特定物体
                    target = query.get('target', '')
                    matched = [
                        obj for obj in self.vision_data
                        if target in obj.get('class', '') or target in obj.get('class_cn', '')
                    ]
                    response = {'found': matched}
                
                elif query_type == 'get_context':
                    # 获取完整上下文
                    response = self._build_fused_context()
                
                else:
                    response = {'error': 'unknown query type'}
            
            # 发布响应
            from std_msgs.msg import String as MsgString
            resp_pub = self.create_publisher(MsgString, '/fusion/query_result', 10)
            resp_msg = MsgString()
            resp_msg.data = json.dumps(response, ensure_ascii=False)
            resp_pub.publish(resp_msg)
            
        except Exception as e:
            self.get_logger().error(f'查询处理失败: {e}')
    
    def _trigger_fusion(self):
        """触发融合计算"""
        if self.nlu_data is None:
            return
        
        # 根据策略进行融合
        if self.fusion_strategy == 'late':
            fused = self._late_fusion()
        elif self.fusion_strategy == 'attention':
            fused = self._attention_fusion()
        else:
            fused = self._simple_fusion()
        
        # 发布融合结果
        fused_msg = String()
        fused_msg.data = json.dumps(fused, ensure_ascii=False)
        self.fusion_pub.publish(fused_msg)
    
    def _late_fusion(self) -> dict:
        """后融合：各模态独立处理后再合并"""
        fused = FusedContext(
            timestamp=self.get_clock().now().to_msg().sec
        )
        
        # 融合 NLU 数据
        if self.nlu_data:
            fused.intent = self.nlu_data.get('intent')
            fused.slots.update(self.nlu_data.get('slots', {}))
            fused.confidence = self.nlu_data.get('confidence', 0.0)
        
        # 融合视觉数据（根据槽位中的物体名称）
        if fused.intent == 'manipulation':
            target_object = fused.slots.get('object_name', '')
            if target_object:
                matched_objects = [
                    obj for obj in self.vision_data
                    if target_object in obj.get('class_cn', '') or 
                       target_object in obj.get('class', '')
                ]
                fused.vision_objects = matched_objects
        
        # 融合机器人状态
        if self.state_data:
            fused.robot_pose = self.state_data.get('pose')
        
        # 生成推理说明
        fused.reasoning = self._generate_reasoning(fused)
        
        return {
            'timestamp': fused.timestamp,
            'intent': fused.intent,
            'slots': fused.slots,
            'vision_objects': fused.vision_objects,
            'robot_pose': fused.robot_pose,
            'confidence': fused.confidence,
            'reasoning': fused.reasoning
        }
    
    def _attention_fusion(self) -> dict:
        """注意力融合：使用注意力机制融合多模态"""
        # 简化实现：基于规则加权
        fused = self._late_fusion()
        fused['fusion_method'] = 'attention'
        return fused
    
    def _simple_fusion(self) -> dict:
        """简单融合：直接拼接"""
        return self._late_fusion()
    
    def _build_fused_context(self) -> dict:
        """构建融合上下文"""
        return self._late_fusion()
    
    def _generate_reasoning(self, fused: FusedContext) -> str:
        """生成推理说明"""
        parts = []
        
        if fused.intent:
            intent_map = {
                'navigation': '用户想要导航到指定位置',
                'manipulation': '用户想要操作物体',
                'information': '用户想要查询信息',
                'greeting': '用户打招呼',
                'control': '用户发送控制命令'
            }
            parts.append(intent_map.get(fused.intent, f'意图: {fused.intent}'))
        
        if fused.vision_objects:
            parts.append(f'检测到相关物体: {len(fused.vision_objects)} 个')
        
        if fused.robot_pose:
            parts.append('机器人位置已知')
        
        return '；'.join(parts) if parts else '无有效信息'

def main(args=None):
    rclpy.init(args=args)
    node = FusionCenter()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

# planner/planner_node.py - 任务规划节点
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import json
from dataclasses import dataclass, field
from typing import List, Dict, Optional, Any

@dataclass
class Action:
    """动作"""
    action_type: str
    params: Dict[str, Any]
    estimated_duration: float = 0.0
    preconditions: List[str] = field(default_factory=list)
    effects: List[str] = field(default_factory=list)

class TaskPlanner(Node):
    """任务规划节点：将高层指令分解为动作序列"""
    
    def __init__(self):
        super().__init__('task_planner')
        
        self.declare_parameter('use_llm_planner', False)
        self.use_llm_planner = self.get_parameter('use_llm_planner').value
        
        # ========== 动作库 ==========
        self.action_library = {
            'go_to': Action(
                action_type='navigation',
                params=['target_position'],
                estimated_duration=10.0,
                preconditions=['robot_at_start'],
                effects=['robot_at_target']
            ),
            'find_object': Action(
                action_type='vision',
                params=['object_name'],
                estimated_duration=5.0,
                preconditions=[],
                effects=['object_localized']
            ),
            'grasp': Action(
                action_type='manipulation',
                params=['object_id', 'grasp_pose'],
                estimated_duration=8.0,
                preconditions=['object_localized', 'robot_near_object'],
                effects=['object_in_hand']
            ),
            'place': Action(
                action_type='manipulation',
                params=['target_position'],
                estimated_duration=5.0,
                preconditions=['object_in_hand'],
                effects=['object_at_target']
            ),
            'speak': Action(
                action_type='tts',
                params=['text'],
                estimated_duration=3.0,
                preconditions=[],
                effects=[]
            )
        }
        
        # ========== ROS2 通信 ==========
        self.intent_sub = self.create_subscription(
            String, '/nlu/intent', self.intent_callback, 10
        )
        self.fusion_sub = self.create_subscription(
            String, '/fusion/context', self.fusion_callback, 10
        )
        self.planner_pub = self.create_publisher(
            String, '/planner/actions', 10
        )
        self.replan_srv = self.create_service(
            String, '/planner/replan', self.replan_callback
        )
        
        self.current_plan: List[Action] = []
        self.fusion_context: Optional[dict] = None
        
        self.get_logger().info('任务规划节点初始化完成')
    
    def intent_callback(self, msg: String):
        """接收意图数据"""
        try:
            intent_data = json.loads(msg.data)
            self._generate_plan(intent_data)
        except json.JSONDecodeError:
            self.get_logger().error('意图数据解析失败')
    
    def fusion_callback(self, msg: String):
        """接收融合上下文"""
        try:
            self.fusion_context = json.loads(msg.data)
        except json.JSONDecodeError:
            pass
    
    def replan_callback(self, request, response):
        """重新规划服务"""
        self.get_logger().info('收到重新规划请求')
        if self.fusion_context:
            self._generate_plan({'intent': self.fusion_context.get('intent'), 'slots': self.fusion_context.get('slots', {})})
        return response
    
    def _generate_plan(self, intent_data: dict):
        """根据意图生成动作序列"""
        intent = intent_data.get('intent', 'unknown')
        slots = intent_data.get('slots', {})
        
        plan: List[Action] = []
        
        if intent == 'navigation':
            plan = self._plan_navigation(slots)
        elif intent == 'manipulation':
            plan = self._plan_manipulation(slots)
        elif intent == 'information':
            plan = self._plan_information(slots)
        elif intent == 'greeting':
            plan = [self.action_library['speak']]
            plan[0].params = {'text': '你好，有什么可以帮你的吗？'}
        elif intent == 'control':
            plan = self._plan_control(slots)
        else:
            plan = [self.action_library['speak']]
            plan[0].params = {'text': '抱歉，我没有理解你的意思。'}
        
        self.current_plan = plan
        self._publish_plan(plan)
    
    def _plan_navigation(self, slots: dict) -> List[Action]:
        """规划导航任务"""
        target_place = slots.get('target_place', '')
        plan = [
            self.action_library['go_to'],
        ]
        plan[0].params = {'target_position': target_place, 'place_name': target_place}
        return plan
    
    def _plan_manipulation(self, slots: dict) -> List[Action]:
        """规划操作任务"""
        object_name = slots.get('object_name', '')
        target_place = slots.get('target_place', '')
        
        plan = [
            self.action_library['go_to'],     # 移动到物体附近
            self.action_library['find_object'],  # 找到物体
            self.action_library['grasp'],     # 抓取
        ]
        
        plan[0].params = {'target_position': slots.get('source_place', 'table')}
        plan[1].params = {'object_name': object_name}
        plan[2].params = {'object_name': object_name}
        
        if target_place:
            plan.append(self.action_library['go_to'])
            plan[3].params = {'target_position': target_place}
            plan.append(self.action_library['place'])
            plan[4].params = {'target_position': target_place}
        
        return plan
    
    def _plan_information(self, slots: dict) -> List[Action]:
        """规划信息查询任务"""
        plan = [self.action_library['speak']]
        plan[0].params = {'text': f'正在查询关于{slots.get("query_topic", "")}的信息。'}
        return plan
    
    def _plan_control(self, slots: dict) -> List[Action]:
        """规划控制任务"""
        action = slots.get('action', '')
        
        if action in ['停止', '暂停']:
            return []
        
        plan = [self.action_library['speak']]
        plan[0].params = {'text': f'好的，{action}。'}
        return plan
    
    def _publish_plan(self, plan: List[Action]):
        """发布动作序列"""
        plan_data = {
            'actions': [
                {
                    'action_type': a.action_type,
                    'params': a.params,
                    'estimated_duration': a.estimated_duration
                }
                for a in plan
            ],
            'total_duration': sum(a.estimated_duration for a in plan)
        }
        
        plan_msg = String()
        plan_msg.data = json.dumps(plan_data, ensure_ascii=False)
        self.planner_pub.publish(plan_msg)
        
        self.get_logger().info(f'发布动作序列: {len(plan)} 个动作')

def main(args=None):
    rclpy.init(args=args)
    node = TaskPlanner()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()

```python
# embodied_voice_assistant/state_manager.py - 状态管理器
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import json
from dataclasses import dataclass, field
from typing import Dict, List, Optional, Any
from datetime import datetime

@dataclass
class ModuleState:
    """模块状态"""
    name: str
    status: str  # ok / warning / error / offline
    message: str = ''
    last_update: float = 0.0

@dataclass
class SystemState:
    """系统状态"""
    state: str = 'initializing'
    modules: Dict[str, ModuleState] = field(default_factory=dict)
    errors: List[str] = field(default_factory=list)
    start_time: Optional[datetime] = None

class StateManager(Node):
    """状态管理器：监控系统各模块状态，处理异常"""
    
    def __init__(self):
        super().__init__('state_manager')
        
        self.declare_parameter('heartbeat_timeout', 10.0)  # 心跳超时（秒）
        self.declare_parameter('max_errors', 10)           # 最大错误数
        
        self.heartbeat_timeout = self.get_parameter('heartbeat_timeout').value
        self.max_errors = self.get_parameter('max_errors').value
        
        # 系统状态
        self.system_state = SystemState(start_time=datetime.now())
        
        # ========== ROS2 通信 ==========
        # 各模块心跳订阅
        self.heartbeat_subs = {}
        self.register_heartbeat_sub('asr', '/asr/heartbeat')
        self.register_heartbeat_sub('nlu', '/nlu/heartbeat')
        self.register_heartbeat_sub('vision', '/vision/heartbeat')
        self.register_heartbeat_sub('motion', '/motion/heartbeat')
        self.register_heartbeat_sub('tts', '/tts/heartbeat')
        self.register_heartbeat_sub('fusion', '/fusion/heartbeat')
        self.register_heartbeat_sub('planner', '/planner/heartbeat')
        
        # 系统状态发布
        self.state_pub = self.create_publisher(
            String, '/system/full_state', 10
        )
        
        # 异常告警发布
        self.alert_pub = self.create_publisher(
            String, '/system/alert', 10
        )
        
        # 定时检查
        self.check_timer = self.create_timer(2.0, self._check_module_health)
        self.publish_timer = self.create_timer(5.0, self._publish_state)
        
        self.get_logger().info('状态管理器初始化完成')
    
    def register_heartbeat_sub(self, module_name: str, topic: str):
        """注册模块心跳订阅"""
        sub = self.create_subscription(
            String, topic,
            lambda msg: self._on_heartbeat(module_name, msg),
            10
        )
        self.heartbeat_subs[module_name] = sub
        
        # 初始化模块状态
        self.system_state.modules[module_name] = ModuleState(
            name=module_name,
            status='offline',
            last_update=0.0
        )
    
    def _on_heartbeat(self, module_name: str, msg: String):
        """处理心跳"""
        try:
            data = json.loads(msg.data)
            status = data.get('status', 'ok')
            message = data.get('message', '')
            
            self.system_state.modules[module_name] = ModuleState(
                name=module_name,
                status=status,
                message=message,
                last_update=self.get_clock().now().to_msg().sec
            )
            
        except json.JSONDecodeError:
            pass
    
    def _check_module_health(self):
        """检查模块健康状态"""
        current_time = self.get_clock().now().to_msg().sec
        alerts = []
        
        for name, module_state in self.system_state.modules.items():
            # 检查超时
            if current_time - module_state.last_update > self.heartbeat_timeout:
                if module_state.status != 'offline':
                    module_state.status = 'offline'
                    alerts.append(f'{name} 模块心跳超时')
        
        # 发布告警
        if alerts:
            alert_msg = String()
            alert_msg.data = json.dumps({
                'alerts': alerts,
                'timestamp': current_time
            }, ensure_ascii=False)
            self.alert_pub.publish(alert_msg)
    
    def _publish_state(self):
        """发布系统状态"""
        state_data = {
            'state': self.system_state.state,
            'modules': {
                name: {
                    'status': m.status,
                    'message': m.message,
                    'last_update': m.last_update
                }
                for name, m in self.system_state.modules.items()
            },
            'error_count': len(self.system_state.errors),
            'uptime': (datetime.now() - self.system_state.start_time).seconds if self.system_state.start_time else 0
        }
        
        state_msg = String()
        state_msg.data = json.dumps(state_data, ensure_ascii=False)
        self.state_pub.publish(state_msg)

def main(args=None):
    rclpy.init(args=args)
    node = StateManager()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
```

### 10.2 简答题答案

**1. 具身智能语音助手的完整工作流程：**

```
用户语音 → 唤醒检测 → VAD切片 → ASR识别 → NLU理解
    → 对话管理 → 任务规划 → 多模态融合 → 执行控制
    → TTS合成 → 语音反馈 → 用户
```

**2. 多模态融合的必要性：**

语音指令往往是不完整或模糊的，例如"把那个东西拿过来"中的"那个东西"需要视觉信息来确认。此外，具身智能需要理解物理环境（物体位置、空间布局），这些信息无法仅从语音获得。多模态融合可以：
- 提高理解准确性
- 消除歧义
- 提供更丰富的上下文
- 实现更智能的决策

**3. ROS2 通信机制对比：**

| 机制 | 特点 | 适用场景 |
|------|------|----------|
| 话题 | 发布-订阅，无返回值，异步 | 持续数据流（传感器数据、状态） |
| 服务 | 请求-应答，同步，1:1 | 一次性查询（查询物体位置） |
| 动作 | 长时间任务，支持反馈，可取消 | 导航、抓取等持续性动作 |

**4. 对话状态机状态说明：**

- **IDLE**：系统空闲状态，等待用户唤醒或输入
- **WAITING_CONFIRM**：当槽位缺失或识别置信度低时，等待用户确认/澄清

这两个状态保证了系统的健壮性和良好的用户体验。

**5. 容错设计举例：**

1. **ASR 识别失败**：降级到关键词匹配模式
2. **导航遇到障碍**：重新规划路径或报告用户
3. **视觉检测失败**：使用深度传感器辅助定位
4. **机械臂抓取失败**：尝试调整抓取策略或报告失败

        # 创建订阅和发布
        self.sub = self.create_subscription(AudioData, '/audio/audio_raw', self.callback)
        self.pub = self.create_publisher(String, '/asr/text')
    
    def callback(self, msg):
        # 音频处理和识别
        result = self.model.transcribe(msg.data)
        self.pub.publish(result['text'])
```

### 10.4 设计题答案要点

**场景："帮我把客厅茶几上的水杯拿过来"**

**1. 意图和槽位理解：**
- 意图：manipulation（操作）
- 槽位：
  - object_name = "水杯"
  - source_place = "客厅茶几"
  - action = "拿过来"

**2. 所需传感器信息：**
- 麦克风（语音输入）
- 摄像头（物体识别）
- 激光雷达（导航避障）
- 里程计（定位）
- 机械臂关节传感器（抓取反馈）

**3. 执行动作步骤：**
```
1. 导航到客厅茶几附近
2. 视觉检测定位水杯
3. 调整机械臂接近水杯
4. 执行抓取动作
5. 导航返回用户位置
6. 放置水杯
7. TTS 反馈"已拿到水杯"
```

**4. 障碍物处理：**
- 导航中使用 Nav2 的动态路径重规划
- 遇到障碍物超时后重新规划或报告用户
- 抓取时使用触觉传感器反馈调整力度

**5. 结果反馈：**
- 成功：TTS "好的，我把水杯拿过来了"
- 失败：TTS "抱歉，水杯抓取失败，可能茶几上没有水杯"
- 部分成功：TTS "我已经拿到水杯了，正在返回"

### B. 常见问题排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| ASR 无法识别 | 麦克风未启动 | 检查音频设备 `arecord -l` |
| 模型加载慢 | 首次运行 | 预加载模型或使用量化版本 |
| 导航失败 | 地图未构建 | 先运行 SLAM 建图 |
| 机械臂不响应 | 服务未启动 | 检查 `/arm/goal` 话题 |
| TTS 无声音 | 音频设备问题 | 检查 `aplay -l` |
| 节点崩溃 | 内存不足 | 关闭其他占用程序 |
