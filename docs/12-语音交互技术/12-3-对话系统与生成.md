## 1. 对话系统概述

### 1.1 什么是对话系统

**对话系统**（Dialogue System / Conversational AI）是能够与人类进行自然语言交互的智能系统。它通过理解用户输入、推理对话状态、生成合理回复，实现人机自然对话。

```
用户："今天天气怎么样？"
         ↓
   [对话系统]
         ↓
系统："今天北京晴，气温25度，适合出行。"
```

对话系统的核心挑战在于：
- **理解**：准确理解用户意图和关键信息
- **推理**：结合上下文进行逻辑推理
- **生成**：生成自然、流畅、符合语境的回复
- **管理**：维护对话状态，实现多轮交互

### 1.2 对话系统的三大类型

根据应用场景和交互模式，对话系统可分为三大类：

| 类型 | 任务型对话 | 问答型对话 | 闲聊型对话 |
|------|-----------|-----------|-----------|
| **目标** | 完成特定任务 | 回答问题 | 情感陪伴 |
| **交互模式** | 多轮、槽位填充 | 单轮/多轮 | 开放域 |
| **典型应用** | 订餐、订票、机器人控制 | 客服问答、知识查询 | 社交聊天、情感陪护 |
| **核心技术** | 对话管理、意图识别 | 检索、阅读理解 | 文本生成、对话策略 |
| **评估指标** | 任务完成率 | 回答准确率 | 流畅度、相关性 |

#### 1.2.1 任务型对话系统

**目标**：帮助用户完成特定任务（如订餐、订票、设备控制）。

```
用户：帮我订一张明天北京到上海的高铁票
系统：请问您想要几点出发的？
用户：上午10点的
系统：好的，已为您找到G101次列车，10:30出发，13:30到达上海虹桥...
用户：就这个吧
系统：已为您下单，请确认支付。
```

**特点**：
- 需要多轮交互收集必填槽位
- 对话状态跟踪（DST）至关重要
- 需要与外部系统（API/数据库）交互
- 错误容忍度低，需精准理解

#### 1.2.2 问答型对话系统

**目标**：回答用户提出的问题。

```
用户：ROS2的节点间通信方式有哪些？
系统：ROS2支持以下几种通信方式：
      1. Topic（话题通信）：发布-订阅模式，适合高频数据传输
      2. Service（服务通信）：请求-应答模式，适合同步调用
      3. Action（动作通信）：适合长时间任务，支持反馈
      4. Parameter（参数服务）：节点参数管理
```

**特点**：
- 通常单轮即可完成
- 依赖知识库或检索系统
- 需要理解问题的语义
- 答案需准确、简洁

#### 1.2.3 闲聊型对话系统

**目标**：与用户进行情感化、开放式的交流。

```
用户：今天心情不好
系统：怎么了？是发生了什么事吗？
用户：工作压力太大了
系统：工作压力大确实很累，适当放松一下很重要。要不要听听音乐缓解一下？
```

**特点**：
- 开放域，无明确任务目标
- 需要理解情感和意图
- 对话风格需多样化
- 评估主观性强

### 1.3 对话系统的核心模块

一个完整的对话系统包含以下核心模块：

```
┌─────────────────────────────────────────────────────────────┐
│                    对话系统架构                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐ │
│  │  用户输入 │ → │   NLU   │ → │ 对话管理 │ → │   NLG   │ → │ 系统回复 │
│  └─────────┘    └─────────┘    └─────────┘    └─────────┘ │
│                      ↓              ↓              ↓         │
│                 意图+槽位      对话状态       自然语言        │
│                                                             │
│                    ┌─────────────┐                        │
│                    │ 知识库/API   │                        │
│                    │（外部资源）  │                        │
│                    └─────────────┘                        │
└─────────────────────────────────────────────────────────────┘
```

**各模块职责**：

| 模块 | 职责 | 输入 | 输出 |
|------|------|------|------|
| **NLU** | 理解用户输入 | 原始文本 | 意图+槽位 |
| **对话管理（DM）** | 维护状态、决定动作 | NLU结果+历史 | 系统动作 |
| **NLG** | 生成自然语言回复 | 系统动作+上下文 | 回复文本 |
| **知识库/API** | 提供领域知识 | 查询请求 | 检索结果 |

### 1.4 机器人对话系统的应用场景

在具身智能机器人中，对话系统承担着重要的人机交互职责：

| 场景 | 典型对话 | 技术要点 |
|------|---------|---------|
| **家庭服务** | "帮我把客厅灯打开"、"播放音乐" | 设备控制、多轮确认 |
| **导览讲解** | "这个展品有什么故事？" | 知识问答、上下文理解 |
| **陪伴聊天** | "今天过得怎么样？" | 情感识别、闲聊生成 |
| **任务指引** | "去厨房怎么走？" | 导航+知识库 |
| **医疗辅助** | "提醒我每天早上8点吃药" | 闹钟+日志记录 |
| **教育互动** | "解释一下什么是ROS2" | 知识问答、多轮解释 |

### 1.5 对话系统发展历程

```
1990s ──→ 2000s ──→ 2010s ──→ 2020s ──→ Today
  │         │         │         │         │
  ▼         ▼         ▼         ▼         ▼
基于规则   统计ML    深度学习   预训练    大模型

```

- **1990s以前**：基于规则的专家系统，模板匹配

- **2020s**：预训练语言模型（BERT、GPT系列）
- **Today**：大语言模型（GPT-4、ChatGLM、Qwen）端到端对话

## 2. 自然语言生成NLG

### 2.1 什么是NLG

**自然语言生成**（Natural Language Generation，NLG）是根据结构化数据或语义表示生成自然语言文本的过程，是对话系统的核心输出模块。

```python
# NLG示例：结构化数据 → 自然语言
# 输入：结构化数据
data = {
    "intent": "light_control",
    "slots": {"location": "客厅", "action": "关", "target": "灯"},
    "success": True
}

# NLG输出
output = "好的，已为您关闭客厅的灯。"
```

**NLG在对话系统中的位置**：

```
对话管理 → [动作/指令] → NLG → [自然语言回复]
              ↓                    ↓
           执行命令              用户可读文本
```

### 2.2 NLG方法的演进

| 时代 | 方法 | 特点 | 代表模型 |
|------|------|------|---------|
| 早期 | 模板填充 | 固定句式，替换槽位 | AIML |
| 中期 | 基于检索 | 从语料库检索相似回复 | 检索式对话 |
| 后期 | Seq2Seq生成 | 端到端神经网络生成 | LSTM-Seq2Seq |
| 现代 | 预训练生成 | 大规模预训练+微调 | GPT、ChatGLM |
| 当代 | 大模型对话 | 指令微调+RLHF | GPT-4、Qwen |

### 2.3 模板生成方法

**模板填充**是最简单也是工业界最常用的NLG方法。

```python
from string import Template

class TemplateBasedNLG:
    """
    基于模板的NLG生成器

    优点：可控、可解释、无幻觉
    缺点：灵活性差、需大量模板
    """

    def __init__(self):
        # 定义意图-模板映射
        self.templates = {
            'light_on': Template("好的，已为您打开${location}的${target}。"),
            'light_off': Template("好的，已为您关闭${location}的${target}。"),
            'music_play': Template("好的，正在为您播放${song}。"),
            'navigation_start': Template("好的，正在为您导航到${destination}。"),
            'alarm_set': Template("已为您设置${time}的闹钟。"),
            'confirm': Template("请问您是想${action}${target}吗？"),
            'greeting': Template("您好！有什么可以帮助您的？"),
            'apology': Template("抱歉，${reason}。请重新描述您的需求。"),
            'unknown': Template("抱歉，我没有理解您说的话，请再说一次？"),
        }

    def generate(self, intent, slots=None, context=None):
        """
        根据意图和槽位生成回复

        Args:
            intent: 意图名称
            slots: 槽位字典
            context: 上下文信息（可选）
        Returns:
            str: 生成的文本
        """
        slots = slots or {}
        template = self.templates.get(intent, self.templates['unknown'])

        try:
            output = template.safe_substitute(slots)
            return output
        except KeyError:
            return self.templates['unknown'].safe_substitute()

# 测试
if __name__ == "__main__":
    nlg = TemplateBasedNLG()

    test_cases = [
        ('light_on', {'location': '客厅', 'target': '灯'}),
        ('light_off', {'location': '卧室', 'target': '灯'}),
        ('music_play', {'song': '周杰伦的歌'}),
        ('navigation_start', {'destination': '北京站'}),
        ('alarm_set', {'time': '明天早上7点'}),
        ('confirm', {'action': '打开', 'target': '灯'}),
    ]

    print("=" * 60)
    print("模板NLG生成结果")
    print("=" * 60)

    for intent, slots in test_cases:
        result = nlg.generate(intent, slots)
        print(f"\n[{intent}] slots={slots}")
        print(f"  → {result}")
```

### 2.4 基于检索的生成

**检索式对话**从预先构建的对话语料库中检索最相似的回复。

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

class RetrievalBasedNLG:
    """
    基于检索的NLG生成器

    工作流程：
    1. 准备（query, response）语料库
    2. 将query向量化
    3. 用户输入时，检索最相似的query
    4. 返回对应的response
    """

    def __init__(self):
        # 对话语料库：(用户query, 系统回复)
        self.corpus = [
            # 灯光控制
            ("打开灯", "好的，已为您开灯。"),
            ("关掉灯", "好的，已为您关灯。"),
            ("把灯打开", "已为您打开灯。"),
            ("关闭客厅灯", "已关闭客厅灯。"),

            # 导航
            ("去厨房怎么走", "厨房在左手边第二个门。"),
            ("导航到门口", "好的，正在为您导航到门口。"),
            ("洗手间在哪", "洗手间在走廊尽头右转。"),

            # 音乐
            ("播放音乐", "好的，正在为您播放音乐。"),
            ("放首歌", "好的，为您播放一首轻音乐。"),
            ("停止播放", "已停止播放。"),

            # 天气
            ("今天天气怎么样", "今天天气晴朗，气温25度。"),
            ("会下雨吗", "根据预报，今天暂时不会下雨。"),

            # 闲聊
            ("你好", "您好！有什么可以帮助您的？"),
            ("你是谁", "我是您的机器人助手，有什么可以帮您的？"),
            ("谢谢", "不客气！还有其他需要吗？"),
            ("再见", "再见，有需要随时叫我！"),
        ]

        # 离线构建TF-IDF索引
        self._build_index()

    def _build_index(self):
        """构建TF-IDF索引"""
        queries = [item[0] for item in self.corpus]
        self.vectorizer = TfidfVectorizer(analyzer='char', ngram_range=(1, 3))
        self.query_vectors = self.vectorizer.fit_transform(queries)
        print(f"✅ 检索索引构建完成，共 {len(self.corpus)} 条语料")

    def retrieve(self, query, top_k=1):
        """
        检索最相似的回复

        Args:
            query: 用户输入
            top_k: 返回top-k个结果
        Returns:
            list: [(相似度, 回复), ...]
        """
        query_vector = self.vectorizer.transform([query])
        similarities = cosine_similarity(query_vector, self.query_vectors)[0]
        top_indices = np.argsort(similarities)[::-1][:top_k]

        results = []
        for idx in top_indices:
            results.append({
                'query': self.corpus[idx][0],
                'response': self.corpus[idx][1],
                'similarity': float(similarities[idx])
            })

        return results

    def generate(self, query, context=None):
        """
        生成回复

        Args:
            query: 用户输入
            context: 对话历史（可选）
        Returns:
            str: 生成的回复
        """
        results = self.retrieve(query, top_k=1)

        if results and results[0]['similarity'] > 0.3:
            return results[0]['response']
        else:
            return "抱歉，我没有理解您的意思，请重新描述一下？"

# 测试
if __name__ == "__main__":
    nlg = RetrievalBasedNLG()

    test_queries = [
        "打开客厅灯",
        "把灯关掉",
        "去厨房怎么走",
        "播放音乐",
        "今天天气好吗",
        "你好啊"
    ]

    print("\n" + "=" * 60)
    print("检索式NLG生成结果")
    print("=" * 60)

    for query in test_queries:
        results = nlg.retrieve(query, top_k=3)
        print(f"\n输入: '{query}'")
        for i, r in enumerate(results):
            print(f"  [{i+1}] sim={r['similarity']:.3f} | query='{r['query']}' → response='{r['response']}'")
```

### 2.5 基于Seq2Seq的生成

**Seq2Seq（Sequence-to-Sequence）**模型是深度学习时代NLG的主流方法，使用编码器-解码器架构。

```python
import torch
import torch.nn as nn
from transformers import BertModel

class Seq2SeqNLG(nn.Module):
    """
    基于Seq2Seq的NLG模型

    架构：Encoder (BERT) + Decoder (LSTM) + Attention
    用于：将语义表示转换为自然语言
    """

    def __init__(self, vocab_size=21128, hidden_size=768, num_layers=2):
        super().__init__()

        # 编码器：使用BERT
        self.encoder = BertModel.from_pretrained("bert-base-chinese")
        self.encoder_hidden_size = hidden_size

        # 注意力机制
        self.attention = nn.MultiheadAttention(
            embed_dim=hidden_size,
            num_heads=8,
            batch_first=True
        )

        # 解码器LSTM
        self.decoder_lstm = nn.LSTM(
            input_size=hidden_size + hidden_size,
            hidden_size=hidden_size,
            num_layers=num_layers,
            batch_first=True
        )

        # 输出层
        self.fc = nn.Linear(hidden_size, vocab_size)

        # 词嵌入
        self.embedding = nn.Embedding(vocab_size, hidden_size)

        # 始末标记
        self.start_token_id = 101  # [CLS]
        self.end_token_id = 102     # [SEP]

    def forward(self, encoder_inputs, decoder_inputs):
        """
        前向传播

        Args:
            encoder_inputs: 编码器输入 (batch, seq_len)
            decoder_inputs: 解码器输入 (batch, seq_len)
        Returns:
            logits: (batch, seq_len, vocab_size)
        """
        encoder_outputs = self.encoder(input_ids=encoder_inputs).last_hidden_state

        batch_size = encoder_inputs.size(0)
        decoder_hidden = (
            torch.zeros(2, batch_size, self.encoder_hidden_size).to(encoder_inputs.device),
            torch.zeros(2, batch_size, self.encoder_hidden_size).to(encoder_inputs.device)
        )

        outputs = []
        max_len = decoder_inputs.size(1)

        for t in range(max_len):
            if t == 0:
                decoder_input = self.embedding(
                    torch.full((batch_size,), self.start_token_id, device=encoder_inputs.device)
                ).unsqueeze(1)
            else:
                decoder_input = self.embedding(decoder_inputs[:, t-1:t])

            attn_output, _ = self.attention(
                query=decoder_input,
                key=encoder_outputs,
                value=encoder_outputs
            )

            lstm_input = torch.cat([decoder_input, attn_output], dim=-1)
            decoder_output, decoder_hidden = self.decoder_lstm(lstm_input, decoder_hidden)
            logits = self.fc(decoder_output)
            outputs.append(logits)

        return torch.cat(outputs, dim=1)


class SimpleSeq2SeqNLG:
    """
    简化版Seq2Seq NLG（用于演示）
    实际应用中建议使用预训练模型
    """

    def __init__(self):
        self.intent_responses = {
            'greet': [
                "你好！有什么我可以帮助你的？",
                "您好！我是机器人助手。",
                "嗨，很高兴为您服务！"
            ],
            'light_on': [
                "好的，已为您打开${location}的${target}。",
                "已打开${location}${target}。"
            ],
            'light_off': [
                "好的，已为您关闭${location}的${target}。",
                "已关闭${location}${target}。"
            ],
            'music_play': [
                "好的，正在为您播放${song}。",
                "已为您开始播放${song}。"
            ],
            'navigation': [
                "好的，正在为您导航到${destination}。",
                "开始导航，目标：${destination}。"
            ],
            'alarm_set': [
                "已为您设置${time}的闹钟。",
                "${time}的闹钟已设定好。"
            ],
            'confirm': [
                "请问您是想${action}${target}吗？",
                "我理解为${action}${target}，对吗？"
            ],
            'thanks': [
                "不客气！还有什么需要帮助的吗？",
                "谢谢您的使用！"
            ],
            'goodbye': [
                "再见，有需要随时叫我！",
                "拜拜，祝您愉快！"
            ],
            'unknown': [
                "抱歉，我没有理解您的意思，请再说一次？",
                "这个我暂时没法处理，您可以换种方式描述吗？"
            ]
        }
        import random
        self.random = random

    def _fill_template(self, template, slots):
        """填充模板"""
        result = template
        for key, value in slots.items():
            result = result.replace(f'${{{key}}}', str(value))
        return result

    def generate(self, intent, slots=None, context=None):
        """
        生成NLG回复

        Args:
            intent: 意图名称
            slots: 槽位字典
            context: 对话历史（可选）
        Returns:
            str: 生成的回复
        """
        slots = slots or {}
        responses = self.intent_responses.get(intent, self.intent_responses['unknown'])
        template = self.random.choice(responses)
        response = self._fill_template(template, slots)
        return response
```

### 2.6 大语言模型（LLM）生成

使用大语言模型进行NLG是当前最主流的方法，支持零样本和少样本生成。

```python
import os
import json

class LLMNLG:
    """
    基于大语言模型的NLG生成器

    支持多种LLM后端：
    - OpenAI GPT
    - ChatGLM（本地部署）
    - Qwen（阿里通义）
    """

    def __init__(self, backend="openai", model=None):
        self.backend = backend
        self.model = model or self._get_default_model()

    def _get_default_model(self):
        """获取默认模型"""
        defaults = {
            "openai": "gpt-3.5-turbo",
            "chatglm": "THUDM/chatglm3-6b",
            "qwen": "Qwen/Qwen-7B-Chat"
        }
        return defaults.get(self.backend, "gpt-3.5-turbo")

    def generate(self, intent, slots=None, context=None, style="礼貌"):
        """
        使用LLM生成NLG回复

        Args:
            intent: 意图名称
            slots: 槽位字典
            context: 对话历史
            style: 回复风格（礼貌/简洁/幽默）
        Returns:
            str: 生成的回复
        """
        slots = slots or {}
        prompt = self._build_prompt(intent, slots, context, style)

        if self.backend == "openai":
            return self._generate_openai(prompt)
        elif self.backend == "chatglm":
            return self._generate_chatglm(prompt)
        else:
            return self._fallback_generate(intent, slots)

    def _build_prompt(self, intent, slots, context, style):
        """构建生成提示"""
        context_text = ""
        if context:
            context_text = "\n对话历史：\n" + "\n".join([
                f"  用户：{h['user']}\n  助手：{h['assistant']}"
                for h in context[-3:]
            ])

        prompt = f"""你是一个机器人助手的自然语言生成模块。

当前对话状态：
- 意图：{intent}
- 槽位信息：{json.dumps(slots, ensure_ascii=False)}
- 回复风格：{style}
{context_text}

请生成一句自然、流畅的回复。直接输出回复内容，不要加引号或解释："""

        return prompt

    def _generate_openai(self, prompt):
        """调用OpenAI API"""
        try:
            from openai import OpenAI
            client = OpenAI(api_key=os.environ.get("OPENAI_API_KEY"))
            response = client.chat.completions.create(
                model=self.model,
                messages=[{"role": "user", "content": prompt}],
                temperature=0.7,
                max_tokens=100
            )
            return response.choices[0].message.content.strip()
        except Exception as e:
            print(f"OpenAI API调用失败: {e}")
            return self._fallback_generate("", {})

    def _generate_chatglm(self, prompt):
        """调用ChatGLM（本地）"""
        try:
            from transformers import AutoTokenizer, AutoModel
            return self._fallback_generate("", {})
        except Exception as e:
            print(f"ChatGLM调用失败: {e}")
            return self._fallback_generate("", {})

    def _fallback_generate(self, intent, slots):
        """备用生成方法"""
        templates = {
            'greet': "您好，有什么可以帮助您的？",
            'light_on': f"好的，已为您打开{slots.get('location', '')}{slots.get('target', '设备')}。",
            'light_off': f"好的，已为您关闭{slots.get('location', '')}{slots.get('target', '设备')}。",
            'music_play': f"好的，正在为您播放{slots.get('song', '音乐')}。",
            'navigation': f"好的，正在导航到{slots.get('destination', '目的地')}。",
            'alarm_set': f"已为您设置{slots.get('time', '')}的闹钟。",
            'confirm': "请问是这个意思吗？",
            'thanks': "不客气！",
            'goodbye': "再见！",
            'unknown': "抱歉，我没有理解，请再说一次。",
        }
        return templates.get(intent, "好的，已收到。")


class HybridNLG:
    """
    混合NLG生成器

    结合模板+检索+LLM三种方法：
    1. 高频简单场景 → 模板（快且准确）
    2. 中频场景 → 检索（有一定多样性）
    3. 复杂/开放场景 → LLM（灵活但慢）
    """

    def __init__(self, llm_backend="openai"):
        self.template_nlg = TemplateBasedNLG()
        self.retrieval_nlg = RetrievalBasedNLG()
        self.llm_nlg = LLMNLG(backend=llm_backend)
        self.template_intents = {'light_on', 'light_off', 'alarm_set'}
        self.retrieval_intents = {'greeting', 'thanks', 'goodbye', 'confirm'}

    def generate(self, intent, slots=None, context=None, user_query=None):
        """
        混合生成

        Args:
            intent: 意图
            slots: 槽位
            context: 对话历史
            user_query: 用户原始query（用于检索）
        Returns:
            str: 生成的回复
        """
        slots = slots or {}

        # 方法1：模板（最快，高频简单场景）
        if intent in self.template_intents:
            return self.template_nlg.generate(intent, slots)

        # 方法2：检索（有一定多样性）
        if intent in self.retrieval_intents and user_query:
            result = self.retrieval_nlg.generate(user_query, context)
            if result != "抱歉，我没有理解您的意思，请重新描述一下？":
                return result

        # 方法3：LLM（最灵活，复杂场景）
        if context or intent not in self.template_intents:
            return self.llm_nlg.generate(intent, slots, context)

        return self.template_nlg.generate(intent, slots)
```

### 3.2 对话状态跟踪（DST）

**对话状态跟踪**（Dialogue State Tracking，DST）是维护对话上下文的核心模块。

```python
class DialogueStateTracker:
    """
    对话状态跟踪器

    维护对话状态：
    - belief_state：用户相信的事实（槽位填充）
    - intent：当前意图
    - history：对话历史
    - turn：当前轮次
    """

    def __init__(self):
        self.state = {
            'turn': 0,
            'intent': None,
            'belief_state': {},
            'history': [],
            'requested_slots': [],
            'system_action': None,
            'confirm_slots': [],
        }

        # 意图对应的必填槽位
        self.required_slots = {
            'light_control': ['target'],
            'navigation': ['destination'],
            'alarm_set': ['time'],
            'music_play': [],
            'weather_query': ['location'],
        }

        self.slot_types = {
            'location': str,
            'target': str,
            'action': str,
            'destination': str,
            'time': str,
            'song': str,
            'device': str,
        }

    def update(self, nlu_result, system_action=None):
        """
        更新对话状态

        Args:
            nlu_result: NLU解析结果，包含intent和slots
            system_action: 系统动作（可选）
        """
        self.state['turn'] += 1

        # 更新意图
        new_intent = nlu_result.get('intent')
        if new_intent and new_intent != 'unknown':
            self.state['intent'] = new_intent

        # 更新信念状态（槽位填充）
        new_slots = nlu_result.get('slots', {})
        self.state['belief_state'].update(new_slots)

        # 记录系统动作
        if system_action:
            self.state['system_action'] = system_action

        # 记录历史
        self.state['history'].append({
            'turn': self.state['turn'],
            'user': nlu_result.get('original_text', ''),
            'intent': new_intent,
            'slots': new_slots.copy(),
            'system_action': system_action,
        })

    def get_state(self):
        """获取当前对话状态"""
        return self.state.copy()

    def get_missing_slots(self):
        """获取未填充的必填槽位"""
        intent = self.state['intent']
        if not intent or intent not in self.required_slots:
            return []

        required = self.required_slots[intent]
        missing = []
        for slot in required:
            if slot not in self.state['belief_state'] or not self.state['belief_state'][slot]:
                missing.append(slot)
        return missing

    def get_last_user_input(self):
        """获取上一轮用户输入"""
        if self.state['history']:
            return self.state['history'][-1].get('user', '')
        return ''

    def reset(self):
        """重置对话状态"""
        self.__init__()

    def to_dict(self):
        """序列化状态（用于调试/日志）"""
        return {
            'turn': self.state['turn'],
            'intent': self.state['intent'],
            'belief_state': self.state['belief_state'],
            'missing_slots': self.get_missing_slots(),
        }
```

### 3.3 对话策略（Policy）

**对话策略**根据当前对话状态决定系统应采取的动作。

```python
class DialoguePolicy:
    """
    对话策略模块

    决定系统动作：
    - 询问缺失槽位
    - 确认用户意图
    - 执行任务
    - 道歉/拒绝
    """

    def __init__(self):
        self.tracker = DialogueStateTracker()

        # 意图到动作的映射
        self.intent_to_action = {
            'light_on': 'execute_light_control',
            'light_off': 'execute_light_control',
            'music_play': 'execute_music_play',
            'navigation': 'execute_navigation',
            'alarm_set': 'execute_alarm_set',
            'stop': 'execute_stop',
        }

        # 追问模板
        self.request_templates = {
            'location': "请问是哪个位置？客厅还是卧室？",
            'target': "请问要控制什么设备？",
            'destination': "请问您要去哪里？",
            'time': "请问要设置几点？",
            'song': "请问想听什么歌曲？",
        }

    def decide(self, nlu_result):
        """
        根据NLU结果决定系统动作

        Args:
            nlu_result: NLU解析结果
        Returns:
            dict: 系统动作
        """
        # 更新对话状态
        self.tracker.update(nlu_result)
        state = self.tracker.get_state()
        intent = state['intent']

        # 策略1：意图未知
        if intent == 'unknown' or not intent:
            return {
                'action': 'clarify',
                'message': "抱歉，我没有理解您的意思，请重新描述一下？"
            }

        # 策略2：检查必填槽位
        missing = self.tracker.get_missing_slots()
        if missing:
            next_slot = missing[0]
            return {
                'action': 'request_slot',
                'slot': next_slot,
                'message': self.request_templates.get(next_slot, f"请提供{next_slot}信息")
            }

        # 策略3：意图置信度低，需要确认
        confidence = nlu_result.get('confidence', 1.0)
        if confidence < 0.7:
            return {
                'action': 'confirm',
                'intent': intent,
                'slots': state['belief_state'],
                'message': self._generate_confirm_message(intent, state['belief_state'])
            }

        # 策略4：执行动作
        if intent in self.intent_to_action:
            return {
                'action': self.intent_to_action[intent],
                'intent': intent,
                'slots': state['belief_state'],
                'message': self._generate_success_message(intent, state['belief_state'])
            }

        # 策略5：闲聊意图
        if intent in ['greeting', 'thanks', 'goodbye']:
            return {
                'action': 'respond',
                'intent': intent,
                'message': self._generate_social_message(intent)
            }

        # 默认
        return {
            'action': 'wait',
            'message': "好的，请继续..."
        }

    def _generate_confirm_message(self, intent, slots):
        """生成确认消息"""
        if intent in ['light_on', 'light_off']:
            loc = slots.get('location', '')
            target = slots.get('target', '灯')
            action = "打开" if intent == 'light_on' else "关闭"
            return f"我理解为：{action}{loc}{target}，对吗？"
        elif intent == 'navigation':
            dest = slots.get('destination', '')
            return f"我理解为：您想去{dest}，对吗？"
        elif intent == 'alarm_set':
            time = slots.get('time', '')
            return f"我理解为：您想设置{time}的闹钟，对吗？"
        else:
            return f"我理解您的意图是{intent}，对吗？"

    def _generate_success_message(self, intent, slots):
        """生成执行成功消息"""
        messages = {
            'light_on': f"好的，已为您打开{slots.get('location', '')}{slots.get('target', '灯')}。",
            'light_off': f"好的，已为您关闭{slots.get('location', '')}{slots.get('target', '灯')}。",
            'music_play': f"好的，正在为您播放{slots.get('song', '音乐')}。",
            'navigation': f"好的，正在为您导航到{slots.get('destination', '目的地')}。",
            'alarm_set': f"已为您设置{slots.get('time', '')}的闹钟。",
            'stop': "好的，已停止当前操作。",
        }
        return messages.get(intent, "好的，已执行您的指令。")

    def _generate_social_message(self, intent):
        """生成社交回复"""
        messages = {
            'greeting': "您好！有什么可以帮助您的？",
            'thanks': "不客气！",
            'goodbye': "再见，有需要随时叫我！"
        }
        return messages.get(intent, "好的。")
```

### 3.4 端到端对话模型

端到端对话模型用单一神经网络直接从用户输入映射到系统回复，省去模块化设计的繁琐。

```python
from transformers import AutoTokenizer, AutoModelForSeq2SeqLM

class EndToEndDialogueModel:
    """
    端到端对话模型

    - 对话理解
    - 状态跟踪
    - 响应生成
    """

    def __init__(self, model_name="google/flan-t5-base"):

        self.dialogue_history = []
```

### 4.2 意图与槽位解析

```python
class TaskOrientedParser:
    """
    任务型对话解析器

    专门处理任务型对话的NLU解析：
    - 意图分类
    - 槽位提取
    - 置信度评估
    """

    def __init__(self):
        # 设备词典
        self.devices = ["灯", "门", "窗", "空调", "电视", "窗帘", "音响"]
        self.locations = ["客厅", "卧室", "厨房", "卫生间", "书房", "阳台", "门口"]
        self.actions = {
            "开": "on", "打开": "on", "关": "off",
            "关闭": "off", "关掉": "off", "关一下": "off"
        }

        # 意图定义
        self.intents = {
            'light_control': ['灯', '灯开', '灯关'],
            'door_control': ['门', '开门', '关门'],
            'ac_control': ['空调', '冷气', '暖气', '空调开', '空调关'],
            'music_control': ['音乐', '歌', '播放', '停止播放'],
            'navigation': ['去', '导航', '怎么走', '在哪'],
            'alarm_set': ['闹钟', '提醒', '定时'],
        }

    def parse(self, text):
        """
        解析用户输入

        Args:
            text: 用户输入文本
        Returns:
            dict: 解析结果
        """
        text = text.strip()
        intent = self._detect_intent(text)
        slots = self._extract_slots(text, intent)

        return {
            'text': text,
            'intent': intent,
            'slots': slots,
            'confidence': 0.9 if intent != 'unknown' else 0.0
        }

    def _detect_intent(self, text):
        """检测意图"""
        for intent, keywords in self.intents.items():
            for kw in keywords:
                if kw in text:
                    return intent
        return 'unknown'

    def _extract_slots(self, text, intent):
        """提取槽位"""
        slots = {}

        # 提取位置
        for loc in self.locations:
            if loc in text:
                slots['location'] = loc
                break

        # 提取操作
        for action, value in self.actions.items():
            if action in text:
                slots['action'] = value
                break

        # 提取设备
        for dev in self.devices:
            if dev in text:
                slots['target'] = dev
                break

        # 导航目的地
        if intent == 'navigation':
            import re
            patterns = [
                r'去(.+?)怎么走',
                r'到(.+?)去',
                r'去(.+?)$',
            ]
            for p in patterns:
                m = re.search(p, text)
                if m:
                    slots['destination'] = m.group(1)
                    break

        return slots
```

```python
from enum import Enum

class DialogueState(Enum):
    """对话状态枚举"""
    IDLE = "idle"                    # 初始状态
    INTENT_DETECTED = "intent_detected"  # 意图已识别
    SLOT_FILLING = "slot_filling"    # 槽位填充中
    CONFIRMATION = "confirmation"    # 等待确认
    EXECUTING = "executing"          # 执行中
    COMPLETED = "completed"          # 完成
    FAILED = "failed"                # 失败


class TaskDialogueStateMachine:
    """
    任务型对话状态机

    状态流转：
    IDLE → INTENT_DETECTED → SLOT_FILLING → CONFIRMATION → EXECUTING → COMPLETED
                  ↓                              ↓
              FAILED                        FAILED
    """

    def __init__(self):
        self.state = DialogueState.IDLE
        self.intent = None
        self.slots = {}
        self.required_slots = {
            'light_control': ['target', 'action'],
            'navigation': ['destination'],
            'alarm_set': ['time'],
        }

    def transition(self, event, data=None):
        """
        状态转换

        Args:
            event: 事件名称
            data: 附加数据
        Returns:
            tuple: (新状态, 系统动作, 回复消息)
        """
        current_state = self.state

        # 状态转换逻辑
        if self.state == DialogueState.IDLE:
            if event == 'user_input':
                intent = data.get('intent')
                if intent and intent != 'unknown':
                    self.intent = intent
                    self.slots.update(data.get('slots', {}))
                    self.state = DialogueState.INTENT_DETECTED

                    # 检查必填槽位
                    missing = self._get_missing_slots()
                    if missing:
                        self.state = DialogueState.SLOT_FILLING
                        return (self.state, 'request_slot', self._get_slot_prompt(missing[0]))
                    else:
                        self.state = DialogueState.CONFIRMATION
                        return (self.state, 'confirm', self._get_confirm_message())
                else:
                    return (self.state, 'clarify', "抱歉，我没有理解您的意思。")

        elif self.state == DialogueState.SLOT_FILLING:
            if event == 'user_input':
                self.slots.update(data.get('slots', {}))
                missing = self._get_missing_slots()

                if missing:
                    return (self.state, 'request_slot', self._get_slot_prompt(missing[0]))
                else:
                    self.state = DialogueState.CONFIRMATION
                    return (self.state, 'confirm', self._get_confirm_message())

        elif self.state == DialogueState.CONFIRMATION:
            if event == 'user_confirm':
                if data.get('confirmed'):
                    self.state = DialogueState.EXECUTING
                    return (self.state, 'execute', self._get_execute_message())
                else:
                    self.state = DialogueState.IDLE
                    return (self.state, 'cancel', "好的，已取消。还有什么需要帮助的吗？")

        elif self.state == DialogueState.EXECUTING:
            if event == 'execution_complete':
                success = data.get('success', False)
                if success:
                    self.state = DialogueState.COMPLETED
                    return (self.state, 'inform', self._get_success_message())
                else:
                    self.state = DialogueState.FAILED
                    return (self.state, 'apology', f"抱歉，{data.get('reason', '执行失败')}。")

        elif self.state == DialogueState.COMPLETED:
            if event == 'user_input':
                # 新对话
                self.reset()
                self.state = DialogueState.INTENT_DETECTED
                self.intent = data.get('intent')
                self.slots.update(data.get('slots', {}))
                missing = self._get_missing_slots()
                if missing:
                    self.state = DialogueState.SLOT_FILLING
                    return (self.state, 'request_slot', self._get_slot_prompt(missing[0]))
                else:
                    self.state = DialogueState.CONFIRMATION
                    return (self.state, 'confirm', self._get_confirm_message())

        return (self.state, 'wait', "请稍候...")

    def _get_missing_slots(self):
        """获取缺失的必填槽位"""
        if not self.intent or self.intent not in self.required_slots:
            return []
        return [s for s in self.required_slots[self.intent] if s not in self.slots]

    def _get_slot_prompt(self, slot):
        """生成槽位追问"""
        prompts = {
            'location': "请问是哪个位置？客厅、卧室还是厨房？",
            'target': "请问要控制什么设备？",
            'action': "请问是要打开还是关闭？",
            'destination': "请问您要去哪里？",
            'time': "请问要设置几点？",
            'song': "请问想听什么歌曲？",
        }
        return prompts.get(slot, f"请提供{slot}信息")

    def _get_confirm_message(self):
        """生成确认消息"""
        if self.intent == 'light_control':
            loc = self.slots.get('location', '')
            target = self.slots.get('target', '灯')
            action = "打开" if self.slots.get('action') == 'on' else "关闭"
            return f"确认执行：{action}{loc}{target}，对吗？"
        elif self.intent == 'navigation':
            return f"确认导航到{self.slots.get('destination', '目的地')}，对吗？"
        elif self.intent == 'alarm_set':
            return f"确认设置{self.slots.get('time', '')}的闹钟，对吗？"
        return f"确认执行{self.intent}，对吗？"

    def _get_execute_message(self):
        """生成执行消息"""
        return "好的，正在执行..."

    def _get_success_message(self):
        """生成成功消息"""
        if self.intent == 'light_control':
            action = "打开" if self.slots.get('action') == 'on' else "关闭"
            return f"已完成：{action}{self.slots.get('location', '')}{self.slots.get('target', '灯')}。"
        elif self.intent == 'navigation':
            return f"已为您导航到{self.slots.get('destination', '目的地')}。"
        elif self.intent == 'alarm_set':
            return f"已设置{self.slots.get('time', '')}的闹钟。"
        return "操作已完成。"

    def reset(self):
        """重置状态机"""
        self.state = DialogueState.IDLE
        self.intent = None
        self.slots = {}
```

### 5.1 Seq2Seq闲聊模型

闲聊型对话系统不需要完成具体任务，重点在于生成自然、连贯、有趣的回复。

```python
import torch
import torch.nn as nn
from transformers import BertTokenizer, BertModel

class Seq2SeqChitchat(nn.Module):
    """
    Seq2Seq闲聊模型

    编码器：BERT
    解码器：LSTM + Attention
    """

    def __init__(self, vocab_size=21128, hidden_size=768):
        super().__init__()
        self.encoder = BertModel.from_pretrained("bert-base-chinese")
        self.hidden_size = hidden_size

        # 注意力
        self.attention = nn.MultiheadAttention(hidden_size, num_heads=8, batch_first=True)

        # 解码器
        self.decoder = nn.LSTM(
            input_size=hidden_size,
            hidden_size=hidden_size,
            num_layers=2,
            batch_first=True
        )

        # 输出
        self.fc = nn.Linear(hidden_size, vocab_size)
        self.embedding = nn.Embedding(vocab_size, hidden_size)

    def forward(self, encoder_input, decoder_input):
        """前向传播"""
        encoder_outputs = self.encoder(input_ids=encoder_input).last_hidden_state
        batch_size = encoder_input.size(0)

        decoder_hidden = (
            torch.zeros(2, batch_size, self.hidden_size).to(encoder_input.device),
            torch.zeros(2, batch_size, self.hidden_size).to(encoder_input.device)
        )

        outputs = []
        for t in range(decoder_input.size(1)):
            if t == 0:
                dec_in = self.embedding(torch.zeros(batch_size, dtype=torch.long, device=encoder_input.device).unsqueeze(1))
            else:
                dec_in = self.embedding(decoder_input[:, t-1:t])

            attn_out, _ = self.attention(dec_in, encoder_outputs, encoder_outputs)
            dec_out, decoder_hidden = self.decoder(attn_out, decoder_hidden)
            logits = self.fc(dec_out)
            outputs.append(logits)

        return torch.cat(outputs, dim=1)
```

```python
            self.tokenizer = AutoTokenizer.from_pretrained(self.model_path, trust_remote_code=True)
            self.model = AutoModelForCausalLM.from_pretrained(self.model_path, torch_dtype=torch.float16).cuda()
            self.model.eval()
            print("✅ Qwen加载成功")
        except Exception as e:
            print(f"⚠️ Qwen加载失败: {e}")
            self.model = None

    @torch.no_grad()
    def chat(self, user_input, history=None):
        """
        对话

        Args:
            user_input: 用户输入
            history: 对话历史 [(user, assistant), ...]
        Returns:
            str: 模型回复
        """
        if self.model is None:
            return "模型未加载，请检查配置。"

        history = history or []

        if self.model_type == "chatglm":
            response, _ = self.model.chat(
                self.tokenizer,
                user_input,
                history=[(h[0], h[1]) for h in history]
            )
            return response

        elif self.model_type == "qwen":
            # Qwen使用messages格式
            messages = []
            for h in history:
                messages.append({"role": "user", "content": h[0]})
                messages.append({"role": "assistant", "content": h[1]})
            messages.append({"role": "user", "content": user_input})

            text = self.tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
            inputs = self.tokenizer([text], return_tensors="pt").to(self.model.device)

            outputs = self.model.generate(**inputs, max_length=2048, temperature=0.7)
            response = self.tokenizer.decode(outputs[0][len(inputs.input_ids[0]):], skip_special_tokens=True)
            return response

        return "未知错误"
```

### 5.3 对话历史处理

```python
class DialogueHistoryManager:
    """
    对话历史管理器

    功能：
    - 历史记录存储
    - 历史摘要（防止过长）
    - 上下文检索
    """

    def __init__(self, max_turns=10, max_tokens=2000):
        self.max_turns = max_turns
        self.max_tokens = max_tokens
        self.history = []
        self.full_history = []

    def add_turn(self, user_input, system_response):
        """添加一轮对话"""
        self.history.append({
            'user': user_input,
            'system': system_response
        })
        self.full_history.append({
            'user': user_input,
            'system': system_response
        })

        # 截断过长历史
        if len(self.history) > self.max_turns:
            self.history = self.history[-self.max_turns:]

    def get_history_for_context(self):
        """
        获取用于构建上下文的对话历史

        Returns:
            list: 最近N轮对话
        """
        return self.history[-self.max_turns:]

    def get_history_text(self, format="simple"):
        """
        获取格式化的历史文本

        Args:
            format: 格式类型 ("simple" | "detailed")
        Returns:
            str: 格式化的历史文本
        """
        if not self.history:
            return ""

        if format == "simple":
            lines = []
            for turn in self.history:
                lines.append(f"用户：{turn['user']}")
                lines.append(f"助手：{turn['system']}")
            return "\n".join(lines)

        elif format == "detailed":
            lines = []
            for i, turn in enumerate(self.history):
                lines.append(f"[轮次 {i+1}]")
                lines.append(f"  用户：{turn['user']}")
                lines.append(f"  助手：{turn['system']}")
            return "\n".join(lines)

        return ""

    def summarize_history(self):
        """
        摘要历史对话（用于超长对话）

        Returns:
            str: 对话摘要
        """
        if len(self.history) <= 3:
            return self.get_history_text()

        # 简单摘要：保留最近3轮
        summary_parts = [f"（早期对话已省略，共{len(self.full_history)}轮）"]
        summary_parts.append("=== 最近对话 ===")
        summary_parts.append(self.get_history_text())

        return "\n".join(summary_parts)

    def search_history(self, keyword):
        """
        在历史中搜索关键词

        Args:
            keyword: 搜索关键词
        Returns:
            list: 匹配的历史轮次
        """
        results = []
        for turn in self.full_history:
            if keyword in turn['user'] or keyword in turn['system']:
                results.append(turn)
        return results

    def clear(self):
        """清空历史"""
        self.history = []
        self.full_history = []
```

```python
        分类情感

        Args:
            text: 输入文本

            dict: 情感结果
```

```python
        """
        对话

        Args:
            message: 用户消息
            system_prompt: 系统提示（可选）
        Returns:
            str: 模型回复
        """
        if self.backend == "openai":
            return self._chat_openai(message, system_prompt)
        elif self.backend == "chatglm":
            return self._chat_chatglm(message)
        elif self.backend == "qwen":
            return self._chat_qwen(message, system_prompt)
        elif self.backend == "kimi":
            return self._chat_kimi(message, system_prompt)
        else:
            return "不支持的后端"

    def _chat_openai(self, message, system_prompt=None):
        """OpenAI GPT对话"""
        from openai import OpenAI
        client = OpenAI(api_key=self.api_key)

        messages = []
        if system_prompt:
            messages.append({"role": "system", "content": system_prompt})
        messages.extend(self.history)
        messages.append({"role": "user", "content": message})

        response = client.chat.completions.create(
            model=self.model,
            messages=messages,
            temperature=0.7,
            max_tokens=1000
        )

        reply = response.choices[0].message.content

        # 更新历史
        self.history.append({"role": "user", "content": message})
        self.history.append({"role": "assistant", "content": reply})

        return reply

    def _chat_chatglm(self, message):
        """ChatGLM对话"""
        try:
            from transformers import AutoTokenizer, AutoModel
            tokenizer = AutoTokenizer.from_pretrained(self.model, trust_remote_code=True)
            model = AutoModel.from_pretrained(self.model, trust_remote_code=True).cuda()
            model.eval()

            response, _ = model.chat(tokenizer, message, history=[
                (h['user'], h['assistant']) for h in self.history
            ])

            self.history.append({"role": "user", "content": message})
            self.history.append({"role": "assistant", "content": response})

            return response
        except Exception as e:
            return f"ChatGLM调用失败: {e}"

    def _chat_qwen(self, message, system_prompt=None):
        """Qwen对话"""
        try:
            from transformers import AutoTokenizer, AutoModelForCausalLM
            tokenizer = AutoTokenizer.from_pretrained(self.model, trust_remote_code=True)
            model = AutoModelForCausalLM.from_pretrained(
                self.model, torch_dtype=torch.float16
            ).cuda()
            model.eval()

            messages = []
            if system_prompt:
                messages.append({"role": "system", "content": system_prompt})
            messages.extend([{"role": "user", "content": h['user']}, {"role": "assistant", "content": h['assistant']}
                           for h in self.history])
            messages.append({"role": "user", "content": message})

            text = tokenizer.apply_chat_template(messages, tokenize=False, add_generation_prompt=True)
            inputs = tokenizer([text], return_tensors="pt").to(model.device)

            outputs = model.generate(**inputs, max_length=2048, temperature=0.7)
            response = tokenizer.decode(outputs[0][len(inputs.input_ids[0]):], skip_special_tokens=True)

            self.history.append({"role": "user", "content": message})
            self.history.append({"role": "assistant", "content": response})

            return response
        except Exception as e:
            return f"Qwen调用失败: {e}"

    def _chat_kimi(self, message, system_prompt=None):
        """Kimi对话"""
        try:
            from openai import OpenAI
            client = OpenAI(api_key=os.environ.get("KIMI_API_KEY"), base_url="https://api.moonshot.cn/v1")

            messages = []
            if system_prompt:
                messages.append({"role": "system", "content": system_prompt})
            messages.extend(self.history)
            messages.append({"role": "user", "content": message})

            response = client.chat.completions.create(
                model=self.model,
                messages=messages,
                temperature=0.7
            )

            reply = response.choices[0].message.content
            self.history.append({"role": "user", "content": message})
            self.history.append({"role": "assistant", "content": reply})

            return reply
        except Exception as e:
            return f"Kimi调用失败: {e}"

    def clear_history(self):
        """清空对话历史"""
        self.history = []
```

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.metrics.pairwise import cosine_similarity
import numpy as np

class RAGDialogueSystem:
    """
    RAG增强对话系统

    检索增强生成（Retrieval-Augmented Generation）：
    1. 从知识库检索相关文档
    2. 将检索结果作为上下文
    3. 使用LLM生成回复
    """

    def __init__(self, llm_backend="openai"):
        # 知识库（文档列表）
        self.documents = [
            {"content": "ROS2使用DDS进行节点间通信，支持话题、服务、动作三种通信方式。", "source": "ROS2基础"},
            {"content": "机器人导航需要使用SLAM技术构建地图，然后使用导航栈进行路径规划。", "source": "机器人导航"},
            {"content": "语音交互流程：ASR语音识别 → NLU语义理解 → 对话管理 → NLG语言生成 → TTS语音合成。", "source": "语音交互"},
            {"content": "常用的中文分词工具有jieba、HanLP、SnowNLP等。", "source": "NLP工具"},
            {"content": "TurtleBot3是一款基于ROS2的教育机器人，支持激光雷达和深度相机。", "source": "TurtleBot3"},
            {"content": "YOLO是一种流行的实时目标检测算法，可用于机器人视觉感知。", "source": "目标检测"},
            {"content": "MoveIt是ROS中常用的机械臂运动规划框架。", "source": "机械臂控制"},
        ]

        # 构建检索索引
        self._build_index()

        # LLM接口
        self.llm = LLMChatInterface(backend=llm_backend)

    def _build_index(self):
        """构建TF-IDF检索索引"""
        self.vectorizer = TfidfVectorizer(analyzer='char', ngram_range=(1, 3))
        doc_texts = [doc['content'] for doc in self.documents]
        self.doc_vectors = self.vectorizer.fit_transform(doc_texts)
        print(f"✅ 知识库索引构建完成，共 {len(self.documents)} 篇文档")

    def retrieve(self, query, top_k=3):
        """
        检索相关文档

        Args:
            query: 查询文本
            top_k: 返回top-k个结果
        Returns:
            list: 检索结果
        """
        query_vector = self.vectorizer.transform([query])
        similarities = cosine_similarity(query_vector, self.doc_vectors)[0]
        top_indices = np.argsort(similarities)[::-1][:top_k]

        results = []
        for idx in top_indices:
            if similarities[idx] > 0.05:
                results.append({
                    'content': self.documents[idx]['content'],
                    'source': self.documents[idx]['source'],
                    'similarity': float(similarities[idx])
                })

        return results

    def chat(self, user_input):
        """
        RAG增强对话

        Args:
            user_input: 用户输入
        Returns:
            str: 助手回复
        """
        # 1. 检索相关文档
        docs = self.retrieve(user_input, top_k=3)

        # 2. 构建增强上下文
        context = ""
        if docs:
            context = "\n参考知识：\n" + "\n".join([
                f"- [{d['source']}] {d['content']}"
                for d in docs
            ])

        # 3. 构建提示
        prompt = f"""你是一个机器人助手，基于以下参考知识回答用户问题。如果参考知识不相关，请基于你的知识回答。

{context}

用户问题：{user_input}

请直接回答："""

        # 4. 调用LLM
        response = self.llm.chat(prompt)
        return response

    def add_document(self, content, source):
        """添加文档到知识库"""
        self.documents.append({"content": content, "source": source})
        # 重建索引
        self._build_index()
        return True

    def remove_document(self, source):
        """从知识库删除文档"""
        self.documents = [d for d in self.documents if d['source'] != source]
        self._build_index()
        return True
```

```python
#!/usr/bin/env python3
"""
ROS2 对话系统节点

功能：
- 订阅ASR识别结果
- 进行NLU解析
- 对话状态管理
- 生成回复
- 发布到TTS

话题接口：
  订阅：/asr/text (std_msgs/String) - 语音识别文本
  发布：/tts/text (std_msgs/String) - 待合成文本
  发布：/dialogue/intent (std_msgs/String) - 解析出的意图
"""
import rclpy
from rclpy.node import Node
from rclpy.qos import QoSProfile
from std_msgs.msg import String
import json

from robot_dialogue.dialogue_manager import DialogueManager
from robot_dialogue.nlg import HybridNLG


class RobotDialogueNode(Node):
    """
    机器人对话节点

    整合NLU + 对话管理 + NLG
    """

    def __init__(self):
        super().__init__('robot_dialogue_node')

        # 参数声明
        self.declare_parameter('use_llm', False)           # 是否使用LLM
        self.declare_parameter('llm_backend', 'openai')    # LLM后端
        self.declare_parameter('confidence_threshold', 0.6) # 置信度阈值
        self.declare_parameter('enable_debug', True)        # 调试模式

        self.use_llm = self.get_parameter('use_llm').value
        self.llm_backend = self.get_parameter('llm_backend').value
        self.confidence_threshold = self.get_parameter('confidence_threshold').value
        self.enable_debug = self.get_parameter('enable_debug').value

        self.get_logger().info(
            f'对话节点启动 (LLM={self.use_llm}, backend={self.llm_backend})'
        )

        # 初始化组件
        self.dialogue_manager = DialogueManager()
        self.nlg = HybridNLG(llm_backend=self.llm_backend if self.use_llm else None)

        # 订阅ASR文本
        self.sub_asr = self.create_subscription(
            String,
            '/asr/text',
            self.on_asr_result,
            QoSProfile(depth=10)
        )

        # 订阅用户确认（用于多轮对话确认）
        self.sub_confirm = self.create_subscription(
            String,
            '/user/confirm',
            self.on_user_confirm,
            QoSProfile(depth=10)
        )

        # 发布TTS文本
        self.pub_tts = self.create_publisher(
            String,
            '/tts/text',
            QoSProfile(depth=10)
        )

        # 发布意图（用于控制模块）
        self.pub_intent = self.create_publisher(
            String,
            '/dialogue/intent',
            QoSProfile(depth=10)
        )

        # 发布对话状态（用于调试）
        if self.enable_debug:
            self.pub_state = self.create_publisher(
                String,
                '/dialogue/state',
                QoSProfile(depth=10)
            )

        self.get_logger().info('对话节点初始化完成')

    def on_asr_result(self, msg):
        """
        处理ASR识别结果的回调

        Args:
            msg: String消息，包含识别的文本
        """
        text = msg.data.strip()

        if not text:
            return

        self.get_logger().info(f'收到ASR文本: "{text}"')

        # 对话处理
        result = self.dialogue_manager.process(text)

        # 发布意图
        if result['intent'] and result['confidence'] >= self.confidence_threshold:
            intent_msg = String()
            intent_msg.data = result['intent']
            self.pub_intent.publish(intent_msg)

        # 生成NLG回复
        nlg_text = self.nlg.generate(
            intent=result['intent'],
            slots=result['slots'],
            context=self.dialogue_manager.history_manager.get_history_for_context(),
            user_query=text
        )

        # 发布TTS文本
        tts_msg = String()
        tts_msg.data = nlg_text
        self.pub_tts.publish(tts_msg)

        # 发布状态（调试）
        if self.enable_debug:
            state_msg = String()
            state_msg.data = json.dumps(result, ensure_ascii=False, indent=2)
            self.pub_state.publish(state_msg)

        self.get_logger().info(f'对话结果: intent={result["intent"]}, reply="{nlg_text}"')

    def on_user_confirm(self, msg):
        """
        处理用户确认

        Args:
            msg: String消息，"是"/"否"或"confirm"/"cancel"
        """
        confirm = msg.data.strip().lower()

        if confirm in ['是', 'yes', 'y', 'confirm', 'ok']:
            # 用户确认
            result = self.dialogue_manager.confirm()
        else:
            # 用户取消
            result = self.dialogue_manager.cancel()

        # 生成回复
        nlg_text = self.nlg.generate(
            intent=result.get('intent', 'unknown'),
            slots=result.get('slots', {}),
        )

        # 发布TTS
        tts_msg = String()
        tts_msg.data = nlg_text
        self.pub_tts.publish(tts_msg)

        self.get_logger().info(f'确认结果: {result}')


def main(args=None):
    """主函数"""
    rclpy.init(args=args)

    try:
        node = RobotDialogueNode()
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
        self.action_executor = ActionExecutor()

    def process(self, user_input):
        """
        处理用户输入

        Args:
            user_input: 用户输入文本
        Returns:
            dict: 处理结果
        """
        # 1. NLU解析
        nlu_result = self.nlu_parser.parse(user_input)

        intent = nlu_result.get('intent', 'unknown')
        slots = nlu_result.get('slots', {})
        confidence = nlu_result.get('confidence', 0.0)

        # 2. 更新状态
        self.current_intent = intent
        self.current_slots.update(slots)

        # 3. 对话策略决策
        if intent == 'unknown' or confidence < 0.3:
            # 意图不清，询问澄清
            self.state = DialogueState.IDLE
            return {
                'intent': 'clarify',
                'slots': {},
                'confidence': 0.0,
                'action': 'ask_clarify',
                'message': "抱歉，我没有理解您的意思，请重新描述一下？"
            }

        # 4. 检查必填槽位
        missing_slots = self._get_missing_slots(intent, slots)
        if missing_slots:
            # 缺少必填槽位，追问
            self.state = DialogueState.PROCESSING
            return {
                'intent': intent,
                'slots': slots,
                'confidence': confidence,
                'action': 'request_slot',
                'missing_slot': missing_slots[0],
                'message': self._get_slot_prompt(missing_slots[0])
            }

        # 5. 置信度中等，需要确认
        if confidence < 0.7:
            self.state = DialogueState.WAITING_CONFIRM
            return {
                'intent': intent,
                'slots': slots,
                'confidence': confidence,
                'action': 'confirm',
                'message': self._get_confirm_message(intent, slots)
            }

        # 6. 执行动作
        self.state = DialogueState.EXECUTING
        exec_result = self.action_executor.execute(intent, slots)

        # 7. 更新历史
        self.history_manager.add_turn(user_input, exec_result.get('message', ''))

        # 8. 重置状态
        self.state = DialogueState.COMPLETED

        return {
            'intent': intent,
            'slots': slots,
            'confidence': confidence,
            'action': 'execute',
            'message': exec_result.get('message', '操作已完成')
        }

    def confirm(self):
        """用户确认，执行动作"""
        if self.state == DialogueState.WAITING_CONFIRM:
            self.state = DialogueState.EXECUTING
            exec_result = self.action_executor.execute(self.current_intent, self.current_slots)
            self.state = DialogueState.COMPLETED
            return {
                'intent': self.current_intent,
                'slots': self.current_slots,
                'action': 'execute',
                'message': exec_result.get('message', '操作已完成')
            }
        return {'message': '当前无需确认'}

    def cancel(self):
        """用户取消"""
        self.state = DialogueState.IDLE
        message = "好的，已取消。"
        return {
            'intent': None,
            'slots': {},
            'action': 'cancel',
            'message': message
        }

    def _get_missing_slots(self, intent, slots):
        """获取缺失的必填槽位"""
        required = {
            'light_control': ['target', 'action'],
            'navigation': ['destination'],
            'alarm_set': ['time'],
        }
        required_slots = required.get(intent, [])
        return [s for s in required_slots if s not in slots or not slots[s]]

    def _get_slot_prompt(self, slot):
        """生成槽位追问"""
        prompts = {
            'location': "请问是哪个位置？客厅、卧室还是厨房？",
            'target': "请问要控制什么设备？",
            'action': "请问是要打开还是关闭？",
            'destination': "请问您要去哪里？",
            'time': "请问要设置几点？",
            'song': "请问想听什么歌曲？",
        }
        return prompts.get(slot, f"请提供{slot}信息")

    def _get_confirm_message(self, intent, slots):
        """生成确认消息"""
        if intent == 'light_control':
            loc = slots.get('location', '')
            target = slots.get('target', '灯')
            action = "打开" if slots.get('action') == 'on' else "关闭"
            return f"我理解为：{action}{loc}{target}，对吗？"
        elif intent == 'navigation':
            return f"我理解为：您想去{slots.get('destination', '目的地')}，对吗？"
        return f"确认执行{intent}，对吗？"


class DialogueHistoryManager:
    """对话历史管理器"""

    def __init__(self, max_turns=10):

    def add_turn(self, user_input, system_response):

    def get_history_for_context(self):
        """获取上下文"""
        return self.history[-3:]

    def clear(self):
        """清空历史"""
        self.history = []
```

```python
"""
NLU解析器模块

import re


class TaskOrientedParser:
    """任务型对话NLU解析器"""

    def __init__(self):
        self.devices = ["灯", "门", "窗", "空调", "电视", "窗帘"]
        self.locations = ["客厅", "卧室", "厨房", "卫生间", "书房", "阳台"]
        self.actions = {
            "开": "on", "打开": "on", "关": "off",
            "关闭": "off", "关掉": "off"
        }
        self.intents = {
            'light_control': ['灯', '灯开', '灯关'],
            'door_control': ['门', '开门', '关门'],
            'music_control': ['音乐', '播放', '歌'],
            'navigation': ['去', '导航', '怎么走'],
            'alarm_set': ['闹钟', '提醒', '定时'],
            'stop': ['停止', '停下', '别动'],
        }

    def parse(self, text):
        """解析用户输入"""
        text = text.strip()
        intent = self._detect_intent(text)
        slots = self._extract_slots(text, intent)
        return {
            'text': text,
            'intent': intent,
            'slots': slots,
            'confidence': 0.9 if intent != 'unknown' else 0.0
        }

    def _detect_intent(self, text):
        for intent, keywords in self.intents.items():
            for kw in keywords:
                if kw in text:
                    return intent
        return 'unknown'

    def _extract_slots(self, text, intent):
        slots = {}
        for loc in self.locations:
            if loc in text:
                slots['location'] = loc
                break
        for action, value in self.actions.items():
            if action in text:
                slots['action'] = value
                break
        for dev in self.devices:
            if dev in text:
                slots['target'] = dev
                break
        if intent == 'navigation':
            m = re.search(r'去(.+?)怎么走|到(.+?)去|去(.+?)$', text)
            if m:
                slots['destination'] = m.group(1) or m.group(2) or m.group(3)
        return slots
```

```python
"""
端到端语音对话机器人

整合：
- ASR（语音识别）

from robot_dialogue.nlu_parser import TaskOrientedParser

class VoiceDialogueRobot(Node):
    """
    端到端语音对话机器人

    完整流程：
    ASR → NLU → DialogueManager → NLG → TTS
    """

    def __init__(self):
        super().__init__('voice_dialogue_robot')

        self.get_logger().info('端到端语音对话机器人启动')

        #

        # 初始化组件
        self.nlu_parser = TaskOrientedParser()
        self.dialogue_manager = DialogueManager()
        self.nlg = HybridNLG()

        # 订阅话题
        self.sub_asr = self.create_subscription(
            String, '/asr/text', self.on_asr, 10
        )
        self.sub_confirm = self.create_subscription(
            String, '/user/confirm', self.on_confirm, 10
        )

        # 发布话题
        self.pub_tts = self.create_publisher(String, '/tts/text', 10)
        self.pub_cmd = self.create_publisher(String, '/robot/cmd', 10)

        """处理语音识别结果"""
        text = msg.data.strip()
        if not text:
            return

        self.get_logger().info(f'[ASR] {text}')

        # 对话处理
        result = self.dialogue_manager.process(text)

        # 发布命令
        if result['action'] == 'execute':
            cmd = self._build_command(result['intent'], result['slots'])
            self.pub_cmd.publish(String(data=cmd))

        # TTS回复
        tts_text = result.get('message', '好的')
        self.pub_tts.publish(String(data=tts_text))

        self.get_logger().info(f'[回复] {tts_text}')

    def on_confirm(self, msg):
        """处理用户确认"""
        confirm = msg.data.strip().lower()
        if confirm in ['是', 'yes', 'y', 'ok']:
            result = self.dialogue_manager.confirm()
        else:
            result = self.dialogue_manager.cancel()

        tts_text = result.get('message', '好的')
        self.pub_tts.publish(String(data=tts_text))

    def _build_command(self, intent, slots):
        """构建机器人命令"""
        if intent == 'light_control':
            action = slots.get('action', 'off')
            loc = slots.get('location', 'all')
            return f"light:{action}:{loc}"
        elif intent == 'navigation':
            dest = slots.get('destination', '')
            return f"nav:{dest}"
        elif intent == 'music_play':
            return "music:play"
        elif intent == 'stop':
            return "stop"
        return ""


def main(args=None):
```

```python
class MultiTurnDialogueManager:
    """
    多轮对话管理器

    特点：
    - 维护对话历史
    - 支持上下文补全
    - 支持意图切换
    - 支持对话中断恢复
    """

    def __init__(self):
        self.state = {
            'turn': 0,
            'intent': None,
            'slots': {},
            'history': [],
            'context': {}  # 额外的上下文信息
        }
        self.nlu_parser = TaskOrientedParser()
        self.nlg = HybridNLG()
        self.max_history = 20

    def process(self, user_input):
        """
        处理用户输入（多轮）

        Args:
            user_input: 用户输入
        Returns:
            dict: 处理结果
        """
        self.state['turn'] += 1

        # NLU解析
        nlu_result = self.nlu_parser.parse(user_input)
        intent = nlu_result['intent']
        slots = nlu_result['slots']

        # 意图检测
        if self._is_intent_switch(intent):
            # 用户切换了意图，重置槽位
            self.get_logger().info(f'意图切换: {self.state["intent"]} → {intent}')
            self.state['intent'] = intent
            self.state['slots'] = slots
        else:
            # 同一意图，补充槽位
            self.state['slots'].update(slots)

        # 追加历史
        self.state['history'].append({
            'turn': self.state['turn'],
            'user': user_input,
            'intent': intent,
            'slots': slots.copy()
        })

        # 截断历史
        if len(self.state['history']) > self.max_history:
            self.state['history'] = self.state['history'][-self.max_history:]

        # 决策
        return self._decide(intent, self.state['slots'])

    def _is_intent_switch(self, new_intent):
        """判断是否切换了意图"""
        if not self.state['intent']:
            return True
        if new_intent != self.state['intent']:
            return True
        return False

    def _decide(self, intent, slots):
        """决策下一步动作"""
        missing = self._get_missing_slots(intent, slots)

        if missing:
            return {
                'action': 'request_slot',
                'intent': intent,
                'slots': slots,
                'missing': missing[0],
                'message': self.nlg.generate('clarify', {})
            }

        # 执行
        return {
            'action': 'execute',
            'intent': intent,
            'slots': slots,
            'message': self.nlg.generate(intent, slots)
        }

    def _get_missing_slots(self, intent, slots):
        """获取缺失槽位"""
        required = {
            'light_control': ['target'],
            'navigation': ['destination'],
            'alarm_set': ['time'],
        }
        req = required.get(intent, [])
        return [s for s in req if s not in slots]

    def reset(self):
        """重置对话"""
        self.state = {
            'turn': 0,
            'intent': None,
            'slots': {},
            'history': [],
            'context': {}
        }

    def get_summary(self):
        """获取对话摘要"""
        return {
            'total_turns': self.state['turn'],
            'current_intent': self.state['intent'],
            'current_slots': self.state['slots'],
            'history_length': len(self.state['history'])
        }
```

```python
class ContextMemory:
    """
    上下文记忆

    用于存储和管理对话上下文信息
    """

    def __init__(self, max_memory=100):
        self.max_memory = max_memory
        self.memory = {}
        self.timestamps = {}

    def store(self, key, value, ttl=None):
        """
        存储记忆

        Args:
            key: 记忆键
            value: 记忆值
            ttl: 生存时间（秒），None表示永久
        """
        self.memory[key] = value
        self.timestamps[key] = {
            'created': time.time(),
            'ttl': ttl
        }

        # 清理过期记忆
        self._cleanup()

    def retrieve(self, key):
        """检索记忆"""
        if key in self.memory:
            # 检查TTL
            ts = self.timestamps[key]
            if ts['ttl'] and (time.time() - ts['created']) > ts['ttl']:
                del self.memory[key]
                del self.timestamps[key]
                return None
            return self.memory[key]
        return None

    def _cleanup(self):
        """清理过期记忆"""
        current_time = time.time()
        expired_keys = [
            k for k, ts in self.timestamps.items()
            if ts['ttl'] and (current_time - ts['created']) > ts['ttl']
        ]
        for k in expired_keys:
            del self.memory[k]
            del self.timestamps[k]

        # 超过最大容量，删除最旧的
        while len(self.memory) > self.max_memory:
            oldest_key = min(self.timestamps.keys(), key=lambda k: self.timestamps[k]['created'])
            del self.memory[oldest_key]
            del self.timestamps[oldest_key]

    def get_all(self):
        """获取所有记忆"""
        self._cleanup()
        return self.memory.copy()

    def clear(self):
        """清空记忆"""
        self.memory = {}
        self.timestamps = {}
```

```python
from collections import Counter
import math

class DialogueEvaluator:
    """
    对话系统质量评估

    评估指标：
    - BLEU：N-gram精确率
    - ROUGE：召回率
    -意图准确率
    - 任务完成率
    """

    def __init__(self):
        self.results = []

    def add_sample(self, reference, hypothesis, intent_pred, intent_true):
        """
        添加评估样本

        Args:
            reference: 参考回复
            hypothesis: 实际回复
            intent_pred: 预测意图
            intent_true: 真实意图
        """
        self.results.append({
            'reference': reference,
            'hypothesis': hypothesis,
            'intent_pred': intent_pred,
            'intent_true': intent_true
        })

    def evaluate(self):
        """
        评估整体质量

        Returns:
            dict: 评估结果
        """
        bleu = self._compute_bleu()
        rouge = self._compute_rouge()
        intent_accuracy = self._compute_intent_accuracy()

        return {
            'bleu': bleu,
            'rouge': rouge,
            'intent_accuracy': intent_accuracy,
            'num_samples': len(self.results)
        }

    def _compute_bleu(self, n=4):
        """
        计算BLEU分数

        BLEU = BP * exp(sum(wn * logpn))
        """
        scores = []
        for r in self.results:
            ref = r['reference']
            hyp = r['hypothesis']
            score = self._sentence_bleu(ref, hyp, n)
            scores.append(score)
        return sum(scores) / len(scores) if scores else 0.0

    def _sentence_bleu(self, reference, hypothesis, n):
        """计算单句BLEU"""
        ref_tokens = reference.split()
        hyp_tokens = hypothesis.split()

        # 统计n-gram
        ref_ngrams = Counter(self._get_ngrams(ref_tokens, n))
        hyp_ngrams = Counter(self._get_ngrams(hyp_tokens, n))

        # 匹配计数
        matched = sum((ref_ngrams & hyp_ngrams).values())
        total = sum(hyp_ngrams.values())

        if total == 0:
            return 0.0

        precision = matched / total

        # Brevity Penalty
        bp = min(1.0, math.exp(1 - len(ref_tokens) / max(1, len(hyp_tokens))))

        return bp * precision

    def _get_ngrams(self, tokens, n):
        """获取n-gram列表"""
        return [tuple(tokens[i:i+n]) for i in range(len(tokens) - n + 1)]

    def _compute_rouge(self, n=1):
        """计算ROUGE分数"""
        scores = []
        for r in self.results:
            ref = set(r['reference'].split())
            hyp = set(r['hypothesis'].split())
            if len(ref) == 0:
                continue
            recall = len(ref & hyp) / len(ref)
            scores.append(recall)
        return sum(scores) / len(scores) if scores else 0.0

    def _compute_intent_accuracy(self):
        """计算意图准确率"""
        if not self.results:
            return 0.0
        correct = sum(1 for r in self.results if r['intent_pred'] == r['intent_true'])
        return correct / len(self.results)

    def print_report(self):
        """打印评估报告"""
        metrics = self.evaluate()
        print("=" * 50)
        print("对话系统质量评估报告")
        print("=" * 50)
        print(f"样本数量: {metrics['num_samples']}")
        print(f"BLEU:    {metrics['bleu']:.4f}")
        print(f"ROUGE:   {metrics['rouge']:.4f}")
        print(f"意图准确率: {metrics['intent_accuracy']:.4f}")
        print("=" * 50)
```

```python
import time
import threading
from concurrent.futures import ThreadPoolExecutor

class ResponseTimeOptimizer:
    """
    响应延迟优化器

    优化策略：
    1. 模型量化
    2. 缓存结果
    3. 异步处理
    4. 批处理
    """

    def __init__(self):
        self.cache = {}
        self.cache_size = 1000

        # 异步执行器
        self.executor = ThreadPoolExecutor(max_workers=4)

        # 统计
        self.latencies = []

    def cached_generate(self, cache_key, generator_func):
        """
        缓存优化

        Args:
            cache_key: 缓存键
            generator_func: 生成函数
        Returns:
            生成结果
        """
        if cache_key in self.cache:
            return self.cache[cache_key]

        result = generator_func()

        # 缓存结果
        if len(self.cache) >= self.cache_size:
            # 删除最早的缓存
            oldest_key = next(iter(self.cache))
            del self.cache[oldest_key]

        self.cache[cache_key] = result
        return result

    def async_generate(self, generator_func, callback):
        """
        异步生成

        Args:
            generator_func: 生成函数
            callback: 回调函数
        """
        def run():
            start = time.time()
            result = generator_func()
            latency = time.time() - start
            self.latencies.append(latency)
            callback(result)

        self.executor.submit(run)

    def get_avg_latency(self):
        """获取平均延迟"""
        if not self.latencies:
            return 0.0
        return sum(self.latencies) / len(self.latencies)

    def get_p95_latency(self):
        """获取P95延迟"""
        if not self.latencies:
            return 0.0
        sorted_latencies = sorted(self.latencies)
        idx = int(len(sorted_latencies) * 0.95)
        return sorted_latencies[min(idx, len(sorted_latencies) - 1)]
```

### 1. 对话系统概述

**三大类型**：
- **任务型**：帮助用户完成特定任务（订餐、控制设备），多轮交互，需槽位填充
- **问答型**：回答用户问题（客服、知识查询），通常单轮，依赖知识库
- **闲聊型**：情感陪伴、开放式交流，开放域，评估主观

**核心模块**：
```
用户输入 → NLU → 对话管理 → NLG → 系统回复
              ↓        ↓         ↓
          意图+槽位  对话状态   自然语言
```

**BLEU vs ROUGE**：
- BLEU：N-gram精确率，衡量生成文本与参考文本的词汇重合度
- ROUGE：N-gram召回率，衡量参考文本中有多少被生成文本覆盖
