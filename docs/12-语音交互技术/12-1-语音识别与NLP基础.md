## 1. 语音识别概述

### 1.1 什么是语音识别（ASR）

**自动语音识别**（Automatic Speech Recognition，ASR）是将人类语音转换为文本的技术。它是语音交互系统的核心组件，让机器人能够"听懂"人类的指令。

```
人类语音 → [麦克风] → [A/D转换] → [ASR模型] → 文本输出
```

### 1.2 语音识别发展历史

| 时代 | 时间 | 技术特点 | 代表成果 |
|------|------|----------|----------|
| 萌芽期 | 1950s-1960s | 基于模板匹配 | Audrey（IBM）、Helen |
| 规则系统 | 1970s-1980s | 专家规则、动态规划 | HARPY、CARSTA |
| 统计模型 | 1990s-2000s | HMM-GMM混合模型 | Sphinx、Julius |
| 深度学习 | 2010s | DNN-HMM、端到端 | DeepSpeech、Wav2Letter |
| 大模型时代 | 2020s-至今 | Transformer、预训练 | Whisper、Conformer |

### 1.3 端到端语音识别原理

端到端ASR模型直接从音频映射到文本，无需复杂的独立组件：

```
音频特征 → [Encoder编码器] → [Attention注意力] → [Decoder解码器] → 文本
```

**核心优势：**
- 简化流水线，减少人工特征工程
- 支持多语言统一建模
- 适应性强，易于迁移

### 1.4 声学模型 vs 语言模型

| 组件 | 声学模型 | 语言模型 |
|------|----------|----------|
| **输入** | 音频特征（MFCC/Mel） | 词序列 |
| **输出** | 音素/字符概率 | 下一个词的概率 |
| **作用** | 建模声音与音素关系 | 约束文本合理性 |
| **技术** | CNN、RNN、Transformer | N-gram、Neural LM |

### 1.5 常用ASR框架对比

| 框架 | 开发方 | 特点 | 适用场景 | 模型大小 |
|------|--------|------|----------|----------|
| **Whisper** | OpenAI | 多语言、大模型、高精度 | 通用场景 | 39M-1.5B |
| **FunASR** | 阿里达摩院 | 中文优化、社区活跃 | 中文场景 | 9M-1.2B |
| **Vosk** | Alphacephei | 轻量离线、实时性好 | 嵌入式、边缘 | 10M-50M |
| **WeNet** | 出门问问 | 流式识别、端到端 | 流式应用 | 10M-120M |
| **Coqui** | Coqui | 开源、TTS+STT | 独立部署 | 30M-220M |

## 2. 语音预处理技术

### 2.1 采样率与量化

**采样率**（Sampling Rate）：每秒采样的点数，常见值有8kHz、16kHz、44.1kHz。

```python
import numpy as np
import librosa

# 加载音频并重采样到16kHz（ASR标准采样率）
audio, sr = librosa.load("input.wav", sr=16000)
print(f"采样率: {sr} Hz")
print(f"时长: {len(audio)/sr:.2f} 秒")
print(f"样本数: {len(audio)}")
```

**量化**（Quantization）：将连续幅值离散化为数字表示。

```python
# 量化位数示例
bit_depth = 16  # 16位量化，65536个离散级别
max_value = 2 ** bit_depth - 1  # 65535
normalized_audio = audio / max_value  # 归一化到[-1, 1]
```

### 2.2 预加重与分帧

**预加重**（Pre-emphasis）：提升高频成分，补偿声门效应。

```python
def pre_emphasis(audio, alpha=0.97):
    """
    预加重滤波
    alpha: 预加重系数，通常取0.95-0.98
    """
    # 高通滤波器：y[n] = x[n] - alpha * x[n-1]
    emphasized = np.append(audio[0], audio[1:] - alpha * audio[:-1])
    return emphasized

# 应用预加重
audio_emphasized = pre_emphasis(audio, alpha=0.97)
```

**分帧**（Framing）：将连续信号切分为短时帧（通常20-40ms）。

```python
def extract_features(audio, sr=16000):
    """
    提取语音特征
    """
    # 提取MFCC特征（梅尔频率倒谱系数）
    mfcc = librosa.feature.mfcc(y=audio, sr=sr, n_mfcc=13)
    
    # 提取Mel频谱
    mel_spec = librosa.feature.melspectrogram(y=audio, sr=sr, n_mels=80)
    mel_spec_db = librosa.power_to_db(mel_spec, ref=np.max)
    
    # 提取Pitch（基频）
    pitches, magnitudes = librosa.piptrack(y=audio, sr=sr)
    
    return {
        'mfcc': mfcc,
        'mel_spec': mel_spec_db,
        'pitch': pitches
    }
```

### 2.3 短时傅里叶变换（STFT）

STFT将时域信号转换到频域，是语音分析的基础工具。

```python
import librosa
import numpy as np
import matplotlib.pyplot as plt

def compute_stft(audio, sr=16000, hop_length=512, n_fft=2048):
    """
    计算短时傅里叶变换
    
    参数:
        audio: 输入音频信号
        sr: 采样率
        hop_length: 帧移
        n_fft: FFT点数
    返回:
        stft_matrix: STFT结果（复数矩阵）
    """
    # 计算STFT
    stft_matrix = librosa.stft(
        audio,
        n_fft=n_fft,
        hop_length=hop_length,
        win_length=n_fft
    )
    
    # 获取幅度谱
    magnitude = np.abs(stft_matrix)
    
    # 获取相位谱
    phase = np.angle(stft_matrix)
    
    return stft_matrix, magnitude, phase

# 可视化频谱
stft_matrix, magnitude, phase = compute_stft(audio)

plt.figure(figsize=(12, 4))
librosa.display.specshow(
    librosa.amplitude_to_db(magnitude, ref=np.max),
    sr=sr,
    hop_length=512,
    x_axis='time',
    y_axis='hz'
)
plt.colorbar(format='%+2.0f dB')
plt.title('STFT频谱图')
plt.tight_layout()
plt.savefig('stft_spectrogram.png', dpi=150)
```

### 2.4 梅尔频谱（Mel Spectrogram）

梅尔频谱模拟人耳对频率的感知，是深度学习ASR的常用输入特征。

```python
def compute_mel_spectrogram(audio, sr=16000, n_mels=80, n_fft=2048):
    """
    计算梅尔频谱
    
    参数:
        n_mels: Mel滤波器组的滤波器数量
    """
    # 计算Mel频谱
    mel_spec = librosa.feature.melspectrogram(
        y=audio,
        sr=sr,
        n_fft=n_fft,
        hop_length=512,
        n_mels=n_mels,  # 80个Mel带
        fmin=0,
        fmax=8000
    )
    
    # 转换为dB单位
    mel_spec_db = librosa.power_to_db(mel_spec, ref=np.max)
    
    return mel_spec_db
```

### 2.5 去噪与VAD（语音活动检测）

**VAD**（Voice Activity Detection）用于检测语音片段，过滤静音。

```python
import webrtcvad

def vad_filter(audio, sr=16000, aggressiveness=2):
    """
    使用WebRTC VAD进行语音活动检测
    
    aggressiveness: 激进程度 0-3，越高越激进过滤
    """
    # 确保采样率为16kHz（WebRTC VAD要求）
    if sr != 16000:
        audio = librosa.resample(audio, orig_sr=sr, target_sr=16000)
        sr = 16000
    
    # 初始化VAD
    vad = webrtcvad.Vad(aggressiveness)
    
    # 将音频分割为30ms帧
    frame_duration = 30  # ms
    frame_size = int(sr * frame_duration / 1000)
    frames = []
    
    for i in range(0, len(audio) - frame_size, frame_size):
        frame = audio[i:i + frame_size]
        # 转换为int16格式
        frame_int = (frame * 32767).astype(np.int16)
        is_speech = vad.is_speech(frame_int.tobytes(), sr)
        frames.append(is_speech)
    
    # 合并语音帧
    speech_mask = np.array(frames)
    speech_audio = []
    frame_idx = 0
    
    for i in range(0, len(audio) - frame_size, frame_size):
        if frame_idx < len(speech_mask) and speech_mask[frame_idx]:
            speech_audio.append(audio[i:i + frame_size])
        frame_idx += 1
    
    return np.concatenate(speech_audio) if speech_audio else np.array([])

# 使用示例
filtered_audio = vad_filter(audio, aggressiveness=2)
```

### 3.4 注意力机制

注意力机制让模型能够"关注"输入序列中的关键部分。

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import math

class SelfAttention(nn.Module):
    """
    自注意力机制实现
    """
    def __init__(self, embed_size, heads):
        super(SelfAttention, self).__init__()
        self.embed_size = embed_size
        self.heads = heads
        self.head_dim = embed_size // heads
        
        assert self.head_dim * heads == embed_size, "embed_size必须能被heads整除"
        
        # Q、K、V 线性变换
        self.values = nn.Linear(embed_size, embed_size)
        self.keys = nn.Linear(embed_size, embed_size)
        self.queries = nn.Linear(embed_size, embed_size)
        self.fc_out = nn.Linear(embed_size, embed_size)
        
    def forward(self, values, keys, query, mask):
        batch_size = query.shape[0]
        
        # 线性变换
        Q = self.queries(query)  # [batch, seq_len, embed_size]
        K = self.keys(keys)
        V = self.values(values)
        
        # 分割多头
        Q = Q.reshape(batch_size, -1, self.heads, self.head_dim).permute(0, 2, 1, 3)
        K = K.reshape(batch_size, -1, self.heads, self.head_dim).permute(0, 2, 1, 3)
        V = V.reshape(batch_size, -1, self.heads, self.head_dim).permute(0, 2, 1, 3)
        
        # 计算注意力分数
        energy = torch.matmul(Q, K.transpose(-1, -2)) / math.sqrt(self.head_dim)
        
        # 应用掩码（可选）
        if mask is not None:
            energy = energy.masked_fill(mask == 0, float("-1e20"))
        
        # Softmax归一化
        attention = F.softmax(energy, dim=-1)
        
        # 加权求和
        out = torch.matmul(attention, V)
        
        # 合并多头
        out = out.permute(0, 2, 1, 3).reshape(batch_size, -1, self.embed_size)
        
        return self.fc_out(out)

# 使用示例
embed_size = 256
heads = 8
attention = SelfAttention(embed_size, heads)

# 模拟输入 [batch=2, seq_len=10, embed_size=256]
x = torch.randn(2, 10, 256)
mask = torch.ones(2, 10)  # 全1掩码
output = attention(x, x, x, mask)
print(f"注意力输出形状: {output.shape}")
```

```python
import whisper
import torch

class WhisperASR:
    """
    Whisper语音识别封装类
    """
    def __init__(self, model_name="base", device=None):
        """
        初始化Whisper ASR
        
        参数:
            model_name: 模型大小 (tiny/base/small/medium/large)
            device: 运行设备 (cuda/cpu)
        """
        self.device = device or ("cuda" if torch.cuda.is_available() else "cpu")
        print(f"加载Whisper {model_name} 模型，设备: {self.device}")
        self.model = whisper.load_model(model_name, device=self.device)
        
    def transcribe(self, audio_path, language="zh", task="transcribe"):
        """
        识别音频文件
        
        参数:
            audio_path: 音频文件路径
            language: 语言代码 (zh=en Chinese)
            task: 任务类型 (transcribe/translate)
        返回:
            dict: 包含text、segments等字段
        """
        # 加载并预处理音频
        audio = whisper.load_audio(audio_path)
        audio = whisper.pad_or_trim(audio)
        
        # 创建Mel频谱
        mel = whisper.log_mel_spectrogram(audio, n_mels=80).to(self.device)
        
        # 推理
        options = whisper.DecodingOptions(
            language=language,
            task=task,
            beam_size=5,
            best_of=5
        )
        result = whisper.decode(self.model, mel, options)
        
        return {
            "text": result.text,
            "language": result.language,
            "no_speech_prob": result.no_speech_prob,
            "avg_logprob": result.avg_logprob
        }
    
    def transcribe_realtime(self, audio_array, sample_rate=16000):
        """
        实时识别音频片段
        
        参数:
            audio_array: numpy音频数组
            sample_rate: 采样率
        """
        # 确保采样率正确
        if sample_rate != 16000:
            audio_array = whisper.pad_or_trim(audio_array)
            
        # 转换为Mel频谱
        mel = whisper.log_mel_spectrogram(audio_array, n_mels=80).to(self.device)
        
        # 解码
        options = whisper.DecodingOptions(language="zh")
        result = whisper.decode(self.model, mel, options)
        
        return result.text

# 使用示例
asr = WhisperASR(model_name="base")
result = asr.transcribe("test_audio.wav", language="zh")
print(f"识别结果: {result['text']}")
```

```python
from funasr import AutoModel
import torch

class FunASRRecognizer:
    """
    FunASR语音识别封装类
    """
    def __init__(self, model_name="paraformer-zh", device=None):
        """
        初始化FunASR
        
        参数:
            model_name: 模型名称
                - paraformer-zh: 中文 Paraformer 模型
                - conformer-zh: 中文 Conformer 模型
                - paraformer-large: 大模型（更高精度）
            device: 运行设备
        """
        self.device = device or ("cuda" if torch.cuda.is_available() else "cpu")
        print(f"加载FunASR模型: {model_name}")
        self.model = AutoModel(
            model=model_name,
            device=self.device,
            disable_update=True  # 禁用自动更新检查
        )
        
    def recognize(self, audio_path):
        """
        识别音频文件
        """
        result = self.model.generate(
            input=audio_path,
            batch_size_s=300,  # 批处理时间（秒）
            hotword="",        # 热词（提高特定词识别率）
            merge_vad=True,    # 合并VAD结果
            merge_length_s=15  # 合并段最大长度
        )
        return result
    
    def recognize_from_array(self, audio_array, sample_rate=16000):
        """
        从numpy数组识别
        """
        result = self.model.generate(
            input=audio_array,
            input_fs=sample_rate,
            hotword=""
        )
        return result

# 使用示例
funasr = FunASRRecognizer(model_name="paraformer-zh")
result = funasr.recognize("test_audio.wav")
print(f"FunASR识别结果: {result[0]['text']}")
```

```python
    """
    ROS2音频采集节点
    
    功能：
    - 从麦克风实时采集音频
    - 发布音频数据到 /audio/raw 话题
    """
    
    def __init__(self):
        super().__init__('audio_capture_node')
        
        # 参数声明
        self.declare_parameter('sample_rate', 16000)
        self.declare_parameter('chunk_size', 4000)  # 每次读取的样本数
        self.declare_parameter('format', 'paInt16')
        self.declare_parameter('channels', 1)
        self.declare_parameter('device_index', None)  # None表示默认设备
        
        self.sample_rate = self.get_parameter('sample_rate').value
        self.chunk_size = self.get_parameter('chunk_size').value
        self.channels = self.get_parameter('channels').value
        self.device_index = self.get_parameter('device_index').value
        
        # 创建发布者
        self.audio_pub = self.create_publisher(
            AudioData,
            '/audio/raw',
            10
        )
        
        # 初始化PyAudio
        self.audio = pyaudio.PyAudio()
        self.stream = None
        self.is_capturing = False
        
        # 音频回调
        def audio_callback(in_data, frame_count, time_info, status):
            """音频流回调函数"""
            if status == pyaudio.paContinue:
                # 发布音频数据
                msg = AudioData()
                msg.data = in_data
                self.audio_pub.publish(msg)
                return (in_data, pyaudio.paContinue)
            else:
                return (in_data, pyaudio.paComplete)
        
        self.audio_callback = audio_callback
        
        self.get_logger().info('音频采集节点初始化完成')
        self.get_logger().info(f'采样率: {self.sample_rate}Hz, 块大小: {self.chunk_size}')
        
    def start_capture(self):
        """开始音频采集"""
        if self.is_capturing:
            self.get_logger().warn('音频采集已在运行')
            return
            
        try:
            self.stream = self.audio.open(
                format=self.audio.get_format_from_width(2),  # 16bit
                channels=self.channels,
                rate=self.sample_rate,
                input=True,
                input_device_index=self.device_index,
                frames_per_buffer=self.chunk_size,
                stream_callback=self.audio_callback
            )
            
            self.stream.start_stream()
            self.is_capturing = True
            self.get_logger().info('音频采集已开始')
            
        except Exception as e:
            self.get_logger().error(f'启动音频采集失败: {e}')
            
    def stop_capture(self):
        """停止音频采集"""
        if not self.is_capturing:
            return
            
        if self.stream:
            self.stream.stop_stream()
            self.stream.close()
            self.stream = None
            
        self.is_capturing = False
        self.get_logger().info('音频采集已停止')
        
    def destroy_node(self):
        """清理资源"""
        self.stop_capture()
        self.audio.terminate()
        super().destroy_node()

        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
# robot_asr/robot_asr/whisper_asr_node.py
"""
Whisper语音识别ROS2节点
订阅音频话题，发布识别结果
"""

import rclpy
from rclpy.node import Node
from std_msgs.msg import String
from sensor_msgs.msg import AudioData
import numpy as np
import whisper
import torch
from collections import deque
import threading


class WhisperASRNode(Node):
    """
    Whisper ASR ROS2节点
    
    订阅话题：
        /audio/raw: 原始音频数据
    
    发布话题：
        /asr/text: 识别文本结果
        /asr/status: 识别状态
    """
    
    def __init__(self):
        super().__init__('whisper_asr_node')
        
        # 参数声明
        self.declare_parameter('model_name', 'base')
        self.declare_parameter('language', 'zh')
        self.declare_parameter('sample_rate', 16000)
        self.declare_parameter('buffer_seconds', 5.0)  # 缓冲音频时长
        self.declare_parameter('confidence_threshold', 0.5)  # 置信度阈值
        
        self.model_name = self.get_parameter('model_name').value
        self.language = self.get_parameter('language').value
        self.sample_rate = self.get_parameter('sample_rate').value
        self.buffer_seconds = self.get_parameter('buffer_seconds').value
        self.confidence_threshold = self.get_parameter('confidence_threshold').value
        
        # 计算缓冲区大小
        self.buffer_size = int(self.sample_rate * self.buffer_seconds)
        
        # 音频缓冲区（存储最近的音频数据）
        self.audio_buffer = deque(maxlen=self.buffer_size)
        
        # 加载Whisper模型
        self.device = "cuda" if torch.cuda.is_available() else "cpu"
        self.get_logger().info(f'加载Whisper模型: {self.model_name}, 设备: {self.device}')
        
        self.model = whisper.load_model(self.model_name, device=self.device)
        self.get_logger().info('Whisper模型加载完成')
        
        # 创建订阅者和发布者
        self.audio_sub = self.create_subscription(
            AudioData,
            '/audio/raw',
            self.audio_callback,
            10
        )
        
        self.text_pub = self.create_publisher(
            String,
            '/asr/text',
            10
        )
        
        self.status_pub = self.create_publisher(
            String,
            '/asr/status',
            10
        )
        
        # 锁，保证线程安全
        self.lock = threading.Lock()
        
        self.get_logger().info('Whisper ASR节点初始化完成')
        
    def audio_callback(self, msg: AudioData):
        """
        音频数据回调
        """
        # 将bytes转换为numpy数组（int16）
        audio_int16 = np.frombuffer(msg.data, dtype=np.int16)
        
        # 转换为float32并归一化
        audio_float = audio_int16.astype(np.float32) / 32768.0
        
        with self.lock:
            # 添加到缓冲区
            self.audio_buffer.extend(audio_float)
            
            # 当缓冲区足够大时进行识别
            if len(self.audio_buffer) >= self.buffer_size:
                self.process_audio()
                
    def process_audio(self):
        """
        处理缓冲区中的音频
        """
        if len(self.audio_buffer) < self.sample_rate:  # 至少1秒音频
            return
            
        # 复制缓冲区数据
        audio = np.array(list(self.audio_buffer), dtype=np.float32)
        
        # 清空缓冲区（避免重复识别）
        self.audio_buffer.clear()
        
        # 发布状态
        status_msg = String()
        status_msg.data = "recognizing"
        self.status_pub.publish(status_msg)
        
        # 推理
        try:
            # 转换为Mel频谱
            audio_tensor = whisper.pad_or_trim(audio)
            mel = whisper.log_mel_spectrogram(audio_tensor, n_mels=80).to(self.device)
            
            # 解码
            options = whisper.DecodingOptions(
                language=self.language,
                fp16=(self.device == "cuda")
            )
            result = whisper.decode(self.model, mel, options)
            
            # 检查置信度
            if result.avg_logprob > np.log(self.confidence_threshold):
                text_msg = String()
                text_msg.data = result.text.strip()
                self.text_pub.publish(text_msg)
                self.get_logger().info(f'识别结果: {result.text}')
            else:
                self.get_logger().warn(f'置信度过低: {result.avg_logprob}')
                
        except Exception as e:
            self.get_logger().error(f'识别失败: {e}')
            
        # 更新状态
        status_msg.data = "idle"
        self.status_pub.publish(status_msg)


def main(args=None):
    """主函数"""
    rclpy.init(args=args)
    node = WhisperASRNode()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

### 6.2 实时流式识别

```python
#!/usr/bin/env python3
# robot_asr/examples/streaming_asr.py
"""
流式语音识别示例
持续监听麦克风，实时输出识别结果
"""

import whisper
import numpy as np
import pyaudio
import threading
import time


class StreamingASR:
    """
    流式语音识别器
    使用VAD检测语音边界，实时输出识别结果
    """
    def __init__(self, model_name="base"):
        self.model = whisper.load_model(model_name)
        self.sample_rate = 16000
        self.chunk_duration = 0.5  # 每块音频时长（秒）
        self.chunk_size = int(self.sample_rate * self.chunk_duration)
        
        # 音频缓冲区
        self.audio_buffer = []
        self.buffer_lock = threading.Lock()
        self.is_recording = False
        
        # 能量阈值（用于简单VAD）
        self.energy_threshold = 0.01
        
        print("[INFO] 流式ASR初始化完成")
        
    def calculate_energy(self, audio):
        """计算音频能量"""
        return np.sqrt(np.mean(audio ** 2))
        
    def audio_capture_thread(self, stream):
        """音频采集线程"""
        while self.is_recording:
            data = stream.read(self.chunk_size, exception_on_overflow=False)
            audio = np.frombuffer(data, dtype=np.float32)
            
            energy = self.calculate_energy(audio)
            
            # 简单的能量检测VAD
            if energy > self.energy_threshold:
                with self.buffer_lock:
                    self.audio_buffer.append(audio)
            else:
                # 检测到静音，检查缓冲区是否有内容
                with self.buffer_lock:
                    if len(self.audio_buffer) > 0:
                        # 清空缓冲区并识别
                        full_audio = np.concatenate(self.audio_buffer)
                        self.audio_buffer = []
                        
                        # 在新线程中识别（避免阻塞采集）
                        threading.Thread(
                            target=self.recognize_chunk,
                            args=(full_audio,)
                        ).start()
                        
    def recognize_chunk(self, audio):
        """识别音频片段"""
        try:
            # 预处理
            if len(audio) < self.sample_rate * 0.3:  # 太短的音频跳过
                return
                
            audio = whisper.pad_or_trim(audio)
            mel = whisper.log_mel_spectrogram(audio, n_mels=80)
            
            # 识别
            options = whisper.DecodingOptions(language="zh")
            result = whisper.decode(self.model, mel, options)
            
            if result.text.strip():
                print(f"[识别] {result.text}")
                
        except Exception as e:
            print(f"[错误] 识别失败: {e}")
            
    def start(self):
        """启动流式识别"""
        p = pyaudio.PyAudio()
        stream = p.open(
            format=pyaudio.paFloat32,
            channels=1,
            rate=self.sample_rate,
            input=True,
            frames_per_buffer=self.chunk_size
        )
        
        print("[INFO] 开始流式识别 (Ctrl+C 退出)")
        self.is_recording = True
        
        # 启动采集线程
        capture_thread = threading.Thread(
            target=self.audio_capture_thread,
            args=(stream,)
        )
        capture_thread.start()
        
        try:
            while True:
                time.sleep(0.1)
        except KeyboardInterrupt:
            print("\n[INFO] 停止流式识别...")
            self.is_recording = False
            
        capture_thread.join()
        stream.stop_stream()
        stream.close()
        p.terminate()


if __name__ == "__main__":
    asr = StreamingASR(model_name="base")
    asr.start()
```

```python
        }
        
        # 位置实体识别
        self.location_patterns = [
            '厨房', '卧室', '客厅', '门口', '走廊',
            '充电桩', '沙发', '桌子', '阳台'
        ]
        
    def parse(self, text):
        """
        解析语音命令
        
        参数:
            text: 识别文本
        返回:
            dict: 解析结果 {intent, entities, original_text}
        """
        text = text.strip()
        
        # 意图识别
        intent = self.recognize_intent(text)
        
        # 实体提取
        entities = self.extract_entities(text)
        
        return {
            'intent': intent,
            'entities': entities,
            'original_text': text,
            'confidence': 1.0
        }
        
    def recognize_intent(self, text):
        """识别意图"""
        intent_scores = {}
        
        for intent, keywords in self.intent_keywords.items():
            score = sum(1 for kw in keywords if kw in text)
            if score > 0:
                intent_scores[intent] = score
                
        if intent_scores:
            return max(intent_scores, key=intent_scores.get)
        return 'unknown'
        
    def extract_entities(self, text):
        """提取实体（位置、物体等）"""
        entities = {'locations': [], 'objects': []}
        
        for loc in self.location_patterns:
            if loc in text:
                entities['locations'].append(loc)
                
        return entities


# 使用示例
parser = VoiceCommandParser()
result = parser.parse("请带我去厨房")
print(f"意图: {result['intent']}")  # navigation
print(f"实体: {result['entities']}")  # {'locations': ['厨房'], 'objects': []}
```

### 8.3 识别效果评估（WER）

**WER**（Word Error Rate，词错误率）是评估ASR系统的主要指标。

```
WER = (S + D + I) / N × 100%

其中:
S = 替换(Substitution)错误数
D = 删除(Deletion)错误数  
I = 插入(Insertion)错误数
N = 参考文本总词数
```

```python
import jiwer

def calculate_wer(reference, hypothesis):
    """
    计算词错误率(WER)
    
    参数:
        reference: 参考文本（ground truth）
        hypothesis: 识别文本（ASR输出）
    返回:
        float: WER值（越小越好）
    """
    # 计算WER
    wer = jiwer.wer(reference, hypothesis)
    return wer

def calculate_cer(reference, hypothesis):
    """
    计算字符错误率(CER)
    适用于中文等字符语言
    """
    cer = jiwer.cer(reference, hypothesis)
    return cer

# 测试示例
ref = "请带我去厨房"
hyp = "请带我去厨房"  # 完全正确

wer = calculate_wer(ref, hyp)
cer = calculate_cer(ref, hyp)
print(f"WER: {wer:.2%}, CER: {cer:.2%}")  # WER: 0.00%, CER: 0.00%

# 错误示例
hyp2 = "请带我去卧室"
wer2 = calculate_wer(ref, hyp2)
cer2 = calculate_cer(ref, hyp2)
print(f"WER: {wer2:.2%}, CER: {cer2:.2%}")  # WER: 50.00%, CER: 16.67%

# 批量评估脚本
def evaluate_asr_system(asr_model, test_audio_files, reference_texts):
    """
    批量评估ASR系统
    
    参数:
        asr_model: ASR模型实例
        test_audio_files: 测试音频文件列表
        reference_texts: 参考文本列表
    返回:
        dict: 评估结果统计
    """
    results = []
    
    for audio_file, ref_text in zip(test_audio_files, reference_texts):
        hyp_text = asr_model.transcribe(audio_file)
        wer = calculate_wer(ref_text, hyp_text)
        cer = calculate_cer(ref_text, hyp_text)
        
        results.append({
            'file': audio_file,
            'reference': ref_text,
            'hypothesis': hyp_text,
            'wer': wer,
            'cer': cer
        })
        
    # 统计
    avg_wer = sum(r['wer'] for r in results) / len(results)
    avg_cer = sum(r['cer'] for r in results) / len(results)
    
    return {
        'results': results,
        'average_wer': avg_wer,
        'average_cer': avg_cer,
        'total_samples': len(results)
    }
```

## 10. 参考答案

### 基础题答案

**1. 选择题**

1.1 **答案：B (16kHz)**
解析：16kHz是ASR的标准采样率，足够覆盖人声频率范围（300Hz-3400Hz），同时不会产生过多冗余数据。

1.2 **答案：C (GMM-HMM)**
解析：GMM-HMM是传统的混合模型，不是端到端模型。Whisper、DeepSpeech、WeNet都是端到端模型。

1.3 **答案：A (梅尔频率倒谱系数)**
解析：MFCC = Mel-Frequency Cepstral Coefficients，即梅尔频率倒谱系数。

**2. 简答题**

2.1 **语音识别基本流程：**
```
原始音频 → 预加重 → 分帧 → 加窗 → FFT → 梅尔滤波器 → MFCC/Mel频谱
     ↓
声学模型（神经网络） → 音素序列
     ↓
语言模型 → 词序列
     ↓
解码输出文本
```

2.2 **VAD的作用：**
- 检测语音活动，过滤静音和噪声段
- 减少无效数据处理，降低计算量
- 用于判断说话开始和结束，实现流式识别
- 提升整体识别准确率

2.3 **声学模型 vs 语言模型：**
| 方面 | 声学模型 | 语言模型 |
|------|----------|----------|
| 输入 | 音频特征(MFCC/Mel) | 词序列 |
| 作用 | 建模"声音-音素"关系 | 建模"词-词"关系 |
| 本质 | 序列到序列的转换 | 序列概率约束 |
| 联系 | 两者结合才能准确识别 | 联合解码得到最终文本 |

```python
def extract_features(audio_path):
```

```python
import numpy as np

def energy_vad(audio, threshold=0.01, frame_size=2048):
    """
    基于能量的语音活动检测
    """
    # 计算每帧能量
    n_frames = len(audio) // frame_size
    energies = []
    
    for i in range(n_frames):
        frame = audio[i*frame_size:(i+1)*frame_size]
        energy = np.sqrt(np.mean(frame ** 2))
        energies.append(energy > threshold)
    
    return np.array(energies)

# 使用
audio = np.random.randn(16000)  # 1秒音频
vad_result = energy_vad(audio)
print(f"检测到语音帧: {vad_result.sum()}/{len(vad_result)}")
```

```python
class SimpleIntentRecognizer:
    def __init__(self):
        self.intents = {
            'navigation': {
                'keywords': ['去', '到', '导航', '前往'],
                'entities': ['厨房', '卧室', '客厅', '门口']
            },
            'stop': {
                'keywords': ['停', '停下', '停止', '取消'],
                'entities': []
            },
            'follow': {
                'keywords': ['跟着', '跟随', '跟'],
                'entities': []
            },
            'greet': {
                'keywords': ['你好', 'hello', 'hi'],
                'entities': []
            }
        }
        
    def recognize(self, text):
        for intent, config in self.intents.items():
            for kw in config['keywords']:
                if kw in text:
                    entities = [e for e in config['entities'] if e in text]
                    return {
                        'intent': intent,
                        'entities': entities,
                        'confidence': 1.0
                    }
        return {'intent': 'unknown', 'entities': [], 'confidence': 0.0}


# 测试

    "请带我去厨房",
    "停下",
    "跟着我",
    "你好"
]

for text in test_cases:
    result = recognizer.recognize(text)
    print(f"'{text}' → {result['intent']}")
```
