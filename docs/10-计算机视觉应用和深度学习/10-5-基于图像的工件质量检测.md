## 1. 工业视觉质检概述

### 1.1 为什么需要工业视觉质检？

在现代化工厂流水线上，产品质量检测是至关重要的一环。传统的人工目检存在以下根本性缺陷：

| 对比维度 | 人工目检 | 机器视觉质检 |
|---------|---------|------------|
| 检测速度 | 1-3件/秒（易疲劳） | 10-100件/秒（不知疲倦） |
| 检测精度 | 受主观因素影响，标准不一致 | 精度稳定，可量化 |
| 工作时间 | 8小时/天，需轮班 | 24小时连续运行 |
| 数据记录 | 人工记录，易出错 | 自动记录，可追溯 |
| 检测一致性 | 不同工人标准不同 | 同一标准，永远一致 |
| 成本 | 人工成本持续上升 | 一次性投入，长期回报 |

**典型应用场景：**

- **汽车零部件**：发动机缸体表面缺陷、螺栓螺纹完整性、焊缝质量检测
- **电子元器件**：PCB板焊点缺陷、元器件贴装位置、芯片外观检测
- **纺织品**：布匹瑕疵（断经、断纬、油污）、印花质量
- **食品包装**：瓶身裂纹、标签印刷错误、封口完整性
- **医药行业**：药片完整性、包装密封性、标签信息

### 1.2 缺陷的分类

工业质检中的"缺陷"种类繁多，但从图像处理的角度，可以分为以下几大类：

| 缺陷类型 | 图像特征 | 典型示例 | 检测方法 |
|---------|---------|---------|---------|
| **划痕** | 细长的线性暗/亮区域，边缘清晰 | 金属表面、玻璃、光学镜片 | 边缘检测+形态学 |
| **斑点/污渍** | 不规则形状的区域，与周围灰度差异明显 | 表面油污、涂料缺损 | 阈值分割+连通域分析 |
| **凹陷/凸起** | 灰度渐变区域，形状不规则 | 冲压件、铸造件表面 | 立体匹配或阴影分析 |
| **裂纹** | 细长的曲线/折线，宽度极小 | 陶瓷、玻璃、金属疲劳裂纹 | 边缘检测+线段检测 |
| **颜色异常** | 颜色偏离标准值 | 喷漆不均、染色不匀 | 颜色空间分析 |
| **形状偏差** | 轮廓与标准形状不一致 | 冲压件变形、塑料件收缩 | 模板匹配、轮廓比较 |

### 1.3 工业视觉质检的流程

```
[图像采集] → [预处理] → [特征提取] → [缺陷检测/分类] → [结果输出]
```

| 环节 | 关键操作 |
|------|---------|
| **图像采集** | 触发信号（光电传感器/编码器）、相机曝光与采集、图像传输到工控机 |
| **预处理** | 畸变校正、亮度归一化、噪声滤波（高斯/中值）、ROI 提取 |
| **特征提取** | 边缘特征（划痕、裂纹）、纹理特征（LBP、Gabor）、颜色特征（HSV） |
| **缺陷检测** | 阈值分割（大津法、自适应阈值）、边缘检测（Canny）、形态学分析 |
| **结果输出** | OK/NG 判定、缺陷类型与位置、数据上传MES、不良品剔除信号 |

## 2. 缺陷检测常用方法

### 2.1 阈值分割

**阈值分割**（Thresholding）的核心思想是：设定灰度阈值 $T$，将图像分为"目标"和"背景"两类：

$$
g(x, y) = \begin{cases} 255 & \text{if } f(x, y) > T \\ 0 & \text{otherwise} \end{cases}
$$

| 方法 | 原理 | 适用场景 | 优点 | 缺点 |
|------|------|---------|------|------|
| **固定阈值** | 手动设定固定值 $T$ | 光照稳定 | 简单快速 | 鲁棒性差 |
| **大津法（Otsu）** | 自动寻找使类间方差最大的阈值 | 双峰分布 | 全局最优自适用 | 对非双峰效果差 |
| **自适应阈值** | 每像素邻域计算阈值 | 光照不均 | 能处理渐变光照 | 计算量较大 |

```python
import cv2
import numpy as np

def thresholding_methods_demo():
    """
    演示三种阈值分割方法的对比
    """
    
    # 创建测试图像（模拟有缺陷的工件）
    # 背景灰度 150，缺陷区域灰度 80（暗缺陷）
    test_image = np.ones((400, 400), dtype=np.uint8) * 150
    
    # 添加不规则暗缺陷区域（模拟斑点）
    cv2.ellipse(test_image, (200, 200), (60, 40), 0, 0, 360, 80, -1)
    
    # 添加细长划痕（模拟划痕）
    cv2.line(test_image, (50, 150), (350, 180), 70, 3)
    
    # 添加高斯噪声模拟真实情况
    noise = np.random.normal(0, 10, test_image.shape).astype(np.int16)
    test_image = np.clip(test_image.astype(np.int16) + noise, 0, 255).astype(np.uint8)
    
    print("使用模拟测试图像")
    
    results = {}
    
    # 1. 固定阈值
    fixed_threshold = 120
    _, fixed_result = cv2.threshold(test_image, fixed_threshold, 255, cv2.THRESH_BINARY)
    results["固定阈值"] = fixed_result
    print(f"\n【固定阈值法】阈值 = {fixed_threshold}")
    
    # 2. 大津法（Otsu）
    _, otsu_result = cv2.threshold(test_image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    results["大津法"] = otsu_result
    print(f"【大津法】自动计算最优阈值 = {_}")
    
    # 3. 自适应阈值（处理光照不均）
    adaptive_result = cv2.adaptiveThreshold(
        test_image, 255,
        cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
        cv2.THRESH_BINARY,
        blockSize=21, C=10
    )
    results["自适应阈值"] = adaptive_result
    print("【自适应阈值法】邻域=21x21, C=10")
    
    print("\n各方法检测到的缺陷像素数量：")
    for method, result in results.items():
        defect_pixels = np.count_nonzero(result == 255)
        print(f"  {method}: {defect_pixels} 像素")
    
    return results

def create_uneven_lighting_image():
    """创建模拟光照不均的图像（中心亮、四周暗）"""
    h, w = 400, 400
    image = np.zeros((h, w), dtype=np.uint8)
    center_x, center_y = w // 2, h // 2
    for y in range(h):
        for x in range(w):
            dist = np.sqrt((x - center_x)**2 + (y - center_y)**2)
            max_dist = np.sqrt(center_x**2 + center_y**2)
            normalized_dist = dist / max_dist
            brightness = int(200 - normalized_dist * 120)
            ellipse_val = 150 if ((x-center_x)**2/100**2 + (y-center_y)**2/60**2) < 1 else brightness
            image[y, x] = ellipse_val
    # 添加缺陷
    cv2.circle(image, (100, 100), 15, 60, -1)   # 左上角暗缺陷
    cv2.circle(image, (300, 300), 15, 240, -1)  # 右下角亮缺陷
    return image

def compare_thresholds_uneven_light():
    """演示光照不均时，固定阈值 vs 自适应阈值的对比"""
    uneven_image = create_uneven_lighting_image()
    
    print("=" * 60)
    print("光照不均情况下的阈值分割对比")
    print("=" * 60)
    
    _, fixed_thresh = cv2.threshold(uneven_image, 130, 255, cv2.THRESH_BINARY)
    _, otsu_thresh = cv2.threshold(uneven_image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
    adaptive_thresh = cv2.adaptiveThreshold(uneven_image, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                             cv2.THRESH_BINARY, blockSize=31, C=5)
    
    fixed_defects = np.count_nonzero(fixed_thresh == 255)
    otsu_defects = np.count_nonzero(otsu_thresh == 255)
    adaptive_defects = np.count_nonzero(adaptive_thresh == 255)
    
    print(f"固定阈值(T=130)检测到: {fixed_defects} 像素")
    print(f"大津法检测到: {otsu_defects} 像素")
    print(f"自适应阈值检测到: {adaptive_defects} 像素")
    print("\n结论：光照不均时，自适应阈值明显优于固定阈值和大津法")
    
    return {"fixed": fixed_defects, "otsu": otsu_defects, "adaptive": adaptive_defects}

threshold_results = thresholding_methods_demo()
uneven_results = compare_thresholds_uneven_light()
```

### 2.2 边缘检测

边缘检测是检测**灰度突变**区域的技术，对于划痕、裂纹、轮廓缺陷等效果显著。

| 算子 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| **Sobel** | 梯度近似（一阶导数） | 对垂直/水平边缘敏感，计算快 | 对噪声敏感，边缘粗 |
| **Laplacian** | 二阶导数 | 对所有方向边缘等效 | 对噪声非常敏感 |
| **Canny** | 多级边缘检测 | 边缘细、定位准、抗噪好 | 参数多（低阈/高阈） |

```python
import cv2
import numpy as np

def edge_detection_comparison():
    """演示不同边缘检测算子的效果对比"""
    
    h, w = 300, 400
    test_image = np.ones((h, w), dtype=np.uint8) * 140
    cv2.rectangle(test_image, (50, 50), (350, 250), 140, -1)  # 工件主体
    cv2.line(test_image, (80, 100), (320, 120), 80, 2)  # 暗划痕
    cv2.line(test_image, (100, 180), (300, 200), 200, 2)  # 亮划痕
    
    noise = np.random.normal(0, 8, test_image.shape).astype(np.int16)
    test_image = np.clip(test_image.astype(np.int16) + noise, 0, 255).astype(np.uint8)
    
    results = {}
    
    # Sobel 边缘检测
    sobel_x = cv2.Sobel(test_image, cv2.CV_64F, 1, 0, ksize=3)
    sobel_y = cv2.Sobel(test_image, cv2.CV_64F, 0, 1, ksize=3)
    sobel_magnitude = np.sqrt(sobel_x**2 + sobel_y**2)
    sobel_magnitude = np.uint8(sobel_magnitude / sobel_magnitude.max() * 255)
    results["Sobel"] = sobel_magnitude
    print("【Sobel算子】计算x和y方向梯度，合并求模值")
    
    # Laplacian 边缘检测
    laplacian = cv2.Laplacian(test_image, cv2.CV_64F, ksize=3)
    laplacian = np.uint8(np.absolute(laplacian))
    results["Laplacian"] = laplacian
    print("【Laplacian算子】二阶导数，对噪声敏感")
    
    # Canny 边缘检测
    canny_100_200 = cv2.Canny(test_image, 100, 200)
    canny_50_150 = cv2.Canny(test_image, 50, 150)
    results["Canny(100,200)"] = canny_100_200
    results["Canny(50,150)"] = canny_50_150
    print("【Canny算子】低阈100/高阈200: 检测到{}个边缘像素".format(np.count_nonzero(canny_100_200)))
    print("【Canny算子】低阈50/高阈150: 检测到{}个边缘像素".format(np.count_nonzero(canny_50_150)))
    
    print("\n" + "=" * 50)
    print("Canny 边缘检测阈值参数选择指南")
    print("=" * 50)
    print("低阈值：过低→伪边缘多，过高→遗漏弱边缘")
    print("高阈值：建议为低阈值的 2-3 倍")
    print("原则：噪声多→提高阈值，边缘对比度低→降低阈值")
    
    return results

edge_results = edge_detection_comparison()
```

### 2.3 形态学分析

| 运算 | 定义 | 效果 | 质检应用 |
|------|------|------|---------|
| **腐蚀（Erosion）** | 减少前景区域 | 去除小的噪点 | 消除伪缺陷 |
| **膨胀（Dilation）** | 扩大前景区域 | 填充电小空洞 | 连接断开的边缘 |
| **开运算** | 先腐蚀后膨胀 | 去噪点，保留形状 | 去除椒盐噪声 |
| **闭运算** | 先膨胀后腐蚀 | 填空洞，保留形状 | 填充裂纹内的暗区 |
| **梯度** | 膨胀-原图 | 提取边缘 | 缺陷边缘检测 |
| **顶帽（Top-hat）** | 原图-开运算 | 提取亮细节 | 提取亮点缺陷 |
| **底帽（Black-hat）** | 闭运算-原图 | 提取暗细节 | 提取暗点/划痕 |

```python
import cv2
import numpy as np

def morphology_demo():
    """形态学运算演示"""
    
    h, w = 300, 400
    test_image = np.ones((h, w), dtype=np.uint8) * 180
    cv2.rectangle(test_image, (40, 40), (360, 260), 180, -1)  # 工件主体
    
    # 散点噪声（腐蚀可去除）
    for _ in range(50):
        x, y = np.random.randint(50, 350, 2)
        test_image[y, x] = 50 if np.random.random() > 0.5 else 230
    
    # 暗色裂纹（闭运算可填充）
    cv2.line(test_image, (100, 100), (300, 150), 60, 4)
    
    # 亮点缺陷（顶帽变换可提取）
    cv2.circle(test_image, (200, 150), 10, 240, -1)
    
    noise = np.random.normal(0, 15, test_image.shape).astype(np.int16)
    test_image = np.clip(test_image.astype(np.int16) + noise, 0, 255).astype(np.uint8)
    
    print("=" * 60)
    print("形态学运算演示")
    print("=" * 60)
    
    for size_name, ksize in [("small(3)", 3), ("medium(7)", 7), ("large(15)", 15)]:
        kernel = np.ones((ksize, ksize), dtype=np.uint8)
        
        # 开运算（先腐后涨）：去除散点噪声
        opened = cv2.morphologyEx(test_image, cv2.MORPH_OPEN, kernel)
        # 闭运算（先涨后腐）：填充裂纹空洞
        closed = cv2.morphologyEx(test_image, cv2.MORPH_CLOSE, kernel)
        # 顶帽：提取亮细节
        tophat = cv2.morphologyEx(test_image, cv2.MORPH_TOPHAT, kernel)
        # 底帽：提取暗细节
        blackhat = cv2.morphologyEx(test_image, cv2.MORPH_BLACKHAT, kernel)
        
        print(f"\n【结构元素 {ksize}x{ksize}】")
        print(f"  开运算: 去除散点噪声，保留主体形状")
        print(f"  闭运算: 填充裂纹空洞，保留主体形状")
        print(f"  顶帽: 提取亮点缺陷 {np.count_nonzero(tophat > 30)} 像素")
        print(f"  底帽: 提取暗点/划痕缺陷 {np.count_nonzero(blackhat > 30)} 像素")
```

## 3. 表面缺陷检测实践

### 3.1 缺陷检测完整 Pipeline

下面实现一个完整的缺陷检测 pipeline，整合所有方法：

```python
import cv2
import numpy as np
from typing import List, Dict, Tuple, Optional


class DefectDetector:
    """
    工业缺陷检测器
    
    功能：
    1. 支持多种缺陷类型：划痕、斑点、裂纹
    2. 提供可配置的检测参数
    3. 输出缺陷位置、大小、类型
    """
    
    def __init__(self, config: Optional[Dict] = None):
        """初始化缺陷检测器"""
        self.config = config or self._default_config()
    
    def _default_config(self) -> Dict:
        """默认配置参数"""
        return {
            "preprocessing": {
                "denoise": True,
                "denoise_strength": 3,
                "normalize_brightness": True,
            },
            "defect_detection": {
                "blob_threshold": 30,
                "blob_min_area": 20,
                "blob_max_area": 5000,
                "scratch_length_threshold": 50,
                "scratch_width_threshold": 5,
                "crack_threshold": 100,
            },
            "classification": {
                "scratch_aspect_ratio": 10.0,
                "blob_circularity": 0.5,
            }
        }
    
    def detect(self, image: np.ndarray) -> Dict:
        """
        执行完整的缺陷检测流程
        
        参数:
            image: 输入图像（灰度图或彩色图）
        
        返回:
            result: {
                "has_defect": bool, 是否有缺陷
                "defects": List[Dict], 缺陷列表
                "defect_count": int, 缺陷数量
                "image_with_defects": np.ndarray, 可视化图像
            }
        """
        # Step 1: 预处理
        processed = self._preprocess(image)
        
        # Step 2: 多方法缺陷检测
        blob_defects = self._detect_blobs(processed)
        scratch_defects = self._detect_scratches(processed)
        crack_defects = self._detect_cracks(processed)
        
        # Step 3: 合并结果
        all_defects = blob_defects + scratch_defects + crack_defects
        
        # Step 4: 生成可视化
        vis_image = self._visualize_defects(image.copy(), all_defects)
        
        return {
            "has_defect": len(all_defects) > 0,
            "defects": all_defects,
            "defect_count": len(all_defects),
            "defect_types": {"blob": len(blob_defects), "scratch": len(scratch_defects), "crack": len(crack_defects)},
            "image_with_defects": vis_image,
        }
    
    def _preprocess(self, image: np.ndarray) -> np.ndarray:
        """图像预处理：转灰度 -> 去噪 -> 亮度归一化"""
        if len(image.shape) == 3:
            gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
        else:
            gray = image.copy()
        
        cfg = self.config["preprocessing"]
        if cfg["denoise"]:
            gray = cv2.fastNlMeansDenoising(gray, None, cfg["denoise_strength"], 7, 21)
        if cfg["normalize_brightness"]:
            clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
            gray = clahe.apply(gray)
        return gray
    
    def _detect_blobs(self, gray: np.ndarray) -> List[Dict]:
        """检测斑点/污渍缺陷：顶帽+底帽变换 -> 连通域分析"""
        cfg = self.config["defect_detection"]
        defects = []
        kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (7, 7))
        
        tophat = cv2.morphologyEx(gray, cv2.MORPH_TOPHAT, kernel)
        blackhat = cv2.morphologyEx(gray, cv2.MORPH_BLACKHAT, kernel)
        
        for name, morph_img in [("亮斑点", tophat), ("暗斑点", blackhat)]:
            _, binary = cv2.threshold(morph_img, cfg["blob_threshold"], 255, cv2.THRESH_BINARY)
            num_labels, labels, stats, centroids = cv2.connectedComponentsWithStats(binary, connectivity=8)
            
            for i in range(1, num_labels):
                area = stats[i, cv2.CC_STAT_AREA]
                if cfg["blob_min_area"] <= area <= cfg["blob_max_area"]:
                    x = stats[i, cv2.CC_STAT_LEFT]
                    y = stats[i, cv2.CC_STAT_TOP]
                    w = stats[i, cv2.CC_STAT_WIDTH]
                    h = stats[i, cv2.CC_STAT_HEIGHT]
                    
                    defects.append({
                        "type": name,
                        "area": int(area),
                        "bbox": (x, y, w, h),
                        "center": (float(centroids[i, 0]), float(centroids[i, 1])),
                        "severity": "high" if area > cfg["blob_max_area"] * 0.5 else "medium"
                    })
        return defects
    
    def _detect_scratches(self, gray: np.ndarray) -> List[Dict]:
        """检测划痕缺陷：形态学梯度 -> 霍夫线变换"""
        cfg = self.config["defect_detection"]
        defects = []
        
        # 形态学梯度：提取边缘
        kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (3, 3))
        gradient = cv2.morphologyEx(gray, cv2.MORPH_GRADIENT, kernel)
        
        # 二值化
        _, binary = cv2.threshold(gradient, cfg["crack_threshold"], 255, cv2.THRESH_BINARY)
        
        # 霍夫线变换检测直线
        lines = cv2.HoughLinesP(binary, rho=1, theta=np.pi/180,
                                 threshold=30, minLineLength=cfg["scratch_length_threshold"],
                                 maxLineGap=5)
        
        if lines is not None:
            for line in lines:
                x1, y1, x2, y2 = line[0]
                length = np.sqrt((x2 - x1)**2 + (y1 - y2)**2)
                
                if length >= cfg["scratch_length_threshold"]:
                    # 计算线宽（通过沿线采样估算）
                    width = self._estimate_line_width(gray, x1, y1, x2, y2)
                    
                    if width <= cfg["scratch_width_threshold"]:
                        defects.append({
                            "type": "划痕",
                            "length": float(length),
                            "width": float(width),
                            "start": (x1, y1),
                            "end": (x2, y2),
                            "severity": "high" if length > 200 else "medium"
                        })
        return defects
    
    def _estimate_line_width(self, gray: np.ndarray, x1: int, y1: int, x2: int, y2: int) -> float:
        """沿线采样估计划痕宽度"""
        # 在线的垂直方向采样灰度剖面，估算宽度
        if abs(x2 - x1) > abs(y2 - y1):
            ys = np.linspace(y1, y2, num=10, dtype=int)
            xs = np.linspace(x1, x2, num=10, dtype=int).astype(float)
            widths = []
            for xi, yi in zip(xs, ys):
                if 0 <= yi < gray.shape[0] and 0 <= int(xi) < gray.shape[1]:
                    profile = gray[yi, :]
                    center_x = int(xi)
                    if center_x > 0 and center_x < len(profile) - 1:
                        baseline = (profile[center_x - 1] + profile[center_x + 1]) / 2
                        diff = abs(int(profile[center_x]) - baseline)
                        widths.append(diff / 2.0)
            return np.mean(widths) if widths else 2.0
        else:
            xs = np.linspace(x1, x2, num=10, dtype=int)
            ys = np.linspace(y1, y2, num=10, dtype=float).astype(int)
            widths = []
            for yi, xi in zip(ys, xs):
                if 0 <= xi < gray.shape[1] and 0 <= yi < gray.shape[0]:
                    profile = gray[:, xi]
                    center_y = yi
                    if center_y > 0 and center_y < len(profile) - 1:
                        baseline = (profile[center_y - 1] + profile[center_y + 1]) / 2
                        diff = abs(int(profile[center_y]) - baseline)
                        widths.append(diff / 2.0)
            return np.mean(widths) if widths else 2.0
    
    def _detect_cracks(self, gray: np.ndarray) -> List[Dict]:
        """检测裂纹缺陷：Canny边缘检测 -> 轮廓分析"""
        cfg = self.config["defect_detection"]
        defects = []
        
        # Canny边缘检测
        edges = cv2.Canny(gray, cfg["crack_threshold"], cfg["crack_threshold"] * 2)
        
        # 形态学处理：闭运算连接断开的边缘
        kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (5, 5))
        closed = cv2.morphologyEx(edges, cv2.MORPH_CLOSE, kernel)
        
        # 轮廓分析
        contours, _ = cv2.findContours(closed, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
        
        for contour in contours:
            area = cv2.contourArea(contour)
            if area >= cfg["blob_min_area"]:
                # 计算轮廓的最小外接矩形
                rect = cv2.minAreaRect(contour)
                (cx, cy), (w, h), angle = rect
                
                # 裂纹通常是细长的，计算长宽比
                longer = max(w, h)
                shorter = min(w, h)
                aspect_ratio = longer / shorter if shorter > 0 else 1
                
                if aspect_ratio >= cfg["classification"]["scratch_aspect_ratio"]:
                    defects.append({
                        "type": "裂纹",
                        "area": float(area),
                        "bbox": cv2.boundingRect(contour),
                        "aspect_ratio": float(aspect_ratio),
                        "severity": "high"
                    })
        return defects
    
    def _visualize_defects(self, image: np.ndarray, defects: List[Dict]) -> np.ndarray:
        """在图像上标注检测到的缺陷"""
        color_map = {
            "亮斑点": (0, 255, 255),   # 黄色
            "暗斑点": (255, 0, 255),   # 紫色
            "划痕": (0, 0, 255),        # 红色
            "裂纹": (255, 0, 0),       # 蓝色
        }
        
        if len(image.shape) == 2:
            image = cv2.cvtColor(image, cv2.COLOR_GRAY2BGR)
        
        for defect in defects:
            color = color_map.get(defect["type"], (0, 255, 0))
            
            if "bbox" in defect:
                x, y, w, h = defect["bbox"]
                cv2.rectangle(image, (x, y), (x + w, y + h), color, 2)
                cv2.putText(image, defect["type"], (x, y - 5),
                            cv2.FONT_HERSHEY_SIMPLEX, 0.5, color, 1)
            
            if "start" in defect and "end" in defect:
                x1, y1 = defect["start"]
                x2, y2 = defect["end"]
                cv2.line(image, (x1, y1), (x2, y2), color, 2)
                cv2.putText(image, f"划痕 L={defect['length']:.0f}px",
                            (x1, y1 - 5), cv2.FONT_HERSHEY_SIMPLEX, 0.5, color, 1)
        
        return image


def demo_defect_detection():
```

### 3.2 连通域分析

**连通域分析**（Connected Component Analysis）是缺陷检测中的核心技术之一。它将二值图像中灰度相连的前景区域划分为不同的连通域，每个连通域对应一个独立的缺陷区域。

**连通性的定义：**

| 类型 | 邻接方式 | 适用场景 |
|------|---------|---------|
| **4连通（4-connectivity）** | 只考虑上下左右4个邻居 | 边缘清晰的缺陷 |
| **8连通（8-connectivity）** | 考虑上下左右+4个对角邻居 | 形状不规则的缺陷 |

```python
import cv2
import numpy as np

def connected_component_analysis_demo():
    """
    连通域分析演示
    """
    
    # 创建测试图像：包含多个缺陷区域
    h, w = 300, 400
    binary = np.zeros((h, w), dtype=np.uint8)
    
    # 添加几个独立的缺陷区域
    cv2.rectangle(binary, (50, 50), (80, 80), 255, -1)    # 缺陷1：矩形
    cv2.circle(binary, (200, 150), 20, 255, -1)           # 缺陷2：圆形
    cv2.circle(binary, (250, 160), 10, 255, -1)          # 缺陷3：小圆
    cv2.ellipse(binary, (150, 220), (30, 15), 0, 0, 360, 255, -1)  # 缺陷4：椭圆
    
    print("=" * 60)
    print("连通域分析演示")
    print("=" * 60)
    
    # ========== 4连通 vs 8连通 ==========
    for connectivity in [4, 8]:
        num_labels, labels, stats, centroids = cv2.connectedComponentsWithStats(
            binary, connectivity=connectivity
        )
        
        print(f"\n【{connectivity}连通】检测到 {num_labels - 1} 个独立缺陷区域")
        print(f"  {'标签ID':<8} {'面积(px)':<12} {'中心坐标':<20} {'占比':<10}")
        
        for i in range(1, num_labels):
            area = stats[i, cv2.CC_STAT_AREA]
            cx, cy = centroids[i]
            ratio = area / (h * w) * 100
            print(f"  {i:<8} {area:<12} ({cx:.1f}, {cy:.1f}){'':<8} {ratio:.2f}%")
    
    print("\n结论：8连通会合并8方向相邻的区域，比4连通检测到的区域更少（对角相邻被合并）")
    
    # ========== 过滤小噪点 ==========
    print("\n" + "=" * 50)
    print("基于面积的连通域过滤")
    print("=" * 50)
    
    min_area_threshold = 200  # 最小面积阈值（像素）
    num_labels, labels, stats, centroids = cv2.connectedComponentsWithStats(binary, connectivity=8)
    
    real_defects = []
    for i in range(1, num_labels):
        area = stats[i, cv2.CC_STAT_AREA]
        if area >= min_area_threshold:
            x = stats[i, cv2.CC_STAT_LEFT]
            y = stats[i, cv2.CC_STAT_TOP]
            w = stats[i, cv2.CC_STAT_WIDTH]
            h_stat = stats[i, cv2.CC_STAT_HEIGHT]
            cx, cy = centroids[i]
            real_defects.append({"id": i, "area": area, "bbox": (x, y, w, h_stat), "center": (cx, cy)})
    
    print(f"原始区域数: {num_labels - 1}")
    print(f"过滤后（面积>{min_area_threshold}px）: {len(real_defects)}")
    
    for defect in real_defects:
        print(f"  区域{defect['id']}: 面积={defect['area']}px, "
              f"BBOX={defect['bbox']}, 中心=({defect['center'][0]:.1f}, {defect['center'][1]:.1f})")
    
    return {"real_defects": real_defects, "total_regions": num_labels - 1}
```

## 4. 尺寸测量与模板匹配

### 4.1 几何尺寸测量

在工业质检中，**尺寸测量**是另一大核心任务。常见需求包括：测量工件的长、宽、孔径、间距等几何参数，判断是否在公差范围内。

```python
import cv2
import numpy as np

def geometric_measurement_demo():
    """
    几何尺寸测量演示
    """
    
    print("=" * 60)
    print("几何尺寸测量")
    print("=" * 60)
    
    # 创建测试图像：模拟带圆孔的金属工件
    h, w = 400, 500
    image = np.ones((h, w), dtype=np.uint8) * 200
    
    # 工件外轮廓（矩形）
    cv2.rectangle(image, (50, 50), (450, 350), 100, 3)
    
    # 圆孔（模拟要测量的特征）
    cv2.circle(image, (150, 200), 40, 100, 3)   # 孔1
    cv2.circle(image, (300, 200), 40, 100, 3)   # 孔2
    
    # 测量参数
    # 假设：每个像素对应实际 0.1mm（通过标定获得）
    pixel_to_mm = 0.1  # mm/px
    
    # ========== 1. 轮廓提取 ==========
    _, binary = cv2.threshold(image, 128, 255, cv2.THRESH_BINARY)
    contours, _ = cv2.findContours(binary, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    print("\n【轮廓检测结果】")
    print(f"检测到 {len(contours)} 个外轮廓")
    
    measurements = []
    
    for i, contour in enumerate(contours):
        # 计算轮廓面积
        area_px = cv2.contourArea(contour)
        area_mm2 = area_px * (pixel_to_mm ** 2)
        
        # 计算周长
        perimeter_px = cv2.arcLength(contour, True)
        perimeter_mm = perimeter_px * pixel_to_mm
        
        # 最小外接矩形
        rect = cv2.minAreaRect(contour)
        (cx, cy), (rect_w, rect_h), angle = rect
        rect_w_mm = rect_w * pixel_to_mm
        rect_h_mm = rect_h * pixel_to_mm
        
        # 等效直径（面积相同的圆的直径）
        equivalent_diameter_px = np.sqrt(4 * area_px / np.pi)
        equivalent_diameter_mm = equivalent_diameter_px * pixel_to_mm
        
        measurements.append({
            "id": i + 1,
            "area_mm2": area_mm2,
            "perimeter_mm": perimeter_mm,
            "bbox_mm": (rect_w_mm, rect_h_mm),
            "equivalent_diameter_mm": equivalent_diameter_mm,
        })
        
        print(f"\n轮廓 {i+1}:")
        print(f"  面积: {area_mm2:.2f} mm²")
        print(f"  周长: {perimeter_mm:.2f} mm")
        print(f"  最小外接矩形: {rect_w_mm:.2f} x {rect_h_mm:.2f} mm")
        print(f"  等效直径: {equivalent_diameter_mm:.2f} mm")
    
    # ========== 2. 圆拟合（测量孔径） ==========
    print("\n【圆拟合 - 测量孔径】")
    
    # 重新检测（用于检测内圆孔）
    contours, _ = cv2.findContours(255 - binary, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    for i, contour in enumerate(contours):
        (cx, cy), radius_px = cv2.minEnclosingCircle(contour)
        radius_mm = radius_px * pixel_to_mm
        diameter_mm = radius_mm * 2
        
        # 使用最小二乘法拟合圆（更精确）
        if len(contour) >= 5:
            (fx, fy), fitted_radius = cv2.minEnclosingCircle(contour)
            fitted_diameter_mm = fitted_radius * 2 * pixel_to_mm
            
            print(f"\n圆孔 {i+1}:")
            print(f"  圆心: ({cx:.1f}, {cy:.1f}) 像素")
            print(f"  拟合直径: {fitted_diameter_mm:.2f} mm")
    
    # ========== 3. 距离测量 ==========
    print("\n【距离测量 - 孔间距】")
    
    # 两个圆孔圆心
    center1 = np.array([150, 200], dtype=np.float32)
    center2 = np.array([300, 200], dtype=np.float32)
    
    distance_px = np.linalg.norm(center2 - center1)
    distance_mm = distance_px * pixel_to_mm
    
    print(f"  孔1圆心: {center1} 像素")
    print(f"  孔2圆心: {center2} 像素")
    print(f"  圆心间距: {distance_mm:.2f} mm")
    
    # 公差判断
    nominal = 150.0  # 公称值 150mm
    tolerance = 0.5  # 公差 ±0.5mm
    
    if abs(distance_mm - nominal) <= tolerance:
        print(f"  判定: OK (距离 {distance_mm:.2f}mm 在 {nominal}±{tolerance}mm 范围内)")
    else:
```

### 4.2 模板匹配

**模板匹配**（Template Matching）是工业质检中检测"形状偏差"的经典方法。其原理是在待检测图像中滑动一个模板图像，找到与模板最相似的位置。

**OpenCV 模板匹配方法对比：**

| 方法 | 公式 | 特点 |
|------|------|------|
| **TM_SQDIFF** | $\sum (I(x,y) - T(x,y))^2$ | 值越小越匹配（差值平方和） |
| **TM_SQDIFF_NORMED** | 归一化SQDIFF | 0=完美匹配，1=完全不匹配 |
| **TM_CCORR** | $\sum I(x,y) \cdot T(x,y)$ | 值越大越匹配 |
| **TM_CCORR_NORMED** | 归一化CCORR | 0=完全不匹配，1=完美匹配 |
| **TM_CCOEFF** | 去均值相关 | 对灰度线性变化鲁棒 |
| **TM_CCOEFF_NORMED** | 归一化相关系数 | 抗灰度增益能力最强，推荐使用 |

```python
import cv2
import numpy as np

def template_matching_demo():
    """
    模板匹配演示：检测工件是否与标准模板一致
    """
    
    print("=" * 60)
    print("模板匹配 - 形状偏差检测")
    print("=" * 60)
    
    # 创建"标准模板"图像
    template_h, template_w = 100, 100
    template = np.ones((template_h, template_w), dtype=np.uint8) * 150
    
    # 模板特征：中心圆（模拟标准工件）
    cv2.circle(template, (50, 50), 30, 100, -1)
    cv2.circle(template, (50, 50), 15, 180, -1)  # 内圆（孔）
    cv2.rectangle(template, (10, 10), (25, 25), 80, -1)  # 左上角标记
    
    # 创建"待检测图像"：包含标准件和变形件
    scene_h, scene_w = 300, 400
    scene = np.ones((scene_h, scene_w), dtype=np.uint8) * 150
    
    # 位置1：标准工件（无缺陷）
    cv2.circle(scene, (70, 70), 30, 100, -1)
    cv2.circle(scene, (70, 70), 15, 180, -1)
    cv2.rectangle(scene, (30, 30), (45, 45), 80, -1)
    
    # 位置2：变形工件（稍大）
    cv2.circle(scene, (200, 70), 33, 100, -1)   # 半径稍大
    cv2.circle(scene, (200, 70), 15, 180, -1)
    cv2.rectangle(scene, (158, 30), (173, 45), 80, -1)
    
    # 位置3：偏移工件（位置不对）
    cv2.circle(scene, (70, 200), 30, 100, -1)
    cv2.circle(scene, (70, 200), 15, 180, -1)
    cv2.rectangle(scene, (30, 160), (45, 175), 80, -1)
    
    # 添加噪声
    noise = np.random.normal(0, 10, scene.shape).astype(np.int16)
    scene = np.clip(scene.astype(np.int16) + noise, 0, 255).astype(np.uint8)
    
    # ========== 模板匹配 ==========
    # 推荐使用 TM_CCOEFF_NORMED（抗灰度线性变化能力最强）
    match_methods = {
        "TM_SQDIFF_NORMED": cv2.TM_SQDIFF_NORMED,   # 值越小越匹配
        "TM_CCORR_NORMED": cv2.TM_CCORR_NORMED,    # 值越大越匹配
        "TM_CCOEFF_NORMED": cv2.TM_CCOEFF_NORMED,  # 值越大越匹配，推荐
    }
    
    print("\n【模板匹配结果】")
    print("在待检测图像中搜索与模板匹配的区域\n")
    
    results = {}
    
    for method_name, method in match_methods.items():
        # 执行模板匹配
        result = cv2.matchTemplate(scene, template, method)
        results[method_name] = result
        
        # 获取最匹配位置
        if method in [cv2.TM_SQDIFF_NORMED]:
            min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)
            best_match_loc = min_loc
            match_score = min_val
            is_perfect = min_val < 0.1  # SQDIFF值越小越好，<0.1认为完美匹配
        else:
            min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)
            best_match_loc = max_loc
            match_score = max_val
            is_perfect = max_val > 0.95  # NCC/CCORR值越大越好，>0.95认为完美匹配
        
        print(f"{method_name}:")
        print(f"  最佳匹配位置: {best_match_loc}")
        print(f"  匹配得分: {match_score:.4f}")
        print(f"  完美匹配: {'是' if is_perfect else '否'}")
    
    # ========== 多尺度模板匹配 ==========
    print("\n" + "=" * 50)
    print("多尺度模板匹配（检测尺寸偏差）")
    print("=" * 50)
    
    # 当工件可能存在尺寸缩放时，需要多尺度搜索
    scale_range = np.linspace(0.8, 1.2, 20)  # 80% ~ 120% 的尺度范围
    best_scale = 1.0
    best_score = -1
    best_location = None
    
    for scale in scale_range:
        # 按比例缩放模板
        scaled_template = cv2.resize(template, None, fx=scale, fy=scale, interpolation=cv2.INTER_LINEAR)
        
        # 执行匹配
        result = cv2.matchTemplate(scene, scaled_template, cv2.TM_CCOEFF_NORMED)
        min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)
        
        if max_val > best_score:
            best_score = max_val
            best_scale = scale
            best_location = max_loc
    
    print(f"最佳尺度: {best_scale:.2f} ({best_scale*100:.0f}%)")
    print(f"最佳匹配得分: {best_score:.4f}")
    print(f"匹配位置: {best_location}")
    
    if 0.95 <= best_scale <= 1.05:
        print("判定: OK (尺度在 95%~105% 范围内)")
    else:
        print("判定: NG (尺度偏差超出 ±5%)")
    
    return results

tm_results = template_matching_demo()
```

### 5.3 ROS2质检节点实现

```python
# ROS2 工业视觉质检节点
# 文件: quality_inspection_node.py

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
from std_msgs.msg import Bool, Int32, String, Float32
from geometry_msgs.msg import Point
from cv_bridge import CvBridge
import cv2
import numpy as np
from typing import List, Dict, Optional


class QualityInspectionNode(Node):
    """
    ROS2 工业视觉质检节点
    
    功能：
    1. 订阅相机图像话题
    2. 执行缺陷检测和尺寸测量
    3. 发布质检结果（OK/NG）
    4. 可选：发布带标注的调试图像
    """
    
    def __init__(self):
        super().__init__('quality_inspection_node')
        
        # CvBridge 用于 ROS 图像消息和 OpenCV 图像之间的转换
        self.bridge = CvBridge()
        
        # 声明参数
        self.declare_parameter('camera_topic', '/image_raw')
        self.declare_parameter('inspection_mode', 'defect_detection')  # defect_detection / measurement
        self.declare_parameter('debug_image', True)
        self.declare_parameter('pixel_to_mm', 0.1)  # 标定参数：像素到毫米的转换
        
        self.camera_topic = self.get_parameter('camera_topic').value
        self.inspection_mode = self.get_parameter('inspection_mode').value
        self.debug_image = self.get_parameter('debug_image').value
        self.pixel_to_mm = self.get_parameter('pixel_to_mm').value
        
        # 初始化缺陷检测器
        self.detector = DefectDetector()
        
        # 订阅原始图像话题
        self.subscription = self.create_subscription(
            Image,
            self.camera_topic,
            self.image_callback,
            10  # 队列大小
        )
        
        # 发布质检结果话题
        self.status_publisher = self.create_publisher(Bool, '/inspection/status', 10)
        self.defect_count_publisher = self.create_publisher(Int32, '/inspection/defect_count', 10)
        self.debug_image_publisher = self.create_publisher(Image, '/inspection/debug_image', 10)
        
        self.get_logger().info(f'工业视觉质检节点已启动')
        self.get_logger().info(f'  订阅话题: {self.camera_topic}')
        self.get_logger().info(f'  检测模式: {self.inspection_mode}')
        
        # 统计
        self.frame_count = 0
        self.total_defects = 0
        self.ok_count = 0
        self.ng_count = 0
    
    def image_callback(self, msg):
        """
        图像回调函数：每次收到新图像时执行质检
        
        参数:
            msg: sensor_msgs/Image 类型的消息
        """
        try:
            # 将 ROS 图像消息转换为 OpenCV 图像
            cv_image = self.bridge.imgmsg_to_cv2(msg, desired_encoding='bgr8')
            
            # 执行缺陷检测
            result = self.detector.detect(cv_image)
            
            # 发布 OK/NG 状态
            status_msg = Bool()
            status_msg.data = not result["has_defect"]
            self.status_publisher.publish(status_msg)
            
            # 发布缺陷数量
            count_msg = Int32()
            count_msg.data = result["defect_count"]
            self.defect_count_publisher.publish(count_msg)
            
            # 发布带标注的调试图像
            if self.debug_image:
                debug_msg = self.bridge.cv2_to_imgmsg(
                    result["image_with_defects"],
                    encoding='bgr8'
                )
                debug_msg.header = msg.header  # 保持时间戳一致
                self.debug_image_publisher.publish(debug_msg)
            
            # 更新统计
            self.frame_count += 1
            if result["has_defect"]:
                self.ng_count += 1
            else:
                self.ok_count += 1
            
            # 每100帧打印一次统计
            if self.frame_count % 100 == 0:
                ok_rate = self.ok_count / self.frame_count * 100
                self.get_logger().info(
                    f'统计(最近100帧): OK={self.ok_count}, NG={self.ng_count}, '
                    f'合格率={ok_rate:.1f}%'
                )
            
        except Exception as e:
            self.get_logger().error(f'质检处理失败: {str(e)}')


def main(args=None):
    """
    节点入口函数
    """
    rclpy.init(args=args)
    
    try:
        node = QualityInspectionNode()
        rclpy.spin(node)  # 保持节点运行
    except KeyboardInterrupt:
        pass
    finally:
        if rclpy.ok():
            rclpy.shutdown()


if __name__ == '__main__':
    main()
```

### 5.5 标定与精度保证

工业视觉质检的**测量精度**是关键指标，需要通过标定来保证：

| 标定项 | 方法 | 精度保证 |
|-------|------|---------|
| **像素到物理单位** | 使用已知尺寸的标定板（玻璃刻尺） | ±0.01mm |
| **镜头畸变校正** | 使用棋盘格标定 | 径向畸变<1% |
| **光源稳定性** | 使用色温计和亮度计监控 | 光源波动<2% |
| **相机-工件距离** | 使用接近传感器监控 | 距离误差<0.5mm |

```python
def calibrate_pixel_to_mm(image, known_object_mm):
    """
    使用已知尺寸的工件进行像素-毫米标定
    
    参数:
        image: 标定图像
        known_object_mm: 工件的实际物理尺寸（毫米）
    
    返回:
        pixel_to_mm: 像素到毫米的转换系数
    """
    # 转换为灰度
    gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    
    # 边缘检测
    edges = cv2.Canny(gray, 50, 150)
    
    # 霍夫圆变换（如果标定物是圆形）
    circles = cv2.HoughCircles(edges, cv2.HOUGH_GRADIENT, dp=1, minDist=100,
                                param1=50, param2=30, minRadius=50, maxRadius=500)
    
    if circles is not None:
        circle = circles[0, 0]
        cx, cy, radius_px = circle
        
        # 计算每个像素对应的毫米数
        # 直径(px) / 直径(mm) = 每毫米像素数
        pixel_to_mm = (radius_px * 2) / known_object_mm
        mm_to_pixel = known_object_mm / (radius_px * 2)
        
        print(f"标定结果：")
        print(f"  测量直径: {radius_px * 2:.2f} 像素")
        print(f"  实际直径: {known_object_mm:.2f} mm")
        print(f"  像素->毫米: {pixel_to_mm:.6f} mm/px")
        print(f"  毫米->像素: {mm_to_pixel:.6f} px/mm")
        
        return pixel_to_mm
    else:
        print("未检测到标定圆，请检查图像")
        return None
```

本章我们系统性地学习了基于图像的工业工件质量检测技术。

在工业视觉质检概述部分，我们了解到机器视觉质检相比人工目检具有速度快、精度高、可追溯等显著优势。工业质检中的缺陷可分为划痕、斑点、裂纹、颜色异常、形状偏差等多种类型，每种缺陷在图像上都有独特的特征。完整的质检 pipeline 包括图像采集、预处理、特征提取、缺陷检测/分类和结果输出五个环节。

在尺寸测量与模板匹配部分，我们学习了如何使用轮廓几何特征进行尺寸测量（面积、周长、拟合圆、等效直径），以及如何使用模板匹配检测形状偏差，并介绍了多尺度模板匹配应对尺寸变化的方法。

## 答案

### 1. 选择题答案

**题目1.1：** **C. 自适应阈值（Adaptive Threshold）**

解析：固定阈值和 Otsu 方法都是基于整幅图像的全局阈值，适用于光照均匀的场景。自适应阈值的特点是为每个像素的邻域计算一个局部阈值，因此可以很好地处理光照渐变不均的图像。场景描述的是"光照不均匀的工业流水线图像"，正是自适应阈值的典型应用。

**题目1.2：** **C. 开运算（先腐蚀后膨胀）**

解析：开运算 = 腐蚀 → 膨胀。先腐蚀可以去除孤立的散点噪声（椒盐噪声中的白点/黑点），同时将主体区域略微缩小；再膨胀将主体区域恢复到接近原始大小。这样既去除了噪声，又基本保留了工件的原始形状。选项 D（闭运算）则相反，会先膨胀填充电小空洞，但也会扩大主体，不适合去噪。

**题目1.3：** **C. TM_CCOEFF_NORMED（归一化相关系数）**

解析：各方法的抗灰度变化能力：TM_SQDIFF 和 TM_SQDIFF_NORMED 对灰度偏移和增益都敏感；TM_CCORR 和 TM_CCORR_NORMED 对灰度增益敏感（乘以常数）；TM_CCOEFF 和 TM_CCOEFF_NORMED 先对图像和模板都做去均值处理，因此对灰度线性变换（$I' = a \cdot I + b$）具有最强的鲁棒性，是工业应用中推荐的方法。

### 2. 填空题答案

**题目2.1：** **大于**；**2-3**（或"2到3"）

解析：在 Canny 算法中：梯度值 > 高阈值 → 强边缘（直接保留）；梯度值 < 低阈值 → 弱边缘（丢弃）；低阈值 ≤ 梯度值 ≤ 高阈值 → 弱边缘，需检查其是否与强边缘连通（8邻域），连通则保留。关于阈值倍数，高阈值通常设为低阈值的 2-3 倍（经验值），具体比例需根据图像内容调整。

**题目2.2：** **4**；**8**；**8**

解析：4连通（4-connectivity）只考虑像素的上下左右4个相邻像素；8连通（8-connectivity）额外包括4个对角方向的邻居。因此，当两个区域仅在对角方向相邻时，使用4连通它们是两个独立区域，使用8连通则会将它们合并。

**题目2.3：** **像素**（或"像素数量"/"像素值"）

解析：相机标定的核心目的就是建立"图像中像素数量"与"物理世界中实际尺寸"之间的对应关系。这个对应关系通常用"像素到毫米的转换系数"（mm/px 或 px/mm）来表示，需要使用已知尺寸的标定物来测量确定。

### 3. 简答题答案

**题目3.1：**

**场景A（检测金属表面油污斑点）：**
- 推荐方法：**阈值分割 + 连通域分析**
- 理由：油污斑点在图像上表现为灰度较周围明显偏暗（或偏亮）的不规则形状区域。使用阈值分割（固定阈值、自适应阈值或底帽变换）可以将油污区域从背景中分离出来，再通过连通域分析计算每个连通域的面积和形状特征，过滤掉过大（不是缺陷）或过小（噪声）的区域。

**场景B（检测玻璃表面细长划痕）：**
- 推荐方法：**边缘检测（Canny）+ 霍夫线

变换 + 形态学处理**
- 理由：划痕的特征是"细长"（长度远大于宽度），在图像上表现为细长的线性边缘。可以通过 Canny 边缘检测提取边缘，再使用霍夫线变换（HoughLinesP）检测直线段，最后根据线段的长度和宽度比筛选出划痕。形态学闭运算可以连接断开的划痕边缘。

**场景C（检测PCB焊点颜色是否正确）：**
- 推荐方法：**HSV颜色空间分析**
- 理由：焊点的颜色偏差反映在HSV颜色空间中，主要是色相（H）不变但饱和度（S）或明度（V）偏离标准值。将图像从RGB转换到HSV后，对焊点区域的H、S、V通道分别计算平均值和标准差，与标准焊点的颜色范围进行比较，超出阈值的判定为颜色异常缺陷。

### 4. 编程题答案

```python
import cv2
import numpy as np
from typing import List, Tuple, Optional

def detect_crack_in_image(image: np.ndarray) -> Tuple[int, List[Tuple[int, int, int, int]]]:
    """
    检测图像中的裂纹缺陷
    
    参数:
        image: 输入图像（彩色或灰度）
    
    返回:
        crack_count: 检测到的裂纹数量
        crack_bboxes: 每个裂纹的边界框列表 [(x, y, w, h), ...]
    
    检测步骤:
        1. 转灰度图
        2. Canny边缘检测
        3. 形态学闭运算连接断开的边缘
        4. 轮廓分析，筛选细长形状（长宽比>10:1）
    """
    # Step 1: 转为灰度图
    if len(image.shape) == 3:
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    else:
        gray = image.copy()
    
    # Step 2: Canny边缘检测（阈值100和200）
    edges = cv2.Canny(gray, threshold1=100, threshold2=200)
    
    # Step 3: 形态学闭运算连接断开的裂纹边缘
    # 使用细长的结构元素（模拟裂纹形状）
    kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (5, 5))
    closed = cv2.morphologyEx(edges, cv2.MORPH_CLOSE, kernel)
    
    # Step 4: 轮廓提取
    contours, _ = cv2.findContours(closed, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
    
    # Step 5: 筛选细长形状（裂纹：长宽比 > 10:1）
    crack_bboxes = []
    aspect_ratio_threshold = 10.0
    
    for contour in contours:
        # 计算最小外接矩形
        rect = cv2.minAreaRect(contour)
        (cx, cy), (w, h), angle = rect
        
        # 计算长宽比（取较长边/较短边）
        longer = max(w, h)
        shorter = min(w, h)
        
        # 跳过太小的区域（面积太小不是裂纹）
        area = cv2.contourArea(contour)
        if area < 50:  # 面积阈值
            continue
        
        # 长宽比 > 10:1 认为是裂纹
        if shorter > 0:
            aspect_ratio = longer / shorter
            if aspect_ratio >= aspect_ratio_threshold:
                bbox = cv2.boundingRect(contour)
                crack_bboxes.append(bbox)
                print(f"裂纹检测: 面积={area:.0f}px², "
                      f"外接矩形={w:.1f}x{h:.1f}px, "
                      f"长宽比={aspect_ratio:.1f}:1, "
                      f"BBOX={bbox}")
    
    crack_count = len(crack_bboxes)
    
    print("\n" + "=" * 50)
    print(f"裂纹检测结果：共检测到 {crack_count} 条裂纹")
    print("=" * 50)
    
    return crack_count, crack_bboxes


def detect_crack_in_image_advanced(image: np.ndarray) -> Tuple[int, List[dict]]:
    """
    增强版裂纹检测（返回更多信息）
    
    返回值包含：裂纹长度、宽度估计值、中心点坐标
    """
    if len(image.shape) == 3:
        gray = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    else:
        gray = image.copy()
    
    # 增强对比度（CLAHE）
    clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
    gray_enhanced = clahe.apply(gray)
    
    # Canny边缘检测（使用较低阈值检测弱边缘）
    edges = cv2.Canny(gray_enhanced, threshold1=50, threshold2=150)
    
    # 形态学处理
    kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (3, 3))
    closed = cv2.morphologyEx(edges, cv2.MORPH_CLOSE, kernel)
    
    # 霍夫线变换（更适合检测细线）
    lines = cv2.HoughLinesP(
        closed,
        rho=1,
        theta=np.pi / 180,
        threshold=30,       # 超过30个像素才认为是直线
        minLineLength=30,   # 最小线段长度
        maxLineGap=5        # 最大线段间隙（连接断开的线）
    )
    
    cracks = []
    
    if lines is not None:
        for line in lines:
            x1, y1, x2, y2 = line[0]
            
            # 计算线段长度
            length = np.sqrt((x2 - x1)**2 + (y1 - y2)**2)
            
            # 计算线段与水平方向的夹角
            angle_rad = np.arctan2(abs(y2 - y1), abs(x2 - x1))
            angle_deg = angle_rad * 180 / np.pi
            
            # 估计线宽（沿线段法线方向采样）
            if abs(x2 - x1) > abs(y2 - y1):
                width_samples = []
                for t in np.linspace(0, 1, 10):
                    x_sample = int(x1 + t * (x2 - x1))
                    y_sample = int(y1 + t * (y2 - y1))
                    if 0 <= y_sample < gray.shape[0] and 0 <= x_sample < gray.shape[1]:
                        profile = gray[y_sample, max(0, x_sample-5):min(gray.shape[1], x_sample+6)]
                        if len(profile) > 0:
                            baseline = (profile[0] + profile[-1]) / 2
                            center_val = profile[len(profile)//2]
                            width_samples.append(abs(int(center_val) - int(baseline)))
                estimated_width = np.mean(width_samples) / 2.0 if width_samples else 1.0
            else:
                width_samples = []
                for t in np.linspace(0, 1, 10):
                    x_sample = int(x1 + t * (x2 - x1))
                    y_sample = int(y1 + t * (y2 - y1))
                    if 0 <= y_sample < gray.shape[0] and 0 <= x_sample < gray.shape[1]:
                        profile = gray[max(0, y_sample-5):min(gray.shape[0], y_sample+6), x_sample]
                        if len(profile) > 0:
                            baseline = (profile[0] + profile[-1]) / 2
                            center_val = profile[len(profile)//2]
                            width_samples.append(abs(int(center_val) - int(baseline)))
                estimated_width = np.mean(width_samples) / 2.0 if width_samples else 1.0
            
            cracks.append({
                "start": (x1, y1),
                "end": (x2, y2),
                "length_px": float(length),
                "width_px": float(estimated_width),
                "angle_deg": float(angle_deg),
                "bbox": (min(x1, x2), min(y1, y2), abs(x2-x1), abs(y2-y1))
            })
            
            print(f"裂纹: 起点{x1,y1}-终点{x2,y2}, "
                  f"长度={length:.1f}px, 宽度≈{estimated_width:.1f}px, "
                  f"角度={angle_deg:.1f}°")
    
    crack_count = len(cracks)
    
    print(f"\n增强版检测：共 {crack_count} 条裂纹")
    
    return crack_count, cracks
```

**答案解析：**

本题考察的是综合运用边缘检测、形态学和轮廓分析进行裂纹检测的能力。

Step 1（灰度化）：裂纹在灰度图上更容易检测，因为裂纹的本质是灰度突变。

Step 2（Canny边缘）：Canny算法能精确定位边缘，同时通过双阈值机制区分强边缘和弱边缘，是检测裂纹的首选。

Step 3（闭运算）：裂纹在图像中往往不是完整的连续线条，可能有断开的地方。闭运算（先膨胀后腐蚀）可以有效地将断开的边缘连接起来，同时不会改变裂纹的粗细特征。

Step 4（轮廓筛选）：通过计算每个轮廓的最小外接矩形的长宽比，可以区分裂纹和其他形状。裂纹的典型特征是"细长"，即长宽比很大（>10:1），这与斑点（接近圆形）和正常边缘（长宽比接近1:1）都有明显区别。
