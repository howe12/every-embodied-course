## 1. 目标跟踪概述

目标跟踪（Object Tracking）是计算机视觉领域的重要研究方向，其目标是在视频序列中连续追踪一个或多个目标的运动轨迹。与目标检测不同，跟踪不仅需要识别目标的位置，还需要为每一帧中的目标分配唯一的身份标识（ID），并在后续帧中保持该标识的一致性。

### 1.1 跟踪任务类型

根据跟踪对象的数量和特点，目标跟踪可以分为以下几类：

**单目标跟踪（Single Object Tracking, SOT）**：
- 只跟踪视频中的一个特定目标
- 目标在第一帧给出，通常使用边界框标注
- 典型算法：Siamese Network、ATOM、DiMP等

**多目标跟踪（Multiple Object Tracking, MOT）**：
- 同时跟踪视频中的多个目标
- 需要处理目标的进入、离开、遮挡等问题
- 典型算法：SORT、DeepSORT、ByteTrack等

**场景跟踪**：
- 关注目标与场景的关系
- 常用于机器人导航和场景理解

### 1.2 跟踪评价指标

多目标跟踪（MOT）常用的评价指标包括：

| 指标 | 名称 | 描述 |
|------|------|------|
| MOTA | Multi-Object Tracking Accuracy | 综合考虑漏检、误检和ID切换的综合指标 |
| MOTP | Multi-Object Tracking Precision | 目标定位精度 |
| IDF1 | ID F1 Score | ID识别正确率 |
| ID Switch | ID Switch | 目标ID切换次数 |
| MT | Mostly Tracked | 长时间跟踪目标的比例 |
| ML | Mostly Lost | 长时间丢失目标的比例 |
| FPS | Frames Per Second | 实时性指标 |

### 1.3 跟踪挑战

目标跟踪面临诸多挑战：

1. **遮挡（Occlusion）**：目标被其他物体部分或完全遮挡
2. **尺度变化（Scale Variation）**：目标在运动过程中距离相机远近变化
3. **运动模糊（Motion Blur）**：快速运动导致图像模糊
4. **外观变化（Appearance Change）**：目标外观随姿态、光照变化
5. **相似目标干扰（Similar Object Interference）**：多个外观相似的目标
6. **实时性要求（Real-time Requirement）**：机器人应用需要实时跟踪

## 2. SORT算法详解

SORT（Simple Online and Realtime Tracking）是由Alex Bewley等人于2017年提出的轻量级多目标跟踪算法。其核心思想是将目标跟踪问题分解为两个独立的子问题：检测和关联。

### 2.1 算法原理

SORT算法采用**检测-跟踪（Detection-based Tracking）**范式，主要包含以下三个组件：

1. **目标检测器**：使用Faster R-CNN、YOLO等检测器获取每帧中的目标
2. **状态估计**：使用卡尔曼滤波器预测目标的运动状态
3. **数据关联**：使用匈牙利算法将检测结果与跟踪轨迹进行匹配

### 2.2 卡尔曼滤波器

SORT使用**卡尔曼滤波器（Kalman Filter）**来预测目标的运动状态。卡尔曼滤波器是一种递归估计器，能够在存在噪声的情况下最优地估计系统状态。

**状态向量定义**：
对于2D跟踪，SORT定义目标状态为：
$$x = [u, v, s, r, \dot{u}, \dot{v}, \dot{s}]^T$$

其中：
- $u, v$：目标边界框中心的坐标
- $s$：边界框面积
- $r$：边界框宽高比
- $\dot{u}, \dot{v}, \dot{s}$：对应变量的速度

**状态预测方程**：
$$\hat{x}_{k|k-1} = F_k \hat{x}_{k-1|k-1}$$

$$\hat{P}_{k|k-1} = F_k P_{k-1|k-1} F_k^T + Q_k$$

其中 $F_k$ 是状态转移矩阵，$Q_k$ 是过程噪声协方差矩阵。

**状态更新方程**：
$$K_k = \hat{P}_{k|k-1} H_k^T (H_k \hat{P}_{k|k-1} H_k^T + R_k)^{-1}$$

$$\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k (z_k - H_k \hat{x}_{k|k-1})$$

$$P_{k|k} = (I - K_k H_k) \hat{P}_{k|k-1}$$

其中 $H_k$ 是观测矩阵，$R_k$ 是观测噪声协方差矩阵，$K_k$ 是卡尔曼增益。

### 2.3 IoU关联

SORT使用**IoU（Intersection over Union）**作为关联度量来匹配检测框和跟踪轨迹。IoU计算公式为：

$$IoU = \frac{Detection \cap Track}{Detection \cup Track}$$

```
┌─────────────────────────────────────────────────────────────────┐
│                        IoU 计算示例                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    Detection Box                   Track Box                   │
│    ┌───────────────┐              ┌───────────────┐            │
│    │               │              │               │            │
│    │    ┌─────┐    │              │    ┌─────┐    │            │
│    │    │     │    │              │    │     │    │            │
│    │    │Intersection│           │    │Intersection│           │
│    │    │     │    │              │    │     │    │            │
│    │    └─────┘    │              │    └─────┘    │            │
│    │               │              │               │            │
│    └───────────────┘              └───────────────┘            │
│                                                                 │
│    Union = Detection + Track - Intersection                     │
│    IoU = Intersection / Union                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**关联匹配步骤**：
1. 计算所有检测框与所有跟踪轨迹的IoU矩阵
2. 使用匈牙利算法在IoU阈值约束下寻找最优匹配
3. 匹配成功的检测框用于更新对应轨迹
4. 未匹配的检测框初始化为新轨迹
5. 未匹配的轨迹标记为丢失（连续丢失超过阈值则删除）

### 2.4 SORT代码实现

以下是SORT算法的核心实现：

```python
import numpy as np
from scipy.optimize import linear_sum_assignment


def iou(bbox1, bbox2):
    """计算两个边界框的IoU
    
    参数:
        bbox1, bbox2: [x1, y1, x2, y2] 格式的边界框
    
    返回:
        IoU值
    """
    x1 = max(bbox1[0], bbox2[0])
    y1 = max(bbox1[1], bbox2[1])
    x2 = min(bbox1[2], bbox2[2])
    y2 = min(bbox1[3], bbox2[3])
    
    intersection = max(0, x2 - x1) * max(0, y2 - y1)
    
    area1 = (bbox1[2] - bbox1[0]) * (bbox1[3] - bbox1[1])
    area2 = (bbox2[2] - bbox2[0]) * (bbox2[3] - bbox2[1])
    
    union = area1 + area2 - intersection
    
    return intersection / union if union > 0 else 0


class KalmanBoxTracker:
    """卡尔曼滤波器用于跟踪边界框"""
    
    count = 0
    
    def __init__(self, bbox):
        """初始化跟踪器
        
        参数:
            bbox: [x1, y1, x2, y2] 格式的边界框
        """
        # 简化的卡尔曼滤波器实现
        self.x = np.zeros(7)  # 状态向量 [x, y, s, r, vx, vy, vs]
        
        # 从边界框初始化状态
        w = bbox[2] - bbox[0]
        h = bbox[3] - bbox[1]
        self.x[0] = bbox[0] + w / 2.0  # cx
        self.x[1] = bbox[1] + h / 2.0  # cy
        self.x[2] = w * h  # area
        self.x[3] = w / h if h > 0 else 1.0  # aspect ratio
        
        self.P = np.eye(7) * 10  # 状态协方差
        self.Q = np.eye(7) * 0.01  # 过程噪声
        self.R = np.eye(4) * 1  # 观测噪声
        
        self.time_since_update = 0
        self.id = KalmanBoxTracker.count
        KalmanBoxTracker.count += 1
        self.history = []
        self.hits = 0
        self.hit_streak = 0
        self.age = 0
        
    def predict(self):
        """预测下一帧状态"""
        # 状态转移 (匀速模型)
        self.x[4] = self.x[0]  # vx = x
        self.x[5] = self.x[1]  # vy = y
        self.x[6] = self.x[2]  # vs = s
        
        # 添加过程噪声
        self.P = self.P + self.Q
        
        self.age += 1
        
        if self.time_since_update > 0:
            self.hit_streak = 0
            
        self.time_since_update += 1
        self.history.append(self.get_state())
        
        return self.history[-1]
    
    def update(self, bbox):
        """更新卡尔曼滤波器
        
        参数:
            bbox: [x1, y1, x2, y2] 格式的边界框
        """
        # 计算观测向量
        w = bbox[2] - bbox[0]
        h = bbox[3] - bbox[1]
        z = np.array([
            bbox[0] + w / 2.0,
            bbox[1] + h / 2.0,
            w * h,
            w / h if h > 0 else 1.0
        ])
        
        # 简化的卡尔曼增益
        H = np.eye(4, 7)
        S = H @ self.P @ H.T + self.R
        K = self.P @ H.T @ np.linalg.inv(S)
        
        # 更新状态
        y = z - H @ self.x  # 观测残差
        self.x = self.x + K @ y
        
        # 更新协方差
        self.P = (np.eye(7) - K @ H) @ self.P
        
        self.time_since_update = 0
        self.history = []
        self.hits += 1
        self.hit_streak += 1
        
    def get_state(self):
        """获取当前边界框 [x1, y1, x2, y2]"""
        w = np.sqrt(self.x[2] * self.x[3])
        h = self.x[2] / w if w > 0 else 0
        return np.array([
            self.x[0] - w / 2.0,
            self.x[1] - h / 2.0,
            self.x[0] + w / 2.0,
            self.x[1] + h / 2.0
        ])


def associate_detections_to_tracks(detections, trackers, iou_threshold=0.3):
    """将检测框关联到跟踪轨迹
    
    参数:
        detections: 检测框列表 [[x1,y1,x2,y2], ...]
        trackers: 跟踪器列表
        iou_threshold: IoU阈值
    
    返回:
        matches: 匹配的列表 [(detection_idx, tracker_idx), ...]
        unmatched_detections: 未匹配的检测框索引
        unmatched_trackers: 未匹配的跟踪器索引
    """
    if len(trackers) == 0:
        return [], list(range(len(detections))), []
    
    if len(detections) == 0:
        return [], [], list(range(len(trackers)))
    
    # 计算IoU矩阵
    iou_matrix = np.zeros((len(detections), len(trackers)), dtype=np.float32)
    for d, det in enumerate(detections):
        for t, trk in enumerate(trackers):
            iou_matrix[d, t] = iou(det, trk)
    
    # 使用匈牙利算法找最优匹配
    row_ind, col_ind = linear_sum_assignment(-iou_matrix)
    
    matches = []
    unmatched_detections = []
    unmatched_trackers = []
    
    for d in range(len(detections)):
        if d not in row_ind:
            unmatched_detections.append(d)
            
    for t in range(len(trackers)):
        if t not in col_ind:
            unmatched_trackers.append(t)
    
    for d, t in zip(row_ind, col_ind):
        if iou_matrix[d, t] < iou_threshold:
            unmatched_detections.append(d)
            unmatched_trackers.append(t)
        else:
            matches.append([d, t])
    
    return matches, unmatched_detections, unmatched_trackers


class Sort:
    """SORT多目标跟踪器"""
    
    def __init__(self, max_age=30, min_hits=3, iou_threshold=0.3):
        """初始化SORT
        
        参数:
            max_age: 最大丢失帧数
            min_hits: 最小命中次数
            iou_threshold: IoU匹配阈值
        """
        self.max_age = max_age
        self.min_hits = min_hits
        self.iou_threshold = iou_threshold
        self.trackers = []
        self.frame_count = 0
        
    def update(self, dets):
        """更新跟踪器
        
        参数:
            dets: 检测框数组，格式 [[x1,y1,x2,y2,score], ...] 或 [[x1,y1,x2,y2], ...]
        
        返回:
            返回格式为 [[x1,y1,x2,y2,id,score], ...] 的结果
        """
        self.frame_count += 1
        
        # 获取检测框
        if dets.shape[1] == 5:
            dets_only = dets[:, :4]
            scores = dets[:, 4]
        else:
            dets_only = dets
            scores = np.ones(len(dets))
        
        # 预测所有跟踪器的状态
        for tracker in self.trackers:
            tracker.predict()
        
        # 关联检测框和跟踪轨迹
        matched, unmatched_dets, unmatched_trks = associate_detections_to_tracks(
            dets_only, 
            [t.get_state() for t in self.trackers],
            self.iou_threshold
        )
        
        # 更新匹配的跟踪器
        for det_idx, trk_idx in matched:
            self.trackers[trk_idx].update(dets_only[det_idx])
        
        # 为未匹配的检测框创建新跟踪器
        for i in unmatched_dets:
            trk = KalmanBoxTracker(dets_only[i])
            self.trackers.append(trk)
        
        # 获取有效结果
        results = []
        i = len(self.trackers)
        for tracker in reversed(self.trackers):
            i -= 1
            if tracker.time_since_update < self.max_age:
                if tracker.hit_streak >= self.min_hits or self.frame_count <= self.min_hits:
                    d = tracker.get_state()
                    results.append([d[0], d[1], d[2], d[3], tracker.id, scores[i] if i < len(scores) else 1.0])
            
            if tracker.time_since_update > self.max_age:
                self.trackers.pop(i)
        
        if len(results) > 0:
            return np.array(results)
        return np.array([])
```

### 2.5 SORT的局限性

SORT算法虽然简单高效，但也存在明显的局限性：

1. **ID切换频繁**：当目标被遮挡或重叠时，容易发生ID切换
2. **仅依赖外观**：SORT主要依赖边界框的几何信息，没有利用目标的外观特征
3. **对遮挡敏感**：遮挡会导致目标丢失后难以找回

这些问题催生了DeepSORT算法的诞生。

---

## 3. DeepSORT算法详解

DeepSORT是SORT算法的改进版本，由Nicolai Wojke等人于2017年提出。核心创新是引入了**外观特征（Appearance Feature）**来增强数据关联的鲁棒性。

### 3.1 核心改进

DeepSORT相对于SORT的主要改进：

1. **外观特征提取**：使用卷积神经网络提取目标的视觉特征
2. **级联匹配**：先进行基于外观特征的匹配，再进行IoU匹配
3. **长方体匹配**：改进了匹配策略，减少ID切换

```
┌─────────────────────────────────────────────────────────────────┐
│                     DeepSORT vs SORT                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  SORT                         DeepSORT                         │
│  ┌─────────┐                  ┌─────────────┐                  │
│  │  检测   │                  │   检测      │                  │
│  └────┬────┘                  └──────┬──────┘                  │
│       │                              │                          │
│       ▼                              ▼                          │
│  ┌─────────┐                  ┌─────────────┐                  │
│  │卡尔曼预测│                  │  特征提取    │                  │
│  │ (IoU)  │                  │  (CNN)      │                  │
│  └────┬────┘                  └──────┬──────┘                  │
│       │                              │                          │
│       ▼                              ▼                          │
│  ┌─────────┐                  ┌─────────────┐                  │
│  │匈牙利匹配│                  │  级联匹配    │                  │
│  │ (IoU)  │                  │  (外观+IoU) │                  │
│  └─────────┘                  └─────────────┘                  │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 外观特征提取

DeepSORT使用一个轻量级的CNN网络（通常是ResNet或MobileNet）来提取目标的外观特征。这些特征具有以下特点：

- **判别性**：同一个目标的特征向量在不同帧中保持相似
- **鲁棒性**：对姿态变化、光照变化具有一定的鲁棒性
- **高效性**：网络轻量，适合实时应用

**余弦相似度计算**：

给定两个特征向量 $f_1$ 和 $f_2$，它们的余弦相似度定义为：

$$\cos(\theta) = \frac{f_1 \cdot f_2}{\|f_1\| \|f_2\|} = \frac{\sum_{i=1}^{n} f_{1i} f_{2i}}{\sqrt{\sum_{i=1}^{n} f_{1i}^2} \sqrt{\sum_{i=1}^{n} f_{2i}^2}}$$

### 3.3 级联匹配

DeepSORT采用**级联匹配**策略来提高匹配精度：

1. **第一级（外观匹配）**：计算检测框特征与跟踪器特征的余弦相似度，使用匈牙利算法进行匹配
2. **第二级（IoU匹配）**：对第一级未匹配的轨迹，使用IoU进行二次匹配

```
┌─────────────────────────────────────────────────────────────────┐
│                     级联匹配流程                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  输入: 检测框集合D, 跟踪轨迹集合T                                │
│                                                                 │
│  ┌───────────────────────────────────────────────────────────┐ │

### 3.4 DeepSORT代码实现

以下是DeepSORT算法的核心实现：

```python
import numpy as np
import torch
import torch.nn as nn
import cv2
from scipy.optimize import linear_sum_assignment


class FeatureExtractor(nn.Module):
    """外观特征提取网络"""
    
    def __init__(self, num_classes=512):
        super().__init__()
        # 使用轻量级网络作为backbone
        self.backbone = nn.Sequential(
            nn.Conv2d(3, 32, 3, 2, 1),
            nn.BatchNorm2d(32),
            nn.ReLU6(inplace=True),
            nn.MaxPool2d(2),
            
            nn.Conv2d(32, 64, 3, 2, 1),
            nn.BatchNorm2d(64),
            nn.ReLU6(inplace=True),
            nn.MaxPool2d(2),
            
            nn.Conv2d(64, 128, 3, 2, 1),
            nn.BatchNorm2d(128),
            nn.ReLU6(inplace=True),
            nn.MaxPool2d(2),
            
            nn.Conv2d(128, 256, 3, 2, 1),
            nn.BatchNorm2d(256),
            nn.ReLU6(inplace=True),
            nn.AdaptiveAvgPool2d((1, 1))
        )
        
        self.embedding = nn.Linear(256, num_classes)
        
    def forward(self, x):
        x = self.backbone(x)
        x = x.view(x.size(0), -1)
        x = self.embedding(x)
        return nn.functional.normalize(x, p=2, dim=1)


def cosine_distance(feat1, feat2):
    """计算余弦距离矩阵
    
    参数:
        feat1, feat2: 特征向量 [N, D] 或 [D,]
    
    返回:
        距离矩阵
    """
    if feat1.ndim == 1:
        feat1 = feat1.unsqueeze(0)
    if feat2.ndim == 1:
        feat2 = feat2.unsqueeze(0)
    
    # 归一化
    feat1 = feat1 / feat1.norm(dim=1, keepdim=True)
    feat2 = feat2 / feat2.norm(dim=1, keepdim=True)
    
    # 余弦相似度
    similarity = torch.mm(feat1, feat2.t())
    
    # 转换为距离
    return 1 - similarity


class DeepSortTracker:
    """DeepSORT多目标跟踪器"""
    
    def __init__(self, 
                 max_age=30, 
                 n_init=3, 
                 max_cosine_distance=0.2,
                 nn_budget=100,
                 iou_threshold=0.7):
        self.max_age = max_age
        self.n_init = n_init
        self.iou_threshold = iou_threshold
        self.min_hits = n_init
        
        # 特征提取器
        self.extractor = FeatureExtractor()
        self.extractor.eval()
        
        self.trackers = []
        self.frame_count = 0
        
    def update(self, detections, ori_img):
        """更新跟踪器
        
        参数:
            detections: 检测框 [[x1,y1,x2,y2,score], ...]
            ori_img: 原始图像 (用于特征提取)
        
        返回:
            结果数组 [[x1,y1,x2,y2,id,score], ...]
        """
        self.frame_count += 1
        
        # 提取检测框特征
        if len(detections) > 0:
            features = self._extract_features(detections, ori_img)
        else:
            features = np.array([])
        
        # 预测
        for tracker in self.trackers:
            tracker.predict()
        
        # 级联匹配
        matches, unmatched_tracks, unmatched_detections = \
            self._cascade_matching(detections, features)
        
        # 更新匹配的跟踪器
        for track_idx, detection_idx in matches:
            self.trackers[track_idx].update(
                detections[detection_idx],
                features[detection_idx]
            )
        
        # IoU匹配（对未匹配的轨迹）
        if len(unmatched_tracks) > 0:
            remaining_dets = detections[unmatched_detections]
            remaining_feats = features[unmatched_detections] if len(unmatched_detections) > 0 else np.array([])
            remaining_trackers = [self.trackers[i] for i in unmatched_tracks]
            
            iou_matches, iou_unmatched_tracks, iou_unmatched_detections = \
                self._iou_matching(remaining_dets, remaining_feats, remaining_trackers)
            
            for track_idx, detection_idx in iou_matches:
                idx = unmatched_tracks[track_idx]
                det_idx = unmatched_detections[detection_idx]
                self.trackers[idx].update(detections[det_idx], features[det_idx])
                
            unmatched_detections = [unmatched_detections[i] for i in iou_unmatched_detections]
        
        # 创建新跟踪器
        for i in unmatched_detections:
            trk = DeepSORTTracker(self.max_age, self.n_init, self.iou_threshold)
            trk.update(detections[i], features[i])
            self.trackers.append(trk)
        
        # 获取有效结果
        results = []
        for tracker in self.trackers:
            if tracker.is_confirmed() and tracker.time_since_update < self.max_age:
                results.append(tracker.to_tlwh_with_score())
        
        return np.array(results) if len(results) > 0 else np.array([])
    
    def _extract_features(self, detections, ori_img):
        """提取检测框的外观特征"""
        features = []
        img_h, img_w = ori_img.shape[:2]
        
        for det in detections:
            x1, y1, x2, y2 = det[:4].astype(int)
            x1, y1 = max(0, x1), max(0, y1)
            x2, y2 = min(img_w, x2), min(img_h, y2)
            
            if x2 > x1 and y2 > y1:
                crop = ori_img[y1:y2, x1:x2]
                crop = cv2.resize(crop, (64, 128))
                crop = crop.astype(np.float32) / 255.0
                crop = torch.from_numpy(crop).permute(2, 0, 1).unsqueeze(0)
                
                with torch.no_grad():
                    feat = self.extractor(crop).numpy()
                features.append(feat[0])
            else:
                features.append(np.zeros(512))
        
        return np.array(features)
    
    def _cascade_matching(self, detections, features):
        """级联匹配"""
        if len(detections) == 0:
            return [], list(range(len(self.trackers))), []
        
        matches = []
        unmatched_tracks = list(range(len(self.trackers)))
        unmatched_detections = list(range(len(detections)))
        
        # 按轨迹年龄进行级联匹配
        for age in range(1, self.max_age + 1):
            if len(unmatched_tracks) == 0:
                break
                
            tracks_at_age = [i for i in unmatched_tracks 
                           if self.trackers[i].time_since_update == age]
            
            if len(tracks_at_age) == 0:
                continue
                
            # 计算外观距离矩阵
            det_features = features[unmatched_detections]
            track_features = np.array([self.trackers[i].feature 
                                      for i in tracks_at_age])
            
            cost_matrix = cosine_distance(
                torch.from_numpy(track_features),
                torch.from_numpy(det_features)
            ).numpy()
            
            # 匈牙利匹配
            row_ind, col_ind = linear_sum_assignment(cost_matrix)
            
            # 更新匹配结果
            new_matches = []
            for r, c in zip(row_ind, col_ind):
                if cost_matrix[r, c] < 0.3:  # 外观相似度阈值
                    new_matches.append((tracks_at_age[r], unmatched_detections[c]))
            
            matches.extend(new_matches)
            
            matched_tracks = set([m[0] for m in new_matches])
            matched_dets = set([m[1] for m in new_matches])
            
            unmatched_tracks = [t for t in unmatched_tracks 
                               if t not in matched_tracks]
            unmatched_detections = [d for d in unmatched_detections 
                                   if d not in matched_dets]
        
        return matches, unmatched_tracks, unmatched_detections
    
    def _iou_matching(self, detections, features, trackers):
        """IoU匹配"""
        if len(detections) == 0 or len(trackers) == 0:
            return [], list(range(len(trackers))), list(range(len(detections)))
        
        # 计算IoU矩阵
        iou_matrix = np.zeros((len(detections), len(trackers)))
        for d, det in enumerate(detections):
            for t, tracker in enumerate(trackers):
                iou_matrix[d, t] = iou(det[:4], tracker.to_tlwh())
        
        # 匈牙利匹配
        row_ind, col_ind = linear_sum_assignment(-iou_matrix)
        
        matches = []
        unmatched_dets = []
        unmatched_trks = []
        
        matched_dets_set = set()
        matched_trks_set = set()
        
        for r, c in zip(row_ind, col_ind):
            if iou_matrix[r, c] >= self.iou_threshold:
                matches.append((c, r))
                matched_dets_set.add(r)
                matched_trks_set.add(c)
        
        unmatched_dets = [i for i in range(len(detections)) if i not in matched_dets_set]
        unmatched_trks = [i for i in range(len(trackers)) if i not in matched_trks_set]
        
        return matches, unmatched_trks, unmatched_dets

        """更新跟踪器
        
        参数:
            bbox: [x1, y1, x2, y2] 边界框
            feature: 外观特征向量
        """
        self.kf.update(bbox)
        
        # 保存特征
        self.features.append(feature)
        if len(self.features) > 100:
            self.features.pop(0)
        self.feature = np.mean(self.features, axis=0)
        
        self.time_since_update = 0
        self.hits += 1
        self.hit_streak += 1
        
        # 更新状态
        if self.state == 'tentative' and self.hits >= self.n_init:
            self.state = 'confirmed'
            
    def is_confirmed(self):
        """检查是否已确认"""
        return self.state == 'confirmed'
    
    def to_tlwh(self):
        """获取边界框 [x, y, w, h]"""
        ret = self.kf.get_state()
        x1, y1, x2, y2 = ret
        return np.array([x1, y1, x2 - x1, y2 - y1])
    
    def to_tlwh_with_score(self):
        """获取边界框和ID [x1, y1, x2, y2, id, score]"""
        ret = self.kf.get_state()
        x1, y1, x2, y2 = ret
        return np.array([x1, y1, x2, y2, self.id, 1.0])

### 3.5 DeepSORT的优势

相对于SORT，DeepSORT具有以下优势：

1. **减少ID切换**：通过外观特征的匹配，即使目标暂时遮挡，也能保持ID一致性
2. **更鲁棒的跟踪**：级联匹配策略提高了在复杂场景下的跟踪稳定性
3. **长轨迹保持**：特征库机制允许目标短暂消失后重新找回
