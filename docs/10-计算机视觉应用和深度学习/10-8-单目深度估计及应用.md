## 1. 单目深度估计概述

### 1.1 什么是单目深度估计

单目深度估计（Monocular Depth Estimation）是指从单张 RGB 图像中推断出每个像素对应的深度值（Distance），即该像素到相机的实际距离。这一过程本质上是将 2D 图像信息"升级"为 3D 空间信息。

设输入图像为 $I \in \mathbb{R}^{H \times W \times 3}$，单目深度估计的输出为深度图 $D \in \mathbb{R}^{H \times W}$，其中每个位置的值表示该像素点到相机平面的垂直距离。

### 1.2 为什么需要单目深度估计

传统的深度获取方法需要额外硬件：
- **立体视觉（Stereo Vision）**：需要两个摄像头和精确的标定
- **结构光（Structured Light）**：需要投影仪和特定的光照条件
- **ToF（Time of Flight）**：需要专门的 ToF 传感器

单目深度估计仅需普通单目摄像头即可工作：
- **成本低廉**：使用普通 RGB 摄像头，无需额外硬件
- **部署灵活**：适用于任何带有单目相机的平台
- **应用广泛**：手机、机器人、无人机、汽车等

### 1.3 单目深度估计的发展历程

单目深度估计技术的发展经历了从传统方法到深度学习的重大变革：

**传统方法时代**：
- 运动恢复结构（SfM）：通过多帧图像的相对运动恢复深度
- 深度学习单帧估计：2014年，Eigen 等人首次使用深度学习从单张图像估计深度

**深度学习时代**：
- Eigen 等人（2014）：首次使用 CNN 从单张图像估计深度
- DispNet（2016）： encoder-decoder 结构，使用立体匹配损失训练
- DPT（2021）：基于 Transformer 的单目深度估计，刷新多项基准
- MiDaS（2022）：多数据集联合训练，实现泛化性能极强的通用深度估计

## 2. 相对深度与绝对深度

### 2.1 基本概念

单目深度估计根据输出深度值的性质可以分为两类：**相对深度（Relative Depth）** 和 **绝对深度（Absolute Depth）**。

**相对深度**：只表示像素之间的深度关系（前后远近），不给出具体的物理距离值。相对深度图中，较暗的区域表示距离较近，较亮的区域表示距离较远，但具体的米数未知。

**绝对深度**：给出像素到相机的实际物理距离（通常以米为单位）。绝对深度图可以直接用于三维重建、导航避障等需要真实尺度信息的应用。

### 2.2 区别对比

| 特性 | 相对深度 | 绝对深度 |
|------|---------|---------|
| 数值含义 | 像素间前后关系 | 到相机的实际距离（米） |
| 物理单位 | 无 | 米（m） |
| 获取方式 | 单目深度估计直接输出 | 需要尺度恢复或传感器融合 |
| 适用场景 | 虚拟场景、3D效果 | 机器人导航、三维重建 |
| 精度评估 | 仅适合相对误差 | 可计算绝对误差 |

### 2.3 相对深度估计

相对深度估计网络学习的是像素之间的深度顺序关系。训练时使用成对的像素点 $(p_i, p_j)$，标签为：
$$r_{ij} = \begin{cases} +1 & \text{若 } d_i < d_j \text{（i 更近）}\\ -1 & \text{若 } d_i > d_j \text{（i 更远）} \end{cases}$$

相对深度损失函数（Relative Depth Loss）：
$$L_{rel} = \sum_{i,j} \max(0, -r_{ij} \cdot (s(d_i) - s(d_j)) + \epsilon)$$

其中 $s(\cdot)$ 是深度值的 sigmoid 变换，$\epsilon$ 是边际（margin）。

### 2.4 绝对深度估计

绝对深度估计直接预测每个像素的深度值（以米为单位）。绝对深度损失通常使用以下几种：

**L1 损失**：
$$L_1 = \sum_{p} |d_p - \hat{d}_p|$$

**L2 损失（均方误差）**：
$$L_2 = \sum_{p} (d_p - \hat{d}_p)^2$$

**Scale-Invariant 损失**（Eigen 等人提出）：
$$L_{si} = \frac{1}{n} \sum_{p} (\log d_p - \log \hat{d}_p)^2 - \frac{1}{n^2} \left(\sum_{p} (\log d_p - \log \hat{d}_p)\right)^2$$

这一损失函数具有尺度不变性，允许网络学习相对关系而无需考虑绝对尺度。

### 2.5 尺度恢复

单目深度估计网络通常只能学习到深度尺度不变的性质，输出的深度值需要经过尺度恢复才能得到绝对深度。

**方法一：在线尺度恢复**
通过融合其他传感器信息（如 IMU、GPS、激光雷达）恢复绝对尺度。

**方法二：后处理尺度恢复**
已知场景中某个物体的实际尺寸，通过该物体在图像中的大小反推尺度。

**方法三：学习方法恢复尺度**
使用额外的网络或损失函数端到端学习绝对尺度。

## 3. MiDaS 详解

### 3.1 MiDaS 概述

MiDaS（Monocular Depth from Any Scale）是 Intel Labs 推出的通用单目深度估计模型。MiDaS 的核心创新在于**多数据集联合训练**，它在多个不同深度的数据集上联合训练，学习到跨数据集的通用深度特征，从而具备极强的泛化能力。

MiDaS 经历了多个版本的迭代：
- **MiDaS v1**（2019）：首次实现多数据集联合训练
- **MiDaS v2**（2020）：改进网络结构，支持更多数据集
- **MiDaS v3**（2022）：引入 Transformer 架构，精度大幅提升

### 3.2 MiDaS 支持的数据集

MiDaS 联合训练使用的数据集包括：

| 数据集 | 场景类型 | 深度获取方式 |
|--------|---------|------------|
| KITTI | 自动驾驶街景 | 激光雷达 |
| NYU Depth V2 | 室内场景 | Kinect 深度相机 |
| SUN RGB-D | 室内场景 | RealSense 等 |
| DIW | 互联网图像 | 人工标注 |
| WSVD | YouTube 视频 | 运动恢复结构 |
| 3D Movies | 电影片段 | 立体匹配 |

### 3.3 MiDaS 网络结构

MiDaS 使用 DPT（Dense Prediction Transformer）作为骨干网络。DPT 将 Transformer 引入密集预测任务，相比传统的 CNN encoder 具有更大的感受野和更好的全局推理能力。

```
输入图像 ──▶ Patch Embedding ──▶ Transformer Encoder ──▶ Reassemble ──▶ 深度头 ──▶ 深度图
```

**Patch Embedding**：将图像分割为固定大小的 patches（通常 16×16），每个 patch 作为一个 token 输入 Transformer。

**Transformer Encoder**：使用多层自注意力机制提取特征，包含多个阶段（MIU 层）。

**Reassemble**：将 Transformer 的 token 输出重新组装为密集特征图。

**深度头**：将特征图上采样到原始分辨率，输出深度值。

### 3.4 MiDaS 的深度归一化

MiDaS 输出的深度值经过归一化处理，深度图的数值范围大致在 0 到某个最大值之间（通常对应 10-100 米的范围）。使用前需要根据场景特点进行反归一化：

```python
# MiDaS 深度值反归一化
depth = depth * depth_max  # depth_max 通常设为 100.0
```

## 4. DPT 详解

### 4.1 DPT 架构

DPT（Dense Prediction Transformer）是 2021 年提出的基于 Transformer 的密集预测网络。DPT 将 ViT（Vision Transformer）改编为密集预测任务，在深度估计和语义分割任务上都取得了 SOTA 效果。

**DPT-DenseNet**：使用 DenseNet-161 作为编码器
**DPT-ViT**：使用 ViT 家族模型作为编码器（DPT-Large, DPT-Hybrid）

DPT-Hybrid 是 MiDaS v3 默认使用的结构，结合了 CNN 的局部特征和 Transformer 的全局推理能力。

### 4.2 DPT 特征融合

DPT 使用多尺度特征融合策略，从 Transformer 的多个层级提取特征：

```
Stage 1 (浅层) ──┐
                 ├──▶ Fusion Block ──▶ 预测头 ──▶ 输出
Stage 4 (深层) ──┘
```

浅层特征保留更多细节信息（边缘、纹理），深层特征包含更丰富的语义信息。DPT 通过学习的方式自适应融合多尺度特征。

### 4.3 DPT 与传统方法的对比

| 特性 | DPT | CNN 方法（Eigen） | 传统方法（SfM） |
|------|-----|-----------------|----------------|
| 感受野 | 全局 | 局部 | 取决于图像序列 |
| 推理速度 | 较慢 | 快 | 慢（需要多帧） |
| 泛化能力 | 强 | 较弱 | 较弱 |
| 绝对尺度 | 需要恢复 | 需要恢复 | 自动获取 |
| 单图像推理 | 支持 | 支持 | 不支持 |

```python
import torch
import cv2
import numpy as np
import matplotlib.pyplot as plt


def load_midas_model(model_type="DPT_Large"):
    """
    加载 MiDaS 模型
    
    参数:
        model_type: 模型类型，DPT_Large/DPT_Hybrid/MiDaS
                   - DPT_Large: 最高精度，较大模型
                   - DPT_Hybrid: 平衡精度和速度
                   - MiDaS: 较小模型，适合边缘设备
    
    返回:
        model: 加载好的模型
        transform: 图像预处理函数
    """
    # 加载预训练模型
    midas = torch.hub.load("intel-isl/MiDaS", model_type)
    
    # 移动到设备
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    midas.to(device)
    midas.eval()
    
    # 获取 transforms
    midas_transforms = torch.hub.load("intel-isl/MiDaS", "transforms")
    
    if model_type == "DPT_Large" or model_type == "DPT_Hybrid":
        transform = midas_transforms.dpt_transform
    else:
        transform = midas_transforms.small_transform
    
    return midas, transform, device


def estimate_depth(model, transform, image, device):
    """
    估计单张图像的深度
    
    参数:
        model: MiDaS 模型
        transform: 预处理函数
        image: BGR 图像
        device: 计算设备
    
    返回:
        depth: 深度图 (H, W)，已反归一化到米为单位
    """
    # BGR 转 RGB
    img_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
    
    # 预处理
    input_batch = transform(img_rgb).to(device)
    
    # 推理
    with torch.no_grad():
        prediction = model(input_batch)
    
    # 处理输出
    prediction = prediction.squeeze().cpu().numpy()
    
    # 反归一化到米
    depth = cv2.resize(prediction, (image.shape[1], image.shape[0]))
    
    # 缩放到合理范围
    depth_min = depth.min()
    depth_max = depth.max()
    
    if depth_max - depth_min > 0:
        depth = (depth - depth_min) / (depth_max - depth_min)
    
    # 最终缩放，假设场景最大深度为 100 米
    depth = depth * 100.0
    
    return depth


def depth_to_colormap(depth, method='numpy'):
    """
    将深度图转换为伪彩色图用于可视化
    
    参数:
        depth: 深度图
        method: 可视化方法
    
    返回:
        colored: 伪彩色深度图
    """
    # 归一化到 0-1
    depth_norm = (depth - depth.min()) / (depth.max() - depth.min() + 1e-8)
    
    # 应用颜色映射
    colored = cv2.applyColorMap((depth_norm * 255).astype(np.uint8), cv2.COLORMAP_INFERNO)
    
    return colored


if __name__ == "__main__":
    # 加载模型
    print("加载 MiDaS 模型...")
    model, transform, device = load_midas_model("DPT_Large")
    print(f"使用设备: {device}")

    # 估计深度
    depth = estimate_depth(model, transform, image, device)
    
    # 可视化
    colored = depth_to_colormap(depth)
    
    cv2.imwrite("depth_result.png", colored)
    print("深度估计完成，结果已保存")
    
    # 打印深度统计
    print(f"深度范围: {depth.min():.2f}m - {depth.max():.2f}m")
    print(f"平均深度: {depth.mean():.2f}m")
```

### 5.3 深度估计类封装

```python
import torch
import cv2
import numpy as np
from typing import Tuple, Optional


class DepthEstimator:
    """单目深度估计器封装类"""
    
    def __init__(
        self,
        model_type: str = "DPT_Large",
        model_path: Optional[str] = None,
        device: Optional[str] = None,
        max_depth: float = 100.0
    ):
        """
        初始化深度估计器
        
        参数:
            model_type: 模型类型，DPT_Large/DPT_Hybrid/MiDaS
            model_path: 本地模型路径（可选）
            device: 计算设备
            max_depth: 假设的最大深度（米），用于归一化
        """
        self.device = torch.device(
            device or ("cuda" if torch.cuda.is_available() else "cpu")
        )
        self.max_depth = max_depth
        
        # 加载模型
        if model_path:
            self.model = torch.load(model_path, map_location=self.device)
        else:
            self.model = torch.hub.load("intel-isl/MiDaS", model_type)
        
        self.model.to(self.device)
        self.model.eval()
        
        # 获取 transforms
        midas_transforms = torch.hub.load("intel-isl/MiDaS", "transforms")
        
        if model_type == "DPT_Large" or model_type == "DPT_Hybrid":
            self.transform = midas_transforms.dpt_transform
        else:
            self.transform = midas_transforms.small_transform
        
        self.input_size = (384, 384) if model_type == "DPT_Large" else (256, 256)
        
        print(f"深度估计器初始化完成，使用设备: {self.device}")
    
    def estimate(self, image: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """
        估计图像深度
        
        参数:
            image: BGR 图像，形状 (H, W, 3)
        
        返回:
            depth: 深度图 (H, W)，单位为米
            depth_colored: 伪彩色深度图用于可视化
        """
        # BGR 转 RGB
        img_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
        
        # 预处理
        input_batch = self.transform(img_rgb).to(self.device)
        
        # 推理
        with torch.no_grad():
            prediction = self.model(input_batch)
        
        # 处理输出
        prediction = prediction.squeeze().cpu().numpy()
        
        # 缩放到原始图像尺寸
        original_h, original_w = image.shape[:2]
        depth = cv2.resize(
            prediction, 
            (original_w, original_h), 
            interpolation=cv2.INTER_CUBIC
        )
        
        # 归一化到 0-max_depth 范围
        depth = (depth - depth.min()) / (depth.max() - depth.min() + 1e-8)
        depth = depth * self.max_depth
        
        # 生成伪彩色图
        depth_norm = (depth / self.max_depth * 255).astype(np.uint8)
        depth_colored = cv2.applyColorMap(depth_norm, cv2.COLORMAP_INFERNO)
        
        return depth, depth_colored
    
    def estimate_batch(self, images: list) -> list:
        """
        批量估计图像深度
        
        参数:
            images: BGR 图像列表
        
        返回:
            depths: 深度图列表
        """
        results = []
        
        for image in images:
            depth, _ = self.estimate(image)
            results.append(depth)
        
        return results
    
    def get_point_cloud(
        self, 
        image: np.ndarray, 
        depth: np.ndarray, 
        intrinsic: np.ndarray
    ) -> np.ndarray:
        """
        从深度图生成点云
        
        参数:
            image: RGB 图像
            depth: 深度图 (H, W)，单位为米
            intrinsic: 相机内参矩阵 (3, 3)
        
        返回:
            points: 点云 (N, 3)，N 为有效点数
        """
        h, w = depth.shape
        
        # 生成像素坐标网格
        u_coords, v_coords = np.meshgrid(np.arange(w), np.arange(h))
        
        # 展平
        u_coords = u_coords.flatten()
        v_coords = v_coords.flatten()
        z = depth.flatten()
        
        # 过滤无效深度
        valid_mask = z > 0
        u_coords = u_coords[valid_mask]
        v_coords = v_coords[valid_mask]
        z = z[valid_mask]
        
        # 反投影到 3D
        fx, fy = intrinsic[0, 0], intrinsic[1, 1]
        cx, cy = intrinsic[0, 2], intrinsic[1, 2]
        
        x = (u_coords - cx) * z / fx
        y = (v_coords - cy) * z / fy
        
        # 合并坐标
        points = np.stack([x, y, z], axis=1)
        
        return points
    
    def get_distance_at_point(self, depth: np.ndarray, x: int, y: int) -> float:
        """
        获取指定像素点的深度值
        
        参数:
            depth: 深度图
            x, y: 像素坐标
        
        返回:
            distance: 深度值（米）
        """
        h, w = depth.shape
        x = max(0, min(x, w - 1))
        y = max(0, min(y, h - 1))
        
        return float(depth[y, x])
    
    def get_depth_stats(self, depth: np.ndarray) -> dict:
        """
        获取深度图统计信息
        
        参数:
            depth: 深度图
        
        返回:
            stats: 统计信息字典
        """
        valid_depth = depth[depth > 0]
        
        if len(valid_depth) == 0:
            return {"error": "No valid depth values"}
        
        return {
            "min": float(valid_depth.min()),
            "max": float(valid_depth.max()),
            "mean": float(valid_depth.mean()),
            "median": float(np.median(valid_depth)),
            "std": float(valid_depth.std()),
            "valid_pixels": len(valid_depth),
            "total_pixels": depth.size,
            "coverage": len(valid_depth) / depth.size
        }


if __name__ == "__main__":
    # 初始化估计器
    estimator = DepthEstimator(model_type="DPT_Large")
    
    # 读取图像
    image = cv2.imread("test.jpg")
    
    # 估计深度
    depth, colored = estimator.estimate(image)
    
    # 打印统计信息
    stats = estimator.get_depth_stats(depth)
    print("深度统计信息:")
    for key, value in stats.items():
        print(f"  {key}: {value:.2f}" if isinstance(value, float) else f"  {key}: {value}")
    
    # 生成点云示例
    intrinsic = np.array([
        [fx, 0, cx],
        [0, fy, cy],
        [0, 0, 1]
    ])  # 需要根据实际相机内参填写
```

## 6. ROS2 集成与机器人应用

### 6.1 深度估计 ROS2 节点架构

```
┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
│   相机驱动节点    │ ────▶│  深度估计节点    │ ────▶│   导航控制节点   │
│                 │      │                 │      │                 │
└─────────────────┘      └─────────────────┘      └─────────────────┘
                               │
                               ▼
                        ┌─────────────────┐
```

### 6.2 深度估计节点实现

```python
# depth_estimation_node.py - ROS2 单目深度估计节点
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image, CameraInfo
from geometry_msgs.msg import PointCloud2, PointField
from cv_bridge import CvBridge
import numpy as np
import torch
import cv2


class DepthEstimationNode(Node):
    """ROS2 单目深度估计节点"""
    
    def __init__(self):
        super().__init__('depth_estimation_node')
        
        self.bridge = CvBridge()
        
        # 初始化设备
        self.device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
        self.get_logger().info(f"使用设备: {self.device}")
        
        # 加载模型
        self._load_model()
        
        # 相机内参
        self.camera_info_received = False
        self.intrinsic = None
        
        # 订阅相机话题
        self.image_sub = self.create_subscription(
            Image,
            'input_image',
            self.image_callback,
            10
        )
        
        self.camera_info_sub = self.create_subscription(
            CameraInfo,
            'camera_info',
            self.camera_info_callback,
            10
        )
        
        # 发布深度话题
        self.depth_pub = self.create_publisher(Image, 'depth_image', 10)
        self.depth_colored_pub = self.create_publisher(Image, 'depth_colored', 10)
        self.pointcloud_pub = self.create_publisher(PointCloud2, 'depth_points', 10)
        
        self.get_logger().info('单目深度估计节点已启动')
    
    def _load_model(self):
        """加载 MiDaS 模型"""
        try:
            self.model = torch.hub.load("intel-isl/MiDaS", "DPT_Large")
            self.model.to(self.device)
            self.model.eval()
            
            # 获取 transforms
            midas_transforms = torch.hub.load("intel-isl/MiDaS", "transforms")
            self.transform = midas_transforms.dpt_transform
            
            self.get_logger().info('MiDaS DPT-Large 模型加载成功')
            
        except Exception as e:
            self.get_logger().error(f'模型加载失败: {str(e)}')
            self.model = None
    
    def camera_info_callback(self, msg):
        """相机信息回调，提取内参"""
        if not self.camera_info_received:
            self.intrinsic = np.array(msg.k).reshape(3, 3)
            self.camera_info_received = True
            self.get_logger().info(f'相机内参: fx={self.intrinsic[0,0]:.2f}, fy={self.intrinsic[1,1]:.2f}')
    
    def estimate_depth(self, image):
        """
        估计图像深度
        
        参数:
            image: BGR 图像
        
        返回:
            depth: 深度图（米）
            depth_norm: 归一化深度图（0-1）
        """
        if self.model is None:
            return None, None
        
        # BGR 转 RGB
        img_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
        
        # 预处理
        input_batch = self.transform(img_rgb).to(self.device)
        
        # 推理
        with torch.no_grad():
            prediction = self.model(input_batch)
        
        # 处理输出
        prediction = prediction.squeeze().cpu().numpy()
        
        # 缩放到原始尺寸
        original_h, original_w = image.shape[:2]
        depth = cv2.resize(
            prediction, 
            (original_w, original_h), 
            interpolation=cv2.INTER_CUBIC
        )
        
        # 归一化到 0-1
        depth_min = depth.min()
        depth_max = depth.max()
        if depth_max - depth_min > 0:
            depth_norm = (depth - depth_min) / (depth_max - depth_min)
        else:
            depth_norm = depth - depth_min
        
        # 缩放到 100 米范围
        depth = depth_norm * 100.0
        
        return depth, depth_norm
    
    def depth_to_pointcloud(self, depth, image, intrinsic):
        """
        将深度图转换为点云
        
        参数:
            depth: 深度图（米）
            image: RGB 图像
            intrinsic: 相机内参 (3, 3)
        
        返回:
            cloud_msg: PointCloud2 消息
        """
        h, w = depth.shape
        fx, fy = intrinsic[0, 0], intrinsic[1, 1]
        cx, cy = intrinsic[0, 2], intrinsic[1, 2]
        
        # 生成坐标网格
        u_coords, v_coords = np.meshgrid(np.arange(w), np.arange(h))
        
        # 展平
        u_coords = u_coords.flatten()
        v_coords = v_coords.flatten()
        z = depth.flatten()
        
        # 过滤无效点
        valid_mask = z > 0.1  # 过滤太近的点
        u_coords = u_coords[valid_mask]
        v_coords = v_coords[valid_mask]
        z = z[valid_mask]
        
        # 反投影
        x = (u_coords - cx) * z / fx
        y = (v_coords - cy) * z / fy
        
        # 获取颜色
        rgb_image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
        colors = rgb_image.reshape(-1, 3)[valid_mask]
        
        # 组合点云数据
        points = np.stack([x, y, z], axis=1)
        
        # 创建 PointCloud2 消息
        from sensor_msgs.msg import PointCloud2, PointField
        
        points_data = np.zeros(len(points), dtype=[(
            'x', np.float32), ('y', np.float32), ('z', np.float32),
            ('r', np.uint8), ('g', np.uint8), ('b', np.uint8)
        ])
        points_data['x'] = x
        points_data['y'] = y
        points_data['z'] = z
        points_data['r'] = colors[:, 0]
        points_data['g'] = colors[:, 1]
        points_data['b'] = colors[:, 2]
        
        cloud_msg = PointCloud2()
        cloud_msg.header.stamp = self.get_clock().now().to_msg()
        cloud_msg.header.frame_id = 'camera_link'
        cloud_msg.height = 1
        cloud_msg.width = len(points)
        cloud_msg.fields = [
            PointField(name='x', offset=0, datatype=PointField.FLOAT32, count=1),
            PointField(name='y', offset=4, datatype=PointField.FLOAT32, count=1),
            PointField(name='z', offset=8, datatype=PointField.FLOAT32, count=1),
            PointField(name='rgb', offset=12, datatype=PointField.UINT32, count=1)
        ]
        cloud_msg.is_bigendian = False
        cloud_msg.point_step = 16
        cloud_msg.row_step = 16 * len(points)
        cloud_msg.data = points_data.tobytes()
        cloud_msg.is_dense = True
        
        return cloud_msg
    
    def image_callback(self, msg):
        """图像回调"""
        try:
            cv_image = self.bridge.imgmsg_to_cv2(msg, desired_encoding='bgr8')
            
            # 估计深度
            depth, depth_norm = self.estimate_depth(cv_image)
            
            if depth is None:
                return
            
            # 发布深度图（米为单位，乘以 1000 转为毫米）
            depth_msg = self.bridge.cv2_to_imgmsg(
                (depth * 1000).astype(np.uint16), 
                encoding='mono16'
            )
            depth_msg.header = msg.header
            self.depth_pub.publish(depth_msg)
            
            # 发布伪彩色深度图
            depth_vis = (depth_norm * 255).astype(np.uint8)
            depth_colored = cv2.applyColorMap(depth_vis, cv2.COLORMAP_INFERNO)
            colored_msg = self.bridge.cv2_to_imgmsg(depth_colored, 'bgr8')
            colored_msg.header = msg.header
            self.depth_colored_pub.publish(colored_msg)
            
            # 发布点云（如果有相机内参）
            if self.camera_info_received:
                cloud_msg = self.depth_to_pointcloud(depth, cv_image, self.intrinsic)
                self.pointcloud_pub.publish(cloud_msg)
            
        except Exception as e:
            self.get_logger().error(f'深度估计失败: {str(e)}')


def main(args=None):
    rclpy.init(args=args)
    node = DepthEstimationNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()


if __name__ == '__main__':
    main()
```

### 6.3 障碍物检测应用

```python
# obstacle_detection_node.py - 基于深度估计的障碍物检测节点
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
from visualization_msgs.msg import MarkerArray
from geometry_msgs.msg import Point, PoseStamped, Twist
from nav_msgs.msg import OccupancyGrid, Odometry
from cv_bridge import CvBridge
import numpy as np
import cv2


class ObstacleDetectionNode(Node):
    """基于单目深度估计的障碍物检测节点"""
    
    def __init__(self, depth_threshold=1.5, height_threshold=0.3):
        """
        初始化障碍物检测节点
        
        参数:
            depth_threshold: 深度阈值（米），超过此距离视为可通行
            height_threshold: 高度阈值（米），用于判断是否为障碍物
        """
        super().__init__('obstacle_detection_node')
        self.bridge = CvBridge()
        
        self.depth_threshold = depth_threshold
        self.height_threshold = height_threshold
        
        # 订阅深度图
        self.depth_sub = self.create_subscription(
            Image, '/depth_image', self.depth_callback, 10
        )
        
        # 发布障碍物标记
        self.obstacle_pub = self.create_publisher(
            MarkerArray, '/obstacles/markers', 10
        )
        
        # 发布代价地图
        self.costmap_pub = self.create_publisher(
            OccupancyGrid, '/obstacles/costmap', 10
        )
        
        # 发布避障速度命令
        self.cmd_pub = self.create_publisher(
            Twist, '/cmd_vel', 10
        )
        
        self.get_logger().info('障碍物检测节点已启动')
        self.get_logger().info(f'深度阈值: {self.depth_threshold}m')
    
    def detect_obstacles(self, depth_image):
        """
        从深度图中检测障碍物
        
        参数:
            depth_image: 深度图（米），形状 (H, W)
        
        返回:
            obstacles: 障碍物列表，每项为 (x, y, distance)
            obstacle_mask: 障碍物掩膜
        """
        h, w = depth_image.shape
        
        # 创建网格坐标
        y_coords, x_coords = np.meshgrid(np.arange(h), np.arange(w), indexing='ij')
        
        # 相机内参（需要根据实际相机配置）
        fx = 525.0  # 焦距 x
        fy = 525.0  # 焦距 y
        cx = w / 2.0  # 主点 x
        cy = h / 2.0  # 主点 y
        
        # 计算每个像素的 3D 坐标
        z = depth_image
        x = (x_coords - cx) * z / fx
        y = (y_coords - cy) * z / fy
        
        # 检测障碍物（距离小于阈值的区域）
        obstacle_mask = (z > 0.1) & (z < self.depth_threshold)
        
        # 计算障碍物中心
        obstacles = []
        if obstacle_mask.sum() > 0:
            # 按距离将图像划分为网格
            grid_size = 20
            grid_h = h // grid_size
            grid_w = w // grid_size
            
            for i in range(grid_size):
                for j in range(grid_size):
                    roi = obstacle_mask[
                        i*grid_h:(i+1)*grid_h, 
                        j*grid_w:(j+1)*grid_w
                    ]
                    
                    if roi.sum() > (grid_h * grid_w * 0.3):  # 超过30%为障碍物
                        # 计算该网格的中心点
                        y_center = (i + 0.5) * grid_h
                        x_center = (j + 0.5) * grid_w
                        
                        # 获取该点的深度
                        dist = float(depth_image[int(y_center), int(x_center)])
                        
                        if dist > 0.1:
                            # 计算 3D 坐标
                            px = (x_center - cx) * dist / fx
                            py = (y_center - cy) * dist / fy
                            
                            obstacles.append((px, py, dist))
        
        return obstacles, obstacle_mask
    
    def create_costmap(self, obstacle_mask, resolution=0.1):
        """
        创建障碍物代价地图
        
        参数:
            obstacle_mask: 障碍物掩膜
            resolution: 地图分辨率（米/像素）
        
        返回:
            costmap: OccupancyGrid 消息
        """
        h, w = obstacle_mask.shape
        
        # 假设相机安装在机器人中心，高度0.5米
        robot_height = 0.5
        
        # 简化的代价地图计算
        # 实际应用中需要更复杂的投影逻辑
        costmap_data = np.zeros((w, h), dtype=np.int8)
        
        # 障碍物区域设为 100
        costmap_data[obstacle_mask.T] = 100
        
        # 代价地图消息
        costmap = OccupancyGrid()
        costmap.header.stamp = self.get_clock().now().to_msg()
        costmap.header.frame_id = 'camera_link'
        costmap.info.resolution = resolution
        costmap.info.width = w
        costmap.info.height = h
        costmap.info.origin.position.x = -w * resolution / 2
        costmap.info.origin.position.y = -h * resolution / 2
        costmap.data = costmap_data.flatten().tolist()
        
        return costmap
    
    def publish_obstacle_markers(self, obstacles):
        """发布障碍物可视化标记"""
        from visualization_msgs.msg import Marker
        
        marker_array = MarkerArray()
        
        for i, (px, py, dist) in enumerate(obstacles):
            marker = Marker()
            marker.header.stamp = self.get_clock().now().to_msg()
            marker.header.frame_id = 'camera_link'
            marker.id = i
            marker.type = Marker.SPHERE
            marker.action = Marker.ADD
            marker.pose.position.x = px
            marker.pose.position.y = py
            marker.pose.position.z = 0
            marker.scale.x = 0.1
            marker.scale.y = 0.1
            marker.scale.z = 0            marker.scale.z = 0.1
            marker.color.r = 1.0
            marker.color.g = 0.0
            marker.color.b = 0.0
            marker.color.a = 1.0
            
            marker_array.markers.append(marker)
        
        self.obstacle_pub.publish(marker_array)
    
    def simple_avoidance(self, obstacles):
        """简单的避障逻辑"""
        from geometry_msgs.msg import Twist
        
        cmd = Twist()
        
        # 找到最近的障碍物
        min_dist = min([obs[2] for obs in obstacles]) if obstacles else float('inf')
        
        if min_dist < 0.5:
            # 太近，停止
            cmd.linear.x = 0.0
            cmd.angular.z = 0.0
        elif min_dist < 1.0:
            # 距离较近，减速并转向
            cmd.linear.x = 0.1
            # 简单策略：向障碍物较少的方向转
            left_count = sum(1 for obs in obstacles if obs[0] < 0)
            right_count = sum(1 for obs in obstacles if obs[0] >= 0)
            cmd.angular.z = 0.5 if left_count < right_count else -0.5
        else:
            # 正常速度前进
            cmd.linear.x = 0.2
            cmd.angular.z = 0.0
        
        self.cmd_pub.publish(cmd)
    
    def depth_callback(self, msg):
        """深度图回调"""
        try:
            # 接收深度图（mono16，单位为毫米）
            depth_mm = self.bridge.imgmsg_to_cv2(msg, desired_encoding='mono16')
            depth = depth_mm.astype(np.float32) / 1000.0  # 转换为米
            
            # 检测障碍物
            obstacles, obstacle_mask = self.detect_obstacles(depth)
            
            # 发布代价地图
            costmap = self.create_costmap(obstacle_mask)
            self.costmap_pub.publish(costmap)
            
            # 发布障碍物标记
            self.publish_obstacle_markers(obstacles)
            
            # 执行避障
            self.simple_avoidance(obstacles)
            
        except Exception as e:
            self.get_logger().error(f'障碍物检测失败: {str(e)}')


def main(args=None):
    rclpy.init(args=args)
    node = ObstacleDetectionNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()


if __name__ == '__main__':
    main()
```

### 选择题

1. 单目深度估计与立体视觉（Stereo Vision）深度获取的主要区别是什么？
   - A) 单目深度估计需要两个摄像头
   - B) 单目深度估计仅需单张图像即可推断深度
   - C) 立体视觉不需要相机标定
   - D) 两者获取的深度精度相同
   
   **答案：B**。单目深度估计仅需单张 RGB 图像即可推断深度信息，而立体视觉需要两个摄像头和精确的标定来计算视差。

2. MiDaS 的核心创新是什么？
   - A) 使用单个数据集训练
   - B) 多数据集联合训练实现强泛化能力
   - C) 仅能估计室内场景深度
   - D) 输出绝对深度值，无需尺度恢复
   
   **答案：B**。MiDaS 的核心创新是通过在多个不同深度的数据集上联合训练，学习跨数据集的通用深度特征，从而具备极强的泛化能力。

3. 单目深度估计输出的是什么类型的深度？
   - A) 绝对深度，单位为米
   - B) 相对深度，仅表示前后关系
   - C) 视差图
   - D)  Disparity Map（视差图）
   
   **答案：B**。单目深度估计通常输出相对深度（尺度不变的深度顺序），绝对深度需要通过尺度恢复才能获得。

4. DPT（Dense Prediction Transformer）相比传统 CNN 编码器的优势是什么？
   - A) 更少的参数量
   - B) 更快的推理速度
   - C) 更大的感受野和更好的全局推理能力
   - D) 不需要Transformer架构
   
   **答案：C**。DPT 使用 Transformer 架构，具有更大的感受野和更好的全局推理能力，在密集预测任务上取得 SOTA 效果。

5. 使用 MiDaS 模型实现一个实时深度估计演示程序，从摄像头读取图像并实时显示深度估计结果。

   **参考答案**：
   
   ```python
   import cv2
   import torch
   import numpy as np
   
   # 加载 MiDaS
   model = torch.hub.load("intel-isl/MiDaS", "DPT_Hybrid")
   device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
   model.to(device)
   model.eval()
   
   midas_transforms = torch.hub.load("intel-isl/MiDaS", "transforms")
   transform = midas_transforms.dpt_transform
   
   # 打开摄像头
   cap = cv2.VideoCapture(0)
   
   while cap.isOpened():
       ret, frame = cap.read()
       if not ret:
           break
       
       # 预处理
       input_batch = transform(frame).to(device)
       
       # 推理
       with torch.no_grad():
           prediction = model(input_batch)
       
       # 处理输出
       depth = prediction.squeeze().cpu().numpy()
       depth = cv2.resize(depth, (frame.shape[1], frame.shape[0]))
       depth = (depth - depth.min()) / (depth.max() - depth.min())
       
       # 伪彩色可视化
       depth_vis = (depth * 255).astype(np.uint8)
       depth_colored = cv2.applyColorMap(depth_vis, cv2.COLORMAP_INFERNO)
       
       # 叠加显示
       result = cv2.addWeighted(frame, 0.5, depth_colored, 0.5, 0)
       
       cv2.imshow('Depth Estimation', result)
       
       if cv2.waitKey(1) & 0xFF == ord('q'):
           break
   
   cap.release()
   cv2.destroyAllWindows()
   ```

6. 修改深度估计节点，添加尺度恢复功能：已知场景中有一个已知尺寸的参照物体（如 A4 纸 21cm × 29.7cm），利用该参照物的尺寸反推绝对尺度。

   **提示**：
   - 在图像中检测参照物的边界
   - 计算参照物在图像中的像素尺寸
   - 通过物理尺寸与像素尺寸的比值恢复绝对尺度
   - 将恢复的尺度应用于整个深度图

## 本章小结

本章我们全面学习了单目深度估计技术及其在机器人中的应用。单目深度估计通过深度学习模型从单张 RGB 图像推断场景的三维深度信息，相比传统立体视觉具有成本低、部署灵活的优势。

我们详细介绍了相对深度与绝对深度的区别：相对深度只表示像素间的前后顺序关系，而绝对深度给出实际的物理距离（米）。单目深度估计网络通常输出尺度不变的深度，需要通过传感器融合或参照物标定来恢复绝对尺度。

MiDaS 作为当前最流行的通用单目深度估计模型，通过多数据集联合训练实现了极强的跨场景泛化能力。DPT 基于 Transformer 架构，在精度上达到了新的 SOTA 水平。

在 ROS2 集成部分，我们展示了如何将 MiDaS 模型与 ROS2 系统对接，实现实时的深度估计功能，并基于深度估计实现了障碍物检测和简单避障功能。这些技术在机器人导航、三维重建、增强现实等领域都有广泛应用。
