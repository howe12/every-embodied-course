## 1. 目标跟随概述

### 1.1 什么是目标跟随？

**目标跟随**（Object Following / Target Tracking）是机器人视觉中的一项核心任务，指机器人通过传感器（通常是相机）持续检测并追踪一个感兴趣的目标，同时根据目标的运动状态调整自身的运动策略，始终与目标保持期望的空间关系（距离、角度、方位）。

```
┌─────────────────────────────────────────────────────────────┐
│                      目标跟随示意图                          │
│                                                             │
│                        目标 (人)                             │
│                           ●                                  │
│                           │                                  │
│                    目标距离 Z                                │
│                    目标偏航角 θ                              │
│                           │                                  │
│                           ▼                                  │
│              ┌─────────────────────┐                         │
│              │      机器人          │                        │
│              │   根据目标位置调整    │                        │
│              │   前进方向和速度      │                        │
│              └─────────────────────┘                         │
│                                                             │
│   跟随目标：保持目标始终在机器人正前方适当距离                 │
│   控制目标：让机器人跟随目标运动，同时避免碰撞                │
└─────────────────────────────────────────────────────────────┘
```

目标跟随与目标检测有何区别？目标检测只告诉机器人"目标在哪里"（单帧结果），而目标跟随不仅要回答"在哪里"，还要回答"它要去哪里"以及"我怎么跟上去"。目标跟随通常包含以下核心能力：

| 能力 | 描述 |
|------|------|
| **目标检测** | 在每一帧图像中找到目标的位置（ bounding box 或像素坐标） |
| **目标跟踪** | 将连续帧中的同一目标关联起来，形成运动轨迹 |
| **距离估计** | 利用深度信息计算目标与机器人的距离 |
| **运动控制** | 根据目标位置和距离，计算机器人的速度和转向 |
| **预测能力** | 预测目标的未来位置，提前调整运动策略 |

### 1.2 应用场景

**（1）机器人跟随（Person Following）**

服务机器人或移动平台跟随人类行走是最常见的跟随应用。机器人需要识别并锁定主人，始终保持在主人身后或侧后方一定距离。当主人突然停下、转弯或加快脚步时，机器人要及时调整。

典型应用场景：
- **商场导购机器人**：跟随导购员移动，同时避开顾客
- **老年辅助机器人**：跟随老年人行走，随时准备提供帮助
- **行李搬运机器人**：在机场、车站跟随旅客，自动搬运行李
- **工厂物料机器人**：跟随工人到不同工位，完成物料配送

```
跟随模式示意：

    主人                  主人停止/转向            主人快走
     ●                       ●                      ●
     │                       │                      │
     ▼  机器人跟随 ▼        │ 机器人停下             ▼▼  机器人加速 ▼▼▼
  ┌─────┐                 ┌─────┐                ┌─────┐
  │  🤖  │                 │  🤖  │                │  🤖  │
  └─────┘                 └─────┘                └─────┘
```

**（2）无人机目标跟踪（Aerial Target Tracking）**

无人机在空中跟踪移动目标（车辆、人物、其他无人机），需要实时估计目标的三维位置（特别是高度和水平距离），并保持安全间隔。

典型应用场景：
- **影视航拍**：跟踪演员或车辆进行运动镜头拍摄
- **安防巡逻**：对可疑目标进行空中持续跟踪
- **交通监控**：从空中跟踪违法车辆并持续记录
- **搜救任务**：跟踪走失人员，无人机持续提供视野

**（3）自动驾驶中的目标跟随**

在自动驾驶和高级辅助驾驶系统（ADAS）中，目标跟随是自适应巡航控制（ACC）和自动紧急制动（AEB）的感知基础。车辆需要识别并持续跟踪前方的车辆，保持安全跟车距离。

| 应用 | 典型跟车距离 | 关键挑战 |
|------|------------|---------|
| 高速公路自适应巡航 | 50-150m | 高速运动、紧急刹车 |
| 城市拥堵辅助 | 5-30m | 频繁启停、加塞检测 |
| 自动泊车 | 1-3m | 狭窄空间、精确控制 |
| 无人机跟随拍摄 | 3-20m | 飞行稳定性、避障 |

一个完整的目标跟随系统通常包含以下模块：


**目标跟随与目标跟踪的关系：**
- **目标跟踪**（Object Tracking）强调的是在时序上对同一目标进行关联和轨迹维护
- **目标跟随**（Object Following）强调的是根据目标的状态控制机器人运动
- 两者通常结合使用：检测提供每帧的位置，跟踪提供目标的身份和轨迹，跟随控制器根据这些信息决策

## 2. 深度信息获取与处理

### 2.1 深度信息的获取方式

在目标跟随系统中，深度信息是计算目标距离的核心。获取深度信息主要有三种方式：

**（1）RGB-D 相机直接获取**

RGB-D 相机（如 Intel RealSense、Microsoft Azure Kinect）可以同时输出 RGB 彩色图像和每个像素的深度值。这是机器人室内目标跟随中最常用的方案。

```python
import numpy as np

def get_depth_from_rgbd(color_image, depth_image, bbox, depth_scale=0.001):
    """
    从 RGB-D 相机获取目标边界框内的平均深度
    
    参数:
        color_image: RGB 图像 (H×W×3)，uint8
        depth_image: 深度图 (H×W)，通常是 uint16，单位取决于 depth_scale
        bbox: 边界框 [x1, y1, x2, y2]
        depth_scale: 深度比例尺，RealSense D455 通常为 0.001（即 mm→m）
    
    返回:
        avg_depth: 边界框内平均深度（米）
        center_depth: 边界框中心点深度（米）
    """
    x1, y1, x2, y2 = bbox
    x1, y1 = max(0, x1), max(0, y1)
    x2, y2 = min(depth_image.shape[1], x2), min(depth_image.shape[0], y2)
    
    # 提取边界框区域的深度
    roi_depth = depth_image[y1:y2, x1:x2].astype(np.float32) * depth_scale
    
    # 过滤无效深度（0 表示无效）
    valid_mask = roi_depth > 0
    if valid_mask.sum() == 0:
        return None, None
    
    avg_depth = roi_depth[valid_mask].mean()
    
    # 中心点深度
    center_u, center_v = (x1 + x2) // 2, (y1 + y2) // 2
    center_depth = depth_image[center_v, center_u] * depth_scale
    
    return avg_depth, center_depth


def rgbd_depth_example():
    """
    RGB-D 深度获取示例
    """
    print("=" * 50)
    print("RGB-D 相机深度获取示例")
    print("=" * 50)
    
    # 模拟 RealSense D455 的深度图（640x480）
    H, W = 480, 640
    depth_image = np.random.randint(500, 3000, (H, W), dtype=np.uint16)  # 模拟深度 mm
    depth_scale = 0.001  # RealSense 深度单位是 mm，乘以 0.001 得到米
    
    # 模拟检测到一个"人"的边界框
    bbox = [250, 150, 390, 400]  # [x1, y1, x2, y2]
    
    avg_depth, center_depth = get_depth_from_rgbd(None, depth_image, bbox, depth_scale)
    
    print(f"\n检测到目标，边界框: {bbox}")
    print(f"边界框内平均深度: {avg_depth:.3f} 米")
    print(f"边界框中心点深度: {center_depth:.3f} 米")
    
    return depth_image, bbox


rgbd_depth_example()
```

**（2）双目立体视觉获取深度**

在没有 RGB-D 相机的情况下，可以使用双目相机通过视差计算深度。首先对左右图像进行立体匹配得到视差图，再利用相机内参和基线计算深度。

```python
import numpy as np

def disparity_to_depth(disparity_map, focal_length_px, baseline_mm):
    """
    将视差图转换为深度图
    
    参数:
        disparity_map: 视差图 (H×W)，单位：像素
        focal_length_px: 焦距（像素单位）
        baseline_mm: 双目基线距离（毫米）
    
    返回:
        depth_map: 深度图（米）
    """
    # 深度（毫米）= (焦距像素 × 基线毫米) / 视差像素
    # 当视差为0时（无穷远），避免除零
    depth_map = np.zeros_like(disparity_map, dtype=np.float32)
    valid_mask = disparity_map > 0
    
    depth_map[valid_mask] = (focal_length_px * baseline_mm) / disparity_map[valid_mask]
    
    return depth_map / 1000.0  # 转换为米


def stereo_depth_example():
    """
    双目深度获取示例
    """
    print("=" * 50)
    print("双目立体视觉深度获取示例")
    print("=" * 50)
    
    # Intel RealSense D455 双目参数
    focal_length = 639.0   # 焦距（像素，720p）
    baseline = 95.0         # 基线距离 95mm
    
    # 模拟视差图
    H, W = 480, 640
    disparity_map = np.zeros((H, W), dtype=np.float32)
    
    # 模拟场景：左边物体视差大（近），右边背景视差小（远）
    for u in range(W):
        disparity_map[:, u] = np.clip(255 - u * 0.35, 0, 255)
    
    # 添加噪声
    noise = np.random.randn(H, W) * 3.0
    disparity_map = np.clip(disparity_map + noise, 0, 255)
    
    # 转换为深度
    depth_map = disparity_to_depth(disparity_map, focal_length, baseline)
    
    print(f"焦距: {focal_length} 像素")
    print(f"基线距离: {baseline} mm")
    print(f"\n深度范围: [{depth_map[depth_map > 0].min():.2f}, {depth_map.max():.2f}] 米")
    
    # 特定位置的深度
    for u in [100, 200, 300, 400]:
        v = 240
        d = disparity_map[v, u]
        z = depth_map[v, u]
        print(f"像素 ({u}, {v}): 视差 {d:.1f} px → 深度 {z:.3f} m")
    
    return disparity_map, depth_map


stereo_depth_example()
```

### 2.2 深度信息滤波

深度相机获取的原始深度数据往往包含噪声和无效值（深度为0通常表示测量失败），需要进行滤波处理才能用于跟随控制。

```python
import numpy as np

def depth_median_filter(depth_image, kernel_size=5):
    """
    深度图的中值滤波
    有效去除深度相机常见的散粒噪声和脉冲噪声
    
    参数:
        depth_image: 深度图 (H×W)，单位米
        kernel_size: 滤波核大小（奇数）
    
    返回:
        filtered: 滤波后的深度图
    """
    from scipy.ndimage import median_filter
    # 中值滤波自动处理边界，0值不会被错误平滑
    filtered = median_filter(depth_image, size=kernel_size)
    return filtered


def depth_bilateral_filter(depth_image, sigma_color=50, sigma_space=5):
    """
    深度图的双边滤波
    保边平滑，在去噪的同时保持深度不连续边缘
    
    参数:
        depth_image: 深度图 (H×W)，单位米
        sigma_color: 颜色/深度差异显著性阈值
        sigma_space: 空间距离显著性阈值
    
    返回:
        filtered: 滤波后的深度图
    """
    # 双边滤波需要有效深度值为正
    # 对于无效深度（<=0），先用邻近有效值填充
    depth_valid = depth_image.copy()
    mask = depth_valid <= 0
    if mask.any():
        # 简单填充：使用最大深度值（最保守）
        depth_valid[mask] = depth_valid[~mask].max() if (~mask).any() else 10.0
    
    # 双边滤波（近似实现，实际建议用 cv2.bilateralFilter）
    H, W = depth_valid.shape
    filtered = depth_valid.copy()
    
    pad = sigma_space * 2
    padded = np.pad(depth_valid, pad, mode='edge')
    
    for i in range(H):
        for j in range(W):
            # 取邻域窗口
            window = padded[i:i+2*pad+1, j:j+2*pad+1]
            center_val = depth_valid[i, j]
            
            # 高斯权重计算（简化版）
            spatial_dist = np.abs(np.arange(-pad, pad+1)[:, None] - np.arange(-pad, pad+1)[None, :])
            range_dist = np.abs(window - center_val)
            
            spatial_weight = np.exp(-spatial_dist**2 / (2 * sigma_space**2))
            range_weight = np.exp(-range_dist**2 / (2 * sigma_color**2))
            
            weight = spatial_weight * range_weight
            filtered[i, j] = (window * weight).sum() / weight.sum()
    
    return filtered


def depth_inpainting(depth_image, invalid_value=0, method='nearest'):
    """
    深度图修复：填补无效深度区域
    
    参数:
        depth_image: 深度图 (H×W)
        invalid_value: 无效深度的标记值
        method: 'nearest'（最近邻填充）或 'median'（中值填充）
    
    返回:
        filled_depth: 修复后的深度图
    """
    from scipy.ndimage import distance_transform_edt
    
    filled_depth = depth_image.copy()
    invalid_mask = (depth_image <= 0)
    
    if not invalid_mask.any():
        return filled_depth  # 没有无效区域
    
    if method == 'nearest':
        # 计算到最近有效点的距离
        valid_mask = ~invalid_mask
        
        # 找到每个无效像素的最近有效像素值（近似：多次扩张）
        temp = valid_mask.astype(float)
        for _ in range(10):  # 最多扩张10次
            for di in [-1, 0, 1]:
                for dj in [-1, 0, 1]:
                    shifted = np.roll(np.roll(temp, di, axis=0), dj, axis=1)
                    temp = np.maximum(temp, shifted)
            if temp[invalid_mask].all():
                break
        
        filled_depth[invalid_mask] = temp[invalid_mask] * filled_depth[valid_mask].mean()
        
    elif method == 'median':

        filled_depth[invalid_mask] = median_depth[invalid_mask]
    
    return filled_depth
```

## 3. 目标检测与深度融合

### 3.1 RGB-D 融合的基本方法

将 RGB 图像中的目标检测结果（二维边界框）与深度图融合，可以获取目标的三维位置。核心思想是：在检测到的边界框区域内，从深度图中提取深度信息。

```
RGB-D 融合示意图：

RGB 图像                          深度图
┌────────────────┐            ┌────────────────┐
│                │            │                │
│   ┌────────┐   │            │   [2.5m][2.3m] │
│   │  检测框 │   │  ──融合──→ │   [2.1m][2.0m] │
│   │  目标   │   │            │   [0.9m][0.8m] │ ← 目标物体较近
│   └────────┘   │            │                │
│                │            │                │
└────────────────┘            └────────────────┘

边界框内深度 → 过滤无效值 → 计算平均/中心深度 → 目标3D位置
```

**融合方法一：边界框内平均深度**

取边界框内所有有效深度像素的平均值，作为目标的大致距离。

```python
import numpy as np

def get_target_3d_from_bbox(depth_image, bbox, camera_intrinsics):
    """
    从边界框和深度图获取目标的3D位置
    
    参数:
        depth_image: 深度图 (H×W)，单位米
        bbox: 边界框 [x1, y1, x2, y2]
        camera_intrinsics: 相机内参字典 {'fx', 'fy', 'cx', 'cy'}
    
    返回:
        target_3d: [X, Y, Z]，目标在相机坐标系下的3D位置（米）
        bbox_depth: 边界框内平均深度（米）
    """
    x1, y1, x2, y2 = bbox
    x1, y1 = max(0, int(x1)), max(0, int(y1))
    x2, y2 = min(depth_image.shape[1], int(x2)), min(depth_image.shape[0], int(y2))
    
    # 提取边界框内的深度
    roi_depth = depth_image[y1:y2, x1:x2]
    
    # 过滤无效深度
    valid_mask = roi_depth > 0
    if valid_mask.sum() == 0:
        return None, None
    
    valid_depths = roi_depth[valid_mask]
    avg_depth = valid_depths.mean()
    
    # 计算边界框中心像素坐标
    center_u = (x1 + x2) // 2
    center_v = (y1 + y2) // 2
    
    # 从深度和像素坐标反投影到3D
    fx = camera_intrinsics['fx']
    fy = camera_intrinsics['fy']
    cx = camera_intrinsics['cx']
    cy = camera_intrinsics['cy']
    
    Z = avg_depth
    X = (center_u - cx) * Z / fx
    Y = (center_v - cy) * Z / fy
    
    return np.array([X, Y, Z]), avg_depth


def get_target_3d_with_median(depth_image, bbox, camera_intrinsics):
    """
    使用中值深度代替平均值，减少异常值影响
    """
    x1, y1, x2, y2 = [int(v) for v in bbox]
    x1, y1 = max(0, x1), max(0, y1)
    x2, y2 = min(depth_image.shape[1], x2), min(depth_image.shape[0], y2)
    
    roi_depth = depth_image[y1:y2, x1:x2]
    valid_mask = roi_depth > 0
    
    if valid_mask.sum() == 0:
        return None, None
    
    median_depth = np.median(roi_depth[valid_mask])
    
    center_u = (x1 + x2) // 2
    center_v = (y1 + y2) // 2
    
    fx = camera_intrinsics['fx']
    fy = camera_intrinsics['fy']
    cx = camera_intrinsics['cx']
    cy = camera_intrinsics['cy']
    
    Z = median_depth
    X = (center_u - cx) * Z / fx
    Y = (center_v - cy) * Z / fy
    
    return np.array([X, Y, Z]), median_depth
```

**融合方法二：点云计算（更精确）**

将边界框内所有有效深度的点反投影到3D，再计算质心，可以得到更精确的目标3D位置。

```python
import numpy as np

def get_target_3d_pointcloud(depth_image, bbox, camera_intrinsics):
    """
    将边界框内所有有效深度的点反投影到3D，计算目标质心
    
    参数:
        depth_image: 深度图 (H×W)，单位米
        bbox: 边界框 [x1, y1, x2, y2]
        camera_intrinsics: 相机内参 {'fx', 'fy', 'cx', 'cy'}
    
    返回:
        centroid: 目标3D质心 (X, Y, Z)，米
        num_points: 用于计算的3D点数
    """
    x1, y1, x2, y2 = [int(v) for v in bbox]
    x1, y1 = max(0, x1), max(0, y1)
    x2, y2 = min(depth_image.shape[1], x2), min(depth_image.shape[0], y2)
    
    roi_depth = depth_image[y1:y2, x1:x2]
    H_roi, W_roi = roi_depth.shape
    
    # 创建像素坐标网格
    u_coords, v_coords = np.meshgrid(
        np.arange(x1, x2),
        np.arange(y1, y2)
    )
    
    # 展平
    u_flat = u_coords.ravel()
    v_flat = v_coords.ravel()
    z_flat = roi_depth.ravel()
    
    # 过滤无效深度
    valid_mask = z_flat > 0
    u_valid = u_flat[valid_mask]
    v_valid = v_flat[valid_mask]
    z_valid = z_flat[valid_mask]
    
    if len(z_valid) == 0:
        return None, 0
    
    # 反投影到3D
    fx = camera_intrinsics['fx']
    fy = camera_intrinsics['fy']
    cx = camera_intrinsics['cx']
    cy = camera_intrinsics['cy']
    
    X = (u_valid - cx) * z_valid / fx
    Y = (v_valid - cy) * z_valid / fy
    Z = z_valid
    
    # 计算质心
    centroid = np.array([X.mean(), Y.mean(), Z.mean()])
    
    return centroid, len(z_valid)
```

## 4. 目标距离估算方法

### 4.1 基于边界框的深度估计

当目标在图像中占据一定像素区域时，可以利用目标在图像中的大小来辅助深度估计。这种方法被称为**结构先验法**——假设我们知道目标的物理尺寸（如人的身高），可以通过目标在图像中的像素大小反推距离。

```
基于尺寸的深度估计原理：

已知：人的平均身高 H_real ≈ 1.7m
已知：相机焦距 fx = 639 像素
已知：目标在图像中的像素高度 h_px

由小孔成像模型：
    h_px / H_real = fx / Z
    → Z = (fx × H_real) / h_px

举例：
  图像中人的高度为 300 像素
  Z = (639 × 1.7) / 300 ≈ 3.6 米
```

```python
import numpy as np

def estimate_depth_from_bbox_size(bbox, real_height, focal_length):
    """
    根据边界框高度估算目标距离（假设知道目标物理高度）
    
    参数:
        bbox: [x1, y1, x2, y2] 边界框
        real_height: 目标的真实物理高度（米），如人的身高 1.7m
        focal_length: 相机焦距（像素）
    
    返回:
        estimated_depth: 估算深度（米）
    """
    x1, y1, x2, y2 = bbox
    pixel_height = y2 - y1  # 边界框的像素高度
    
    if pixel_height <= 0:
        return None
    
    # Z = (fx × H_real) / h_px
    estimated_depth = (focal_length * real_height) / pixel_height
    
    return estimated_depth
```

### 4.2 深度滤波与时序平滑

单帧深度估计容易受到噪声影响，使用**时序滤波**可以显著提高深度估计的稳定性和精度。常用的方法包括指数移动平均（EMA）和卡尔曼滤波。

```python
import numpy as np

class DepthFilter:
    """
    深度滤波器：使用指数移动平均对深度进行时序平滑
    """
    
    def __init__(self, alpha=0.3, initial_value=None):
        """
        参数:
            alpha: 平滑系数，范围 [0, 1]
                   alpha 越小，平滑越强（响应越慢）
                   alpha 越大，响应越快（平滑越弱）
            initial_value: 初始深度值
        """
        self.alpha = alpha
        self.depth = initial_value
        self.history = []
    
    def update(self, new_depth):
        """
        输入新的深度观测，返回平滑后的深度
        
        参数:
            new_depth: 新的深度观测值（米），None 表示无效
        """
        if new_depth is None or new_depth <= 0:
            # 无效观测，保留历史值
            self.history.append(self.depth)
            return self.depth
        
        if self.depth is None:
            # 第一次有效观测
            self.depth = new_depth
        else:
            # 指数移动平均：depth_new = alpha * obs + (1-alpha) * depth_old
            self.depth = self.alpha * new_depth + (1 - self.alpha) * self.depth
        
        self.history.append(self.depth)
        return self.depth
    
    def get_depth(self):
        return self.depth


class KalmanDepthFilter:
    """
    一维卡尔曼滤波器：用于深度估计的时序融合
    比 EMA 更理论化，可以同时估计深度和深度变化率
    """
    
    def __init__(self, process_noise=0.1, measurement_noise=0.5):
        """
        参数:
            process_noise: 过程噪声标准差（目标移动带来的不确定性）
            measurement_noise: 测量噪声标准差（深度传感器的不确定性）
        """
        self.x = None      # 状态：深度
        self.v = None      # 状态：深度变化率
        self.P = None      # 状态协方差
        
        self.Q = process_noise ** 2   # 过程噪声方差
        self.R = measurement_noise ** 2  # 测量噪声方差
    
    def predict(self, dt=1):
        """
        预测步骤：根据上一状态预测当前状态
        
        参数:
            dt: 时间步长（帧间隔）
        """
        if self.x is None:
            return
        
        # 状态转移：x_new = x + v * dt
        # v_new = v（假设匀速）
        F = np.array([[1, dt], [0, 1]])  # 状态转移矩阵
        x_pred = F @ np.array([self.x, self.v])
        
        # 协方差预测：P_new = F @ P @ F.T + Q
        self.P = F @ self.P @ F.T + self.Q * np.eye(2)
        
        self.x, self.v = x_pred
    
    def update(self, measurement):
        """
        更新步骤：融合新的测量值
        
        参数:
            measurement: 深度测量值（米）
        """
        if measurement is None or measurement <= 0:
            return
        
        if self.x is None:
            # 初始化
            self.x = measurement
            self.v = 0.0
            self.P = np.eye(2) * self.R
            return
        
        # 测量矩阵：z = H @ x + noise
        H = np.array([[1, 0]])
        
        # 测量残差（创新）
        z = measurement
        y = z - H @ np.array([self.x, self.v])  # 残差
        
        # 残差协方差：S = H @ P @ H.T + R
        S = H @ self.P @ H.T + self.R
        
        # 卡尔曼增益：K = P @ H.T @ S^-1
        K = self.P @ H.T / S
        
        # 状态更新：x_new = x_old + K @ y
        state = np.array([self.x, self.v]) + K * y
        self.x, self.v = state
        
        # 协方差更新：P_new = (I - K @ H) @ P
        I_KH = np.eye(2) - np.outer(K, H)
        self.P = I_KH @ self.P
    
    def get_depth(self):
        return self.x
    
    def get_velocity(self):
        return self.v


def depth_filtering_comparison():
```

## 5. 目标跟随控制策略

### 5.1 跟随坐标系与误差定义

在讨论跟随控制策略之前，我们需要明确机器人与目标之间的空间关系。目标跟随通常在**机器人坐标系**下描述问题：

```
跟随坐标系示意图（俯视图）：

                      机器人前进方向
                         (+X_robot)
                            ↑
                            │
              目标偏右角 θ  │  目标在右前方
                         │/
                   目标 ● ─┼───● 机器人
                        /  │  /
                       /   │ /
                      /    │/
              目标距离 Z    θ
              
    偏航角 θ > 0：目标在机器人右侧
    偏航角 θ < 0：目标在机器人左侧
    
    控制目标：
      1. 距离误差 e_dist = Z_target - Z_desired → 控制前进速度 v
      2. 偏航角误差 e_yaw = θ_target → 控制转向速度 ω
```

```python
import numpy as np

def compute_following_errors(target_3d, desired_distance=1.5):
    """
    计算跟随误差（机器人坐标系下）
    
    参数:
        target_3d: 目标在相机坐标系下的3D位置 [X, Y, Z]，通常需要转换到机器人坐标系
        desired_distance: 期望跟随距离（米）
    
    返回:
        distance: 目标距离（米）
        yaw_angle: 目标偏航角（弧度）
        distance_error: 距离误差（米），正值表示目标太远
        yaw_error: 偏航角误差（弧度），正值表示目标偏右
    """
    X, Y, Z = target_3d
    
    # 距离：到机器人的水平距离（忽略高度 Y）
    # 实际跟随中通常只考虑水平距离
    distance = np.sqrt(X**2 + Z**2)
    
    # 偏航角：目标相对机器人正前方的角度
    # atan2(X, Z)：X 是侧向（机器人右侧为正），Z 是前方（机器人正前方为正）
    yaw_angle = np.arctan2(X, Z)  # 弧度
    
    # 误差计算
    distance_error = distance - desired_distance
    yaw_error = yaw_angle  # 期望偏航角为 0（正前方）
    
    return {
        'distance': distance,
        'yaw_angle': yaw_angle,
        'yaw_angle_deg': np.degrees(yaw_angle),
        'distance_error': distance_error,
        'yaw_error': yaw_error,
    }
```

### 5.2 纯距离跟随（Bang-Bang 控制）

最简单的跟随策略是**Bang-Bang 控制**（也称开关控制）：当目标太远时全速前进，当目标太近时全速后退，当目标在合适范围内时停止。这种控制简单但效果粗糙，会导致机器人运动不平滑。

```python
import numpy as np

def bangbang_controller(target_distance, desired_distance,
                         distance_tolerance=0.2,
                         speed_forward=0.3,
                         speed_backward=0.2):
    """
    Bang-Bang 跟随控制器
    
    策略：
      - 距离误差 > tolerance：全速前进
      - 距离误差 < -tolerance：全速后退
      - |距离误差| <= tolerance：停止
    
    参数:
        target_distance: 目标当前距离（米）
        desired_distance: 期望跟随距离（米）
        distance_tolerance: 距离容差（米）
        speed_forward: 前进速度（m/s）
        speed_backward: 后退速度（m/s）
    
    返回:
        v: 线速度 (m/s)，正值前进
    """
    error = target_distance - desired_distance
    
    if error > distance_tolerance:
        v = speed_forward  # 目标太远，前进
    elif error < -distance_tolerance:
        v = -speed_backward  # 目标太近，后退
    else:
        v = 0.0  # 距离合适，停止
    
    return v


def pure_yaw_controller(yaw_error, max_angular_velocity=1.0):
    """
    纯方向控制器：目标偏右则右转，偏左则左转
    
    参数:
        yaw_error: 偏航角误差（弧度），正值=目标偏右
        max_angular_velocity: 最大角速度 (rad/s)
    
    返回:
        omega: 角速度 (rad/s)
    """
    # 简单的比例控制
    omega = np.clip(yaw_error * 2.0, -max_angular_velocity, max_angular_velocity)
    return omega
```

### 5.3 PID 控制跟随

**PID 控制**（比例-积分-微分控制）是工程中最广泛使用的闭环控制方法。在目标跟随中，PID 可以分别控制距离和方向：

```python
import numpy as np

class PIDController:
    """
    PID 控制器
    
    控制律：u(t) = Kp * e(t) + Ki * ∫e(t)dt + Kd * de(t)/dt
    
    在离散系统中：
        u[k] = Kp * e[k] + Ki * Σe[i] + Kd * (e[k] - e[k-1])
    """
    
    def __init__(self, kp=1.0, ki=0.0, kd=0.0, output_limits=None):
        """
        参数:
            kp: 比例增益
            ki: 积分增益
            kd: 微分增益
            output_limits: 输出限幅 (min, max)
        """
        self.kp = kp
        self.ki = ki
        self.kd = kd
        
        self.integral = 0.0   # 积分累积
        self.prev_error = None  # 上一时刻误差
        self.output_limits = output_limits
    
    def compute(self, error, dt=1.0):
        """
        计算 PID 控制输出
        
        参数:
            error: 当前误差
            dt: 时间步长（秒）
        
        返回:
            output: 控制输出
        """
        # 比例项
        p_term = self.kp * error
        
        # 积分项（带积分限幅，防止积分饱和）
        self.integral += error * dt
        i_term = self.ki * self.integral
        
        # 微分项
        if self.prev_error is not None:
            d_term = self.kd * (error - self.prev_error) / dt
        else:
            d_term = 0.0
        
        # 总输出
        output = p_term + i_term + d_term
        
        # 输出限幅
        if self.output_limits is not None:
            output = np.clip(output, self.output_limits[0], self.output_limits[1])
        
        self.prev_error = error
        
        return output
    
    def reset(self):
        """重置控制器状态"""
        self.integral = 0.0
        self.prev_error = None


class FollowingController:
    """
    基于 PID 的目标跟随控制器
    
    包含两个独立的 PID 控制器：
    - 距离 PID：控制机器人与目标的距离
    - 方向 PID：控制机器人朝向目标
    """
    
    def __init__(self,
                 desired_distance=1.5,
                 max_linear_speed=0.5,
                 max_angular_speed=1.5):
        """
        参数:
            desired_distance: 期望跟随距离（米）
            max_linear_speed: 最大线速度 (m/s)
            max_angular_speed: 最大角速度 (rad/s)
        """
        self.desired_distance = desired_distance
        
        # 距离 PID：Kp=0.8, Ki=0.05, Kd=0.3
        self.distance_pid = PIDController(
            kp=0.8, ki=0.05, kd=0.3,
            output_limits=(-max_linear_speed, max_linear_speed)
        )
        
        # 方向 PID：Kp=2.0, Ki=0.0, Kd=0.1
        self.yaw_pid = PIDController(
            kp=2.0, ki=0.0, kd=0.1,
            output_limits=(-max_angular_speed, max_angular_speed)
        )
        
        # 速度平滑（低通滤波）
        self.v_filtered = 0.0
        self.omega_filtered = 0.0
        self.alpha = 0.7  # 速度平滑系数
    
    def compute(self, target_3d, dt=0.1):
        """
        计算跟随控制指令
        
        参数:
            target_3d: 目标3D位置 [X, Y, Z]，米
            dt: 时间步长（秒）
        
        返回:
            v: 线速度 (m/s)
            omega: 角速度 (rad/s)
        """
        X, Y, Z = target_3d
        
        # 计算当前距离和偏航角
        current_distance = np.sqrt(X**2 + Z**2)
        yaw_error = np.arctan2(X, Z)
        
        # 距离误差
        distance_error = current_distance - self.desired_distance
        
        # PID 控制
        v = self.distance_pid.compute(distance_error, dt)
        omega = self.yaw_pid.compute(yaw_error, dt)
        
        # 速度平滑（减少抖动）
        self.v_filtered = self.alpha * v + (1 - self.alpha) * self.v_filtered
        self.omega_filtered = self.alpha * omega + (1 - self.alpha) * self.omega_filtered
        
        return self.v_filtered, self.omega_filtered
    
    def reset(self):
        self.distance_pid.reset()
        self.yaw_pid.reset()
        self.v_filtered = 0.0
        self.omega_filtered = 0.0


def pid_following_demo():
    """
    PID 跟随控制演示
    """

    controller = FollowingController(
        desired_distance=1.5,
        max_linear_speed=0.5,
        max_angular_speed=1.5
    )

    np.random.seed(42)
    target_positions = []
    
    # 目标从3米处逐渐走近，然后左右移动
    for i in range(15):
        if i < 5:
            # 目标逐渐靠近
            z = 3.0 - i * 0.3
            x = 0.0
        elif i < 10:
            # 目标左右摆动
            z = 1.5
            x = 0.5 * np.sin((i - 5) * 0.8)
        else:
            # 目标后退
            z = 1.5 + (i - 10) * 0.2
            x = 0.0
        target_positions.append([x, 0.0, z])
    
    print(f"\n{'帧':<4} {'目标位置 [X,Z]':<16} {'距离':<8} {'偏航角°':<10} "
          f"{'v (m/s)':<10} {'ω (rad/s)':<12} {'机器人动作'}")
    print("-" * 85)
    
    for i, pos in enumerate(target_positions):
        X, Y, Z = pos
        dist = np.sqrt(X**2 + Z**2)
        yaw_deg = np.degrees(np.arctan2(X, Z))
        
        v, omega = controller.compute(pos, dt=0.1)
        
        # 描述机器人动作
        actions = []
        if abs(v) > 0.01:
            actions.append(f"{'前进' if v > 0 else '后退'}{abs(v):.2f}")
        if abs(omega) > 0.05:
            actions.append(f"{'右转' if omega > 0 else '左转'}{abs(omega):.2f}")
        if not actions:
            actions.append("停止")
        
        action_str = ", ".join(actions)
        print(f"{i:<4} [{X:+.2f}, {Z:.2f}]      {dist:<8.2f} {yaw_deg:<10.1f} "
              f"{v:<10.3f} {omega:<12.3f} {action_str}")
    
    print()
    print("PID 控制的优点：")
    print("  - 运动平滑：速度连续变化，无 Bang-Bang 的突变抖动")
    print("  - 鲁棒性：积分项消除稳态误差，微分项抑制超调")
    print("  - 参数调节：Kp/Ki/Kd 三个参数可独立调节")


pid_following_demo()
```

## 6. 多目标场景处理

### 6.1 目标选择策略

当相机视野中存在多个潜在目标时，机器人需要做出选择：

| 策略 | 描述 | 优点 | 缺点 |
|------|------|------|------|
| **最近目标** | 跟随距离最近的目标 | 简单直接 | 可能频繁切换目标 |
| **最大目标** | 跟随图像中最大的目标 | 稳定 | 不考虑实际需求 |
| **指定类别** | 跟随特定类别（如只跟人） | 可定制 | 需要类别检测能力 |
| **指定特征** | 跟随特定颜色或外观的目标 | 可区分同类别目标 | 外观可能变化 |
| **指定 ID** | 跟随被分配的特定目标 ID | 最稳定 | 需要目标跟踪器 |

```python
import numpy as np

def select_target_by_strategy(detections, strategy='closest', target_class='person'):
    if not detections:
        return None
    
    valid_detections = [d for d in detections
                        if d.get('distance') is not None and d.get('distance') > 0]
    
    if not valid_detections:
        return None
    
    if strategy == 'closest':
        selected = min(valid_detections, key=lambda d: d['distance'])
    elif strategy == 'largest':
        def bbox_area(d):
            x1, y1, x2, y2 = d['bbox']
            return (x2 - x1) * (y2 - y1)
        selected = max(valid_detections, key=bbox_area)
    elif strategy == 'class_only':
        class_detections = [d for d in valid_detections if d['class_name'] == target_class]
        if not class_detections:
            return None
        selected = min(class_detections, key=lambda d: d['distance'])
    elif strategy == 'confidence':
        selected = max(valid_detections, key=lambda d: d['confidence'])
    else:
        selected = valid_detections[0]
    
    return selected
```

### 6.2 目标跟踪（Data Association）

当存在多个候选目标时，仅靠单帧检测无法保证跟随的稳定性。**目标跟踪**（Tracking）通过在连续帧之间维护目标的身份，解决"哪个检测结果对应哪个目标"的问题。

```python
import numpy as np

class SimpleTargetTracker:
    """
    简单的目标跟踪器（基于贪心匹配）
    
    功能：
    - 维护已知目标列表（包含 ID、位置、速度）
    - 将新检测与已知目标关联
    - 创建新目标或删除消失目标
    """
    
    def __init__(self, max_age=10, min_hits=3):
        self.next_id = 0
        self.tracks = {}
        self.max_age = max_age
        self.min_hits = min_hits
    
    def update(self, detections):
        """更新跟踪器，detections: 当前帧检测列表，每个元素 [X, Y, Z]"""
        if not self.tracks:
            for det in detections:
                track_id = self.next_id
                self.next_id += 1
                self.tracks[track_id] = {
                    'position': np.array(det[:3]),
                    'velocity': np.zeros(3),
                    'age': 0,
                    'hits': 1,
                }
            return self._get_confirmed()
        
        # Step 1: 预测所有跟踪目标的位置（匀速模型）
        for track in self.tracks.values():
            track['position'] = track['position'] + track['velocity'] * 0.1
            track['age'] += 1
        
        # Step 2: 贪心匹配
        matched_tracks = set()
        matched_detections = set()
        det_positions = [np.array(d[:3]) for d in detections]
        
        for track_id, track in self.tracks.items():
            best_match_idx = -1
            best_dist = float('inf')
            
            for i, det_pos in enumerate(det_positions):
                if i in matched_detections:
                    continue
                dist = np.linalg.norm(track['position'] - det_pos)
                if dist < best_dist and dist < 1.0:
                    best_dist = dist
                    best_match_idx = i
            
            if best_match_idx >= 0:
                matched_detections.add(best_match_idx)
                matched_tracks.add(track_id)
                
                new_pos = det_positions[best_match_idx]
                old_pos = track['position']
                track['velocity'] = 0.5 * (new_pos - old_pos) / 0.1 + 0.5 * track['velocity']
                track['position'] = new_pos
                track['age'] = 0
                track['hits'] += 1
        
        # Step 3: 未匹配的检测 -> 创建新目标
        for i, det_pos in enumerate(det_positions):
            if i not in matched_detections:
                track_id = self.next_id
                self.next_id += 1
                self.tracks[track_id] = {
                    'position': det_pos,
                    'velocity': np.zeros(3),
                    'age': 0,
                    'hits': 1,
                }
        
        # Step 4: 删除过旧的目标
        to_delete = [tid for tid, track in self.tracks.items()
                      if track['age'] > self.max_age]
        for tid in to_delete:
            del self.tracks[tid]
        
        return self._get_confirmed()
    
    def _get_confirmed(self):
        confirmed = []
        for tid, track in self.tracks.items():
            if track['hits'] >= self.min_hits:
                confirmed.append({
                    'id': tid,
                    'position': track['position'].copy(),
                    'velocity': track['velocity'].copy(),
                    'hits': track['hits'],
                })
        return confirmed


def target_tracker_demo():
    print("=" * 50)

        info_str = "; ".join(track_info) if track_info else "无"
        print(f" {i:<3} | {len(dets):<6} | {len(confirmed):<10} | {info_str}")
```

```python
import numpy as np
import cv2

class DepthObjectDetector:
    """
    深度目标检测与3D定位器
    
    整合：
    - RGB图像目标检测（YOLO）
    - 深度图融合（获取目标3D位置）
    - 时序滤波（平滑深度估计）
    """
    
    def __init__(self, camera_intrinsics, target_class='person', depth_filter_alpha=0.3):
        self.camera_intrinsics = camera_intrinsics
        self.target_class = target_class
        self.depth_filters = {}
    
    def detect_and_localize(self, color_image, depth_image, detections):
        """对检测结果进行深度融合，获取3D位置"""
        localized = []
        
        for det in detections:
            if det.get('class_name') != self.target_class:
                continue
            
            bbox = det['bbox']
            x1, y1, x2, y2 = [int(v) for v in bbox]
            
            h, w = depth_image.shape[:2]
            x1, y1 = max(0, x1), max(0, y1)
            x2, y2 = min(w, x2), min(h, y2)
            
            if x2 <= x1 or y2 <= y1:
                continue
            
            roi_depth = depth_image[y1:y2, x1:x2].astype(np.float32)
            valid_mask = roi_depth > 0
            
            if not valid_mask.any():
                continue
            
            median_depth = np.median(roi_depth[valid_mask])
            
            # 时序滤波
            track_id = det.get('track_id', id(det))
            if track_id not in self.depth_filters:
                self.depth_filters[track_id] = DepthFilter(alpha=depth_filter_alpha)
            
            filtered_depth = self.depth_filters[track_id].update(median_depth)
            
            # 计算3D位置
            center_u = (x1 + x2) // 2
            center_v = (y1 + y2) // 2
            
            fx = self.camera_intrinsics['fx']
            fy = self.camera_intrinsics['fy']
            cx = self.camera_intrinsics['cx']
            cy = self.camera_intrinsics['cy']
            
            Z = filtered_depth
            X = (center_u - cx) * Z / fx
            Y = (center_v - cy) * Z / fy
            
            localized_det = det.copy()
            localized_det['position_3d'] = np.array([X, Y, Z])
            localized_det['distance'] = Z
            localized_det['filtered_depth'] = filtered_depth
            
            localized.append(localized_det)
        
        return localized


class DepthFilter:
    """深度时序滤波器：指数移动平均"""
    def __init__(self, alpha=0.3):
        self.alpha = alpha
        self.depth = None
    
    def update(self, new_depth):
        if new_depth is None or new_depth <= 0:
            return self.depth
        if self.depth is None:
            self.depth = new_depth
        else:
            self.depth = self.alpha * new_depth + (1 - self.alpha) * self.depth
        return self.depth
```

```python
import numpy as np

class PIDControllerSingleVar:
    """单变量 PID 控制器"""
    def __init__(self, kp=1.0, ki=0.0, kd=0.0):
        self.kp = kp
        self.ki = ki
        self.kd = kd
        self.integral = 0.0
        self.prev_error = None
        self.output_min = None
        self.output_max = None
    
    def set_limits(self, min_val, max_val):
        self.output_min = min_val
        self.output_max = max_val
    
    def compute(self, error, dt=1.0):
        p_term = self.kp * error
        self.integral += error * dt
        i_term = self.ki * self.integral
        d_term = 0.0
        if self.prev_error is not None:
            d_term = self.kd * (error - self.prev_error) / dt
        output = p_term + i_term + d_term
        if self.output_min is not None:
            output = max(self.output_min, min(self.output_max, output))
        self.prev_error = error
        return output
    
    def reset(self):
        self.integral = 0.0
        self.prev_error = None


class ObjectFollowingController:
    """
    完整的目标跟随控制器
    
    功能：
    1. 接收目标3D位置
    2. 计算跟随误差（距离误差 + 偏航角误差）
    3. PID 控制计算速度指令
    4. 输出 (v, omega) 给机器人底盘
    """
    
    def __init__(self, desired_distance=1.5,
                 max_linear_speed=0.5, max_angular_speed=1.5,
                 kp_distance=0.8, kp_yaw=2.0):
        self.desired_distance = desired_distance
        
        self.distance_pid = PIDControllerSingleVar(kp=kp_distance, ki=0.05, kd=0.3)
        self.yaw_pid = PIDControllerSingleVar(kp=kp_yaw, ki=0.0, kd=0.1)
        self.distance_pid.set_limits(-max_linear_speed, max_linear_speed)
        self.yaw_pid.set_limits(-max_angular_speed, max_angular_speed)
        
        self.max_linear_speed = max_linear_speed
        self.max_angular_speed = max_angular_speed
        
        self.v_filtered = 0.0
        self.omega_filtered = 0.0
    
    def compute_velocity(self, target_3d, dt=0.1):
        """
        计算跟随速度指令
        
        参数:
            target_3d: 目标3D位置 [X, Y, Z]（米）
            dt: 时间步长（秒）
        
        返回:
            v: 线速度 (m/s)
            omega: 角速度 (rad/s)
        """
        if target_3d is None:
            self.v_filtered *= 0.8
            self.omega_filtered *= 0.8
            return self.v_filtered, self.omega_filtered
        
        X, Y, Z = target_3d
        
        current_distance = np.sqrt(X**2 + Z**2)
        yaw_error = np.arctan2(X, Z)
        distance_error = current_distance - self.desired_distance
        
        v = self.distance_pid.compute(distance_error, dt)
        omega = self.yaw_pid.compute(yaw_error, dt)
        
        # 速度平滑
        alpha = 0.7
        self.v_filtered = alpha * v + (1 - alpha) * self.v_filtered
        self.omega_filtered = alpha * omega + (1 - alpha) * self.omega_filtered
        
        return self.v_filtered, self.omega_filtered
    
    def reset(self):
        self.distance_pid.reset()
        self.yaw_pid.reset()
        self.v_filtered = 0.0
        self.omega_filtered = 0.0


def following_controller_demo():

            x = 0.0
        target_positions.append(np.array([x, 0.0, z]))
    
    print(f"\n{'帧':<4} {'目标 [X,Z]':<16} {'距离':<8} {'偏航角':<10} "
          f"{'v (m/s)':<10} {'w (rad/s)':<12}")
    print("-" * 75)
    
    for i, pos in enumerate(target_positions):
        v, omega = controller.compute_velocity(pos, dt=0.1)
        
        dist = np.sqrt(pos[0]**2 + pos[2]**2)
```

```python
        self.camera_info_received = False
        
        # 深度图和工具
        self.latest_depth = None
        self.bridge = CvBridge()
        
        # 当前跟踪目标
        self.tracked_target = None
        self.target_position_history = []
        
        # PID 控制器
        self.distance_pid = PIDControllerSingleVar(kp=0.8, ki=0.05, kd=0.3)
        self.yaw_pid = PIDControllerSingleVar(kp=2.0, ki=0.0, kd=0.1)
        self.distance_pid.set_limits(-self.max_linear_speed, self.max_linear_speed)
        self.yaw_pid.set_limits(-self.max_angular_speed, self.max_angular_speed)
        
        # 速度平滑
        self.v_filtered = 0.0
        self.omega_filtered = 0.0
        
        # 50Hz 控制循环
        self.control_timer = self.create_timer(0.02, self.control_callback)
        
        # 订阅深度图像
        self.depth_sub = self.create_subscription(
            Image,
            '/camera/aligned_depth/image_raw',
            self.depth_callback,
            10
        )
        
        # 订阅相机内参
        self.info_sub = self.create_subscription(
            CameraInfo,
            '/camera/depth/camera_info',
            self.camera_info_callback,
            10
        )
        
        # 订阅目标检测结果
        self.detection_sub = self.create_subscription(
            Int32MultiArray,
            '/tracking/detections',
            self.detection_callback,
            10
        )
        
        # 发布速度指令
        self.cmd_vel_pub = self.create_publisher(Twist, '/cmd_vel', 10)
        
        self.get_logger().info('目标跟随控制节点已启动')
        self.get_logger().info(f'期望跟随距离: {self.desired_distance}m')
    
    def camera_info_callback(self, msg):
        """从 CameraInfo 消息中提取相机内参"""
        if self.camera_info_received:
            return
        self.camera_intrinsics['fx'] = msg.k[0]
        self.camera_intrinsics['fy'] = msg.k[4]
        self.camera_intrinsics['cx'] = msg.k[2]
        self.camera_intrinsics['cy'] = msg.k[5]
        self.camera_info_received = True
        self.get_logger().info(f'相机内参已更新: {self.camera_intrinsics}')
    
    def depth_callback(self, msg):
        """接收深度图像"""
        try:
            # 转换为 numpy 数组（深度单位 mm，需要 /1000 转为米）
            depth = self.bridge.imgmsg_to_cv2(msg, desired_encoding='passthrough')
            self.latest_depth = depth.astype(np.float32) / 1000.0
        except Exception as e:
            self.get_logger().error(f'深度图像转换失败: {e}')
    
    def detection_callback(self, msg):
        """接收目标检测结果（格式：[track_id, x1, y1, x2, y2, class_id]）"""
        detections = []
        data = list(msg.data)
        for i in range(0, len(data), 6):
            if i + 5 < len(data):
                track_id, x1, y1, x2, y2, class_id = data[i:i+6]
                detections.append({
                    'track_id': track_id,
                    'bbox': [x1, y1, x2, y2],
                    'class_id': class_id,
                })
        if detections:
            self.process_detections(detections)
    
    def process_detections(self, detections):
        """处理检测结果：深度融合 + 目标选择"""
        if self.latest_depth is None:
            return
        
        class_detections = [d for d in detections if d['class_id'] == self.target_class]
        if not class_detections:
            self.tracked_target = None
            return
        
        localized_detections = []
        for det in class_detections:
            pos_3d, dist = self.get_3d_position(det['bbox'])
            if pos_3d is not None:
                localized_detections.append({
                    **det,
                    'position_3d': pos_3d,
                    'distance': dist,
                })
        
        if not localized_detections:
            return
        
        # 优先选择当前跟踪的目标 ID，其次选择最近的
        if self.tracked_target is not None:
            matched = [d for d in localized_detections
                      if d['track_id'] == self.tracked_target['track_id']]
            selected = matched[0] if matched else min(localized_detections, key=lambda d: d['distance'])
        else:
            selected = min(localized_detections, key=lambda d: d['distance'])
        
        self.tracked_target = {
            'track_id': selected['track_id'],
            'position_3d': selected['position_3d'],
            'distance': selected['distance'],
        }
        
        self.target_position_history.append(selected['position_3d'])
        if len(self.target_position_history) > 10:
            self.target_position_history.pop(0)
    
    def get_3d_position(self, bbox):
        """从边界框和深度图获取目标3D位置"""
        x1, y1, x2, y2 = [int(v) for v in bbox]
        h, w = self.latest_depth.shape[:2]
        x1, y1 = max(0, x1), max(0, y1)
        x2, y2 = min(w, x2), min(h, y2)
        
        if x2 <= x1 or y2 <= y1:
            return None, None
        
        roi_depth = self.latest_depth[y1:y2, x1:x2]
        valid_mask = roi_depth > 0
        
        if not valid_mask.any():
            return None, None
        
        median_depth = np.median(roi_depth[valid_mask])
        
        center_u = (x1 + x2) // 2
        center_v = (y1 + y2) // 2
        
        fx = self.camera_intrinsics['fx']
        fy = self.camera_intrinsics['fy']
        cx = self.camera_intrinsics['cx']
        cy = self.camera_intrinsics['cy']
        
        Z = median_depth
        X = (center_u - cx) * Z / fx
        Y = (center_v - cy) * Z / fy
        
        return np.array([X, Y, Z]), median_depth
    
    def control_callback(self):
        """50Hz 控制循环"""
        if not self.enable_following:
            return
        
        if self.tracked_target is None:
            # 无目标，缓慢停止
            self.v_filtered *= 0.8
            self.omega_filtered *= 0.8
        else:
            target_3d = self.tracked_target['position_3d']
            current_distance = np.sqrt(target_3d[0]**2 + target_3d[2]**2)
            yaw_error = np.arctan2(target_3d[0], target_3d[2])
            distance_error = current_distance - self.desired_distance
            
            v = self.distance_pid.compute(distance_error, dt=0.02)
            omega = self.yaw_pid.compute(yaw_error, dt=0.02)
            
            alpha = 0.7
            self.v_filtered = alpha * v + (1 - alpha) * self.v_filtered
            self.omega_filtered = alpha * omega + (1 - alpha) * self.omega_filtered
        
        # 发布速度指令
        twist = Twist()
        twist.linear.x = self.v_filtered
        twist.angular.z = self.omega_filtered
        self.cmd_vel_pub.publish(twist)


def main(args=None):
    rclpy.init(args=args)
    node = FollowingControlNode()
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

## 本章小结

本章我们系统性地学习了基于深度信息的目标跟随技术。

在目标跟随概述部分，我们了解到目标跟随是机器人视觉中的核心任务之一，广泛应用于服务机器人跟随、无人机跟踪、自动驾驶等场景。一个完整的目标跟随系统包含目标检测、深度融合、目标跟踪和跟随控制四大模块。

在深度信息获取与处理部分，我们学习了三种获取深度的方式：RGB-D 相机直接获取、双目立体视觉和单目深度估计（深度学习），以及深度图的滤波处理（中值滤波、双边滤波、深度修复）。

在目标检测与深度融合部分，我们学习了如何将 YOLO 等目标检测器的二维检测结果与深度图融合，获取目标的三维位置。包括基于边界框内平均/中值深度的方法，以及将所有有效深度点反投影到3D后计算质心的精确方法。

在目标距离估算方法部分，我们学习了基于边界框尺寸的深度估计（结构先验法），以及深度滤波与时序平滑技术（EMA、卡尔曼滤波），显著提高了深度估计的稳定性和精度。

在目标跟随控制策略部分，我们从简单的 Bang-Bang 控制出发，逐步过渡到 PID 控制，分析了各自的优缺点和适用场景。PID 控制通过独立控制距离和偏航角，实现了平滑稳定的跟随效果。

在多目标场景处理部分，我们学习了多目标选择策略（最近、最大、指定类别等）和目标跟踪（Data Association），解决了多目标环境下的目标身份维护问题。

在 ROS2 实现部分，我们设计了一个完整的目标跟随系统架构，包含相机驱动节点、目标检测节点和跟随控制节点，并在 ROS2 中实现了完整的跟随控制节点代码，包括参数配置、深度融合、PID 控制和速度发布。
