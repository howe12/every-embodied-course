# 10-计算机视觉应用

> 本章节讲解计算机视觉在机器人中的应用

## 章节简介

本章节涵盖图像处理、目标检测、人体识别、语义分割、实例分割、深度学习CNN、以及深度估计等计算机视觉技术，为机器人赋予视觉感知能力。
