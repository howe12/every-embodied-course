## 📚 实验课程体系

### 为什么需要5个实验？

深度学习网络就像搭积木，每个模块都有独特作用。本课程通过**控制变量法**，让学生亲手验证每个模块的效果。

**核心思想**：每个实验只比上一个实验多一个模块，这样才能精确知道每个模块的贡献。

```
实验1（Baseline）
    ↓ +卷积层
实验2（+Conv+Pool）
    ↓ +BatchNorm
实验3（+BN）
    ↓ +Dropout
实验4（+Dropout）
    ↓ +数据增强
实验5（+增强）
    ↓ 自由调参
实验6（竞赛）
```

### 模块积木对照表

| 模块 | PyTorch | 核心参数 | 首次出现 |
|------|---------|---------|---------|
| 全连接层 | `nn.Linear` | in/out_features | 实验1 |
| 卷积层 | `nn.Conv2d` | in/out_channels, kernel, stride, padding | 实验2 |

## 实验1：Baseline - 纯全连接网络

### 📖 实验说明

**实验目标**：建立纯全连接网络的基准线，理解深度学习训练的基本流程

**网络结构**：
```
输入(784) → Linear → ReLU → Linear(10) → 输出
```

**本实验首次引入**：
- `nn.Linear`：全连接层
- `nn.CrossEntropyLoss`：交叉熵损失
- `torch.optim.Adam`：Adam优化器
- 前向传播、反向传播概念

### 💡 基础知识讲解

#### 1. 什么是全连接层（nn.Linear）？

```python
self.fc1 = nn.Linear(784, 256)  # 输入784维 → 输出256维
```

- **含义**：每一层的每个神经元都与上一层的所有神经元相连
- **参数**：
  - `in_features`：输入维度（MNIST是28×28=784）
  - `out_features`：输出维度（本层神经元数）
- **本质**：做一次线性变换 y = W·x + b

#### 2. 什么是交叉熵损失（nn.CrossEntropyLoss）？

```python
criterion = nn.CrossEntropyLoss()
```

- **作用**：衡量模型预测与真实标签的差异
- **适用**：多分类问题（MNIST是10分类）
- **注意**：内部已包含Softmax，不需要在输出层再加激活

#### 3. 什么是优化器（torch.optim.Adam）？

```python
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
```

- **作用**：根据梯度更新参数，使损失最小化
- **参数**：
  - `model.parameters()`：模型的所有参数
  - `lr=0.001`：学习率，控制更新步长
- **Adam优点**：收敛快、对学习率不那么敏感

#### 4. 什么是前向传播和反向传播？

```
前向传播：输入 → 各层计算 → 输出预测

反向传播：计算损失 → 反向求梯度 → 更新参数
```

### 💻 完整代码

```python
"""
实验1：Baseline - 纯全连接网络
==============================
这是所有实验的基准线。
网络结构：输入(784) → 隐藏层(256) → 输出(10)
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

# ========== 步骤1：数据加载 ==========
# transform：数据预处理流水线
transform = transforms.Compose([
    transforms.ToTensor(),              # 转为张量：像素[0,255] → [0,1]
    transforms.Normalize((0.5,), (0.5,))  # 标准化：均值0.5，标准差0.5
])

# 加载MNIST数据集
trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)

trainloader = DataLoader(trainset, batch_size=64, shuffle=True, num_workers=2)
testloader = DataLoader(testset, batch_size=100, shuffle=False, num_workers=2)

print(f"训练集: {len(trainset)}张 | 测试集: {len(testset)}张")

# ========== 步骤2：定义网络 ==========
class BaselineNet(nn.Module):
    """
    基准网络：纯全连接结构
    
    结构：输入(784) → 隐藏层(256) → 输出(10)
    """
    
    def __init__(self):
        super(BaselineNet, self).__init__()
        # 全连接层1：784 → 256
        self.fc1 = nn.Linear(in_features=784, out_features=256)
        # 全连接层2：256 → 10
        self.fc2 = nn.Linear(in_features=256, out_features=10)
    
    def forward(self, x):
        # 1. 展平：从 (batch, 1, 28, 28) → (batch, 784)
        x = x.view(x.size(0), -1)
        # 2. 全连接层 + ReLU激活
        x = F.relu(self.fc1(x))
        # 3. 输出层（10分类，不需要激活）
        x = self.fc2(x)
        return x

# ========== 步骤3：训练函数 ==========
def train_model(model, trainloader, testloader, num_epochs=5, lr=0.001):
    """
    训练函数
    
    参数：
        model：神经网络模型
        trainloader：训练数据
        testloader：测试数据
        num_epochs：训练轮数
        lr：学习率
    """
    # 损失函数：交叉熵（多分类）
    criterion = nn.CrossEntropyLoss()
    # 优化器：Adam
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    
    for epoch in range(num_epochs):
        # ====== 训练阶段 ======
        model.train()
        running_loss = 0.0
        correct = total = 0
        
        for images, labels in trainloader:
            # 前向传播：计算预测
            outputs = model(images)
            # 计算损失
            loss = criterion(outputs, labels)
            
            # 反向传播：计算梯度
            optimizer.zero_grad()  # 清除上一步的梯度
            loss.backward()         # 反向传播
            optimizer.step()        # 更新参数
            
            running_loss += loss.item()
            _, predicted = torch.max(outputs.data, 1)  # 取最大值索引
            total += labels.size(0)
            correct += (predicted == labels).sum().item()
        
        train_loss = running_loss / len(trainloader)
        train_acc = 100 * correct / total
        
        # ====== 测试阶段 ======
        model.eval()
        correct = total = 0
        with torch.no_grad():  # 不计算梯度（省内存加速）
            for images, labels in testloader:
                outputs = model(images)
                _, predicted = torch.max(outputs.data, 1)
                total += labels.size(0)
                correct += (predicted == labels).sum().item()
        test_acc = 100 * correct / total
        
        print(f'Epoch [{epoch+1}/{num_epochs}] Loss: {train_loss:.4f} | Train: {train_acc:.2f}% | Test: {test_acc:.2f}%')
    
    return model

# ========== 步骤4：主程序 ==========
if __name__ == '__main__':
    model = BaselineNet()
    
    # 统计参数量
    # fc1: 784×256 + 256 = 200,960
    # fc2: 256×10 + 10 = 2,570
    # 总计：203,530
    params = sum(p.numel() for p in model.parameters())
    print(f"网络参数量: {params:,}")
    
    print("\n开始训练 Baseline...")
    model = train_model(model, trainloader, testloader, num_epochs=5)
```

## 实验2：+卷积层 - 引入卷积特征提取

### 📖 实验说明

**实验目标**：在实验1基础上添加卷积层和池化层，理解卷积如何提取图像特征

**网络结构**：
```
输入(1,28,28) → Conv→ReLU→Pool → Conv→ReLU→Pool → Flatten → FC → FC → 输出
```

**本实验新增模块**：
- `nn.Conv2d`：卷积层，提取空间特征
- `nn.MaxPool2d`：池化层，降维防过拟合
- `F.relu`：ReLU激活函数

**对比重点**：Conv+Pool vs 纯FC，参数量和精度的变化

#### 1. 模型加载（torch.load）

```python
model.load_state_dict(torch.load('model.pth'))
```

- **含义**：加载保存的模型参数
- **注意**：模型结构必须与保存时一致

```python
"""
实验7：模型加载与预测可视化
==============================
加载各实验训练的模型，对随机图片进行预测并可视化
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader
import matplotlib.pyplot as plt
import numpy as np
import random

# ========== 可修改配置 ==========
MODEL_PATH = 'exp5_final.pth'  # 修改这里选择要加载的模型
NUM_IMAGES = 10  # 显示的图片数量
# ================================

# ========== 数据加载 ==========
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)

# ========== 根据模型路径选择网络结构 ==========
def get_model_by_path(model_path):
    """根据模型路径返回对应的网络类"""
    
    if 'exp1' in model_path:
        # 实验1：纯全连接网络
        class BaselineNet(nn.Module):
            def __init__(self):
                super(BaselineNet, self).__init__()
                self.fc1 = nn.Linear(784, 256)
                self.fc2 = nn.Linear(256, 10)
            def forward(self, x):
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                return self.fc2(x)
        return BaselineNet()
    
    elif 'exp2' in model_path:
        # 实验2：卷积网络（Conv+Pool，无BN和Dropout）
        class ConvNet(nn.Module):
            def __init__(self):
                super(ConvNet, self).__init__()
                self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
                self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
                self.pool = nn.MaxPool2d(2, 2)
                self.fc1 = nn.Linear(64 * 7 * 7, 128)
                self.fc2 = nn.Linear(128, 10)
            def forward(self, x):
                x = self.pool(F.relu(self.conv1(x)))
                x = self.pool(F.relu(self.conv2(x)))
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                return self.fc2(x)
        return ConvNet()
    
    elif 'exp3' in model_path:
        # 实验3：卷积+BatchNorm（无Dropout）
        class ConvNetBN(nn.Module):
            def __init__(self):
                super(ConvNetBN, self).__init__()
                self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
                self.bn1 = nn.BatchNorm2d(32)
                self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
                self.bn2 = nn.BatchNorm2d(64)
                self.pool = nn.MaxPool2d(2, 2)
                self.fc1 = nn.Linear(64 * 7 * 7, 128)
                self.fc2 = nn.Linear(128, 10)
            def forward(self, x):
                x = self.pool(F.relu(self.bn1(self.conv1(x))))
                x = self.pool(F.relu(self.bn2(self.conv2(x))))
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                return self.fc2(x)
        return ConvNetBN()
    
    else:
        # 实验4/5：卷积+BatchNorm+Dropout
        class ConvNetBNDropout(nn.Module):
            def __init__(self):
                super(ConvNetBNDropout, self).__init__()
                self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
                self.bn1 = nn.BatchNorm2d(32)
                self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
                self.bn2 = nn.BatchNorm2d(64)
                self.pool = nn.MaxPool2d(2, 2)
                self.dropout = nn.Dropout(p=0.5)
                self.fc1 = nn.Linear(64 * 7 * 7, 128)
                self.fc2 = nn.Linear(128, 10)
            def forward(self, x):
                x = self.pool(F.relu(self.bn1(self.conv1(x))))
                x = self.pool(F.relu(self.bn2(self.conv2(x))))
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                x = self.dropout(x)
                return self.fc2(x)
        return ConvNetBNDropout()

# ========== 加载模型 ==========
def load_model(model_path):
    """加载模型权重"""
    model = get_model_by_path(model_path)
    if model is None:
        print(f"❌ 无法识别模型类型: {model_path}")
        return None
    
    try:
        state_dict = torch.load(model_path, map_location='cpu')
        model.load_state_dict(state_dict)
        print(f"✅ 成功加载模型: {model_path}")
        print(f"   模型结构: {model.__class__.__name__}")
    except FileNotFoundError:
        print(f"❌ 文件不存在: {model_path}")
        print(f"   请先运行对应实验生成模型权重文件")
        return None
    except Exception as e:
        print(f"❌ 加载失败: {e}")
        return None
    
    model.eval()
    return model

# ========== 预测并可视化 ==========
def predict_and_visualize(model, dataset, num_images=10):
    """随机获取图片，预测并可视化"""
    if model is None:
        return
    
    # 随机选择图片索引
    random_indices = random.sample(range(len(dataset)), num_images)
    
    # 创建画布：2行5列显示10张图片
    fig, axes = plt.subplots(2, 5, figsize=(12, 5))
    axes = axes.flatten()
    
    for idx, ax in zip(random_indices, axes):
        # 获取图片和标签
        image, true_label = dataset[idx]
        
        # 预测（需要扩展batch维度）
        with torch.no_grad():
            image_batch = image.unsqueeze(0)  # (1, 1, 28, 28)
            output = model(image_batch)
            probs = torch.softmax(output, dim=1)  # 转概率
            pred_label = torch.argmax(probs, dim=1).item()
            confidence = probs[0][pred_label].item() * 100
        
        # 反归一化用于显示
        img_display = image.squeeze().numpy() * 0.5 + 0.5  # (28, 28)
        
        # 显示图片
        ax.imshow(img_display, cmap='gray')
        
        # 设置标题（正确绿色，错误红色）
        color = 'green' if pred_label == true_label else 'red'
        ax.set_title(f'真: {true_label} | 预: {pred_label}\n置信度: {confidence:.1f}%', 
                    color=color, fontsize=10)
        ax.axis('off')
    
    plt.suptitle(f'模型预测结果: {MODEL_PATH}', fontsize=14)
    plt.tight_layout()
    plt.savefig('prediction_result.png', dpi=150)  # 保存结果
    print(f"📊 预测结果已保存到: prediction_result.png")
    plt.show()

# ========== 主程序 ==========
if __name__ == '__main__':
    print(f"加载模型: {MODEL_PATH}")
    print(f"显示图片数: {NUM_IMAGES}")
    print("=" * 50)
    
    # 加载模型
    model = load_model(MODEL_PATH)
    
    # 预测并可视化
    predict_and_visualize(model, testset, NUM_IMAGES)
    
    print("\n" + "=" * 50)
    print("💡 可修改 MODEL_PATH 加载不同实验的模型：")
    print("   - exp1_baseline.pth（纯FC网络）")
    print("   - exp2_convnet.pth（卷积网络）")
    print("   - exp3_convnet_bn.pth（+BatchNorm）")
    print("   - exp5_final.pth（完整网络）")
```

## 附录：GPU加速训练

### 📖 GPU vs CPU

深度学习训练需要大量矩阵运算，GPU比CPU有更多核心，更适合并行计算。

| 设备 | 优点 | 缺点 |
|------|------|------|
| CPU | 兼容性好，调试方便 | 并行度低，速度慢 |
| GPU | 并行计算强，速度快10-100倍 | 需要NVIDIA显卡 |

### 💻 GPU代码添加方法

#### 1. 在文件开头添加设备检测

```python
# ========== GPU设置 ==========
# 检测是否有可用的GPU
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
print(f"使用设备: {device}")

# 如果显示 "使用设备: cpu" 说明没有GPU或CUDA配置有问题
# 如果显示 "使用设备: cuda" 说明可以使用GPU加速
```

#### 2. 将模型移到GPU

```python
if __name__ == '__main__':
    model = BaselineNet()
    model = model.to(device)  # 将模型参数移到GPU
```

#### 3. 将数据移到GPU（训练函数中）

```python
def train_model(model, trainloader, testloader, num_epochs=5, lr=0.001):
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    
    for epoch in range(num_epochs):
        model.train()
        running_loss = 0.0
        correct = total = 0
        
        for images, labels in trainloader:
            # ===== 新增：将数据移到GPU =====
            images, labels = images.to(device), labels.to(device)
            # ==================================
            
            outputs = model(images)
            loss = criterion(outputs, labels)
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            # ...
```

### ⚠️ 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `cuda` not available | 驱动太旧 | 更新NVIDIA驱动或用CPU版本 |
| `Expected all tensors to be on the same device` | 数据和模型不在同一设备 | 确保都调用 `.to(device)` |
| 显存不足 | Batch size太大 | 减小batch_size或用CPU |

### 📝 完整的主程序模板（带GPU）

```python
if __name__ == '__main__':
    # GPU设置
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    print(f"使用设备: {device}")
    
    # 创建模型并移到GPU
    model = BaselineNet().to(device)
    print(f"网络参数量: {sum(p.numel() for p in model.parameters()):,}")
    
    # 开始训练
    print("\n开始训练...")
    model = train_model(model, trainloader, testloader, num_epochs=5)
```
