## 实验1：Baseline - 纯全连接网络

### 📖 实验说明

**实验目标**：建立纯全连接网络的基准线，理解深度学习训练的基本流程

**网络结构**：
```
输入(784) → Linear → ReLU → Linear(10) → 输出
```

**本实验首次引入**：
- `nn.Linear`：全连接层
- `nn.CrossEntropyLoss`：交叉熵损失
- `torch.optim.Adam`：Adam优化器
- 前向传播、反向传播概念

### 💡 基础知识讲解

#### 1. 什么是全连接层（nn.Linear）？

```python
self.fc1 = nn.Linear(784, 256)  # 输入784维 → 输出256维
```

- **含义**：每一层的每个神经元都与上一层的所有神经元相连
- **参数**：
  - `in_features`：输入维度（MNIST是28×28=784）
  - `out_features`：输出维度（本层神经元数）
- **本质**：做一次线性变换 y = W·x + b

#### 2. 什么是交叉熵损失（nn.CrossEntropyLoss）？

```python
criterion = nn.CrossEntropyLoss()
```

- **作用**：衡量模型预测与真实标签的差异
- **适用**：多分类问题（MNIST是10分类）
- **注意**：内部已包含Softmax，不需要在输出层再加激活

#### 3. 什么是优化器（torch.optim.Adam）？

```python
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
```

- **作用**：根据梯度更新参数，使损失最小化
- **参数**：
  - `model.parameters()`：模型的所有参数
  - `lr=0.001`：学习率，控制更新步长
- **Adam优点**：收敛快、对学习率不那么敏感

#### 4. 什么是前向传播和反向传播？

```
前向传播：输入 → 各层计算 → 输出预测

反向传播：计算损失 → 反向求梯度 → 更新参数
```

### 💻 完整代码

```python
"""
实验1：Baseline - 纯全连接网络
==============================
这是所有实验的基准线。
网络结构：输入(784) → 隐藏层(256) → 输出(10)
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

# ========== 步骤1：数据加载 ==========
# transform：数据预处理流水线
transform = transforms.Compose([
    transforms.ToTensor(),              # 转为张量：像素[0,255] → [0,1]
    transforms.Normalize((0.5,), (0.5,))  # 标准化：均值0.5，标准差0.5
])

# 加载MNIST数据集
trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)

trainloader = DataLoader(trainset, batch_size=64, shuffle=True, num_workers=2)
testloader = DataLoader(testset, batch_size=100, shuffle=False, num_workers=2)

print(f"训练集: {len(trainset)}张 | 测试集: {len(testset)}张")

# ========== 步骤2：定义网络 ==========
class BaselineNet(nn.Module):
    """
    基准网络：纯全连接结构
    
    结构：输入(784) → 隐藏层(256) → 输出(10)
    """
    
    def __init__(self):
        super(BaselineNet, self).__init__()
        # 全连接层1：784 → 256
        self.fc1 = nn.Linear(in_features=784, out_features=256)
        # 全连接层2：256 → 10
        self.fc2 = nn.Linear(in_features=256, out_features=10)
    
    def forward(self, x):
        # 1. 展平：从 (batch, 1, 28, 28) → (batch, 784)
        x = x.view(x.size(0), -1)
        # 2. 全连接层 + ReLU激活
        x = F.relu(self.fc1(x))
        # 3. 输出层（10分类，不需要激活）
        x = self.fc2(x)
        return x

# ========== 步骤3：训练函数 ==========
def train_model(model, trainloader, testloader, num_epochs=5, lr=0.001):
    """
    训练函数
    
    参数：
        model：神经网络模型
        trainloader：训练数据
        testloader：测试数据
        num_epochs：训练轮数
        lr：学习率
    """
    # 损失函数：交叉熵（多分类）
    criterion = nn.CrossEntropyLoss()
    # 优化器：Adam
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    
    for epoch in range(num_epochs):
        # ====== 训练阶段 ======
        model.train()
        running_loss = 0.0
        correct = total = 0
        
        for images, labels in trainloader:
            # 前向传播：计算预测
            outputs = model(images)
            # 计算损失
            loss = criterion(outputs, labels)
            
            # 反向传播：计算梯度
            optimizer.zero_grad()  # 清除上一步的梯度
            loss.backward()         # 反向传播
            optimizer.step()        # 更新参数
            
            running_loss += loss.item()
            _, predicted = torch.max(outputs.data, 1)  # 取最大值索引
            total += labels.size(0)
            correct += (predicted == labels).sum().item()
        
        train_loss = running_loss / len(trainloader)
        train_acc = 100 * correct / total
        
        # ====== 测试阶段 ======
        model.eval()
        correct = total = 0
        with torch.no_grad():  # 不计算梯度（省内存加速）
            for images, labels in testloader:
                outputs = model(images)
                _, predicted = torch.max(outputs.data, 1)
                total += labels.size(0)
                correct += (predicted == labels).sum().item()
        test_acc = 100 * correct / total
        
        print(f'Epoch [{epoch+1}/{num_epochs}] Loss: {train_loss:.4f} | Train: {train_acc:.2f}% | Test: {test_acc:.2f}%')
    
    return model

# ========== 步骤4：主程序 ==========
if __name__ == '__main__':
    model = BaselineNet()
    
    # 统计参数量
    # fc1: 784×256 + 256 = 200,960
    # fc2: 256×10 + 10 = 2,570
    # 总计：203,530
    params = sum(p.numel() for p in model.parameters())
    print(f"网络参数量: {params:,}")
    
    print("\n开始训练 Baseline...")
    model = train_model(model, trainloader, testloader, num_epochs=5)
```

## 实验2：+卷积层 - 引入卷积特征提取

### 📖 实验说明

**实验目标**：在实验1基础上添加卷积层和池化层，理解卷积如何提取图像特征

**网络结构**：
```
输入(1,28,28) → Conv→ReLU→Pool → Conv→ReLU→Pool → Flatten → FC → FC → 输出
```

**本实验新增模块**：
- `nn.Conv2d`：卷积层，提取空间特征
- `nn.MaxPool2d`：池化层，降维防过拟合
- `F.relu`：ReLU激活函数

**对比重点**：Conv+Pool vs 纯FC，参数量和精度的变化

### 💡 新增模块讲解

#### 1. 什么是卷积层（nn.Conv2d）？

```python
self.conv1 = nn.Conv2d(in_channels=1, out_channels=32, kernel_size=3, padding=1)
```

- **含义**：在图像上滑动3×3卷积核，提取局部特征
- **核心参数**：
  - `in_channels=1`：输入通道数（灰度图=1）
  - `out_channels=32`：输出通道数（卷积核数量）
  - `kernel_size=3`：卷积核大小（3×3）
  - `padding=1`：边缘填充1圈，保持尺寸不变
- **作用**：通过局部连接和权重共享提取空间特征

#### 2. 什么是池化层（nn.MaxPool2d）？

```python
self.pool = nn.MaxPool2d(kernel_size=2, stride=2)
```

- **含义**：2×2窗口取最大值，尺寸减半
- **核心参数**：
  - `kernel_size=2`：窗口大小
  - `stride=2`：步长2，不重叠
- **作用**：降维、减少计算量、增强平移不变性

#### 3. 形状变化计算

```
输入图片：28×28

Conv1(3×3, pad=1) → 保持28×28 → [32, 28, 28]
MaxPool(2×2) → 28÷2=14 → [32, 14, 14]
Conv2(3×3, pad=1) → 保持14×14 → [64, 14, 14]
MaxPool(2×2) → 14÷2=7 → [64, 7, 7]

展平：64×7×7 = 3136
```

### 💻 完整代码

```python
"""
实验2：+卷积层 - 引入卷积特征提取
==============================
在实验1基础上，将全连接层替换为卷积+池化结构
对比：Conv+Pool vs 纯FC
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

# ========== 数据加载（与实验1相同）==========
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)

trainloader = DataLoader(trainset, batch_size=64, shuffle=True, num_workers=2)
testloader = DataLoader(testset, batch_size=100, shuffle=False, num_workers=2)

# ========== 定义网络 ==========
class ConvNet(nn.Module):
    """
    卷积神经网络（在BaselineNet基础上改用Conv+Pool）
    
    结构：Conv→ReLU→Pool → Conv→ReLU→Pool → FC → FC
    """
    
    def __init__(self):
        super(ConvNet, self).__init__()
        # 卷积层1：1通道 → 32通道
        # in_channels=1: 灰度图输入
        # out_channels=32: 32个卷积核，输出32通道特征图
        # kernel_size=3: 3×3卷积核
        # padding=1: 边缘填充，保持28×28
        self.conv1 = nn.Conv2d(in_channels=1, out_channels=32, kernel_size=3, padding=1)
        
        # 卷积层2：32通道 → 64通道
        self.conv2 = nn.Conv2d(in_channels=32, out_channels=64, kernel_size=3, padding=1)
        
        # 池化层：2×2最大池化，尺寸减半
        self.pool = nn.MaxPool2d(kernel_size=2, stride=2)
        
        # 全连接层
        # 输入：64通道 × 7×7 = 3136
        self.fc1 = nn.Linear(in_features=64 * 7 * 7, out_features=128)
        self.fc2 = nn.Linear(in_features=128, out_features=10)
    
    def forward(self, x):
        # Conv1 → ReLU → Pool
        # 输出尺寸：28×28 → 14×14
        x = self.pool(F.relu(self.conv1(x)))
        
        # Conv2 → ReLU → Pool
        # 输出尺寸：14×14 → 7×7
        x = self.pool(F.relu(self.conv2(x)))
        
        # 展平： (batch, 64, 7, 7) → (batch, 3136)
        x = x.view(x.size(0), -1)
        
        # 全连接层
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x

# ========== 训练函数（与实验1相同）==========
def train_model(model, trainloader, testloader, num_epochs=5, lr=0.001):
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    
    for epoch in range(num_epochs):
        model.train()
        running_loss = 0.0
        correct = total = 0
        
        for images, labels in trainloader:
            outputs = model(images)
            loss = criterion(outputs, labels)
            
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            
            running_loss += loss.item()
            _, predicted = torch.max(outputs.data, 1)
            total += labels.size(0)
            correct += (predicted == labels).sum().item()
        
        train_loss = running_loss / len(trainloader)
        train_acc = 100 * correct / total
        
        model.eval()
        correct = total = 0
        with torch.no_grad():
            for images, labels in testloader:
                outputs = model(images)
                _, predicted = torch.max(outputs.data, 1)
                total += labels.size(0)
                correct += (predicted == labels).sum().item()
        test_acc = 100 * correct / total
        
        print(f'Epoch [{epoch+1}/{num_epochs}] Loss: {train_loss:.4f} | Train: {train_acc:.2f}% | Test: {test_acc:.2f}%')
    
    return model

# ========== 主程序 ==========
if __name__ == '__main__':
    model = ConvNet()
    
    # 参数量对比
    # Conv1: 1×32×3×3 + 32 = 320
    # Conv2: 32×64×3×3 + 64 = 18,496
    # FC1: 3136×128 + 128 = 401,536
    # FC2: 128×10 + 10 = 1,290
    # 总计：约 421,642
    params = sum(p.numel() for p in model.parameters())
    print(f"ConvNet参数量: {params:,}")
    print(f"对比Baseline: 203,530")
    print(f"精度预期：~97%（vs Baseline ~92%）")
    
    print("\n开始训练 ConvNet...")
    model = train_model(model, trainloader, testloader, num_epochs=5)
```

## 实验3：+BatchNorm - 批归一化加速收敛

### 📖 实验说明

**实验目标**：在实验2基础上添加BatchNorm层，理解归一化对训练稳定性的作用

**网络结构**：
```
输入 → Conv→BN→ReLU→Pool → Conv→BN→ReLU→Pool → FC → FC → 输出
```

**本实验新增模块**：
- `nn.BatchNorm2d`：批归一化层

**对比重点**：观察BatchNorm如何让训练更稳定、更快地收敛

### 💡 新增模块讲解

#### 什么是BatchNorm（nn.BatchNorm2d）？

```python
self.bn1 = nn.BatchNorm2d(32)  # 对32通道做归一化
```

- **含义**：对batch内的数据进行标准化，让每层输入分布稳定
- **核心参数**：
  - `num_features`：通道数（与上一层输出通道数相同）
- **位置**：通常放在卷积层之后、激活函数之前
- **作用**：
  1. 加速收敛（梯度更稳定）
  2. 缓解梯度消失
  3. 对初始值不那么敏感

### 💻 完整代码

```python
"""
实验3：+BatchNorm - 批归一化加速收敛
==============================
在实验2基础上添加BatchNorm层
对比：观察训练曲线是否更稳定
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

# ========== 数据加载（与实验1、2相同）==========
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)

trainloader = DataLoader(trainset, batch_size=64, shuffle=True, num_workers=2)
testloader = DataLoader(testset, batch_size=100, shuffle=False, num_workers=2)

# ========== 定义网络 ==========
class ConvNetBN(nn.Module):
    """
    卷积神经网络 + BatchNorm（在ConvNet基础上添加BN）
    
    结构：Conv→BN→ReLU→Pool → Conv→BN→ReLU→Pool → FC → FC
    """
    
    def __init__(self):
        super(ConvNetBN, self).__init__()
        # 卷积层1 + BatchNorm
        self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
        self.bn1 = nn.BatchNorm2d(32)  # 新增：对32通道归一化
        
        # 卷积层2 + BatchNorm
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
        self.bn2 = nn.BatchNorm2d(64)  # 新增：对64通道归一化
        
        self.pool = nn.MaxPool2d(2, 2)
        self.fc1 = nn.Linear(64 * 7 * 7, 128)
        self.fc2 = nn.Linear(128, 10)
    
    def forward(self, x):
        # Conv → BN → ReLU → Pool
        # 注意顺序：Conv → BN → ReLU（不是BN → Conv）
        x = self.pool(F.relu(self.bn1(self.conv1(x))))
        x = self.pool(F.relu(self.bn2(self.conv2(x))))
        x = x.view(x.size(0), -1)
        x = F.relu(self.fc1(x))
        x = self.fc2(x)
        return x

    print(f"{'='*50}")
    
    for epoch in range(num_epochs):

            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            
            running_loss += loss.item()
            _, predicted = torch.max(outputs.data, 1)
            total += labels.size(0)
            correct += (predicted == labels).sum().item()
        
        train_loss = running_loss / len(trainloader)
        train_acc = 100 * correct / total
        
        model.eval()
        correct = total = 0
        with torch.no_grad():
```

## 实验4：+Dropout - 防止过拟合

### 📖 实验说明

**实验目标**：在实验3基础上添加Dropout层，理解正则化如何防止过拟合

**网络结构**：
```
输入 → Conv→BN→ReLU→Pool → Conv→BN→ReLU→Pool → FC→Dropout → FC → 输出
```

**本实验新增模块**：
- `nn.Dropout`：随机丢弃层

**对比重点**：观察Dropout如何减少Train-Test Gap，防止过拟合

### 💡 新增模块讲解

#### 什么是Dropout（nn.Dropout）？

```python
self.dropout = nn.Dropout(p=0.5)  # 50%概率丢弃
```

- **含义**：训练时随机将一部分神经元的输出置零
- **核心参数**：
  - `p`：丢弃概率（0.5表示50%神经元被丢弃）
- **作用**：
  1. 防止过拟合（让网络不依赖特定神经元）
  2. 起到集成学习的效果
- **注意**：
  - 训练时启用，评估时自动关闭
  - 需要调用`model.train()`和`model.eval()`

### 💻 完整代码

```python
"""
实验4：+Dropout - 防止过拟合
==============================
在实验3基础上添加Dropout层
对比：观察Gap是否减小
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
import torchvision
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

# ========== 数据加载 ==========
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize((0.5,), (0.5,))
])

trainset = torchvision.datasets.MNIST(root='./data', train=True, download=True, transform=transform)
testset = torchvision.datasets.MNIST(root='./data', train=False, download=True, transform=transform)

trainloader = DataLoader(trainset, batch_size=64, shuffle=True, num_workers=2)
testloader = DataLoader(testset, batch_size=100, shuffle=False, num_workers=2)

# ========== 定义网络 ==========
class ConvNetBNDropout(nn.Module):
    """
    卷积神经网络 + BatchNorm + Dropout（在ConvNetBN基础上添加Dropout）
    
    结构：Conv→BN→ReLU→Pool → Conv→BN→ReLU→Pool → FC→Dropout → FC
    """
    
    def __init__(self, dropout_p=0.5):
        super(ConvNetBNDropout, self).__init__()
        self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
        self.bn1 = nn.BatchNorm2d(32)
        self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
        self.bn2 = nn.BatchNorm2d(64)
        self.pool = nn.MaxPool2d(2, 2)
        
        # Dropout层：放在全连接层之间
        self.dropout = nn.Dropout(p=dropout_p)
        
        self.fc1 = nn.Linear(64 * 7 * 7, 128)
        self.fc2 = nn.Linear(128, 10)
    
    def forward(self, x):
        x = self.pool(F.relu(self.bn1(self.conv1(x))))
        x = self.pool(F.relu(self.bn2(self.conv2(x))))
        x = x.view(x.size(0), -1)
        x = F.relu(self.fc1(x))
        x = self.dropout(x)  # 训练时随机丢弃
        x = self.fc2(x)
        return x

# ========== 训练函数 ==========
def train_and_record(model, trainloader, testloader, name, num_epochs=10):
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
    
    print(f"\n{'='*55}")
    print(f"训练模型: {name}")
    print(f"{'='*55}")

    for epoch in range(num_epochs):
        model.train()
        correct = total = 0
        for images, labels in trainloader:
            outputs = model(images)
            loss = criterion(outputs, labels)
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            _, predicted = torch.max(outputs.data, 1)
            total += labels.size(0)
            correct += (predicted == labels).sum().item()
        train_acc = 100 * correct / total
        
        model.eval()
        correct = total = 0
        with torch.no_grad():
            for images, labels in testloader:
                outputs = model(images)
                _, predicted = torch.max(outputs.data, 1)
                total += labels.size(0)
                correct += (predicted == labels).sum().item()
        test_acc = 100 * correct / total
        
        gap = train_acc - test_acc
        if test_acc > best_test_acc:
            best_test_acc = test_acc
        
        print(f'Epoch [{epoch+1:2d}/{num_epochs}] Train: {train_acc:.2f}% | Test: {test_acc:.2f}% | Gap: {gap:.2f}%')
    
    return best_test_acc

# ========== 主程序 ==========
if __name__ == '__main__':
```

```python
                x = self.pool(F.relu(self.conv1(x)))
                x = self.pool(F.relu(self.conv2(x)))
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                return self.fc2(x)
        return ConvNet()
    
    elif 'exp3' in model_path:
        # 实验3：卷积+BatchNorm（无Dropout）
        class ConvNetBN(nn.Module):
            def __init__(self):
                super(ConvNetBN, self).__init__()
                self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
                self.bn1 = nn.BatchNorm2d(32)
                self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
                self.bn2 = nn.BatchNorm2d(64)
                self.pool = nn.MaxPool2d(2, 2)
                self.fc1 = nn.Linear(64 * 7 * 7, 128)
                self.fc2 = nn.Linear(128, 10)
            def forward(self, x):
                x = self.pool(F.relu(self.bn1(self.conv1(x))))
                x = self.pool(F.relu(self.bn2(self.conv2(x))))
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                return self.fc2(x)
        return ConvNetBN()
    
    else:
        # 实验4/5：卷积+BatchNorm+Dropout
        class ConvNetBNDropout(nn.Module):
            def __init__(self):
                super(ConvNetBNDropout, self).__init__()
                self.conv1 = nn.Conv2d(1, 32, 3, padding=1)
                self.bn1 = nn.BatchNorm2d(32)
                self.conv2 = nn.Conv2d(32, 64, 3, padding=1)
                self.bn2 = nn.BatchNorm2d(64)
                self.pool = nn.MaxPool2d(2, 2)
                self.dropout = nn.Dropout(p=0.5)
                self.fc1 = nn.Linear(64 * 7 * 7, 128)
                self.fc2 = nn.Linear(128, 10)
            def forward(self, x):
                x = self.pool(F.relu(self.bn1(self.conv1(x))))
                x = self.pool(F.relu(self.bn2(self.conv2(x))))
                x = x.view(x.size(0), -1)
                x = F.relu(self.fc1(x))
                x = self.dropout(x)
                return self.fc2(x)
        return ConvNetBNDropout()

def load_model(model_path):

    try:

    except FileNotFoundError:

        print(f"   请先运行对应实验生成模型权重文件")
        return None
    except Exception as e:
        print(f"❌ 加载失败: {e}")
        return None
    
    model.eval()
    return model

def predict_and_visualize(model, dataset, num_images=10):
    if model is None:
        return
    
    random_indices = random.sample(range(len(dataset)), num_images)
    fig, axes = plt.subplots(2, 5, figsize=(12, 5))
    axes = axes.flatten()
    
    for idx, ax in zip(random_indices, axes):
        image, true_label = dataset[idx]
        
        with torch.no_grad():
            image_batch = image.unsqueeze(0)
            output = model(image_batch)
            probs = torch.softmax(output, dim=1)
            pred_label = torch.argmax(probs, dim=1).item()
            confidence = probs[0][pred_label].item() * 100
        
        img_display = image.squeeze().numpy() * 0.5 + 0.5
        
        ax.imshow(img_display, cmap='gray')
        color = 'green' if pred_label == true_label else 'red'
        ax.set_title(f'真: {true_label} | 预: {pred_label}\n置信度: {confidence:.1f}%', 
                    color=color, fontsize=10)
        ax.axis('off')
    
    plt.suptitle(f'模型预测结果: {MODEL_PATH}', fontsize=14)
    plt.tight_layout()
    plt.savefig('prediction_result.png', dpi=150)
```

## 附录：GPU加速训练

### 📖 GPU vs CPU

深度学习训练需要大量矩阵运算，GPU比CPU有更多核心，更适合并行计算。

| 设备 | 优点 | 缺点 |
|------|------|------|
| CPU | 兼容性好，调试方便 | 并行度低，速度慢 |
| GPU | 并行计算强，速度快10-100倍 | 需要NVIDIA显卡 |

### 💻 GPU代码添加方法

#### 1. 在文件开头添加设备检测

```python
# ========== GPU设置 ==========
# 检测是否有可用的GPU
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
print(f"使用设备: {device}")

# 如果显示 "使用设备: cpu" 说明没有GPU或CUDA配置有问题
# 如果显示 "使用设备: cuda" 说明可以使用GPU加速
```

#### 2. 将模型移到GPU

```python
if __name__ == '__main__':
    model = BaselineNet()
    model = model.to(device)  # 将模型参数移到GPU
```

#### 3. 将数据移到GPU（训练函数中）

```python
def train_model(model, trainloader, testloader, num_epochs=5, lr=0.001):
    criterion = nn.CrossEntropyLoss()
    optimizer = torch.optim.Adam(model.parameters(), lr=lr)
    
    for epoch in range(num_epochs):
        model.train()
        running_loss = 0.0
        correct = total = 0
        
        for images, labels in trainloader:
            # ===== 新增：将数据移到GPU =====
            images, labels = images.to(device), labels.to(device)
            # ==================================
            
            outputs = model(images)
            loss = criterion(outputs, labels)
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            # ...
```

### ⚠️ 常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `cuda` not available | 驱动太旧 | 更新NVIDIA驱动或用CPU版本 |
| `Expected all tensors to be on the same device` | 数据和模型不在同一设备 | 确保都调用 `.to(device)` |
| 显存不足 | Batch size太大 | 减小batch_size或用CPU |

### 📝 完整的主程序模板（带GPU）

```python
if __name__ == '__main__':
    # GPU设置

    model = BaselineNet().to(device)
    print(f"网络参数量: {sum(p.numel() for p in model.parameters()):,}")
```
