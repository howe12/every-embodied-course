## 2. 理论部分

### 2.1 拉格朗日法原理

**拉格朗日函数 $L$** 定义为系统总动能减去总势能：
$$
L(q, \dot{q}) = K(q, \dot{q}) - P(q)
$$

**欧拉-拉格朗日方程**：
$$
\frac{d}{dt} \left( \frac{\partial L}{\partial \dot{q}_i} \right) - \frac{\partial L}{\partial q_i} = \tau_i
$$

这个方程描述了：能量如何随关节变化 → 产生力矩。

> **生活类比**：拉格朗日法就像"能量会计"——记录系统的动能和势能，然后推导出力矩。

### 2.2 动力学标准方程

经过推导，机器人动力学方程可写为标准形式：
$$
M(q)\ddot{q} + C(q, \dot{q})\dot{q} + G(q) = \tau
$$

| 符号 | 名称 | 物理含义 |
|------|------|----------|
| $M(q)$ | **惯性矩阵** | 质量/惯量对运动的阻碍 |
| $C(q, \dot{q})$ | **科里奥利 & 离心力项** | 运动速度导致的附加力 |
| $G(q)$ | **重力项** | 重力对关节的拉力 |
| $\tau$ | **关节力矩** | 电机提供的驱动力 |

### 2.3 各矩阵的物理意义

#### 2.3.1 惯性矩阵 $M(q)$

- 是 $n \times n$ 的对称正定矩阵
- 对角元素：各关节自身的惯性
- 非对角元素：关节之间的耦合惯性（动一个关节会影响其他关节）

**性质**：$M(q)$ 始终是正定的，即 $x^T M x > 0$ 对任何非零向量 $x$ 成立。

#### 2.3.2 科里奥利项 $C(q, \dot{q})$

包含两种力：
- **科里奥利力**：物体在旋转参考系中运动时受到的力
- **离心力**：物体在旋转参考系中静止时受到的力

> **生活类比**：当你站在旋转木马上走动时，会感受到"被向外推"的感觉——这就是科里奥利力和离心力！

#### 2.3.3 重力项 $G(q)$

- 只与关节位置 $q$ 有关
- 与速度 $\dot{q}$ 无关
- 在水平面上工作的机器人 $G(q) = 0$

## 3. 代码实现

### 3.1 单摆动力学推导（SymPy）

```python
import sympy as sp

def pendulum_dynamics():
    """
    使用拉格朗日法推导单摆动力学方程
    """
    # 1. 定义符号
    t = sp.symbols('t')
    m, l, g = sp.symbols('m l g', real=True, positive=True)
    theta = sp.Function('theta')(t)  # 角度随时间变化

    # 速度 = 角速度 = dθ/dt
    theta_dot = theta.diff(t)
    # 加速度 = d²θ/dt²
    theta_ddot = theta_dot.diff(t)

    # 2. 计算能量
    # 线速度 v = l * ω
    v = l * theta_dot
    # 动能 K = 1/2 * m * v²
    K = sp.Rational(1, 2) * m * v**2
    
    # 势能 P = m * g * h
    # 假设最低点 h=0，最高点 h=2l
    # 以转轴为原点，y轴向上，则 h = -l*cos(θ)
    h = -l * sp.cos(theta)
    P = m * g * h

    # 3. 拉格朗日函数 L = K - P
    L = K - P

    # 4. 欧拉-拉格朗日方程
    # d/dt(∂L/∂θ̇) - ∂L/∂θ = τ
    dL_dthetadot = sp.diff(L, theta_dot)
    term1 = sp.diff(dL_dthetadot, t)
    term2 = sp.diff(L, theta)
    
    # 如果没有驱动力，τ = 0
    tau = term1 - term2

    # 5. 简化结果
    tau = sp.simplify(tau)

    print("=" * 50)
    print("单摆动力学方程 (符号推导)")
    print("=" * 50)
    print("\n拉格朗日函数 L:")
    sp.pprint(L)
    print("\n力矩方程 τ:")
    sp.pprint(tau)
    
    # 6. 整理为标准形式
    # τ = m*l²*θ̈ + m*g*l*sin(θ) = 0
    print("\n整理后的方程:")
    print("  m * l² * θ̈ + m * g * l * sin(θ) = τ")
    print("\n因此:")
    print(f"  惯性项系数 M = {sp.simplify(tau.coeff(theta_ddot))}")
    print(f"  重力项系数 G = {sp.simplify(-tau.coeff(theta_ddot).subs(theta_ddot, 0))}")


if __name__ == "__main__":
    pendulum_dynamics()
```

### 3.2 2R 机械臂动力学推导

```python
import sympy as sp
import numpy as np

def robot_2r_dynamics():
    """
    使用拉格朗日法推导 2R 机械臂动力学方程
    """
    t = sp.symbols('t')
    
    # 符号定义
    m1, m2 = sp.symbols('m1 m2', positive=True)
    l1, l2 = sp.symbols('l1 l2', positive=True)
    l1c, l2c = sp.symbols('l1c l2c', positive=True)  # 质心距离
    I1, I2 = sp.symbols('I1 I2', positive=True)     # 转动惯量
    g = sp.symbols('g', positive=True)
    
    # 关节变量
    theta1 = sp.Function('theta1')(t)
    theta2 = sp.Function('theta2')(t)
    theta1_dot = theta1.diff(t)
    theta2_dot = theta2.diff(t)
    
    # 简化符号
    c1 = sp.cos(theta1)
    c2 = sp.cos(theta2)
    c12 = sp.cos(theta1 + theta2)
    s1 = sp.sin(theta1)
    s2 = sp.sin(theta2)
    s12 = sp.sin(theta1 + theta2)
    
    # ===== 连杆 1 动能 =====
    # 平动动能 + 转动动能
    v1_sq = l1c**2 * theta1_dot**2
    K1 = sp.Rational(1, 2) * m1 * v1_sq + sp.Rational(1, 2) * I1 * theta1_dot**2
    
    # ===== 连杆 2 动能 =====
    # 连杆2末端位置对时间的导数较复杂，使用简化的耦合形式
    # 2R机械臂的完整动能表达式
    K2 = sp.Rational(1, 2) * m2 * (
        l1**2 * theta1_dot**2 + 
        l2**2 * (theta1_dot + theta2_dot)**2 + 
        2 * l1 * l2 * theta1_dot * (theta1_dot + theta2_dot) * sp.cos(theta2)
    ) + sp.Rational(1, 2) * I2 * (theta1_dot + theta2_dot)**2
    
    # 总动能
    K = sp.simplify(K1 + K2)
    
    # ===== 势能 =====
    # 以基座为势能零点，y轴向上
    P1 = m1 * g * l1c * s1
    P2 = m2 * g * (l1 * s1 + l2 * s12)
    P = sp.simplify(P1 + P2)
    
    # ===== 拉格朗日函数 =====
    L = K - P
    
    print("=" * 60)
    print("2R 机械臂动力学方程")
    print("=" * 60)
    
    # 动能表达式
    print("\n【总动能 K】")
    sp.pprint(K)
    
    print("\n【总势能 P】")
    sp.pprint(P)
    
    # 对 theta1 求导
    dL_dq1 = sp.diff(L, theta1)
    dL_dq1_dot = sp.diff(L, theta1_dot)
    ddt_dL_dq1_dot = dL_dq1_dot.diff(t)
    tau1 = sp.simplify(ddt_dL_dq1_dot - dL_dq1)
    
    # 对 theta2 求导
    dL_dq2 = sp.diff(L, theta2)
    dL_dq2_dot = sp.diff(L, theta2_dot)
    ddt_dL_dq2_dot = dL_dq2_dot.diff(t)
    tau2 = sp.simplify(ddt_dL_dq2_dot - dL_dq2)
    
    print("\n【关节1力矩 τ₁】")
    sp.pprint(tau1)
    
    print("\n【关节2力矩 τ₂】")
    sp.pprint(tau2)
    
    print("\n" + "=" * 60)
    print("简化形式（提取 M, C, G 矩阵元素）")
    print("=" * 60)
    
    # 提取系数
    q1_ddot = theta1.diff(t).diff(t)
    q2_ddot = theta2.diff(t).diff(t)
    
    # tau1 中的加速度系数 -> M11, M12
    M11 = sp.simplify(tau1.coeff(q1_ddot))
    M12 = sp.simplify(tau1.coeff(q2_ddot))
    
    # tau2 中的加速度系数 -> M21, M22
    M21 = sp.simplify(tau2.coeff(q1_ddot))
    M22 = sp.simplify(tau2.coeff(q2_ddot))
    
    print(f"\n惯性矩阵 M:")
    print(f"  M11 = {M11}")
    print(f"  M12 = {M12}")
    print(f"  M21 = {M21}")
    print(f"  M22 = {M22}")


if __name__ == "__main__":
    robot_2r_dynamics()
```

## 5. 练习题

### 5.1 基础题

1. **概念理解**：解释为什么惯性矩阵 $M(q)$ 必须是正定的。
2. **单摆分析**：对于单摆 $ml^2\ddot{\theta} + mgl\sin(\theta) = 0$，当 $\theta$ 很小时近似为什么？
3. **重力项**：在太空（无重力）环境中，动力学方程会有什么变化？

### 5.2 进阶题

1. **倒立摆**：推导倒立摆的动力学方程。
2. **添加阻尼**：在单摆动力学方程中添加阻尼项 $-b\dot{\theta}$。
3. **数值验证**：使用数值积分验证 SymPy 推导的动力学方程。

### 5.3 答案

**基础题 2**：
当 $\theta$ 很小时，$\sin(\theta) \approx \theta$：
$$
ml^2\ddot{\theta} + mgl\theta = 0
$$
或
$$
\ddot{\theta} + \frac{g}{l}\theta = 0
$$
这就是简谐运动方程！

**基础题 3**：
无重力时 $G(q) = 0$，方程简化为：
$$
M(q)\ddot{q} + C(q, \dot{q})\dot{q} = \tau
$$

## 6. 本章小结

| 概念 | 说明 |
|------|------|
| **拉格朗日法** | 基于能量守恒：$L = K - P$，$\frac{d}{dt}(\frac{\partial L}{\partial \dot{q}}) - \frac{\partial L}{\partial q} = \tau$ |
| **动力学方程** | $M(q)\ddot{q} + C(q,\dot{q})\dot{q} + G(q) = \tau$ |
| **惯性矩阵 $M$** | $n \times n$ 对称正定矩阵，描述质量/惯量分布 |
| **科里奥利项 $C$** | 与速度相关的非线性项，包括离心力和科里奥利力 |
| **重力项 $G$** | 仅与位置有关的保守力 |
| **正动力学** | 已知 $\tau$ → 求运动轨迹 |
| **逆动力学** | 已知运动 → 求 $\tau$（用于控制） |
