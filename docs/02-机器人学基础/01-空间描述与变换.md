### 1.1 什么是空间描述

想象一下，你的朋友问你：**"你的手机在哪里？"**

你会怎么回答？

| 回答方式 | 说明 |
|----------|------|
| "在我口袋里" | 相对于自己的身体 |
| "在桌子上" | 相对于房间的物体 |
| "在我右手边1米处" | 相对于自己的相对位置+距离 |

**机器人也是一样的道理**——它需要知道：
- 自己在哪里（自定位）
- 目标物体在哪里（目标定位）
- 怎么从A点到B点（路径规划）

## 2. 理论部分

### 2.1 位置描述

#### 2.1.1 点与向量

| 概念 | 说明 |
|------|------|
| **点** | 空间中的基本元素，没有长度，没有体积 |
| **向量** | 从某点指向另一点的"箭头"，表示方向和大小 |

> **重要**：不要把"向量"和"坐标"混淆！只有在指定了坐标系后，才能谈论向量在该坐标系下的坐标。

#### 2.1.2 坐标表示

一旦建立起坐标系后，我们能用一个 **3×1 向量**来描述点：

$$^AP = \begin{bmatrix} p_x \\ p_y \\ p_z \end{bmatrix}$$

其中三个元素代表向量 $P$ 在坐标系A三个轴上的**投影**。

```
        Z (上)
        |
        |
        |
        +------------> Y (前)
       /
      /
     /
    X (右)
```

这是**右手坐标系**——记住伸出右手：
- 拇指 → X轴
- 食指 → Y轴
- 中指 → Z轴

### 2.2 姿态描述（方向）

#### 什么是姿态？

光知道位置不够，机器人还需要知道**朝向**——就像你知道手机在桌上，但手机是平放还是竖着放？

#### 2.2.1 旋转矩阵

在物体上固定一个坐标系，给出此坐标系相对于参考坐标系的描述，这就是**姿态**。

设坐标系B相对于坐标系A的旋转矩阵为 ${^A_B}R$：

$${^A_B}R = \begin{bmatrix} ^Ax_B & ^Ay_B & ^Az_B \end{bmatrix} = \begin{bmatrix} r_{11} & r_{12} & r_{13} \\ r_{21} & r_{22} & r_{23} \\ r_{31} & r_{32} & r_{33} \end{bmatrix}$$

其中每列是B坐标系主轴方向单位向量在A坐标系中的表达。

**绕单个轴旋转的基本旋转矩阵**：

**绕Z轴旋转 $\theta$ 角：**

$$R_z(\theta) = \begin{bmatrix} \cos\theta & -\sin\theta & 0 \\ \sin\theta & \cos\theta & 0 \\ 0 & 0 & 1 \end{bmatrix}$$

**绕X轴旋转 $\theta$ 角：**

$$R_x(\theta) = \begin{bmatrix} 1 & 0 & 0 \\ 0 & \cos\theta & -\sin\theta \\ 0 & \sin\theta & \cos\theta \end{bmatrix}$$

**绕Y轴旋转 $\theta$ 角：**

$$R_y(\theta) = \begin{bmatrix} \cos\theta & 0 & \sin\theta \\ 0 & 1 & 0 \\ -\sin\theta & 0 & \cos\theta \end{bmatrix}$$

> **注意**：这里用的是**弧度制**，不是角度制！
> - $360^\circ = 2\pi$ 弧度
> - $180^\circ = \pi$ 弧度
> - $90^\circ = \pi/2$ 弧度
> - $45^\circ = \pi/4$ 弧度

#### 2.2.2 旋转矩阵的性质

| 性质 | 公式 | 说明 |
|------|------|------|
| **正交矩阵** | $R^T R = I$ | 行向量和列向量都是单位向量且相互正交 |
| **行列式** | $\det(R) = 1$ | 保持坐标系手性 |
| **逆矩阵** | $R^{-1} = R^T$ | 转置等于逆矩阵 |

这意味着：$$R^{-1} = R^T = \begin{bmatrix} r_{11} & r_{21} & r_{31} \\ r_{12} & r_{22} & r_{32} \\ r_{13} & r_{23} & r_{33} \end{bmatrix}$$

### 2.3 位姿 = 位置 + 姿态

**位姿**就是同时描述位置和方向：

$$\text{位姿} = \begin{bmatrix} \text{位置} \\ \text{姿态} \end{bmatrix}$$

举例：告诉你"手机在桌子右边20cm处，屏幕朝上"
- 位置：右边20cm
- 姿态：屏幕朝上（这个就是方向）

## 3. 坐标变换（映射）

### 3.1 坐标平移

当坐标系A与坐标系B的姿态相同，只有原点平移时：

$${^A}P = {^B}P + {^A_B}p$$

其中 ${^A_B}p$ 是B坐标系原点相对于A坐标系的位置向量。

### 3.2 坐标旋转

当两个坐标系原点重合，但姿态不同时：

$${^A}P = {^A_B}R \cdot {^B}P$$

### 3.3 一般变换（重点！）

当坐标系原点不重合且姿态不同时（最常见的情况）：

$${^A}P = {^A_B}R \cdot {^B}P + {^A_B}p$$

或者写成齐次形式：

$${^A}P = {^A_B}T \cdot {^B}P$$

其中齐次变换矩阵：

$$T = \begin{bmatrix} R & p \\ 0 & 1 \end{bmatrix} = \begin{bmatrix} r_{11} & r_{12} & r_{13} & p_x \\ r_{21} & r_{22} & r_{23} & p_y \\ r_{31} & r_{32} & r_{33} & p_z \\ 0 & 0 & 0 & 1 \end{bmatrix}$$

### 3.4 复合变换

已知坐标系C相对于B，B相对于A，求C相对于A：

$${^A_B}T \cdot {^B_C}T = {^A_C}T$$

变换顺序很重要：$T_A = T_1 \cdot T_2 \neq T_2 \cdot T_1$

### 3.5 逆变换

已知 ${^A_B}T$，求 ${^B_A}T$：

$${^B_A}T = \begin{bmatrix} R^T & -R^T \cdot p \\ 0 & 1 \end{bmatrix}$$

## 4. 欧拉角

### 4.1 什么是欧拉角？

旋转矩阵和旋转向量虽然能描述旋转，但对人类不直观。**欧拉角**用**三个角度**表示旋转，更容易理解。

### 4.2 RPY角（最常用）

| 角度 | 绕轴 | 运动 |
|------|------|------|
| **偏航角(Yaw)** | Z轴 | 左右转向 |
| **俯仰角(Pitch)** | Y轴 | 上下俯仰 |
| **翻滚角(Roll)** | X轴 | 前后倾斜 |

旋转顺序：Z → Y → X

$$R = R_z(\text{yaw}) \cdot R_y(\text{pitch}) \cdot R_x(\text{roll})$$

### 4.3 万向锁问题（重要！）

**万向锁(Gimbal Lock)**：当俯仰角为 $\pm 90^\circ$ 时，第一次旋转和第三次旋转使用同一个轴，系统**丢失一个自由度**。

这是欧拉角的**奇异性问题**——用3个实数表示3维旋转必然存在。

> **注意**：由于这个原因，欧拉角不适用于插值和迭代，通常只用于人机交互。

---

## 5. 坐标变换注意事项

| 注意事项 | 说明 | 常见错误 |
|----------|------|----------|
| **注意坐标顺序** | 先旋转后平移，和先平移后旋转结果不同 | 搞混顺序 |
| **注意参考坐标系** | 一定要明确"相对于谁" | 没弄清楚参考系 |
| **注意单位** | 通常用米，不是厘米 | 忘记转换单位 |
| **注意角度制式** | 程序中通常用弧度制，不是角度制 | 用错单位 |
| **注意矩阵乘法顺序** | $A \cdot B \neq B \cdot A$ | 顺序颠倒 |

## 6. 实践部分

### 6.1 实践目标

用Python编写一个**坐标变换计算器**，实现：
1. 计算绕各轴旋转后的坐标
2. 构建齐次变换矩阵
3. 计算复合变换

### 6.2 程序

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
坐标变换计算器
功能：计算刚体在空间中的位姿变换
"""

import numpy as np


class CoordinateTransform:
    """坐标变换计算器"""
    
    def __init__(self):
        print("坐标变换计算器已创建")
    
    def rotation_matrix_x(self, theta):
        """绕X轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [1, 0, 0],
            [0, c, -s],
            [0, s, c]
        ])
    
    def rotation_matrix_y(self, theta):
        """绕Y轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [c, 0, s],
            [0, 1, 0],
            [-s, 0, c]
        ])
    
    def rotation_matrix_z(self, theta):
        """绕Z轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [c, -s, 0],
            [s, c, 0],
            [0, 0, 1]
        ])
    
    def rpy_to_rotation(self, roll, pitch, yaw):
        """RPY角转旋转矩阵 (Z-Y-X顺序)"""
        R_x = self.rotation_matrix_x(roll)
        R_y = self.rotation_matrix_y(pitch)
        R_z = self.rotation_matrix_z(yaw)
        return R_z @ R_y @ R_x
    
    def homogeneous_transform(self, R, p):
        """构建齐次变换矩阵"""
        T = np.eye(4)
        T[0:3, 0:3] = R
        T[0:3, 3] = p
        return T
    
    def transform_point(self, T, point):
        """变换点的坐标"""
        point_h = np.array([point[0], point[1], point[2], 1])
        new_point_h = T @ point_h
        return new_point_h[0:3]
    
    def inverse_transform(self, T):
        """求逆变换"""
        R = T[0:3, 0:3]
        p = T[0:3, 3]
        T_inv = np.eye(4)
        T_inv[0:3, 0:3] = R.T
        T_inv[0:3, 3] = -R.T @ p
        return T_inv


def demo():
    """演示"""
    t = CoordinateTransform()
    
    # 1. 基本旋转
    print("\n=== 演示1: 绕Z轴旋转90度 ===")
    point = np.array([1.0, 0.0, 0.0])
    R = t.rotation_matrix_z(np.pi / 2)
    new_point = R @ point
    print(f"原始点: {point}")
    print(f"旋转后: {new_point}")
    
    # 2. 齐次变换
    print("\n=== 演示2: 齐次变换矩阵 ===")
    R = t.rotation_matrix_z(np.pi / 6)  # 30度
    p = np.array([1.0, 0.0, 0.0])
    T = t.homogeneous_transform(R, p)
    print("齐次变换矩阵:")
    print(np.round(T, 3))
    
    # 3. RPY角
    print("\n=== 演示3: RPY角转旋转矩阵 ===")
    roll = np.radians(10)
    pitch = np.radians(20)
    yaw = np.radians(30)
    R = t.rpy_to_rotation(roll, pitch, yaw)
    print(f"Roll={np.degrees(roll):.1f}°, Pitch={np.degrees(pitch):.1f}°, Yaw={np.degrees(yaw):.1f}°")
    print("旋转矩阵:")
    print(np.round(R, 3))
    
    # 4. 逆变换
    print("\n=== 演示4: 逆变换 ===")
    T_inv = t.inverse_transform(T)
    print("逆变换矩阵:")
    print(np.round(T_inv, 3))


if __name__ == '__main__':
    demo()
```

```bash
python3 coordinate_transform.py
```

## 本章小结

| 概念 | 说明 |
|------|------|
| 向量 | 有方向和大小的量，需要坐标系来描述坐标 |
| 旋转矩阵 | 3×3正交矩阵，$R^{-1} = R^T$ |
| 齐次变换 | 4×4矩阵，统一表示旋转和平移 |
| 欧拉角 | yaw-pitch-roll，易理解但存在万向锁 |
| 变换顺序 | $T_A = T_1 \cdot T_2 \neq T_2 \cdot T_1$ |
