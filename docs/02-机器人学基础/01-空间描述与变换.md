## 2. 理论部分

### 2.1 刚体的位姿描述

刚体在空间中的状态用**位姿**表示：

$$\text{位姿} = \text{位置} + \text{姿态}$$

| 组成部分 | 含义 | 表示方法 |
|----------|------|----------|
| 位置 | 刚体原点的坐标 | $(x, y, z)$ |
| 姿态 | 刚体相对于参考坐标系的朝向 | 旋转矩阵/四元数/欧拉角 |

### 2.2 坐标系

机器人中常用的坐标系：

| 坐标系 | 符号 | 说明 |
|--------|------|------|
| 世界坐标系 | $\{W\}$ 或 $\{0\}$ | 固定不动的参考系 |
| 基座坐标系 | $\{B\}$ | 机器人基座 |
| 工具坐标系 | $\{T\}$ | 机器人末端 |
| 目标坐标系 | $\{G\}$ | 目标位置 |

### 2.3 旋转矩阵

绕各轴旋转的**基本旋转矩阵**：

**绕X轴旋转 $\theta$ 角：**

$$R_x(\theta) = \begin{bmatrix}
1 & 0 & 0 \\
0 & \cos\theta & -\sin\theta \\
0 & \sin\theta & \cos\theta
\end{bmatrix}$$

**绕Y轴旋转 $\theta$ 角：**

$$R_y(\theta) = \begin{bmatrix}
\cos\theta & 0 & \sin\theta \\
0 & 1 & 0 \\
-\sin\theta & 0 & \cos\theta
\end{bmatrix}$$

**绕Z轴旋转 $\theta$ 角：**

$$R_z(\theta) = \begin{bmatrix}
\cos\theta & -\sin\theta & 0 \\
\sin\theta & \cos\theta & 0 \\
0 & 0 & 1
\end{bmatrix}$$

### 2.4 齐次变换矩阵

将旋转和平移统一到一个矩阵中：

$$T = \begin{bmatrix}
R & p \\
0 & 1
\end{bmatrix} = \begin{bmatrix}
r_{11} & r_{12} & r_{13} & p_x \\
r_{21} & r_{22} & r_{23} & p_y \\
r_{31} & r_{32} & r_{33} & p_z \\
0 & 0 & 0 & 1
\end{bmatrix}$$

其中：
- $R$：3×3 旋转矩阵
- $p$：3×1 平移向量

### 2.5 坐标变换

**复合变换**：

$${^A}T_B \cdot {^B}T_C = {^A}T_C$$

表示：先从坐标系C变换到B，再从B变换到A。

## 3. 实践部分

### 3.1 实践：Python坐标变换计算

#### 程序功能

- 计算复合变换矩阵
- 可视化结果

#### 程序位置

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
坐标变换计算器
功能：计算刚体在空间中的位姿变换

使用方法：
  python coordinate_transform.py
"""

import numpy as np
import matplotlib.pyplot as plt

class CoordinateTransform:
    """坐标变换计算器"""
    
    def __init__(self):
        pass
    
    def rotation_matrix_x(self, theta):
        """绕X轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [1, 0, 0],
            [0, c, -s],
            [0, s, c]
        ])
    
    def rotation_matrix_y(self, theta):
        """绕Y轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [c, 0, s],
            [0, 1, 0],
            [-s, 0, c]
        ])
    
    def rotation_matrix_z(self, theta):
        """绕Z轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [c, -s, 0],
            [s, c, 0],
            [0, 0, 1]
        ])
    
    def homogeneous_transform(self, R, p):
        """
        构建齐次变换矩阵
        
        Args:
            R: 3x3旋转矩阵
            p: 3x1平移向量
            
        Returns:
            T: 4x4齐次变换矩阵
        """
        T = np.eye(4)
        T[0:3, 0:3] = R
        T[0:3, 3] = p
        return T
    
    def transform_point(self, T, point):
        """
        变换点坐标
        
        Args:
            T: 4x4齐次变换矩阵
            point: [x, y, z] 原始坐标
            
        Returns:
            new_point: [x, y, z] 变换后坐标
        """
        point_h = np.array([point[0], point[1], point[2], 1])
        new_point_h = T @ point_h
        return new_point_h[0:3]
    
    def composite_transform(self, T1, T2):
        """
        复合变换：T1 * T2
        
        Args:
            T1, T2: 4x4齐次变换矩阵
            
        Returns:
            T: 复合变换矩阵
        """
        return T1 @ T2


def demo_rotation():
    """演示绕各轴旋转"""
    transform = CoordinateTransform()
    
    # 测试点
    point = np.array([1, 0, 0])
    
    print("=" * 50)
    print("坐标变换演示")
    print("=" * 50)
    print(f"原始点: {point}")
    
    # 绕Z轴旋转90度
    R_z = transform.rotation_matrix_z(np.pi/2)
    new_point = R_z @ point
    print(f"绕Z轴旋转90°: {new_point}")
    
    # 绕Y轴旋转90度
    R_y = transform.rotation_matrix_y(np.pi/2)
    new_point = R_y @ point
    print(f"绕Y轴旋转90°: {new_point}")


def demo_homogeneous():
    """演示齐次变换"""
    transform = CoordinateTransform()
    
    print("\n" + "=" * 50)
    print("齐次变换矩阵演示")
    print("=" * 50)
    
    # 旋转矩阵（绕Z轴30度）
    R = transform.rotation_matrix_z(np.pi/6)
    
    # 平移向量（沿X轴1米）
    p = np.array([1, 0, 0])
    
    # 构建齐次变换矩阵
    T = transform.homogeneous_transform(R, p)
    
    print("旋转矩阵（绕Z轴30°）：")
    print(np.round(R, 3))
    print("\n平移向量：[1, 0, 0] 米")
    print("\n齐次变换矩阵：")
    print(np.round(T, 3))
    
    # 变换点
    point = np.array([0.5, 0, 0])
    new_point = transform.transform_point(T, point)
    print(f"\n原始点: {point}")
    print(f"变换后: {new_point}")


def demo_composite():
    """演示复合变换"""
    transform = CoordinateTransform()
    
    print("\n" + "=" * 50)
    print("复合变换演示")
    print("=" * 50)
    
    # 变换1：绕Z轴旋转30度
    R1 = transform.rotation_matrix_z(np.pi/6)
    p1 = np.array([0, 0, 0])
    T1 = transform.homogeneous_transform(R1, p1)
    
    # 变换2：沿X轴移动1米
    R2 = np.eye(3)
    p2 = np.array([1, 0, 0])
    T2 = transform.homogeneous_transform(R2, p2)
    
    # 复合变换：先旋转后平移
    T = transform.composite_transform(T2, T1)
    
    print("变换1：绕Z轴旋转30°")
    print("变换2：沿X轴移动1米")
    print("\n复合变换矩阵：")
    print(np.round(T, 3))


def visualize():
    """可视化"""
    fig = plt.figure(figsize=(12, 4))
    
    transform = CoordinateTransform()
    point = np.array([1, 0, 0])
    
    # 原始点
    ax1 = fig.add_subplot(131, projection='3d')
    ax1.scatter(*point, c='blue', s=100)
    ax1.set_xlim(-1.5, 1.5)
    ax1.set_ylim(-1.5, 1.5)
    ax1.set_zlim(-1.5, 1.5)
    ax1.set_title('原始点')
    ax1.set_xlabel('X')
    ax1.set_ylabel('Y')
    ax1.set_zlabel('Z')
    
    # 旋转后
    ax2 = fig.add_subplot(132, projection='3d')
    R_z = transform.rotation_matrix_z(np.pi/2)
    new_point = R_z @ point
    ax2.scatter(*new_point, c='red', s=100)
    ax2.set_xlim(-1.5, 1.5)
    ax2.set_ylim(-1.5, 1.5)
    ax2.set_zlim(-1.5, 1.5)
    ax2.set_title('绕Z轴旋转90°')
    
    # 变换后
    ax3 = fig.add_subplot(133, projection='3d')
    T = transform.homogeneous_transform(R_z, np.array([0.5, 0, 0]))
    final_point = transform.transform_point(T, point)
    ax3.scatter(*final_point, c='green', s=100)
    ax3.set_xlim(-1.5, 1.5)
    ax3.set_ylim(-1.5, 1.5)
    ax3.set_zlim(-1.5, 1.5)
    ax3.set_title('旋转+平移')
    
    plt.tight_layout()
    plt.show()


if __name__ == '__main__':
    demo_rotation()
    demo_homogeneous()
    demo_composite()
