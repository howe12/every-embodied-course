## 2. 场景一：轮式机器人（差分驱动）

### 2.1 什么是差分驱动？

**差分驱动**是最常见的移动机器人底盘形式：
- 两个轮子独立驱动（左右各一个）
- 第三个轮子（万向轮）用于支撑

> **生活类比**：就像坦克的履带——左右履带速度不同，车就会转向！

### 2.2 运动学模型

```
        ┌─────────────┐
        │             │
   ○←───●─────────────●←───○
  (左轮)            (右轮)
        │      ↑      │
        │    中心     │
        │             │
        └─────────────┘
```

**运动学方程**：

| 参数 | 符号 | 说明 |
|------|------|------|
| 轮距 | $L$ | 左右轮之间的距离 |
| 左轮速度 | $v_L$ | 左轮线速度 (m/s) |
| 右轮速度 | $v_R$ | 右轮线速度 (m/s) |
| 机器人线速度 | $v$ | 机器人中心线速度 |
| 机器人角速度 | $\omega$ | 机器人转向角速度 |

**运动学公式**：

$$
v = \frac{v_R + v_L}{2}
$$

$$
\omega = \frac{v_R - v_L}{L}
$$

### 2.3 正运动学（轮速 → 位姿）

已知 $(v_L, v_R)$，求机器人位姿 $(x, y, \theta)$：

```python
def differential_drive_kinematics(vl, vr, L, dt, x, y, theta):
    """
    差分驱动正运动学
    
    参数:
        vl, vr: 左右轮速度 (m/s)
        L: 轮距 (m)
        dt: 时间步长 (s)
        x, y, theta: 当前位姿
    
    返回:
        更新后的位姿 (x, y, theta)
    """
    # 线速度和角速度
    v = (vr + vl) / 2
    omega = (vr - vl) / L
    
    # 更新角度
    theta_new = theta + omega * dt
    
    # 更新位置
    x_new = x + v * np.cos(theta) * dt
    y_new = y + v * np.sin(theta) * dt
    
    return x_new, y_new, theta_new
```

### 2.4 逆运动学（目标速度 → 轮速）

已知目标 $(v, \omega)$，求 $(v_L, v_R)$：

$$
v_L = v - \frac{\omega L}{2}
$$

$$
v_R = v + \frac{\omega L}{2}
$$

## 3. 场景二：2R 机械臂

### 3.1 2R 机械臂回顾

2R 机械臂是最经典的机械臂模型：
- 两个旋转关节
- 两个连杆长度 $L_1, L_2$

**正运动学**（从上一节已知）：
$$
x = L_1\cos\theta_1 + L_2\cos(\theta_1 + \theta_2)
$$
$$
y = L_1\sin\theta_1 + L_2\sin(\theta_1 + \theta_2)
$$

### 3.2 工作空间分析

**工作空间**是机械臂末端能达到的所有位置集合。

对于 2R 机械臂：
- **可达工作空间**：内半径 $|L_1 - L_2|$，外半径 $L_1 + L_2$ 的圆环区域
- **灵活工作空间**：2R 机械臂没有姿态灵活性（只有一种构型到达某点）

```python
def compute_workspace(L1, L2, resolution=100):
    """
    计算 2R 机械臂的工作空间
    """
    theta1_range = np.linspace(-np.pi, np.pi, resolution)
    theta2_range = np.linspace(-np.pi, np.pi, resolution)
    
    points_x = []
    points_y = []
    
    for t1 in theta1_range:
        for t2 in theta2_range:
            x = L1 * np.cos(t1) + L2 * np.cos(t1 + t2)
            y = L1 * np.sin(t1) + L2 * np.sin(t1 + t2)
            points_x.append(x)
            points_y.append(y)
    
    return np.array(points_x), np.array(points_y)
```

## 4. 完整仿真代码

### 4.1 轮式机器人仿真

```python
import numpy as np
import matplotlib.pyplot as plt

def simulate_diff_drive():
    """
    仿真差分驱动机器人走一个正方形轨迹
    """
    # 参数
    L = 0.5  # 轮距 (m)
    dt = 0.01  # 时间步长 (s)
    total_time = 20  # 总时间 (s)
    
    # 初始化位姿
    x, y, theta = 0, 0, 0
    
    # 存储轨迹
    trajectory = [(x, y, theta)]
    
    # 目标：走正方形 (每边 2m)
    # 阶段 1: 前进步行 2m
    # 阶段 2: 左转 90度
    # 阶段 3: 前进步行 2m
    # 阶段 4: 左转 90度
    # ... 重复
    
    steps = int(total_time / dt)
    
    for i in range(steps):
        t = i * dt
        
        # 根据时间确定目标动作
        phase = int(t / 5) % 4  # 每5秒换一个动作
        phase_time = t % 5
        
        if phase == 0:  # 前进
            v, omega = 0.4, 0
        elif phase == 1:  # 左转
            v, omega = 0, -np.pi/2 / 2  # 2秒转90度
        elif phase == 2:  # 前进
            v, omega = 0.4, 0
        else:  # 左转
            v, omega = 0, -np.pi/2 / 2
        
        # 逆运动学：求轮速
        vl = v - omega * L / 2
        vr = v + omega * L / 2
        
        # 正运动学：更新位姿
        x, y, theta = differential_drive_kinematics(vl, vr, L, dt, x, y, theta)
        
        trajectory.append((x, y, theta))
    
    # 绘制轨迹
    trajectory = np.array(trajectory)
    
    plt.figure(figsize=(8, 8))
    plt.plot(trajectory[:, 0], trajectory[:, 1], 'b-', linewidth=2)
    plt.plot(trajectory[0, 0], trajectory[0, 1], 'go', markersize=10, label='起点')
    plt.plot(trajectory[-1, 0], trajectory[-1, 1], 'ro', markersize=10, label='终点')
    
    plt.axis('equal')
    plt.xlabel('X (m)')
    plt.ylabel('Y (m)')
    plt.title('差分驱动机器人正方形轨迹')
    plt.legend()
    plt.grid(True)
    
    print(f"起点: ({trajectory[0, 0]:.2f}, {trajectory[0, 1]:.2f})")
    print(f"终点: ({trajectory[-1, 0]:.2f}, {trajectory[-1, 1]:.2f})")
    
    # plt.show()
    return trajectory


def differential_drive_kinematics(vl, vr, L, dt, x, y, theta):
    """差分驱动正运动学"""
    v = (vr + vl) / 2
    omega = (vr - vl) / L
    
    theta_new = theta + omega * dt
    x_new = x + v * np.cos(theta) * dt
    y_new = y + v * np.sin(theta) * dt
    
    return x_new, y_new, theta_new


if __name__ == "__main__":
    simulate_diff_drive()
```

### 4.2 2R 机械臂仿真

```python
def simulate_2r_arm():
    """
    仿真 2R 机械臂末端轨迹跟踪
    """
    # 参数
    L1, L2 = 1.0, 1.0  # 连杆长度
    
    # 目标：画一个圆
    radius = 1.2
    center_x, center_y = 1.0, 0.0
    omega = 0.5  # 角速度
    
    # 仿真
    dt = 0.01
    total_time = 4 * np.pi  # 画两个圆
    steps = int(total_time / dt)
    
    # 简化的逆运动学（使用 Jacobian 伪逆）
    theta1, theta2 = np.radians(45), np.radians(45)  # 初始角度
    
    trajectory = []
    
    for i in range(steps):
        t = i * dt
        
        # 目标位置（圆轨迹）
        target_x = center_x + radius * np.cos(omega * t)
        target_y = center_y + radius * np.sin(omega * t)
        
        # 当前末端位置
        x = L1 * np.cos(theta1) + L2 * np.cos(theta1 + theta2)
        y = L1 * np.sin(theta1) + L2 * np.sin(theta1 + theta2)
        
        # 位置误差
        ex = target_x - x
        ey = target_y - y
        
        # 雅可比矩阵（线速度部分）
        Jv = np.array([
            [-L1*np.sin(theta1) - L2*np.sin(theta1+theta2), -L2*np.sin(theta1+theta2)],
            [ L1*np.cos(theta1) + L2*np.cos(theta1+theta2),  L2*np.cos(theta1+theta2)]
        ])
        
        # 伪逆求解关节速度
        Jv_pinv = np.linalg.pinv(Jv)
        dq = Jv_pinv @ np.array([ex, ey]) * 5  # 增益 5
        
        # 更新关节角度
        theta1 += dq[0] * dt
        theta2 += dq[1] * dt
        
        trajectory.append((x, y, theta1, theta2))
    
    # 绘制
    trajectory = np.array(trajectory)
    
    plt.figure(figsize=(10, 5))
    
    # 子图 1: 轨迹
    plt.subplot(1, 2, 1)
    plt.plot(trajectory[:, 0], trajectory[:, 1], 'b-', linewidth=2, label='实际轨迹')
    
    # 目标圆
    theta_circle = np.linspace(0, 2*np.pi, 100)
    plt.plot(center_x + radius*np.cos(theta_circle), 
             center_y + radius*np.sin(theta_circle), 
             'r--', label='目标轨迹')
    
    plt.axis('equal')
    plt.xlabel('X (m)')
    plt.ylabel('Y (m)')
    plt.title('2R 机械臂轨迹跟踪')
    plt.legend()
    plt.grid(True)
    
    # 子图 2: 关节角度
    plt.subplot(1, 2, 2)
    plt.plot(np.degrees(trajectory[:, 2]), label='θ₁')
    plt.plot(np.degrees(trajectory[:, 3]), label='θ₂')
    plt.xlabel('时间步')
    plt.ylabel('角度 (°)')
    plt.title('关节角度变化')
    plt.legend()
    plt.grid(True)
    
    plt.tight_layout()
    # plt.show()
    
    return trajectory


if __name__ == "__main__":
    simulate_2r_arm()
```

## 5. 双场景对比

| 场景 | 轮式机器人 | 2R 机械臂 |
|------|-----------|-----------|
| **运动类型** | 平面移动 + 旋转 | 关节旋转 |
| **自由度** | 2 (x, y, θ 但受约束) | 2 |
| **工作空间** | 无界（只要地面够大） | 圆环区域 |
| **控制变量** | (v_L, v_R) 轮速 | (θ₁, θ₂) 关节角 |
| **运动学难点** | 运动约束 | 逆运动学求解 |
| **典型应用** | 扫地机器人、送餐机器人 | 工业装配、焊接 |

## 思考题答案

### 基础题答案

1. **轮式机器人**：如果左轮速度为 0.5 m/s，右轮速度为 0.3 m/s，轮距 0.5m，计算机器人的旋转半径。
   - 线速度 $v = (0.5 + 0.3) / 2 = 0.4$ m/s
   - 角速度 $\omega = (0.3 - 0.5) / 0.5 = -0.4$ rad/s
   - 旋转半径 $R = v / |\omega| = 0.4 / 0.4 = 1$ m
   - 负号表示向左（逆时针）转向

2. **机械臂**：2R 机械臂 $L_1=L_2=1$m 的可达工作空间内半径是多少？
   - 内半径 = $|L_1 - L_2| = |1 - 1| = 0$ m
   - 外半径 = $L_1 + L_2 = 2$ m
   - 所以工作空间是一个内半径为0、外半径为2的圆盘

### 进阶题答案

1. **三轮全向机器人**：调研并实现三轮全向轮机器人的运动学模型。
   - 全向机器人使用全向轮，可以同时实现前后和左右移动
   - 运动学模型需要考虑三个轮子的角度分布
   - 通常使用麦克纳姆轮或球轮

2. **3R 机械臂**：扩展代码实现 3R 机械臂的正逆运动学。
   - 正运动学：三个DH矩阵连乘 $T_{0}^{3} = T_{0}^{1} \times T_{1}^{2} \times T_{2}^{3}$
   - 逆运动学：通常需要数值解法（如牛顿法或雅可比伪逆）

3. **轨迹规划**：为机械臂添加圆弧插值或直线插值轨迹。
   - 直线插值：在起始点和目标点之间线性插值
   - 圆弧插值：需要指定圆心和半径，或使用三点确定圆弧
