## 2. 理论部分

### 2.1 位置描述

#### 2.1.1 点与向量

| 概念 | 说明 |
|------|------|
| **点** | 空间中的基本元素，没有长度，没有体积 |
| **向量** | 从某点指向另一点的"箭头"，表示方向和大小 |

> **重要**：不要把"向量"和"坐标"混淆！只有在指定了坐标系后，才能谈论向量在该坐标系下的坐标。

#### 2.1.2 坐标表示

一旦建立起坐标系后，我们能用一个 **3×1 向量**来描述点：

$$
^AP = \begin{bmatrix} p_x \\ p_y \\ p_z \end{bmatrix}
$$

其中三个元素代表向量 $P$ 在坐标系A三个轴上的**投影**。

```
        Z (上)
        |
        |
        |
        +------------> Y (前)
       /
      /
     /
    X (右)
```

这是**右手坐标系**——记住伸出右手：
- 拇指 → X轴
- 食指 → Y轴
- 中指 → Z轴

### 2.2 姿态描述（方向）

#### 什么是姿态？

光知道位置不够，机器人还需要知道**朝向**——就像你知道手机在桌上，但手机是平放还是竖着放？

#### 2.2.1 旋转矩阵

在物体上固定一个坐标系，给出此坐标系相对于参考坐标系的描述，这就是**姿态**。

设坐标系B相对于坐标系A的旋转矩阵为 ${^A_B}R$：

$$
{^A_B}R = \begin{bmatrix} ^Ax_B & ^Ay_B & ^Az_B \end{bmatrix} = \begin{bmatrix} 
r_{11} & r_{12} & r_{13} \\ 
r_{21} & r_{22} & r_{23} \\ 
r_{31} & r_{32} & r_{33} 
\end{bmatrix}
$$

其中每列是B坐标系主轴方向单位向量在A坐标系中的表达。

**绕单个轴旋转的基本旋转矩阵**：

**绕Z轴旋转 $\theta$ 角：**

$$
R_z(\theta) = \begin{bmatrix} 
\cos\theta & -\sin\theta & 0 \\ 
\sin\theta & \cos\theta & 0 \\ 
0 & 0 & 1 
\end{bmatrix}
$$

**绕X轴旋转 $\theta$ 角：**

$$
R_x(\theta) = \begin{bmatrix} 
1 & 0 & 0 \\ 
0 & \cos\theta & -\sin\theta \\ 
0 & \sin\theta & \cos\theta 
\end{bmatrix}
$$

**绕Y轴旋转 $\theta$ 角：**

$$
R_y(\theta) = \begin{bmatrix} 
\cos\theta & 0 & \sin\theta \\ 
0 & 1 & 0 \\ 
-\sin\theta & 0 & \cos\theta 
\end{bmatrix}
$$

> **注意**：这里用的是**弧度制**，不是角度制！
> - $360^\circ = 2\pi$ 弧度
> - $180^\circ = \pi$ 弧度
> - $90^\circ = \pi/2$ 弧度
> - $45^\circ = \pi/4$ 弧度

#### 2.2.2 旋转矩阵的性质

| 性质 | 公式 | 说明 |
|------|------|------|
| **正交矩阵** | $R^T R = I$ | 行向量和列向量都是单位向量且相互正交 |
| **行列式** | $\det(R) = 1$ | 保持坐标系手性 |
| **逆矩阵** | $R^{-1} = R^T$ | 转置等于逆矩阵 |

这意味着：
$$
R^{-1} = R^T = \begin{bmatrix} 
r_{11} & r_{21} & r_{31} \\ 
r_{12} & r_{22} & r_{32} \\ 
r_{13} & r_{23} & r_{33} 
\end{bmatrix}
$$

### 2.3 位姿 = 位置 + 姿态

**位姿**就是同时描述位置和方向：

$$
\text{位姿} = \begin{bmatrix} \text{位置} \\ \text{姿态} \end{bmatrix}
$$

举例：告诉你"手机在桌子右边20cm处，屏幕朝上"
- 位置：右边20cm
- 姿态：屏幕朝上（这个就是方向）

## 3. 坐标变换（映射）

### 3.1 坐标平移

当坐标系A与坐标系B的姿态相同，只有原点平移时：

$${^A}P = {^B}P + {^A_B}p$$

其中 ${^A_B}p$ 是B坐标系原点相对于A坐标系的位置向量。

### 3.2 坐标旋转

当两个坐标系原点重合，但姿态不同时：

$${^A}P = {^A_B}R \cdot {^B}P$$

### 3.3 一般变换（重点！）

当坐标系原点不重合且姿态不同时（最常见的情况）：

$${^A}P = {^A_B}R \cdot {^B}P + {^A_B}p$$

或者写成齐次形式：

$${^A}P = {^A_B}T \cdot {^B}P$$

其中齐次变换矩阵：

$$
T = \begin{bmatrix} R & p \\ 0 & 1 \end{bmatrix} = \begin{bmatrix} 
r_{11} & r_{12} & r_{13} & p_x \\ 
r_{21} & r_{22} & r_{23} & p_y \\ 
r_{31} & r_{32} & r_{33} & p_z \\ 
0 & 0 & 0 & 1 
\end{bmatrix}
$$

### 3.4 复合变换

已知坐标系C相对于B，B相对于A，求C相对于A：

$${^A_B}T \cdot {^B_C}T = {^A_C}T$$

变换顺序很重要：$T_A = T_1 \cdot T_2 \neq T_2 \cdot T_1$

### 3.5 逆变换

已知 ${^A_B}T$，求 ${^B_A}T$：

$$
{^B_A}T = \begin{bmatrix} R^T & -R^T \cdot p \\ 0 & 1 \end{bmatrix}
$$

## 4. 欧拉角

### 4.1 什么是欧拉角？

旋转矩阵和旋转向量虽然能描述旋转，但对人类不直观。**欧拉角**用**三个角度**表示旋转，更容易理解。

### 4.2 RPY角（最常用）

| 角度 | 绕轴 | 运动 |
|------|------|------|
| **偏航角(Yaw)** | Z轴 | 左右转向 |
| **俯仰角(Pitch)** | Y轴 | 上下俯仰 |
| **翻滚角(Roll)** | X轴 | 前后倾斜 |

旋转顺序：Z → Y → X

$$R = R_z(\text{yaw}) \cdot R_y(\text{pitch}) \cdot R_x(\text{roll})$$

### 4.3 万向锁问题（重要！）

**万向锁(Gimbal Lock)**：当俯仰角为 $\pm 90^\circ$ 时，第一次旋转和第三次旋转使用同一个轴，系统**丢失一个自由度**。

这是欧拉角的**奇异性问题**——用3个实数表示3维旋转必然存在。

> **注意**：由于这个原因，欧拉角不适用于插值和迭代，通常只用于人机交互。

---

## 5. 坐标变换注意事项

| 注意事项 | 说明 | 常见错误 |
|----------|------|----------|
| **注意坐标顺序** | 先旋转后平移，和先平移后旋转结果不同 | 搞混顺序 |
| **注意参考坐标系** | 一定要明确"相对于谁" | 没弄清楚参考系 |
| **注意单位** | 通常用米，不是厘米 | 忘记转换单位 |
| **注意角度制式** | 程序中通常用弧度制，不是角度制 | 用错单位 |
| **注意矩阵乘法顺序** | $A \cdot B \neq B \cdot A$ | 顺序颠倒 |

## 6. 实践部分

### 6.1 实践目标

用Python编写一个**坐标变换计算器**，实现：
1. 计算绕各轴旋转后的坐标
2. 构建齐次变换矩阵
3. 计算复合变换

### 6.2 程序

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
坐标变换计算器
功能：计算刚体在空间中的位姿变换
"""

import numpy as np


class CoordinateTransform:
    """坐标变换计算器"""
    
    def __init__(self):
        print("坐标变换计算器已创建")
    
    def rotation_matrix_x(self, theta):
        """绕X轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [1, 0, 0],
            [0, c, -s],
            [0, s, c]
        ])
    
    def rotation_matrix_y(self, theta):
        """绕Y轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [c, 0, s],
            [0, 1, 0],
            [-s, 0, c]
        ])
    
    def rotation_matrix_z(self, theta):
        """绕Z轴旋转矩阵"""
        c, s = np.cos(theta), np.sin(theta)
        return np.array([
            [c, -s, 0],
            [s, c, 0],
            [0, 0, 1]
        ])
    
    def rpy_to_rotation(self, roll, pitch, yaw):
        """RPY角转旋转矩阵 (Z-Y-X顺序)"""
        R_x = self.rotation_matrix_x(roll)
        R_y = self.rotation_matrix_y(pitch)
        R_z = self.rotation_matrix_z(yaw)
        return R_z @ R_y @ R_x
    
    def homogeneous_transform(self, R, p):
        """构建齐次变换矩阵"""
        T = np.eye(4)
        T[0:3, 0:3] = R
        T[0:3, 3] = p
        return T
    
    def transform_point(self, T, point):
        """变换点的坐标"""
        point_h = np.array([point[0], point[1], point[2], 1])
        new_point_h = T @ point_h
        return new_point_h[0:3]
    
    def inverse_transform(self, T):
        """求逆变换"""
        R = T[0:3, 0:3]
        p = T[0:3, 3]
        T_inv = np.eye(4)
        T_inv[0:3, 0:3] = R.T
        T_inv[0:3, 3] = -R.T @ p
        return T_inv


def demo():
    """演示"""
    t = CoordinateTransform()
    
    # 1. 基本旋转
    print("\n=== 演示1: 绕Z轴旋转90度 ===")
    point = np.array([1.0, 0.0, 0.0])
    R = t.rotation_matrix_z(np.pi / 2)
    new_point = R @ point
    print(f"原始点: {point}")
    print(f"旋转后: {new_point}")
    
    # 2. 齐次变换
    print("\n=== 演示2: 齐次变换矩阵 ===")
    R = t.rotation_matrix_z(np.pi / 6)  # 30度
    p = np.array([1.0, 0.0, 0.0])
    T = t.homogeneous_transform(R, p)
    print("齐次变换矩阵:")
    print(np.round(T, 3))
    
    # 3. RPY角
    print("\n=== 演示3: RPY角转旋转矩阵 ===")
    roll = np.radians(10)
    pitch = np.radians(20)
    yaw = np.radians(30)
    R = t.rpy_to_rotation(roll, pitch, yaw)
    print(f"Roll={np.degrees(roll):.1f}°, Pitch={np.degrees(pitch):.1f}°, Yaw={np.degrees(yaw):.1f}°")
    print("旋转矩阵:")
    print(np.round(R, 3))
    
    # 4. 逆变换
    print("\n=== 演示4: 逆变换 ===")
    T_inv = t.inverse_transform(T)
    print("逆变换矩阵:")
    print(np.round(T_inv, 3))


if __name__ == '__main__':
    demo()
```

## 7. 练习题

### 基础题

1. **向量概念**：点 $(3, 4, 0)$ 到原点的距离是多少？
2. **旋转**：绕Y轴旋转180度，点 $(1, 0, 0)$ 变成什么？
3. **单位换算**：90度等于多少弧度？

### 进阶题

4. **坐标变换**：B相对于A平移 $(2, 0, 0)$，点 $P_B = (1, 1, 0)$ 在A中的坐标？
5. **复合变换**：先绕Z轴旋转90度，再沿X轴移动1米，点 $(0, 1, 0)$ 在哪里？

---

## 思考题答案

### 基础题答案

1. **向量概念**：点 $(3, 4, 0)$ 到原点的距离是多少？
   - 距离 = $\sqrt{3^2 + 4^2 + 0^2} = \sqrt{9 + 16} = \sqrt{25} = 5$

2. **旋转**：绕Y轴旋转180度，点 $(1, 0, 0)$ 变成什么？
   - 绕Y轴旋转180°: $\cos(180°) = -1, \sin(180°) = 0$
   - $x' = x\cos\theta + z\sin\theta = 1 \times (-1) + 0 \times 0 = -1$
   - $y' = y = 0$
   - $z' = -x\sin\theta + z\cos\theta = 1 \times 0 + 0 \times (-1) = 0$
   - 结果：$(-1, 0, 0)$

3. **单位换算**：90度等于多少弧度？
   - $90° \times \frac{\pi}{180°} = \frac{\pi}{2} \approx 1.5708$ 弧度

### 进阶题答案

4. **坐标变换**：B相对于A平移 $(2, 0, 0)$，点 $P_B = (1, 1, 0)$ 在A中的坐标？
   - ${^A}P = {^B}P + {^A_B}p = (1, 1, 0) + (2, 0, 0) = (3, 1, 0)$

5. **复合变换**：先绕Z轴旋转90度，再沿X轴移动1米，点 $(0, 1, 0)$ 在哪里？
   - 绕Z轴旋转90°: $(0, 1, 0) \rightarrow (-1, 0, 0)$
   - 沿X轴移动1米: $(-1, 0, 0) + (1, 0, 0) = (0, 0, 0)$
   - 结果：$(0, 0, 0)$

### 8.3 程序代码

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
"""
坐标变换计算器 - 核心算法模块
功能：实现空间坐标变换的数学计算

设计原理：
    1. 旋转矩阵：3x3正交矩阵，R^T * R = I
    2. 齐次变换矩阵：4x4矩阵，统一表示旋转和平移
    3. 坐标变换：P_A = R_AB * P_B + p_AB

程序运行流程：
    transform_calculator.py (主程序)
    ├── __init__() - 初始化计算器
    ├── rotation_matrix_x/y/z() - 计算绕各轴旋转矩阵
    ├── rpy_to_rotation() - RPY角转旋转矩阵
    ├── quaternion_to_rotation() - 四元数转旋转矩阵
    ├── rotation_to_quaternion() - 旋转矩阵转四元数
    ├── homogeneous_transform() - 构建齐次变换矩阵
    ├── transform_point() - 变换点坐标
    ├── inverse_transform() - 求逆变换
    └── compose_transforms() - 复合变换

可供优化的参数：
    - 角度单位：弧度/角度切换 (USE_DEGREES)
    - 数值精度：小数位数 (DECIMAL_PLACES)
    - 奇异值分解公差 (SVD_TOLERANCE)

作者：霍海杰
"""

import numpy as np
from typing import Tuple, Union

# ============== 可调参数 ==============
USE_DEGREES = False        # True: 使用角度制, False: 使用弧度制
DECIMAL_PLACES = 4         # 输出小数位数
SVD_TOLERANCE = 1e-6       # SVD奇异值分解公差
# ======================================


class TransformCalculator:
    """
    坐标变换计算器
    
    支持：
    - 绕X/Y/Z轴的旋转矩阵计算
    - RPY角（欧拉角）与旋转矩阵互转
    - 四元数与旋转矩阵互转
    - 齐次变换矩阵构建与求逆
    - 点的坐标变换与复合变换
    """
    
    def __init__(self, use_degrees: bool = USE_DEGREES):
        self.use_degrees = use_degrees
        self.decimal_places = DECIMAL_PLACES
        print(f"[TransformCalculator] 初始化完成，使用{'角度' if use_degrees else '弧度'}制")
    
    def _to_radians(self, angle: float) -> float:
        if self.use_degrees:
            return np.radians(angle)
        return angle
    
    def rotation_matrix_x(self, theta: float) -> np.ndarray:
        """绕X轴旋转矩阵"""
        theta = self._to_radians(theta)
        c, s = np.cos(theta), np.sin(theta)
        return np.array([[1, 0, 0], [0, c, -s], [0, s, c]])
    
    def rotation_matrix_y(self, theta: float) -> np.ndarray:
        """绕Y轴旋转矩阵"""
        theta = self._to_radians(theta)
        c, s = np.cos(theta), np.sin(theta)
        return np.array([[c, 0, s], [0, 1, 0], [-s, 0, c]])
    
    def rotation_matrix_z(self, theta: float) -> np.ndarray:
        """绕Z轴旋转矩阵"""
        theta = self._to_radians(theta)
        c, s = np.cos(theta), np.sin(theta)
        return np.array([[c, -s, 0], [s, c, 0], [0, 0, 1]])
    
    def rotation_matrix(self, axis: str, theta: float) -> np.ndarray:
        """通用旋转矩阵计算"""
        axis = axis.lower()
        if axis == 'x': return self.rotation_matrix_x(theta)
        elif axis == 'y': return self.rotation_matrix_y(theta)
        elif axis == 'z': return self.rotation_matrix_z(theta)
        else: raise ValueError(f"Invalid axis: {axis}")
    
    def rpy_to_rotation(self, roll: float, pitch: float, yaw: float, order: str = 'ZYX') -> np.ndarray:
        """RPY角转旋转矩阵"""
        roll = self._to_radians(roll)
        pitch = self._to_radians(pitch)
        yaw = self._to_radians(yaw)
        R_x = self.rotation_matrix_x(roll)
        R_y = self.rotation_matrix_y(pitch)
        R_z = self.rotation_matrix_z(yaw)
        if order == 'ZYX': return R_z @ R_y @ R_x
        elif order == 'XYZ': return R_x @ R_y @ R_z
        elif order == 'YXZ': return R_y @ R_x @ R_z
        else: raise ValueError(f"Invalid order: {order}")
    
    def rotation_to_rpy(self, R: np.ndarray) -> Tuple[float, float, float]:
        """旋转矩阵转RPY角"""
        pitch = np.arcsin(-R[2, 0])
        if np.abs(np.cos(pitch)) > SVD_TOLERANCE:
            roll = np.arctan2(R[2, 1], R[2, 2])
            yaw = np.arctan2(R[1, 0], R[0, 0])
        else:
            roll = 0
            yaw = np.arctan2(-R[0, 1], R[1, 1])
        return roll, pitch, yaw
    
    def quaternion_to_rotation(self, q: Union[list, np.ndarray]) -> np.ndarray:
        """四元数转旋转矩阵"""
        if len(q) == 4:
            qw, qx, qy, qz = q
        else:
            raise ValueError("四元数必须包含4个分量")
        norm = np.sqrt(qw**2 + qx**2 + qy**2 + qz**2)
        qw, qx, qy, qz = qw/norm, qx/norm, qy/norm, qz/norm
        R = np.array([
            [1 - 2*(qy**2 + qz**2), 2*(qx*qy - qw*qz), 2*(qx*qz + qw*qy)],
            [2*(qx*qy + qw*qz), 1 - 2*(qx**2 + qz**2), 2*(qy*qz - qw*qx)],
            [2*(qx*qz - qw*qy), 2*(qy*qz + qw*qx), 1 - 2*(qx**2 + qy**2)]
        ])
        return R
    
    def rotation_to_quaternion(self, R: np.ndarray) -> np.ndarray:
        """旋转矩阵转四元数"""
        tr = np.trace(R)
        if tr > 0:
            S = np.sqrt(tr + 1.0) * 2
            qw = 0.25 * S
            qx = (R[2, 1] - R[1, 2]) / S
            qy = (R[0, 2] - R[2, 0]) / S
            qz = (R[1, 0] - R[0, 1]) / S
        elif R[0, 0] > R[1, 1] and R[0, 0] > R[2, 2]:
            S = np.sqrt(1.0 + R[0, 0] - R[1, 1] - R[2, 2]) * 2
            qw = (R[2, 1] - R[1, 2]) / S
            qx = 0.25 * S
            qy = (R[0, 1] + R[1, 0]) / S
            qz = (R[0, 2] + R[2, 0]) / S
        elif R[1, 1] > R[2, 2]:
            S = np.sqrt(1.0 + R[1, 1] - R[0, 0] - R[2, 2]) * 2
            qw = (R[0, 2] - R[2, 0]) / S
            qx = (R[0, 1] + R[1, 0]) / S
            qy = 0.25 * S
            qz = (R[1, 2] + R[2, 1]) / S
        else:
            S = np.sqrt(1.0 + R[2, 2] - R[0, 0] - R[1, 1]) * 2
            qw = (R[1, 0] - R[0, 1]) / S
            qx = (R[0, 2] + R[2, 0]) / S
            qy = (R[1, 2] + R[2, 1]) / S
            qz = 0.25 * S
        return np.array([qw, qx, qy, qz])
    
    def homogeneous_transform(self, R: np.ndarray, p: Union[list, np.ndarray]) -> np.ndarray:
        """构建齐次变换矩阵"""
        p = np.array(p).flatten()
        T = np.eye(4)
        T[0:3, 0:3] = R
        T[0:3, 3] = p
        return T
    
    def extract_R_and_p(self, T: np.ndarray) -> Tuple[np.ndarray, np.ndarray]:
        """从齐次变换矩阵提取R和p"""
        return T[0:3, 0:3], T[0:3, 3]
    
    def transform_point(self, T: np.ndarray, point: Union[list, np.ndarray]) -> np.ndarray:
        """变换点的坐标"""
        point = np.array(point).flatten()
        point_h = np.append(point, 1)
        return (T @ point_h)[0:3]
    
    def inverse_transform(self, T: np.ndarray) -> np.ndarray:
        """求逆变换"""
        R, p = self.extract_R_and_p(T)
        T_inv = np.eye(4)
        T_inv[0:3, 0:3] = R.T
        T_inv[0:3, 3] = -R.T @ p
        return T_inv
    
    def compose_transforms(self, T1: np.ndarray, T2: np.ndarray) -> np.ndarray:
        """复合变换"""
        return T1 @ T2
    
    def print_transform(self, T: np.ndarray, name: str = "变换矩阵"):
        print(f"\n{'='*50}\n{name}:\n{'='*50}")
        print(np.round(T, self.decimal_places))
    
    def print_rotation_matrix(self, R: np.ndarray, name: str = "旋转矩阵"):
        print(f"\n{name}:\n{np.round(R, self.decimal_places)}")
        is_orthogonal = np.allclose(R @ R.T, np.eye(3), atol=SVD_TOLERANCE)
        det_R = np.linalg.det(R)
        print(f"  - 正交性: {'✓' if is_orthogonal else '✗'}")
        print(f"  - 行列式: {det_R:.6f} ({'右手系' if det_R > 0 else '左手系'})")


def demo():
    print("\n" + "="*60)
    print(" 坐标变换计算器 - 功能演示 ")
    print("="*60)
    
    calc = TransformCalculator(use_degrees=False)
    
    print("\n>>> 演示1: 绕Z轴旋转90度")
    point = np.array([1.0, 0.0, 0.0])
    R_z = calc.rotation_matrix_z(np.pi / 2)
    new_point = R_z @ point
    print(f"原始点: {point}\n旋转后: {new_point}")
    calc.print_rotation_matrix(R_z, "旋转矩阵 R_z(90°)")
    
    print("\n>>> 演示2: 齐次变换矩阵")
    R = calc.rotation_matrix_z(np.pi / 6)
    p = np.array([1.0, 0.0, 0.0])
    T = calc.homogeneous_transform(R, p)
    calc.print_transform(T, "齐次变换矩阵 T")
    
    print("\n>>> 演示3: 点坐标变换")
    point_in_B = np.array([0.0, 1.0, 0.0])
    point_in_A = calc.transform_point(T, point_in_B)
    print(f"B坐标系中的点: {point_in_B}\n变换到A坐标系: {point_in_A}")
    
    print("\n>>> 演示4: RPY角转旋转矩阵")
    roll, pitch, yaw = 0.1, 0.2, 0.3
    R_rpy = calc.rpy_to_rotation(roll, pitch, yaw)
    calc.print_rotation_matrix(R_rpy, "RPY旋转矩阵")
    r, p, y = calc.rotation_to_rpy(R_rpy)
    print(f"转回RPY: roll={r:.3f}, pitch={p:.3f}, yaw={y:.3f}")
    
    print("\n>>> 演示5: 四元数与旋转矩阵互转")
    q = [0.877, 0.479, 0, 0]
    R_quat = calc.quaternion_to_rotation(q)
    calc.print_rotation_matrix(R_quat, "四元数转旋转矩阵")
    q_back = calc.rotation_to_quaternion(R_quat)
    print(f"转回四元数: {np.round(q_back, 3)}")
    
    print("\n>>> 演示6: 逆变换")
    T_inv = calc.inverse_transform(T)
    calc.print_transform(T_inv, "逆变换矩阵 T^(-1)")
    print(f"验证 T @ T^(-1) ≈ I: {np.allclose(T @ T_inv, np.eye(4), atol=SVD_TOLERANCE)}")
    
    print("\n>>> 演示7: 复合变换")
    R_z30 = calc.rotation_matrix_z(np.pi / 6)
    T_rotate = calc.homogeneous_transform(R_z30, [0, 0, 0])
    T_translate = calc.homogeneous_transform(np.eye(3), [1, 0, 0])
    T_combined = calc.compose_transforms(T_translate, T_rotate)
    original = np.array([0.5, 0.5, 0])
    transformed = calc.transform_point(T_combined, original)
    print(f"原始点: {original}\n复合变换后: {transformed}")
    
    print("\n" + "="*60)
    print(" 演示完成 ")
    print("="*60 + "\n")
```

```python
#!/usr/bin/env python3
"""
tf2_ros 坐标系发布示例
"""

import rclpy
from rclpy.node import Node
from geometry_msgs.msg import TransformStamped
from tf2_ros import TransformBroadcaster
import numpy as np


class TFBroadcaster(Node):
    def __init__(self):
        super().__init__('tf_broadcaster')
        self.tf_broadcaster = TransformBroadcaster(self)
        self.timer = self.create_timer(0.1, self.timer_callback)
        from coordinate_transform.transform_calculator import TransformCalculator
        self.calc = TransformCalculator()
        
    def timer_callback(self):
        t = TransformStamped()
        t.header.stamp = self.get_clock().now().to_msg()
        t.header.frame_id = 'base_link'
        t.child_frame_id = 'end_effector'
        t.transform.translation.x = 0.5
        t.transform.translation.y = 0.3
        t.transform.translation.z = 0.2
        q = self.calc.rotation_to_quaternion(
            self.calc.rpy_to_rotation(0.0, 0.0, np.pi/4)
        )
        t.transform.rotation.x = q[1]
        t.transform.rotation.y = q[2]
        t.transform.rotation.z = q[3]
        t.transform.rotation.w = q[0]
        self.tf_broadcaster.sendTransform(t)


def main(args=None):
    rclpy.init(args=args)
    node = TFBroadcaster()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()


if __name__ == '__main__':
    main()
```
