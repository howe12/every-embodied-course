## 1. 模仿学习概述

### 1.1 什么是模仿学习

模仿学习的核心思想是**通过专家示范（Demonstration）来学习策略**。与强化学习不同，模仿学习不依赖奖励函数，而是直接从人类或专家提供的状态-动作配对数据中学习。智能体观察环境状态，模仿专家应该采取的动作，从而复现专家行为。

```
专家示范 → (状态, 动作) 配对数据集 → 监督学习 → 策略网络 π(a|s)
```

**典型应用场景**：

| 应用 | 专家示范方式 | 学习目标 |
|------|------------|---------|
| 手术机器人 | 医生遥操作演示 | 复现精细切割/缝合动作 |
| 无人机编队 | 人工飞行轨迹 | 复现稳定飞行/编队变换 |
| 家庭机器人 | 人类动作捕捉 | 复现折叠衣物/整理桌面 |
| 自动驾驶 | 驾驶员行车记录 | 复现变道/转弯/停车决策 |

### 1.2 模仿学习 vs 强化学习

| 对比维度 | 模仿学习 | 强化学习 |
|----------|----------|----------|
| **数据来源** | 专家示范轨迹 | 智能体自主探索 |
| **奖励函数** | 不需要 | 通常需要手动设计 |
| **样本效率** | 高（数据一次性收集） | 低（需要大量试错） |
| **探索风险** | 无（只复现已知的专家行为） | 高（探索可能损坏机器人） |
| **策略上限** | 受限于专家演示质量 | 理论上可超越人类 |
| **分布偏移** | 容易发生（compounding error） | 通过奖励引导自然探索 |

### 1.3 模仿学习主要方法

| 方法 | 核心思想 | 优点 | 缺点 |
|------|---------|------|------|
| **行为克隆（BC）** | 将示范数据当作监督学习 | 简单直接，实现容易 | 分布偏移，误差累积 |
| **DAgger** | 迭代式查询专家策略 | 在线修正，可适应新状态 | 需要持续接入专家 |
| **GAIL** | 生成对抗模仿学习 | 可学习隐式奖励，无奖励工程 | 训练不稳定 |
| **IRL（逆向强化学习）** | 从示范推断奖励函数 | 可迁移到新环境 | 计算代价高 |

## 2. 行为克隆（Behavior Cloning）

### 2.1 原理详解

行为克隆将模仿学习问题建模为**标准的监督分类/回归问题**。给定专家示范数据集 $\mathcal{D} = \{(s_i, a_i)\}_{i=1}^N$，学习一个策略网络 $\pi_\theta(a|s)$，使得在状态 $s_i$ 下的预测动作 $\hat{a}_i$ 尽可能接近专家动作 $a_i$。

**损失函数**（分类问题，多分类动作）：

$$
\mathcal{L}(\theta) = -\sum_{i=1}^N \log \pi_\theta(a_i | s_i)
$$

**回归问题（连续动作）**使用均方误差：

$$
\mathcal{L}(\theta) = \sum_{i=1}^N \| a_i - \pi_\theta(s_i) \|^2
$$

### 2.2 数据收集

| 采集方式 | 工具/设备 | 适用场景 |
|---------|---------|---------|
| **遥操作（Teleoperation）** | 示教器、手柄、VR | 精细操作任务 |
| **动作捕捉（Motion Capture）** | OptiTrack、VICON | 人体运动复现 |
| **动觉示教（Kinesthetic Teaching）** | 直接拖动机器人 | 简单示教场景 |
| **动觉示教（Kinesthetic Teaching）** | 触觉反馈设备 | 精确力控任务 |
| **人类视频（Third-Person）** | RGB相机 | 开放场景，低成本 |

**数据预处理要点**：
- 状态对齐：确保专家状态与智能体感知一致
- 动作映射：专家动作空间 → 机器人动作空间的映射
- 时序对齐：处理示范中的时间步对齐问题

### 2.3 代码实现：基于PyTorch的行为克隆

```python
"""
行为克隆（Behavior Cloning）实现
基于PyTorch + Gymnasium机器人环境
"""
import torch
import torch.nn as nn
import torch.optim as optim
import gymnasium as gym
import numpy as np
from collections import deque
import pickle
import os

# ============ 1. 策略网络定义 ============
class PolicyNetwork(nn.Module):
    """策略网络：从状态预测动作"""
    def __init__(self, state_dim, action_dim, hidden_dims=[256, 256]):
        super().__init__()
        layers = []
        prev_dim = state_dim
        for h_dim in hidden_dims:
            layers.extend([
                nn.Linear(prev_dim, h_dim),
                nn.ReLU()
            ])
            prev_dim = h_dim
        layers.append(nn.Linear(prev_dim, action_dim))
        self.network = nn.Sequential(*layers)
    
    def forward(self, state):
        return self.network(state)


class DiscretePolicyNetwork(nn.Module):
    """离散动作策略网络"""
    def __init__(self, state_dim, action_dim, hidden_dims=[256, 256]):
        super().__init__()
        layers = []
        prev_dim = state_dim
        for h_dim in hidden_dims:
            layers.extend([
                nn.Linear(prev_dim, h_dim),
                nn.ReLU()
            ])
            prev_dim = h_dim
        layers.append(nn.Linear(prev_dim, action_dim))
        layers.append(nn.Softmax(dim=-1))
        self.network = nn.Sequential(*layers)
    
    def forward(self, state):
        return self.network(state)


# ============ 2. 专家策略（用于生成示范数据） ============
class ExpertPolicy:
    """专家策略：基于规则的控制器（CartPole平衡）"""
    def __init__(self, action_dim):
        self.action_dim = action_dim
    
    def get_action(self, state):
        """CartPole专家规则：小车位偏左推杆向左，位偏右推杆向右"""
        x, x_dot, theta, theta_dot = state
        if theta < 0:
            return 0  # 杆偏左推左
        else:
            return 1  # 杆偏右推右


# ============ 3. 数据采集 ============
def collect_demonstrations(env, expert, num_episodes=50):
    """收集专家示范数据"""
    demonstrations = []
    
    for episode in range(num_episodes):
        state, _ = env.reset()
        episode_data = {'states': [], 'actions': []}
        terminated = False
        truncated = False
        episode_reward = 0
        
        while not (terminated or truncated):
            action = expert.get_action(state)
            episode_data['states'].append(state)
            episode_data['actions'].append(action)
            
            state, reward, terminated, truncated, _ = env.step(action)
            episode_reward += reward
        
        episode_data['states'] = np.array(episode_data['states'])
        episode_data['actions'] = np.array(episode_data['actions'])
        demonstrations.append(episode_data)
        print(f"Episode {episode+1}/{num_episodes}: reward={episode_reward}")
    
    return demonstrations


# ============ 4. 行为克隆训练 ============
def train_behavior_cloning(demonstrations, state_dim, action_dim, 
                           epochs=50, batch_size=64, lr=1e-3):
    """训练行为克隆策略"""
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    
    # 构建数据集
    all_states = np.concatenate([demo['states'] for demo in demonstrations])
    all_actions = np.concatenate([demo['actions'] for demo in demonstrations])
    
    # 转为PyTorch张量
    states_tensor = torch.FloatTensor(all_states).to(device)
    actions_tensor = torch.LongTensor(all_actions).to(device)
    
    # 策略网络
    policy = DiscretePolicyNetwork(state_dim, action_dim).to(device)
    optimizer = optim.Adam(policy.parameters(), lr=lr)
    criterion = nn.CrossEntropyLoss()
    
    dataset_size = len(states_tensor)
    print(f"\n数据集大小: {dataset_size} 样本")
    
    for epoch in range(epochs):
        # Mini-batch随机梯度下降
        indices = torch.randperm(dataset_size)
        total_loss = 0.0
        num_batches = 0
        
        for i in range(0, dataset_size, batch_size):
            batch_idx = indices[i:i+batch_size]
            batch_states = states_tensor[batch_idx]
            batch_actions = actions_tensor[batch_idx]
            
            # 前向传播
            action_probs = policy(batch_states)
            loss = criterion(action_probs, batch_actions)
            
            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            optimizer.step()
            
            total_loss += loss.item()
            num_batches += 1
        
        if (epoch + 1) % 10 == 0:
            avg_loss = total_loss / num_batches
            # 计算训练准确率
            with torch.no_grad():
                preds = policy(states_tensor).argmax(dim=1)
                accuracy = (preds == actions_tensor).float().mean().item()
            print(f"Epoch {epoch+1}/{epochs}: Loss={avg_loss:.4f}, Accuracy={accuracy:.4f}")
    
    return policy


# ============ 5. 评估策略 ============
def evaluate_policy(env, policy, num_episodes=20):
    """评估学习到的策略"""
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    policy.eval()
    
    total_rewards = []
    
    for episode in range(num_episodes):
        state, _ = env.reset()
        episode_reward = 0
        terminated = False
        truncated = False
        
        with torch.no_grad():
            while not (terminated or truncated):
                state_t = torch.FloatTensor(state).unsqueeze(0).to(device)
                action_prob = policy(state_t)
                action = action_prob.argmax(dim=1).item()
                
                state, reward, terminated, truncated, _ = env.step(action)
                episode_reward += reward
        
        total_rewards.append(episode_reward)
    
    mean_reward = np.mean(total_rewards)
    std_reward = np.std(total_rewards)
    print(f"\n评估结果: {num_episodes} episodes")
    print(f"平均奖励: {mean_reward:.2f} ± {std_reward:.2f}")
    return mean_reward


# ============ 6. 主函数 ============
if __name__ == "__main__":
    # 创建环境
    env = gym.make("CartPole-v1")
    state_dim = env.observation_space.shape[0]
    action_dim = env.action_space.n
    
    print(f"状态空间维度: {state_dim}, 动作空间维度: {action_dim}")
    
    # Step 1: 收集专家示范数据
    expert = ExpertPolicy(action_dim)
    demonstrations = collect_demonstrations(env, expert, num_episodes=50)
    
    # 保存数据（可选）
    os.makedirs("data", exist_ok=True)
    with open("data/cartpole_demonstrations.pkl", "wb") as f:
        pickle.dump(demonstrations, f)
    print(f"\n已保存 {len(demonstrations)} 条示教轨迹")
    
    # Step 2: 训练行为克隆策略
    policy = train_behavior_cloning(demonstrations, state_dim, action_dim,
                                     epochs=100, batch_size=64, lr=1e-3)
    
    # Step 3: 评估策略
    eval_reward = evaluate_policy(env, policy, num_episodes=20)
    
    # 保存策略
    torch.save(policy.state_dict(), "models/bc_policy_cartpole.pth")
    print("\n策略已保存至 models/bc_policy_cartpole.pth")
    
    env.close()
```

### 2.4 分布偏移问题（Compounding Error）

行为克隆的核心问题是**分布偏移（Distribution Shift）**。训练时策略见过所有示范状态，但实际部署时微小误差会累积，导致智能体进入从未见过的状态分布。

```
专家轨迹:     s1 → s2 → s3 → s4 → s5（目标达成）
克隆轨迹:     s1 → s2' → s3'' → s4''' → s5''''（偏移累积）
                ↑     ↑      ↑        ↑         ↑
              误差1  误差2   误差3     误差4      到达未知区域
```

**解决方案对比**：

| 方案 | 核心思想 | 实现难度 |
|------|---------|---------|
| **数据增强** | 增加噪声扰动下的示范数据 | 低 |
| **DAgger** | 在线迭代查询专家修正 | 中 |
| **随机平滑** | 在状态输入层添加Dropout/噪声 | 低 |
| **多专家聚合** | 混合多个专家策略增加鲁棒性 | 中 |

## 3. DAgger（Dataset Aggregation）

### 3.1 原理详解

DAgger（Dataset Aggregation）由 Ross 等人于 2011 年提出，核心思想是**迭代式地收集专家修正数据**，逐步缩小学习策略与专家策略之间的差距。

**算法流程**：

```
初始化: 
  π_0 ← 随机初始化策略
  D ← ∅ (空数据集)

迭代 (对于每个回合):
  1. 用当前策略 π_i 收集轨迹: {τ_j}
  2. 在每个状态 s ∈ τ_j 查询专家获取最优动作: a* = π*(s)
  3. 将 (s, a*) 加入数据集: D ← D ∪ {(s, π_i(s), π*(s))}
  4. 用 D 重新训练策略 π_i
```

**关键公式**：

经过 $N$ 次迭代后，DAgger的累积误差上界为：

$$
\epsilon_N \leq O\left(\frac{1}{N}\right) + O\left(\frac{T \cdot \epsilon_{\text{bc}}}{N}\right)
$$

其中 $T$ 是轨迹长度，$\epsilon_{\text{bc}}$ 是单步行为克隆误差。

### 3.2 代码实现：DAgger

```python
"""
DAgger（Dataset Aggregation）实现
在CartPole上迭代训练专家修正
"""
import torch
import torch.nn as nn
import torch.optim as optim
import gymnasium as gym
import numpy as np

class DiscretePolicyNetwork(nn.Module):
    """离散动作策略网络"""
    def __init__(self, state_dim, action_dim, hidden_dims=[128, 128]):
        super().__init__()
        layers = []
        prev_dim = state_dim
        for h_dim in hidden_dims:
            layers.extend([nn.Linear(prev_dim, h_dim), nn.ReLU()])
            prev_dim = h_dim
        layers.extend([nn.Linear(prev_dim, action_dim), nn.Softmax(dim=-1)])
        self.network = nn.Sequential(*layers)
    
    def forward(self, state):
        return self.network(state)


class ExpertPolicy:
    """CartPole专家策略"""
    def get_action(self, state):
        x, x_dot, theta, theta_dot = state
        return 0 if theta < 0 else 1


def train_dagger(env, expert, state_dim, action_dim,
                 num_iterations=10, episodes_per_iteration=10,
                 epochs_per_iteration=20, batch_size=64):
    """DAgger训练主循环"""
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    
    # 初始化策略
    policy = DiscretePolicyNetwork(state_dim, action_dim).to(device)
    optimizer = optim.Adam(policy.parameters(), lr=1e-3)
    criterion = nn.CrossEntropyLoss()
    
    # 数据集（存储专家动作）
    dataset_states = []
    dataset_actions = []
    
    print("=" * 50)
    print("开始 DAgger 训练")
    print("=" * 50)
    
    for iteration in range(num_iterations):
        print(f"\n--- Iteration {iteration + 1}/{num_iterations} ---")
        
        # Step 1: 用当前策略收集轨迹
        all_states_this_iter = []
        all_expert_actions_this_iter = []
        
        for ep in range(episodes_per_iteration):
            state, _ = env.reset()
            terminated = False
            truncated = False
            ep_reward = 0
            
            while not (terminated or truncated):
                # 保存当前状态
                all_states_this_iter.append(state)
                
                # 用当前策略选动作
                with torch.no_grad():
                    state_t = torch.FloatTensor(state).unsqueeze(0).to(device)
                    action_prob = policy(state_t)
                    action = action_prob.argmax(dim=1).item()
                
                # 查询专家，获取该状态下的最优动作
                expert_action = expert.get_action(state)
                all_expert_actions_this_iter.append(expert_action)
                
                state, reward, terminated, truncated, _ = env.step(action)
                ep_reward += reward
            
            print(f"  Episode {ep+1}: reward={ep_reward}")
        
        # Step 2: 将新数据加入总数据集
        dataset_states.extend(all_states_this_iter)
        dataset_actions.extend(all_expert_actions_this_iter)
        
        print(f"  累计数据集大小: {len(dataset_states)} 样本")
        
        # Step 3: 用聚合数据集重新训练策略
        states_np = np.array(dataset_states)
        actions_np = np.array(dataset_actions)
        
        states_tensor = torch.FloatTensor(states_np).to(device)
        actions_tensor = torch.LongTensor(actions_np).to(device)
        
        for epoch in range(epochs_per_iteration):
            indices = torch.randperm(len(states_tensor))
            total_loss = 0.0
            num_batches = 0
            
            for i in range(0, len(states_tensor), batch_size):
                batch_idx = indices[i:i+batch_size]
                batch_states = states_tensor[batch_idx]
                batch_actions = actions_tensor[batch_idx]
                
                action_probs = policy(batch_states)
                loss = criterion(action_probs, batch_actions)
                
                optimizer.zero_grad()
                loss.backward()
                optimizer.step()
                
                total_loss += loss.item()
                num_batches += 1
            
            if (epoch + 1) % 10 == 0:
                print(f"    Epoch {epoch+1}: loss={total_loss/num_batches:.4f}")
        
        # Step 4: 评估当前策略
        mean_reward = evaluate_current_policy(env, policy, device)
        print(f"  当前策略评估: mean_reward={mean_reward:.1f}")
    
    return policy


def evaluate_current_policy(env, policy, device, num_episodes=5):
    """快速评估策略"""
    policy.eval()
    rewards = []
    
    with torch.no_grad():
        for _ in range(num_episodes):
            state, _ = env.reset()
            ep_reward = 0
            terminated = False
            truncated = False
            
            while not (terminated or truncated):
                state_t = torch.FloatTensor(state).unsqueeze(0).to(device)
                action = policy(state_t).argmax(dim=1).item()
                state, reward, terminated, truncated, _ = env.step(action)
                ep_reward += reward
            
            rewards.append(ep_reward)
    
    return np.mean(rewards)


if __name__ == "__main__":
    env = gym.make("CartPole-v1")
    state_dim = env.observation_space.shape[0]
    action_dim = env.action_space.n
    
    expert = ExpertPolicy()
    
    policy = train_dagger(
        env, expert, state_dim, action_dim,
        num_iterations=8,
        episodes_per_iteration=10,
        epochs_per_iteration=30,
        batch_size=128
    )
    
    print("\n" + "=" * 50)
    print("DAgger训练完成，最终评估:")
    evaluate_current_policy(env, policy, 
                            torch.device("cuda" if torch.cuda.is_available() else "cpu"),
                            num_episodes=20)
    
    torch.save(policy.state_dict(), "models/dagger_policy_cartpole.pth")
    env.close()
```

## 4. GAIL（生成对抗模仿学习）

### 4.1 原理详解

GAIL（Generative Adversarial Imitation Learning）由 Ho 和 Ermon 于 2016 年提出，将**生成对抗网络（GAN）**的思想引入模仿学习。

**核心思想**：
- 训练一个判别器 $D$，区分专家数据 $(s, a)$ 与智能体生成数据 $(s, \pi(a|s))$
- 训练一个生成器（策略网络）$\pi$，欺骗判别器使其认为自己的数据来自专家分布
- 判别器的输出作为隐式奖励，引导策略学习

**判别器损失**：

$$
\min_D \mathbb{E}_{ (s,a) \sim \pi_E } [\log D(s,a)] + \mathbb{E}_{ (s,a) \sim \pi_\theta } [\log(1 - D(s,a))]
$$

**策略奖励（来自判别器）**：

$$
r_t(s, a) = -\log(1 - D(s, a))
$$

### 4.2 GAIL vs BC 对比

| 维度 | 行为克隆（BC） | GAIL |
|------|--------------|------|
| 学习目标 | 最小化动作预测误差 | 匹配专家状态-动作分布 |
| 奖励函数 | 不需要 | 隐式地从判别器获得 |
| 分布偏移处理 | 无法处理 | 通过对抗训练动态修正 |
| 训练稳定性 | 稳定 | 需要GAN技巧（WGAN-GP等） |
| 计算代价 | 低 | 高（额外判别器网络） |

### 4.3 代码实现：GAIL

```python
"""
GAIL（生成对抗模仿学习）实现
基于PyTorch + CartPole环境
"""
import torch
import torch.nn as nn
import torch.optim as optim
import gymnasium as gym
import numpy as np
import os

# ============ 1. 判别器网络 ============
class Discriminator(nn.Module):
    """判别器：区分专家轨迹 vs 策略轨迹"""
    def __init__(self, state_dim, action_dim, hidden_dims=[128, 128]):
        super().__init__()
        self.action_dim = action_dim
        # 状态+动作的拼接输入
        input_dim = state_dim + action_dim
        layers = []
        prev_dim = input_dim
        for h_dim in hidden_dims:
            layers.extend([nn.Linear(prev_dim, h_dim), nn.ReLU()])
            prev_dim = h_dim
        layers.append(nn.Linear(prev_dim, 1))
        layers.append(nn.Sigmoid())
        self.network = nn.Sequential(*layers)
    
    def forward(self, state, action):
        """
        Args:
            state: (batch, state_dim)
            action: (batch,) 整数索引，自动转为one-hot
        Returns:
            D(s,a): 专家概率估计 (0~1)
        """
        # action 整数转 one-hot（离散动作）
        if action.dim() == 1:
            action_onehot = torch.nn.functional.one_hot(
                action.long(), num_classes=self.action_dim
            ).float()
        else:
            action_onehot = action
        concat = torch.cat([state, action_onehot], dim=-1)
        return self.network(concat)


# ============ 2. 策略网络 ============
class PolicyNetwork(nn.Module):
    """离散策略网络"""
    def __init__(self, state_dim, action_dim, hidden_dims=[128, 128]):
        super().__init__()
        self.action_dim = action_dim
        layers = []
        prev_dim = state_dim
        for h_dim in hidden_dims:
            layers.extend([nn.Linear(prev_dim, h_dim), nn.ReLU()])
            prev_dim = h_dim
        layers.append(nn.Linear(prev_dim, action_dim))
        layers.append(nn.Softmax(dim=-1))
        self.network = nn.Sequential(*layers)
    
    def forward(self, state):
        return self.network(state)
    
    def get_action(self, state):
        with torch.no_grad():
            probs = self.forward(state)
            return probs.argmax(dim=-1).item()


# ============ 3. 专家策略 ============
class ExpertPolicy:
    def get_action(self, state):
        x, x_dot, theta, theta_dot = state
        return 0 if theta < 0 else 1


# ============ 4. 收集轨迹数据 ============
def collect_expert_demonstrations(env, expert, num_episodes=50):
    """收集专家示教轨迹"""
    demonstrations = []
    
    for _ in range(num_episodes):
        state, _ = env.reset()
        episode_data = {'states': [], 'actions': []}
        terminated = False
        truncated = False
        
        while not (terminated or truncated):
            action = expert.get_action(state)
            episode_data['states'].append(state)
            episode_data['actions'].append(action)
            state, _, terminated, truncated, _ = env.step(action)
        
        episode_data['states'] = np.array(episode_data['states'])
        episode_data['actions'] = np.array(episode_data['actions'])
        demonstrations.append(episode_data)
    
    return demonstrations


def collect_policy_trajectories(env, policy, device, episodes=5):
    """用当前策略收集轨迹"""
    trajectories = []
    
    for _ in range(episodes):
        state, _ = env.reset()
        episode_data = {'states': [], 'actions': []}
        terminated = False
        truncated = False
        
        while not (terminated or truncated):
            state_t = torch.FloatTensor(state).unsqueeze(0).to(device)
            action_prob = policy(state_t)
            action = action_prob.argmax(dim=1).item()
            
            episode_data['states'].append(state)
            episode_data['actions'].append(action)
            state, _, terminated, truncated, _ = env.step(action)
        
        episode_data['states'] = np.array(episode_data['states'])
        episode_data['actions'] = np.array(episode_data['actions'])
        trajectories.append(episode_data)
    
    return trajectories


# ============ 5. GAIL训练 ============
def train_gail(env, expert_demos, state_dim, action_dim=2,
               num_iterations=100, trajectories_per_iter=5,
               discriminator_epochs=5, policy_epochs=5,
               lr_d=1e-3, lr_pi=1e-3):
    """GAIL训练主循环"""
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    print(f"使用设备: {device}")
    
    # 准备专家数据
    expert_states = np.concatenate([demo['states'] for demo in expert_demos])
    expert_actions = np.concatenate([demo['actions'] for demo in expert_demos])
    expert_states_t = torch.FloatTensor(expert_states).to(device)
    expert_actions_t = torch.LongTensor(expert_actions).to(device)
    
    # 网络初始化
    discriminator = Discriminator(state_dim, action_dim).to(device)
    policy = PolicyNetwork(state_dim, action_dim).to(device)
    
    optim_d = optim.Adam(discriminator.parameters(), lr=lr_d)
    optim_pi = optim.Adam(policy.parameters(), lr=lr_pi)
    
    criterion = nn.BCELoss()
    
    print("=" * 60)
    print("开始 GAIL 训练")
    print("=" * 60)
    
    for iteration in range(num_iterations):
        # ----- Step 1: 收集策略轨迹 -----
        policy_trajectories = collect_policy_trajectories(env, policy, device, trajectories_per_iter)
        
        pi_states = np.concatenate([t['states'] for t in policy_trajectories])
        pi_actions = np.concatenate([t['actions'] for t in policy_trajectories])
        pi_states_t = torch.FloatTensor(pi_states).to(device)
        pi_actions_t = torch.LongTensor(pi_actions).to(device)
        
        # ----- Step 2: 训练判别器 -----
        for _ in range(discriminator_epochs):
            optim_d.zero_grad()
            
            # 专家数据标签=1，策略数据标签=0
            expert_prob = discriminator(expert_states_t, expert_actions_t)
            policy_prob = discriminator(pi_states_t, pi_actions_t)
            
            loss_d = (criterion(expert_prob, torch.ones_like(expert_prob)) +
                       criterion(policy_prob, torch.zeros_like(policy_prob)))
            
            loss_d.backward()
            optim_d.step()
        
        # ----- Step 3: 训练策略（最大化判别器奖励） -----
        for _ in range(policy_epochs):
            optim_pi.zero_grad()
            
            # GAIL奖励 r = -log(1 - D)，希望策略数据被判别器认为是专家
            policy_prob = discriminator(pi_states_t, pi_actions_t)
            gail_reward = -torch.log(1 - policy_prob + 1e-8)
            loss_pi = -gail_reward.mean()  # 最大化奖励 = 最小化负奖励
            
            loss_pi.backward()
            optim_pi.step()
        
        # ----- 评估 -----
        if (iteration + 1) % 20 == 0:
            with torch.no_grad():
                expert_score = discriminator(expert_states_t, expert_actions_t).mean().item()
                policy_score = discriminator(pi_states_t, pi_actions_t).mean().item()
            
            avg_steps = np.mean([len(t['states']) for t in policy_trajectories])
            print(f"Iter {iteration+1:3d}/{num_iterations} | "
                  f"D_expert={expert_score:.3f} D_policy={policy_score:.3f} | "
                  f"avg_steps={avg_steps:.0f}")
    
    return policy, discriminator


if __name__ == "__main__":
    env = gym.make("CartPole-v1")
    state_dim = env.observation_space.shape[0]
    action_dim = env.action_space.n
    expert = ExpertPolicy()
    
    print("收集专家示教数据...")
    expert_demos = collect_expert_demonstrations(env, expert, num_episodes=50)
    print(f"专家数据: {sum(len(d['states']) for d in expert_demos)} 样本")
    
    policy, discriminator = train_gail(
        env, expert_demos, state_dim, action_dim=2,
        num_iterations=100,
        trajectories_per_iter=5,
        discriminator_epochs=5,
        policy_epochs=5
    )
    
    # 最终评估
    print("\n最终评估（20 episodes）:")
    device = torch.device("cpu")  # 评估用CPU
    policy.eval()
    
    eval_rewards = []
    for _ in range(20):
        state, _ = env.reset()
        steps = 0
        terminated = False
        truncated = False
        while not (terminated or truncated):
            state_t = torch.FloatTensor(state).unsqueeze(0).to(device)
            action = policy.get_action(state_t)
            state, _, terminated, truncated, _ = env.step(action)
            steps += 1
        eval_rewards.append(steps)
    
    print(f"平均存活步数: {np.mean(eval_rewards):.1f} ± {np.std(eval_rewards):.1f}")
    
    os.makedirs("models", exist_ok=True)
    torch.save(policy.state_dict(), "models/gail_policy_cartpole.pth")
    torch.save(discriminator.state_dict(), "models/gail_discriminator_cartpole.pth")
    print("模型已保存")
    
    env.close()
```

FRANKA 机械臂行为克隆示教学习
基于ROS2 + franka_msgs + PyTorch

import torch
import torch.nn as nn
import numpy as np
import pickle
import os

# ============ 1. 策略网络 ============
class PandaPolicyNetwork(nn.Module):
    """7自由度Panda机械臂策略网络"""
    def __init__(self, state_dim=14, action_dim=7, hidden_dims=[256, 256, 128]):
        """
        state_dim=14: 7关节位置 + 7关节速度
        action_dim=7: 7关节目标位置增量
        """
        super().__init__()
        layers = []
        prev_dim = state_dim
        for h_dim in hidden_dims:
            layers.extend([nn.Linear(prev_dim, h_dim), nn.ReLU()])
            prev_dim = h_dim
        layers.append(nn.Linear(prev_dim, action_dim))
        layers.append(nn.Tanh())  # 输出 [-1, 1]
        self.network = nn.Sequential(*layers)
        
        # 关节限位（Franka Panda）
        self.joint_limits_upper = np.array([2.8973, 1.7628, 2.8973, -0.0698, 2.8973, 3.7525, 2.8973])
        self.joint_limits_lower = np.array([-2.8973, -1.7628, -2.8973, -3.0718, -2.8973, -0.0175, -2.8973])
    
    def forward(self, state):
        return self.network(state)
    
    def action_to_joint_targets(self, action, current_joint_pos):
        """将归一化动作 [-1,1] 转换为关节目标位置"""
        delta = action * 0.1  # 最大步长 0.1 rad
        target = current_joint_pos + delta
        # 限位裁剪
        target = np.clip(target, self.joint_limits_lower, self.joint_limits_upper)
        return target

### 简答题答案

**6. 分布偏移问题的产生与DAgger的解决思路**

**问题产生**：行为克隆将模仿学习当作监督分类/回归，数据集 $\mathcal{D} = \{(s_i, a_i)\}$ 来自专家轨迹。训练时策略在专家访问过的状态分布上优化，但实际执行时，策略的动作误差会使下一状态偏离训练分布。这种偏差随时间步累积，最终导致策略进入从未训练过的区域，性能急剧下降。

**DAgger的解决思路**：不依赖一次性收集的固定数据集，而是在线迭代：
1. 用当前策略 $\pi_i$ 收集轨迹 $\tau$
2. 在 $\tau$ 的每个状态 $s$ 处查询专家得到最优动作 $a^* = \pi^*(s)$
3. 将 $(s, a^*)$ 加入训练集

这样即使策略偏离了专家轨迹，也能立即获得该偏离状态下的专家修正数据，从而逐步适应更广的状态分布。

---

**7. BC、DAgger、GAIL 处理分布偏移的对比**

| 方法 | 策略 | 分布偏移处理能力 |
|------|------|----------------|
| **BC** | 固定数据集，一次性监督学习 | 弱，误差累积无法修正 |
| **DAgger** | 在线迭代查询专家，动态扩展数据集 | 强，但需要专家全程在线 |
| **GAIL** | 对抗训练，判别器作为隐式奖励 | 强，能隐式匹配专家分布，但训练不稳定 |

核心差异：BC是**被动接受**已有数据，DAgger是**主动查询修正**，GAIL是**对抗式自适应**。

---

**8. 家庭服务机器人模仿学习系统设计**

**数据采集层**：
- 动作捕捉方案：Azure Kinect / RealSense D455 采集人体操作视频
- 示教方式：人类演示家务动作（折叠衣物、摆放餐具），同步录制RGB-D + 关节状态
- 动作映射：人体17关键点 → 机械臂7关节，通过逆运动学或末端执行器对齐

**算法选择**：
- 主算法：GAIL（学习隐式奖励，无需手动标注动作优劣）
- 在线优化：DAgger微调（处理新场景中的分布偏移）
- 安全层：基于力传感的阻抗控制，碰撞时自动停止

**部署流程**：
1. 离线训练：用采集的示教数据训练GAIL策略
2. 仿真验证：在Gazebo/PyBullet中验证动作安全性
3. 零样本部署：Sim-to-Real迁移（领域随机化：光照、物体位置）
4. 在线适应：运行时通过触觉反馈微调策略

## 本章小结

| 概念 | 要点 |
|------|------|
| **模仿学习** | 通过专家示范数据学习策略，不依赖奖励函数 |
| **行为克隆（BC）** | 监督分类/回归，简单但存在分布偏移 |
| **DAgger** | 迭代式在线查询专家修正，动态适应新状态 |
| **GAIL** | 生成对抗模仿学习，判别器输出隐式奖励 |
| **示教采集** | 遥操作/动觉示教/动作捕捉/视频学习 |
| **分布偏移** | 误差累积是BC的核心瓶颈，DAgger/GAIL从不同角度解决 |
| **Sim-to-Real** | 领域随机化 + 在线微调弥合仿真-现实差距 |
