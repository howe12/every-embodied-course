## 1. 规划器接口详解

Nav2采用插件化架构，规划器通过统一的接口进行交互。理解这些接口是开发自定义规划器的基础。

### 1.1 Nav2规划器架构概述

Nav2中的路径规划分为两个层次：**全局路径规划（Global Planner）**和**局部路径规划（Local Planner）**。全局规划器负责生成从起点到目标点的完整路径，局部规划器则负责根据实时感知信息生成避障轨迹。

```
┌─────────────────────────────────────────────────────────────────┐
│                        Navigation Stack                          │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ┌──────────────┐         ┌──────────────┐                    │
│   │  Global      │         │   Local      │                    │
│   │  Planner     │────────▶│   Planner    │                    │
│   │ (插件化)      │         │  (插件化)     │                    │
│   └──────┬───────┘         └──────┬───────┘                    │
│          │                        │                             │
│          ▼                        ▼                             │
│   ┌──────────────┐         ┌──────────────┐                    │
│   │   Costmap2D  │         │   Costmap2D  │                    │
│   │  (全局代价图) │         │  (局部代价图) │                    │
│   └──────────────┘         └──────────────┘                    │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

**全局规划器的职责**：
- 接收起点和目标点
- 在全局代价地图上搜索可行路径
- 生成一系列路点（waypoints）
- 优化路径长度和平滑度

**局部规划器的职责**：
- 接收全局路径作为参考
- 根据局部代价地图实时调整
- 生成短期速度和转向命令
- 处理动态障碍物避让

### 1.2 全局规划器接口

全局规划器需要继承`nav2_core::GlobalPlanner`基类，并实现以下核心方法：

| 方法 | 返回类型 | 描述 |
|------|----------|------|
| `configure()` | void | 初始化规划器参数和资源 |
| `cleanup()` | void | 清理分配的资源 |
| `activate()` | void | 激活规划器（开始工作） |
| `deactivate()` | void | 停用规划器（暂停工作） |
| `createPlan()` | geometry_msgs::msg::Path | 创建路径 |

`createPlan()`方法的签名如下：

```cpp
// nav2_core::GlobalPlanner::createPlan
geometry_msgs::msg::Path createPlan(
  const geometry_msgs::msg::PoseStamped & start,
  const geometry_msgs::msg::PoseStamped & goal
);
```

**参数说明**：
- `start`：起始位姿，包含位置和方向
- `goal`：目标位姿
- 返回值：包含一系列路点的Path消息

### 1.3 局部规划器接口

局部规划器需要继承`nav2_controller::Controller`基类（注意：Nav2中局部规划器也称为Controller）：

| 方法 | 返回类型 | 描述 |
|------|----------|------|
| `configure()` | void | 初始化控制器参数 |
| `cleanup()` | void | 清理资源 |
| `activate()` | void | 激活控制器 |
| `deactivate()` | void | 停用控制器 |
| `computeVelocityCommands()` | geometry_msgs::msg::Twist | 计算速度命令 |
| `setPlan()` | void | 接收新的路径参考 |

`computeVelocityCommands()`方法的核心签名：

```cpp
// nav2_controller::Controller::computeVelocityCommands
geometry_msgs::msg::Twist computeVelocityCommands(
  const geometry_msgs::msg::PoseStamped & pose,    // 当前机器人位姿
  const geometry_msgs::msg::Twist & velocity      // 当前速度
);
```

### 1.4 规划器配置参数结构

全局规划器通常需要以下配置参数：

```yaml
planner_server:
  ros__parameters:
    # 全局规划器类型
    planner_plugin_types: ['nav2_smac_planner/SmacPlanner']
    planner_plugin_ids: ['grid_global']
    
    # SmacPlanner参数
    grid_global:
      tolerance: 0.5  # 目标容差（米）
      downsample_costmap: false
      downsampling_factor: 1
      max_iterations: 1000000
      max_on_approach_iterations: 1000
      allow_unknown: true
```

局部规划器配置：

```yaml
controller_server:
  ros__parameters:
    # 局部规划器（控制器）类型
    controller_plugin_types: ['nav2_dwb_controller/DWBLocalPlanner']
    controller_plugin_ids: ['FollowPath']
    
    # DWB参数
    FollowPath:
      min_vel_x: 0.0
      max_vel_x: 0.26
      max_vel_theta: 1.0
      min_speed_theta: 0.0
```

## 2. 插件开发流程

Nav2使用插件系统（pluginlib）加载规划器，这使得开发者可以将自定义规划器编译为共享库，运行时动态加载。

### 2.1 插件系统概述

ROS2的插件系统基于`pluginlib`库，实现了以下功能：
- **运行时加载**：无需重新编译核心导航栈
- **热插拔**：可以随时切换不同的规划器
- **解耦设计**：规划器与导航框架独立开发

```
┌─────────────────┐
│  Nav2 Core      │  ← 核心导航框架
│  (pluginlib)    │
└────────┬────────┘
         │
         │ 动态加载
         ▼
┌─────────────────┐    ┌─────────────────┐
│  My Planner     │    │  Built-in       │
│  (plugin.so)    │    │  Planner        │
└─────────────────┘    └─────────────────┘
```

## 3. 全局规划器实现

本节详细讲解如何实现一个自定义全局规划器。我们将实现一个基于A*算法的全局规划器。

### 3.1 头文件定义

```cpp
// include/my_planner_plugins/my_global_planner.hpp
#ifndef MY_GLOBAL_PLANNER_HPP_
#define MY_GLOBAL_PLANNER_HPP_

#include <string>
#include <memory>
#include <vector>

#include "rclcpp/rclcpp.hpp"
#include "nav2_core/global_planner.hpp"
#include "geometry_msgs/msg/pose_stamped.hpp"
#include "nav_msgs/msg/path.hpp"
#include "nav2_costmap_2d/costmap_2d_ros.hpp"

namespace my_planner_plugins
{

/**
 * @brief 自定义全局路径规划器
 * 
 * 实现基于A*算法的全局路径规划
 */
class MyGlobalPlanner : public nav2_core::GlobalPlanner
{
public:
  /**
   * @brief 构造函数
   */
  MyGlobalPlanner() = default;

  /**
   * @brief 析构函数
   */
  ~MyGlobalPlanner() override = default;

  /**
   * @brief 配置规划器
   * @param parent 父节点指针
   * @param name 节点名称
   * @param tf 坐标变换缓存
   * @param costmap_ros 全局代价地图
   */
  void configure(
    rclcpp_lifecycle::LifecycleNode::SharedPtr parent,
    std::string name,
    std::shared_ptr<tf2_ros::Buffer> tf,
    std::shared_ptr<nav2_costmap_2d::Costmap2DROS> costmap_ros) override;

  /**
   * @brief 清理资源
   */
  void cleanup() override;

  /**
   * @brief 激活规划器
   */
  void activate() override;

  /**
   * @brief 停用规划器
   */
  void deactivate() override;

  /**
   * @brief 创建路径规划
   * @param start 起始位姿
   * @param goal 目标位姿
   * @return 规划的路径
   */
  geometry_msgs::msg::Path createPlan(
    const geometry_msgs::msg::PoseStamped & start,
    const geometry_msgs::msg::PoseStamped & goal) override;

private:
  /**
   * @brief A*搜索算法
   * @param start_x 起始x坐标（网格坐标）
   * @param start_y 起始y坐标
   * @param goal_x 目标x坐标
   * @param goal_y 目标y坐标
   * @return 路径点集合
   */
  std::vector<std::pair<int, int>> aStarSearch(
    int start_x, int start_y,
    int goal_x, int goal_y);

  /**
   * @brief 将网格坐标转换为世界坐标
   */
  geometry_msgs::msg::PoseStamped gridToWorld(
    unsigned int mx, unsigned int my,
    const geometry_msgs::msg::PoseStamped & goal);

  /**
   * @brief 检查网格点是否可行
   */
  bool isCellValid(unsigned int mx, unsigned int my);

  // 节点和tf
  rclcpp_lifecycle::LifecycleNode::SharedPtr parent_;
  std::string name_;
  std::shared_ptr<tf2_ros::Buffer> tf_;
  
  // 代价地图
  nav2_costmap_2d::Costmap2D * costmap_;
  unsigned int global_frame_id_;

  // 参数
  double tolerance_;
  bool allow_unknown_;
  int max_iterations_;
};

}  // namespace my_planner_plugins

#endif  // MY_GLOBAL_PLANNER_HPP_
```

### 3.2 源文件实现

```cpp
// src/my_global_planner.cpp
#include "my_planner_plugins/my_global_planner.hpp"

#include <cmath>
#include <queue>
#include <vector>

void MyGlobalPlanner::configure(
  rclcpp_lifecycle::LifecycleNode::SharedPtr parent,
  std::string name,
  std::shared_ptr<tf2_ros::Buffer> tf,
  std::shared_ptr<nav2_costmap_2d::Costmap2DROS> costmap_ros)
{
  parent_ = parent;
  name_ = name;
  tf_ = tf;
  costmap_ = costmap_ros->getCostmap();
  global_frame_id_ = costmap_ros->getGlobalFrameID();

  // 获取参数
  nav2_util::declare_parameter_if_not_declared(
    parent_, name_ + ".tolerance", rclcpp::ParameterValue(0.5));
  nav2_util::declare_parameter_if_not_declared(
    parent_, name_ + ".allow_unknown", rclcpp::ParameterValue(true));
  nav2_util::declare_parameter_if_not_declared(
    parent_, name_ + ".max_iterations", rclcpp::ParameterValue(1000000));

  parent_->get_parameter(name_ + ".tolerance", tolerance_);
  parent_->get_parameter(name_ + ".allow_unknown", allow_unknown_);
  parent_->get_parameter(name_ + ".max_iterations", max_iterations_);

  RCLCPP_INFO(parent_->get_logger(), 
    "MyGlobalPlanner configured with tolerance=%.2f", tolerance_);
}

  RCLCPP_INFO(parent_->get_logger(), "MyGlobalPlanner activated");

geometry_msgs::msg::Path MyGlobalPlanner::createPlan(
  const geometry_msgs::msg::PoseStamped & start,
  const geometry_msgs::msg::PoseStamped & goal)
{
  geometry_msgs::msg::Path path;

  // 验证坐标系
  if (start.header.frame_id != global_frame_id_ ||
      goal.header.frame_id != global_frame_id_)
  {
    RCLCPP_ERROR(parent_->get_logger(),
      "Start or goal frame mismatch. Expected: %s, Got: %s and %s",
      global_frame_id_.c_str(),
      start.header.frame_id.c_str(),
      goal.header.frame_id.c_str());
    return path;
  }

  // 获取起点和终点的网格坐标
  unsigned int start_mx, start_my, goal_mx, goal_my;
  if (!costmap_->worldToMap(start.pose.position.x, start.pose.position.y,
                            start_mx, start_my) ||
      !costmap_->worldToMap(goal.pose.position.x, goal.pose.position.y,
                            goal_mx, goal_my))
  {
    RCLCPP_WARN(parent_->get_logger(),
      "Start or goal position out of bounds");
    return path;
  }

  // 检查目标点是否可行
  if (!isCellValid(goal_mx, goal_my) || costmap_->getCost(goal_mx, goal_my) == nav2_costmap_2d::LETHAL_OBSTACLE)
  {
    RCLCPP_WARN(parent_->get_logger(), "Goal is in an obstacle");
    return path;
  }

  // 执行A*搜索
  RCLCPP_INFO(parent_->get_logger(),
    "Planning from (%d,%d) to (%d,%d)", start_mx, start_my, goal_mx, goal_my);

  auto cells = aStarSearch(start_mx, start_my, goal_mx, goal_my);

  if (cells.empty())
  {
    RCLCPP_WARN(parent_->get_logger(), "No path found!");
    return path;
  }

  // 将网格路径转换为PoseStamped序列
  for (auto it = cells.rbegin(); it != cells.rend(); ++it)
  {
    auto pose = gridToWorld(it->first, it->second, goal);
    path.poses.push_back(pose);
  }

  // 设置路径头信息
  path.header.stamp = parent_->now();
  path.header.frame_id = global_frame_id_;

  RCLCPP_INFO(parent_->get_logger(), "Path found with %zu points", path.poses.size());

  return path;
}

// A*算法实现
struct Node
{
  int x, y;
  double g;  // 从起点到当前点的代价
  double h;  // 从当前点到目标的启发式代价
  double f;  // 总代价 = g + h
  Node* parent;

  Node(int x_, int y_, double g_, double h_, Node* parent_)
    : x(x_), y(y_), g(g_), h(h_), f(g_ + h_), parent(parent_) {}

  bool operator<(const Node& other) const
  {
    return f > other.f;  // 最小堆
  }
};

std::vector<std::pair<int, int>> MyGlobalPlanner::aStarSearch(
  int start_x, int start_y,
  int goal_x, int goal_y)
{
  std::priority_queue<Node> open_set;
  std::unordered_map<int, Node*> all_nodes;
  std::set<std::pair<int, int>> closed_set;

  auto cost = [&](int x, int y) -> double {
    unsigned char c = costmap_->getCost(x, y);
    if (c == nav2_costmap_2d::LETHAL_OBSTACLE) return 1e9;
    if (c == nav2_costmap_2d::NO_INFORMATION) {
      return allow_unknown_ ? 1.0 : 1e9;
    }
    return static_cast<double>(c) / 254.0;
  };

  auto heuristic = [&](int x, int y) -> double {
    return std::hypot(x - goal_x, y - goal_y);
  };

  auto hash = [&](int x, int y) -> int {
    return y * costmap_->getSizeInCellsX() + x;
  };

  // 创建起点节点
  Node* start_node = new Node(start_x, start_y, 0.0, heuristic(start_x, start_y), nullptr);
  open_set.push(*start_node);
  all_nodes[hash(start_x, start_y)] = start_node;

  // 8连通方向
  const int dx[8] = {-1, 0, 1, -1, 1, -1, 0, 1};
  const int dy[8] = {-1, -1, -1, 0, 0, 1, 1, 1};

  int iterations = 0;
  while (!open_set.empty() && iterations < max_iterations_)
  {
    iterations++;
    Node current = open_set.top();
    open_set.pop();

    int cx = current.x;
    int cy = current.y;

    // 检查是否到达目标
    if (cx == goal_x && cy == goal_y)
    {
      std::vector<std::pair<int, int>> path;
      Node* node = all_nodes[hash(cx, cy)];
      while (node != nullptr)
      {
        path.push_back({node->x, node->y});
        node = node->parent;
      }

      // 清理内存
      for (auto& pair : all_nodes) {
        delete pair.second;
      }
      return path;
    }

    closed_set.insert({cx, cy});

    // 扩展邻居节点
    for (int i = 0; i < 8; i++)
    {
      int nx = cx + dx[i];
      int ny = cy + dy[i];

      // 检查边界
      if (nx < 0 || nx >= static_cast<int>(costmap_->getSizeInCellsX()) ||
          ny < 0 || ny >= static_cast<int>(costmap_->getSizeInCellsY()))
        continue;

      // 检查是否已访问或不可通行
      if (closed_set.count({nx, ny}) || !isCellValid(nx, ny))
        continue;

      double move_cost = (i < 4) ? 1.0 : 1.414;  // 对角线代价
      double new_g = current.g + cost(nx, ny) * move_cost;

      Node* existing = all_nodes[hash(nx, ny)];
      if (existing == nullptr || new_g < existing->g)
      {
        Node* neighbor = new Node(nx, ny, new_g, heuristic(nx, ny),
                                   all_nodes[hash(cx, cy)]);
        all_nodes[hash(nx, ny)] = neighbor;
        open_set.push(*neighbor);
      }
    }
  }

  // 清理内存
  for (auto& pair : all_nodes) {
    delete pair.second;
  }

  RCLCPP_WARN(parent_->get_logger(), "A* search exhausted");
  return {};
}

bool MyGlobalPlanner::isCellValid(unsigned int mx, unsigned int my)
{
  unsigned char cost = costmap_->getCost(mx, my);
  return cost != nav2_costmap_2d::LETHAL_OBSTACLE;
}

geometry_msgs::msg::PoseStamped MyGlobalPlanner::gridToWorld(
  unsigned int mx, unsigned int my,
  const geometry_msgs::msg::PoseStamped & goal)
{
  geometry_msgs::msg::PoseStamped pose;
  pose.header.frame_id = global_frame_id_;
  pose.header.stamp = parent_->now();

  double wx, wy;
  costmap_->mapToWorld(mx, my, wx, wy);
  pose.pose.position.x = wx;
  pose.pose.position.y = wy;
  pose.pose.position.z = 0.0;
  pose.pose.orientation.x = 0.0;
  pose.pose.orientation.y = 0.0;
  pose.pose.orientation.z = 0.0;
  pose.pose.orientation.w = 1.0;

  return pose;
}
```

```cpp
geometry_msgs::msg::Twist MyLocalPlanner::computeVelocityCommands(
  const geometry_msgs::msg::PoseStamped & pose,
  const geometry_msgs::msg::Twist & velocity)
{
  geometry_msgs::msg::Twist cmd;

  if (global_path_.poses.empty())
  {
    RCLCPP_WARN(node_->get_logger(), "No path to follow");
    return cmd;
  }

  // 找到最近的路点
  double min_dist = std::numeric_limits<double>::max();
  size_t closest_idx = current_goal_index_;
  
  for (size_t i = current_goal_index_; i < global_path_.poses.size(); ++i)
  {
    double dx = pose.pose.position.x - global_path_.poses[i].pose.position.x;
    double dy = pose.pose.position.y - global_path_.poses[i].pose.position.y;
    double dist = std::hypot(dx, dy);
    
    if (dist < min_dist)
    {
      min_dist = dist;
      closest_idx = i;
    }
  }

  // 更新当前目标索引
  current_goal_index_ = closest_idx;

  // 检查是否到达最终目标
  if (current_goal_index_ >= global_path_.poses.size() - 1)
  {
    double dx = pose.pose.position.x - global_path_.poses.back().pose.position.x;
    double dy = pose.pose.position.y - global_path_.poses.back().pose.position.y;
    double dist = std::hypot(dx, dy);
    
    if (dist < goal_tolerance_)
    {
      goal_reached_ = true;
      RCLCPP_INFO(node_->get_logger(), "Goal reached!");
      return cmd;
    }
  }

  // 获取目标路点
  size_t target_idx = std::min(
    current_goal_index_ + 1,
    global_path_.poses.size() - 1
  );
  auto target_pose = global_path_.poses[target_idx];

  // 计算朝向误差
  double heading_error = computeHeadingError(pose.pose, target_pose);

  // 简单的P控制器
  double angular_gain = 2.0;
  double linear_gain = 0.5;

  cmd.angular.z = std::clamp(
    angular_gain * heading_error,
    -max_angular_vel_,
    max_angular_vel_
  );

  // 线性速度根据角速度调整
  double speed_factor = 1.0 - std::abs(heading_error) / M_PI;
  cmd.linear.x = max_linear_vel_ * speed_factor * linear_gain;
  cmd.linear.x = std::clamp(cmd.linear.x, min_linear_vel_, max_linear_vel_);

  // 障碍物检测
  if (detectObstacle())
  {
    cmd.linear.x = 0.0;
    cmd.angular.z = max_angular_vel_ * ((heading_error > 0) ? 1.0 : -1.0);
    RCLCPP_WARN(node_->get_logger(), "Obstacle detected, rotating in place");
  }

  return cmd;
}

bool MyLocalPlanner::isGoalReached()
{
  return goal_reached_;
}

double MyLocalPlanner::computeHeadingError(
  const geometry_msgs::msg::Pose & current,
  const geometry_msgs::msg::PoseStamped & target)
{
  // 计算目标方向
  double target_yaw = std::atan2(
    target.pose.position.y - current.position.y,
    target.pose.position.x - current.position.x
  );

  // 获取当前朝向
  tf2::Quaternion q;
  tf2::fromMsg(current.orientation, q);
  tf2::Matrix3x3 m(q);
  double roll, pitch, current_yaw;
  m.getRPY(roll, pitch, current_yaw);

  // 计算角度差
  double error = target_yaw - current_yaw;

  // 归一化到[-π, π]
  while (error > M_PI) error -= 2 * M_PI;
  while (error < -M_PI) error += 2 * M_PI;

  return error;
}

bool MyLocalPlanner::detectObstacle()
{
  nav2_costmap_2d::Costmap2D * costmap = costmap_ros_->getCostmap();
  unsigned int mx, my;
  
  // 检查机器人前方的障碍物
  costmap->worldToMap(
    costmap->getOriginX() + costmap->getSizeInMetersX() / 2,
    costmap->getOriginY() + costmap->getSizeInMetersY() / 2,
    mx, my
  );

  unsigned char cost = costmap->getCost(mx, my);
  return cost >= nav2_costmap_2d::LETHAL_OBSTACLE;
}
```

## 6. 实战：自定义DWA局部规划器

本节实现一个更完整的DWA（Dynamic Window Approach）局部规划器。

### 6.1 DWA算法原理

DWA算法的核心是在速度空间中采样可行的速度组合，评估每个速度组合的得分，选择最优的速度命令。

**速度空间采样**：

$$V_s = \{v \in [v_{min}, v_{max}] \times \omega \in [\omega_{min}, \omega_{max}] | v \cdot \Delta t \leq d(v, \omega)\}$$

其中 $d(v, \omega)$ 是给定速度下机器人能在制动距离内停下的距离。

**评价函数**：

$$G(v, \omega) = \sigma(\alpha \cdot heading(v, \omega) + \beta \cdot dist(v, \omega) + \gamma \cdot velocity(v, \omega))$$

- $heading$：朝向目标的角度得分
- $dist$：到最近障碍物的距离得分
- $velocity$：速度大小得分

### 6.2 DWA规划器实现

```cpp

geometry_msgs::msg::Twist DWAPlanner::computeVelocityCommands(
  const geometry_msgs::msg::PoseStamped & pose,
  const geometry_msgs::msg::Twist & velocity)
{
  geometry_msgs::msg::Twist cmd;

  // 获取当前速度
  double vx = velocity.linear.x;
  double vy = velocity.linear.y;
  double vtheta = velocity.angular.z;

  // 计算动态窗口
  double min_v = std::max(min_speed_xy_, vx - acc_lim_x_ * sim_time_);
  double max_v = std::min(max_speed_xy_, vx + acc_lim_x_ * sim_time_);
  double min_theta = std::max(min_speed_theta_, vtheta - acc_lim_theta_ * sim_time_);
  double max_theta = std::min(max_speed_theta_, vtheta + acc_lim_theta_ * sim_time_);

  // 采样速度空间
  double best_score = -std::numeric_limits<double>::infinity();
  double best_vx = 0.0;
  double best_vtheta = 0.0;

  int vx_samples = vx_samples_;
  int vtheta_samples = vtheta_samples_;

  for (int i = 0; i < vx_samples; i++)
  {
    double v = min_v + (max_v - min_v) * i / (vx_samples - 1);
    
    for (int j = 0; j < vtheta_samples; j++)
    {
      double vth = min_theta + (max_theta - min_theta) * j / (vtheta_samples - 1);
      
      // 模拟轨迹
      Trajectory traj = simulateTrajectory(pose, v, vth);
      
      // 评估得分
      double score = scoreTrajectory(traj, pose);
      
      if (score > best_score)
      {
        best_score = score;
        best_vx = v;
        best_vtheta = vth;
      }
    }
  }

  cmd.linear.x = best_vx;
  cmd.linear.y = 0.0;
  cmd.angular.z = best_vtheta;

  return cmd;
}

Trajectory DWAPlanner::simulateTrajectory(
  const geometry_msgs::msg::PoseStamped & pose,
  double vx, double vtheta)
{
  Trajectory traj;
  
  unsigned int num_steps = static_cast<unsigned int>(sim_time_ / sim_granularity_);
  
  double x = pose.pose.position.x;
  double y = pose.pose.position.y;
  
  // 获取当前偏航角
  tf2::Quaternion q;
  tf2::fromMsg(pose.pose.orientation, q);
  tf2::Matrix3x3 m(q);
  double yaw, pitch, roll;
  m.getRPY(roll, pitch, yaw);
  
  nav2_costmap_2d::Costmap2D * costmap = costmap_ros_->getCostmap();
  
  for (unsigned int i = 0; i < num_steps; i++)
  {
    double t = i * sim_granularity_;
    
    // 积分计算位置
    double dx = vx * std::cos(yaw) * sim_granularity_;
    double dy = vx * std::sin(yaw) * sim_granularity_;
    double dyaw = vtheta * sim_granularity_;
    
    x += dx;
    y += dy;
    yaw += dyaw;
    
    // 检查是否撞墙
    unsigned int mx, my;
    if (costmap->worldToMap(x, y, mx, my))
    {
      unsigned char cost = costmap->getCost(mx, my);
      traj.cost = cost;
      if (cost >= nav2_costmap_2d::LETHAL_OBSTACLE)
      {
        break;
      }
    }
    
    traj.points.push_back({x, y, yaw});
  }
  
  return traj;
}

double DWAPlanner::scoreTrajectory(
  const Trajectory & traj,
  const geometry_msgs::msg::PoseStamped & pose)
{
  if (traj.points.empty())
    return -std::numeric_limits<double>::infinity();
  
  // 障碍物距离得分
  double occdist = 0.0;
  if (traj.cost >= 0)
  {
    occdist = 1.0 - (traj.cost / 255.0);
  }
  
  // 目标朝向得分
  double last_point = traj.points.back();
  double target_yaw = std::atan2(
    last_point.y - pose.pose.position.y,
    last_point.x - pose.pose.position.x
  );
  
  tf2::Quaternion q;
  tf2::fromMsg(pose.pose.orientation, q);
  tf2::Matrix3x3 m(q);
  double roll, pitch, yaw;
  m.getRPY(roll, pitch, yaw);
  
  double heading = M_PI - std::abs(normalizeAngle(target_yaw - yaw));
  
  // 速度得分
  double velocity = std::sqrt(traj.points.back().x * traj.points.back().x + 
                              traj.points.back().y * traj.points.back().y);
  
  // 综合得分
  double score = path_distance_bias_ * occdist + 
                goal_distance_bias_ * heading + 
                occdist_scale_ * velocity;
  
  return score;
}

double DWAPlanner::normalizeAngle(double angle)
{
  while (angle > M_PI) angle -= 2 * M_PI;
  while (angle < -M_PI) angle += 2 * M_PI;
  return angle;
}

}  // namespace my_planner_plugins

// 注册插件
#include <pluginlib/class_list_macros.hpp>
PLUGINLIB_EXPORT_CLASS(my_planner_plugins::DWAPlanner, nav2_controller::Controller)
```

## 7. 调试与优化技巧

### 7.1 常见问题排查

**问题1：规划器插件无法加载**

```bash
# 检查插件是否注册
ros2 plugin list | grep planner

# 查看详细的加载错误
ros2 launch nav2_bringup navigation_launch.py --ros-args -r __log_level:=debug
```

**问题2：路径规划失败**

- 检查代价地图是否正确生成
- 验证起点和目标点是否在可通行区域
- 增加`max_iterations`参数值

**问题3：机器人行为不稳定**

- 调整局部规划器的采样参数
- 增加`sim_time`以获得更长的预测视界
- 调整评价函数的权重参数

### 7.2 性能优化建议

1. **代价地图分辨率**：在保证导航精度的前提下，尽量使用较低的分辨率
2. **采样点数**：减少`vx_samples`和`vtheta_samples`可以提高计算速度
3. **预测时间**：`sim_time`不宜过长，否则会增加计算负担
4. **多线程**：对于复杂算法，可以考虑使用OpenMP并行化
