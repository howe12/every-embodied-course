## 1. 高级参数调优概述

导航系统的性能不仅取决于定位精度和环境感知质量，规划算法和控制器的参数配置同样至关重要。本节将介绍Nav2导航框架中的高级参数调优方法，涵盖全局规划器、局部规划器和轨迹跟踪控制器的核心参数。

### 1.1 参数调优的重要性

在机器人导航系统中，参数调优直接影响以下关键指标：

| 指标 | 影响因素 |
|------|----------|
| **路径长度** | 全局规划器启发函数、障碍物成本权重 |
| **执行时间** | 规划算法复杂度、采样分辨率 |
| **平滑度** | 路径平滑参数、控制器增益 |
| **安全性** | 障碍物膨胀、预测时域、控制约束 |
| **能耗** | 路径曲率、启停频率、加速度限制 |

合理的参数配置可以使机器人在复杂环境中高效、安全地自主导航；而不当的参数则可能导致路径不平滑、频繁停止甚至碰撞。

### 1.2 Nav2参数层次结构

Nav2的参数配置采用层次化结构，主要分为三个级别：

```yaml
# 完整的Nav2参数层次
nav2:
  bt_navigator:
    ...  # 行为树配置
  
  planner_server:
    planner_plugins: ["grid_based"]
    grid_based:
      ...  # 全局规划器参数
  
  controller_server:
    controller_plugins: ["FollowPath"]
    FollowPath:
      ...  # 局部控制器参数
  
  smoother_server:
    smoother_plugins: ["smoother"]
    smoother:
      ...  # 路径平滑参数
  
  costmap_providers:
    obstacle_layer:
      ...  # 代价地图参数
```

理解这个层次结构是进行有效参数调优的基础。

## 2. 规划器参数详解

全局规划器负责生成从起点到目标点的完整路径，而局部规划器则负责在局部范围内进行动态避障。本节将深入讲解这两类规划器的核心参数。

### 2.1 全局规划器参数

全局规划器（通常使用SMAC或NavFn）负责在静态地图上搜索可行路径。以下是关键的全局规划器参数：

#### 2.1.1 SMAC Planner参数

SMAC（Symbolic Model Abstraction Cache） planner是一种基于搜索的全局规划器，支持2D和混合A*算法。

```yaml
# smac_planner.yaml
planner_server:
  ros__parameters:
    planner_plugins: ["grid_based"]
    
    grid_based:
      plugin: "nav2_smac_planner/SmacPlanner"
      
      # 基础参数
      tolerance: 0.5  # 目标点容差（米）
      downsample_costmap: true  # 是否降采样代价地图
      downsampling_factor: 2  # 降采样因子
      
      # A*搜索参数
      max_iterations: 1000000  # 最大迭代次数
      max_on_approach_iterations: 1000  # 接近目标时的最大迭代
      allow_unknown: true  # 是否允许穿越未知区域
      
      # 成本权重
      neutral_cost: 50  # 中性代价值
      cost_multiplier: 3.0  # 成本乘数（越高越倾向绕行）
      
      # 启发函数
      heuristic_function: "2d"  # 启发函数类型：2d, diagonal, null
      
      # 角度参数
      angle_quantization_bins: 72  # 角度量化bin数
      promote_planner: true  # 是否使用改进版 planner
      
      # 缓存与优化
      cache_obstacle_heuristic: true  # 缓存障碍物启发值
      reverse_penalty: 2.1  # 反向移动惩罚因子
      change_penalty: 1.2  # 方向改变惩罚因子
      non_straight_penalty: 1.05  # 非直行惩罚
      cost_before_correction: 1.0  # 修正前的成本
```

**参数调优建议**：

| 参数 | 推荐值范围 | 调整原则 |
|------|-----------|----------|
| `tolerance` | 0.2-1.0 | 环境越复杂，容差越大 |
| `cost_multiplier` | 1.0-5.0 | 高值=安全优先，低值=路径最短 |
| `max_iterations` | 100000-10000000 | 根据地图大小调整，避免超时 |
| `angle_quantization_bins` | 16-72 | 精细转向需要更多bins |

#### 2.1.2 全局规划器性能优化

在大面积地图中，全局规划的计算可能成为瓶颈。以下是一些优化策略：

```yaml
# 全局规划器优化配置
grid_based:
  # 1. 降采样 - 减少搜索空间
  downsample_costmap: true
  downsampling_factor: 2  # 分辨率降低2倍
  
  # 2. 使用改进启发函数
  cache_obstacle_heuristic: true  # 缓存已计算启发值
  heuristic_function: "2d"  # 2D欧几里得启发最快
  
  # 3. 限制搜索范围
  max_iterations: 500000  # 限制最大迭代
  max_on_approach_iterations: 500  # 快速终止
  
  # 4. 允许穿越未知区域（在探索场景）
  allow_unknown: true
```

## 4. 性能评估指标

科学评估导航系统的性能需要量化的指标。本节将介绍导航性能评估的核心指标及其计算方法。

### 4.1 路径质量指标

#### 4.1.1 路径长度

路径长度是衡量导航效率的基本指标：

$$L = \sum_{i=1}^{n-1} \sqrt{(x_{i+1}-x_i)^2 + (y_{i+1}-y_i)^2}$$

其中 $(x_i, y_i)$ 是路径上第 $i$ 个点的坐标。

```python
# 计算路径长度
def calculate_path_length(path):
    """计算路径总长度
    
    Args:
        path: nav_msgs/Path 消息
        
    Returns:
        float: 路径长度（米）
    """
    if len(path.poses) < 2:
        return 0.0
    
    total_length = 0.0
    for i in range(1, len(path.poses)):
        prev = path.poses[i-1].pose.position
        curr = path.poses[i].pose.position
        dx = curr.x - prev.x
        dy = curr.y - prev.y
        total_length += math.sqrt(dx*dx + dy*dy)
    
    return total_length
```

#### 4.1.2 路径平滑度

路径平滑度反映路径的平滑程度，可以用曲率或方向变化来衡量：

$$S = \frac{1}{n-2} \sum_{i=1}^{n-2} |\theta_{i+1} - \theta_i|$$

其中 $\theta_i = \arctan2(y_{i+1}-y_i, x_{i+1}-x_i)$ 是路径方向角。

```python
def calculate_path_smoothness(path):
    """计算路径平滑度
    
    Returns:
        float: 平均方向变化（弧度）
    """
    if len(path.poses) < 3:
        return 0.0
    
    angle_changes = []
    for i in range(1, len(path.poses) - 1):
        prev = path.poses[i-1].pose.position
        curr = path.poses[i].pose.position
        next_p = path.poses[i+1].pose.position
        
        angle1 = math.atan2(curr.y - prev.y, curr.x - prev.x)
        angle2 = math.atan2(next_p.y - curr.y, next_p.x - curr.x)
        
        diff = abs(angle2 - angle1)
        if diff > math.pi:  # 处理角度跳变
            diff = 2 * math.pi - diff
        angle_changes.append(diff)
    
    return sum(angle_changes) / len(angle_changes) if angle_changes else 0.0
```

#### 4.1.3 路径效率

路径效率定义为最短路径与实际路径的比值：

$$\eta = \frac{D_{euclidean}}{L_{actual}} \times 100\%$$

其中 $D_{euclidean}$ 是起点到终点的欧几里得距离。

### 4.2 运动性能指标

#### 4.2.1 平均速度

$$v_{avg} = \frac{L_{total}}{T_{total}}$$

其中 $T_{total}$ 是总执行时间。

#### 4.2.2 最大速度

记录导航过程中达到的最大线速度和角速度。

```python
def analyze_navigation_performance(path, feedback):
    """分析导航性能
    
    Args:
        path: 执行的路径
        feedback: nav2_msgs/NavigationFeedback
        
    Returns:
        dict: 性能指标字典
    """
    metrics = {}
    
    # 路径长度
    metrics['path_length'] = calculate_path_length(path)
    
    # 直线距离
    start = path.poses[0].pose.position
    end = path.poses[-1].pose.position
    euclidean_dist = math.sqrt(
        (end.x - start.x)**2 + (end.y - start.y)**2
    )
    metrics['euclidean_distance'] = euclidean_dist
    
    # 路径效率
    metrics['path_efficiency'] = euclidean_dist / metrics['path_length'] * 100
    
    # 平滑度
    metrics['smoothness'] = calculate_path_smoothness(path)
    
    # 执行时间（如果feedback可用）
    if feedback and feedback.elapsed_time:
        metrics['execution_time'] = feedback.elapsed_time.sec
        metrics['average_speed'] = metrics['path_length'] / feedback.elapsed_time.sec
    
    return metrics
```

### 4.3 安全性指标

#### 4.3.1 最小障碍物距离

记录整个导航过程中与最近障碍物的距离：

$$d_{min} = \min_{t \in [0, T]} d_{obstacle}(t)$$

#### 4.3.2 碰撞次数

统计导航过程中发生碰撞的次数。

```python
def check_collision(costmap, position, footprint):
    """检查是否发生碰撞
    
    Args:
        costmap: 代价地图
        position: 机器人位置
        footprint: 机器人footprint
        
    Returns:
        bool: 是否碰撞
    """
    # 将footprint转换为地图坐标
    # 检查所有footprint点是否在LETHAL区域
    for point in footprint:
        map_x, map_y = world_to_map(position, point)
        cost = costmap[map_y][map_x]
        if cost >= 128:  # LETHAL cost
            return True
    return False
```

### 4.4 综合性能评分

可以将多个指标综合为一个综合评分：

$$Score = w_1 \cdot \eta + w_2 \cdot \frac{d_{min}}{d_{safe}} + w_3 \cdot \frac{v_{avg}}{v_{max}} + w_4 \cdot (1 - \frac{collisions}{n})$$

其中 $w_1, w_2, w_3, w_4$ 是各指标的权重，满足 $\sum w_i = 1$。

### 5.2 测试执行框架

```python
#!/usr/bin/env python3
"""导航性能Benchmark框架"""

import rclpy
from rclpy.node import Node
from nav2_msgs.action import NavigateToPose
from rclpy.action import ActionClient
import time
import json
from geometry_msgs.msg import PoseStamped

class NavigationBenchmark(Node):
    def __init__(self):
        super().__init__('navigation_benchmark')
        
        # 动作客户端
        self.nav_client = ActionClient(self, NavigateToPose, 'navigate_to_pose')
        
        # 测试结果存储
        self.results = []
        
        # 等待服务
        self.nav_client.wait_for_server()
        self.get_logger().info('导航服务已就绪')
        
    def run_test(self, test_name, start_pose, goal_pose):
        """执行单个测试"""
        self.get_logger().info(f'开始测试: {test_name}')
        
        # 发送目标
        goal_msg = NavigateToPose.Goal()
        goal_msg.pose = goal_pose
        
        start_time = time.time()
        send_goal_future = self.nav_client.send_goal_async(goal_msg)
        rclpy.spin_until_future_complete(self, send_goal_future)
        
        goal_handle = send_goal_future.result()
        if not goal_handle.accepted:
            self.get_logger().error('目标被拒绝')
            return None
            
        result_future = goal_handle.get_result_async()
        rclpy.spin_until_future_complete(self, result_future)
        
        end_time = time.time()
        result = result_future.result()
        
        # 记录结果
        test_result = {
            'test_name': test_name,
            'success': result.result.success,
            'execution_time': end_time - start_time,
            'distance_traveled': result.result.distance_remaining,
            # 可以添加更多指标...
        }
        
        self.results.append(test_result)
        return test_result
        
    def run_suite(self, test_scenarios):
        """运行测试套件"""
        for scenario in test_scenarios:
            result = self.run_test(
                scenario['name'],
                scenario['start'],
                scenario['goal']
            )
            time.sleep(1)  # 测试间隔
            
    def save_results(self, filename):
        """保存结果到文件"""
        with open(filename, 'w') as f:
            json.dump(self.results, f, indent=2)
        self.get_logger().info(f'结果已保存到 {filename}')
        
    def print_summary(self):
        """打印测试摘要"""
        successful = sum(1 for r in self.results if r['success'])
        total = len(self.results)
        
        print(f"\n{'='*50}")
        print(f"测试摘要: {successful}/{total} 成功")
        print(f"{'='*50}")
        
        for result in self.results:
            status = "✓" if result['success'] else "✗"
            print(f"{status} {result['test_name']}: {result['execution_time']:.2f}s")
```

   import math
   
   def calculate_path_metrics(path):
       """计算路径质量指标
       
       Args:
           path: nav_msgs/Path 消息
           
       Returns:
           dict: 包含长度、效率、平滑度的字典

       # 1. 计算路径长度
       path_length = 0.0
       for i in range(1, len(path.poses)):
           p1 = path.poses[i-1].pose.position
           p2 = path.poses[i].pose.position
           path_length += math.sqrt(
               (p2.x - p1.x)**2 + (p2.y - p1.y)**2
           )
       
       # 2. 计算欧几里得距离（直线距离）
       start = path.poses[0].pose.position
       end = path.poses[-1].pose.position
       euclidean_dist = math.sqrt(
           (end.x - start.x)**2 + (end.y - start.y)**2
       )
       
       # 3. 计算路径效率
       path_efficiency = (euclidean_dist / path_length * 100) if path_length > 0 else 0
       
       # 4. 计算平滑度（平均方向变化）
       angle_changes = []
       for i in range(1, len(path.poses) - 1):
           p0 = path.poses[i-1].pose.position

           diff = abs(angle2 - angle1)
