# 15-模仿学习

> 本章节讲解机器人模仿学习技术

## 章节简介

本章节涵盖行为克隆、DAgger、逆向强化学习（IRL）、GAIL 等模仿学习技术，让机器人从人类演示中学习技能。模仿学习通过专家示范数据学习策略映射，无需设计奖励函数，适合试错代价高的真实机器人场景。

## 前置知识

- 强化学习基础（MDP、策略梯度）

## 学习目标

- 理解模仿学习基本原理，与强化学习的本质区别
- 掌握行为克隆（BC）方法及分布偏移（compounding error）问题
- 掌握 DAgger 在线修正算法
- 理解逆强化学习（IRL）从专家轨迹推断奖励函数的数学框架
- 掌握 GAIL 生成对抗模仿学习方法
- 具备机器人示教学习系统开发能力
