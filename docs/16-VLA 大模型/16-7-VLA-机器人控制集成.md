## 1. VLA机器人集成概述

### 1.1 VLA与ROS/ROS2的关系

ROS（Robot Operating System）和ROS2是机器人软件开发的实际标准框架，为机器人提供了硬件抽象、消息传递、工具链等基础能力。VLA模型本身是一个推理引擎，负责根据视觉输入和语言指令输出动作决策。将VLA集成到ROS/ROS2系统中，意味着让VLA成为机器人控制pipeline中的一个计算节点。

| 特性 | ROS | ROS2 |
|------|-----|------|
| 通信机制 | TCPROS/UDPROS | DDS（数据分发服务） |
| 实时性 | 非实时，需要补丁 | 原生实时支持 |
| 生命周期管理 | 节点独立 | 节点有生命周期状态机 |
| QoS质量服务 | 不支持 | 支持，可配置可靠性、持久性 |

对于VLA机器人控制，**ROS2是更推荐的选择**，原因如下：
- DDS的实时性更适合控制循环的确定性要求
- QoS机制可以保证视觉数据传递的可靠性
- 生命周期管理便于优雅地启动/停止VLA推理节点
- ROS2的Action机制天然适合长时间的运动任务

### 1.2 控制循环架构

VLA机器人控制系统的核心是一个多层嵌套的控制循环架构：

```
┌─────────────────────────────────────────────────┐
│           任务规划层（Task Planning）              │
│  接收高层指令 → 分解为可执行的动作序列              │
└─────────────────────┬───────────────────────────┘
                      │
┌─────────────────────▼───────────────────────────┐
│          VLA 推理层（VLA Inference）              │
│  视觉+语言 → 动作预测（action prediction）         │
└─────────────────────┬───────────────────────────┘
                      │
┌─────────────────────▼───────────────────────────┐
│         运动控制层（Motion Control）              │
│  动作命令 → 关节力矩/末端速度 → 电机驱动           │
└─────────────────────┬───────────────────────────┘
                      │
┌─────────────────────▼───────────────────────────┐
│         传感反馈层（Sensor Feedback）             │
│  视觉/力觉/关节状态 → 更新环境模型                 │
└─────────────────────────────────────────────────┘
```

| 层级 | 频率 | 示例 | 典型延迟容忍 |
|------|------|------|-------------|
| 任务规划层 | 0.1-1 Hz | "去厨房拿杯子" | 允许秒级延迟 |
| VLA推理层 | 1-10 Hz | 每帧图像推理 | 10-100ms可接受 |
| 运动控制层 | 100-1000 Hz | PID控制 | 必须<10ms |
| 传感反馈层 | 30-120 Hz | 相机图像采集 | 实时要求 |

### 1.3 实时性要求分析

总延迟分解：

$$L_{total} = L_{capture} + L_{preprocess} + L_{inference} + L_{postprocess} + L_{control}$$

| 环节 | 平均延迟 | P99延迟 | 优化手段 |
|------|---------|---------|---------|
| 图像采集 | 5-15ms | 30ms | USB3.0/GigE相机 |
| 图像预处理 | 2-5ms | 10ms | GPU加速 |
| VLA推理 | 50-200ms | 500ms | TensorRT/ONNX加速、模型量化 |
| 动作后处理 | 1-3ms | 5ms | 查表、并行计算 |
| 控制命令发送 | 1-5ms | 10ms | ROS2 QoS |

**总延迟目标**：桌面级机械臂100-300ms，移动机器人50-150ms。

## 2. 视觉感知模块

### 2.1 相机配置

机器人视觉系统通常配备多种相机：

**单目相机**：成本低、适合物体识别、场景理解。分辨率640×480到1920×1080，帧率30-60fps。

**深度相机**：获取深度信息，适合抓取、避障。典型产品：
- Intel RealSense D415/D435i：RGBD，深度精度约2mm@2m
- Azure Kinect DK：RGBD+IMU，深度精度1mm@1m

**腕部相机**：安装在机械臂末端，提供近距离、高分辨率视觉信息。

```python
import rclpy
```

### 2.2 图像预处理

VLA模型对输入图像有特定格式要求，需要预处理：

```python
import numpy as np
import cv2

class ImagePreprocessor:
    """VLA图像预处理器 - 将相机图像预处理为模型输入格式"""
    
    def __init__(self, target_size=(224, 224),
                 mean=(0.485, 0.456, 0.406),
                 std=(0.229, 0.224, 0.225)):
        self.target_size = target_size
        self.mean = np.array(mean).reshape(1, 1, 3)
        self.std = np.array(std).reshape(1, 1, 3)
    
    def preprocess(self, image):
        """预处理单张图像"""
        # BGR转RGB
        image = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
        # 尺寸调整
        image = cv2.resize(image, self.target_size, interpolation=cv2.INTER_LINEAR)
        # 归一化到[0, 1]
        image = image.astype(np.float32) / 255.0
        # 标准化
        image = (image - self.mean) / self.std
        return image
```

### 2.3 物体检测与跟踪

物体检测与跟踪为VLA提供额外的目标位置信息：

```python
import numpy as np

class ObjectTracker:
    """物体跟踪器 - 使用IOU匹配实现多目标跟踪"""
    
    def __init__(self, max_disappeared=5, iou_threshold=0.3):
        self.max_disappeared = max_disappeared
        self.iou_threshold = iou_threshold
        self.tracked_objects = {}
        self.next_object_id = 0
    
    def compute_iou(self, box1, box2):
        """计算两个边界框的IOU"""
        x1_min, y1_min, x1_max, y1_max = box1
        x2_min, y2_min, x2_max, y2_max = box2
        
        inter_xmin, inter_ymin = max(x1_min, x2_min), max(y1_min, y2_min)
        inter_xmax, inter_ymax = min(x1_max, x2_max), min(y1_max, y2_max)
        
        if inter_xmax < inter_xmin or inter_ymax < inter_ymin:
            return 0.0
        
        inter_area = (inter_xmax - inter_xmin) * (inter_ymax - inter_ymin)
        box1_area = (x1_max - x1_min) * (y1_max - y1_min)
        box2_area = (x2_max - x2_min) * (y2_max - y2_min)
        union_area = box1_area + box2_area - inter_area
        
        return inter_area / union_area if union_area > 0 else 0.0
    
    def update(self, detections):
        """更新跟踪器状态"""
        if len(detections) == 0:
            for obj_id in list(self.tracked_objects.keys()):
                self.tracked_objects[obj_id]["disappeared"] += 1
                if self.tracked_objects[obj_id]["disappeared"] > self.max_disappeared:
                    del self.tracked_objects[obj_id]
            return self.tracked_objects
        
        detected_boxes = [det[:4] for det in detections]
        
        if len(self.tracked_objects) == 0:
            for det in detections:
                obj_id = self.next_object_id
                self.next_object_id += 1
                x_min, y_min, x_max, y_max = det[:4]
                self.tracked_objects[obj_id] = {
                    "bbox": det[:4],
                    "center": [(x_min + x_max) / 2, (y_min + y_max) / 2],
                    "disappeared": 0
                }
            return self.tracked_objects
        
        # IOU匹配
        object_ids = list(self.tracked_objects.keys())
        object_boxes = [self.tracked_objects[oid]["bbox"] for oid in object_ids]
        
        iou_matrix = np.zeros((len(object_ids), len(detected_boxes)))
        for i, obj_box in enumerate(object_boxes):
            for j, det_box in enumerate(detected_boxes):
                iou_matrix[i, j] = self.compute_iou(obj_box, det_box)
        
        matched_objects, matched_detections = set(), set()
        for _ in range(min(len(object_ids), len(detected_boxes))):
            max_iou_idx = np.unravel_index(np.argmax(iou_matrix), iou_matrix.shape)
            if iou_matrix[max_iou_idx] < self.iou_threshold:
                break
            obj_idx, det_idx = max_iou_idx
            obj_id = object_ids[obj_idx]
            x_min, y_min, x_max, y_max = detected_boxes[det_idx]
            self.tracked_objects[obj_id] = {
                "bbox": detected_boxes[det_idx],
                "center": [(x_min + x_max) / 2, (y_min + y_max) / 2],
                "disappeared": 0
            }
            matched_objects.add(obj_idx)
            matched_detections.add(det_idx)
            iou_matrix[max_iou_idx] = 0
        
        for i, obj_id in enumerate(object_ids):
            if i not in matched_objects:
                self.tracked_objects[obj_id]["disappeared"] += 1
                if self.tracked_objects[obj_id]["disappeared"] > self.max_disappeared:
                    del self.tracked_objects[obj_id]
        
        for j, det_box in enumerate(detected_boxes):
            if j not in matched_detections:
                obj_id = self.next_object_id
                self.next_object_id += 1
                x_min, y_min, x_max, y_max = det_box
                self.tracked_objects[obj_id] = {
                    "bbox": det_box,
                    "center": [(x_min + x_max) / 2, (y_min + y_max) / 2],
                    "disappeared": 0
                }
        
        return self.tracked_objects
```

## 3. 语音指令解析

### 3.1 语音识别

使用Whisper进行语音识别，将语音转换为文本指令：

```python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
from sensor_msgs.msg import AudioData
import numpy as np

class SpeechRecognitionNode(Node):
    """语音识别节点 - 通过Whisper模型进行语音识别"""
    
    def __init__(self, model_name="base"):
        super().__init__('speech_recognition_node')
        
        self.get_logger().info(f'加载Whisper模型: {model_name}')
        import whisper
        self.model = whisper.load_model(model_name)
        self.get_logger().info('Whisper模型加载完成')
        
        self.audio_buffer = []
        self.sample_rate = 16000
        self.max_buffer_duration = 30
        self.silence_threshold = 0.01
        self.silence_frames = 0
        self.is_speaking = False
        
        self.audio_sub = self.create_subscription(
            AudioData, '/audio/raw', self.audio_callback, 10
        )
        self.text_pub = self.create_publisher(String, '/speech/text', 10)
        
        self.get_logger().info('语音识别节点已初始化')
    
    def audio_callback(self, audio_msg):
        """音频数据回调"""
        audio_data = np.frombuffer(audio_msg.data, dtype=np.int16)
        audio_float = audio_data.astype(np.float32) / 32768.0
        energy = np.sqrt(np.mean(audio_float ** 2))
        
        if energy > self.silence_threshold:
            self.silence_frames = 0
            self.is_speaking = True
            self.audio_buffer.append(audio_float)
        else:
            self.silence_frames += 1
            if self.silence_frames > 30 and self.is_speaking and len(self.audio_buffer) > 0:
                self.process_audio()
                self.audio_buffer = []
                self.is_speaking = False
                self.silence_frames = 0
    
    def process_audio(self):
        """处理音频并进行识别"""
        if not self.audio_buffer:
            return
        audio = np.concatenate(self.audio_buffer)
        if len(audio) > self.sample_rate * self.max_buffer_duration:
            audio = audio[:self.sample_rate * self.max_buffer_duration]
        
        result = self.model.transcribe(audio, language='zh', fp16=False)
        text = result["text"].strip()
        
        if text:
            self.get_logger().info(f'识别结果: {text}')
            msg = String()
            msg.data = text
            self.text_pub.publish(msg)
```

### 3.2 指令理解与解析

将识别的文本解析为结构化指令：

```python
import re
from enum import Enum
from dataclasses import dataclass
from typing import List, Optional

class RobotAction(Enum):
    """机器人动作枚举"""
    MOVE_TO = "move_to"
    GRASP = "grasp"
    PLACE = "place"
    PUSH = "push"
    PULL = "pull"
    POUR = "pour"
    OPEN = "open"
    CLOSE = "close"
    LOOK_AT = "look_at"
    WAIT = "wait"
    HOME = "home"

@dataclass
class ParsedInstruction:
    """解析后的指令数据结构"""
    action: RobotAction
    target_object: Optional[str] = None
    location: Optional[str] = None
    speed: Optional[str] = None
    confidence: float = 1.0
    raw_text: str = ""

class InstructionParser:
    """自然语言指令解析器 - 将文本解析为结构化机器人指令"""
    
    ACTION_PATTERNS = {
        RobotAction.GRASP: [r"抓[取]?", r"拿起", r"拾取"],
        RobotAction.PLACE: [r"放下", r"放置", r"放到"],
        RobotAction.MOVE_TO: [r"移动到", r"走到", r"去.*那边"],
        RobotAction.PUSH: [r"推", r"按"],
        RobotAction.PULL: [r"拉", r"拽"],
        RobotAction.POUR: [r"倒", r"倾倒"],
        RobotAction.OPEN: [r"打开", r"开"],
        RobotAction.CLOSE: [r"关闭", r"关上"],
        RobotAction.LOOK_AT: [r"看", r"观察", r"查看"],
        RobotAction.WAIT: [r"等[待一下]", r"停[止留]"],
        RobotAction.HOME: [r"回零", r"复位", r"归位"],
    }
    
    SPEED_PATTERNS = {
        "slow": [r"慢[速的]?", r"小心"],
        "medium": [r"正常", r"中速"],
        "fast": [r"快[速的]?", r"迅速"],
    }
    
    def __init__(self):
        self.action_regexes = {
            action: [re.compile(p, re.IGNORECASE) for p in patterns]
            for action, patterns in self.ACTION_PATTERNS.items()
        }
        self.speed_regexes = {
            speed: [re.compile(p, re.IGNORECASE) for p in patterns]
            for speed, patterns in self.SPEED_PATTERNS.items()
        }
    
    def parse(self, text: str) -> List[ParsedInstruction]:
        """解析文本指令"""
        results = []
        text = text.strip()
        
        detected_actions = []
        for action, regexes in self.action_regexes.items():
            for regex in regexes:
                match = regex.search(text)
                if match:
                    detected_actions.append((action, match.start()))
                    break
        
        if not detected_actions:
            return [ParsedInstruction(action=RobotAction.WAIT, confidence=0.0, raw_text=text)]
        
        detected_actions.sort(key=lambda x: x[1])
        
        for action, start_pos in detected_actions:
            target_object = None
            color_match = re.search(r"(红|黄|蓝|绿|白|黑)(色)的?", text)
            if color_match:
                target_object = color_match.group(0)
            
            speed = None
            for speed_val, regexes in self.speed_regexes.items():
                if any(r.search(text) for r in regexes):
                    speed = speed_val
                    break
            
            location = None
            loc_match = re.search(r"(到|在|去)(.{1,20})", text[start_pos:])
            if loc_match:
                location = loc_match.group(2).rstrip("的")
            
            results.append(ParsedInstruction(
                action=action,
                target_object=target_object,
                location=location,
                speed=speed,
                confidence=0.8,
                raw_text=text
            ))
        
        return results
```

### 3.3 任务分解

将高层指令分解为VLA可执行的子动作序列：

```python
from typing import List

class TaskDecomposer:
    """任务分解器 - 将高层指令分解为子动作序列"""
    
    TASK_TEMPLATES = {
        "抓取并放置": [
            {"action": "MOVE_TO", "target": "approach_{object}"},
            {"action": "MOVE_TO", "target": "above_{object}"},
            {"action": "LOWER"},
            {"action": "GRASP", "target": "{object}"},
            {"action": "RAISE"},
            {"action": "MOVE_TO", "target": "approach_{location}"},
            {"action": "MOVE_TO", "target": "above_{location}"},
            {"action": "LOWER"},
            {"action": "RELEASE"},
            {"action": "RAISE"},
            {"action": "HOME"},
        ],
    }
    
    def decompose(self, instruction) -> List[dict]:
        """分解指令为子动作"""
        action = instruction.action.value
        target = instruction.target_object or ""
        location = instruction.location or ""
        
        for key in self.TASK_TEMPLATES:
            if key in instruction.raw_text:
                template = self.TASK_TEMPLATES[key]
                sub_actions = []
                for step in template:
                    step_copy = step.copy()
                    if "{object}" in str(step_copy):
                        step_copy["target"] = step_copy["target"].format(object=target)
                    if "{location}" in str(step_copy):
                        step_copy["target"] = step_copy["target"].format(location=location)
                    sub_actions.append(step_copy)
                return sub_actions
        
        # 简单动作直接映射
        if action == "grasp":
            return [
                {"action": "MOVE_TO", "target": f"above_{target}"},
                {"action": "LOWER"},
                {"action": "GRASP", "target": target},
                {"action": "RAISE"},
            ]
        elif action == "place":
            return [
                {"action": "MOVE_TO", "target": f"above_{location}"},
                {"action": "LOWER"},
                {"action": "RELEASE"},
                {"action": "RAISE"},
            ]
        else:
            return [{"action": action.upper(), "target": target}]
```

## 4. 动作执行

### 4.1 关节控制

关节控制是最底层的运动控制：

```python
import rclpy
from rclpy.node import Node
from trajectory_msgs.msg import JointTrajectory, JointTrajectoryPoint
from sensor_msgs.msg import JointState

class JointController(Node):
    """关节控制器 - 支持位置控制和力矩控制"""
    
    def __init__(self):
        super().__init__('joint_controller')
        
        self.joint_names = [
            'shoulder_pan_joint', 'shoulder_lift_joint', 'elbow_joint',
            'wrist_1_joint', 'wrist_2_joint', 'wrist_3_joint',
        ]
        
        self.joint_state_sub = self.create_subscription(
            JointState, '/joint_states', self.joint_state_callback, 10
        )
        self.current_joint_states = {}
        
        self.trajectory_pub = self.create_publisher(
            JointTrajectory, '/joint_trajectory_controller/joint_trajectory', 10
        )
        
        self.effort_pub = self.create_publisher(
            JointState, '/effort_controller/command', 10
        )
        
        self.get_logger().info('关节控制器节点已初始化')
    
    def joint_state_callback(self, msg):
        """更新当前关节状态"""
        for name, pos, vel, eff in zip(msg.name, msg.position, msg.velocity, msg.effort):
            self.current_joint_states[name] = {'position': pos, 'velocity': vel, 'effort': eff}
    
    def send_joint_positions(self, positions, duration=2.0):
        """发送关节位置控制指令"""
        traj = JointTrajectory()
        traj.joint_names = self.joint_names
        
        point = JointTrajectoryPoint()
        point.positions = list(positions)
        point.velocities = [0.0] * len(positions)
        point.time_from_start = rclpy.duration.Duration(seconds=duration).to_msg()
        
        traj.points.append(point)
        self.trajectory_pub.publish(traj)
        self.get_logger().info(f'发送关节位置: {positions}')
    
    def send_joint_efforts(self, efforts):
        """发送关节力矩控制指令"""
        effort_cmd = JointState()
        effort_cmd.name = self.joint_names
        effort_cmd.effort = list(efforts)
        self.effort_pub.publish(effort_cmd)
    
    def get_current_positions(self):
        """获取当前关节角度"""
        return [self.current_joint_states.get(name, {}).get('position', 0.0) 
                for name in self.joint_names]
```

```python
import rclpy
from rclpy.node import Node
from std_msgs.msg import Float64

class EndEffectorController(Node):
    """末端执行器控制器 - 控制夹爪开闭"""
    
    def __init__(self):
```

### 4.3 碰撞检测

碰撞检测是保证机器人安全运行的关键：

```python
import numpy as np

class CollisionDetector:
    """碰撞检测器 - 基于关节力矩的碰撞检测"""
    
    def __init__(self, torque_threshold=10.0, window_size=10):
        """
        Args:
            torque_threshold: 力矩突变阈值，超过此值判定为碰撞
            window_size: 滑动窗口大小
        """
        self.torque_threshold = torque_threshold
        self.window_size = window_size
        self.torque_history = []
        self.collision_callbacks = []
    
    def add_collision_callback(self, callback):
        """注册碰撞回调函数"""
        self.collision_callbacks.append(callback)
    
    def check_collision(self, joint_states):
        """
        检测碰撞
        
        Args:
            joint_states: 关节状态 dict，键为关节名，值为 {'position', 'velocity', 'effort'}
            
        Returns:
            True if collision detected, False otherwise
        """
        torques = [state['effort'] for state in joint_states.values()]
        self.torque_history.append(torques)
        
        if len(self.torque_history) > self.window_size:
            self.torque_history.pop(0)
        
        if len(self.torque_history) < 2:
            return False
        
        # 计算力矩变化率
        torque_array = np.array(self.torque_history)
        torque_diff = np.abs(torque_array[-1] - torque_array[0])
        
        # 碰撞判定：力矩突变超过阈值
        if np.any(torque_diff > self.torque_threshold):
            for callback in self.collision_callbacks:
                callback(joint_states)
            return True
        
        return False
    
    def get_joint_load(self):
        """获取当前关节负载估计"""
        if not self.torque_history:
            return {}
        return {'mean_torque': np.mean(self.torque_history, axis=0).tolist()}
```

```python
            self.state_callbacks[new_state]()

        pass
    
    def on_executing(self):
        """执行状态处理"""
        pass

        """错误状态处理"""
```

import traceback

from typing import Callable, Any

class FaultHandler:
    """故障处理器 - 统一管理系统中的各类故障"""
    
    def __init__(self, node):
        self.node = node
        self.fault_count = {}
        self.last_fault_time = {}
        self.fault_callbacks = {}
        self.recovery_strategies = {}
    
    def register_fault_type(self, fault_type: str, callback: Callable, recovery: Callable = None):
        """
        注册故障类型及处理策略
        
        Args:
            fault_type: 故障类型名称
            callback: 故障发生时的回调
            recovery: 故障恢复策略
        """
        self.fault_callbacks[fault_type] = callback
        self.recovery_strategies[fault_type] = recovery
        self.fault_count[fault_type] = 0
    
    def handle_fault(self, fault_type: str, context: Any = None):
        """
        处理故障
        
        Args:
            fault_type: 故障类型
            context: 故障上下文信息
        """
        import time
        
        self.fault_count[fault_type] = self.fault_count.get(fault_type, 0) + 1
        self.last_fault_time[fault_type] = time.time()
        
        self.node.get_logger().error(f'检测到故障 [{fault_type}]: {context}')
        
        # 调用故障回调
        if fault_type in self.fault_callbacks:
            try:
                self.fault_callbacks[fault_type](context)
            except Exception as e:
                self.node.get_logger().error(f'故障回调执行失败: {e}')
        
        # 尝试自动恢复
        if fault_type in self.recovery_strategies and self.recovery_strategies[fault_type]:
            self.attempt_recovery(fault_type)
    
    def attempt_recovery(self, fault_type: str):
        """尝试恢复"""
        try:
            recovery = self.recovery_strategies[fault_type]
            self.node.get_logger().info(f'尝试恢复策略: {fault_type}')
            recovery()
            self.node.get_logger().info(f'恢复成功: {fault_type}')
        except Exception as e:
            self.node.get_logger().error(f'恢复失败: {e}')
    
    def get_fault_stats(self):
        """获取故障统计"""
        return {
            'counts': self.fault_count.copy(),
            'last_times': self.last_fault_time.copy()

## 8. 参考答案

### 8.1 概念理解答案

**题目1答案**：

总延迟组成：
$$L_{total} = L_{capture} + L_{preprocess} + L_{inference} + L_{postprocess} + L_{control}$$

| 环节 | 典型量级 | 优化方法 |
|------|---------|---------|
| $L_{capture}$ | 5-15ms | USB3.0/GigE相机、硬件触发 |
| $L_{preprocess}$ | 2-5ms | GPU加速、批量处理 |
| $L_{inference}$ | 50-200ms | TensorRT/ONNX、模型量化 |
| $L_{postprocess}$ | 1-3ms | 查表、并行计算 |
| $L_{control}$ | 1-5ms | ROS2 QoS、本地通信 |

**题目2答案**：

VLA推理层频率设计在1-10Hz的原因：
- **够快的环境响应**：机器人在动态环境中需要及时响应变化，1Hz可以感知主要变化，10Hz可以跟踪快速运动
- **计算资源约束**：VLA模型较大，推理耗时较长（50-200ms），10Hz已是较高频率
- **层级解耦**：VLA推理不需要像底层控制那样达到100-1000Hz的实时性，它属于战术级决策

**题目3答案**：

ROS2相比ROS的优势：
1. **DDS通信**：提供实时性支持和可靠的数据传输
2. **QoS机制**：可配置消息传递的可靠性、持久性
3. **生命周期管理**：便于优雅地启动/停止节点
4. **Action机制**：原生支持长时间运动任务
5. **安全性**：内置安全特性支持

**题目4答案**：

`JointTrajectoryPoint`的必要字段：
- `positions`：目标关节角度（必须）
- `velocities`：目标关节速度（可选，通常设为0）
- `accelerations`：目标关节加速度（可选）
- `time_from_start`：从轨迹开始到到达此点的时间，用于轨迹插值

```python
import numpy as np

class SimpleTracker:
    """简化的物体跟踪器"""
    
    def __init__(self, iou_threshold=0.3):
        self.iou_threshold = iou_threshold
        self.tracked = {}  # {id: bbox}
        self.next_id = 0
    
    def compute_iou(self, box1, box2):
        """计算IOU"""
        x1_min, y1_min, x1_max, y1_max = box1
        x2_min, y2_min, x2_max, y2_max = box2
        
        inter_xmin, inter_ymin = max(x1_min, x2_min), max(y1_min, y2_min)
        inter_xmax, inter_ymax = min(x1_max, x2_max), min(y1_max, y2_max)
        
        if inter_xmax <= inter_xmin or inter_ymax <= inter_ymin:
            return 0.0
        
        inter_area = (inter_xmax - inter_xmin) * (inter_ymax - inter_ymin)
        area1 = (x1_max - x1_min) * (y1_max - y1_min)
        area2 = (x2_max - x2_min) * (y2_max - y2_min)
        union_area = area1 + area2 - inter_area
        
        return inter_area / union_area if union_area > 0 else 0.0
    
    def update(self, detections):
        """更新跟踪"""
        if len(self.tracked) == 0:
            # 初始化
            for det in detections:
                self.tracked[self.next_id] = det[:4]
                self.next_id += 1
            return self.tracked
        
        # IOU匹配
        matched = set()
        for obj_id, obj_bbox in list(self.tracked.items()):
            best_iou = 0
            best_det_idx = -1
            for i, det in enumerate(detections):
                if i in matched:
                    continue
                iou = self.compute_iou(obj_bbox, det[:4])
                if iou > best_iou:
                    best_iou = iou
                    best_det_idx = i
            
            if best_iou >= self.iou_threshold:
                self.tracked[obj_id] = detections[best_det_idx][:4]
                matched.add(best_det_idx)
            else:
                del self.tracked[obj_id]
        
        # 新目标
        for i, det in enumerate(detections):
            if i not in matched:
                self.tracked[self.next_id] = det[:4]
                self.next_id += 1
        
        return self.tracked
```

```python
import re

class CommandParser:
    """指令解析器"""
    
    PATTERNS = {
        'grasp': [r'抓', r'拿', r'拾'],
        'place': [r'放', r'置'],
        'move_to': [r'移动到', r'走到', r'去'],
    }
    
    def __init__(self):
        self.compiled = {
            action: [re.compile(p) for p in patterns]
            for action, patterns in self.PATTERNS.items()
        }
    
    def parse(self, text):
        """解析指令"""
        result = {'action': 'unknown', 'target': None, 'location': None}
        
        for action, regexes in self.compiled.items():
            for regex in regexes:
                match = regex.search(text)
                if match:
                    result['action'] = action
                    # 提取颜色+物体
                    color_obj = re.search(r'((红|黄|蓝|绿|白|黑)(色)的?)?(.{1,10})', text[match.end():])
                    if color_obj:
                        target = color_obj.group(0) if color_obj.group(0) else None
                        if result['action'] == 'place':
                            result['location'] = target
                        else:
                            result['target'] = target
                    # 提取位置
                    loc = re.search(r'(到|在)(.{1,10})', text)
                    if loc:
                        result['location'] = loc.group(2)
                    break
        
        return result
```

import time
import threading

class SimpleWatchdog:
    """简单的看门狗监控"""
    
    def __init__(self, timeout=5.0):
        self.timeout = timeout
        self.last_heartbeat = {}
        self.callbacks = {}
        self.running = False
        self.thread = None
    
    def register(self, node_name, callback):
        """注册节点"""
        self.callbacks[node_name] = callback
    
    def start

    def start(self):
        self.running = True
        self.thread = threading.Thread(target=self._loop, daemon=True)
        self.thread.start()
    
    def stop(self):
        self.running = False
    
    def heartbeat(self, node_name):
        self.last_heartbeat[node_name] = time.time()
    
    def _loop(self):
        while self.running:
            now = time.time()
            for node_name, last_time in list(self.last_heartbeat.items()):
                if now - last_time > self.timeout:
                    if node_name in self.callbacks:
                        self.callbacks[node_name]()
            time.sleep(0.5)
