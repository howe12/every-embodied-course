## 1. VLA训练概述

### 1.1 训练挑战

VLA（视觉-语言-动作）模型的训练是一项极具挑战性的系统工程任务。与纯视觉或纯语言模型的训练不同，VLA需要同时解决三个核心问题：

**模态对齐挑战**：视觉特征、语言语义和机器人动作三者来自完全不同的表示空间。图像是密集的像素级表征，语言是离散的符号序列，机器人动作则是连续的数值向量。如何让模型在预训练阶段建立起这三个模态之间的语义关联，是VLA训练的首要难题。

**计算资源需求巨大**：一个7B参数的VLA模型，在单张A100 GPU上进行全参数微调需要数周时间。典型的VLA训练流程需要数百到数千GPU时的计算资源。以OpenVLA为例，其预训练使用了256块A100 GPU，训练约两周才完成。

**动作表示的连续性与离散性矛盾**：机器人动作本质上是连续值（关节角度或末端位姿），而语言模型天然处理离散token。如何在Transformer架构中优雅地表示动作，是模型设计的关键问题。

**数据稀缺问题**：高质量的"图像-语言-动作"三元组数据极为稀缺。获取大量机器人演示数据需要真实机器人采集，成本高昂；而互联网视频虽然丰富，却缺乏精确的动作标注。

### 1.2 计算资源需求

VLA训练的计算资源需求可以大致分为以下几个级别：

| 训练阶段 | 参数量 | GPU类型 | GPU数量 | 训练时间 |
|----------|--------|---------|---------|----------|
| 预训练（互联网数据） | 7B | A100 80GB | 64-256 | 1-4周 |
| 预训练（机器人数据） | 7B | A100 80GB | 32-128 | 3-7天 |
| 动作微调（LoRA） | 7B | A100 80GB | 1-8 | 6-72小时 |
| 全参数微调 | 7B | A100 80GB | 8-32 | 2-5天 |

对于研究者和个人开发者而言，使用**LoRA微调**是更为实际的选择。相比全参数训练，LoRA可以将可训练参数量减少100-1000倍，同时保持90%以上的性能。

### 1.3 数据规模

VLA训练通常需要多阶段、不同来源的数据：

**预训练阶段**：
- 互联网图像-文本对：约10M-100M条（如COYO、LAION）
- 机器人演示数据：约100K-1M条轨迹
- 动作标注数据：约10K-100K条

**微调阶段**：
- 任务特定机器人数据：1K-10K条高质量演示
- 人类遥操作数据：500-5K条
- 仿真到现实迁移数据：1K-50K条

VLA预训练的典型数据配比为：**互联网视觉-语言数据占90%以上**，机器人动作数据虽然占比小，但却是赋予机器人操控能力的关键。

## 2. 训练数据准备

### 2.1 机器人轨迹采集

机器人轨迹采集是VLA训练数据准备的核心环节。采集方式主要分为以下几类：

**人类遥操作（Teleoperation）**：通过设备让人类操作者实时控制机器人，记录操作过程中的所有状态信息。这种方式采集的数据质量最高，但效率较低。常用设备包括：
- **SpaceMouse**：通过三维鼠标控制机械臂末端姿态
- **VR手柄**：通过VR手柄的6DOF位姿控制机器人
- **数据手套**：通过手套的角度传感器控制机器人关节
- **视觉动捕**：通过摄像头捕捉人体动作映射到机器人

**主动示范学习（Demonstration Learning）**：人类操作者提前录制好演示轨迹，机器人通过模仿学习掌握技能。这种方式效率更高，适合批量数据采集。

**仿真数据生成（Sim-to-Real）**：在仿真环境（如MuJoCo、PyBullet）中通过运动规划或强化学习自动生成大量轨迹数据，再通过域随机化（Domain Randomization）提升泛化能力。

```python
# 示例：使用SpaceMouse采集机器人轨迹数据
import numpy as np
from pathlib import Path

class RobotTrajectoryCollector:
    """机器人轨迹采集器
    
    使用SpaceMouse设备采集人类操作员的控制指令，
    并将其与对应的机器人状态（图像、关节角度）一起记录。
    """
    
    def __init__(self, robot_interface, camera_interface):
        self.robot = robot_interface      # 机器人通信接口
        self.camera = camera_interface     # 相机接口
        self.trajectories = []            # 存储所有轨迹
        
    def collect_demonstration(self, task_instruction: str, duration_sec: int = 30):
        """采集单条演示轨迹
        
        Args:
            task_instruction: 任务文本描述，如"将红色方块放到蓝色盒子里"
            duration_sec: 采集持续时间（秒）
        """
        trajectory = {
            'instruction': task_instruction,   # 语言指令
            'observations': [],                  # 每帧图像观察
            'actions': [],                       # 每帧执行的动作
            'states': [],                       # 每帧的机器人状态
        }
        
        # 初始化采集
        self.robot.set_mode('teleoperation')  # 设置遥操作模式
        self.camera.start_grabbing()           # 开始图像采集
        
        # 以30Hz频率记录数据
        sampling_rate = 30  # Hz
        num_samples = duration_sec * sampling_rate
        
        for step in range(num_samples):
            # 获取当前观测
            image = self.camera.get_frame()           # 获取当前图像
            joint_positions = self.robot.get_joints()  # 获取关节角度
            ee_pose = self.robot.get_ee_pose()         # 获取末端执行器位姿
            
            # 获取人类操作员的控制指令
            action = self.spacemouse.get_action()      # 获取6DOF控制量
            
            # 存储数据
            trajectory['observations'].append(image)
            trajectory['actions'].append(action)
            trajectory['states'].append({
                'joints': joint_positions,
                'ee_pose': ee_pose,
                'timestamp': step / sampling_rate
            })
        
        self.camera.stop_grabbing()
        self.trajectories.append(trajectory)
        
        return trajectory
    
    def save_dataset(self, output_path: Path):
        """将采集的数据保存为标准格式"""
        dataset = {
            'version': '1.0',
            'num_trajectories': len(self.trajectories),
            'trajectories': self.trajectories
        }
        
        import json
        with open(output_path / 'dataset.json', 'w') as f:
            json.dump(dataset, f, indent=2)
        
        print(f"已保存 {len(self.trajectories)} 条轨迹到 {output_path}")
```

### 2.2 视觉-语言-动作对齐

视觉-语言-动作三模态对齐是VLA训练的核心问题。数据准备阶段需要确保三个模态的信息在语义上保持一致。

**时间同步**：图像帧、动作指令和语言描述必须严格对齐。常见策略是以图像帧的时间戳为基准，将相邻时间窗口内的动作和语言描述进行关联。

**语义一致性**：同一条轨迹可能对应多条不同的语言描述。例如，机器人抓取苹果的动作可以描述为"抓住苹果"或"用夹爪拾取水果"。VLA训练中通常保留这种多样性，让模型学习同一动作的不同表达方式。

**动作归一化**：不同机器人的动作空间差异巨大（关节数量、控制范围、力矩限制等）。在数据准备阶段，需要对动作进行归一化处理，使其映射到统一的空间（如[-1, 1]）。

```python
# 示例：视觉-语言-动作数据对齐处理
import numpy as np
from typing import List, Dict

class VLADatasetAligner:
    """VLA数据集对齐器
    
    负责将原始采集的轨迹数据转换为VLA训练所需的标准格式，
    包括动作归一化、语言指令编码、时间步对齐等。
    """
    
    def __init__(self, action_dim: int = 7, image_size: tuple = (224, 224)):
        self.action_dim = action_dim         # 动作维度（通常7=6DOF+夹爪）
        self.image_size = image_size         # 图像归一化尺寸
        self.action_bounds = None            # 动作数值范围
        
    def normalize_action(self, action: np.ndarray, 
                         action_min: np.ndarray, 
                         action_max: np.ndarray) -> np.ndarray:
        """将动作值线性归一化到[-1, 1]区间
        
        Args:
            action: 原始动作值，形状为 (action_dim,)
            action_min: 各维度最小值
            action_max: 各维度最大值
        
        Returns:
            归一化后的动作值
        """
        # 防止除零
        action_range = action_max - action_min
        action_range = np.where(action_range < 1e-6, 1.0, action_range)
        
        # 线性归一化到[-1, 1]
        normalized = 2.0 * (action - action_min) / action_range - 1.0
        
        return normalized.astype(np.float32)
    
    def denormalize_action(self, normalized_action: np.ndarray,
                           action_min: np.ndarray,
                           action_max: np.ndarray) -> np.ndarray:
        """将归一化的动作值还原到原始数值范围"""
        action_range = action_max - action_min
        action_range = np.where(action_range < 1e-6, 1.0, action_range)
        
        original = (normalized_action + 1.0) / 2.0 * action_range + action_min
        return original.astype(np.float32)
    
    def align_trajectory(self, trajectory: Dict) -> Dict:
        """对齐单条轨迹的视觉-语言-动作数据"""
        aligned = {
            'instruction': trajectory['instruction'],
            'image_paths': [],
            'normalized_actions': [],
            'timestamps': [],
        }
        
        # 统计动作范围
        all_actions = np.array(trajectory['actions'])
        action_min = all_actions.min(axis=0)
        action_max = all_actions.max(axis=0)
        
        # 逐帧处理
        for obs, action, state in zip(
            trajectory['observations'],
            trajectory['actions'],
            trajectory['states']
        ):
            # 处理图像
            image_path = self.save_processed_image(obs)
            aligned['image_paths'].append(image_path)
            
            # 归一化动作
            norm_action = self.normalize_action(action, action_min, action_max)
            aligned['normalized_actions'].append(norm_action)
            aligned['timestamps'].append(state['timestamp'])
        
        aligned['action_bounds'] = {
            'min': action_min.tolist(),
            'max': action_max.tolist()
        }
        
        return aligned
    
    def save_processed_image(self, image: np.ndarray) -> str:
        """预处理并保存图像"""
        import cv2
        resized = cv2.resize(image, self.image_size)
        return resized  # 实际存储时保存到磁盘，返回路径
```

### 2.3 数据格式定义

VLA训练数据的标准格式需要满足以下要求：机器可读、人类可理解、支持高效加载。业界常用的格式包括：

**OpenVLA格式**：基于HuggingFace的datasets库格式，每条数据包含图像路径、指令文本和归一化动作向量。

**RT-2格式**：Google开源的格式，包含图像tokens、文本tokens和动作tokens的序列。

```python
# 示例：VLA训练数据格式定义（符合OpenVLA标准）
import json
from dataclasses import dataclass
from typing import List, Optional
import numpy as np

@dataclass
class VLATrainingSample:
    """VLA训练样本标准格式
    
    每个样本包含一条完整的(图像, 语言, 动作)训练数据。
    遵循OpenVLA开源项目的数据格式规范。
    """
    # 图像信息
    image_path: str              # 图像文件路径
    image_features: Optional[np.ndarray] = None  # 预计算的视觉特征
    
    # 语言指令
    instruction: str            # 任务描述文本，如"打开抽屉"
    instruction_tokens: Optional[List[int]] = None  # 文本token化结果
    
    # 动作标签
    action: np.ndarray          # 归一化动作向量，形状为(action_dim,)
    action_raw: Optional[np.ndarray] = None  # 原始动作值（未归一化）
    
    # 元数据
    dataset_name: str = ""      # 数据集来源名称
    robot_type: str = ""        # 机器人类型（如"widowx", "franka"）
    task_category: str = ""     # 任务类别（如"manipulation", "navigation"）
    
    def to_dict(self) -> dict:
        """序列化为字典格式"""
        return {
            'image_path': self.image_path,
            'instruction': self.instruction,
            'action': self.action.tolist() if isinstance(self.action, np.ndarray) else self.action,
            'dataset_name': self.dataset_name,
            'robot_type': self.robot_type,
            'task_category': self.task_category,
        }
    
    @classmethod
    def from_dict(cls, data: dict) -> 'VLATrainingSample':
        """从字典格式反序列化"""
        return cls(
            image_path=data['image_path'],
            instruction=data['instruction'],
            action=np.array(data['action'], dtype=np.float32),
            dataset_name=data.get('dataset_name', ''),
            robot_type=data.get('robot_type', ''),
            task_category=data.get('task_category', ''),
        )


@dataclass
class VLADatasetManifest:
    """VLA数据集清单文件
    
    描述整个数据集的元信息，便于分布式加载和版本管理。
    类似于ImageNet的meta.mat文件。
    """
    version: str = "1.0"
    dataset_name: str = ""
    total_samples: int = 0
    samples: List[dict] = None  # 所有样本的简要信息列表
    
    def __post_init__(self):
        if self.samples is None:
            self.samples = []
    
    def save(self, output_path: str):
        """保存清单文件为JSON格式"""
        with open(output_path, 'w') as f:
            json.dump({
                'version': self.version,
                'dataset_name': self.dataset_name,
                'total_samples': self.total_samples,
                'samples': self.samples
            }, f, indent=2)
    
    @classmethod
    def load(cls, input_path: str) -> 'VLADatasetManifest':
        """加载清单文件"""
        with open(input_path, 'r') as f:
            data = json.load(f)
        return cls(
            version=data['version'],
            dataset_name=data['dataset_name'],
            total_samples=data['total_samples'],
            samples=data['samples']
        )
```

## 3. 预训练阶段

### 3.1 视觉语言对齐

预训练的首要目标是建立视觉特征与语言语义之间的关联。这一阶段通常采用**对比学习**（Contrastive Learning）或**掩码重建**（Masked Modeling）策略。

**SigLIP方案**：Google DeepMind在OpenVLA中使用的方案，通过大规模图像-文本对学习视觉编码器。损失函数为：

$$
\mathcal{L}_{contrastive} = -\log \frac{\exp(\text{sim}(I, T)/\tau)}{\sum_{j=1}^{B} \exp(\text{sim}(I, T_j)/\tau)}
$$

其中 $I$ 为图像特征，$T$ 为对应文本特征，$\tau$ 为温度参数，$B$ 为批量大小。

**LLaVA方案**：通过多模态指令微调，将冻结的视觉编码器与语言模型对齐。视觉特征通过投影层映射到语言模型的 embedding 空间。

```python
# 示例：视觉-语言对齐训练（简化版）
import torch
import torch.nn as nn
import torch.nn.functional as F

class VisionLanguageAligner(nn.Module):
    """视觉-语言对齐模型
    
    将视觉编码器的输出映射到语言模型的embedding空间，
    使得视觉特征可以与文本token直接拼接。
    """
    
    def __init__(self, vision_dim: int = 768, language_dim: int = 4096, 
                 projection_dim: int = 4096):
        super().__init__()
        
        # 两层MLP进行特征变换
        self.projection = nn.Sequential(
            nn.Linear(vision_dim, projection_dim),
            nn.GELU(),                          # GELU激活函数
            nn.Linear(projection_dim, language_dim),
        )
        
        # 可学习的温度参数（用于对比学习）
        self.logit_scale = nn.Parameter(torch.ones([]) * np.log(1 / 0.07))
    
    def forward(self, vision_features: torch.Tensor, 
                text_features: torch.Tensor) -> dict:
        """
        Args:
            vision_features: 视觉特征，形状 (batch_size, vision_dim)
            text_features: 文本特征，形状 (batch_size, language_dim)
        
        Returns:
            包含对比损失和投影特征的字典
        """
        # 投影视觉特征到语言空间
        projected_vision = self.projection(vision_features)
        
        # L2归一化
        projected_vision = F.normalize(projected_vision, p=2, dim=-1)
        text_features = F.normalize(text_features, p=2, dim=-1)
        
        # 计算相似度矩阵
        logit_scale = self.logit_scale.exp()
        logits_per_vision = logit_scale * projected_vision @ text_features.T
        logits_per_text = logits_per_vision.T
        
        # 对比学习损失（InfoNCE）
        batch_size = vision_features.shape[0]
        labels = torch.arange(batch_size, device=vision_features.device)
        
        loss_vision = F.cross_entropy(logits_per_vision, labels)
        loss_text = F.cross_entropy(logits_per_text, labels)
        contrastive_loss = (loss_vision + loss_text) / 2
        
        return {
            'loss': contrastive_loss,
            'projected_vision': projected_vision,
            'logits': logits_per_vision,
        }
```

### 3.2 互联网数据

预训练阶段使用的大量互联网图像-文本对数据，帮助VLA模型获得强大的视觉理解和语言理解能力。这类数据不需要动作标签，只需图像和描述性文本即可。

**常用数据集**：
- **COYO-700M**：韩国NAVER开源的7亿图像-文本对数据集
- **LAION-2B**：LAION开源的20亿规模数据集
- **AltCLIP数据**：多语言图像-文本对数据

**数据过滤策略**：原始互联网数据噪声很大，需要过滤：
- 图像分辨率过滤：移除分辨率过低的图像
- 文本长度过滤：移除过短或过长的文本描述
- CLIP分数过滤：使用预训练CLIP模型计算图文相似度，只保留高相似度样本
- 安全过滤：移除包含不当内容的图像-文本对

```python
# 示例：互联网数据过滤
import torch
from PIL import Image
from transformers import CLIPProcessor, CLIPModel

class InternetDataFilter:
    """互联网数据质量过滤器
    
    使用CLIP模型评估图文匹配度，
    过滤掉低质量的图像-文本对。
    """
    
    def __init__(self, device: str = "cuda"):
        self.device = device
        # 加载CLIP模型
        self.clip_model = CLIPModel.from_pretrained("openai/clip-vit-base-patch32")
        self.clip_processor = CLIPProcessor.from_pretrained("openai/clip-vit-base-patch32")
        self.clip_model.to(device)
        self.clip_model.eval()
        
        # 过滤阈值
        self.min_clip_score = 0.25      # 最小CLIP分数
        self.min_image_size = 128       # 最小图像尺寸
        self.max_text_length = 300      # 最大文本长度
    
    def filter_sample(self, image: Image.Image, text: str) -> bool:
        """判断单条数据是否通过过滤
        
        Returns:
            True表示通过，可以用于训练；False表示被过滤掉
        """
        # 1. 检查图像尺寸
        if image.width < self.min_image_size or image.height < self.min_image_size:
            return False
        
        # 2. 检查文本长度
        if len(text) < 5 or len(text) > self.max_text_length:
            return False
        
        # 3. 计算CLIP分数
        try:
            inputs = self.clip_processor(text=[text], images=[image], 
                                         return_tensors="pt", padding=True)
            inputs = {k: v.to(self.device) for k, v in inputs.items()}
            
            with torch.no_grad():
                outputs = self.clip_model(**inputs)
            
            clip_score = outputs.logits_per_image.item()
            
            if clip_score < self.min_clip_score:
                return False
                
        except Exception:
            # 处理图像无法解码等情况
            return False
        
        return True
```

### 3.3 动作表示学习

在预训练阶段引入动作标签，让模型学习"看到图像、理解语言后应该执行什么动作"。动作表示学习的核心问题是如何将连续动作离散化或编码为模型可以处理的形式。

**连续动作直接回归**：最简单的方式是让模型直接输出连续动作值，损失函数为MSE：

$$
\mathcal{L}_{action} = \frac{1}{T} \sum_{t=1}^{T} \| a_t - \hat{a}_t \|^2
$$

其中 $a_t$ 为真实动作，$\hat{a}_t$ 为模型预测的动作。

**动作离散化（RT-2方案）**：将连续动作空间划分为离散bins，每个bin对应一个特殊的action token：

$$
a_t^{discrete} = \text{round}\left( \frac{a_t - a_{min}}{a_{max} - a_{min}} \times (N-1) \right)
$$

其中 $N$ 为离散化的bin数量（RT-2使用255个bins）。

**动作chunking（OpenVLA方案）**：将动作向量拼接成字符串，通过语言模型的自回归方式生成：

$$
P(a_t | I, L) = \prod_{i=1}^{D} P(a_{t,i} | I, L, a_{t,<i})
$$

其中 $D$ 为动作维度，$I$ 为图像，$L$ 为语言指令。

```python
# 示例：动作表示学习
import torch
import torch.nn as nn

class ActionRepresentationLearner(nn.Module):
    """动作表示学习模块
    
    支持两种模式：
    1. 连续回归：直接预测连续动作值
    2. 离散化：预测动作的离散bins（用于RT-2风格训练）
    """
    
    def __init__(self, input_dim: int = 4096, action_dim: int = 7,
                 mode: str = 'continuous', num_bins: int = 255):
        super().__init__()
        
        self.mode = mode                      # 'continuous' 或 'discrete'
        self.action_dim = action_dim          # 动作维度（如7DOF机械臂）
        self.num_bins = num_bins              # 离散化bin数量
        
        if mode == 'continuous':
            # 连续模式：直接输出动作值
            self.action_head = nn.Sequential(
                nn.Linear(input_dim, input_dim // 2),
                nn.GELU(),
                nn.Dropout(0.1),
                nn.Linear(input_dim // 2, action_dim),
                nn.Tanh()  # 输出限制在[-1, 1]
            )
        else:
            # 离散模式：每个动作维度输出一个logits
            self.action_heads = nn.ModuleList([
                nn.Sequential(
                    nn.Linear(input_dim, input_dim // 4),
                    nn.GELU(),
                    nn.Linear(input_dim // 4, num_bins)
                )
                for _ in range(action_dim)
            ])
    
    def discretize_action(self, action: torch.Tensor, 
                         action_min: float = -1.0,
                         action_max: float = 1.0) -> torch.Tensor:
        """将连续动作离散化
        
        Args:
            action: 连续动作值，形状 (batch_size, action_dim)
            action_min: 动作空间下界
            action_max: 动作空间上界
        
        Returns:
            离散化后的动作，形状 (batch_size, action_dim)
        """
        # 归一化到[0, 1]
        normalized = (action - action_min) / (action_max - action_min)
        
        # 映射到[0, num_bins-1]
        scaled = normalized * (self.num_bins - 1)
        
        # 四舍五入取整
        discrete = torch.round(scaled).long()
        
        # 限制在有效范围内
        discrete = torch.clamp(discrete, 0, self.num_bins - 1)
        
        return discrete
    
    def forward(self, features: torch.Tensor, 
                target_actions: torch.Tensor = None) -> dict:
        """统一前向接口"""
        if self.mode == 'continuous':
            predicted_actions = self.action_head(features)
            result = {'predicted_actions': predicted_actions}
            
            if target_actions is not None:
                loss = nn.functional.mse_loss(predicted_actions, target_actions)
                result['loss'] = loss
        else:
            all_logits = []
            for head in self.action_heads:
                logits = head(features)  # 形状 (batch_size, num_bins)
                all_logits.append(logits)
            
            logits_stacked = torch.stack(all_logits, dim=1)
            result = {'action_logits': logits_stacked}
            
            if target_actions is not None:
                discrete_targets = self.discretize_action(target_actions)
                batch_size, action_dim = discrete_targets.shape
                logits_flat = logits_stacked.view(batch_size * action_dim, self.num_bins)
                targets_flat = discrete_targets.view(batch_size * action_dim)
                
                loss = nn.functional.cross_entropy(logits_flat, targets_flat)
                result['loss'] = loss
        
        return result
```

## 4. 动作微调

### 4.1 低秩适配（LoRA）

LoRA（Low-Rank Adaptation）是目前VLA微调中最常用的参数高效微调方法。其核心思想是假设预训练模型的权重更新矩阵是低秩的，因此只需学习两个小的低秩矩阵的乘积，而不是整个权重矩阵。

对于预训练权重矩阵 $W_0 \in \mathbb{R}^{d \times k}$，LoRA的更新公式为：

$$
W = W_0 + \Delta W = W_0 + BA
$$

其中 $B \in \mathbb{R}^{d \times r}$，$A \in \mathbb{R}^{r \times k}$，$r \ll \min(d, k)$ 为低秩维度。

LoRA的训练目标函数为：

$$
\mathcal{L}_{LoRA} = \mathcal{L}_{task} + \frac{\lambda}{r} \|A\|_F^2
$$

其中 $\mathcal{L}_{task}$ 为下游任务损失，$\lambda$ 为正则化系数。

```python
# 示例：LoRA微调VLA模型
import torch
import torch.nn as nn
from typing import Dict, List

class LoRALinear(nn.Module):
    """LoRA线性层
    
    在标准的线性层旁边添加低秩支路，
    冻结原权重，只训练低秩矩阵A和B。
    """
    
    def __init__(self, in_features: int, out_features: int, 
                 rank: int = 8, alpha: float = 16.0, dropout: float = 0.0):
        super().__init__()
        
        self.rank = rank                        # 低秩维度r
        self.alpha = alpha                      # 缩放因子
        
        # 冻结原始权重
        self.weight = nn.Parameter(
            torch.randn(out_features, in_features), 
            requires_grad=False
        )
        self.bias = nn.Parameter(
            torch.zeros(out_features), 
            requires_grad=False
        )
        
        # LoRA低秩矩阵
        # A: (r, in_features)，使用随机初始化
        self.lora_A = nn.Parameter(torch.randn(rank, in_features) * 0.01)
        # B: (out_features, r)，初始化为零（确保训练开始时与原模型行为一致）
        self.lora_B = nn.Parameter(torch.zeros(out_features, rank))
        
        # 可选的Dropout
        self.lora_dropout = nn.Dropout(p=dropout) if dropout > 0 else None
        
    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """前向传播
        
        原始路径：W_0 @ x + b
        LoRA路径：W_0 @ x + (alpha/r) * B @ A @ x
        """
        # 原始权重计算（冻结）
        original_output = nn.functional.linear(x, self.weight, self.bias)
        
        # LoRA路径
        # A: (r, in_features), x: (batch, in_features)
        # B: (out_features, r)
        x_dropped = self.lora_dropout(x) if self.lora_dropout else x
        lora_output = (self.lora_B @ self.lora_A) @ x_dropped.T
        
        # 缩放LoRA输出
        scaled_lora = (self.alpha / self.rank) * lora_output.T
        
        return original_output + scaled_lora


class VLABlockWithLoRA(nn.Module):
    """在VLA Transformer块中应用LoRA
    
    通常对Q、K、V投影和FFN层应用LoRA，
    这些是最需要适配的部分。
    """
    
    def __init__(self, hidden_size: int, num_heads: int, 
                 intermediate_size: int, rank: int = 8):
        super().__init__()
        
        # 自注意力层（使用LoRA）
        self.q_proj = LoRALinear(hidden_size, hidden_size, rank=rank)
        self.k_proj = LoRALinear(hidden_size, hidden_size, rank=rank)
        self.v_proj = LoRALinear(hidden_size, hidden_size, rank=rank)
        self.o_proj = LoRALinear(hidden_size, hidden_size, rank=rank)
        
        # 前馈网络（使用LoRA）
        self.gate_proj = LoRALinear(hidden_size, intermediate_size, rank=rank)
        self.up_proj = LoRALinear(intermediate_size, hidden_size, rank=rank)
        
        self.hidden_size = hidden_size
        self.num_heads = num_heads
    
    def forward(self, hidden_states: torch.Tensor) -> torch.Tensor:
        """带LoRA的Transformer块前向传播"""
        batch_size, seq_len, hidden_size = hidden_states.shape
        
        # 自注意力
        Q = self.q_proj(hidden_states)
        K = self.k_proj(hidden_states)
        V = self.v_proj(hidden_states)
        
        head_dim = hidden_size // self.num_heads
        Q = Q.view(batch_size, seq_len, self.num_heads, head_dim).transpose(1, 2)
        K = K.view(batch_size, seq_len, self.num_heads, head_dim).transpose(1, 2)
        V = V.view(batch_size, seq_len, self.num_heads, head_dim).transpose(1, 2)
        
        attn_weights = torch.matmul(Q, K.transpose(-2, -1)) / (head_dim ** 0.5)
        attn_weights = torch.softmax(attn_weights, dim=-1)
        attn_output = torch.matmul(attn_weights, V)
        attn_output = attn_output.transpose(1, 2).contiguous()
        attn_output = attn_output.view(batch_size, seq_len, hidden_size)
        hidden_states = hidden_states + self.o_proj(attn_output)
        
        # SwiGLU前馈网络
        gate = self.gate_proj(hidden_states)
        up = self.up_proj(hidden_states)
        hidden_states = hidden_states + torch.nn.functional.silu(gate) * up
        
        return hidden_states
```

### 4.2 梯度累积

当GPU显存不足以容纳大批量数据时，梯度累积是解决这一问题的标准技术。通过在多个小批次上累积梯度，然后在累积足够多的梯度后再更新模型参数，可以模拟大批量训练的效果。

梯度累积的数学原理很简单：设原始批次大小为 $B$，目标有效批次大小为 $B_{eff} = B \times N$，其中 $N$ 为累积步数。则：

$$
\theta_{t+1} = \theta_t - \frac{\eta}{N} \sum_{i=1}^{N} \nabla \mathcal{L}_i(\theta_t)
$$

即先对 $N$ 个小批次的梯度求和，最后用 $\frac{\eta}{N}$ 的学习率进行一次参数更新。

```python
# 示例：梯度累积实现
import torch
from torch.nn.optimizer import Optimizer
from typing import List, Callable

class GradientAccumulator:
    """梯度累积管理器
    
    将多个小批次的梯度累积起来，
    实现等效的大批次训练效果。
    """
    
    def __init__(self, model: torch.nn.Module, 
                 optimizer: Optimizer,
                 accumulation_steps: int = 4,
                 device: str = "cuda"):
        self.model = model
        self.optimizer = optimizer
        self.accumulation_steps = accumulation_steps  # 累积步数N
        self.device = device
        self.step_count = 0
        
        # 将模型移到目标设备
        self.model.to(device)
    
    def training_step(self, batch: dict) -> dict:
        """执行一个训练步骤（带梯度累积）
        
        Args:
            batch: 包含图像、指令、动作的批次数据
        
        Returns:
            包含损失值的字典
        """
        # 前向传播
        outputs = self.model(
            images=batch['images'].to(self.device),
            instructions=batch['instructions'],
            actions=batch['actions'].to(self.device)
        )
        
        # 计算单个样本的损失
        loss = outputs['loss'] / self.accumulation_steps
        
        # 反向传播（累积梯度）
        loss.backward()
        
        self.step_count += 1
        
        # 当累积足够步数后，执行参数更新
        if self.step_count % self.accumulation_steps == 0:
            # 梯度裁剪（可选，防止梯度爆炸）
            torch.nn.utils.clip_grad_norm_(self.model.parameters(), max_norm=1.0)
            
            # 执行优化器步骤
            self.optimizer.step()
            self.optimizer.zero_grad()
        
        return {'loss': loss.item() * self.accumulation_steps}
    
    def compute_effective_batch_size(self, micro_batch_size: int) -> int:
        """计算有效批次大小
        
        Args:
            micro_batch_size: 单个GPU上的小批次大小
        
        Returns:
            有效批次大小
        """
        num_gpus = torch.cuda.device_count()
        return micro_batch_size * self.accumulation_steps * num_gpus
```

### 4.3 学习率调度

学习率调度对VLA微调的效果有显著影响。常用的学习率调度策略包括：

**余弦退火（Cosine Annealing）**：学习率按照余弦曲线从初始值缓慢衰减到最小值：

$$
\eta_t = \eta_{min} + \frac{1}{2}(\eta_{max} - \eta_{min}) \left(1 + \cos\left(\frac{t\pi}{T}\right)\right)
$$

其中 $\eta_{max}$ 为最大学习率，$\eta_{min}$ 为最小学习率，$T$ 为总训练步数。

**预热+余弦衰减（Warmup + Cosine Decay）**：先进行线性预热，然后在余弦衰减：

$$
\eta_t = \begin{cases} 
\eta_{max} \cdot \frac{t}{T_{warmup}} & t < T_{warmup} \\
\eta_{min} + \frac{1}{2}(\eta_{max} - \eta_{min}) \left(1 + \cos\left(\frac{(t-T_{warmup})\pi}{T-T_{warmup}}\right)\right) & t \geq T_{warmup}
\end{cases}
$$

```python
# 示例：学习率调度器实现
import math
from torch.optim.lr_scheduler import _LRScheduler

class WarmupCosineScheduler(_LRScheduler):
    """预热 + 余弦退火学习率调度器
    
    适用于VLA微调场景：
    - 预热阶段：让模型逐渐适应大学习率
    - 余弦衰减：平滑地降低学习率
    """
    
    def __init__(self, optimizer, warmup_epochs: int, min_lr: float = 1e-6, warmup_ratio: float = 0.1):
        super().__init__(optimizer)
        
        self.total_steps = total_steps              # 总训练步数
        self.warmup_steps = int(total_steps * warmup_ratio)  # 预热步数
        self.min_lr = min_lr                        # 最小学习率
        self.base_lrs = [group['lr'] for group in optimizer.param_groups]
    
    def get_lr(self) -> List[float]:
        """计算当前学习率"""
        step = self.last_epoch
        
        if step < self.warmup_steps:
            # 线性预热阶段
            warmup_factor = step / self.warmup_steps
            return [base_lr * warmup_factor for base_lr in self.base_lrs]
        else:
            # 余弦退火阶段
            progress = (step - self.warmup_steps) / (self.total_steps - self.warmup_steps)
            cosine_factor = 0.5 * (1 + math.cos(math.pi * progress))
            return [self.min_lr + (base_lr - self.min_lr) * cosine_factor 
                    for base_lr in self.base_lrs]


class VLALoRATrainer:
    """VLA LoRA微调训练器
    
    整合了LoRA模型、梯度累积、学习率调度等组件，
    提供完整的VLA微调训练流程。
    """
    
    def __init__(self, model, train_loader, val_loader, config):
        self.model = model
        self.train_loader = train_loader
        self.val_loader = val_loader
        self.config = config
        
        # 设置优化器（只优化LoRA参数）
        self.optimizer = torch.optim.AdamW(
            self._get_lora_parameters(),
            lr=config.lora.lr,
            weight_decay=config.lora.weight_decay
        )
        
        # 学习率调度器
        total_steps = len(train_loader) * config.training.num_epochs
        self.scheduler = WarmupCosineScheduler(
            self.optimizer,
            total_steps=total_steps,
            warmup_ratio=config.training.warmup_ratio
        )
        
        # 梯度累积
        self.accumulator = GradientAccumulator(
            model=self.model,
            optimizer=self.optimizer,
            accumulation_steps=config.training.gradient_accumulation_steps
        )
        
        self.current_epoch = 0
        self.best_val_loss = float('inf')
    
    def _get_lora_parameters(self):
        """获取所有LoRA可训练参数"""
        trainable_params = []
        for name, param in self.model.named_parameters():
            if 'lora_' in name:
                param.requires_grad = True
                trainable_params.append(param)
            else:
                param.requires_grad = False
        return trainable_params
    
    def train_epoch(self):
        """训练一个epoch"""
        self.model.train()
        total_loss = 0
        
        for batch_idx, batch in enumerate(self.train_loader):
            loss_dict = self.accumulator.training_step(batch)
            total_loss += loss_dict['loss']
            
            # 更新学习率
            self.scheduler.step()
        
        avg_loss = total_loss / len(self.train_loader)
        return avg_loss
    
    def validate(self):
        """验证"""
        self.model.eval()
        total_loss = 0
        
        with torch.no_grad():
            for batch in self.val_loader:
                outputs = self.model(
                    images=batch['images'].cuda(),
                    instructions=batch['instructions'],
                    actions=batch['actions'].cuda()
                )
                total_loss += outputs['loss'].item()
        
        avg_loss = total_loss / len(self.val_loader)
        return avg_loss
    
    def train(self, num_epochs: int):
        """完整训练流程"""
        for epoch in range(num_epochs):
            self.current_epoch = epoch
            
            train_loss = self.train_epoch()
            val_loss = self.validate()
            
            print(f"Epoch {epoch+1}/{num_epochs}")
            print(f"  Train Loss: {train_loss:.4f}")
            print(f"  Val Loss:   {val_loss:.4f}")
            
            # 保存最佳模型
            if val_loss < self.best_val_loss:
                self.best_val_loss = val_loss
                self.save_checkpoint('best_model.pt')
    
    def save_checkpoint(self, path: str):
        """保存LoRA权重检查点"""
        checkpoint = {
            'epoch': self.current_epoch,
            'model_state_dict': {
                name: param for name, param in self.model.named_parameters() 
                if 'lora_' in name
            },
            'optimizer_state_dict': self.optimizer.state_dict(),
            'scheduler_state_dict': self.scheduler.state_dict(),
            'best_val_loss': self.best_val_loss
        }
        torch.save(checkpoint, path)
        print(f"检查点已保存: {path}")
```

### 5.2 多GPU训练

多GPU训练的核心挑战是如何高效地将数据和模型分布到多个GPU上，并协调梯度同步。常用的并行策略包括：

**数据并行（Data Parallel, DP）**：每个GPU持有完整的模型副本，处理不同的数据批次，然后同步梯度。简单高效，但显存占用大。

**张量并行（Tensor Parallel, TP）**：将模型的一个层（如线性变换）的权重矩阵切分到多个GPU上，适合单层计算量大的场景（如Transformer的矩阵乘法）。

**流水线并行（Pipeline Parallel, PP）**：将模型的不同层分配到不同的GPU上，形成流水线。适合超深模型，但存在流水线气泡（Bubble）问题。

```python
# 示例：使用PyTorch原生DistributedDataParallel进行多GPU训练
import torch
import torch.distributed as dist
from torch.nn.parallel import DistributedDataParallel as DDP
import torch.multiprocessing as mp

def setup_ddp(rank: int, world_size: int):
    """设置分布式训练环境"""
    import os
    os.environ['MASTER_ADDR'] = 'localhost'
    os.environ['MASTER_PORT'] = '12355'
    
    dist.init_process_group("nccl", rank=rank, world_size=world_size)
    torch.cuda.set_device(rank)

def cleanup_ddp():
    """清理分布式训练环境"""
    dist.destroy_process_group()

def ddp_training_worker(rank: int, world_size: int, config: dict):
    """每个GPU上的训练进程"""
    setup_ddp(rank, world_size)
    
    # 加载模型并包装为DDP
    model = YourVLAModel(config)
    model = model.cuda()
    model = DDP(model, device_ids=[rank])
    
    # 使用DistributedSampler进行数据分片
    train_dataset = VLADataset(...)
    sampler = torch.utils.data.DistributedSampler(
        train_dataset,
        num_replicas=world_size,
        rank=rank,
        shuffle=True
    )
    train_loader = torch.utils.data.DataLoader(
        train_dataset,
        batch_size=config.batch_size_per_gpu,
        sampler=sampler,
        num_workers=4,
        pin_memory=True
    )
    
    # 优化器
    optimizer = torch.optim.AdamW(model.parameters(), lr=config.lr)
    
    # 训练循环
    model.train()
    for epoch in range(config.num_epochs):
        sampler.set_epoch(epoch)  # 保证每个epoch数据打乱方式不同
        
        for batch_idx, batch in enumerate(train_loader):
            # 将数据移到对应GPU
            batch = {k: v.cuda(rank) if isinstance(v, torch.Tensor) else v 
                     for k, v in batch.items()}
            
            optimizer.zero_grad()
            outputs = model(**batch)
            loss = outputs['loss']
            
            # DDP自动处理梯度同步
            loss.backward()
            optimizer.step()
            
            if rank == 0 and batch_idx % 100 == 0:
                print(f"Epoch {epoch}, Step {batch_idx}, Loss: {loss.item():.4f}")
    
    cleanup_ddp()

    mp.spawn(

        args=(world_size, config),
        nprocs=world_size,
        join=True
    )

### 5.3 混合精度

混合精度训练通过使用FP16（半精度）和FP32（单精度）混合计算，在保持模型精度的同时大幅减少显存占用和加速训练。

**动态损失缩放（Dynamic Loss Scaling）**：FP16的表示范围有限，直接使用可能导致梯度下溢。动态损失缩放通过自动调整缩放因子来解决这个问题：

$$
\hat{L} = s \cdot L, \quad \nabla W = \frac{1}{s} \cdot \frac{\partial \hat{L}}{\partial W}
$$

其中 $L$ 为原始损失，$s$ 为损失缩放因子。

```python
# 示例：PyTorch原生混合精度训练
import torch
from torch.cuda.amp import autocast, GradScaler

class VLAMixedPrecisionTrainer:
    """VLA混合精度训练器
    
    使用torch.cuda.amp实现自动混合精度（AMP）训练，
    减少显存占用的同时加速训练。
    """
    
    def __init__(self, model, config):
        self.model = model
        self.config = config
        
        # 优化器
        self.optimizer = torch.optim.AdamW(
            model.parameters(),
            lr=config.lr,
            weight_decay=config.weight_decay
        )
        
        # 损失缩放器（用于FP16训练）
        self.scaler = GradScaler()
        
    def train_step(self, batch: dict) -> dict:
        """执行一个混合精度的训练步骤"""
        
        self.optimizer.zero_grad()
        
        # 前向传播使用FP16
        with autocast(dtype=torch.float16):
            outputs = self.model(
                images=batch['images'],
                instructions=batch['instructions'],
                actions=batch['actions']
            )
            loss = outputs['loss']
        
        # 反向传播使用FP16，但梯度用FP32存储
        self.scaler.scale(loss).backward()
        
        # 梯度裁剪
        self.scaler.unscale_(self.optimizer)
        torch.nn.utils.clip_grad_norm_(self.model.parameters(), max_norm=1.0)
        
        # 优化器步骤
        self.scaler.step(self.optimizer)
        self.scaler.update()
        
        return {'loss': loss.item()}

### 6.1 动作质量评估

VLA模型的动作质量评估是训练过程中的重要环节。常用的评估指标包括：

**均方误差（MSE）**：衡量预测动作与真实动作的数值差异：

$$
MSE = \frac{1}{T \cdot D} \sum_{t=1}^{T} \sum_{d=1}^{D} (a_{t,d} - \hat{a}_{t,d})^2
$$

**平均绝对误差（MAE）**：

$$
MAE = \frac{1}{T \cdot D} \sum_{t=1}^{T} \sum_{d=1}^{D} |a_{t,d} - \hat{a}_{t,d}|
$$

**余弦相似度**：衡量动作向量的方向一致性：

$$
\cos\theta = \frac{\sum_{d=1}^{D} a_d \cdot \hat{a}_d}{\|a\| \cdot \|\hat{a}\|}
$$

```python
# 示例：VLA动作质量评估器
import numpy as np
import torch
from typing import Dict, List

class VLAActionEvaluator:
    """VLA动作质量评估器
    
    从多个维度评估VLA模型预测动作的质量。
    """
    
    def __init__(self, action_dim: int = 7):
        self.action_dim = action_dim
    
    def compute_metrics(self, predicted_actions: torch.Tensor, 
                       target_actions: torch.Tensor) -> Dict[str, float]:
        """计算所有评估指标
        
        Args:
            predicted_actions: 预测动作，形状 (batch_size, action_dim)
            target_actions: 目标动作，形状 (batch_size, action_dim)
        
        Returns:
            包含各项指标的字典
        """
        pred = predicted_actions.detach().cpu().numpy()
        target = target_actions.detach().cpu().numpy()
        
        # 均方误差
        mse = np.mean((pred - target) ** 2)
        
        # 平均绝对误差
        mae = np.mean(np.abs(pred - target))
        
        # 余弦相似度
        cos_sim = np.mean([
            np.dot(pred[i], target[i]) / (np.linalg.norm(pred[i]) * np.linalg.norm(target[i]) + 1e-8)
            for i in range(len(pred))
        ])
        
        # 动作各维度的MSE
        mse_per_dim = {
            f'dim_{i}': np.mean((pred[:, i] - target[:, i]) ** 2)
            for i in range(self.action_dim)
        }
        
        return {
            'mse': float(mse),
            'mae': float(mae),
            'cosine_similarity': float(cos_sim),
            **mse_per_dim
        }
    
    def evaluate_trajectory(self, predicted_traj: List[torch.Tensor],
                          target_traj: List[torch.Tensor]) -> Dict[str, float]:
        """评估整条轨迹的预测质量
        
        Args:
            predicted_traj: 预测轨迹，动作列表
            target_traj: 目标轨迹，动作列表
        """
        pred_stack = torch.stack(predicted_traj)
        target_stack = torch.stack(target_traj)
        
        metrics = self.compute_metrics(pred_stack, target_stack)
        
        # 额外指标：轨迹终点误差
        final_pred = pred_stack[-1]
        final_target = target_stack[-1]
        metrics['final_position_error'] = float(np.linalg.norm(
            (final_pred - final_target).cpu().numpy()
        ))
        
        return metrics
```

### 6.2 成功率和泛化性测试

动作质量评估只能反映模型的数值误差，真正的VLA能力需要通过任务级别的成功率和泛化性测试来衡量。

**成功率测试**：在真实机器人或仿真环境中执行任务，统计成功率。常用测试集包括：
- **CALVIN基准**：长期操控任务测试
- **RLBench**：多任务仿真基准
- **真实机器人测试**： WidowX、Franka等实体机器人

**泛化性测试**：评估模型在以下维度的泛化能力：
- **视觉泛化**：新背景、新光照、新物体

- **任务泛化**：训练时未见过的新任务组合

### 7.1 VLA训练数据处理

完整的数据处理流程演示，从原始轨迹到可训练的VLA数据格式。

```python
# 示例：完整的VLA训练数据处理流程
import json
import numpy as np
from pathlib import Path
from typing import List, Dict
from concurrent.futures import ProcessPoolExecutor, as_completed

class VLADataProcessor:
    """VLA训练数据完整处理流程
    
    将原始机器人轨迹数据处理为VLA模型可直接使用的训练格式。
    包含数据清洗、归一化、增强等步骤。
    """
    
    def __init__(self, 
                 action_dim: int = 7,
                 image_size: tuple = (224, 224),
                 num_workers: int = 4):
        self.action_dim = action_dim
        self.image_size = image_size
        self.num_workers = num_workers
        
        # 动作全局归一化参数（从数据中统计）
        self.action_mean = None
        self.action_std = None
    
    def process_dataset(self, 
                       input_path: str,
                       output_path: str,
                       train_split: float = 0.9):
        """处理完整数据集
        
        Args:
            input_path: 原始数据路径
            output_path: 处理后数据保存路径
            train_split: 训练集比例
        """
        # Step 1: 加载原始轨迹数据
        raw_trajectories = self.load_raw_trajectories(input_path)
        print(f"加载了 {len(raw_trajectories)} 条原始轨迹")
        
        # Step 2: 过滤低质量轨迹
        filtered_trajectories = self.filter_trajectories(raw_trajectories)
        print(f"过滤后剩余 {len(filtered_trajectories)} 条轨迹")
        
        # Step 3: 统计动作归一化参数
        self.compute_action_statistics(filtered_trajectories)
        
        # Step 4: 并行处理每条轨迹
        processed_samples = []
        with ProcessPoolExecutor(max_workers=self.num_workers) as executor:
            futures = {
                executor.submit(self.process_single_trajectory, traj): traj 
                for traj in filtered_trajectories
            }
            
            for future in as_completed(futures):
                samples = future.result()
                processed_samples.extend(samples)
        
        print(f"生成了 {len(processed_samples)} 个训练样本")
        
        # Step 5: 划分训练集和验证集
        np.random.shuffle(processed_samples)
        split_idx = int(len(processed_samples) * train_split)
        train_samples = processed_samples[:split_idx]
        val_samples = processed_samples[split_idx:]
        
        # Step 6: 保存处理后的数据
        output_dir = Path(output_path)
        output_dir.mkdir(parents=True, exist_ok=True)
        
        self.save_jsonl(train_samples, output_dir / 'train.jsonl')
        self.save_jsonl(val_samples, output_dir / 'val.jsonl')
        
        # 保存归一化参数
        with open(output_dir / 'action_stats.json', 'w') as f:
            json.dump({
                'mean': self.action_mean.tolist(),
                'std': self.action_std.tolist(),
                'action_dim': self.action_dim
            }, f)
        
        print(f"数据处理完成！")
        print(f"  训练集: {len(train_samples)} 样本")
        print(f"  验证集: {len(val_samples)} 样本")
        print(f"  保存路径: {output_dir}")
    
    def load_raw_trajectories(self, path: str) -> List[Dict]:
        """加载原始轨迹数据"""
        with open(path, 'r') as f:
            data = json.load(f)
        return data['trajectories']
    
    def filter_trajectories(self, trajectories: List[Dict]) -> List[Dict]:
        """过滤低质量轨迹
        
        过滤条件：
        - 轨迹长度不足（少于10帧）
        - 动作方差为零（机器人未移动）
        - 图像读取失败
        """
        filtered = []
        
        for traj in trajectories:
            # 检查轨迹长度
            if len(traj['observations']) < 10:
                continue
            
            # 检查动作方差
            actions = np.array(traj['actions'])
            if np.std(actions) < 1e-4:
                continue
            
            # 检查指令是否为空
            if not traj.get('instruction', '').strip():
                continue
            
            filtered.append(traj)
        
        return filtered
    
    def compute_action_statistics(self, trajectories: List[Dict]):
        """计算动作归一化的均值和标准差"""
        all_actions = []
        
        for traj in trajectories:
            actions = np.array(traj['actions'], dtype=np.float32)
            all_actions.append(actions)
        
        all_actions = np.concatenate(all_actions, axis=0)
        
        self.action_mean = np.mean(all_actions, axis=0, dtype=np.float32)
        self.action_std = np.std(all_actions, axis=0, dtype=np.float32)
        
        # 防止除零
        self.action_std = np.where(self.action_std < 1e-6, 1.0, self.action_std)
    
    def process_single_trajectory(self, trajectory: Dict) -> List[Dict]:
        """处理单条轨迹，生成多个训练样本"""
        samples = []
        
        instructions = trajectory['instructions'] if isinstance(
            trajectory['instructions'], list) else [trajectory['instructions']]
        
        actions = np.array(trajectory['actions'], dtype=np.float32)
        observations = trajectory['observations']
        
        num_frames = len(observations)
        
        for t in range(num_frames):
            # 归一化动作
            normalized_action = (actions[t] - self.action_mean) / self.action_std
            
            sample = {
                'image_path': observations[t],
                'instruction': np.random.choice(instructions),  # 随机选一条指令
                'action': normalized_action.tolist(),
                'timestep': t,
                'trajectory_id': id(trajectory)
            }
            samples.append(sample)
        
        return samples
    
    def save_jsonl(self, samples: List[Dict], path: Path):
        """保存为JSONL格式（每行一个JSON对象）"""
        with open(path, 'w') as f:
            for sample in samples:
                f.write(json.dumps(sample) + '\n')
    
    def load_jsonl(self, path: Path) -> List[Dict]:
        """加载JSONL格式数据"""
        samples = []
        with open(path, 'r') as f:
            for line in f:
                samples.append(json.loads(line))
        return samples


# 数据处理入口

### 7.2 LoRA微调VLA

完整LoRA微调代码，整合前述所有组件。

```python
# 示例：完整LoRA微调VLA代码
import torch
import torch.nn as nn
from torch.utils.data import DataLoader
from pathlib import Path
from typing import Dict, Optional
import json

# ============ LoRA 实现 ============

class LoRALinear(nn.Module):
    """LoRA线性层"""
    def __init__(self, in_features, out_features, rank=8, alpha=16.0, dropout=0.0):
        super().__init__()
        self.rank = rank
        self.alpha = alpha
        self.weight = nn.Parameter(torch.randn(out_features, in_features), requires_grad=False)
        self.bias = nn.Parameter(torch.zeros(out_features), requires_grad=False)
        self.lora_A = nn.Parameter(torch.randn(rank, in_features) * 0.01)
        self.lora_B = nn.Parameter(torch.zeros(out_features, rank))
        self.lora_dropout = nn.Dropout(p=dropout) if dropout > 0 else None

    def forward(self, x):
        original = nn.functional.linear(x, self.weight, self.bias)
        x_d = self.lora_dropout(x) if self.lora_dropout else x
        lora = (self.lora_B @ self.lora_A) @ x_d.T
        return original + (self.alpha / self.rank) * lora.T


# ============ 简化的VLA模型（用于演示） ============

class SimpleVLAModel(nn.Module):
    """简化的VLA模型（用于演示LoRA微调流程）
    
    实际使用时应替换为OpenVLA等真实模型。
    """
    def __init__(self, vision_dim=768, lang_dim=4096, action_dim=7, rank=8):
        super().__init__()
        self.action_dim = action_dim
        
        # 视觉编码器（简化）
        self.vision_proj = nn.Linear(vision_dim, lang_dim)
        
        # 语言编码器（简化）
        self.lang_proj = nn.Linear(lang_dim, lang_dim)
        
        # Transformer融合层
        self.transformer = nn.TransformerEncoder(
            nn.TransformerEncoderLayer(d_model=lang_dim, nhead=8, dim_feedforward=lang_dim*4),
            num_layers=4
        )
        
        # 动作预测头（带LoRA）
        self.action_head = LoRALinear(lang_dim, action_dim, rank=rank)
        
    def forward(self, vision_features, lang_features, actions=None):
        # 投影
        vision_emb = self.vision_proj(vision_features).unsqueeze(1)  # (B, 1, D)
        lang_emb = self.lang_proj(lang_features).unsqueeze(1)         # (B, 1, D)
        
        # 拼接融合
        combined = torch.cat([vision_emb, lang_emb], dim=1)           # (B, 2, D)
        
        # Transformer处理
        fused = self.transformer(combined)                           # (B, 2, D)
        
        # 取最后一个隐状态预测动作
        final_features = fused[:, -1, :]                              # (B, D)
        
        # 预测动作
        predicted_action = self.action_head(final_features)           # (B, action_dim)
        predicted_action = torch.tanh(predicted_action)                # 限制在[-1, 1]
        
        result = {'predicted_actions': predicted_action}
        
        if actions is not None:
            loss = nn.functional.mse_loss(predicted_action, actions)
            result['loss'] = loss
        
        return result

def train_vla_with_lora(
    model: nn.Module,
    train_loader: DataLoader,
    val_loader: DataLoader,
    output_dir: str,
    num_epochs: int = 10,
    lr: float = 1e-3,
    device: str = "cuda"
):
    """LoRA微调VLA主训练循环"""
    
    # 冻结非LoRA参数
    for name, param in model.named_parameters():
        if 'lora_' not in name:
            param.requires_grad = False
    
    # 统计可训练参数
    trainable_params = sum(p.numel() for p in model.parameters() if p.requires_grad)
    total_params = sum(p.numel() for p in model.parameters())
    print(f"可训练参数量: {trainable_params:,} / {total_params:,} ({100*trainable_params/total_params:.2f}%)")
    
    # 优化器（只优化LoRA参数）
    optimizer = torch.optim.AdamW(
        filter(lambda p: p.requires_grad, model.parameters()),
        lr=lr,
        weight_decay=0.01
    )
    
    # 学习率调度器
    scheduler = torch.optim.lr_scheduler.CosineAnnealingLR(
        optimizer, T_max=num_epochs * len(train_loader)
    )
    
    # 混合精度
    scaler = torch.amp.GradScaler('cuda')
    
    best_val_loss = float('inf')
    output_path = Path(output_dir)
    output_path.mkdir(parents=True, exist_ok=True)
    
    for epoch in range(num_epochs):
        # 训练阶段
        model.train()
        train_losses = []
        
        for batch in train_loader:
            vision = batch['vision_features'].to(device)
            lang = batch['lang_features'].to(device)
            actions = batch['actions'].to(device)
            
            optimizer.zero_grad()
            
            with torch.amp.autocast('cuda'):
                outputs = model(vision, lang, actions)
                loss = outputs['loss']
            
            scaler.scale(loss).backward()
            scaler.unscale_(optimizer)
            torch.nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
            scaler.step(optimizer)
            scaler.update()
            
            scheduler.step()
            train_losses.append(loss.item())
        
        avg_train_loss = sum(train_losses) / len(train_losses)
        
        # 验证阶段
        model.eval()
        val_losses = []
        
        with torch.no_grad():
            for batch in val_loader:
                vision = batch['vision_features'].to(device)
                lang = batch['lang_features'].to(device)
                actions = batch['actions'].to(device)
                
                with torch.amp.autocast('cuda'):
                    outputs = model(vision, lang, actions)
                    loss = outputs['loss']
                
                val_losses.append(loss.item())
        
        avg_val_loss = sum(val_losses) / len(val_losses)
        
        print(f"Epoch {epoch+1}/{num_epochs}")
        print(f"  Train Loss: {avg_train_loss:.4f}")
        print(f"  Val Loss:   {avg_val_loss:.4f}")
        print(f"  LR:         {scheduler.get_last_lr()[0]:.2e}")
        
        # 保存最佳模型
        if avg_val_loss < best_val_loss:
            best_val_loss = avg_val_loss
            torch.save({
                'epoch': epoch,
                'model_state_dict': {
                    k: v for k, v in model.state_dict().items() if 'lora_' in k
                },
                'optimizer_state_dict': optimizer.state_dict(),
                '

                "best_val_loss": best_val_loss
            }, output_path / "best_lora_model.pt")
            print(f"  -> 保存最佳模型到 best_lora_model.pt")

    print(f"训练完成！最佳验证损失: {best_val_loss:.4f}")


if __name__ == "__main__":

### 简答题答案

**6. 答：** 预训练阶段数据配比为互联网视觉-语言数据占90%以上，机器人动作数据占少量（10%以下）。这是因为：
- 互联网数据规模大（数亿到数十亿级别），可以帮助VLA建立强大的视觉-语言基础能力
- 机器人动作数据稀缺且获取成本高，但却是赋予机器人操控能力的关键
- 预训练的目标是让模型具备通用的视觉理解和语言理解能力

微调阶段则几乎全部使用机器人动作数据（任务特定演示数据），因为：
- 微调的目标是让模型学会具体的操控技能
- 需要高质量的动作标注数据
- 数据量需求相对较小（1K-10K条即可）

**7. 答：** Warmup + Cosine Decay分为两个阶段：
- **预热阶段（Warmup）**：学习率从0线性增长到最大学习率。原因：模型参数从随机初始化开始，大幅度学习率可能导致训练不稳定；预热让模型逐步适应，减少参数更新的振荡。
- **余弦退火阶段（Cosine Decay）**：学习率按余弦曲线从最大学习率缓慢衰减到最小学习率。余弦曲线比阶梯式衰减更平滑，让模型在后期用更小的学习率进行细粒度调整。

**8. 答：** MSE损失函数公式为：

$$\mathcal{L}_{MSE} = \frac{1}{T \cdot D} \sum_{t=1}^{T} \sum_{d=1}^{D} (a_{t,d} - \hat{a}_{t,d})^2$$

MSE适用于连续动作预测的原因：
- 动作值是连续数值，MSE对大误差惩罚更重，符合实际需求
- 梯度计算简单，优化稳定
- 对称性：对过高和过低的预测一视同仁

**9. 答：** 在RTX 4090（24GB）上微调7B VLA的优化策略：
- **LoRA/QLoRA微调**：只训练1-5%的参数，显存占用降低100倍
- **梯度累积**：使用小批次（如4）通过累积模拟大批次
- **混合精度（FP16/BF16）**：减少一半显存占用
- **梯度检查点（Gradient Checkpointing）**：用时间换空间
- **DeepSpeed ZeRO-2/3**：分片优化器状态和梯度
- **启用CPU Offload**：优化器状态offload到CPU
- **减少图像分辨率**：如从224降至168

**10. 答：**
- **数据并行（DP）**：每个GPU持有完整模型，处理不同数据批次，训练结束后同步梯度。简单高效，适合模型能放入单卡显存但batch size不足的场景。
- **张量并行（TP）**：将模型的单层权重矩阵切分到多个GPU，如将一个大型矩阵乘法A@B的A按列切分到不同GPU。通信密集，适合单层计算量大的Transformer场景，可以训练参数量超过单卡显存的大模型。
- **主要区别**：DP是数据分布，TP是模型分布；DP通信在梯度同步，TP通信在每层前反向传播中。

import numpy as np

def normalize_actions(actions, action_min, action_max):
    """将动作值线性归一化到[-1, 1]区间"""
    if actions.ndim == 1:
        actions = actions.reshape(1, -1)
        single_input = True
    else:
        single_input = False

    action_range = action_max - action_min
    action_range = np.where(action_range < 1e-6, 1.0, action_range)

    normalized = 2.0 * (actions - action_min) / action_range - 1.0

    if single_input:
        normalized = normalized.squeeze(0)

    return normalized.astype(np.float32)

**12. 简化的LoRA层实现：**

```python
import torch
import torch.nn as nn

class LoRALayer(nn.Module):
    def __init__(self, in_features, out_features, rank=8, alpha=16.0):
        super().__init__()
        self.in_features = in_features
        self.out_features = out_features
        self.rank = rank
        self.alpha = alpha

        self.weight = nn.Parameter(torch.randn(out_features, in_features), requires_grad=False)
        self.bias = nn.Parameter(torch.zeros(out_features), requires_grad=False)
        self.lora_A = nn.Parameter(torch.randn(rank, in_features) * 0.01)
        self.lora_B = nn.Parameter(torch.zeros(out_features, rank))

    def forward(self, x):
        original = nn.functional.linear(x, self.weight, self.bias)
        lora = (self.lora_B @ self.lora_A) @ x.T
        scaled = (self.alpha / self.rank) * lora.T
        return original + scaled

    def get_trainable_params(self):
        trainable = {n: p.numel() for n, p in self.named_parameters() if p.requires_grad}
        total = sum(trainable.values())
        return {
            "trainable_params": trainable,
            "total_trainable": total,
            "compression_ratio": (self.in_features * self.out_features) / total if total > 0 else 0
        }


if __name__ == "__main__":
    layer = LoRALayer(in_features=512, out_features=512, rank=8)
    x = torch.randn(4, 512)
    y = layer(x)
    print(f"Input: {x.shape}, Output: {y.shape}")
    stats = layer.get_trainable_params()
    print(f"Trainable: {stats['total_trainable']:,}, Ratio: {stats['compression_ratio']:.1f}x")
```
