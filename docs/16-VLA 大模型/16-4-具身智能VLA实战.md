## 1. 实战项目概述

### 1.1 项目目标

本课程以一个完整的 **VLA控制机械臂抓取任务** 为实战项目，带领读者从零搭建一套可运行的具身智能系统。项目目标是让机器人能够：

- 接收自然语言指令（如"把红色方块放到蓝色盒子里"）
- 通过视觉感知定位目标物体和场景要素
- 由VLA模型理解任务并生成连续动作
- 控制机械臂完成抓取和放置动作
- 具备失败重试和安全保护机制

**核心能力矩阵**：

| 能力维度 | 具体表现 |
|---------|---------|
| 语言理解 | 解析自然语言指令中的目标物体、位置关系、动作意图 |
| 视觉感知 | 目标检测、深度获取、物体6D位姿估计 |
| 动作生成 | VLA输出连续动作序列（末端位置+夹爪开合） |
| 运动控制 | MoveIt2轨迹规划与执行 |
| 实时反馈 | 闭环控制：动作执行中持续接收视觉反馈 |
| 异常处理 | 抓取失败检测与重试策略 |

## 2. 系统搭建

### 2.1 相机标定

**内参标定**（获取相机固有参数）：

相机内参描述了相机成像的几何模型，是将像素坐标转换为归一化相机坐标的基础。

**内参矩阵**：

$$
\mathbf{K} = \begin{bmatrix} f_x & 0 & c_x \\ 0 & f_y & c_y \\ 0 & 0 & 1 \end{bmatrix}
$$

其中 $f_x, f_y$ 为焦距（像素单位），$c_x, c_y$ 为主点坐标。

**标定流程**：

1. 打印标准棋盘格标定板（建议 9×6 或 11×8 角点）
2. 从多个角度拍摄 20-30 张标定图像
3. 使用 ROS2 标定工具或 OpenCV `calibrateCamera` 进行标定

```python
"""
相机内参标定脚本

使用 OpenCV 对 RealSense D435i 进行内参标定
"""

import cv2
import numpy as np
import glob
import yaml


def calibrate_camera(images_path="calib_images/*.png", 
                     checkerboard=(9, 6)):
    """
    相机内参标定
    
    Args:
        images_path: 标定板图像路径
        checkerboard: 棋盘格角点数量 (列, 行)
    Returns:
        mtx: 内参矩阵
        dist: 畸变系数
        rvecs, tvecs: 旋转向量和平移向量
    """
    # 终止条件：迭代30次或误差变化小于0.001
    criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 30, 0.001)
    
    # 准备棋盘格世界坐标 (z=0平面)
    objp = np.zeros((checkerboard[0] * checkerboard[1], 3), np.float32)
    objp[:, :2] = np.mgrid[0:checkerboard[0], 0:checkerboard[1]].T.reshape(-1, 2)
    
    # 存储棋盘格角点的世界坐标和像素坐标
    objpoints = []  # 3D点
    imgpoints = []  # 2D点
    
    images = glob.glob(images_path)
    print(f"找到 {len(images)} 张标定图像")
    
    for fname in images:
        img = cv2.imread(fname)
        gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
        
        # 查找棋盘格角点
        ret, corners = cv2.findChessboardCorners(gray, checkerboard, None)
        
        if ret:
            objpoints.append(objp)
            
            # 亚像素精细化（提高角点精度到子像素级别）
            corners2 = cv2.cornerSubPix(
                gray, corners, (11, 11), (-1, -1), criteria
            )
            imgpoints.append(corners2)
            
            # 可视化角点检测结果
            cv2.drawChessboardCorners(img, checkerboard, corners2, ret)
            cv2.imshow('Corner Detection', img)
            cv2.waitKey(100)
        else:
            print(f"警告：{fname} 未检测到棋盘格")
    
    cv2.destroyAllWindows()
    
    # 执行标定
    ret, mtx, dist, rvecs, tvecs = cv2.calibrateCamera(
        objpoints, imgpoints, gray.shape[::-1], None, None
    )
    
    print(f"\n=== 标定结果 ===")
    print(f"内参矩阵 K:\n{mtx}")
    print(f"畸变系数 dist: {dist.ravel()}")
    print(f"重投影误差: {ret:.4f} 像素")
    
    # 计算每个标定板的重投影误差
    total_error = 0
    for i in range(len(objpoints)):
        imgpoints2, _ = cv2.projectPoints(objpoints[i], rvecs[i], 
                                          tvecs[i], mtx, dist)
        error = cv2.norm(imgpoints[i], imgpoints2, cv2.NORM_L2) / len(imgpoints2)
        total_error += error
    mean_error = total_error / len(objpoints)
    print(f"平均重投影误差: {mean_error:.4f} 像素")
    
    return mtx, dist, rvecs, tvecs


def save_calibration(mtx, dist, rvecs, tvecs, save_path="camera_calib.yaml"):
    """保存标定结果到 YAML 文件"""
    calib_data = {
        'camera_matrix': mtx.tolist(),
        'dist_coeffs': dist.tolist(),
        'image_width': 640,
        'image_height': 480,
    }
    with open(save_path, 'w') as f:
        yaml.dump(calib_data, f)
    print(f"标定结果已保存至: {save_path}")


if __name__ == "__main__":
    mtx, dist, rvecs, tvecs = calibrate_camera(
        images_path="calib_images/*.png",
        checkerboard=(9, 6)
    )
    save_calibration(mtx, dist, rvecs, tvecs, "config/camera_calib.yaml")
```

**标定质量评估**：

| 指标 | 合格标准 | 说明 |
|------|---------|------|
| 重投影误差 | < 0.5 像素 | 越低越好，一般 < 1.0 可接受 |
| 标定图像数量 | ≥ 20 张 | 覆盖不同角度和距离 |
| 焦距一致性 | $f_x \approx f_y$ | 理想情况下应接近相等 |

### 2.2 手眼标定

手眼标定是建立相机坐标系与机械臂基坐标系之间变换关系的关键步骤。根据相机安装位置的不同，分为两类：

| 类型 | 相机位置 | 标定目的 |
|------|---------|---------|
| **Eye-in-Hand** | 相机固定在机械臂末端 | 求末端到相机的变换 $_e^c\mathbf{T}$ |
| **Eye-to-Hand** | 相机固定在基座旁 | 求基座到相机的变换 $_b^c\mathbf{T}$ |

本项目采用 **Eye-in-Hand** 构型。

**手眼标定数学原理**：

设机械臂末端在基坐标系下的位姿为 $_b^e\mathbf{T}$，我们要求相机在末端坐标系下的位姿 $_e^c\mathbf{T}$。

对于任意标定板位姿 $_c^p\mathbf{T}$（由相机识别标定板得到），有：

$$
{}_b^p\mathbf{T} = {}_b^e\mathbf{T} \cdot {}_e^c\mathbf{T} \cdot {}_c^p\mathbf{T}
$$

通过采集多组数据（$N \geq 10$ 组），构建 **AX = XB** 问题，使用 **Tsai-Lenz** 算法求解 ${}_e^c\mathbf{T}$。

```python
"""
手眼标定脚本 (Eye-in-Hand)

通过ROS2话题获取机械臂末端位姿和标定板位姿，
计算相机到机械臂末端的变换矩阵
"""

import rclpy
from rclpy.node import Node
from geometry_msgs.msg import TransformStamped, PoseStamped
import tf_transformations as tf_trans
import numpy as np
from scipy.spatial.transform import Rotation as Rot


class HandEyeCalibrator(Node):
    """
    手眼标定节点
    
    订阅机械臂末端位姿和标定板位姿，
    收集多组数据后计算手眼变换
    """
    
    def __init__(self):
        super().__init__('hand_eye_calibrator')
        
        # 标定数据存储
        self.T_cb_list = []  # 标定板在相机坐标系下的位姿列表
        self.T_eb_list = []  # 末端在基座坐标系下的位姿列表
        
        # 订阅标定板位姿（来自AprilTag或棋盘格识别）
        self.create_subscription(
            PoseStamped,
            '/apriltag/pose',
            self.tag_pose_callback,
            10
        )
        
        # 订阅机械臂末端位姿（从TF获取）
        self.create_subscription(
            TransformStamped,
            '/tf',
            self.tf_callback,
            10
        )
        
        self.latest_robot_pose = None   # 当前机械臂末端在基座下的位姿
        self.latest_tag_pose = None     # 当前标定板在相机下的位姿
        
        self.get_logger().info("手眼标定节点已启动，请移动机械臂采集数据")
        
    def tag_pose_callback(self, msg: PoseStamped):
        """收到标定板位姿时记录数据"""
        self.latest_tag_pose = self.pose_to_matrix(msg)
        
    def tf_callback(self, msg: TransformStamped):
        """从TF获取机械臂末端在基座下的位姿"""
        # 根据实际的frame_id调整
        if msg.header.frame_id == 'base_link' and msg.child_frame_id == 'tool0':
            self.latest_robot_pose = self.tf_to_matrix(msg.transform)
            
    @staticmethod
    def pose_to_matrix(pose_msg: PoseStamped) -> np.ndarray:
        """将 Pose 消息转换为 4x4 变换矩阵"""
        q = np.array([
            pose_msg.pose.orientation.x,
            pose_msg.pose.orientation.y,
            pose_msg.pose.orientation.z,
            pose_msg.pose.orientation.w
        ])
        t = np.array([
            pose_msg.pose.position.x,
            pose_msg.pose.position.y,
            pose_msg.pose.position.z
        ])
        mat = np.eye(4)
        mat[:3, :3] = tf_trans.quaternion_matrix(q)[:3, :3]
        mat[:3, 3] = t
        return mat
    
    @staticmethod
    def tf_to_matrix(transform) -> np.ndarray:
        """将 TF 消息转换为 4x4 变换矩阵"""
        q = np.array([
            transform.rotation.x,
            transform.rotation.y,
            transform.rotation.z,
            transform.rotation.w
        ])
        t = np.array([
            transform.translation.x,
            transform.translation.y,
            transform.translation.z
        ])
        mat = np.eye(4)
        mat[:3, :3] = tf_trans.quaternion_matrix(q)[:3, :3]
        mat[:3, 3] = t
        return mat
    
    def collect_sample(self) -> bool:
        """
        手动采集一组标定数据
        
        调用时机：机械臂移动到新位置且标定板在相机视野内时
        
        Returns:
            是否采集成功
        """
        if self.latest_robot_pose is None or self.latest_tag_pose is None:
            self.get_logger().warn("数据未就绪，无法采集")
            return False
        
        # T_cb: 标定板在相机坐标系下的位姿
        T_cb = self.latest_tag_pose
        
        # T_eb: 基座在末端坐标系下的位姿（即末端到基座的逆）
        T_eb = np.linalg.inv(self.latest_robot_pose)
        
        self.T_cb_list.append(T_cb)
        self.T_eb_list.append(T_eb)
        
        self.get_logger().info(f"已采集第 {len(self.T_cb_list)} 组数据")
        return True
    
    def solve_hand_eye(self):
        """
        使用 Tsai-Lenz 算法求解手眼标定 AX = XB
        
        Returns:
            T_ec: 末端到相机的变换矩阵 (相机挂在末端时的安装关系)
        """
        if len(self.T_cb_list) < 10:
            self.get_logger().error("数据不足，需要至少10组数据")
            return None
        
        # 方法1: 使用 OpenCV 的 handEyeCalibrate
        # 需要将位姿转为 rvec/tvec 格式
        R_cam2base, t_cam2base = [], []
        R_target2end, t_target2end = [], []
        
        for i in range(len(self.T_cb_list)):
            # T_cb: 标定板在相机下 -> 转置得到相机在标定板下(不需要)
            # T_eb: 基座在末端下 -> 即末端到基座
            
            # OpenCV handEyeCalibrate 的输入格式：
            # R_gripper2base, t_gripper2base: 机械臂末端相对于基座的旋转和平移
            # R_target2cam, t_target2cam: 标定板相对于相机的旋转和平移
            R_gripper2base = self.T_eb_list[i][:3, :3]
            t_gripper2base = self.T_eb_list[i][:3, 3]
            
            R_target2cam = self.T_cb_list[i][:3, :3]
            t_target2cam = self.T_cb_list[i][:3, 3]
            
            R_cam2base.append(R_gripper2base)
            t_cam2base.append(t_gripper2base)
            R_target2end.append(R_target2cam)
            t_target2end.append(t_target2cam)
        
        # 调用 OpenCV 手眼标定
        R_cam2base_arr = np.array(R_cam2base, dtype=np.float64)
        t_cam2base_arr = np.array(t_cam2base, dtype=np.float64).reshape(-1, 1, 3)
        R_target2end_arr = np.array(R_target2end, dtype=np.float64)
        t_target2end_arr = np.array(t_target2end, dtype=np.float64).reshape(-1, 1, 3)
        
        try:
            R_cam2gripper, t_cam2gripper = cv2.calibrateHandEye(
                R_cam2base_arr, t_cam2base_arr,
                R_target2end_arr, t_target2end_arr,
                method=cv2.CALIB_HAND_EYE_TSAI
            )
        except Exception as e:
            self.get_logger().error(f"手眼标定失败: {e}")
            return None
        
        # 组装变换矩阵
        T_ec = np.eye(4)
        T_ec[:3, :3] = R_cam2gripper
        T_ec[:3, 3] = t_cam2gripper.ravel()
        
        self.get_logger().info(f"手眼标定完成!\n末端->相机 变换矩阵:\n{T_ec}")
        
        # 保存结果
        self._save_calibration(T_ec)
        
        return T_ec
    
    def _save_calibration(self, T_ec: np.ndarray, 
                           save_path="config/hand_eye_calib.yaml"):
        """保存手眼标定结果"""
        import yaml
        
        # 将旋转矩阵转为欧拉角（更易读）
        euler = tf_trans.euler_from_matrix(T_ec, 'sxyz')
        
        calib_data = {
            'T_ec': T_ec.tolist(),  # 末端到相机的变换矩阵
            'translation': T_ec[:3, 3].tolist(),
            'euler_deg': [np.rad2deg(e) for e in euler],
            'rotation_matrix': T_ec[:3, :3].tolist(),
        }
        
        with open(save_path, 'w') as f:
            yaml.dump(calib_data, f, default_flow_style=False)
        
        self.get_logger().info(f"手眼标定结果已保存至: {save_path}")


def main():
    rclpy.init()
    calibrator = HandEyeCalibrator()
    rclpy.spin(calibrator)
    calibrator.destroy_node()
    rclpy.shutdown()


if __name__ == "__main__":
    main()
```

| 模型 | 参数量 | 动作维度 | 动作空间 | 特色 | 部署难度 |

### 3.1 视觉感知

**目标检测（YOLOv8）**：

YOLOv8 用于实时检测场景中的目标物体，提供类别、置信度和边界框。

```python
"""
YOLOv8 目标检测模块

检测场景中的目标物体，结合深度图获取3D位置
"""

from ultralytics import YOLO
import cv2
import numpy as np
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image, CameraInfo
from vision_msgs.msg import Detection2DArray
from geometry_msgs.msg import PoseStamped
import message_filters


class ObjectDetector(Node):
    """
    目标检测节点
    
    功能：
    - YOLOv8 实时目标检测
    - 融合深度图获取3D位置
    - 发布检测结果话题
    """
    
    def __init__(self, model_path="models/yolov8_grasp.pt"):
        super().__init__('object_detector')
        
        # 加载YOLOv8模型
        self.model = YOLO(model_path)
        self.class_names = self.model.names  # 获取类别名称
        
        # 相机内参（从CameraInfo话题获取）
        self.camera_matrix = None
        self.dist_coeffs = None
        
        # 深度图像缓存
        self.latest_depth = None
        
        # 订阅相机话题（使用时间同步确保RGB和深度配对）
        self.color_sub = message_filters.Subscriber(
            self, Image, '/camera/color/image_raw'
        )
        self.depth_sub = message_filters.Subscriber(
            self, Image, '/camera/aligned_depth_to_color/image_raw'
        )
        self.create_subscription(
            CameraInfo, '/camera/color/camera_info', 
            self.camera_info_callback, 10
        )
        
        # 近似时间同步（slop=0.05s，允许50ms误差）
        ts = message_filters.ApproximateTimeSynchronizer(
            [self.color_sub, self.depth_sub], 
            queue_size=10, slop=0.05
        )
        ts.registerCallback(self.sync_callback)
        
        # 发布检测结果
        self.det_pub = self.create_publisher(
            Detection2DArray, '/detected_objects', 10
        )
        self.obj_pos_pub = self.create_publisher(
            PoseStamped, '/target_object_position', 10
        )
        
        self.get_logger().info(f"目标检测节点已启动，模型: {model_path}")
    
    def camera_info_callback(self, msg: CameraInfo):
        """获取并缓存相机内参"""
        if self.camera_matrix is None:
            self.camera_matrix = np.array(msg.k).reshape(3, 3)
            self.dist_coeffs = np.array(msg.d)
            self.get_logger().info(f"相机内参:\n{self.camera_matrix}")
    
    def sync_callback(self, color_msg: Image, depth_msg: Image):
        """同步处理RGB和深度图"""
        # 转换图像格式
        color_np = np.frombuffer(color_msg.data, dtype=np.uint8).reshape(
            color_msg.height, color_msg.width, 3
        )
        depth_np = np.frombuffer(depth_msg.data, dtype=np.uint16).reshape(
            depth_msg.height, depth_msg.width
        ).astype(np.float32) / 1000.0  # mm -> m
        
        self.latest_depth = depth_np
        
        # 执行YOLOv8检测
        results = self.model(color_np, verbose=False)
        
        # 处理检测结果
        detections = []
        for result in results:
            boxes = result.boxes
            for box in boxes:
                x1, y1, x2, y2 = box.xyxy[0].cpu().numpy()
                conf = float(box.conf[0])
                cls_id = int(box.cls[0])
                label = self.class_names[cls_id]
                
                # 计算边界框中心
                cx, cy = int((x1 + x2) / 2), int((y1 + y2) / 2)
                
                # 从深度图获取3D位置
                if self.latest_depth is not None:
                    depth_val = self.latest_depth[cy, cx]
                    if 0.01 < depth_val < 5.0:  # 有效深度范围
                        pos_3d = self._pixel_to_3d(cx, cy, depth_val)
                    else:
                        pos_3d = None
                else:
                    pos_3d = None
                
                detections.append({
                    'label': label,
                    'confidence': conf,
                    'bbox': (x1, y1, x2, y2),
                    'center': (cx, cy),
                    'depth': depth_val,
                    'position_3d': pos_3d
                })
        
        self._publish_detections(detections, color_msg.header)
    
    def _pixel_to_3d(self, u: int, v: int, depth: float) -> np.ndarray:
        """
        将像素坐标 + 深度转换为相机坐标系下的3D坐标
        
        Args:
            u, v: 像素坐标
            depth: 深度值 (m)
        Returns:
            position: 3D位置 [x, y, z] (m)，相机坐标系下
        """
        if self.camera_matrix is None:
            return None
        
        fx, fy = self.camera_matrix[0, 0], self.camera_matrix[1, 1]
        cx, cy = self.camera_matrix[0, 2], self.camera_matrix[1, 2]
        
        # 针孔相机模型反投影
        x = (u - cx) * depth / fx
        y = (v - cy) * depth / fy
        z = depth
        
        return np.array([x, y, z])
    
    def _publish_detections(self, detections, header):
        """发布检测结果"""
        from vision_msgs.msg import Detection2DArray, BoundingBox2D
        from std_msgs.msg import Header
        
        msg = Detection2DArray()
        msg.header = header
        
        for det in detections:
            det_msg = Detection2D()
            det_msg.bbox.center.position.x = det['center'][0]
            det_msg.bbox.center.position.y = det['center'][1]
            det_msg.bbox.size_x = det['bbox'][2] - det['bbox'][0]
            det_msg.bbox.size_y = det['bbox'][3] - det['bbox'][1]
            msg.detections.append(det_msg)
        
        self.det_pub.publish(msg)
        
        # 发布高置信度目标的位置（用于VLA控制）
        for det in detections:
            if det['confidence'] > 0.8 and det['position_3d'] is not None:
                pose = PoseStamped()
                pose.header = header
                pose.pose.position.x, pose.pose.position.y, pose.pose.position.z = det['position_3d']
                self.obj_pos_pub.publish(pose)
                break


if __name__ == "__main__":
    rclpy.init()
    node = ObjectDetector(model_path="models/yolov8_grasp.pt")
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()
```

### 3.2 自然语言指令解析

**基于规则的指令解析**（快速可复现）：

```python
"""
自然语言指令解析模块

将自然语言指令解析为结构化的任务描述
"""

import re
from dataclasses import dataclass
from typing import Optional, List


@dataclass
class GraspCommand:
    """抓取命令结构化表示"""
    action: str              # 动作: "pick", "place", "push"
    target_object: str       # 目标物体
    source_location: Optional[str] = None   # 源位置
    target_location: Optional[str] = None  # 目标位置
    object_color: Optional[str] = None       # 物体颜色
    constraints: Optional[List[str]] = None   # 额外约束


class InstructionParser:
    """
    指令解析器
    
    支持常见指令模式：
    - "把红色的杯子拿起来"
    - "把方块放到蓝色盒子里"
    - "将红色方块从桌面左侧移到右侧"
    """
    
    # 动作关键词映射
    ACTION_MAP = {
        'pick': ['拿起', '抓取', '拿', '捡起', 'grasp', 'pick'],
        'place': ['放下', '放置', '放到', '放到...里', 'place', 'put'],
        'push': ['推', '移动', 'push', 'move'],
        'pour': ['倒', '倾倒', 'pour'],
    }
    
    # 颜色关键词
    COLOR_MAP = {
        '红色': 'red', '蓝色': 'blue', '绿色': 'green',
        '黄色': 'yellow', '白色': 'white', '黑色': 'black',
        '橙色': 'orange', '紫色': 'purple',
    }
    
    # 介词（用于识别位置关系）
    PREPOSITIONS = ['到', '在', '向', '往', 'from', 'to', 'into']
    
    def parse(self, instruction: str) -> GraspCommand:
        """
        将自然语言指令解析为结构化命令
        
        Args:
            instruction: 自然语言指令
        Returns:
            GraspCommand: 结构化命令
        """
        instruction = instruction.strip().lower()
        
        # 1. 识别动作类型
        action = self._extract_action(instruction)
        
        # 2. 识别颜色
        color = self._extract_color(instruction)
        
        # 3. 识别目标物体
        target = self._extract_target(instruction)
        
        # 4. 识别位置关系（源位置和目标位置）
        source, target_loc = self._extract_locations(instruction)
        
        return GraspCommand(
            action=action,
            target_object=target,
            source_location=source,
            target_location=target_loc,
            object_color=color,
        )
    
    def _extract_action(self, text: str) -> str:
        """识别动作类型"""
        for action, keywords in self.ACTION_MAP.items():
            for kw in keywords:
                if kw in text:
                    return action
        return 'pick'  # 默认动作为抓取
    
    def _extract_color(self, text: str) -> Optional[str]:
        """识别颜色"""
        for cn, ce in self.COLOR_MAP.items():
            if cn in text or ce in text:
                return ce
        return None
    
    def _extract_target(self, text: str) -> str:
        """识别目标物体"""
        # 常见物体关键词
        common_objects = [
            '杯子', 'cup', '方块', 'block', 'cube',
            '盒子', 'box', '瓶子', 'bottle', '苹果', 'apple',
            '香蕉', 'banana', '碗', 'bowl', '盘', 'plate'
        ]
        for obj in common_objects:
            if obj in text:
                return obj
        return 'object'  # 默认物体
    
    def _extract_locations(self, text: str) -> tuple:
        """识别源位置和目标位置"""
        # 位置关键词
        locations = {
            'source': ['左侧', '左边', '左侧', '左', 'right', 'from'],
            'target': ['右侧', '右边', '右侧', 'box', '容器', 'container', '盒子里'],
        }
        
        source, target_loc = None, None
        
        for loc_name, keywords in locations.items():
            for kw in keywords:
                if kw in text:
                    if loc_name == 'source':
                        source = kw
                    else:
                        target_loc = kw
        
        return source, target_loc
    
    def build_vla_instruction(self, cmd: GraspCommand) -> str:
        """
        将结构化命令转换为 VLA 模型可理解的指令
        
        Args:
            cmd: 结构化命令
        Returns:
            VLA prompt 指令字符串
        """
        parts = []
        
        # 构建简洁的英文指令（OpenVLA使用英文指令）
        if cmd.action == 'pick':
            parts.append(f"pick up the")
        elif cmd.action == 'place':
            parts.append(f"place the")
        elif cmd.action == 'push':
            parts.append(f"move the")
        
        if cmd.object_color:
            parts.append(f"{cmd.object_color} {cmd.target_object}")
        else:
            parts.append(cmd.target_object)
        
        if cmd.target_location:
            if cmd.action == 'pick':
                parts.append(f"from {cmd.target_location}")
            else:
                parts.append(f"to {cmd.target_location}")
        
        return ' '.join(parts)

if __name__ == "__main__":
```

### 3.3 VLA动作生成

**OpenVLA 推理引擎封装**：

```python
"""
OpenVLA 推理引擎

封装 OpenVLA 模型的加载和推理，提供 ROS2 兼容的接口
"""

import torch
import numpy as np
from PIL import Image
from typing import Optional, Tuple
from transformers import AutoProcessor, AutoModelForVision2Seq
from collections import deque
import time


class OpenVLAEngine:
    """
    OpenVLA 推理引擎
    
    支持 FP16 / INT8 / INT4 量化
    提供动作平滑和性能监控
    """
    
    def __init__(
        self,
        model_id: str = "openvla/openvla-7b-widowx",
        quant_mode: str = "int8",
        device: str = "cuda",
        action_buffer_size: int = 3,
        blend_alpha: float = 0.7,
    ):
        """
        Args:
            model_id: HuggingFace 模型 ID
            quant_mode: 量化模式 ('fp16', 'int8', 'int4')
            device: 计算设备 ('cuda', 'cpu')
            action_buffer_size: 动作平滑缓冲大小
            blend_alpha: 动作平滑系数
        """
        self.model_id = model_id
        self.quant_mode = quant_mode
        self.device = device if torch.cuda.is_available() else "cpu"
        self.blend_alpha = blend_alpha
        
        # 动作平滑缓冲
        self.action_buffer = deque(maxlen=action_buffer_size)
        
        # 性能统计
        self.inference_times = deque(maxlen=100)
        
        # 加载模型
        self.model, self.processor = self._load_model()
        print(f"[OpenVLA] 模型加载完成 | 设备: {self.device} | 量化: {quant_mode}")
    
    def _load_model(self):
        """加载模型（支持多种量化模式）"""
        print(f"[OpenVLA] 正在加载模型: {self.model_id}")
        
        if self.quant_mode == "int8":
            # INT8 量化：显存减半，速度提升约1.5-2x
            model = AutoModelForVision2Seq.from_pretrained(
                self.model_id,
                load_in_8bit=True,
                device_map="auto",
                trust_remote_code=True,
            )
        elif self.quant_mode == "int4":
            # INT4 量化：显存减少75%，速度提升2-4x，精度损失较大
            from transformers import BitsAndBytesConfig
            bnb_config = BitsAndBytesConfig(
                load_in_4bit=True,
                bnb_4bit_quant_type="nf4",
                bnb_4bit_compute_dtype=torch.float16,
            )
            model = AutoModelForVision2Seq.from_pretrained(
                self.model_id,
                quantization_config=bnb_config,
                device_map="auto",
                trust_remote_code=True,
            )
        else:
            # FP16：原始精度，显存占用最大
            model = AutoModelForVision2Seq.from_pretrained(
                self.model_id,
                torch_dtype=torch.float16,
                device_map="auto",
                trust_remote_code=True,
            )
        
        processor = AutoProcessor.from_pretrained(
            self.model_id,
            trust_remote_code=True,
        )
        
        model.eval()
        
        return model, processor
    
    @torch.no_grad()
    def predict_action(
        self,
        image: Image.Image,
        instruction: str,
    ) -> Tuple[np.ndarray, float]:
        """
        预测机器人动作（核心推理函数）
        
        Args:
            image: 当前相机图像 (PIL Image)
            instruction: 自然语言指令 (英文)
        
        Returns:
            action: 归一化动作向量 [-1, 1]，形状 (7,)
            inference_time: 推理耗时 (秒)
        """
        start_time = time.time()
        
        # 构建 VLA prompt（格式：Task: {指令}. Bot: In:）
        prompt = f"Task: {instruction}. Bot: In:"
        
        # 预处理：图像 + 文本 -> token IDs
        inputs = self.processor(
            text=prompt,
            images=image,
            return_tensors="pt",
            padding=True,
        )
        inputs = {
            k: v.to(self.device) if isinstance(v, torch.Tensor) else v
            for k, v in inputs.items()
        }
        
        # 自回归生成动作 tokens
        outputs = self.model.generate(
            **inputs,
            max_new_tokens=8,    # 7个关节 + 1个夹爪 = 8 tokens
            do_sample=False,      # 贪心解码，保证确定性
            output_scores=False,
        )
        
        # 解析生成的动作 token
        generated_ids = outputs.sequences[0]
        action_token_ids = generated_ids[-8:]  # 取最后8个token为动作
        
        # 使用模型内置的动作解码器（将token转为物理动作值）
        action = self.model.decode_action(action_token_ids)
        
        inference_time = time.time() - start_time
        self.inference_times.append(inference_time)
        
        # 动作平滑
        action = self._smooth_action(action)
        
        return action, inference_time
    
    def _smooth_action(self, new_action: np.ndarray) -> np.ndarray:
        """
        动作平滑：对新动作与历史动作进行指数滑动平均
        
        平滑公式：a_smooth = alpha * a_new + (1-alpha) * a_buffer_avg
        
        Args:
            new_action: 新预测的动作
        Returns:
            平滑后的动作
        """
        self.action_buffer.append(new_action)
        
        if len(self.action_buffer) == 1:
            return new_action
        
        # 对缓冲中的动作取加权平均
        weights = np.array([
            self.blend_alpha * (1 - self.blend_alpha) ** (len(self.action_buffer) - i - 1)
            for i in range(len(self.action_buffer))
        ])
        weights = weights / weights.sum()
        
        smoothed = np.average(list(self.action_buffer), axis=0, weights=weights)
        return smoothed.astype(np.float32)
    
    def get_stats(self) -> dict:
        """获取推理性能统计"""
        if not self.inference_times:
            return {}
        
        times = list(self.inference_times)
        return {
            'mean_time': np.mean(times),
            'std_time': np.std(times),
            'min_time': np.min(times),
            'max_time': np.max(times),
            'fps': 1.0 / np.mean(times) if np.mean(times) > 0 else 0,
        }
    
    def denormalize_action(
        self,
        action: np.ndarray,
        joint_min: list = None,
        joint_max: list = None,
    ) -> np.ndarray:
        """
        将归一化动作 [-1, 1] 反归一化到物理关节空间
        
        Args:
            action: 归一化动作 (7,)
            joint_min: 各关节最小值
            joint_max: 各关节最大值
        Returns:
            物理关节空间的动作值
        """
        if joint_min is None:
            joint_min = [-3.14, -3.14, -3.14, -3.14, -3.14, -3.14, 0.0]
        if joint_max is None:
            joint_max = [3.14, 3.14, 3.14, 3.14, 3.14, 3.14, 0.085]
        
        joint_min = np.array(joint_min)
        joint_max = np.array(joint_max)
        
        # 反归一化: x_phys = (x_norm + 1) / 2 * (max - min) + min
        # 假设 action 范围为 [-1, 1]
        phys_action = (action + 1.0) / 2.0 * (joint_max - joint_min) + joint_min
        
        return phys_action
```

### 4.1 动作平滑处理

VLA 直接输出的动作序列往往存在高频抖动，直接执行会导致机械臂振动和定位不准。

**常用平滑方法对比**：

| 方法 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|---------|
| **指数移动平均 (EMA)** | $a_t = \alpha a_t^{VLA} + (1-\alpha)a_{t-1}$ | 简单、实时性好 | 有延迟 | 实时控制 |
| **卡尔曼滤波** | 状态估计 + 观测更新 | 最优估计 | 需调参 | 高精度场景 |
| **Savitzky-Golay滤波** | 多项式拟合 | 保形性好 | 计算量稍大 | 离线处理 |
| **动作缓冲+插值** | 历史动作缓冲，插值生成 | 轨迹平滑 | 增加延迟 | 轨迹跟踪 |

### 4.2 失败重试机制

```python
"""
失败检测与重试策略

检测常见失败模式并自动重试或回退
"""

import time
from enum import Enum
from dataclasses import dataclass
from typing import Optional


class FailureMode(Enum):
    """失败模式枚举"""
    NONE = "none"
    GRASP_FAILED = "grasp_failed"       # 抓取失败（夹爪合拢后物体未在预期位置）
    COLLISION = "collision"             # 碰撞检测
    TIMEOUT = "timeout"                # 执行超时
    STUCK = "stuck"                     # 动作卡住（连续单调动作）
    VISION_LOST = "vision_lost"         # 目标丢失


@dataclass
class GraspResult:
    """抓取结果"""
    success: bool
    failure_mode: FailureMode = FailureMode.NONE
    attempts: int = 0
    message: str = ""


class FailureDetector:
    """
    失败检测器
    
    监控执行过程中的异常情况
    """
    
    def __init__(self):
        # 动作历史（用于检测卡住）
        self.action_history = []
        self.max_history = 20
        
        # 统计
        self.consecutive_same_direction = 0
        self.last_action = None
        
    def update(self, action: np.ndarray, 
               gripper_width: float,
               target_visible: bool = True):
        """
        更新状态，检测异常
        
        Args:
            action: 当前动作
            gripper_width: 当前夹爪宽度
            target_visible: 目标是否在视野内
        Returns:
            FailureMode: 检测到的失败模式
        """
        # 1. 检查目标是否丢失
        if not target_visible:
            return FailureMode.VISION_LOST
        
        # 2. 检查是否卡住（连续动作方向相同且幅度相近）
        if self.last_action is not None:
            delta = np.abs(action - self.last_action).mean()
            if delta < 0.005:  # 动作变化极小
                self.consecutive_same_direction += 1
                if self.consecutive_same_direction > 10:
                    return FailureMode.STUCK
            else:
                self.consecutive_same_direction = 0
        
        self.last_action = action.copy()
        
        # 3. 检查夹爪状态
        if gripper_width < 0.005:
            # 夹爪几乎闭合但目标不可见 = 抓取失败
            if not target_visible:
                return FailureMode.GRASP_FAILED
        
        return FailureMode.NONE


class RetryController:
    """
    重试控制器
    
    根据失败模式决定重试策略
    """
    
    # 最大重试次数
    MAX_RETRIES = 3
    
    # 重试间隔 (秒)
    RETRY_DELAY = 1.0
    
    def __init__(self):
        self.attempt_count = 0
        self.failure_history = []
    
    def should_retry(self, failure_mode: FailureMode) -> bool:
        """判断是否应该重试"""
        if self.attempt_count >= self.MAX_RETRIES:
            return False
        
        # 可重试的失败模式
        retryable = [
            FailureMode.GRASP_FAILED,
            FailureMode.STUCK,
            FailureMode.TIMEOUT,
        ]
        
        return failure_mode in retryable
    
    def get_recovery_action(self, failure_mode: FailureMode) -> str:
        """
        根据失败模式获取恢复动作
        
        Args:
            failure_mode: 失败模式
        Returns:
            恢复动作描述
        """
        recovery_map = {
            FailureMode.GRASP_FAILED: "retract_and_reapproach",
            FailureMode.STUCK: "go_home_and_retry",
            FailureMode.TIMEOUT: "go_home_and_retry",
            FailureMode.COLLISION: "stop_and_wait",
            FailureMode.VISION_LOST: "search_for_target",
        }
        
        return recovery_map.get(failure_mode, "stop_and_wait")
    
    def record_failure(self, failure_mode: FailureMode):
        """记录失败"""
        self.failure_history.append({
            'time': time.time(),
            'mode': failure_mode,
        })
        self.attempt_count += 1
    
    def reset(self):
        """重置重试计数器"""
        self.attempt_count = 0
        self.failure_history = []
```

**跳帧策略实现**：

```python
"""
实时性优化：跳帧推理

每隔 N 帧执行一次 VLA 推理，中间帧复用上一个动作
适用于 VLA 推理频率低于控制频率的场景
"""

class FrameSkipController:
    """
    跳帧控制器
    
    VLA 推理频率和控制频率不匹配时，
    通过跳帧减少推理次数，提高实时性
    """
    
    def __init__(self, skip_frames: int = 2):
        """
        Args:
            skip_frames: 跳帧数
                        控制频率 30Hz，skip=2 -> 推理频率 10Hz
        """
        self.skip_frames = skip_frames
        self.frame_counter = 0
        self.last_action = None
    
    def should_infer(self) -> bool:
        """判断当前帧是否需要推理"""
        self.frame_counter += 1
        return (self.frame_counter - 1) % (self.skip_frames + 1) == 0
    
    def get_action(self, new_action: np.ndarray) -> np.ndarray:
        """
        获取当前帧应执行的动作
        
        Args:
            new_action: 新推理的动作（仅在 should_infer()=True 时有效）
        Returns:
            当前帧应执行的动作
        """
        if self.should_infer():
            self.last_action = new_action
        
        return self.last_action


class AsyncInferenceEngine:
    """
    异步推理引擎
    
    在独立线程中运行 VLA 推理，不阻塞主控制循环
    """
    
    def __init__(self, vla_engine: OpenVLAEngine):
        import threading
        from queue import Queue
        
        self.vla_engine = vla_engine
        self.inference_queue = Queue(maxsize=1)  # 最新图像帧
        self.result_queue = Queue(maxsize=1)    # 最新推理结果
        
        self.running = True
        
        # 启动推理线程
        self.inference_thread = threading.Thread(
            target=self._inference_loop,
            daemon=True
        )
        self.inference_thread.start()
    
    def _inference_loop(self):
        """推理线程主循环"""
        while self.running:
            try:
                # 获取最新图像（阻塞，最多等 0.1 秒）
                image, instruction = self.inference_queue.get(timeout=0.1)
                
                # 执行推理
                action, inf_time = self.vla_engine.predict_action(
                    image, instruction
                )
                
                # 放入结果队列（覆盖旧结果，只保留最新）
                if self.result_queue.full():
                    try:
                        self.result_queue.get_nowait()
                    except:
                        pass
                self.result_queue.put((action, inf_time))
                
            except:
                continue  # 队列为空，继续等待
    
    def submit(self, image, instruction: str):
        """提交推理任务"""
        if self.inference_queue.full():
            try:
                self.inference_queue.get_nowait()
            except:
                pass
        self.inference_queue.put((image, instruction))
    
    def get_result(self, timeout: float = 0.05):
        """获取推理结果"""
        try:
            return self.result_queue.get(timeout=timeout)
        except:
            return None
    
    def stop(self):
        """停止推理线程"""
        self.running = False
        self.inference_thread.join(timeout=1.0)
```

```python
        # ========== 4. 状态管理 ==========
        
        self.current_instruction = None   # 当前指令
        self.current_image = None         # 当前图像
        self.current_depth = None         # 当前深度图
        self.target_position = None       # 目标物体位置
        self.is_executing = False         # 是否正在执行
        self.execution_start_time = None  # 执行开始时间
        
        # 性能统计
        self.stats = {
            'inference_times': deque(maxlen=50),
            'total_attempts': 0,
            'success_count': 0,
            'failure_count': 0,
        }
        
        self.get_logger().info("VLA 机械臂抓取节点已启动！")
        self.get_logger().info("请发布指令: ros2 topic pub /grasp_instruction std_msgs/String 'data: 把红色方块拿起来'")
        
        # ========== 5. 指令订阅 ==========
        from std_msgs.msg import String
        self.create_subscription(
            String,
            '/grasp_instruction',
            self.instruction_callback,
            10
        )
    
    def instruction_callback(self, msg):
        """接收自然语言指令"""
        instruction = msg.data
        self.get_logger().info(f"收到指令: {instruction}")
        
        # 解析指令
        cmd = self.instruction_parser.parse(instruction)
        vla_instr = self.instruction_parser.build_vla_instruction(cmd)
        
        self.current_instruction = vla_instr
        self.get_logger().info(f"VLA指令: {vla_instr}")
        
        # 启动执行
        self.start_grasp(cmd)
    
    def target_callback(self, msg: PoseStamped):
        """接收目标物体位置（来自感知模块）"""
        self.target_position = np.array([
            msg.pose.position.x,
            msg.pose.position.y,
            msg.pose.position.z,
        ])
    
    def image_callback(self, rgb_msg: Image, depth_msg: Image):
        """同步处理相机图像"""
        # 转换 ROS 图像为 PIL Image
        rgb_np = np.frombuffer(rgb_msg.data, dtype=np.uint8).reshape(
            rgb_msg.height, rgb_msg.width, 3
        )
        self.current_image = PILImage.fromarray(rgb_np)
        
        depth_np = np.frombuffer(depth_msg.data, dtype=np.uint16).reshape(
            depth_msg.height, depth_msg.width
        ).astype(np.float32) / 1000.0
        self.current_depth = depth_np
        
        # 如果正在执行且有指令，执行 VLA 推理
        if self.is_executing and self.current_instruction:
            self._execute_vla_step()
    
    def start_grasp(self, cmd: GraspCommand):
        """开始抓取任务"""
        self.is_executing = True
        self.execution_start_time = time.time()
        self.retry_controller.reset()
        self.failure_detector = FailureDetector()
        
        self.get_logger().info(f"开始执行抓取任务，尝试次数: {self.retry_controller.attempt_count + 1}")
    
    def _execute_vla_step(self):
        """执行一个 VLA 推理-控制步骤"""
        if self.current_image is None or self.current_instruction is None:
            return
        
        # ===== 1. VLA 推理 =====
        try:
            action, inf_time = self.vla_engine.predict_action(
                self.current_image,
                self.current_instruction,
            )
            self.stats['inference_times'].append(inf_time)
            
        except Exception as e:
            self.get_logger().error(f"VLA 推理失败: {e}")
            return
        
        # ===== 2. 动作反归一化 =====
        phys_action = self.vla_engine.denormalize_action(action)
        
        # ===== 3. 安全检查 =====
        if not self._safety_check(phys_action):
            self.get_logger().warn("动作未通过安全检查")
            # 执行安全回退
            self._execute_recovery("go_home")
            return
        
        # ===== 4. 发布动作命令 =
        # ===== 4. 发布动作命令 =====
        self._publish_action(phys_action)
        
        # ===== 5. 失败检测 =====
        failure_mode = self.failure_detector.update(
            action=action,
            gripper_width=phys_action[6] if len(phys_action) > 6 else 0.0,
            target_visible=(self.target_position is not None),
        )
        
        # ===== 6. 执行超时检测 =====
        elapsed = time.time() - self.execution_start_time
        if elapsed > 60.0:  # 60秒超时
            self.get_logger().warn("执行超时")
            self._handle_failure(FailureMode.TIMEOUT)
            return
        
        # ===== 7. 任务完成检测 =====
        if self._check_success():
            self._handle_success()
    
    def _safety_check(self, action: np.ndarray) -> bool:
        """执行安全检查"""
        # 关节限位检查
        joint_min = np.array([-3.14, -3.14, -3.14, -3.14, -3.14, -3.14, 0.0])
        joint_max = np.array([3.14, 3.14, 3.14, 3.14, 3.14, 3.14, 0.085])
        
        if np.any(action < joint_min) or np.any(action > joint_max):
            return False
        
        # 夹爪范围检查
        if action[6] < 0 or action[6] > 0.085:
            return False
        
        return True
    
    def _publish_action(self, action: np.ndarray):
        """发布机械臂动作"""
        traj = JointTrajectory()
        traj.joint_names = [
            'shoulder_pan_joint', 'shoulder_lift_joint', 'elbow_joint',
            'wrist_1_joint', 'wrist_2_joint', 'wrist_3_joint'
        ]
        
        point = JointTrajectoryPoint()
        point.positions = action[:6].tolist()
        point.time_from_start = rclpy.duration.Duration(seconds=0.1).to_msg()
        traj.points.append(point)
        
        self.action_pub.publish(traj)
        
        # 发布夹爪命令
        gripper_msg = Float64()
        gripper_msg.data = float(action[6])
        self.gripper_pub.publish(gripper_msg)
    
    def _check_success(self) -> bool:
        """检查任务是否成功完成"""
        # 检测夹爪是否闭合到目标宽度（表示已抓取）
        # 实际中应结合力矩传感器或视觉反馈综合判断
        return False  # 需要根据具体任务定义
    
    def _handle_success(self):
        """处理任务成功"""
        self.is_executing = False
        self.stats['success_count'] += 1
        self.get_logger().info("抓取任务成功！")
        
        avg_inf_time = np.mean(list(self.stats['inference_times']))
        self.get_logger().info(
            f"性能统计: 平均推理时间={avg_inf_time*1000:.1f}ms, "
            f"FPS={1.0/avg_inf_time:.1f}"
        )
    
    def _handle_failure(self, failure_mode: FailureMode):
        """处理任务失败"""
        self.stats['failure_count'] += 1
        self.get_logger().warn(f"任务失败: {failure_mode.value}")
        
        # 记录失败并决定是否重试
        self.retry_controller.record_failure(failure_mode)
        
        if self.retry_controller.should_retry(failure_mode):
            recovery = self.retry_controller.get_recovery_action(failure_mode)
            self.get_logger().info(f"执行恢复动作: {recovery}")
            self._execute_recovery(recovery)
        else:
            self.is_executing = False
            self.get_logger().error("已达到最大重试次数，任务终止")
    
    def _execute_recovery(self, recovery_type: str):
        """执行恢复动作"""
        if recovery_type == "go_home":
            # 回退到初始位置
            home_action = np.array([0.0, -1.57, 1.57, 0.0, 0.0, 0.0, 0.0])
            self._publish_action(home_action)
            
        elif recovery_type == "retract_and_reapproach":
            # 稍微后退然后重新接近
            pass
        
        # 重置状态，等待下一帧
        self.is_executing = True


def main():
    rclpy.init()
    node = VLAGraspNode()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == "__main__":
    main()
```

**8. VLA 边缘部署的实时性与安全性手段**

| 手段 | 实时性效果 | 安全性效果 | 说明 |
|------|-----------|-----------|------|
| **INT8 量化** | 推理速度 1.5-2x | 无直接影响 | 显存减半，支持更大批次 |
| **跳帧推理** | 推理频率减半 | 无直接影响 | 控制频率不变，中间帧复用 |
| **异步推理** | 消除推理阻塞 | 无直接影响 | 推理在独立线程，不卡控制循环 |
| **动作平滑 (EMA)** | 无直接影响 | 防止动作突变 | alpha=0.7 兼顾响应与平滑 |
| **动作限幅** | 无直接影响 | 限制单步最大位移 | joint_delta < 0.15 rad |
| **图像降分辨率** | 显著减少计算量 | 无直接影响 | 224×224 → 160×128 |
| **TensorRT 加速** | 速度 2-5x | 无直接影响 | NVIDIA 专用，需模型转换 |

**组合方案推荐**：
- 实时性：INT8 量化 + 异步推理 + 跳帧策略
- 安全性：EMA 平滑 + 单步限幅 + 安全检查

| 碰撞检测触发 | 立即停止 | 立即停止 |
| 右臂超出工作空间 | 等待 | 强制回退 |
| 协作力超过阈值 | 增力/保持 | 减力/释放 |
| 通信超时 (>100ms) | 保持当前 | 保持当前，等待恢复 |

**异常恢复策略**：

```python
class DualArmRecoveryController:
    """双臂异常恢复控制器"""
    
    def handle_collision(self):
        """碰撞后恢复"""
        self.left_arm.stop()
        self.right_arm.stop()
        self.left_arm.retract(safe_distance=0.1)
        self.right_arm.retract(safe_distance=0.1)
        self.log_collision()
        return 'DUAL_ARM_COLLISION_HANDLED'
    
    def handle_sync_failure(self):
        """双臂动作不同步"""
        # 计算位置差
        pos_diff = np.linalg.norm(
            self.left_arm.position - self.right_arm.position
        )
        if pos_diff > 0.3:  # 超过30cm差异
            self.right_arm.match_position(self.left_arm.position)
        return 'POSITIONS_SYNCED'
    
    def handle_communication_timeout(self):
        """通信超时"""
        self.left_arm.hold()
        self.right_arm.hold()
        time.sleep(0.5)
        if not self.check_connection():
            self.execute_emergency_stop()
        return 'TIMEOUT_HANDLED'
```
