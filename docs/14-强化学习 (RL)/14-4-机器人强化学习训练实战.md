## 1. 机器人强化学习应用场景

### 1.1 机械臂控制（Manipulation）

机械臂控制是强化学习在机器人领域最经典的应用之一：

| 任务类型 | 描述 | 典型环境 |
|----------|------|----------|
| **抓取（ Grasping）** | 学会将末端执行器移动到目标物体并抓取 | Shadow-Tactile、Gripper |
| **放置（ Placing）** | 将抓取的物体放到指定位置 | FetchReach、Sawyer |
| **开门/抽屉（Door Opening）** | 操作带有铰链的物体 | KukaDoor |
| **装配（ Assembly）** | 多步精细操作任务 | PegaTable |

**状态空间设计**：
```python
# 机械臂控制典型状态空间
state = {
    "joint_positions":      # 关节角度 [6-7维]
    "joint_velocities":     # 关节速度
    "end_effector_pose":   # 末端位置+姿态 [6维]
    "target_object_pose":  # 目标物体位置 [3维]
    "gripper_state":       # 夹爪开合状态
}
```

**动作空间设计**：
- **连续动作空间**：每个关节的角度增量或速度指令（适合PPO、SAC）
- **离散动作空间**：预定义的动作序列（适合DQN）

### 1.2 机器人导航（Navigation）

移动机器人的强化学习导航任务：

| 任务类型 | 描述 | 典型环境 |
|----------|------|----------|
| **点对点导航** | 从起点移动到目标点 | PointBot、FetchNav |
| **避障导航** | 在动态/静态障碍物中穿行 | NarrowCorridor |
| **探索（Exploration）** | 最大化地图覆盖面积 | ExploreGrid |
| **社交导航** | 在人群中安全行走 | SocialNav |

**观测空间设计**：
```python
# 激光雷达+视觉导航典型观测
observation = {
    "laser_scan":        # 360°激光雷达距离 [360-1080维]
    "robot_pose":        # 机器人当前位姿 (x, y, theta)
    "goal_position":    # 目标点相对位置 (dx, dy)
    "velocity":          # 当前速度 (vx, vy, omega)
    # 可选：RGB-D相机图像
}
```

**动作空间设计**：
- **连续速度**：线速度 + 角速度 $(v, \omega)$
- **离散动作**：前进、左转、右转、后退等

### 1.3 足式机器人运动（Locomotion）

### 2.2 ROS2 Gymnasium 环境封装

以下代码展示如何将ROS2控制的机器人封装为Gymnasium兼容环境：

```python
"""
ROS2 + Gymnasium 机器人强化学习环境封装
"""
import gymnasium as gym
import numpy as np
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import JointState
from geometry_msgs.msg import Pose
from trajectory_msgs.msg import JointTrajectory, JointTrajectoryPoint


class ROS2RobotEnv(gym.Env):
    """
    ROS2机器人强化学习环境基类
    封装ROS2通信为Gymnasium标准接口
    """
    
    def __init__(self, ns="", episode_max_steps=100):
        """
        初始化ROS2环境
        
        参数:
            ns: 机器人命名空间
            episode_max_steps: 每回合最大步数
        """
        super().__init__()
        
        # 初始化ROS2
        rclpy.init(args=None)
        self.node = Node(f'rl_env_{ns}')
        
        # 状态空间：7维关节位置 + 7维关节速度 + 3维目标位置
        self.observation_space = gym.spaces.Box(
            low=-np.inf, high=np.inf, shape=(17,), dtype=np.float32
        )
        # 动作空间：7维关节速度控制
        self.action_space = gym.spaces.Box(
            low=-1.0, high=1.0, shape=(7,), dtype=np.float32
        )
        
        self.episode_max_steps = episode_max_steps
        self.current_step = 0
        
        # ROS2发布者/订阅者
        self.joint_pub = self.node.create_publisher(
            JointTrajectory, '/joint_trajectory', 10
        )
        self.joint_sub = self.node.create_subscription(
            JointState, '/joint_states', self._joint_callback, 10
        )
        
        # 状态缓存
        self._joint_positions = np.zeros(7)
        self._joint_velocities = np.zeros(7)
        self._target_position = np.zeros(3)
        
        # 等待连接就绪
        self.node.get_logger().info("等待ROS2连接...")
        for _ in range(10):
            rclpy.spin_once(self.node, timeout_sec=0.1)
        
        self.node.get_logger().info("ROS2环境初始化完成")
    
    def _joint_callback(self, msg: JointState):
        """更新关节状态"""
        self._joint_positions = np.array(msg.position[:7])
        self._joint_velocities = np.array(msg.velocity[:7])
    
    def reset(self, seed=None, options=None):
        """重置环境，返回初始观测"""
        super().reset(seed=seed)
        self.current_step = 0
        
        # 发送重置指令（关节回到初始位置）
        self._send_joint_command(np.zeros(7))
        
        # 随机生成新的目标位置
        self._target_position = self.np_random.uniform(
            low=[0.3, -0.3, 0.1],
            high=[0.7, 0.3, 0.5]
        )
        
        observation = self._get_observation()
        info = {}
        
        return observation, info
    
    def step(self, action):
        """
        执行动作，返回 (next_obs, reward, terminated, truncated, info)
        """
        self.current_step += 1
        
        # 将动作（-1,1）映射到实际关节速度范围
        joint_velocities = action * 0.5  # 最大0.5 rad/s
        
        # 发送关节速度命令
        self._send_joint_command(joint_velocities)
        
        # 等待状态更新（实际中需要精确时序控制）
        for _ in range(5):
            rclpy.spin_once(self.node, timeout_sec=0.01)
        
        # 获取新观测
        observation = self._get_observation()
        
        # 计算奖励
        reward, terminated = self._compute_reward()
        
        # 检查截断条件
        truncated = self.current_step >= self.episode_max_steps
        
        info = {"is_success": terminated}
        
        return observation, reward, terminated, truncated, info
    
    def _get_observation(self):
        """拼接观测向量"""
        end_effector = self._forward_kinematics(self._joint_positions)
        return np.concatenate([
            self._joint_positions,       # 7维
            self._joint_velocities,       # 7维
            self._target_position         # 3维
        ], dtype=np.float32)
    
    def _send_joint_command(self, velocities):
        """发送关节速度命令"""
        msg = JointTrajectory()
        msg.joint_names = [
            'shoulder_pan_joint', 'shoulder_lift_joint', 'elbow_joint',
            'wrist_1_joint', 'wrist_2_joint', 'wrist_3_joint',
            'finger_left_joint'  # 夹爪
        ]
        point = JointTrajectoryPoint()
        point.velocities = velocities.tolist()
        point.time_from_start = rclpy.duration.Duration(
            seconds=0.1
        ).to_msg()
        msg.points.append(point)
        self.joint_pub.publish(msg)
    
    def _forward_kinematics(self, joint_positions):
        """
        简化版正运动学（实际应使用URDF或KDL）
        返回末端执行器位置
        """
        # 假设末端位置与关节位置线性相关（实际需精确计算）
        return joint_positions[:3] * 0.2  # 简化模型
    
    def _compute_reward(self):
        """
        计算奖励函数
        
        任务目标：末端执行器移动到目标位置
        """
        end_effector = self._forward_kinematics(self._joint_positions)
        distance = np.linalg.norm(end_effector - self._target_position)
        
        # 距离奖励：越近越高
        reward = -distance * 10.0
        
        # 成功判定：距离小于5cm
        terminated = distance < 0.05
        
        if terminated:
            reward += 100.0  # 成功奖励
        
        return reward, terminated
    
    def close(self):
        """清理ROS2资源"""
        self.node.destroy_node()
        rclpy.shutdown()
```

### 3.2 奖励函数设计

抓取任务的奖励函数设计至关重要：

```python
def compute_fetch_reach_reward(observation, action):
    """
    FetchReach任务奖励函数
    
    目标：使机械臂末端移动到目标位置
    """
    achieved_goal = observation["achieved_goal"]
    desired_goal = observation["desired_goal"]
    
    # 计算距离
    distance = np.linalg.norm(achieved_goal - desired_goal)
    
    # 密集奖励：距离越近奖励越高
    reward = -distance
    
    # 成功判定
    success = distance < 0.05  # 5cm精度
    
    return reward, success


def compute_fetch_pick_place_reward(observation, action):
    """
    FetchPickAndPlace任务奖励函数
    
    目标：将物体从初始位置移动到目标位置
    """
    achieved_goal = observation["achieved_goal"]
    desired_goal = observation["desired_goal"]
    object_pos = observation["achieved_goal"]  # 物体当前位置
    
    # 物块到目标距离
    object_to_goal = np.linalg.norm(object_pos - desired_goal)
    
    # 末端到物块距离
    gripper_to_object = np.linalg.norm(
        observation["observation"][:3] - object_pos
    )
    
    # 多层次奖励设计
    reward = 0.0
    
    # 1. 接近物块
    if gripper_to_object > 0.1:
        reward -= gripper_to_object * 2.0
    else:
        reward += 10.0  # 接近奖励
    
    # 2. 抓取物块（夹爪接近物块）
    if gripper_to_object < 0.05:
        reward += 20.0
    
    # 3. 举动物块
    if object_pos[2] > 0.2:
        reward += 15.0
    
    # 4. 移动到目标
    reward -= object_to_goal * 5.0
    
    success = object_to_goal < 0.05
    
    return reward, success
```

### 3.3 SAC算法训练机械臂

```python
"""
SAC（Soft Actor-Critic）算法训练Fetch机械臂抓取
SAC擅长连续控制任务，且不需要过多的超参数调优
"""
import gymnasium as gym
import numpy as np
from stable_baselines3 import SAC
from stable_baselines3.common.vec_env import DummyVecEnv, VecNormalize


def make_fetch_env():
    """创建Fetch抓取环境"""
    env = gym.make("FetchPickAndPlaceDense-v2")
    return env


def train_sac_fetch():
    log_dir = "./logs/sac_fetch/"
    
    env = DummyVecEnv([make_fetch_env])
    env = VecNormalize(env, norm_obs=True, norm_reward=False)
    
    # SAC配置
    model = SAC(
        "MultiInputPolicy",  # 支持字典观测空间
        env,
        learning_rate=3e-4,
        buffer_size=100000,
        learning_starts=1000,
        batch_size=256,
        tau=0.005,           # 软更新系数
        gamma=0.99,
        train_freq=1,
        gradient_steps=1,
        ent_coef="auto",     # 自动调节熵系数
        verbose=1,
        tensorboard_log=log_dir + "tb/",
        device="cuda"       # 使用GPU加速
    )
    
    print("开始训练 SAC (FetchPickAndPlace)...")
    model.learn(
        total_timesteps=1000000,
        log_interval=10,
        progress_bar=True
    )
    
    model.save(log_dir + "sac_fetch_pick_place")
    env.save(log_dir + "vecnormalize.pkl")
    print("训练完成！")


def evaluate(model, env, num_episodes=20):
    """评估训练好的智能体"""
    successes = []
    
    for episode in range(num_episodes):
        obs, info = env.reset()
        episode_reward = 0
        done = False
        
        while not done:
            action, _ = model.predict(obs, deterministic=True)
            obs, reward, terminated, truncated, info = env.step(action)
            episode_reward += reward
            done = terminated or truncated
        
        successes.append(info.get("is_success", False))
        print(f"Episode {episode+1}: Reward={episode_reward:.2f}, "
              f"Success={info.get('is_success', False)}")
    
    success_rate = np.mean(successes)
    print(f"\n评估结果: {num_episodes}个回合 平均成功率={success_rate*100:.1f}%")
    return success_rate


if __name__ == "__main__":
    train_sac_fetch()
```

### 4.2 自定义激光雷达导航环境

以下代码展示如何构建基于激光雷达的导航环境：

```python
"""
基于激光雷达的移动机器人导航强化学习环境
"""
import gymnasium as gym
import numpy as np
from gymnasium import spaces
import math


class LidarNavEnv(gym.Env):
    """
    移动机器人激光雷达导航环境
    
    任务：机器人从起点移动到目标点，避开障碍物
    """
    
    metadata = {"render_modes": ["human", "rgb_array"]}
    
    def __init__(self, num_lidar_beams=360, max_steps=500):
        super().__init__()
        
        self.num_lidar_beams = num_lidar_beams
        self.max_steps = max_steps
        
        # 动作空间：线速度 + 角速度（连续）
        self.action_space = spaces.Box(
            low=np.array([0.0, -1.57]),   # [v_min, omega_min]
            high=np.array([0.5, 1.57]),   # [v_max, omega_max]
            dtype=np.float32
        )
        
        # 观测空间：激光雷达 + 目标相对位置
        self.observation_space = spaces.Box(
            low=0.0, high=10.0,
            shape=(num_lidar_beams + 4,),  # lidar + [dx, dy, v, omega]
            dtype=np.float32
        )
        
        # 环境状态
        self.robot_pos = np.zeros(2)      # x, y
        self.robot_theta = 0.0            # 朝向角
        self.goal_pos = np.zeros(2)
        self.obstacles = []
        self.current_step = 0
        
        self.render_mode = "human"
    
    def reset(self, seed=None, options=None):
        super().reset(seed=seed)
        self.current_step = 0
        
        # 随机放置机器人起点（原点附近）
        self.robot_pos = self.np_random.uniform(
            low=[-2.0, -2.0], high=[-1.0, -1.0]
        )
        self.robot_theta = self.np_random.uniform(-math.pi, math.pi)
        
        # 随机放置目标点
        self.goal_pos = self.np_random.uniform(
            low=[1.0, 1.0], high=[2.0, 2.0]
        )
        
        # 生成随机障碍物
        self._generate_obstacles(num=5)
        
        return self._get_observation(), {}
    
    def _generate_obstacles(self, num):
        """生成随机圆形障碍物"""
        self.obstacles = []
        for _ in range(num):
            obstacle = self.np_random.uniform(
                low=[-1.5, -1.5], high=[1.5, 1.5]
            )
            # 确保障碍物不在起点或终点
            if (np.linalg.norm(obstacle - self.robot_pos) > 0.5 and
                np.linalg.norm(obstacle - self.goal_pos) > 0.5):
                self.obstacles.append({"pos": obstacle, "radius": 0.3})
    
    def step(self, action):
        self.current_step += 1
        
        v, omega = action
        
        # 更新机器人位姿（简化的运动学模型）
        dt = 0.1
        self.robot_theta += omega * dt
        self.robot_pos[0] += v * math.cos(self.robot_theta) * dt
        self.robot_pos[1] += v * math.sin(self.robot_theta) * dt
        
        # 获取激光雷达数据
        lidar = self._get_lidar_scan()
        
        # 计算奖励
        reward, terminated = self._compute_reward()
        
        # 检查截断
        truncated = self.current_step >= self.max_steps
        
        info = {"is_success": terminated}
        
        return self._get_observation(), reward, terminated, truncated, info
    
    def _get_lidar_scan(self):
        """
        模拟激光雷达扫描
        返回360维距离数组
        """
        lidar = np.full(self.num_lidar_beams, 10.0)  # 最大距离10m
        
        for i in range(self.num_lidar_beams):
            angle = self.robot_theta + (2 * math.pi * i / self.num_lidar_beams)
            ray_dir = np.array([math.cos(angle), math.sin(angle)])
            
            # 检测障碍物碰撞
            for obs in self.obstacles:
                dist_to_obs = np.linalg.norm(self.robot_pos - obs["pos"])
                if dist_to_obs < obs["radius"]:
                    hit_dist = dist_to_obs - obs["radius"]
                    if hit_dist < lidar[i]:
                        lidar[i] = hit_dist
            
            # 检测边界
            boundary_dist = 5.0 - np.dot(ray_dir, self.robot_pos)
            if 0 < boundary_dist < lidar[i]:
                lidar[i] = max(0, boundary_dist)
        
        return lidar
    
    def _get_observation(self):
        """拼接观测向量"""
        # 相对目标位置
        dx = self.goal_pos[0] - self.robot_pos[0]
        dy = self.goal_pos[1] - self.robot_pos[1]
        
        # 目标角度
        goal_angle = math.atan2(dy, dx) - self.robot_theta
        goal_dist = np.linalg.norm([dx, dy])
        
        lidar = self._get_lidar_scan()
        
        return np.concatenate([
            lidar / 10.0,          # 归一化激光雷达
            [dx / 5.0, dy / 5.0, goal_angle / math.pi, goal_dist / 5.0]
        ], dtype=np.float32)
    
    def _compute_reward(self):
        """计算奖励"""
        dist_to_goal = np.linalg.norm(self.robot_pos - self.goal_pos)
        
        # 密集奖励
        reward = -dist_to_goal * 2.0
        
        # 检查是否到达目标
        terminated = dist_to_goal < 0.15  # 15cm精度
        
        if terminated:
            reward += 100.0
        
        # 碰撞障碍物惩罚
        for obs in self.obstacles:
            if np.linalg.norm(self.robot_pos - obs["pos"]) < obs["radius"]:
                reward -= 50.0
                terminated = True
                break
        
        # 边界惩罚
        if np.linalg.norm(self.robot_pos) > 4.5:
            reward -= 20.0
            terminated = True
        
        return reward, terminated
    
    def render(self):
        if self.render_mode == "human":
            try:
                import matplotlib.pyplot as plt
                plt.clf()
                
                # 绘制激光雷达扫描
                lidar = self._get_lidar_scan()
                angles = np.linspace(0, 2*math.pi, self.num_lidar_beams)
                xs = self.robot_pos[0] + lidar * np.cos(angles + self.robot_theta)
                ys = self.robot_pos[1] + lidar * np.sin(angles + self.robot_theta)
                plt.plot(xs, ys, 'g.', alpha=0.3, markersize=1)
                
                # 绘制机器人
                plt.plot(self.robot_pos[0], self.robot_pos[1], 'bo', markersize=10)
                dir_x = self.robot_pos[0] + 0.3 * math.cos(self.robot_theta)
                dir_y = self.robot_pos[1] + 0.3 * math.sin(self.robot_theta)
                plt.plot([self.robot_pos[0], dir_x], [self.robot_pos[1], dir_y], 'b-')
                
                # 绘制目标
                plt.plot(self.goal_pos[0], self.goal_pos[1], 'r*', markersize=15)
                
                # 绘制障碍物
                for obs in self.obstacles:
                    circle = plt.Circle(
                        obs["pos"], obs["radius"],
                        color='gray', alpha=0.5
                    )
                    plt.gca().add_patch(circle)
                
                plt.xlim(-5, 5)
                plt.ylim(-5, 5)
                plt.grid(True)
                plt.pause(0.01)
            except ImportError:
                pass
    
    def close(self):
        pass
```

## 5. Sim-to-Real：仿真到真实机器人迁移

### 5.1 Domain Randomization

**Domain Randomization (DR)** 是Sim-to-Real最常用的技术，通过在仿真中随机化物理参数，使策略被迫学习对变化鲁棒：

```python
"""
Domain Randomization 配置示例
"""
class DomainRandomizer:
    """仿真参数随机化"""
    
    def __init__(self):
        self.params = {
            # 物理参数
            "robot_mass":       {"base": 5.0, "range": 1.0},    # 机器人质量 ±1kg
            "joint_friction":  {"base": 0.1, "range": 0.05},   # 关节摩擦力
            "motor_strength":   {"base": 1.0, "range": 0.2},     # 电机强度
            "gravity":          {"base": 9.81, "range": 0.5},   # 重力加速度
            
            # 传感器噪声
            "lidar_noise":      {"base": 0.0, "range": 0.05},  # 激光雷达噪声
            "joint_noise":      {"base": 0.0, "range": 0.02},   # 关节位置噪声
            "camera_noise":     {"base": 0.0, "range": 0.1},    # 相机噪声
            
            # 视觉参数
            "light_intensity":  {"base": 1.0, "range": 0.3},
            "light_position":   {"base": [0, 0, 2], "range": 0.5},
            "object_color":     {"base": [1, 1, 1], "range": 0.2}
        }
    
    def sample(self, rng=None):
        """采样一组随机参数"""
        if rng is None:
            rng = np.random.default_rng()
        
        sampled = {}
        for name, cfg in self.params.items():
            base = cfg["base"]
            range_val = cfg["range"]
            
            if isinstance(base, (int, float)):
                sampled[name] = base + rng.uniform(-range_val, range_val)
            elif isinstance(base, list):
                sampled[name] = [
                    b + rng.uniform(-range_val, range_val)
                    for b in base
                ]
        
        return sampled
    
    def apply_to_sim(self, sim, params):
        """将随机参数应用到仿真器"""
        sim.set_robot_mass(params["robot_mass"])
        sim.set_joint_friction(params["joint_friction"])
        sim.set_motor_strength(params["motor_strength"])
        sim.set_gravity(params["gravity"])
        sim.set_lidar_noise(params["lidar_noise"])
        # ... 其他参数应用
```

```python
"""
机器人系统辨识流程
"""
import numpy as np


def identify_robot_properties(robot, num_samples=100):
    """
    通过实验数据辨识机器人物理参数
    
    参数:
        robot: 真实机器人接口
        num_samples: 实验采样次数
    
    返回:
        identified_params: 辨识得到的参数字典
    """
    identified_params = {}
    
    # 1. 辨识关节摩擦力（通过恒定速度运动）
    print("辨识关节摩擦力...")
    friction_data = []
    for _ in range(num_samples):
        velocity, torque = robot.measure_velocity_torque()
        friction_data.append((velocity, torque))
    
    # 使用库仑摩擦模型拟合
    velocities = [d[0] for d in friction_data]
    torques = [d[1] for d in friction_data]
    friction_coef = np.polyfit(velocities, torques, 1)
    identified_params["friction_coef"] = friction_coef[0]
    
    # 2. 辨识关节惯性（通过阶跃响应）
    print("辨识关节惯性...")
    identified_params["inertia"] = robot.measure_inertia_from_step_response()
    
    # 3. 辨识末端执行器位置偏移
    print("辨识末端执行器偏移...")
    identified_params["ee_offset"] = robot.calibrate_ee_offset()
    
    return identified_params


def apply_identified_params(sim, params):
    """将辨识参数应用到仿真"""
    sim.set_friction_model(params["friction_coef"])
    sim.set_joint_inertia(params["inertia"])
    sim.set_ee_offset(params["ee_offset"])
    print(f"已应用辨识参数: {params}")
```

### 6.3 进阶技术

**1. ADR (Automatic Domain Randomization)**
自动逐步增加随机化范围，无需人工调参：

```python
class AdaptiveDomainRandomizer:
    """自适应Domain Randomization"""
    
    def __init__(self, initial_range=0.1, adaptation_rate=1.02):
        self.range = initial_range
        self.adaptation_rate = adaptation_rate
    
    def adapt(self, success_rate):
        """
        根据成功率自适应调整随机化范围
        
        参数:
            success_rate: 最近N个回合的平均成功率
        """
        if success_rate > 0.8:
            # 策略表现好，增加随机化范围
            self.range *= self.adaptation_rate
            print(f"增加随机化范围: {self.range:.4f}")
        elif success_rate < 0.5:
            # 策略表现差，减小随机化范围
            self.range /= self.adaptation_rate
            print(f"减小随机化范围: {self.range:.4f}")
```

**2. RLBC (Reinforcement Learning from Bootstrapped Corrections)**

通过人类校正来微调策略，弥补Sim-to-Real差距：

```python
def rlbc_correction(model, robot, num_corrections=100):
    """
    RLBC：收集人类校正数据并微调策略
    
    每隔N步让人类操作者接管，记录校正动作
    """
    corrections = []
    
    obs, _ = robot.reset()
    for i in range(num_corrections * 100):
        action, _ = model.predict(obs, deterministic=True)
        
        # 人类操作者可以通过摇杆接管
        human_action = robot.get_human_action()
        
        if human_action is not None:
            # 记录校正
            corrections.append((obs, human_action, action))
            print(f"收集校正数据: {len(corrections)}/{num_corrections}")
        
        # 执行动作
        if human_action is not None:
            robot.execute_action(human_action)
        else:
            robot.execute_action(action)
        
        obs, _, done, _, _ = robot.step(None)
        
        if done:
            obs, _ = robot.reset()
    
    # 使用校正数据微调
    if corrections:
        model.fine_tune(corrections)
        print("策略微调完成")
```

### 练习2：设计机械臂抓取奖励函数

假设你要训练一个机械臂抓取桌面上物体的策略，状态包括：末端执行器位置、物块位置、夹爪状态。设计一个合理的奖励函数，并说明每个奖励项的作用。

**答案**：

```python
def grasp_reward(obs, action):
    """
    机械臂抓取奖励函数
    
    状态包含：
    - gripper_pos: 末端执行器位置
    - object_pos: 物块位置
    - gripper_width: 夹爪开合程度
    """
    gripper_pos = obs["gripper_pos"]
    object_pos = obs["object_pos"]
    gripper_width = obs["gripper_width"]
    
    # 奖励项1：接近物块（阶段1）
    dist_to_object = np.linalg.norm(gripper_pos - object_pos)
    reward = -dist_to_object * 10.0  # 距离越近奖励越高
    
    # 奖励项2：接近后张开夹爪准备抓取
    if dist_to_object < 0.1:
        if gripper_width > 0.05:  # 夹爪处于张开状态
            reward += 5.0
    
    # 奖励项3：抓取物块（夹爪合拢且包裹物块）
    if gripper_width < 0.02 and dist_to_object < 0.05:
        reward += 20.0
    
    # 奖励项4：举起物块
    if object_pos[2] > 0.15:  # 物块高度超过阈值
        reward += 15.0
    
    # 奖励项5：移动物块到目标位置
    goal_pos = obs["goal_pos"]
    dist_to_goal = np.linalg.norm(object_pos - goal_pos)
    reward -= dist_to_goal * 20.0
    
    # 奖励项6：成功放置
    if dist_to_goal < 0.05:
        reward += 100.0
    
    return reward
```

**各奖励项的作用**：
- **接近物块**：引导末端执行器向物块移动
- **张开夹爪**：确保在接近物块时夹爪处于正确状态
- **抓取物块**：奖励精确抓取动作
- **举起物块**：确保物块被成功夹起
- **移动到目标**：引导物块向目标位置移动
- **成功放置**：最终任务目标奖励

   - 真实值5.0kg在[4.0, 6.0]范围内 ✓
   - 误差范围±20%，对大多数控制器是合理的
   - 如果真实质量测量准确，且机器人构造稳定，20%的随机范围通常足够

2. **策略在真实机器人上效果不好的可能原因**：

| 可能原因 | 说明 | 检查方法 |
|----------|------|----------|
| 质量测量不准确 | 真实质量可能与测量值有偏差 | 重新测量；使用秤 |
| 重心位置未随机化 | 仅随机化质量但忽略了重心偏移 | 在DR中加入重心随机化 |
| 摩擦力差异 | 关节摩擦力与仿真不一致 | 录制数据对比摩擦损耗 |
| 传感器标定误差 | 关节位置传感器有零偏 | 静止时检查读数 |
| 控制器带宽差异 | 真实电机响应与仿真模型不符 | 测试阶跃响应对比 |
| 动作延迟过大 | 通信延迟导致动作滞后 | 测量并补偿延迟 |

### 练习4：导航任务策略评估

你在FetchNav环境中训练了一个导航策略，在仿真测试中成功率为92%，但在真实机器人测试时成功率仅60%。请列出可能原因并给出改进建议。

**答案**：

**可能原因**：

1. **观测差异**：
   - 仿真中目标位置是精确已知的，真实环境中目标定位可能有噪声/误差
   - 激光雷达在真实环境中可能有阳光干扰、镜面反射等

2. **物理差异**：
   - 真实机器人运动学模型与仿真URDF不完全一致
   - 轮子打滑、地形不平整

3. **时序差异**：
   - 仿真控制周期固定（如50Hz），真实系统可能有抖动
   - 观测-决策-动作链路存在延迟

4. **稀疏奖励陷阱**：
   - 仿真测试时奖励函数可能过于密集，掩盖了真实环境中的策略缺陷

**改进建议**：

| 改进方向 | 具体措施 |
|----------|----------|
| 增加观测噪声DR | 在仿真中加入GPS/视觉定位噪声 |
| 增加地形DR | 在仿真中加入斜坡、不平坦地面 |
| 增加延迟模拟 | 在仿真动作执行中加入随机延迟 |
| 目标位置模糊化 | 使用概率地图而非精确坐标 |
| 课程学习 | 从简单场景逐步过渡到复杂场景 |
| 真实环境微调 | 使用DAgger等方法在真实环境微调 |

### 练习5：比较PPO和SAC在机器人任务中的适用性

针对以下三个任务，选择你认为更适合的算法并说明理由：
1. 四足机器人复杂地形行走
2. 机械臂精细装配（如插销任务）
3. 移动机器人高速避障导航

**答案**：

| 任务 | 推荐算法 | 理由 |
|------|----------|------|
| **四足机器人复杂地形行走** | **PPO** | 地形行走需要稳定、确定性的步态策略；PPO的训练稳定性更好，适合高风险物理任务；样本效率适中，可通过仿真积累大量数据 |
| **机械臂精细装配** | **SAC** | 精细装配需要探索多种精确动作组合；SAC的随机策略更适合探索；连续动作空间的样本效率高；自动熵调节减少人工调参 |
| **移动机器人高速避障** | **PPO** | 高速避障需要快速响应；PPO的决策延迟相对较低；稳定的策略梯度更新适合需要泛化的复杂动态环境 |

**综合对比**：
- PPO：更稳定，适合安全关键应用（如足式机器人）
- SAC：更灵活，适合精细操作任务（如装配）
