## 1. 机器人RL概述

### 1.1 连续动作空间的挑战

传统DQN适用于**离散动作空间**（左/右、上/下），但机器人控制的动作空间几乎全是连续的：

| 应用场景 | 动作维度 | 动作类型 |
|----------|----------|----------|
| 机械臂关节控制 | 6-7维 | 关节角度/扭矩 |
| 移动机器人 | 2维 | 线速度 + 角速度 |
| 四足机器人 | 12维 | 12个关节角度 |
| 人形机器人 | 30+维 | 全身关节 |

**连续动作空间的核心挑战**：

1. **无法枚举动作**：$\max_a Q(s,a)$ 在连续空间不可行（无法遍历所有动作）
2. **控制精度要求高**：机器人需要精确的力/位置指令，误差容忍度低
3. **时序耦合**：当前动作影响未来状态，动作序列而非单步决策

**解决方案**：
- **Actor-Critic 架构**：用神经网络直接输出动作（策略函数 $\pi_\theta(a|s)$）
- **确定性策略**：$a = \pi_\theta(s)$，直接输出动作值
- **随机策略**：$\pi_\theta(a|s) = \mathcal{N}(\mu_\theta(s), \sigma^2)$，输出动作分布

### 1.2 Sim2Real：仿真到真实迁移

**Sim2Real** 是机器人RL的核心难题——在仿真中训练的策略往往无法直接迁移到真实机器人。

```
仿真环境                    真实机器人
┌──────────────┐          ┌──────────────┐
│  精确模型    │          │  未知误差    │
│  完美传感器  │  ──迁移──▶  │  噪声干扰    │
│  确定性物理  │          │ 磨损/松动    │
└──────────────┘          └──────────────┘
```

**Sim2Real 差距的来源**：

| 类别 | 仿真 | 真实 |
|------|------|------|
| **物理参数** | 标称值（质量、摩擦、惯性） | 实际值偏差 |
| **传感器** | 无噪声、零延迟 | 噪声、漂移、延迟 |
| **执行器** | 理想响应 | 死区、饱和、滞后 |
| **接触力学** | 简化碰撞模型 | 复杂摩擦/弹性接触 |

### 1.3 域随机化（Domain Randomization）

**域随机化（Domain Randomization, DR）** 是解决Sim2Real差距的主流方法：

**核心思想**：在训练时**随机化**仿真器的物理/传感器参数，强制策略学会**对变化鲁棒**，从而在真实环境中也能有效泛化。

$$
\theta^* = \arg\max_\theta \mathbb{E}_{d \sim D(\phi)} \left[ \sum_{t=0}^{T} r(s_t, a_t) \right]
$$

其中 $d$ 是从参数分布 $D(\phi)$ 中采样的域参数（质量、摩擦、噪声等）。

**DR 的随机化维度**：

```python
class DomainRandomizer:
    """域随机化配置"""
    
    def __init__(self):
        self.randomization_config = {
            # 物理参数随机化
            "robot_mass": {"base": 3.5, "range": 1.0},       # 质量 ±1kg
            "joint_friction": {"base": 0.1, "range": 0.08}, # 关节摩擦力
            "motor_strength": {"base": 1.0, "range": 0.3},  # 电机强度
            "gravity": {"base": 9.81, "range": 0.5},        # 重力加速度
            
            # 传感器噪声随机化
            "lidar_noise": {"base": 0.0, "range": 0.05},   # 激光雷达噪声
            "joint_position_noise": {"base": 0.0, "range": 0.01},  # 关节位置噪声
            "imu_noise": {"base": 0.0, "range": 0.02},      # IMU噪声
            
            # 视觉随机化
            "camera_exposure": {"base": 0.5, "range": 0.3},  # 相机曝光
            "light_intensity": {"base": 1.0, "range": 0.4}, # 光照强度
        }
```

**DR 的变种**：

| 方法 | 原理 | 适用场景 |
|------|------|----------|
| **均匀随机化** | 固定范围内均匀采样 | 初期探索 |
| **自适应DR (ADR)** | 根据成功率动态调整范围 | 训练稳定后扩展 |
| **guided DR** | 用真实数据引导随机化 | 有少量真实数据 |
| **tsallis DR** | 基于Tsallis统计的随机化 | 非高斯噪声 |

### 2.3 ROS2 通信接口实现

```python
"""
ROS2 机器人 RL 环境基类
将 ROS2 通信接口封装为 Gymnasium 标准环境
"""
import gymnasium as gym
import numpy as np
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import JointState
from geometry_msgs.msg import Twist
from trajectory_msgs.msg import JointTrajectory, JointTrajectoryPoint


class ROS2RobotEnv(gym.Env):
    """
    ROS2 机器人强化学习环境基类
    
    提供标准的 Gymnasium 接口：
    - reset() -> observation, info
    - step(action) -> observation, reward, terminated, truncated, info
    - close()
    """
    
    def __init__(self, ns="", episode_max_steps=100):
        """
        参数:
            ns: 机器人命名空间
            episode_max_steps: 每回合最大步数
        """
        super().__init__()
        
        # 初始化 ROS2 节点
        rclpy.init(args=None)
        self.node = Node(f'rl_env_{ns}')
        
        # 定义观测空间和动作空间（子类需根据具体机器人重写）
        self.observation_space = gym.spaces.Box(
            low=-np.inf, high=np.inf, shape=(17,), dtype=np.float32
        )
        self.action_space = gym.spaces.Box(
            low=-1.0, high=1.0, shape=(7,), dtype=np.float32
        )
        
        self.episode_max_steps = episode_max_steps
        self.current_step = 0
        
        # ROS2 发布者/订阅者初始化（子类中具体实现）
        self._init_ros_comm()
        
        # 状态缓存
        self._state_cache = {}
        
        self.node.get_logger().info("ROS2 RL 环境初始化完成")
    
    def _init_ros_comm(self):
        """初始化 ROS2 通信接口（子类实现）"""
        raise NotImplementedError("子类必须实现 ROS2 通信接口")
    
    def reset(self, seed=None, options=None):
        """重置环境，返回初始观测"""
        super().reset(seed=seed)
        self.current_step = 0
        
        # 发送重置指令
        self._send_reset_command()
        
        # 随机生成新的任务目标
        self._sample_new_goal()
        
        observation = self._get_observation()
        info = {}
        
        return observation, info
    
    def step(self, action):
        """
        执行动作，返回 (next_obs, reward, terminated, truncated, info)
        
        参数:
            action: 动作向量（由 RL 算法输出）
        """
        self.current_step += 1
        
        # 将动作映射为具体的机器人命令
        self._send_action_command(action)
        
        # 等待状态更新
        self._spin_once()
        
        # 获取新观测
        observation = self._get_observation()
        
        # 计算奖励
        reward, terminated = self._compute_reward(observation)
        
        # 检查截断条件（步数超限）
        truncated = self.current_step >= self.episode_max_steps
        
        info = {"is_success": terminated, "step": self.current_step}
        
        return observation, reward, terminated, truncated, info
    
    def _get_observation(self):
        """获取当前观测向量（子类实现）"""
        raise NotImplementedError
    
    def _send_action_command(self, action):
        """发送动作命令到机器人（子类实现）"""
        raise NotImplementedError
    
    def _compute_reward(self, observation):
        """计算奖励（子类实现）"""
        raise NotImplementedError
    
    def _send_reset_command(self):
        """发送重置命令"""
        pass
    
    def _sample_new_goal(self):
        """采样新任务目标"""
        pass
    
    def _spin_once(self):
        """处理一次 ROS2 回调"""
        rclpy.spin_once(self.node, timeout_sec=0.01)
    
    def close(self):
        """清理 ROS2 资源"""
        self.node.destroy_node()
        rclpy.shutdown()
```

## 3. 典型机器人RL任务

### 3.1 移动机器人导航

**TurtleBot3** 是最流行的ROS2学习平台，使用激光雷达进行自主导航：

| 组件 | 规格 |
|------|------|
| 处理器 | Raspberry Pi / Jetson Nano |
| 激光雷达 | LDS-02 (360°, 12m) |
| 电机 | 两轮差速驱动 |
| 动作空间 | 2维: $(v, \omega)$ |
| 观测空间 | 激光雷达(360维) + 目标相对位置(2维) |

**导航任务定义**：
- **目标**：从起点移动到指定目标点
- **奖励**：接近目标正奖励，碰撞障碍物负奖励
- **终止条件**：到达目标（距离<0.3m）或碰撞

### 3.2 机械臂操控

**UR5** 是 6 自由度协作机械臂，广泛用于 RL 研究：

| 参数 | 值 |
|------|---|
| 工作半径 | 850mm |
| 自由度 | 6 (RRRRRR) |
| 负载 | 5kg |
| 重复精度 | ±0.03mm |
| 动作空间 | 6维关节位置/速度 |
| 观测空间 | 关节状态 + 末端位姿 + 目标位置 |

**典型任务**：
- **Reach**：末端到达目标点
- **Pick**：抓取物体
- **Place**：将物体放置到目标位置

### 3.3 双足行走

**双足行走**是RL最难的任务之一，需要全身协调控制：

### 4.2 TurtleBot3 PPO 导航环境

```python
"""
TurtleBot3 激光雷达导航强化学习环境
使用 PPO 算法训练移动机器人自主导航
"""
import gymnasium as gym
import numpy as np
from gymnasium import spaces
import math
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import LaserScan
from geometry_msgs.msg import Twist
from nav_msgs.msg import Odometry
from std_srvs.srv import Empty


class TurtleBot3NavEnv(gym.Env):
    """
    TurtleBot3 导航环境
    
    任务：使用激光雷达感知环境，训练策略控制机器人从起点导航到目标点
    
    动作空间: 2维连续 (线速度 v, 角速度 omega)
    观测空间: 激光雷达(360维) + 目标相对位置(2维) + 当前速度(2维)
    """
    
    metadata = {"render_modes": ["human", "rgb_array"]}
    
    def __init__(self, ns="turtlebot3", num_lidar=360, max_steps=500, use_ros2=False):
        super().__init__()
        
        self.ns = ns
        self.num_lidar = num_lidar
        self.max_steps = max_steps
        self.current_step = 0
        self.use_ros2 = use_ros2
        
        # 动作空间: [v, omega]
        self.action_space = spaces.Box(
            low=np.array([0.0, -1.57]),   # 最小线速度0，最大角速度-90°/s
            high=np.array([0.26, 1.57]),  # 最大线速度0.26m/s，最大角速度+90°/s
            dtype=np.float32
        )
        
        # 观测空间: 激光雷达 + 目标相对位置 + 当前速度
        self.observation_space = spaces.Box(
            low=0.0, high=10.0,
            shape=(num_lidar + 4,),
            dtype=np.float32
        )
        
        # 状态变量
        self.robot_pos = np.zeros(2)      # x, y
        self.robot_theta = 0.0           # 朝向角
        self.robot_vel = np.zeros(2)     # v, omega
        self.goal_pos = np.zeros(2)      # 目标位置
        self.lidar_data = np.full(num_lidar, 10.0)  # 激光雷达数据
        self.last_lidar = np.full(num_lidar, 10.0)   # 上一帧激光雷达
        
        # 障碍物地图（仿真用）
        self.obstacles = []
        
        # ROS2 接口
        if use_ros2:
            rclpy.init(args=None)
            self.node = Node(f'nav_rl_{ns}')
            self._init_ros2_comm()
        
        self.render_mode = "human"
        self.viewer = None
    
    def _init_ros2_comm(self):
        """初始化 ROS2 通信"""
        # 订阅激光雷达
        self.lidar_sub = self.node.create_subscription(
            LaserScan,
            f'/{self.ns}/scan',
            self._lidar_callback,
            10
        )
        
        # 订阅里程计
        self.odom_sub = self.node.create_subscription(
            Odometry,
            f'/{self.ns}/odom',
            self._odom_callback,
            10
        )
        
        # 发布速度命令
        self.vel_pub = self.node.create_publisher(Twist, f'/{self.ns}/cmd_vel', 10)
        
        self.node.get_logger().info("ROS2 通信初始化完成")
    
    def _lidar_callback(self, msg: LaserScan):
        """激光雷达数据回调"""
        ranges = np.array(msg.ranges)
        # 处理无效值（inf/nan -> 最大距离）
        ranges = np.nan_to_num(ranges, nan=10.0, posinf=10.0)
        self.lidar_data = np.clip(ranges, 0.0, 10.0)
    
    def _odom_callback(self, msg: Odometry):
        """里程计数据回调"""
        # 获取位置
        self.robot_pos[0] = msg.pose.pose.position.x
        self.robot_pos[1] = msg.pose.pose.position.y
        
        # 获取朝向（四元数转欧拉角）
        q = msg.pose.pose.orientation
        self.robot_theta = math.atan2(
            2.0 * (q.w * q.z + q.x * q.y),
            1.0 - 2.0 * (q.y * q.y + q.z * q.z)
        )
        
        # 获取速度
        self.robot_vel[0] = msg.twist.twist.linear.x
        self.robot_vel[1] = msg.twist.twist.angular.z
    
    def reset(self, seed=None, options=None):
        """重置环境"""
        super().reset(seed=seed)
        self.current_step = 0
        
        # 随机放置机器人起点（地图左侧）
        self.robot_pos = self.np_random.uniform(
            low=[-3.0, -3.0], high=[-2.0, -2.0]
        )
        self.robot_theta = self.np_random.uniform(-math.pi, math.pi)
        self.robot_vel = np.zeros(2)
        
        # 随机放置目标点（地图右侧）
        self.goal_pos = self.np_random.uniform(
            low=[2.0, 2.0], high=[3.0, 3.0]
        )
        
        # 生成随机障碍物（仅仿真模式）
        if not self.use_ros2:
            self._generate_obstacles(num=6)
        
        return self._get_observation(), {}
    
    def _generate_obstacles(self, num):
        """生成随机圆形障碍物"""
        self.obstacles = []
        for _ in range(num):
            pos = self.np_random.uniform(low=[-2.5, -2.5], high=[2.5, 2.5])
            # 确保障碍物不在起点终点
            if (np.linalg.norm(pos - self.robot_pos) > 1.0 and
                np.linalg.norm(pos - self.goal_pos) > 1.0):
                self.obstacles.append({"pos": pos, "radius": 0.3})
    
    def _get_observation(self):
        """拼接观测向量"""
        # 目标相对位置（变换到机器人坐标系）
        dx = self.goal_pos[0] - self.robot_pos[0]
        dy = self.goal_pos[1] - self.robot_pos[1]
        goal_dist = np.linalg.norm([dx, dy])
        goal_angle = math.atan2(dy, dx) - self.robot_theta
        
        # 归一化激光雷达数据
        lidar = self.lidar_data / 10.0
        
        return np.concatenate([
            lidar,                                          # 360维激光雷达
            [dx / 5.0, dy / 5.0, goal_angle / math.pi, goal_dist / 5.0]  # 4维目标信息
        ], dtype=np.float32)
    
    def step(self, action):
        """执行动作"""
        self.current_step += 1
        
        v, omega = np.clip(action, self.action_space.low, self.action_space.high)
        
        if self.use_ros2:
            # ROS2 模式：发送速度命令
            self._send_velocity_ros2(v, omega)
        else:
            # 仿真模式：直接更新状态
            self._update_robot_state(v, omega)
        
        # 仿真模式：计算激光雷达
        if not self.use_ros2:
            self._simulate_lidar()
        
        # 获取观测
        obs = self._get_observation()
        
        # 计算奖励
        reward, terminated = self._compute_reward()
        
        # 检查截断
        truncated = self.current_step >= self.max_steps
        
        info = {"is_success": terminated}
        
        return obs, reward, terminated, truncated, info
    
    def _send_velocity_ros2(self, v, omega):
        """通过 ROS2 发送速度命令"""
        msg = Twist()
        msg.linear.x = v
        msg.angular.z = omega
        self.vel_pub.publish(msg)
    
    def _update_robot_state(self, v, omega):
        """仿真模式：更新机器人状态"""
        dt = 0.1  # 控制周期 10Hz
        
        # 差速驱动运动学模型
        self.robot_theta += omega * dt
        self.robot_pos[0] += v * math.cos(self.robot_theta) * dt
        self.robot_pos[1] += v * math.sin(self.robot_theta) * dt
        
        self.robot_vel = np.array([v, omega])
    
    def _simulate_lidar(self):
        """仿真模式：模拟激光雷达数据"""
        self.lidar_data = np.full(self.num_lidar, 10.0)
        
        for i in range(self.num_lidar):
            angle = self.robot_theta + (2 * math.pi * i / self.num_lidar)
            ray_dir = np.array([math.cos(angle), math.sin(angle)])
            
            # 检测障碍物
            for obs in self.obstacles:
                dist = np.linalg.norm(self.robot_pos - obs["pos"])
                if dist < obs["radius"] + 0.1:
                    hit = dist - obs["radius"]
                    if 0 < hit < self.lidar_data[i]:
                        self.lidar_data[i] = hit
            
            # 检测边界（5m x 5m 地图）
            for axis in [0, 1]:
                boundary = 4.5 - self.robot_pos[axis]
                if ray_dir[axis] > 0:
                    t = boundary / ray_dir[axis]
                    if 0 < t < self.lidar_data[i]:
                        other_axis = self.robot_pos[1 - axis] + t * ray_dir[1 - axis]
                        if abs(other_axis) < 5.0:
                            self.lidar_data[i] = t
            
            self.lidar_data[i] = max(0.01, self.lidar_data[i])
    
    def _compute_reward(self):
        """计算奖励"""
        dist_to_goal = np.linalg.norm(self.robot_pos - self.goal_pos)
        reward = 0.0
        terminated = False
        
        # 接近目标正奖励
        reward += -dist_to_goal * 2.0
        
        # 成功到达目标
        if dist_to_goal < 0.3:
            reward += 100.0
            terminated = True
        
        # 碰撞障碍物惩罚
        for obs in self.obstacles:
            if np.linalg.norm(self.robot_pos - obs["pos"]) < obs["radius"] + 0.05:
                reward -= 50.0
                terminated = True
                break
        
        # 边界惩罚
        if np.linalg.norm(self.robot_pos) > 4.5:
            reward -= 20.0
            terminated = True
        
        return reward, terminated
    
    def render(self):
        """渲染环境（matplotlib 可视化）"""
        if self.render_mode == "human":
            try:
                import matplotlib.pyplot as plt
                
                plt.clf()
                
                # 绘制激光雷达扫描
                angles = np.linspace(0, 2 * math.pi, self.num_lidar)
                xs = self.robot_pos[0] + self.lidar_data * np.cos(angles + self.robot_theta)
                ys = self.robot_pos[1] + self.lidar_data * np.sin(angles + self.robot_theta)
                plt.scatter(xs, ys, c=self.lidar_data, cmap='gray_r', s=1, alpha=0.3)
                
                # 绘制机器人
                plt.plot(self.robot_pos[0], self.robot_pos[1], 'b^', markersize=10)
                dir_x = self.robot_pos[0] + 0.2 * math.cos(self.robot_theta)
                dir_y = self.robot_pos[1] + 0.2 * math.sin(self.robot_theta)
                plt.plot([self.robot_pos[0], dir_x], [self.robot_pos[1], dir_y], 'b-', linewidth=2)
                
                # 绘制目标
                plt.plot(self.goal_pos[0], self.goal_pos[1], 'g*', markersize=15)
                
                # 绘制障碍物
                for obs in self.obstacles:
                    circle = plt.Circle(
                        obs["pos"], obs["radius"],
                        color='red', alpha=0.3
                    )
                    plt.gca().add_patch(circle)
                
                plt.xlim(-5, 5)
                plt.ylim(-5, 5)
                plt.gca().set_aspect('equal')
                plt.grid(True)
                plt.title(f"Step: {self.current_step}")
                plt.pause(0.01)
            except ImportError:
                pass
    
    def close(self):
        if self.use_ros2 and rclpy.ok():
            self.node.destroy_node()
            rclpy.shutdown()
```

"""
TurtleBot3 PPO 导航训练脚本
使用 Stable-Baselines3 的 PPO 算法训练移动机器人导航策略
"""
import os
import time
import numpy as np
from stable_baselines3 import PPO
from stable_baselines3.common.vec_env import DummyVecEnv, VecNormalize
from stable_baselines3.common.callbacks import EvalCallback, CheckpointCallback
from stable_baselines3.common.logger import configure

# 导入自定义环境
from turtlebot3_nav_env import TurtleBot3NavEnv


def make_nav_env(use_ros2=False, render=False):
    """
    创建导航环境工厂函数
    
    参数:
        use_ros2: 是否使用 ROS2 连接真实/Gazebo 仿真
        render: 是否渲染
    """
    env = TurtleBot3NavEnv(
        ns="turtlebot3",
        num_lidar=360,
        max_steps=500,
        use_ros2=use_ros2
    )
    if render:
        env.render_mode = "human"
    else:
        env.render_mode = None
    return env


def train_ppo_navigation(
    total_timesteps=2000000,
    use_ros2=False,
    save_path="./logs/turtlebot3_ppo/"
):
    """
    训练 PPO 导航策略
    
    参数:
        total_timesteps: 总训练步数
        use_ros2: 是否连接 ROS2 仿真/真实机器人
        save_path: 模型保存路径
    """
    os.makedirs(save_path, exist_ok=True)
    
    # 创建训练环境（仿真模式用于训练）
    env = DummyVecEnv([lambda: make_nav_env(use_ros2=False, render=False)])
    
    # 添加观测和奖励归一化（RL 训练的关键技巧）
    env = VecNormalize(env, norm_obs=True, norm_reward=True)
    
    # PPO 超参数配置（针对导航任务优化）
    model = PPO(
        "MlpPolicy",               # 多层感知机策略网络
        env,
        learning_rate=3e-4,        # 学习率
        n_steps=4096,              # 每次采集的步数（越大越稳定但越慢）
        batch_size=512,            # 批次大小（2的幂次）
        n_epochs=10,               # 每次更新的 epoch 数
        gamma=0.99,                # 折扣因子
        gae_lambda=0.95,           # GAE 参数（方差-偏差平衡）
        clip_range=0.2,            # PPO 裁剪边界
        ent_coef=0.01,            # 熵系数（鼓励探索）
        max_grad_norm=0.5,         # 梯度裁剪阈值
        verbose=1,
        tensorboard_log=save_path + "tb/",
        device="cuda"             # 使用 GPU 加速
    )
    
    # 评估回调：定期评估策略性能
    eval_env = DummyVecEnv([lambda: make_nav_env(use_ros2=use_ros2, render=False)])
    eval_env = VecNormalize(eval_env, norm_obs=True, norm_reward=True)
    
    eval_callback = EvalCallback(
        eval_env,
        best_model_save_path=save_path + "best_model/",
        log_path=save_path + "eval_results/",
        eval_freq=10000,           # 每10000步评估一次
        n_eval_episodes=50,        # 每次评估50个回合
        deterministic=True,        # 确定性策略（无随机探索）
        render=False
    )
    
    # 检查点回调：定期保存模型
    checkpoint_callback = CheckpointCallback(
        save_freq=50000,
        save_path=save_path + "checkpoints/",
        name_prefix="ppo_turtlebot3_nav"
    )
    
    print("=" * 60)
    print("开始训练 PPO TurtleBot3 导航策略")
    print(f"总训练步数: {total_timesteps}")
    print(f"保存路径: {save_path}")
    print("=" * 60)
    
    start_time = time.time()
    
    model.learn(
        total_timesteps=total_timesteps,
        callback=[eval_callback, checkpoint_callback],
        progress_bar=True,
        log_interval=10
    )
    
    elapsed = time.time() - start_time
    print(f"\n训练完成！耗时: {elapsed / 60:.1f} 分钟")
    
    # 保存最终模型
    model.save(save_path + "ppo_turtlebot3_nav_final")
    env.save(save_path + "vecnormalize.pkl")
    print(f"模型已保存至 {save_path}")
    
    return model


def evaluate_trained_model(model_path, env_path=None, num_episodes=20, use_ros2=False):
    """
    评估训练好的导航模型
    
    参数:
        model_path: 模型路径
        env_path: 环境归一化参数路径
        num_episodes: 评估回合数
        use_ros2: 是否使用 ROS2
    """
    from stable_baselines3.common.vec_env import VecNormalize
    
    # 加载模型
    model = PPO.load(model_path)
    
    # 创建评估环境
    env = DummyVecEnv([lambda: make_nav_env(use_ros2=use_ros2, render=True)])
    env = VecNormalize(env, norm_obs=True, norm_reward=False)
    
    if env_path and os.path.exists(env_path):
        env = VecNormalize.load(env_path, env)
    
    successes = []
    episode_rewards = []
    
    print("=" * 60)
    print("开始评估模型")
    print("=" * 60)
    
    for episode in range(num_episodes):
        obs, _ = env.reset()
        episode_reward = 0
        done = False
        
        while not done:
            action, _ = model.predict(obs, deterministic=True)
            obs, reward, terminated, truncated, info = env.step(action)
            episode_reward += reward
            done = terminated or truncated
            
            # 渲染
            env.render()
        
        successes.append(info.get("is_success", False))
        episode_rewards.append(episode_reward)
        print(f"Episode {episode+1:3d}/{num_episodes}: "
              f"Reward={episode_reward:8.2f}, Success={info.get('is_success', False)}")
    
    success_rate = np.mean(successes)
    avg_reward = np.mean(episode_rewards)
    
    print("=" * 60)
    print(f"评估结果 ({num_episodes} 个回合)")
    print(f"  平均奖励: {avg_reward:.2f}")
    print(f"  成功率:   {success_rate*100:.1f}%")
    print("=" * 60)
    
    return success_rate, avg_reward


if __name__ == "__main__":
    import argparse
    
    parser = argparse.ArgumentParser(description="TurtleBot3 PPO 导航训练")
    parser.add_argument("--mode", type=str, default="train", choices=["train", "eval"])

### 4.4 PPO 训练调参指南

| 超参数 | 默认值 | 调整建议 | 影响 |
|--------|--------|----------|------|
| `learning_rate` | 3e-4 | 训练不稳定时降低到 1e-4 | 收敛速度、稳定性 |
| `n_steps` | 2048 | 复杂任务可增大到 4096-8192 | 样本多样性、内存占用 |
| `batch_size` | 64 | 使用 2 的幂次，如 128/256/512 | 梯度估计稳定性 |
| `n_epochs` | 10 | 简单任务可降低到 4-6 | 更新效率、过拟合风险 |
| `gamma` | 0.99 | 长期任务可提高到 0.999 | 长期规划能力 |
| `clip_range` | 0.2 | 训练初期可用 0.3，后期降到 0.1 | 探索-利用平衡 |
| `ent_coef` | 0.01 | 稀疏奖励任务可增大到 0.05 | 探索程度 |

### 5.2 UR5 机械臂抓取环境

```python
"""
UR5 机械臂抓取强化学习环境
使用 SAC 算法训练 UR5 完成物体抓取任务
"""
import gymnasium as gym
import numpy as np
from gymnasium import spaces
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import JointState
from geometry_msgs.msg import Pose
from trajectory_msgs.msg import JointTrajectory, JointTrajectoryPoint
from std_msgs.msg import Float64MultiArray
import tf_transformations


class UR5GraspEnv(gym.Env):
    """
    UR5 机械臂抓取环境
    
    任务：控制 UR5 机械臂抓取桌面上的物体并放置到目标位置
    
    动作空间: 6维关节位置控制
    观测空间: 关节角度(6) + 关节速度(6) + 末端位姿(6) + 目标位置(3) + 物体位置(3) + 夹爪状态(1)
    """
    
    def __init__(self, ns="ur5", episode_max_steps=200, use_ros2=False):
        super().__init__()
        
        self.ns = ns
        self.episode_max_steps = episode_max_steps
        self.current_step = 0
        self.use_ros2 = use_ros2
        
        # 动作空间：6 个关节的目标位置（归一化到 [-1, 1]）
        self.action_space = spaces.Box(
            low=-1.0, high=1.0, shape=(6,), dtype=np.float32
        )
        
        # 观测空间: 25维
        self.observation_space = spaces.Box(
            low=-np.inf, high=np.inf,
            shape=(25,),
            dtype=np.float32
        )
        
        # UR5 关节名称
        self.joint_names = [
            'shoulder_pan_joint', 'shoulder_lift_joint', 'elbow_joint',
            'wrist_1_joint', 'wrist_2_joint', 'wrist_3_joint'
        ]
        
        # 关节限位（UR5）
        self.joint_limits_lower = np.array([-3.14, -3.14, -3.14, -3.14, -3.14, -3.14])
        self.joint_limits_upper = np.array([3.14, 3.14, 3.14, 3.14, 3.14, 3.14])
        
        # 关节状态
        self.joint_positions = np.zeros(6)
        self.joint_velocities = np.zeros(6)
        self.ee_pose = np.zeros(6)  # x, y, z, roll, pitch, yaw
        
        # 任务目标
        self.object_pos = np.zeros(3)
        self.target_pos = np.zeros(3)
        self.gripper_state = 0.0  # 0=张开, 1=闭合
        
        # 仿真物体位置
        self.sim_object_pos = np.zeros(3)
        
        # ROS2 通信
        if use_ros2:
            rclpy.init(args=None)
            self.node = Node(f'ur5_grasp_{ns}')
            self._init_ros2()
        
        self.render_mode = "human"
    
    def _init_ros2(self):
        """初始化 ROS2 通信"""
        # 关节状态订阅
        self.joint_sub = self.node.create_subscription(
            JointState,
            f'/{self.ns}/joint_states',
            self._joint_callback,
            10
        )
        
        # 关节命令发布
        self.joint_pub = self.node.create_publisher(
            JointTrajectory,
            f'/{self.ns}/joint_trajectory',
            10
        )
        
        # 夹爪命令发布
        self.gripper_pub = self.node.create_publisher(
            Float64MultiArray,
            f'/{self.ns}/gripper_command',
            10
        )
        
        self.node.get_logger().info("UR5 ROS2 环境初始化完成")
    
    def _joint_callback(self, msg: JointState):
        """关节状态回调"""
        for i, name in enumerate(msg.name):
            if name in self.joint_names:
                idx = self.joint_names.index(name)
                if i < len(msg.position):
                    self.joint_positions[idx] = msg.position[i]
                if i < len(msg.velocity):
                    self.joint_velocities[idx] = msg.velocity[i]
    
    def reset(self, seed=None, options=None):
        """重置环境"""
        super().reset(seed=seed)
        self.current_step = 0
        
        # 重置关节到初始位置
        self.joint_positions = np.array([0.0, -1.57, 1.57, 0.0, 1.57, 0.0])
        self.joint_velocities = np.zeros(6)
        
        # 随机放置物体（在机械臂可达范围内）
        self.object_pos = self.np_random.uniform(
            low=[0.3, -0.2, 0.01],
            high=[0.5, 0.2, 0.01]
        )
        
        # 随机放置目标位置
        self.target_pos = self.np_random.uniform(
            low=[0.3, -0.3, 0.01],
            high=[0.5, 0.3, 0.01]
        )
        
        # 重置夹爪（张开）
        self.gripper_state = 0.0
        
        # 发送初始位置命令
        if self.use_ros2:
            self._send_joint_command(self.joint_positions)
            self._send_gripper_command(0.0)
        
        return self._get_observation(), {}
    
    def _get_observation(self):
        """获取观测向量"""
        # 简化：使用关节位置代替末端位姿（完整实现需 FK）
        ee_pos = self.joint_positions[:3] * 0.2  # 简化正运动学
        
        return np.concatenate([
            self.joint_positions / np.pi,         # 6维（归一化关节位置）
            self.joint_velocities / np.pi,         # 6维（归一化关节速度）
            ee_pos / 1.0,                         # 3维（末端位置近似）
            self.object_pos / 1.0,                # 3维（物体位置）
            self.target_pos / 1.0,                # 3维（目标位置）
            [self.gripper_state]                   # 1维（夹爪状态）
        ], dtype=np.float32)
    
    def step(self, action):
        """执行动作"""
        self.current_step += 1
        
        # 将归一化动作映射到关节位置增量
        action = np.clip(action, -1.0, 1.0)
        target_joints = self.joint_positions + action * 0.1  # 每步最大增量 0.1 rad
        target_joints = np.clip(target_joints, self.joint_limits_lower, self.joint_limits_upper)
        
        # 发送关节命令
        if self.use_ros2:
            self._send_joint_command(target_joints)
        else:
            # 仿真模式：直接更新关节位置
            self.joint_positions = target_joints
        
        # 获取观测
        obs = self._get_observation()
        
        # 计算奖励
        reward, terminated = self._compute_reward(obs)
        
        # 检查截断
        truncated = self.current_step >= self.episode_max_steps
        
        info = {"is_success": terminated}
        
        return obs, reward, terminated, truncated, info
    
    def _send_joint_command(self, positions):
        """发送关节位置命令"""
        msg = JointTrajectory()
        msg.joint_names = self.joint_names
        point = JointTrajectoryPoint()
        point.positions = positions.tolist()
        point.time_from_start = rclpy.duration.Duration(seconds=0.1).to_msg()
        msg.points = [point]
        self.joint_pub.publish(msg)
    
    def _send_gripper_command(self, position):
        """发送夹爪命令"""
        msg = Float64MultiArray()
        msg.data = [position]
        self.gripper_pub.publish(msg)
    
    def _compute_reward(self, obs):
        """
        计算抓取任务奖励
        
        奖励设计原则：
        1. 接近物体 -> 正奖励
        2. 抓取物体 -> 正奖励
        3. 举动物体 -> 正奖励
        4. 到达目标 -> 大量正奖励
        """
        reward = 0.0
        terminated = False
        
        # 解析观测
        ee_pos = obs[12:15]  # 末端位置
        object_pos = obs[15:18]  # 物体位置
        target_pos = obs[18:21]  # 目标位置
        gripper = obs[24]  # 夹爪状态
        
        # 计算距离
        dist_to_object = np.linalg.norm(ee_pos - object_pos)
        dist_to_target = np.linalg.norm(object_pos - target_pos)
        
        # 奖励项1：接近物体
        reward -= dist_to_object * 5.0
        
        # 奖励项2：抓取（夹爪接近物体且夹爪闭合）
        if dist_to_object < 0.05 and gripper > 0.5:
            reward += 20.0
        
        # 奖励项3：举起物体
        if object_pos[2] > 0.1:
            reward += 10.0
        
        # 奖励项4：到达目标
        if dist_to_target < 0.08:
            reward += 100.0
            terminated = True
        
        return reward, terminated
    
    def close(self):
        if self.use_ros2 and rclpy.ok():
            self.node.destroy_node()
            rclpy.shutdown()

"""
Sim2Real 域随机化完整实现
用于训练能在真实机器人上泛化的策略
"""
import numpy as np
from typing import Dict, List, Tuple


class DomainRandomizer:
    """
    域随机化器
    
    在每个 episode 开始时随机化仿真参数，
    使策略学会对参数变化鲁棒
    """
    
    def __init__(self, config: Dict):
        """
        参数:
            config: 随机化参数配置字典
        """
        self.config = config
        self.current_params = {}
        self.episode_count = 0
    
    def sample_params(self, rng=None) -> Dict:
        """
        采样一组随机化参数
        
        返回:
            params: 随机化后的参数字典
        """
        if rng is None:
            rng = np.random.default_rng()
        
        params = {}
        for name, cfg in self.config.items():
            base = cfg["base"]
            range_val = cfg["range"]
            
            if isinstance(base, (int, float)):
                # 均匀采样
                sampled = base + rng.uniform(-range_val, range_val)
                params[name] = sampled
            elif isinstance(base, list):
                # 向量参数独立采样
                params[name] = [
                    b + rng.uniform(-range_val, range_val) for b in base
                ]
            elif isinstance(base, dict):
                # 高斯采样（可选）
                mean = base.get("mean", 0)
                std = base.get("std", range_val)
                params[name] = rng.normal(mean, std)
        
        self.current_params = params
        return params
    
    def apply_to_sim(self, sim) -> None:
        """
        将随机化参数应用到仿真器
        
        参数:
            sim: 仿真器对象（需要实现对应的 set_xxx 方法）
        """
        if not self.current_params:
            return
        
        # 物理参数应用
        if "robot_mass" in self.current_params:
            sim.set_robot_mass(self.current_params["robot_mass"])
        
        if "joint_friction" in self.current_params:
            sim.set_joint_friction(self.current_params["joint_friction"])
        
        if "motor_strength" in self.current_params:
            sim.set_motor_strength(self.current_params["motor_strength"])
        
        if "gravity" in self.current_params:
            sim.set_gravity(self.current_params["gravity"])
        
        # 传感器噪声应用
        if "lidar_noise" in self.current_params:
            sim.set_lidar_noise(self.current_params["lidar_noise"])
        
        if "joint_position_noise" in self.current_params:
            sim.set_joint_position_noise(self.current_params["joint_position_noise"])
        
        if "imu_noise" in self.current_params:
            sim.set_imu_noise(self.current_params["imu_noise"])
        
        # 视觉参数应用
        if "camera_exposure" in self.current_params:
            sim.set_camera_exposure(self.current_params["camera_exposure"])
        
        if "light_intensity" in self.current_params:
            sim.set_light_intensity(self.current_params["light_intensity"])


class AdaptiveDomainRandomizer(DomainRandomizer):
    """
    自适应域随机化 (ADR)
    
    根据训练进度自动调整随机化范围：
    - 策略表现好 -> 增大随机化范围（提高难度）
    - 策略表现差 -> 减小随机化范围（降低难度）
    """
    
    def __init__(
        self,
        config: Dict,
        initial_range_scale: float = 0.5,
        adaptation_rate: float = 1.02,
        success_threshold: float = 0.8,
        min_threshold: float = 0.5
    ):
        """
        参数:
            config: 随机化参数配置
            initial_range_scale: 初始随机化范围（相对于配置范围的缩放比例）
            adaptation_rate: 适应速率（每次调整的倍数）
            success_threshold: 成功率上界（超过此值增大范围）
            min_threshold: 成功率下界（低于此值减小范围）
        """
        super().__init__(config)
        self.range_scale = initial_range_scale
        self.adaptation_rate = adaptation_rate
        self.success_threshold = success_threshold
        self.min_threshold = min_threshold
        self.success_history = []
    
    def update(self, success: bool):
        """
        根据单个 episode 的成功与否更新随机化范围
        
        参数:
            success: 当前 episode 是否成功
        """
        self.success_history.append(success)
        
        # 使用滑动窗口计算成功率
        window_size = 20
        if len(self.success_history) > window_size:
            self.success_history = self.success_history[-window_size:]
        
        success_rate = np.mean(self.success_history)
        
        if success_rate > self.success_threshold:
            # 成功率太高，增加随机化范围
            self.range_scale *= self.adaptation_rate
            self._clamp_range_scale()
            print(f"[ADR] 成功率 {success_rate:.2%} > {self.success_threshold:.2%}, "
                  f"增大随机化范围: x{self.range_scale:.3f}")
        
        elif success_rate < self.min_threshold:
            # 成功率太低，减小随机化范围
            self.range_scale /= self.adaptation_rate
            self._clamp_range_scale()
            print(f"[ADR] 成功率 {success_rate:.2%} < {self.min_threshold:.2%}, "
                  f"减小随机化范围: x{self.range_scale:.3f}")
    
    def _clamp_range_scale(self):
        """限制随机化范围在 [0.1, 2.0] 之间"""
        self.range_scale = np.clip(self.range_scale, 0.1, 2.0)
    
    def sample_params(self, rng=None) -> Dict:
        """使用缩放后的范围采样参数"""
        if rng is None:
            rng = np.random.default_rng()
        
        params = {}
        for name, cfg in self.config.items():
            base = cfg["base"]
            range_val = cfg["range"] * self.range_scale  # 应用缩放
            
            if isinstance(base, (int, float)):
                sampled = base + rng.uniform(-range_val, range_val)
                params[name] = sampled
            elif isinstance(base, list):
                params[name] = [
                    b + rng.uniform(-range_val, range_val) for b in base
                ]
        
        self.current_params = params
        return params

class ROS2RobotEnv(gym.Env):
    """关键接口"""
    
    def reset(self):
        """
        数据流：
        1. 发送 ROS2 Service 调用（如 /reset 话题）请求 Gazebo 重置
        2. 订阅 /odom、/scan 等话题获取初始状态
        3. 返回 (observation, info)
        """
        self._send_reset_service()  # ROS2 Service call
        self._spin_untilUpdated()    # 等待状态更新
        obs = self._get_observation()
        return obs, {}
    
    def step(self, action):
        """
        数据流：
        1. 将 RL 动作 (v, omega) 转换为 Twist 消息
        2. 发布到 /cmd_vel 话题
        3. Gazebo/机器人执行动作
        4. 订阅 /scan、/odom 获取新状态
        5. 计算奖励并返回
        """
        # 动作执行
        twist = Twist()
        twist.linear.x = action[0]
        twist.angular.z = action[1]
        self.vel_pub.publish(twist)  # ROS2 Publisher
        
        # 状态获取（异步回调）
        self._spin_once()  # rclpy.spin_once()
        obs = self._get_observation()
        reward, terminated = self._compute_reward(obs)
        
        return obs, reward, terminated, truncated, info

def grasp_reward_function(obs, action=None):
    """
    UR5 抓取任务奖励函数
    
    状态 (obs):
    - ee_pos: 末端执行器位置 [x, y, z]
    - object_pos: 物体位置 [x, y, z]
    - target_pos: 目标位置 [x, y, z]
    - gripper_state: 夹爪状态 [0=张开, 1=闭合]
    """
    ee_pos = obs[0:3]
    object_pos = obs[3:6]
    target_pos = obs[6:9]
    gripper = obs[9]  # 0~1
    
    reward = 0.0
    
    # ---------- 奖励项 1：接近物体 ----------
    # 作用：引导末端执行器向物体移动
    dist_to_object = np.linalg.norm(ee_pos - object_pos)
    reward -= dist_to_object * 8.0  # 越近奖励越高
    
    # ---------- 奖励项 2：接近后张开夹爪 ----------
    # 作用：确保在接近物块时夹爪处于正确状态
    if dist_to_object < 0.15:
        if gripper < 0.3:
            reward += 5.0  # 接近物体且夹爪张开
        elif gripper > 0.7:
            reward -= 3.0  # 接近物体但夹爪闭合（错误状态）
    
    # ---------- 奖励项 3：抓取动作 ----------
    # 作用：奖励成功的抓取动作（夹爪合拢且包裹物体）
    if dist_to_object < 0.05 and gripper > 0.7:
        reward += 25.0  # 抓取成功
    
    # ---------- 奖励项 4：举起物体 ----------
    # 作用：确保物体被成功夹起
    if gripper > 0.7 and object_pos[2] > 0.15:
        reward += 15.0  # 物体被举起
    
    # ---------- 奖励项 5：移动物体到目标 ----------
    # 作用：引导物体向目标位置移动
    dist_to_target = np.linalg.norm(object_pos - target_pos)
    reward -= dist_to_target * 10.0
    
    # ---------- 奖励项 6：成功放置 ----------
    # 作用：最终任务目标奖励
    if dist_to_target < 0.08:
        reward += 100.0  # 任务成功
    
    return reward
```

**稀疏奖励 vs 密集奖励的权衡**：

| 类型 | 优点 | 缺点 |
|------|------|------|
| **稀疏奖励** | 易于设计、避免 reward hacking | 学习困难、收敛极慢 |
| **密集奖励** | 学习信号丰富、收敛快 | 可能与真正目标不一致、reward hacking 风险 |

建议采用**稀疏 + 密集混合**设计：核心成功条件用稀疏奖励（大奖励），中间过程用密集奖励（塑形）。

### 练习3：域随机化参数选择

**1. 各参数的随机化范围设计**：

```python
DR_CONFIG_TURTLEBOT3 = {
    # 质量偏差约 ±20%，取 ±1.5kg（覆盖 2.7-4.5 kg 真实范围）
    "robot_mass": {"base": 3.5, "range": 1.5},
    
    # 摩擦系数偏差约 ±30%，取 ±0.4（覆盖 0.4-1.2 范围）
    "wheel_friction": {"base": 0.8, "range": 0.4},
    
    # 激光雷达噪声覆盖测量值（0.08m RMS），留有余量
    "lidar_noise": {"base": 0.0, "range": 0.12},
    
    # 里程计漂移，覆盖 0.05m/10m 误差，留 2x 余量
    "odom_drift": {"base": [0.0, 0.0, 0.0], "range": 0.1},
}
```

**2. 为什么某些参数需要较大随机化范围**：

| 参数 | 随机化范围建议 | 原因 |
|------|---------------|------|
| 激光雷达噪声 | 较大 (±0.12m) | 环境依赖强（阳光、地板反射、灰尘） |
| 里程计漂移 | 较大 | 轮子磨损、地面材质变化影响大 |
| 质量 | 中等 (±1.5kg) | 电池电量变化导致质量变化 |
| 摩擦系数 | 中等 | 地面材质变化、轮子磨损 |

**核心原则**：随机化范围应**覆盖真实环境的变化范围 × 1.5-2x 安全系数**。

**3. 真实环境失效的最可能原因**：

| 可能原因 | 检查方法 | 解决方案 |
|----------|----------|----------|
| 激光雷达噪声差异过大 | 对比训练/真实噪声分布 | 增大 lidar_noise 范围 |
| 里程计累积漂移 | 测试长时间运行轨迹误差 | 加入里程计漂移补偿 |
| 控制延迟 | 测量真实响应时间 | 在仿真中加入 action_delay |
| 碰撞模型不一致 | 对比仿真/真实碰撞行为 | 减小障碍物碰撞半径 |
| 物理参数未覆盖 | 逐一对比测量值与训练范围 | 调整 DR 范围覆盖测量值 |
