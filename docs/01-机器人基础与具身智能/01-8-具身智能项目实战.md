## 1. 项目实战概述

### 1.1 具身智能完整项目流程

做一个具身智能项目，和做普通的软件开发有本质区别——因为你面对的不只是代码，还有真实的物理世界。一个完整的企业级具身智能项目，通常会经历以下六个阶段：

| 阶段 | 名称 | 核心任务 | 产出物 |
|------|------|----------|--------|
| 1 | **需求分析** | 明确机器人要"做什么"，在"什么环境"做 | 需求文档、技术指标 |
| 2 | **系统设计** | 决定"怎么做"，划分模块，选择技术栈 | 系统架构图、技术方案 |
| 3 | **算法开发** | 实现感知、决策、控制等核心算法 | 模型文件、代码模块 |
| 4 | **仿真验证** | 在虚拟环境中测试功能和性能 | 仿真报告、Bug清单 |
| 5 | **硬件集成** | 把软件部署到真实机器人上 | 部署文档、标定参数 |
| 6 | **迭代优化** | 在真实环境中发现问题并改进 | 优化报告、用户手册 |

**为什么仿真验证必不可少？**

很多初学者恨不得一步到位，直接在真实机器人上编程。但这样做风险极高：

- 机器人摔坏了怎么办？
- 算法有Bug，机器人横冲直撞，伤到人怎么办？
- 调试一次要充电两小时，效率极低

仿真环境（如Gazebo、MuJoCo、Isaac Sim）可以在虚拟世界中反复测试，等算法稳定后再上真机。这是具身智能项目的基本规范。

### 1.2 从需求分析到部署的完整路径

让我们用一个具体案例来理解这个流程：**开发一个能在办公室环境中自主导航送快递的移动机器人**。

**阶段1：需求分析**

你首先要想清楚：这个机器人要做什么？

```
需求文档摘要：
- 机器人需要在办公室环境中自主移动
- 能够识别目标人物的脸或工牌
- 能够避开障碍物（人、椅子、垃圾桶等）
- 能够乘坐电梯（多楼层场景）
- 一次充电续航至少4小时
- 支持远程监控和紧急停止
```

**阶段2：系统设计**

根据需求，你开始设计系统架构：

```
系统架构：
├── 感知层：激光雷达 + 深度相机 + IMU
├── 定位层：SLAM建图 + AMCL定位
├── 规划层：全局路径规划 + 局部路径规划
├── 控制层：底盘运动控制
├── 交互层：语音交互 + APP控制
└── 安全层：紧急停止 + 虚拟围栏
```

**阶段3：算法开发**

每个模块都需要具体实现：

- SLAM建图：使用Cartographer或GMapping
- 目标识别：使用YOLO检测工牌
- 路径规划：使用A*或Dijkstra全局规划 + DWA局部规划
- 人脸识别：使用FaceNet或ArcFace

**阶段4：仿真验证**

在Gazebo中搭建办公室仿真环境，逐一测试每个模块。

**阶段5：硬件集成**

把算法部署到真实机器人上，进行标定（相机内参、激光雷达外参、轮式里程计标定）。

**阶段6：迭代优化**

真实环境中发现的问题：
- 光照变化导致视觉识别率下降
- 电梯门关闭太快，机器人进不去
- 人多时路径规划频繁重规划，效率低

针对这些问题进行优化迭代。

#### 2.1.3 SLAM建图

**SLAM（Simultaneous Localization and Mapping）** 是移动机器人的核心技术——机器人在移动过程中同时估计自己的位置并构建环境地图。

下面是一个使用Cartographer实现2D SLAM的示例代码：

```python
#!/usr/bin/env python3
"""
SLAM建图节点 - 使用Cartographer进行2D地图构建
功能：启动Cartographer SLAM，实时构建环境地图
"""

import rclpy
from rclpy.node import Node
from nav_msgs.msg import OccupancyGrid  # 地图消息类型
from geometry_msgs.msg import PoseStamped  # 位姿消息类型
import numpy as np
import time


class SLAMMapBuilder(Node):
    """SLAM地图构建器节点
    
    职责：
    1. 订阅激光雷达扫描数据
    2. 订阅机器人位姿数据
    3. 实时发布栅格地图
    """
    
    def __init__(self):
        super().__init__('slam_map_builder')
        
        # ===== 参数定义 =====
        self.map_resolution = 0.05  # 地图分辨率：5cm/像素
        self.map_size_x = 200       # 地图宽度：200像素 = 10米
        self.map_size_y = 200       # 地图高度：200像素 = 10米
        self.map_origin_x = -5.0    # 地图原点X坐标（世界坐标系）
        self.map_origin_y = -5.0    # 地图原点Y坐标（世界坐标系）
        
        # 初始化地图（-1表示未知，0表示空闲，100表示占用）
        self.map_data = np.full(
            (self.map_size_x, self.map_size_y), 
            -1, 
            dtype=np.int8
        )
        
        # 机器人初始估计位姿
        self.robot_pose = {
            'x': 0.0, 
            'y': 0.0, 
            'theta': 0.0
        }
        
        # 激光雷达参数
        self.laser_range_min = 0.1   # 最小测量距离（米）
        self.laser_range_max = 12.0  # 最大测量距离（米）
        self.laser_angle_min = -2.356  # 激光扫描起始角度（弧度）
        self.laser_angle_max = 2.356    # 激光扫描结束角度（弧度）
        
        # ===== 订阅者（监听传感器数据）=====
        # 订阅激光雷达扫描数据
        self.scan_sub = self.create_subscription(
            LaserScan,      # 消息类型
            '/scan',        # 话题名
            self.laser_callback,  # 回调函数
            qos_profile=10
        )
        
        # 订阅机器人位姿（来自里程计或SLAM本身）
        self.pose_sub = self.create_subscription(
            PoseStamped,
            '/robot_pose',
            self.pose_callback,
            qos_profile=10
        )
        
        # ===== 发布者（发布地图数据）=====
        # 发布栅格地图
        self.map_pub = self.create_publisher(
            OccupancyGrid,
            '/map',
            qos_profile=10
        )
        
        # 定时器：定期发布地图（1Hz）
        self.timer = self.create_timer(1.0, self.publish_map)
        
        self.get_logger().info('SLAM地图构建器已启动')
    
    def pose_callback(self, msg: PoseStamped):
        """位姿回调函数 - 更新机器人当前位置
        
        Args:
            msg: 机器人位姿消息，包含位置(x,y,z)和朝向
        """
        # 从消息中提取位置信息（2D导航只用x,y和yaw角）
        self.robot_pose['x'] = msg.pose.position.x
        self.robot_pose['y'] = msg.pose.position.y
        
        # 从四元数提取yaw角（偏航角）
        q = msg.pose.orientation
        self.robot_pose['theta'] = self.quaternion_to_yaw(q.w, q.z)
    
    def laser_callback(self, msg: LaserScan):
        """激光雷达回调函数 - 处理激光扫描数据并更新地图
        
        这是SLAM的核心：当激光雷达返回一帧数据时，
        根据机器人当前位置，将激光点击转换为地图上的占用格子
        
        Args:
            msg: 激光扫描消息，包含ranges（距离数组）和角度信息
        """
        ranges = np.array(msg.ranges)  # 距离数组
        angle_increment = msg.angle_increment  # 角度分辨率
        
        # 遍历每一条激光射线
        for i, r in enumerate(ranges):
            # 跳过无效数据（太近或太远）
            if r < self.laser_range_min or r > self.laser_range_max:
                continue
            
            # 计算这条激光射线的角度（世界坐标系）
            angle = msg.angle_min + i * angle_increment
            world_angle = angle + self.robot_pose['theta']
            
            # 计算激光击中点的世界坐标
            hit_x = self.robot_pose['x'] + r * np.cos(world_angle)
            hit_y = self.robot_pose['y'] + r * np.sin(world_angle)
            
            # 将世界坐标转换为地图坐标（像素）
            map_x = int((hit_x - self.map_origin_x) / self.map_resolution)
            map_y = int((hit_y - self.map_origin_y) / self.map_resolution)
            
            # 检查是否在地图范围内
            if 0 <= map_x < self.map_size_x and 0 <= map_y < self.map_size_y:
                # 将该格子标记为占用（100）
                self.map_data[map_x, map_y] = 100
                
                # 沿射线标记空闲格子（从机器人到击中点之间的格子设为0）
                self.mark_line_free(map_x, map_y)
    
    def mark_line_free(self, end_x: int, end_y: int):
        """Bresenham算法 - 标记两点连线上的所有格子为空闲
        
        原理：激光从机器人发出到击中障碍物，路径上的格子应该是空闲的
        使用Bresenham直线算法高效地标记这条路径
        
        Args:
            end_x: 终点像素X坐标（激光击中点）
            end_y: 终点像素Y坐标
        """
        # 将机器人位置转换为地图坐标
        start_x = int((self.robot_pose['x'] - self.map_origin_x) / self.map_resolution)
        start_y = int((self.robot_pose['y'] - self.map_origin_y) / self.map_resolution)
        
        # Bresenham直线算法
        dx = abs(end_x - start_x)
        dy = abs(end_y - start_y)
        sx = 1 if start_x < end_x else -1
        sy = 1 if start_y < end_y else -1
        err = dx - dy
        
        x, y = start_x, start_y
        
        while True:
            if (x, y) == (start_x, start_y) or (x, y) == (end_x, end_y):
                pass  # 跳过起点和终点
            else:
                if self.map_data[x, y] == -1:  # 只更新未知格子
                    self.map_data[x, y] = 0    # 标记为空闲
            
            if x == end_x and y == end_y:
                break
            
            e2 = 2 * err
            if e2 > -dy:
                err -= dy
                x += sx
            if e2 < dx:
                err += dx
                y += sy
    
    def publish_map(self):
        """定时发布地图 - 将内存中的地图数据包装成ROS消息发布"""
        msg = OccupancyGrid()
        msg.header.stamp = self.get_clock().now().to_msg()
        msg.header.frame_id = 'map'
        
        # 设置地图元数据
        msg.info.resolution = self.map_resolution
        msg.info.width = self.map_size_x
        msg.info.height = self.map_size_y
        msg.info.origin.position.x = self.map_origin_x
        msg.info.origin.position.y = self.map_origin_y
        
        # 转换地图数据格式（numpy数组转一维列表）
        msg.data = self.map_data.flatten().tolist()
        
        self.map_pub.publish(msg)
        self.get_logger().debug(f'地图已发布：尺寸 {self.map_size_x}x{self.map_size_y}')
    
    @staticmethod
    def quaternion_to_yaw(w: float, z: float) -> float:
        """从四元数提取yaw角（偏航角）
        
        Args:
            w: 四元数的w分量
            z: 四元数的z分量
        
        Returns:
            float: yaw角（弧度）
        """
        # 对于只有yaw旋转的情况：(w, z) = (cos(θ/2), sin(θ/2))
        return 2 * np.arctan2(z, w)


def main(args=None):
    """主函数 - 启动SLAM节点"""
    rclpy.init(args=args)
    node = SLAMMapBuilder()
    
    try:
        rclpy.spin(node)  # 保持节点运行，持续处理回调
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

**代码说明**：

这个SLAM实现虽然简化了，但包含了核心思想：
1. **订阅激光雷达数据**：获取环境障碍物信息
2. **订阅机器人位姿**：知道"我在哪里"
3. **坐标变换**：将激光雷达的极坐标（距离+角度）转换为世界坐标系中的笛卡尔坐标
4. **栅格地图更新**：将障碍物位置标记为占用，沿射线标记空闲区域
5. **发布地图**：定时将地图数据发布出去，供其他节点使用

#### 2.1.4 路径规划与导航

路径规划分为全局规划和局部规划两层：

```python
#!/usr/bin/env python3
"""
导航节点 - 实现全局路径规划和局部路径规划
功能：接收目标点，输出安全的机器人运动控制指令
"""

import rclpy
from rclpy.node import Node
from geometry_msgs.msg import PoseStamped, Twist  # 位姿和速度消息
from nav_msgs.msg import Path, Odometry           # 路径和里程计消息
import numpy as np


class NavigationController(Node):
    """导航控制器 - 融合全局规划和局部规划的完整导航系统
    
    核心思想：
    - 全局规划器：在大尺度上找到从起点到终点的最优路径（类似导航软件）
    - 局部规划器：在小尺度上实时避障，生成安全的运动指令（类似驾驶员的实时反应）
    """
    
    def __init__(self):
        super().__init__('navigation_controller')
        
        # ===== 地图和规划器参数 =====
        self.map_resolution = 0.05  # 地图分辨率（米/像素）
        self.map_size = (200, 200)   # 地图尺寸（像素）
        
        # 机器人当前位置
        self.robot_x = 0.0
        self.robot_y = 0.0
        self.robot_theta = 0.0
        
        # 当前目标点
        self.goal_x = 0.0
        self.goal_y = 0.0
        self.goal_received = False
        
        # 路径跟踪参数
        self.max_linear_speed = 0.5    # 最大线速度（米/秒）
        self.max_angular_speed = 1.0   # 最大角速度（弧度/秒）
        self.lookahead_distance = 0.5 # 前看距离：距离目标点多远时开始减速
        
        # ===== 订阅者 =====
        # 订阅里程计（获取机器人当前位置）
        self.odom_sub = self.create_subscription(
            Odometry,
            '/odom',
            self.odom_callback,
            qos_profile=10
        )
        
        # 订阅目标点
        self.goal_sub = self.create_subscription(
            PoseStamped,
            '/goal_pose',
            self.goal_callback,
            qos_profile=10
        )
        
        # 订阅全局路径（来自全局规划器）
        self.global_path_sub = self.create_subscription(
            Path,
            '/global_path',
            self.global_path_callback,
            qos_profile=10
        )
        
        # ===== 发布者 =====
        # 发布速度控制指令
        self.cmd_vel_pub = self.create_publisher(
            Twist,
            '/cmd_vel',
            qos_profile=10
        )
        
        # 定时器：控制频率10Hz
        self.timer = self.create_timer(0.1, self.control_loop)
        
        # 全局路径
        self.global_path = []
        
        self.get_logger().info('导航控制器已启动')
    
    def odom_callback(self, msg: Odometry):
        """里程计回调 - 更新机器人当前位置"""
        self.robot_x = msg.pose.pose.position.x
        self.robot_y = msg.pose.pose.position.y
        
        # 从四元数提取yaw角
        q = msg.pose.pose.orientation
        self.robot_theta = 2 * np.arctan2(q.z, q.w)
    
    def goal_callback(self, msg: PoseStamped):
        """目标点回调 - 接收新的导航目标"""
        self.goal_x = msg.pose.position.x
        self.goal_y = msg.pose.position.y
        self.goal_received = True
        self.get_logger().info(f'收到新目标: ({self.goal_x:.2f}, {self.goal_y:.2f})')
    
    def global_path_callback(self, msg: Path):
        """全局路径回调 - 更新全局规划路径"""
        self.global_path = [(p.pose.position.x, p.pose.position.y) 
                           for p in msg.poses]
        self.get_logger().debug(f'全局路径更新，共{len(self.global_path)}个路径点')
    
    def control_loop(self):
        """控制主循环 - 定时器驱动，10Hz执行
        
        核心逻辑：
        1. 检查是否收到目标点
        2. 查找全局路径上最近的点
        3. 计算前看点
        4. 生成局部避障路径
        5. 跟踪路径，输出速度指令
        """
        if not self.goal_received or len(self.global_path) == 0:
            # 没有目标或路径时，停止机器人
            self.publish_cmd_vel(0.0, 0.0)
            return
        
        # ===== 步骤1：找到路径上离机器人最近的点 =====
        min_dist = float('inf')
        closest_idx = 0
        
        for i, (px, py) in enumerate(self.global_path):
            dist = np.sqrt((px - self.robot_x)**2 + (py - self.robot_y)**2)
            if dist < min_dist:
                min_dist = dist
                closest_idx = i
        
        # ===== 步骤2：找到前看点（lookahead point）=====
        # 前看点是路径上距离机器人一定距离的前方点
        lookahead_idx = closest_idx
        for i in range(closest_idx, len(self.global_path)):
            dist = np.sqrt(
                (self.global_path[i][0] - self.robot_x)**2 + 
                (self.global_path[i][1] - self.robot_y)**2
            )
            if dist >= self.lookahead_distance:
                lookahead_idx = i
                break
        
        lookahead_x, lookahead_y = self.global_path[lookahead_idx]
        
        # ===== 步骤3：计算速度指令 =====
        # 计算前看点的相对角度
        dx = lookahead_x - self.robot_x
        dy = lookahead_y - self.robot_y
        target_angle = np.arctan2(dy, dx)
        
        # 计算角度误差
        angle_error = target_angle - self.robot_theta
        
        # 归一化角度误差到[-π, π]
        while angle_error > np.pi:
            angle_error -= 2 * np.pi
        while angle_error < -np.pi:
            angle_error += 2 * np.pi
        
        # 计算目标距离
        goal_dist = np.sqrt(
            (self.goal_x - self.robot_x)**2 + 
            (self.goal_y - self.robot_y)**2
        )
        
        # ===== 步骤4：到达判断 =====
        if goal_dist < 0.2:  # 距离目标点小于20cm，认为到达
            self.get_logger().info('已到达目标点！')
            self.publish_cmd_vel(0.0, 0.0)
            self.goal_received = False
            return
        
        # ===== 步骤5：生成速度指令 =====
        # 角度控制：使用比例控制器
        angular_vel = angle_error * 1.5  # P控制器
        
        # 限制角速度范围
        angular_vel = np.clip(
            angular_vel, 
            -self.max_angular_speed, 
            self.max_angular_speed
        )
        
        # 速度控制：距离目标越近，速度越慢（安全减速）
        if goal_dist < self.lookahead_distance:
            linear_vel = goal_dist * 1.5  # 线性减速
        else:
            linear_vel = self.max_linear_speed
        
        # 限制线速度范围
        linear_vel = np.clip(linear_vel, 0.0, self.max_linear_speed)
        
        # 发布速度指令
        self.publish_cmd_vel(linear_vel, angular_vel)
    
    def publish_cmd_vel(self, linear: float, angular: float):
        """发布速度控制指令
        
        Args:
            linear: 线速度（米/秒）
            angular: 角速度（弧度/秒）
        """
        msg = Twist()
        msg.linear.x = linear
        msg.angular.z = angular
        self.cmd_vel_pub.publish(msg)


def main(args=None):
    rclpy.init(args=args)
    node = NavigationController()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

**代码说明**：

这个导航控制器实现了Pure Pursuit算法的核心思想：
1. **订阅全局路径**：接收来自全局规划器的路径
2. **Pure Pursuit算法**：在路径上找一个"前看点"，让机器人追着这个点运动
3. **P控制器**：将角度误差转化为角速度指令
4. **安全减速**：距离目标越近，速度越慢，避免撞击

#### 2.1.5 视觉感知集成

```python
#!/usr/bin/env python3
"""
视觉感知节点 - 集成目标检测和深度估计
功能：使用YOLO检测物体，结合深度相机获取物体三维位置
"""

import rclpy
from rclpy.node import Node
from vision_msgs.msg import Detection2DArray, BoundingBox3D  # 视觉消息类型
from sensor_msgs.msg import Image, CameraInfo  # 图像和相机信息
import numpy as np


class VisionPerception(Node):
    """视觉感知节点 - 融合目标检测和深度信息
    
    核心功能：
    1. 接收RGB图像，用YOLO进行目标检测
    2. 接收深度图像，获取检测物体的距离
    3. 将2D检测结果转换为3D空间位置
    """
    
    def __init__(self):
        super().__init__('vision_perception')
        
        # 深度图像（从深度相机接收）
        self.depth_image = None
        
        # 相机内参矩阵（从相机标定获得）
        self.fx = 615.0  # 焦距x
        self.fy = 615.0  # 焦距y
        self.cx = 320.0  # 主点x
        self.cy = 240.0  # 主点y
        
        # ===== 订阅者 =====
        # 订阅深度图像
        self.depth_sub = self.create_subscription(
            Image,
            '/camera/depth/image_raw',
            self.depth_callback,
            qos_profile=10
        )
        
        # 订阅2D检测结果（来自YOLO）
        self.detection_sub = self.create_subscription(
            Detection2DArray,
            '/yolo/detections',
            self.detection_callback,
            qos_profile=10
        )
        
        # ===== 发布者 =====
        # 发布带3D位置的检测结果
        self.detection3d_pub = self.create_publisher(
            BoundingBox3D,
            '/detections_3d',
            qos_profile=10
        )
        
        self.get_logger().info('视觉感知节点已启动')
    
    def depth_callback(self, msg: Image):
        """深度图像回调 - 更新深度图像缓存
        
        Args:
            msg: 深度图像消息（单位：米）
        """
        # 将ROS深度图像消息转换为numpy数组
        self.depth_image = np.frombuffer(msg.data, dtype=np.float32).reshape(
            msg.height, msg.width
        )
    
    def detection_callback(self, msg: Detection2DArray):
        """检测结果回调 - 将2D检测转换为3D位置
        
        核心逻辑：
        1. 遍历每个2D检测框
        2. 取检测框中心的像素坐标
        3. 从深度图像获取该像素的深度值
        4. 用相机内参将像素坐标转换为相机坐标系下的3D坐标
        
        Args:
            msg: YOLO输出的2D检测结果数组
        """
        if self.depth_image is None:
            return  # 没有深度图像时跳过
        
        for detection in msg.detections:
            # 获取2D边界框中心像素坐标
            bbox = detection.bbox
            center_x = int(bbox.center.position.x)
            center_y = int(bbox.center.position.y)
            
            # 获取检测类别和置信度
            class_id = detection.results[0].id
            score = detection.results[0].score
            
            # 跳过低置信度检测
            if score < 0.5:
                continue
            
            # ===== 2D到3D转换 =====
            # 从深度图像获取该像素点的深度值（米）
            depth = self.get_depth_at(center_x, center_y)
            
            if depth is None or depth <= 0:
                continue  # 无效深度值，跳过
            
            # 使用相机内参将像素坐标转换为相机坐标系下的3D坐标
            # 公式：X = (u - cx) * Z / fx, Y = (v - cy) * Z / fy
            x_3d = (center_x - self.cx) * depth / self.fx
            y_3d = (center_y - self.cy) * depth / self.fy
            z_3d = depth
            
            # 构建并发布3D检测结果
            self.publish_3d_detection(
                class_id=class_id,
                score=score,
                x=x_3d, y=y_3d, z=z_3d
            )
    
    def get_depth_at(self, u: int, v: int) -> float:
        """获取指定像素点的深度值
        
        Args:
            u: 像素X坐标（列）
            v: 像素Y坐标（行）
        
        Returns:
            float: 深度值（米），如果无效返回None
        """
        h, w = self.depth_image.shape[:2]
        
        # 检查坐标是否在图像范围内
        if 0 <= u < w and 0 <= v < h:
            depth = self.depth_image[v, u]
            
            # 检查深度值是否有效（非零、非NaN）
            if np.isfinite(depth) and depth > 0:
                return float(depth)
        
        return None
    
    def publish_3d_detection(self, class_id: str, score: float, 
                              x: float, y: float, z: float):
        """发布3D检测结果
        
        Args:
            class_id: 类别ID（如"person", "chair"等）
            score: 置信度分数
            x, y, z: 物体在相机坐标系下的3D位置（米）
        """
        msg = BoundingBox3D()
        
        # 设置3D位置
        msg.center.position.x = x
        msg.center.position.y = y
        msg.center.position.z = z
        msg.size.z = 0.1  # 假设物体厚度10cm
        
        self.detection3d_pub.publish(msg)
        
        self.get_logger().debug(
            f'3D检测: class={class_id}, pos=({x:.2f}, {y:.2f}, {z:.2f}), '
            f'score={score:.2f}'
        )


def main(args=None):
    rclpy.init(args=args)
    node = VisionPerception()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

#### 2.2.2 视觉识别系统

```python
#!/usr/bin/env python3
"""
抓取检测节点 - 目标检测 + 6D位姿估计 + 抓取点生成
功能：输入RGBD图像，输出抓取姿态
"""

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
import numpy as np
import cv2
import torch
from PIL import Image as PILImage


class GraspDetector(Node):
    """抓取检测器 - 整合检测、位姿估计、抓取点生成三个功能
    
    核心算法：GraspNet / Contact-GraspNet
    1. 输入RGBD图像
    2. 目标检测：找出"要抓什么"
    3. 6D位姿估计：确定物体"在哪里"以及"朝向如何"
    4. 抓取点生成：计算"怎么抓"（抓取位置+抓取角度）
    """
    
    def __init__(self):
        super().__init__('grasp_detector')
        
        # 相机内参（从标定获得）
        self.fx, self.fy = 615.0, 615.0
        self.cx, self.cy = 320.0, 240.0
        
        # 机械臂参数
        self.grasp_depth_min = 0.02   # 最小抓取深度（米）
        self.grasp_depth_max = 0.15   # 最大抓取深度（米）
        self.grasp_width_max = 0.08    # 最大抓取宽度（米）
        
        # ===== 订阅者 =====
        self.rgb_sub = self.create_subscription(
            Image, '/camera/color/image_raw',
            self.rgb_callback, 10
        )
        self.depth_sub = self.create_subscription(
            Image, '/camera/depth/image_raw',
            self.depth_callback, 10
        )
        
        # ===== 发布者 =====
        self.grasp_pub = self.create_publisher(
            GraspPoseArray, '/grasp_poses', 10
        )
        
        self.get_logger().info('抓取检测器已启动')
    
    def detect_grasp(self, rgb, depth):
        """执行抓取检测
        
        Args:
            rgb: RGB图像 (H, W, 3)
            depth: 深度图像 (H, W)
        
        Returns:
            list: 抓取姿态列表 [(x, y, z, roll, pitch, yaw, width), ...]
        """
        # ===== 步骤1：目标检测 =====
        # 使用YOLO检测物体边界框
        boxes = self.yolo_detect(rgb)  # 返回 [(class_id, x1, y1, x2, y2), ...]
        
        grasps = []
        for class_id, x1, y1, x2, y2 in boxes:
            # ===== 步骤2：计算物体中心点的深度 =====
            center_u = int((x1 + x2) / 2)
            center_v = int((y1 + y2) / 2)
            z = depth[center_v, center_u] / 1000.0  # 毫米转米
            
            if z <= 0 or z > 1.0:
                continue  # 无效深度
            
            # ===== 步骤3：将像素坐标转换为3D坐标 =====
            x = (center_u - self.cx) * z / self.fx
            y = (center_v - self.cy) * z / self.fy
            
            # ===== 步骤4：计算抓取宽度 =====
            width = min((x2 - x1) * z / self.fx, self.grasp_width_max)
            
            # ===== 步骤5：计算抓取角度 =====
            # 简单策略：抓取角度设为朝向物体中心
            grasp_yaw = np.arctan2(y, x)
            
            grasp = (x, y, z, 0.0, 0.0, grasp_yaw, width)
            grasps.append(grasp)
        
        return grasps
    
    def yolo_detect(self, rgb):
        """YOLO目标检测（简化版）
        
        实际项目中应该使用真正的YOLO模型进行检测
        这里返回模拟的检测结果用于演示
        """
        # 实际应用中：
        # model = torch.hub.load('ultralytics/yolov8', 'yolov8n')
        # results = model(rgb)
        # return results boxes
        
        # 模拟：检测图像中心区域
        h, w = rgb.shape[:2]
        return [('object', w//4, h//4, 3*w//4, 3*h//4)]
    
    def rgb_callback(self, msg):
        self.latest_rgb = np.frombuffer(msg.data, dtype=np.uint8).reshape(
            msg.height, msg.width, 3)
    
    def depth_callback(self, msg):
        self.latest_depth = np.frombuffer(msg.data, dtype=np.float32).reshape(
            msg.height, msg.width)
        
        if hasattr(self, 'latest_rgb'):
            grasps = self.detect_grasp(self.latest_rgb, self.latest_depth)
            self.publish_grasps(grasps)

```python
#!/usr/bin/env python3
"""
机械臂运动规划节点 - 使用MoveIt2进行运动规划
功能：接收抓取姿态，输出机械臂关节轨迹
"""

import rclpy
from rclpy.node import Node
from moveit_interface import MoveGroupInterface
import numpy as np


class ArmMotionPlanner(Node):
    """机械臂运动规划器
    
    使用MoveIt2进行运动规划：
    1. 抓取前姿态规划（PREGRASP）：移动到物体上方
    2. 抓取姿态规划（GRASP）：下降到抓取位置
    3. 抓取后姿态规划（RETREAT）：抬起并移动到放置位置
    """
    
    def __init__(self):
        super().__init__('arm_motion_planner')
        
        # 初始化MoveIt接口
        self.arm = MoveGroupInterface('arm_group', 'robot_description')
        
        # 机械臂关节限制
        self.joint_limits = {
            'shoulder_pan_joint': (-np.pi, np.pi),
            'shoulder_lift_joint': (-np.pi/2, np.pi/2),
            'elbow_joint': (-np.pi, 0),
            'wrist_1_joint': (-np.pi, np.pi),
            'wrist_2_joint': (-np.pi, np.pi),
            'wrist_3_joint': (-np.pi, np.pi),
        }
        
        # 抓取参数
        self.pregrasp_offset = 0.15   # 抓取前，机械臂提前停留的高度偏移（米）
        self.grasp_offset = 0.02      # 抓取时，机械臂距离物体的距离（米）
        
        self.get_logger().info('机械臂运动规划器已启动')
    
    def plan_to_grasp(self, target_pose):
        """规划抓取路径
        
        路径规划：当前位置 → 抓取前位置 → 抓取位置
        
        Args:
            target_pose: 目标抓取姿态 (x, y, z, roll, pitch, yaw)
        
        Returns:
            bool: 规划是否成功
        """
        x, y, z, roll, pitch, yaw = target_pose
        
        # ===== 步骤1：规划抓取前位置（PREGRASP）=====
        # 机械臂先移动到物体上方一段距离的位置
        pregrasp_pose = self.create_pose(
            x=x, y=y, z=z + self.pregrasp_offset,
            roll=roll, pitch=pitch, yaw=yaw
        )
        
        self.get_logger().info(f'规划PREGRASP位置: ({x:.3f}, {y:.3f}, {z+self.pregrasp_offset:.3f})')
        
        # 执行PREGRASP规划
        success_pre = self.arm.go_to_pose_goal(pregrasp_pose, wait=True)
        if not success_pre:
            self.get_logger().error('PREGRASP规划失败')
            return False
        
        # ===== 步骤2：规划抓取位置（GRASP）=====
        # 下降到抓取位置
        grasp_pose = self.create_pose(
            x=x, y=y, z=z - self.grasp_offset,
            roll=roll, pitch=pitch, yaw=yaw
        )
        
        self.get_logger().info(f'规划GRASP位置: ({x:.3f}, {y:.3f}, {z-self.grasp_offset:.3f})')
        
        # 执行GRASP规划（注意：这里通常会分多步下降，每步检查力反馈）
        success_grasp = self.arm.go_to_pose_goal(grasp_pose, wait=True)
        if not success_grasp:
            self.get_logger().error('GRASP规划失败')
            return False
        
        self.get_logger().info('抓取路径规划成功！')
        return True
    
    def plan_to_place(self, place_pose):
        """规划放置路径
        
        Args:
            place_pose: 放置位置姿态 (x, y, z, roll, pitch, yaw)
        
        Returns:
            bool: 规划是否成功
        """
        x, y, z, roll, pitch, yaw = place_pose
        
        # ===== 步骤1：先抬起（RETREAT）=====
        # 机械臂先抬起一段距离
        retreat_pose = self.create_pose(
            x=x, y=y, z=z + 0.1,
            roll=roll, pitch=pitch, yaw=yaw
        )
        
        success_retreat = self.arm.go_to_pose_goal(retreat_pose, wait=True)
        if not success_retreat:
            self.get_logger().error('RETREAT规划失败')
            return False
        
        # ===== 步骤2：移动到放置位置上方 =====
        preplace_pose = self.create_pose(
            x=x, y=y, z=z + self.pregrasp_offset,
            roll=roll, pitch=pitch, yaw=yaw
        )
        
        success_preplace = self.arm.go_to_pose_goal(preplace_pose, wait=True)
        if not success_preplace:
            self.get_logger().error('PREPLACE规划失败')
            return False
        
        # ===== 步骤3：下降到放置位置 =====
        place_pose_full = self.create_pose(
            x=x, y=y, z=z,
            roll=roll, pitch=pitch, yaw=yaw
        )
        
        success_place = self.arm.go_to_pose_goal(place_pose_full, wait=True)
        if not success_place:
            self.get_logger().error('PLACE规划失败')
            return False
        
        self.get_logger().info('放置路径规划成功！')
        return True
    
    @staticmethod
    def create_pose(x, y, z, roll, pitch, yaw):
        """创建位姿消息
        
        使用欧拉角创建四元数
        
        Args:
            x, y, z: 位置（米）
            roll, pitch, yaw: 欧拉角（弧度）
        
        Returns:
            geometry_msgs.msg.Pose
        """
        from geometry_msgs.msg import Pose
        from tf_transformations import quaternion_from_euler
        
        # 欧拉角转四元数
        q = quaternion_from_euler(roll, pitch, yaw)
        
        pose = Pose()
        pose.position.x = x
        pose.position.y = y
        pose.position.z = z
        pose.orientation.x = q[0]
        pose.orientation.y = q[1]
        pose.orientation.z = q[2]
        pose.orientation.w = q[3]
        
        return pose


def main(args=None):
    rclpy.init(args=args)
    planner = ArmMotionPlanner()
    try:
        rclpy.spin(planner)
    finally:
        planner.destroy_node()
        rclpy.shutdown()

#!/usr/bin/env python3
"""
协同控制节点 - 协调视觉、机械臂、夹爪三个子系统
功能：状态机控制整个抓取流程
"""

import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import time


class GraspingController(Node):
    """抓取协同控制器 - 状态机协调多个子系统
    
    抓取流程状态机：
    
    IDLE → DETECT → PLAN_PREGRASP → MOVE_PREGRASP 
         → PLAN_GRASP → MOVE_GRASP → GRASP 
         → PLAN_RETREAT → MOVE_RETREAT → DONE
    
    每个状态：
    - 检测前置条件
    - 执行对应动作
    - 判断是否成功，失败则进入恢复流程
    """
    
    # 状态定义
    STATE_IDLE          = 'IDLE'
    STATE_DETECT        = 'DETECT'
    STATE_PLAN_PREGRASP = 'PLAN_PREGRASP'
    STATE_MOVE_PREGRASP = 'MOVE_PREGRASP'
    STATE_PLAN_GRASP    = 'PLAN_GRASP'
    STATE_MOVE_GRASP    = 'MOVE_GRASP'
    STATE_GRASP         = 'GRASP'
    STATE_PLAN_RETREAT  = 'PLAN_RETREAT'
    STATE_MOVE_RETREAT  = 'MOVE_RETREAT'
    STATE_DONE          = 'DONE'
    STATE_ERROR         = 'ERROR'
    
    def __init__(self):
        super().__init__('grasping_controller')
        
        # 当前状态
        self.current_state = self.STATE_IDLE
        
        # 检测到的抓取目标
        self.current_grasp = None
        self.place_position = (0.3, 0.0, 0.1)  # 放置位置
        
        # 夹爪状态
        self.gripper_open = True
        
        # ===== 订阅者 =====
        # 订阅抓取检测结果
        self.grasp_sub = self.create_subscription(
            GraspPoseArray, '/grasp_poses',
            self.grasp_callback, 10
        )
        
        # ===== 发布者 =====
        # 发布控制指令
        self.cmd_pub = self.create_publisher(String, '/controller_cmd', 10)
        
        # 定时器：状态机主循环（10Hz）
        self.timer = self.create_timer(0.1, self.state_machine_loop)
        
        self.get_logger().info('抓取协同控制器已启动')
    
    def grasp_callback(self, msg):
        """接收抓取检测结果"""
        if msg.grasps and self.current_state == self.STATE_DETECT:
            # 选择最佳抓取目标（选择最近的）
            best_grasp = min(msg.grasps, key=lambda g: g.pose.position.z)
            self.current_grasp = best_grasp
            self.get_logger().info('收到抓取目标，开始规划...')
    
    def state_machine_loop(self):
        """状态机主循环"""
        self.get_logger().debug(f'当前状态: {self.current_state}')
        
        if self.current_state == self.STATE_IDLE:
            # 等待启动指令（实际项目中可能是UI按钮或语音指令）
            pass
        
        elif self.current_state == self.STATE_DETECT:
            # 等待视觉检测结果
            if self.current_grasp is not None:
                self.transition_to(self.STATE_PLAN_PREGRASP)
        
        elif self.current_state == self.STATE_PLAN_PREGRASP:
            # 规划PREGRASP路径
            self.plan_pregrasp()
        
        elif self.current_state == self.STATE_MOVE_PREGRASP:
            # 等待PREGRASP移动完成
            if self.arm_reached_target():
                self.transition_to(self.STATE_PLAN_GRASP)
        
        elif self.current_state == self.STATE_PLAN_GRASP:
            # 规划GRASP路径
            self.plan_grasp()
        
        elif self.current_state == self.STATE_MOVE_GRASP:
            # 等待GRASP移动完成
            if self.arm_reached_target():
                self.transition_to(self.STATE_GRASP)
        
        elif self.current_state == self.STATE_GRASP:
            # 执行抓取：关闭夹爪
            self.execute_grasp()
        
        elif self.current_state == self.STATE_PLAN_RETREAT:
            # 规划撤退路径
            self.plan_retreat()
        
        elif self.current_state == self.STATE_MOVE_RETREAT:
            # 等待撤退移动完成
            if self.arm_reached_target():
                self.transition_to(self.STATE_DONE)
        
        elif self.current_state == self.STATE_DONE:
            # 抓取成功，可以开始下一个任务
            self.get_logger().info('抓取任务完成！')
            time.sleep(2)
            self.reset()
    
    def transition_to(self, new_state):
        """状态转换"""
        self.get_logger().info(f'状态转换: {self.current_state} → {new_state}')
        self.current_state = new_state
    
    def plan_pregrasp(self):
        """规划PREGRASP路径"""
        # 实际项目中调用MoveIt2进行规划
        self.publish_cmd('PLAN_PREGRASP')
        self.transition_to(self.STATE_MOVE_PREGRASP)
    
    def plan_grasp(self):
        """规划GRASP路径"""
        self.publish_cmd('PLAN_GRASP')
        self.transition_to(self.STATE_MOVE_GRASP)
    
    def execute_grasp(self):

        self.publish_cmd('CLOSE_GRIPPER')
        self.gripper_open = False
        time.sleep(0.5)  # 等待夹爪关闭
        self.transition_to(self.STATE_PLAN_RETREAT)
    
    def plan_retreat(self):
        """规划撤退路径"""
        self.publish_cmd('PLAN_RETREAT')
        self.transition_to(self.STATE_MOVE_RETREAT)
    
    def arm_reached_target(self):
        """检查机械臂是否到达目标位置"""
        # 实际项目中检查MoveIt反馈
        return True
    
    def publish_cmd(self, cmd):
        """发布控制指令"""
        msg = String()
        msg.data = cmd
        self.cmd_pub.publish(msg)
    
    def reset(self):
        """重置状态机"""
        self.current_state = self.STATE_IDLE
        self.current_grasp = None
        self.gripper_open = True

```python
#!/usr/bin/env python3
"""
多机器人通信节点 - 使用ROS2分布式通信实现多机器人数据共享
功能：机器人之间共享位置、任务状态、环境信息
"""

import rclpy
from rclpy.node import Node
from geometry_msgs.msg import PoseStamped, Twist
from std_msgs.msg import String, Float32
import socket
import json


class MultiRobot通信(Node):
    """多机器人通信节点
    
    通信架构：
    - 每个机器人是一个ROS2节点
    - 使用DDS（Data Distribution Service）进行话题广播
    - 共享的话题：
      /robot_{id}/pose      - 机器人当前位置
      /robot_{id}/status    - 机器人状态（空闲/工作中/充电）
      /robot_{id}/cmd       - 接收其他机器人的指令
      /task_pool            - 共享任务池
    
    实际项目中可以使用：
    - ROS2原生的话题通信（简单场景）
    - ROS2的Action Server（需要协调的场景）
    - 专门的中间件如MQTT（跨网络场景）
    """
    
    def __init__(self, robot_id):
        super().__init__(f'multi_robot_comm_{robot_id}')
        
        self.robot_id = robot_id  # 机器人ID（1, 2, 3...）
        
        # 机器人状态
        self.current_pose = None
        self.current_status = 'idle'  # idle / working / charging
        self.current_task = None
        
        # 邻居机器人信息
        self.robots = {}  # {robot_id: {'pose': Pose, 'status': str}}
        
        # ===== 发布者 =====
        # 发布自己的位置
        self.pose_pub = self.create_publisher(
            PoseStamped, f'/robot_{robot_id}/pose', 10
        )
        
        # 发布自己的状态
        self.status_pub = self.create_publisher(
            String, f'/robot_{robot_id}/status', 10
        )
        
        # ===== 订阅者 =====
        # 订阅其他机器人的位置（实际项目中使用通配符订阅）
        for other_id in range(1, 4):
            if other_id != robot_id:
                self.create_subscription(
                    PoseStamped,
                    f'/robot_{other_id}/pose',
                    lambda msg, rid=other_id: self.pose_callback(msg, rid),
                    10
                )
                self.create_subscription(
                    String,
                    f'/robot_{other_id}/status',
                    lambda msg, rid=other_id: self.status_callback(msg, rid),
                    10
                )
        
        # 订阅任务池
        self.create_subscription(
            String, '/task_pool',
            self.task_pool_callback, 10
        )
        
        # 定时器：广播自己的状态（5Hz）
        self.create_timer(0.2, self.broadcast_status)
        
        self.get_logger().info(f'Robot {robot_id} 多机器人通信节点已启动')
    
    def pose_callback(self, msg, robot_id):
        """接收邻居机器人的位置"""
        self.robots[robot_id] = self.robots.get(robot_id, {})
        self.robots[robot_id]['pose'] = msg.pose
        self.check_collision()
    
    def status_callback(self, msg, robot_id):
        """接收邻居机器人的状态"""
        self.robots[robot_id] = self.robots.get(robot_id, {})
        self.robots[robot_id]['status'] = msg.data
    
    def task_pool_callback(self, msg):
        """接收任务池更新"""
        tasks = json.loads(msg.data)
        self.assign_task(tasks)
    
    def broadcast_status(self):
        """广播自己的状态给所有机器人"""
        # 发布位置
        pose_msg = PoseStamped()
        pose_msg.header.stamp = self.get_clock().now().to_msg()
        pose_msg.header.frame_id = 'map'
        pose_msg.pose = self.current_pose or Pose()
        self.pose_pub.publish(pose_msg)
        
        # 发布状态
        status_msg = String()
        status_msg.data = json.dumps({
            'robot_id': self.robot_id,
            'status': self.current_status,
            'task': self.current_task
        })
        self.status_pub.publish(status_msg)
    
    def assign_task(self, tasks):
        """任务分配算法
        
        分配策略：最短距离 + 负载均衡
        """
        if self.current_status != 'idle':
            return  # 忙碌中，不接新任务
        
        # 找出所有未分配的任务
        unassigned_tasks = [t for t in tasks if not t.get('assigned')]
        
        if not unassigned_tasks:
            return
        
        # 选择距离最近的任务
        best_task = None
        best_distance = float('inf')
        
        for task in unassigned_tasks:
            task_pos = task['goal_position']
            if self.current_pose:
                dist = self.calculate_distance(
                    self.current_pose.position,
                    task_pos
                )
                if dist < best_distance:
                    best_distance = dist
                    best_task = task
        
        if best_task:
            self.accept_task(best_task)
    
    def accept_task(self, task):
        """接受任务"""
        self.current_status = 'working'
        self.current_task = task
        self.get_logger().info(f'Robot {self.robot_id} 接受任务: {task["id"]}')
    
    def check_collision(self):
        """碰撞检测：检查是否与其他机器人距离太近"""
        if not self.current_pose:
            return
        
        for robot_id, info in self.robots.items():
            if 'pose' not in info:
                continue
            
            other_pose = info['pose']
            dist = self.calculate_distance(
                self.current_pose.position,
                other_pose.position
            )
            
            if dist < 0.5:  # 距离小于50cm，可能碰撞
                self.get_logger().warn(
                    f'与Robot {robot_id}距离过近: {dist:.2f}m，触发避障！'
                )
                self.trigger_avoidance(robot_id)
    
    def trigger_avoidance(self, other_robot_id):
        """触发避障行为"""
        # 实际项目中，这里会触发局部路径重规划
        self.get_logger().info(f'执行避障策略：等待Robot {other_robot_id}通过')
    
    @staticmethod
    def calculate_distance(pos1, pos2):
        """计算两点之间的距离"""
        import math
        return math.sqrt(
            (pos1.x - pos2.x)**2 + 
            (pos1.y - pos2.y)**2
        )

        rclpy.spin(node)

```python
#!/usr/bin/env python3
"""
协同决策节点 - 多机器人任务协调与冲突解决
功能：统一的调度中心，负责任务分配和冲突解决
"""

import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import json
import heapq


class TaskScheduler(Node):
    """任务调度器 - 集中式任务分配和协调
    
    功能：
    1. 维护任务池
    2. 监听所有机器人的状态
    3. 动态分配任务
    4. 处理冲突（如两个机器人抢同一个任务）
    5. 处理异常（如机器人故障）
    
    使用优先级队列管理任务：
    - 紧急任务：高优先级
    - 普通任务：低优先级
    """
    
    def __init__(self):
        super().__init__('task_scheduler')
        
        # 任务池（按优先级组织的最小堆）
        self.task_queue = []
        
        # 机器人状态表
        self.robot_states = {}  # {robot_id: {'status': str, 'pose': Pose}}
        
        # 任务分配表
        self.task_assignments = {}  # {task_id: robot_id}
        
        # ===== 发布者 =====
        # 发布任务池（广播给所有机器人）
        self.task_pool_pub = self.create_publisher(
            String, '/task_pool', 10
        )
        
        # 发布指令给特定机器人
        self.cmd_pubs = {
            i: self.create_publisher(String, f'/robot_{i}/cmd', 10)
            for i in range(1, 4)
        }
        
        # ===== 订阅者 =====
        # 监听所有机器人的状态
        for i in range(1, 4):
            self.create_subscription(
                String, f'/robot_{i}/status',
                lambda msg, rid=i: self.status_callback(msg, rid), 10
            )
        
        # 监听新任务到达
        self.create_subscription(
            String, '/new_task',
            self.new_task_callback, 10
        )
        
        # 定时器：定期更新任务分配（2Hz）
        self.create_timer(0.5, self调度_loop)
        
        self.get_logger().info('任务调度器已启动')
    
    def status_callback(self, msg, robot_id):
        """接收机器人状态更新"""
        data = json.loads(msg.data)
        self.robot_states[robot_id] = data
        
        # 如果机器人完成了任务，从分配表中移除
        if data.get('status') == 'idle' and robot_id in self.task_assignments.values():
            completed_task = [k for k, v in self.task_assignments.items() if v == robot_id]
            for task_id in completed_task:
                del self.task_assignments[task_id]
    
    def new_task_callback(self, msg):
        """接收新任务"""
        task = json.loads(msg.data)
        priority = task.get('priority', 5)  # 默认优先级5
        
        # 添加到优先级队列
        # heapq是最小堆，数值越小优先级越高
        heapq.heappush(self.task_queue, (priority, task['id'], task))
        
        self.get_logger().info(f'收到新任务: {task["id"]}, 优先级: {priority}')
    
    def dispatch_loop(self):
        """调度主循环"""
        # 重新发布任务池（带分配信息）
        tasks_with_assignments = []
        for _, _, task in self.task_queue:
            task_id = task['id']
            task_copy = task.copy()
            task_copy['assigned'] = task_id in self.task_assignments
            task_copy['assigned_to'] = self.task_assignments.get(task_id)
            tasks_with_assignments.append(task_copy)
        
        # 发布任务池
        pool_msg = String()
        pool_msg.data = json.dumps(tasks_with_assignments)
        self.task_pool_pub.publish(pool_msg)
        
        # 尝试分配新任务
        self.assign_tasks()
    
    def assign_tasks(self):
        """任务分配算法"""
        # 找出所有空闲的机器人
        idle_robots = [
            rid for rid, state in self.robot_states.items()
            if state.get('status') == 'idle' and 
               rid not in [v for v in self.task_assignments.values()]
        ]
        
        if not idle_robots or not self.task_queue:
            return  # 没有空闲机器人或没有任务
        
        # 取出最高优先级任务
        _, _, task = heapq.heappop(self.task_queue)
        
        # 选择距离最近的空闲机器人
        best_robot = min(
            idle_robots,
            key=lambda rid: self.distance_to_task(rid, task)
        )
        
        # 分配任务
        self.task_assignments[task['id']] = best_robot
        
        # 发送指令给选中的机器人
        cmd = {
            'command': 'EXECUTE_TASK',
            'task': task
        }
        cmd_msg = String()
        cmd_msg.data = json.dumps(cmd)
        self.cmd_pubs[best_robot].publish(cmd_msg)
        
        self.get_logger().info(
            f'任务 {task["id"]} 分配给 Robot {best_robot}'
        )
    
    def distance_to_task(self, robot_id, task):
        """计算机器人到任务的距离"""
        robot_state = self.robot_states.get(robot_id, {})
        robot_pose = robot_state.get('pose', {})
        
        task_pos = task.get('goal_position', {})
        
        import math
        return math.sqrt(
            (robot_pose.get('x', 0) - task_pos.get('x', 0))**2 +
            (robot_pose.get('y', 0) - task_pos.get('y', 0))**2
        )


def main(args=None):
    rclpy.init(args=args)
    scheduler = TaskScheduler()
    try:
        rclpy.spin(scheduler)
    finally:
        scheduler.destroy_node()
        rclpy.shutdown()

    Attributes:
        max_speed: 最大速度（米/秒）
        min_speed: 最小速度（米/秒）
    """
    
    def __init__(self, max_speed: float = 1.0, min_speed: float = 0.0) -> None:
        """初始化导航控制器
        
        Args:
            max_speed: 最大允许速度
            min_speed: 最小允许速度
        """
        self.max_speed = max_speed
        self.min_speed = min_speed
    
    def compute_velocity(
        self, 
        current_pose: Tuple[float, float, float],
        target_pose: Tuple[float, float, float]
    ) -> Tuple[float, float]:
        """计算速度指令
        
        使用Pure Pursuit算法计算机器人的线速度和角速度。
        
        Args:
            current_pose: 当前位姿 (x, y, theta)
            target_pose: 目标位姿 (x, y, theta)
        
        Returns:
            Tuple[float, float]: (线速度, 角速度)
        
        Raises:
            ValueError: 如果速度超出限制范围
        
        Example:
            >>> controller = NavigationController()
            >>> vx, vw = controller.compute_velocity((0, 0, 0), (1, 0, 0))
            >>> print(f"速度指令: vx={vx}, vw={vw}")
        """
        # 计算距离和角度误差
        dx = target_pose[0] - current_pose[0]
        dy = target_pose[1] - current_pose[1]
        dist = np.sqrt(dx**2 + dy**2)
        angle_error = np.arctan2(dy, dx) - current_pose[2]
        
        # 归一化角度
        angle_error = np.arctan2(np.sin(angle_error), np.cos(angle_error))
        
        # 计算速度（简单的P控制器）
        vx = np.clip(dist * 1.0, self.min_speed, self.max_speed)
        vw = angle_error * 1.5
        
        return vx, vw

**2. SLAM建图的核心原理是什么？**

<details>
<summary>点击查看答案</summary>

SLAM（Simultaneous Localization and Mapping）的核心原理：
- **定位**：机器人通过传感器数据（激光雷达、相机等）估计自己在地图中的位置
- **建图**：同时根据机器人位置，将传感器观测到的环境障碍物标记到地图上
- **闭环检测**：当机器人回到曾经到过的地方时，修正累计误差

核心公式：机器人位姿 = f(传感器观测, 控制输入, 地图)

</details>

**3. Pure Pursuit算法的作用是什么？**

<details>
<summary>点击查看答案</summary>

Pure Pursuit算法是一种路径跟踪算法，核心思想是：
1. 在全局路径上找一个"前看点"（lookahead point）
2. 计算机器人到前看点的角度误差
3. 将角度误差作为角速度指令，使机器人"追着"前看点运动
4. 当机器人接近目标时，自动减速

优点：实现简单、路径跟踪平滑、鲁棒性好

</details>

**4. ROS2的`rclpy`和`rclcpp`有什么区别？**

<details>
<summary>点击查看答案</summary>

- **rclpy**：ROS2的Python客户端库，用于编写Python节点
- **rclcpp**：ROS2的C++客户端库，用于编写C++节点

选择建议：
- 快速原型验证、脚本工具、数据处理 → 使用 **rclpy**（Python）
- 性能关键的实时控制、嵌入式 → 使用 **rclcpp**（C++）
- 实际项目中：感知、规划等用Python，高频控制用C++

</details>

### 进阶题

**1. 为什么具身智能项目必须经过仿真验证才能上真机？**

<details>
<summary>点击查看答案</summary>

原因有以下几点：

1. **安全性**：真机上Bug可能导致机器人失控，损坏硬件或伤人
2. **效率**：调试一次真机需要充电、复位，而仿真可以无限次重置
3. **成本**：真机损耗是真实成本，仿真几乎是零成本
4. **可重复性**：仿真环境是完全可重复的，真机受环境影响大
5. **覆盖Corner Case**：仿真可以轻松模拟危险场景，真机难以复现

最佳实践：**TDD（Test-Driven Development）**
- 先写测试用例
- 在仿真中通过测试
- 再部署到真机验证

</details>

**2. 多机器人协作中的冲突检测和解决策略有哪些？**

<details>
<summary>点击查看答案</summary>

**冲突检测**：
- **距离检测**：检测两个机器人之间的距离是否小于安全阈值
- **路径交叉**：检测两条规划路径是否会在同一时间到达同一位置
- **资源竞争**：检测两个任务是否需要同一个物理资源（如电梯）

**解决策略**：

| 策略 | 原理 | 适用场景 |
|------|------|----------|
| **优先级调度** | 高优先级机器人优先通过 | 固定优先级场景 |
| **先来先服务** | 先申请路径的机器人优先 | 任务到达时间重要 |
| **动态绕行** | 一个机器人主动绕行 | 临时冲突 |
| **速度调整** | 加快或放慢其中一个机器人 | 避免碰撞时间点 |
| **重新规划** | 触发局部路径重规划 | 复杂环境 |

**实现方式**：
- **集中式**：调度中心统一分配路径（如本案例的TaskScheduler）
- **分布式**：机器人之间协商（如分布式一致性算法）

</details>

**3. 如何评估一个具身智能项目的性能？**

<details>
<summary>点击查看答案</summary>

**性能指标体系**：

| 类别 | 指标 | 测量方法 |
|------|------|----------|
| **准确性** | 定位精度、建图精度、抓取成功率 | 地面真值对比 |
| **实时性** | 控制频率、感知延迟、规划耗时 | ROS2话题延迟统计 |
| **鲁棒性** | 成功率、平均故障间隔 | 长时运行统计 |
| **效率** | 任务完成时间、路径长度、能耗 | 实际任务计时 |
| **安全性** | 碰撞次数、紧急停止次数 | 传感器和日志 |

**评估方法**：

1. **仿真评估**：在Gazebo中运行标准化测试场景
2. **真机评估**：在真实环境中运行实际任务
3. **压力测试**：在极端条件下测试系统稳定性
4. **用户评估**：真人用户操作，收集主观反馈

**benchmark工具**：
- ROS2性能测试：`ros2 topic hz`、`ros2 topic delay`
- 导航benchmark：Nav2提供的`nav2_benchmark`
- 抓取benchmark：GraspNet benchmark

</details>
