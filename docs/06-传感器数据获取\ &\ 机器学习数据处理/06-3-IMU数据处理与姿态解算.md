## 1. IMU原理

惯性测量单元（Inertial Measurement Unit，IMU）是机器人系统中用于测量载体运动状态的核心传感器。一个典型的IMU通常包含**加速度计**和**陀螺仪**，有些高级IMU还集成**磁力计**形成九轴传感器。

### 1.1 加速度计工作原理

加速度计测量的是**比力**（specific force），即除重力以外作用在物体上的力。其基本原理基于**牛顿第二定律**：

$$F = ma$$

在加速度计内部，有一个质量块通过弹簧悬挂在外壳上。当载体加速运动时，质量块由于惯性会相对外壳产生位移。这个位移被转换为电信号输出，从而得到加速度值。

```
┌─────────────────────┐
│     加速度计内部     │
│  ┌───────────────┐  │
│  │   质量块(m)   │◀─┼── 弹簧
│  └───────────────┘  │
│         ↓          │
│   位移传感器        │
│         ↓          │
│   输出电压 ∝ 加速度 │
└─────────────────────┘
```

加速度计的输出值包括三个轴向的加速度：
- $a_x$：X轴加速度（前后方向）
- $a_y$：Y轴加速度（左右方向）  
- $a_z$ ：Z轴加速度（垂直方向）

**重要特性**：加速度计对振动敏感，但测量低频运动较为准确。在静止状态下，$a_z$ 接近重力加速度 $g \approx 9.81 m/s^2$。

### 1.2 陀螺仪工作原理

陀螺仪测量的是**角速度**（angular velocity），即载体绕各轴旋转的速率。其工作原理基于**科里奥利效应**：

当一个质量块在旋转的参考系中移动时，会受到一个垂直于运动方向和旋转轴的力——科里奥利力。陀螺仪通过测量这个力来确定角速度。

```
┌─────────────────────┐
│      陀螺仪内部      │
│                     │
│   ══════════════    │  ← 振动质量块
│        ↑↑↑↑         │
│        科里奥利力    │
│        ↓↓↓↓         │
│   检测电容变化       │
│        ↓            │
│   输出电压 ∝ 角速度  │
└─────────────────────┘
```

陀螺仪的输出值包括三个轴向的角速度：
- $\omega_x$：绕X轴旋转速度（roll rate）
- $\omega_y$：绕Y轴旋转速度（pitch rate）
- $\omega_z$ ：绕Z轴旋转速度（yaw rate）

**重要特性**：陀螺仪对高频运动响应好，但存在**漂移**（drift）问题——即使载体静止，输出也会随时间缓慢累积误差。

### 1.3 坐标系定义

理解IMU数据需要明确以下几个坐标系：

| 坐标系 | 说明 |
|--------|------|
| **惯性坐标系（I系）** | 固定不动的参考坐标系，通常以地球为参考 |
| **载体坐标系（B系）** | 固定在机器人本体上的坐标系 |
| **ENU坐标系** | 东北天坐标系，X指向东，Y指向北，Z指向天 |

ROS2中常用的坐标系约定：
- **base_link**：机器人载体坐标系
- **imu_link**：IMU传感器坐标系
- **world/map**：世界/地图坐标系

### 1.4 ROS2中的IMU消息

ROS2使用`sensor_msgs/Imu`消息类型传输IMU数据：

```yaml

```

## 2. 姿态表示方法

机器人姿态是指载体坐标系相对于参考坐标系的旋转关系。常用的姿态表示方法有：**欧拉角**、**旋转矩阵**和**四元数**。

### 2.1 欧拉角

欧拉角使用三个旋转角度表示姿态：

- **Roll（横滚）**：绕X轴旋转，$\phi$
- **Pitch（俯仰）**：绕Y轴旋转，$\theta$
- **Yaw（偏航）**：绕Z轴旋转，$\psi$

```
         Y (北)
         ↑
         │
    ─────┼─────→ X (东)
         │
         │
         ↓
        Z (天)
         
Roll(φ): 绕X轴旋转 - 飞机左右倾斜
Pitch(θ): 绕Y轴旋转 - 飞机抬头/低头
Yaw(ψ):  绕Z轴旋转 - 飞机机头左右偏转
```

欧拉角的优势是直观易懂，但存在**万向节锁**（Gimbal Lock）问题——当俯仰角接近±90°时，会丢失一个自由度。

### 2.2 四元数

四元数（Quaternion）是解决万向节锁问题的最佳方案，使用四个参数表示三维旋转：

$$q = w + xi + yj + zk = [w, x, y, z]^T$$

其中：
$$w^2 + x^2 + y^2 + z^2 = 1$$

四元数的优点：
- 避免万向节锁
- 插值平滑
- 计算效率高
- 数值稳定

ROS2的`Imu`消息中使用四元数表示姿态。

### 2.3 姿态转换

**四元数转欧拉角**：

```python
def quaternion_to_euler(q):
    """四元数转欧拉角（弧度）"""
    w, x, y, z = q.w, q.x, q.y, q.z
    
    # Roll (x-axis rotation)
    sinr_cosp = 2 * (w * x + y * z)
    cosr_cosp = 1 - 2 * (x * x + y * y)
    roll = math.atan2(sinr_cosp, cosr_cosp)
    
    # Pitch (y-axis rotation)
    sinp = 2 * (w * y - z * x)
    if abs(sinp) >= 1:
        pitch = math.copysign(math.pi / 2, sinp)  # use 90 degrees if out of range
    else:
        pitch = math.asin(sinp)
    
    # Yaw (z-axis rotation)
    siny_cosp = 2 * (w * z + x * y)
    cosy_cosp = 1 - 2 * (y * y + z * z)
    yaw = math.atan2(siny_cosp, cosy_cosp)
    
    return roll, pitch, yaw
```

**欧拉角转四元数**：

```python
def euler_to_quaternion(roll, pitch, yaw):
    """欧拉角转四元数"""
    cy = math.cos(yaw * 0.5)
    sy = math.sin(yaw * 0.5)
    cp = math.cos(pitch * 0.5)
    sp = math.sin(pitch * 0.5)
    cr = math.cos(roll * 0.5)
    sr = math.sin(roll * 0.5)
    
    w = cr * cp * cy + sr * sp * sy
    x = sr * cp * cy - cr * sp * sy
    y = cr * sp * cy + sr * cp * sy
    z = cr * cp * sy - sr * sp * cy
    
    return [w, x, y, z]
```

## 3. 姿态解算算法

从IMU原始数据获取稳定姿态需要使用算法进行处理。本节介绍两种主流算法：**互补滤波**和**卡尔曼滤波**。

### 3.1 互补滤波（Complementary Filter）

互补滤波的核心思想是：**加速度计适合测量低频姿态，陀螺仪适合测量高频姿态**。通过将两者融合，取长补短，得到更准确的姿态估计。

算法公式：

$$\theta_{fusion} = \alpha \cdot \theta_{gyro} + (1 - \alpha) \cdot \theta_{accel}$$

其中：
- $\alpha$ 为融合系数，通常取 0.9~0.98
- $\theta_{gyro}$ 为陀螺仪积分得到的角度
- $\theta_{accel}$ 为加速度计计算得到的角度

**陀螺仪积分**：

$$\theta_{gyro}(t) = \theta_{gyro}(t-1) + \omega \cdot \Delta t$$

**加速度计计算角度**（简化模型）：

$$\theta_{accel} = \arctan2(a_y, a_z)$$

$$\phi_{accel} = \arctan2(-a_x, \sqrt{a_y^2 + a_z^2})$$

互补滤波的优点是计算简单、实时性好，适用于大多数嵌入式系统。缺点是无法完全消除陀螺仪漂移。

```
┌──────────────┐     ┌──────────────┐
│  加速度计     │     │   陀螺仪     │
│  (低频准确)   │     │  (高频准确)  │
└──────┬───────┘     └──────┬───────┘
       │                    │
       ▼                    ▼
┌──────────────┐     ┌──────────────┐
│ 低通滤波器   │     │ 高通滤波器   │
└──────┬───────┘     └──────┬───────┘
       │                    │
       └────────┬───────────┘
                ▼
        ┌──────────────┐
        │   姿态融合   │
        └──────────────┘
```

### 3.2 卡尔曼滤波（Kalman Filter）

卡尔曼滤波是一种**最优估计**算法，它在存在噪声的情况下，能够从一系列包含噪声的测量值中估计系统的最优状态。

卡尔曼滤波的核心是两个方程：

**预测方程**：
$$\hat{x}_{k|k-1} = F_k \hat{x}_{k-1|k-1} + B_k u_k$$

$$P_{k|k-1} = F_k P_{k-1|k-1} F_k^T + Q_k$$

**更新方程**：
$$K_k = P_{k|k-1} H_k^T (H_k P_{k|k-1} H_k^T + R_k)^{-1}$$

$$\hat{x}_{k|k} = \hat{x}_{k|k-1} + K_k (z_k - H_k \hat{x}_{k|k-1})$$

$$P_{k|k} = (I - K_k H_k) P_{k|k-1}$$

其中：
- $\hat{x}$：状态估计
- $P$：估计协方差
- $Q$：过程噪声协方差
- $R$：测量噪声协方差
- $K$：卡尔曼增益

**在IMU姿态估计中的应用**：

状态向量通常包含：
$$x = [\phi, \theta, \psi, \delta\omega_x, \delta\omega_y, \delta\omega_z]^T$$

其中 $\phi, \theta, \psi$ 为姿态角，$\delta\omega$ 为陀螺仪偏置。

卡尔曼滤波的优点是理论上最优，缺点是计算复杂度高，需要较精确的系统模型和噪声参数。

### 3.3 算法对比

| 特性 | 互补滤波 | 卡尔曼滤波 |
|------|----------|------------|
| 计算复杂度 | 低 | 高 |
| 实时性 | 好 | 一般 |
| 陀螺仪漂移处理 | 一般 | 好 |
| 参数调优 | 简单（只需α） | 复杂（Q、R矩阵） |
| 适用场景 | 嵌入式、实时控制 | 精度要求高 |

### 5.2 互补滤波实现

```python
# imu_fusion_node.py
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Imu
from geometry_msgs.msg import Vector3
import math
import numpy as np


class ComplementaryFilter:
    """互补滤波器"""
    
    def __init__(self, alpha=0.98):
        """
        初始化互补滤波器
        
        Args:
            alpha: 融合系数，值越大越相信陀螺仪 (0.9~0.98)
        """
        self.alpha = alpha
        self.roll = 0.0
        self.pitch = 0.0
        self.yaw = 0.0
        
    def update(self, ax, ay, az, gx, gy, gz, dt):
        """
        更新姿态估计
        
        Args:
            ax, ay, az: 加速度计数据 (m/s²)
            gx, gy, gz: 陀螺仪数据 (rad/s)
            dt: 时间间隔 (s)
        """
        # 从加速度计计算姿态
        pitch_acc = math.atan2(-ax, math.sqrt(ay*ay + az*az))
        roll_acc = math.atan2(ay, az)
        
        # 陀螺仪积分
        self.roll = self.alpha * (self.roll + gx * dt) + (1 - self.alpha) * roll_acc
        self.pitch = self.alpha * (self.pitch + gy * dt) + (1 - self.alpha) * pitch_acc
        self.yaw = self.yaw + gz * dt  # Yaw只能从陀螺仪获取
        
        return self.roll, self.pitch, self.yaw
    
    def reset(self):
        """重置姿态"""
        self.roll = 0.0
        self.pitch = 0.0
        self.yaw = 0.0


class ImuFusionNode(Node):
    """IMU融合节点"""
    
    def __init__(self):
        super().__init__('imu_fusion_node')
        
        # 参数
        self.declare_parameter('alpha', 0.98)
        self.declare_parameter('frame_id', 'imu_link')
        self.declare_parameter('input_topic', '/imu/data_raw')
        self.declare_parameter('output_topic', '/imu/fusion')
        
        self.alpha = self.get_parameter('alpha').value
        self.frame_id = self.get_parameter('frame_id').value
        input_topic = self.get_parameter('input_topic').value
        output_topic = self.get_parameter('output_topic').value
        
        # 初始化滤波器
        self.filter = ComplementaryFilter(alpha=self.alpha)
        
        # 状态变量
        self.last_time = None
        
        # 订阅原始IMU数据
        self.imu_sub = self.create_subscription(
            Imu,
            input_topic,
            self.imu_callback,
            10
        )
        
        # 发布融合后的姿态（使用Vector3模拟欧拉角输出）
        self.orientation_pub = self.create_publisher(
            Vector3,
            output_topic,
            10
        )
        
        # 同时发布完整的Imu消息（带融合后的姿态）
        self.imu_pub = self.create_publisher(
            Imu,
            output_topic + '_imu',
            10
        )
        
        self.get_logger().info(f'IMU Fusion node started')
        self.get_logger().info(f'Input topic: {input_topic}')
        self.get_logger().info(f'Output topic: {output_topic}')
        self.get_logger().info(f'Filter alpha: {self.alpha}')
        
    def imu_callback(self, msg: Imu):
        """IMU数据回调"""
        current_time = msg.header.stamp
        
        if self.last_time is None:
            self.last_time = current_time
            return
            
        # 计算时间间隔（秒）
        dt = (current_time.nanoseconds - self.last_time.nanoseconds) / 1e9
        self.last_time = current_time
        
        # 防止时间间隔过大（避免积分发散）
        if dt > 0.5 or dt <= 0:
            dt = 0.02  # 默认20ms
            
        # 获取加速度计数据
        ax = msg.linear_acceleration.x
        ay = msg.linear_acceleration.y
        az = msg.linear_acceleration.z
        
        # 获取陀螺仪数据
        gx = msg.angular_velocity.x
        gy = msg.angular_velocity.y
        gz = msg.angular_velocity.z
        
        # 更新姿态
        roll, pitch, yaw = self.filter.update(ax, ay, az, gx, gy, gz, dt)
        
        # 发布欧拉角
        orientation_msg = Vector3()
        orientation_msg.x = roll
        orientation_msg.y = pitch
        orientation_msg.z = yaw
        self.orientation_pub.publish(orientation_msg)
        
        # 发布带姿态的IMU消息
        fused_msg = Imu()
        fused_msg.header = msg.header
        fused_msg.header.frame_id = self.frame_id
        
        # 转换欧拉角到四元数
        q = self.euler_to_quaternion(roll, pitch, yaw)
        fused_msg.orientation.x = q[1]
        fused_msg.orientation.y = q[2]
        fused_msg.orientation.z = q[3]
        fused_msg.orientation.w = q[0]
        
        # 保留原始角速度
        fused_msg.angular_velocity = msg.angular_velocity
        fused_msg.linear_acceleration = msg.linear_acceleration
        
        self.imu_pub.publish(fused_msg)
        
    def euler_to_quaternion(self, roll, pitch, yaw):
        """欧拉角转四元数"""
        cy = math.cos(yaw * 0.5)
        sy = math.sin(yaw * 0.5)
        cp = math.cos(pitch * 0.5)
        sp = math.sin(pitch * 0.5)
        cr = math.cos(roll * 0.5)
        sr = math.sin(roll * 0.5)
        
        w = cr * cp * cy + sr * sp * sy
        x = sr * cp * cy - cr * sp * sy
        y = cr * sp * cy + sr * cp * sy
        z = cr * cp * sy - sr * sp * cy
        
        return [w, x, y, z]


def main(args=None):
    rclpy.init(args=args)
    node = ImuFusionNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()


if __name__ == '__main__':
    main()
```

### 5.3 卡尔曼滤波实现

```python
# imu_fusion_node.py (扩展版 - 卡尔曼滤波)
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Imu
from geometry_msgs.msg import Vector3
import math
import numpy as np


class KalmanFilter:
    """扩展卡尔曼滤波器用于IMU姿态估计"""
    
    def __init__(self):
        # 状态向量: [roll, pitch, yaw, gyro_bias_x, gyro_bias_y, gyro_bias_z]
        self.x = np.zeros(6)
        
        # 状态协方差矩阵
        self.P = np.eye(6) * 0.1
        
        # 过程噪声协方差
        self.Q = np.eye(6) * 0.001
        self.Q[0:3, 0:3] *= 0.01  # 姿态的过程噪声
        self.Q[3:6, 3:6] *= 0.001  # 偏置的过程噪声
        
        # 测量噪声协方差（从IMU消息获取或使用默认值）
        self.R = np.eye(3) * 0.1
        
        # 雅可比矩阵
        self.F = np.eye(6)  # 状态转移矩阵
        self.H = np.eye(3, 6)  # 测量矩阵
        
    def predict(self, gx, gy, gz, dt):
        """预测步骤"""
        # 状态转移（简化模型：角度积分，偏置不变）
        self.x[0] += (gx - self.x[3]) * dt
        self.x[1] += (gy - self.x[4]) * dt
        self.x[2] += (gz - self.x[5]) * dt
        
        # 更新状态转移矩阵
        self.F[0, 3] = -dt
        self.F[1, 4] = -dt
        self.F[2, 5] = -dt
        
        # 协方差预测
        self.P = self.F @ self.P @ self.F.T + self.Q
        
    def update(self, ax, ay, az):
        """更新步骤"""
        # 从加速度计计算测量值
        z_measured = np.array([
            math.atan2(ay, az),  # roll
            math.atan2(-ax, math.sqrt(ay*ay + az*az)),  # pitch
            0.0  # yaw无法从加速度计获取
        ])
        
        # 测量残差
        z_pred = self.x[0:3]
        y = z_measured - z_pred
        
        # 卡尔曼增益
        S = self.H @ self.P @ self.H.T + self.R
        K = self.P @ self.H.T @ np.linalg.inv(S)
        
        # 状态更新
        self.x = self.x + K @ y
        
        # 协方差更新
        self.P = (np.eye(6) - K @ self.H) @ self.P
        
        return self.x[0], self.x[1], self.x[2]


class ImuFusionNode(Node):
    """IMU融合节点 - 卡尔曼滤波版本"""
    
    def __init__(self):
        super().__init__('imu_fusion_node')
        
        # 参数
        self.declare_parameter('frame_id', 'imu_link')
        self.declare_parameter('input_topic', '/imu/data_raw')
        self.declare_parameter('output_topic', '/imu/fusion')
        
        self.frame_id = self.get_parameter('frame_id').value
        input_topic = self.get_parameter('input_topic').value
        output_topic = self.get_parameter('output_topic').value
        
        # 初始化卡尔曼滤波器
        self.filter = KalmanFilter()
        
        # 状态变量
        self.last_time = None
        
        # 订阅原始IMU数据
        self.imu_sub = self.create_subscription(
            Imu,
            input_topic,
            self.imu_callback,
            10
        )
        
        # 发布融合后的姿态
        self.orientation_pub = self.create_publisher(
            Vector3,
            output_topic,
            10
        )
        
        self.get_logger().info(f'IMU Fusion node started (Kalman Filter)')
        
    def imu_callback(self, msg: Imu):
        """IMU数据回调"""
        current_time = msg.header.stamp
        
        if self.last_time is None:
            self.last_time = current_time
            return
            
        # 计算时间间隔
        dt = (current_time.nanoseconds - self.last_time.nanoseconds) / 1e9
        self.last_time = current_time
        
        if dt > 0.5 or dt <= 0:
            dt = 0.02
            
        # 获取传感器数据
        ax = msg.linear_acceleration.x
        ay = msg.linear_acceleration.y
        az = msg.linear_acceleration.z
        
        gx = msg.angular_velocity.x
        gy = msg.angular_velocity.y
        gz = msg.angular_velocity.z
        
        # 卡尔曼滤波预测和更新
        self.filter.predict(gx, gy, gz, dt)
        roll, pitch, yaw = self.filter.update(ax, ay, az)
        
        # 发布
        orientation_msg = Vector3()
        orientation_msg.x = roll
        orientation_msg.y = pitch
        orientation_msg.z = yaw
        self.orientation_pub.publish(orientation_msg)


def main(args=None):
    rclpy.init(args=args)
    node = ImuFusionNode()
    rclpy.spin(node)
    node.destroy_node()
    rclpy.shutdown()


if __name__ == '__main__':
    main()
```

## 6. 进阶：传感器噪声建模

要获得更好的姿态估计效果，需要对传感器噪声进行准确建模。

### 6.1 噪声分析

**加速度计噪声源**：
- 白噪声：测量不确定性
- 偏置不稳定性：低速漂移
- 振动：高频干扰

**陀螺仪噪声源**：
- 角度随机游走（ARW）：积分后的角度漂移
- 偏置不稳定性：低速漂移
- 角度偏置：恒定偏移

### 6.2 Allan方差分析

Allan方差是分析IMU噪声特性的标准方法：

```
┌─────────────────────────────────────┐
│         Allan方差曲线               │
│                                     │
│   log(σ)                           │
│     │     ╱                         │
│     │   ╱   ← 角度随机游走          │
│     │ ╱                             │
│     │╱                              │
│     └──────────────────→ log(τ)    │
│         1   10  100  1000          │
└─────────────────────────────────────┘
```

通过Allan方差分析可以得到：
- **角度随机游走系数**：$\sqrt{N}$
- **偏置不稳定性**：$\sqrt{B}$

```python
# 噪声参数配置示例
class ImuParameters:
    """IMU噪声参数"""
    
    # MPU6050典型参数
    gyro_noise_density = 0.005  # rad/s/√Hz

    accel_random_walk = 0.001  # m/s³/√Hz
    
    # 用于卡尔曼滤波的协方差矩阵
    @staticmethod
    def get_gyro_covariance():
        return np.eye(3) * ImuParameters.gyro_noise_density ** 2
    
    @staticmethod
    def get_accel_covariance():
        return np.eye(3) * ImuParameters.accel_noise_density ** 2
```

   - B) 位移

2. 互补滤波中，融合系数α的取值范围和含义是？
   - A) 0~1，α越大越相信加速度计
   - B) 0~1，α越大越相信陀螺仪
   - C) -1~1，α越大响应越快
   - D) 0~1，α越小滤波效果越好
   
   **答案：B**。α越大表示越相信陀螺仪的短期准确性，加速度计的权重越小。

3. 卡尔曼滤波中的Q矩阵代表什么？
   - A) 测量噪声协方差
   - B) 过程噪声协方差
   - C) 状态协方差
   - D) 卡尔曼增益
   
   **答案：B**。Q矩阵是过程噪声协方差矩阵，描述系统模型的不确定性。

5. 修改互补滤波实现，添加磁力计融合来修正Yaw角。
   
   **参考答案**：
   
   ```python
   class ComplementaryFilterWithMag:
       """带磁力计的互补滤波器"""
       
       def __init__(self, alpha=0.98, beta=0.02):
           self.alpha = alpha  # 陀螺仪权重
           self.beta = beta    # 磁力计权重
           self.roll = 0.0
           self.pitch = 0.0
           self.yaw = 0.0
           
       def update(self, ax, ay, az, mx, my, mz, gx, gy, gz, dt):
           # 加速度计算姿态
           pitch_acc = math.atan2(-ax, math.sqrt(ay*ay + az*az))
           roll_acc = math.atan2(ay, az)
           
           # 磁力计计算Yaw（需要补偿Roll和Pitch）
           # 简化计算
           yaw_mag = math.atan2(my, mx)
           
           # 陀螺仪积分
           roll_gyro = self.roll + gx * dt
           pitch_gyro = self.pitch + gy * dt
           yaw_gyro = self.yaw + gz * dt
           
           # 融合
           self.roll = self.alpha * roll_gyro + (1 - self.alpha) * roll_acc
           self.pitch = self.alpha * pitch_gyro + (1 - self.alpha) * pitch_acc
           self.yaw = (1 - self.beta) * yaw_gyro + self.beta * yaw_mag
           
           return self.roll, self.pitch, self.yaw
   ```
