## 1. 项目概述与背景

### 1.1 什么是VLM视觉-语言导航

VLM（Vision-Language Model）视觉-语言导航是指机器人通过视觉感知与自然语言指令的深度融合，实现自主导航的一项技术。与传统SLAM导航依赖精确几何地图不同，VLM导航让机器人能够"看懂"环境——识别物体、理解空间关系、遵循自然语言指令完成复杂导航任务。

**核心问题定义**：

给定一个自然语言指令（如"走进厨房，在蓝色沙发旁边停下"）和机器人的第一人称视觉观测，导航智能体需要预测一系列动作（如前进、左转、停止），使机器人最终到达目标区域。

```
┌─────────────────────────────────────────────────────────────┐
│               VLM 视觉-语言导航问题定义                      │
└─────────────────────────────────────────────────────────────┘

输入：
  - 自然语言指令 I："走进厨房，在蓝色沙发旁边停下"
  - 第一人称视觉观测 V：RGB-D 图像序列
  - 可选：拓扑/几何地图 M

输出：
  - 动作序列 A = {a_1, a_2, ..., a_n}
    其中 a_i ∈ {前进, 后退, 左转, 右转, 停止, 注视}

目标：
  - 到达指令描述的目标位置
  - 最大化成功率（Success Rate）
  - 最小化路径长度
```

### 1.2 VLM导航的优势

相比传统导航方法，VLM导航具有以下核心优势：

| 特性 | 传统SLAM导航 | VLM视觉-语言导航 |
|------|-------------|----------------|
| **指令理解** | 依赖预定义目标点坐标 | 自然语言灵活描述目标 |
| **环境感知** | 几何特征（点云/激光） | 语义感知（物体/场景/关系） |
| **泛化能力** | 需预先建图，泛化差 | 零样本/少样本泛化 |
| **人机交互** | 复杂坐标/命令 | 自然语言对话 |
| **容错能力** | 地图漂移难恢复 | 视觉语义纠错 |
| **部署成本** | 高精度传感器要求 | 依赖模型泛化能力 |

### 1.3 端到端视觉语言导航

端到端VLM导航将视觉感知与语言理解统一到单一模型中，直接从"图像+文本"映射到"动作输出"，避免了传统流水线的误差累积问题：

```
┌─────────────────────────────────┐     ┌─────────────────────────────────┐
│     传统流水线导航架构            │     │     端到端VLM导航架构            │
├─────────────────────────────────┤     ├─────────────────────────────────┤
│  感知 → 建图 → 定位 → 规划 → 控制 │     │  图像 + 文本 → VLM → 动作        │
│                                 │     │                                 │
│  问题：各模块独立优化，误差累积   │     │  问题：对模型容量和训练数据依赖大  │
│  优势：各模块可解释性强           │     │  优势：全局优化，泛化能力强        │
└─────────────────────────────────┘     └─────────────────────────────────┘
```

**端到端VLM导航的关键技术**：

1. **视觉编码器**：将图像编码为特征向量（如 ResNet/ViT/Swin Transformer）
2. **语言编码器**：将自然语言指令编码为语义向量（如 BERT/CLIP Text Encoder）
3. **跨模态融合**：对齐视觉与语言特征（如 Cross-Attention/Concat+MLP）
4. **动作预测头**：从融合特征预测下一步动作（如 MLP/Transformer Decoder）

### 1.4 技术选型

#### 1.4.1 VLM模型选择

| 模型 | 参数量 | 视觉语言能力 | 本地部署难度 | 推荐场景 |
|------|--------|-------------|-------------|---------|
| **GPT-4V** | ~1.8T（API） | 极强 | 需API | 云端高精度场景 |

## 3. 核心技术详解

### 3.1 VLM模型原理

#### 3.1.1 LLaVA架构详解

LLaVA（Large Language and Vision Assistant）是最经典的开源VLM之一，采用简化的视觉语言融合架构：

```python
# LLaVA 核心架构伪代码
class LLaVA(nn.Module):
    def __init__(self, vision_encoder, language_model, projector):
        self.vision_encoder = vision_encoder    # CLIP ViT
        self.language_model = language_model      # Vicuna/LLaMA
        self.projector = projector                 # 线性投影层
    
    def forward(self, images, input_ids, pixel_values):
        # 1. 视觉编码
        image_features = self.vision_encoder(pixel_values)
        image_features = self.projector(image_features)
        
        # 2. 文本嵌入
        text_embeddings = self.language_model.get_input_embeddings()(input_ids)
        
        # 3. 跨模态融合（直接拼接）
        # 图像特征与文本嵌入在 embedding 空间直接拼接
        # 序列：[TEXT] [IMAGE] [IMAGE] ... [IMAGE]
        inputs_embeds = torch.cat([text_embeddings, image_features], dim=1)
        
        # 4. LLM 自回归生成
        outputs = self.language_model(
            inputs_embeds=inputs_embeds,
            labels=labels
        )
        
        return outputs
```

#### 3.1.2 视觉语言对齐

VLM的核心挑战是将视觉特征空间与语言特征空间对齐。LLaVA通过**投影层（Projector）** 实现这一目标：

```
视觉编码器输出 (CLIP ViT):  shape = [B, 256, 1024]
        │
        ▼
   投影层 (MLP)
   Linear(1024 → 4096)
   GELU()
   Linear(4096 → 4096)
        │
        ▼
投影后视觉特征:            shape = [B, 256, 4096]
        │
        ▼
   与文本嵌入 [B, SeqLen, 4096] 拼接
        │
        ▼
   输入 LLM 进行生成
```

**为什么投影层有效？**
- CLIP ViT 是在大规模图文对数据上训练的，已具备良好的视觉-语言对齐能力
- 投影层将 CLIP 特征映射到 LLM 的 embedding 空间，实现跨模态语义对齐
- 训练时冻结视觉编码器，仅训练投影层和少量 LLM 层（LoRA），降低训练成本

#### 3.2.2 动作预测头的设计

动作预测头（Action Head）是VLM导航模型的最后一层，负责从融合特征预测动作概率：

```python
class ActionHead(nn.Module):
    """动作预测头：将VLM输出特征映射为动作概率分布"""
    
    def __init__(self, input_dim=4096, num_actions=4, hidden_dim=512):
        super().__init__()
        
        self.net = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Dropout(0.1),
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Dropout(0.1),
            nn.Linear(hidden_dim // 2, num_actions)
        )
    
    def forward(self, vlm_features):
        """
        Args:
            vlm_features: [B, D] VLM输出特征
        
        Returns:
            action_logits: [B, num_actions] 每个动作的未归一化分数
            action_probs: [B, num_actions] 每个动作的概率分布
        """
        action_logits = self.net(vlm_features)
        action_probs = F.softmax(action_logits, dim=-1)
        
        return action_logits, action_probs
```

#### 3.2.3 动作选择的贪心与采样策略

```python
def select_action(action_probs, strategy='greedy', temperature=1.0):
    """选择导航动作
    
    Args:
        action_probs: [num_actions] 动作概率分布
        strategy: 'greedy'（贪心）或 'sample'（采样）
        temperature: 采样温度，>1 更随机，<1 更确定
    
    Returns:
        selected_action: 选中的动作索引
    """
    if strategy == 'greedy':
        # 贪心策略：选择概率最高的动作
        return torch.argmax(action_probs).item()
    
    elif strategy == 'sample':
        # 采样策略：根据概率分布随机采样
        # 应用温度调整
        adjusted_probs = F.softmax(action_probs / temperature, dim=-1)
        return torch.multinomial(adjusted_probs, 1).item()
    
    elif strategy == 'topk':
        # Top-K 策略：限制在概率最高的K个动作中采样
        k = 2
        topk_probs, topk_indices = torch.topk(action_probs, k)
        topk_probs = F.softmax(topk_probs / temperature, dim=-1)
        selected_idx = torch.multinomial(topk_probs, 1).item()
        return topk_indices[selected_idx].item()
```

### 3.3 跨模态融合策略

#### 3.3.1 融合方式对比

| 融合方式 | 原理 | 优点 | 缺点 | 适用场景 |
|---------|------|------|------|---------|
| **早期融合** | 图像与文本 embedding 直接拼接 | 简单，计算量小 | 可能丢失模态特异性 | LLaVA |
| **晚期融合** | 各模态独立编码后再融合 | 保留模态特性 | 融合能力有限 | CLIP |
| **跨注意力融合** | Cross-Attention 交叉融合 | 强大的模态交互 | 计算量大 | Flamingo |
| **门控融合** | 门控机制动态加权模态 | 可解释性强 | 训练复杂 | |

**本项目采用早期融合（Concat + MLP）**，因其实现简单、效果好、LLaVA验证充分。

        try:
            # 优先尝试本地 LLaVA 模型
            from llava.model import LlavaForCausalLM
            from llava.mm_utils import tokenizer_image_token
            from transformers import AutoProcessor, AutoTokenizer

            self.get_logger().info(f'正在加载 LLaVA 模型: {self.vlm_model_path}')

            self.processor = AutoProcessor.from_pretrained(
                self.vlm_model_path,

                self.vlm_model_path,
                trust_remote_code=True

                self.vlm_model_path,

    def _on_image_received(self, msg: Image):

            # 将 ROS Image 消息转换为 OpenCV 图像
            cv_image = self.bridge.imgmsg_to_cv2(msg, desired_encoding='bgr8')
            
            # 图像预处理：缩放到合适大小
            cv_image = cv2.resize(cv_image, (336, 336))
            
            # 维护图像缓冲区（保留最近 N 帧）
            self.current_images.append(cv_image)
            if len(self.current_images) > self.image_top_k:
                self.current_images.pop(0)
            
        except Exception as e:
            self.get_logger().error(f'图像接收处理失败: {str(e)}')
    
    def _on_depth_received(self, msg: Image):
        """接收深度图像（可选）"""
        # 深度图像处理逻辑，可用于障碍物检测
        pass
    
    def _on_instruction_received(self, msg: String):
        """接收用户导航指令"""
        instruction = msg.data.strip()
        if not instruction:
            return
        
        self.current_instruction = instruction
        self.is_navigating = True
        self.action_history.clear()  # 新指令，清空历史
        
        self.get_logger().info(f'收到导航指令: "{instruction}"')
        
        # 发布状态反馈
        self._publish_status('instruction_received', {'instruction': instruction})
    
    def _on_odom_received(self, msg: Odometry):
        """接收里程计数据"""
        # 可用于记录机器人的移动轨迹
        self.current_position = {
            'x': msg.pose.pose.position.x,
            'y': msg.pose.pose.position.y,
            'theta': 2.0 * msg.pose.pose.orientation.z
        }
    
    def _on_nav_enable_received(self, msg: Bool):
        """接收导航使能信号"""
        self.is_navigating = msg.data
        status = '启动导航' if msg.data else '暂停导航'
        self.get_logger().info(status)
        self._publish_status('nav_enable_changed', {'enabled': msg.data})
    
    def _navigation_step(self):
        """导航步骤主循环（定时器触发）"""
        if not self.is_navigating:
            return
        
        if not self.current_instruction:
            self.get_logger().warn_throttle(5.0, '无导航指令，等待中...')
            return
        
        if len(self.current_images) == 0:
            self.get_logger().warn_throttle(5.0, '无视觉图像，等待中...')
            return
        
        # 构建 VLM 输入
        prompt = self._build_prompt()
        image = self.current_images[-1]  # 使用最新一帧
        
        # 调用 VLM 推理
        action = self._predict_action(prompt, image)
        
        if action is None:
            self.get_logger().warn('VLM 推理失败，跳过本轮')
            return
        
        # 执行动作
        self._execute_action(action)
        
        # 记录动作历史
        self.action_history.append({
            'action': action,
            'timestamp': time.time(),
            'instruction': self.current_instruction
        })
        
        # 限制历史长度
        if len(self.action_history) > self.history_length:
            self.action_history.pop(0)
        
        # 发布动作
        self._publish_action(action)
    
    def _build_prompt(self) -> str:
        """构建 VLM 推理提示词"""
        base_prompt = f"""你是一个室内机器人导航助手。

你可执行的动作有：
- forward: 前进（约0.25米）
- turn_left: 左转（约15度）
- turn_right: 右转（约15度）
- stop: 停止（已到达目标位置）

用户指令：{self.current_instruction}"""
        
        # 添加历史上下文
        if self.enable_history and len(self.action_history) > 0:
            history_str = '\n\n最近的动作历史：\n'
            for i, entry in enumerate(self.action_history[-3:]):  # 最近3个动作
                history_str += f"- {entry['action']}\n"
            base_prompt += history_str
            base_prompt += "\n根据历史动作和当前视觉，观察机器人是否需要调整方向。"
        
        base_prompt += "\n\n仔细观察图像，输出一个动作词（仅输出：forward / turn_left / turn_right / stop）："
        
        return base_prompt
    
    def _predict_action(self, prompt: str, image: np.ndarray) -> Optional[str]:
        """调用 VLM 预测动作
        
        Args:
            prompt: 提示词
            image: 当前图像 (H, W, 3)
        
        Returns:
            预测的动作名称，或 None（推理失败）
        """
        if self.vlm_client == 'llava':
            return self._predict_action_llava(prompt, image)
        else:
            return self._predict_action_mock(prompt, image)
    
    def _predict_action_llava(self, prompt: str, image: np.ndarray) -> Optional[str]:
        """使用 LLaVA 模型预测动作"""
        try:
            import torch
            from llava.mm_utils import tokenizer_image_token
            from llava.constants import IMAGE_TOKEN, DEFAULT_IMAGE_TOKEN
            
            # 准备输入：图像 + 文本
            # LLaVA 使用 IMAGE_TOKEN 占位符表示图像位置
            conversation = [
                {
                    'from': 'human',
                    'value': f'{DEFAULT_IMAGE_TOKEN}\n{prompt}'
                },
                {
                    'from': 'gpt',
                    'value': ''
                }
            ]
            
            # 使用 processor 处理输入
            text_prompt = f'{DEFAULT_IMAGE_TOKEN}\n{prompt}'
            inputs = self.processor(
                text=text_prompt,
                images=image,
                return_tensors='pt'
            ).to(self.vlm_device)
            
            # VLM 推理
            with torch.no_grad():
                output_ids = self.vlm_model.generate(
                    **inputs,
                    max_new_tokens=20,
                    do_sample=True,
                    temperature=self.temperature
                )
            
            # 解码输出
            outputs = self.processor.batch_decode(
                output_ids, skip_special_tokens=True
            )[0]
            
            # 解析动作
            action = self._parse_action_from_output(outputs)
            return action
        
        except Exception as e:
            self.get_logger().error(f'LLaVA 推理失败: {str(e)}')
            return None
    
    def _predict_action_mock(self, prompt: str, image: np.ndarray) -> Optional[str]:
        """模拟 VLM 推理（用于演示和测试）
        
        模拟策略：根据指令关键词简单判断动作
        """
        instruction = self.current_instruction.lower()
        
        # 简单规则判断
        if '停' in instruction or '到达' in instruction or '到了' in instruction:
            return 'stop'
        elif '左转' in instruction or '左拐' in instruction:
            return 'turn_left'
        elif '右转' in instruction or '右拐' in instruction:
            return 'turn_right'
        else:
            # 随机选择前进或其他动作
            import random
            actions = ['forward', 'forward', 'forward', 'turn_left', 'turn_right']
            return random.choice(actions)
    
    def _parse_action_from_output(self, output: str) -> Optional[str]:
        """从 VLM 输出中解析动作词
        
        Args:
            output: VLM 原始输出文本
        
        Returns:
            解析到的动作名称，或 None
        """
        valid_actions = ['forward', 'turn_left', 'turn_right', 'stop']
        
        output_lower = output.lower().strip()
        
        for action in valid_actions:
            if action in output_lower:
                return action
        
        # 如果无法解析，默认前进
        self.get_logger().warn(f'无法解析 VLM 输出: "{output}"，默认前进')
        return 'forward'
    
    def _execute_action(self, action: str):
        """执行导航动作
        
        Args:
            action: 动作名称
        """
        twist = Twist()
        
        # 动作参数配置
        action_params = {
            'forward': {'linear': 0.2, 'angular': 0.0},
            'turn_left': {'linear': 0.0, 'angular': 0.5},
            'turn_right': {'linear': 0.0, 'angular': -0.5},
            'stop': {'linear': 0.0, 'angular': 0.0}
        }
        
        params = action_params.get(action, action_params['forward'])
        twist.linear.x = params['linear']
        twist.angular.z = params['angular']
        
        # 发布速度命令
        self.cmd_vel_publisher.publish(twist)
        
        self.get_logger().info(f'执行动作: {action} (linear={params["linear"]}, angular={params["angular"]})')
        
        # 如果是停止动作，关闭导航状态
        if action == 'stop':
            self.is_navigating = False
            self._publish_status('navigation_completed', {
                'total_actions': len(self.action_history),
                'instruction': self.current_instruction
            })
    
    def _publish_action(self, action: str):
        """发布动作到 ROS2 topic"""
        msg = String()
        msg.data = json.dumps({
            'action': action,
            'timestamp': time.time(),
            'instruction': self.current_instruction
        })
        self.action_publisher.publish(msg)
    
    def _publish_status(self, status: str, data: Dict):
        """发布状态信息"""
        msg = String()
        msg.data = json.dumps({
            'status': status,
            'data': data,
            'timestamp': time.time()
        })
        self.status_publisher.publish(msg)
    
    def destroy_node(self):
        """清理资源"""
        # 停止机器人
        twist = Twist()
        twist.linear.x = 0.0
        twist.angular.z = 0.0
        self.cmd_vel_publisher.publish(twist)
        super().destroy_node()

```python
        if not instruction:
            return
        
        self.current_instruction = instruction
        self.is_navigating = True
        self.action_history.clear()
        self.get_logger().info(f'收到导航指令: "{instruction}"')
    
    def _on_nav_enable_received(self, msg: Bool):
        """接收导航使能信号"""
        self.is_navigating = msg.data
        status = '启动导航' if msg.data else '暂停导航'
        self.get_logger().info(status)
    
    def _navigation_step(self):
        """导航步骤主循环"""
        if not self.is_navigating or not self.current_instruction:
            return
        
        if len(self.current_images) == 0:
            return
        
        prompt = self._build_prompt()
        image = self.current_images[-1]
        action = self._predict_action(prompt, image)
        
        if action is None:
            return
        
        self._execute_action(action)
        self.action_history.append({'action': action, 'timestamp': time.time()})
        
        if len(self.action_history) > self.history_length:
            self.action_history.pop(0)
        
        self._publish_action(action)
    
    def _build_prompt(self) -> str:
        """构建 VLM 推理提示词"""
        base_prompt = f"""你是一个室内机器人导航助手。

你可执行的动作有：
- forward: 前进（约0.25米）
- turn_left: 左转（约15度）
- turn_right: 右转（约15度）
- stop: 停止（已到达目标位置）

用户指令：{self.current_instruction}"""
        
        if self.enable_history and len(self.action_history) > 0:
            history_str = '\n\n最近的动作历史：\n'
            for entry in self.action_history[-3:]:
                history_str += f"- {entry['action']}\n"
            base_prompt += history_str
            base_prompt += "\n根据历史动作和当前视觉，观察机器人是否需要调整方向。"
        
        base_prompt += "\n\n仔细观察图像，输出一个动作词（仅输出：forward / turn_left / turn_right / stop）："
        return base_prompt
    
    def _predict_action(self, prompt: str, image: np.ndarray) -> Optional[str]:
        """调用 VLM 预测动作"""
        if self.vlm_client == 'llava':
            return self._predict_action_llava(prompt, image)
        else:
            return self._predict_action_mock(prompt, image)
    
    def _predict_action_llava(self, prompt: str, image: np.ndarray) -> Optional[str]:
        """使用 LLaVA 模型预测动作"""
        try:
            import torch
            from llava.constants import DEFAULT_IMAGE_TOKEN
            
            text_prompt = f'{DEFAULT_IMAGE_TOKEN}\n{prompt}'
            inputs = self.processor(text=text_prompt, images=image, return_tensors='pt').to(self.vlm_device)
            
            with torch.no_grad():
                output_ids = self.vlm_model.generate(
                    **inputs, max_new_tokens=20, do_sample=True, temperature=self.temperature
                )
            
            outputs = self.processor.batch_decode(output_ids, skip_special_tokens=True)[0]
            return self._parse_action_from_output(outputs)
        
        except Exception as e:
            self.get_logger().error(f'LLaVA 推理失败: {str(e)}')
            return None
    
    def _predict_action_mock(self, prompt: str, image: np.ndarray) -> Optional[str]:
        """模拟 VLM 推理（用于演示）"""
        instruction = self.current_instruction.lower()
        
        if '停' in instruction or '到达' in instruction or '到了' in instruction:
            return 'stop'
        elif '左转' in instruction or '左拐' in instruction:
            return 'turn_left'
        elif '右转' in instruction or '右拐' in instruction:
            return 'turn_right'
        else:
            import random
            actions = ['forward', 'forward', 'forward', 'turn_left', 'turn_right']
            return random.choice(actions)
    
    def _parse_action_from_output(self, output: str) -> Optional[str]:
        """从 VLM 输出中解析动作词"""
        valid_actions = ['forward', 'turn_left', 'turn_right', 'stop']
        output_lower = output.lower().strip()
        
        for action in valid_actions:
            if action in output_lower:
                return action
        
        self.get_logger().warn(f'无法解析 VLM 输出: "{output}"，默认前进')
        return 'forward'
    
    def _execute_action(self, action: str):
        """执行导航动作"""
        twist = Twist()
        
        action_params = {
            'forward': {'linear': 0.2, 'angular': 0.0},
            'turn_left': {'linear': 0.0, 'angular': 0.5},
            'turn_right': {'linear': 0.0, 'angular': -0.5},
            'stop': {'linear': 0.0, 'angular': 0.0}
        }
        
        params = action_params.get(action, action_params['forward'])
        twist.linear.x = params['linear']
        twist.angular.z = params['angular']
        
        self.cmd_vel_publisher.publish(twist)
        self.get_logger().info(f'执行动作: {action}')
        
        if action == 'stop':
            self.is_navigating = False
    
    def _publish_action(self, action: str):
        """发布动作到 ROS2 topic"""
        msg = String()
        msg.data = json.dumps({
            'action': action,
            'timestamp': time.time(),
            'instruction': self.current_instruction
        })
        self.action_publisher.publish(msg)
    
    def destroy_node(self):
        """清理资源"""
        twist = Twist()
        twist.linear.x = 0.0
        twist.angular.z = 0.0
        self.cmd_vel_publisher.publish(twist)
        super().destroy_node()
```

### 5.2 指令解析模块

指令解析模块将自然语言导航指令拆解为可执行的语义单元：

```python
# instruction_parser.py
"""
指令解析模块：将自然语言导航指令解析为结构化语义
支持：地点识别、物体识别、方向理解、关系理解
"""

import re
from typing import Dict, List, Optional, Tuple
import json


class InstructionParser:
    """自然语言导航指令解析器"""
    
    def __init__(self):
        # 可识别的地点关键词
        self.room_keywords = {
            '厨房': ['厨房', '灶台', '做饭'],
            '客厅': ['客厅', '沙发', '电视'],
            '卧室': ['卧室', '床', '睡觉'],
            '洗手间': ['洗手间', '厕所', '卫生间', '浴室'],
            '阳台': ['阳台', '晒台'],
            '门口': ['门口', '玄关', '大门', '入口'],
            '餐厅': ['餐厅', '餐桌', '饭桌']
        }
        
        # 可识别的物体关键词
        self.object_keywords = {
            '水杯': ['水杯', '杯子', '马克杯', '茶杯'],
            '手机': ['手机', '电话', 'iphone'],
            '钥匙': ['钥匙', '钥匙扣'],
            '书本': ['书', '书本', '书籍'],
            '遥控器': ['遥控器', '遥控', '电视遥控']
        }
        
        # 方向关键词
        self.direction_keywords = {
            '左转': ['左转', '左拐', '向左', '左'],
            '右转': ['右转', '右拐', '向右', '右'],
            '直走': ['直走', '直行', '往前', '前进', '向前'],
            '后退': ['后退', '后退走', '倒车'],
            '停下': ['停下', '停止', '停', '到达', '到了']
        }
        
        # 关系关键词
        self.relation_keywords = {
            '旁边': ['旁边', '附近', '靠'],
            '前面': ['前面', '前方', '前'],
            '后面': ['后面', '后方', '后'],
            '对面': ['对面', '正对面', '另一边']
        }
    
    def parse(self, instruction: str) -> Dict:
        """解析导航指令
        
        Args:
            instruction: 自然语言导航指令
        
        Returns:
            结构化解析结果
        """
        instruction = instruction.strip()
        
        result = {
            'raw_instruction': instruction,
            'target_room': self._extract_room(instruction),
            'target_object': self._extract_object(instruction),
            'directions': self._extract_directions(instruction),
            'relations': self._extract_relations(instruction),
            'stop_condition': self._extract_stop_condition(instruction),
            'confidence': 0.0
        }
        
        # 计算解析置信度
        result['confidence'] = self._compute_confidence(result)
        
        return result
    
    def _extract_room(self, text: str) -> Optional[str]:
        """提取目标房间"""
        for room, keywords in self.room_keywords.items():
            for kw in keywords:
                if kw in text:
                    return room
        return None
    
    def _extract_object(self, text: str) -> Optional[str]:
        """提取目标物体"""
        for obj, keywords in self.object_keywords.items():
            for kw in keywords:
                if kw in text:
                    return obj
        return None
    
    def _extract_directions(self, text: str) -> List[str]:
        """提取方向指令"""
        directions = []
        for direction, keywords in self.direction_keywords.items():
            for kw in keywords:
                if kw in text:
                    if direction not in directions:
                        directions.append(direction)
                    break
        return directions
    
    def _extract_relations(self, text: str) -> List[Dict]:
        """提取空间关系"""
        relations = []
        for relation, keywords in self.relation_keywords.items():
            for kw in keywords:
                if kw in text:
                    relations.append({'type': relation, 'keyword': kw})
                    break
        return relations
    
    def _extract_stop_condition(self, text: str) -> Optional[str]:
        """提取停止条件"""
        stop_keywords = ['停下', '停止', '停', '到达', '到了', '之后']
        for kw in stop_keywords:
            if kw in text:
                return kw
        return None
    
    def _compute_confidence(self, parsed: Dict) -> float:
        """计算解析置信度"""
        score = 0.0
        
        if parsed['target_room']:
            score += 0.3
        if parsed['target_object']:
            score += 0.2
        if parsed['directions']:
            score += 0.2 * len(parsed['directions'])
        if parsed['relations']:
            score += 0.15 * len(parsed['relations'])
        if parsed['stop_condition']:
            score += 0.15
        
        return min(score, 1.0)
    
    def to_navigation_goal(self, parsed: Dict) -> Dict:
        """将解析结果转换为导航目标描述
        
        Args:
            parsed: parse() 的输出
        
        Returns:
            导航目标描述（用于构建 VLM 提示词）
        """
        goal_parts = []
        
        if parsed['target_room']:
            goal_parts.append(f"目标地点：{parsed['target_room']}")
        
        if parsed['target_object']:
            goal_parts.append(f"目标物体：{parsed['target_object']}")
        
        if parsed['relations']:
            rel_str = '、'.join([r['type'] for r in parsed['relations']])
            goal_parts.append(f"空间关系：{rel_str}")
        
        if parsed['directions']:
            dir_str = ' → '.join(parsed['directions'])
            goal_parts.append(f"移动方向：{dir_str}")
        
        return {
            'goal_description': '；'.join(goal_parts),
            'stop_on_find_object': parsed['target_object'] is not None,
            'raw_instruction': parsed['raw_instruction']
        }
```

### 5.3 视觉语言融合核心实现

```python
# vlm_fusion.py
"""
视觉语言融合模块：实现图像与文本特征的深度融合
核心策略：早期融合（Early Fusion）— 投影后拼接
"""

import torch
import torch.nn as nn
import torch.nn.functional as F
from typing import Optional, Tuple


class VisionLanguageFusion(nn.Module):
    """视觉语言融合模块
    
    采用早期融合策略：
    1. 视觉编码器（CLIP ViT）提取图像特征
    2. 文本编码器（LLM Embedding）提取文本特征
    3. 通过投影层将视觉特征映射到文本空间
    4. 直接拼接后送入 LLM
    """
    
    def __init__(
        self,
        vision_hidden_size: int = 1024,    # CLIP ViT 隐藏层维度
        text_hidden_size: int = 4096,      # LLM 隐藏层维度
        fusion_hidden_size: int = 4096,    # 融合层维度
        dropout: float = 0.1
    ):
        super().__init__()
        
        # 视觉投影层：CLIP 特征 → LLM 文本空间
        self.vision_projector = nn.Sequential(
            nn.Linear(vision_hidden_size, fusion_hidden_size),
            nn.GELU(),
            nn.Dropout(dropout),
            nn.Linear(fusion_hidden_size, fusion_hidden_size)
        )
        
        # 可选：视觉-文本交叉注意力层（增强融合效果）
        self.cross_attention = CrossAttentionLayer(
            query_dim=text_hidden_size,
            key_dim=fusion_hidden_size,
            value_dim=fusion_hidden_size,
            num_heads=8,
            dropout=dropout
        )
        
        # 模态嵌入（用于区分视觉和文本 tokens）
        self.vision_token_embed = nn.Parameter(torch.randn(1, 1, fusion_hidden_size))
        self.text_token_embed = nn.Parameter(torch.randn(1, 1, fusion_hidden_size))
    
    def forward(
        self,
        vision_features: torch.Tensor,    # [B, N_vision, D_vision]
        text_embeddings: torch.Tensor      # [B, N_text, D_text]
    ) -> torch.Tensor:
        """
        融合视觉和文本特征
        
        Args:
            vision_features: CLIP 提取的视觉特征
            text_embeddings: LLM 的文本嵌入
        
        Returns:
            fused_features: 融合后的特征 [B, N_fused, D_fusion]
        """
        # Step 1: 视觉特征投影到 LLM 空间
        projected_vision = self.vision_projector(vision_features)
        
        # Step 2: 添加视觉 token 类型嵌入
        vision_type_embed = self.vision_token_embed.expand_as(projected_vision)
        projected_vision = projected_vision + vision_type_embed
        
        # Step 3: 添加文本 token 类型嵌入
        text_type_embed = self.text_token_embed.expand_as(text_embeddings)
        text_embeddings = text_embeddings + text_type_embed
        
        # Step 4: 跨模态注意力交互（可选）
        # 将文本作为 query，视觉作为 key/value
        if hasattr(self, 'cross_attention'):
            # 调整维度以适配 cross attention
            projected_vision_adapted = nn.Linear(
                projected_vision.shape[-1], text_embeddings.shape[-1]
            ).to(projected_vision.device)(projected_vision)
            
            fused = self.cross_attention(
                query=text_embeddings,
                key=projected_vision_adapted,
                value=projected_vision_adapted
            )
        else:
            fused = text_embeddings
        
        # Step 5: 直接拼接（早期融合）
        # 顺序：[文本 tokens..., 视觉 tokens...]
        fused_features = torch.cat([fused, projected_vision], dim=1)
        
        return fused_features


class CrossAttentionLayer(nn.Module):
    """跨模态交叉注意力层"""
    
    def __init__(self, query_dim: int, key_dim: int, value_dim: int,
                 num_heads: int = 8, dropout: float = 0.1):
        super().__init__()
        
        self.num_heads = num_heads
        self.head_dim = query_dim // num_heads
        self.scale = self.head_dim ** -0.5
        
        self.q_proj = nn.Linear(query_dim, query_dim)
        self.k_proj = nn.Linear(key_dim, query_dim)
        self.v_proj = nn.Linear(value_dim, query_dim)
        self.out_proj = nn.Linear(query_dim, query_dim)
        
        self.dropout = nn.Dropout(dropout)
    
    def forward(
        self,
        query: torch.Tensor,    # [B, N_q, D_q]
        key: torch.Tensor,       # [B, N_k, D_k]
        value: torch.Tensor       # [B, N_v, D_v]
    ) -> torch.Tensor:
        """跨注意力前向传播"""
        B = query.size(0)
        
        # 投影
        q = self.q_proj(query)
        k = self.k_proj(key)
        v = self.v_proj(value)
        
        # Reshape 为多头形式
        q = q.view(B, -1, self.num_heads, self.head_dim).transpose(1, 2)
        k = k.view(B, -1, self.num_heads, self.head_dim).transpose(1, 2)
        v = v.view(B, -1, self.num_heads, self.head_dim).transpose(1, 2)
        
        # 计算注意力
        attn = (q @ k.transpose(-2, -1)) * self.scale
        attn = F.softmax(attn, dim=-1)
        attn = self.dropout(attn)
        
        # 应用注意力到 value
        out = (attn @ v).transpose(1, 2).contiguous().view(B, -1, self.num_heads * self.head_dim)
        out = self.out_proj(out)
        
        return out


class ActionPredictionNetwork(nn.Module):
    """动作预测网络：基于融合特征预测导航动作"""
    
    def __init__(
        self,
        input_dim: int = 4096,
        num_actions: int = 4,
        hidden_dim: int = 512,
        dropout: float = 0.1
    ):
        super().__init__()
        
        self.num_actions = num_actions
        
        # 动作预测MLP
        self.action_head = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.LayerNorm(hidden_dim),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(hidden_dim // 2, num_actions)
        )
        
        # 动作置信度预测（用于不确定性估计）
        self.confidence_head = nn.Sequential(
            nn.Linear(input_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, 1),
            nn.Sigmoid()
        )
    
    def forward(self, fused_features: torch.Tensor) -> Tuple[torch.Tensor, torch.Tensor]:
        """
        预测动作概率分布
        
        Args:
            fused_features: [B, D] 融合后的全局特征
        
        Returns:
            action_logits: [B, num_actions] 动作未归一化分数
            action_probs: [B, num_actions] 动作概率分布
            confidence: [B, 1] 动作置信度
        """
        action_logits = self.action_head(fused_features)
        action_probs = F.softmax(action_logits, dim=-1)
        confidence = self.confidence_head(fused_features)
        
        return action_logits, action_probs, confidence
```

## 6. 导航策略与动作预测

### 6.1 导航策略设计

VLM导航的策略设计需要考虑安全性、效率和可解释性：

```python
# navigation_policy.py
"""
导航策略管理器：负责动作的安全性检查、历史记忆、策略选择
"""

from typing import List, Dict, Optional, Tuple
import numpy as np
import json


class NavigationPolicyManager:
    """导航策略管理器"""
    
    def __init__(self, safety_threshold: float = 0.3):
        """
        Args:
            safety_threshold: 安全障碍物距离阈值（米）
        """
        self.safety_threshold = safety_threshold
        
        # 动作历史
        self.action_history: List[Dict] = []
        self.max_history = 10
        
        # 连续同方向旋转检测
        self.consecutive_turns = 0
        self.max_consecutive_turns = 5
        
        # 导航状态
        self.state = 'idle'  # idle / navigating / paused / completed
    
    def select_action(
        self,
        vlm_action: str,
        vlm_confidence: float,
        depth_image: Optional[np.ndarray] = None,
        collision_risk: bool = False
    ) -> str:
        """
        选择最终执行动作（带安全检查）
        
        Args:
            vlm_action: VLM 预测的动作
            vlm_confidence: VLM 预测置信度
            depth_image: 深度图像（用于障碍物检测）
            collision_risk: 是否检测到碰撞风险
        
        Returns:
            最终选择的动作
        """
        final_action = vlm_action
        
        # 安全检查1：碰撞风险检测
        if collision_risk and vlm_action == 'forward':
            self.state = 'paused'
            final_action = 'stop'
            self.state = 'navigating'
            return final_action
        
        # 安全检查2：深度图像障碍物检测
        if depth_image is not None and vlm_action == 'forward':
            min_depth = np.min(depth_image)
            if min_depth < self.safety_threshold:
                final_action = 'stop'
                return final_action
        
        # 安全检查3：连续旋转检测（防止卡死）
        if vlm_action in ['turn_left', 'turn_right']:
            self.consecutive_turns += 1
            if self.consecutive_turns >= self.max_consecutive_turns:
                # 强制插入前进，打破旋转循环
                final_action = 'forward'
                self.consecutive_turns = 0
        else:
            self.consecutive_turns = 0
        
        # 安全检查4：低置信度时增加保守策略
        if vlm_confidence < 0.4 and vlm_action == 'forward':
            # 低置信度前进时，缩短步长（此处简化为直接停止）
            final_action = 'stop'
        
        # 安全检查5：历史循环检测
        if self._detect_loop():
            final_action = 'stop'
            self.state = 'paused'
        
        # 记录动作
        self.action_history.append({
            'action': final_action,
            'vlm_action': vlm_action,
            'confidence': vlm_confidence,
            'timestamp': None  # 实际应用中填入时间戳
        })
        
        if len(self.action_history) > self.max_history:
            self.action_history.pop(0)
        
        return final_action
    
    def _detect_loop(self) -> bool:
        """检测是否陷入导航循环"""
        if len(self.action_history) < 6:
            return False
        
        # 检测最近的动作序列是否形成循环
        recent = [h['action'] for h in self.action_history[-6:]]
        pattern_1 = ['turn_left', 'forward', 'turn_right', 'forward', 'turn_left', 'forward']
        pattern_2 = ['turn_right', 'forward', 'turn_left', 'forward', 'turn_right', 'forward']
        
        return recent == pattern_1 or recent == pattern_2
    
    def get_navigation_status(self) -> Dict:
        """获取导航状态摘要"""
        return {
            'state': self.state,
            'total_actions': len(self.action_history),
            'consecutive_turns': self.consecutive_turns,
            'recent_actions': [h['action'] for h in self.action_history[-3:]]
        }
    
    def reset(self):
        """重置策略状态"""
        self.action_history.clear()
        self.consecutive_turns = 0
        self.state = 'idle'
```

```python
"""
导航动作序列生成器
基于 VLM 输出生成可执行的动作序列
"""

from typing import List, Dict, Optional, Tuple
import json


class ActionSequenceGenerator:
    """导航动作序列生成器"""
    
    # 动作空间定义
    ACTION_SPACE = {
        'forward': {
            'distance': 0.25,
            'linear_vel': 0.2,
            'angular_vel': 0.0,
            'duration': 1.25
        },
        'turn_left': {
            'angle': 15.0,
            'linear_vel': 0.0,
            'angular_vel': 0.26,  # ~15度
            'duration': 1.0
        },
        'turn_right': {
            'angle': -15.0,
            'linear_vel': 0.0,
            'angular_vel': -0.26,
            'duration': 1.0
        },
        'stop': {
            'distance': 0.0,
            'linear_vel': 0.0,
            'angular_vel': 0.0,
            'duration': 0.0
        }
    }
    
    def __init__(self):
        self.current_sequence: List[Dict] = []
        self.execution_index = 0
    
    def generate_from_vlm_response(
        self,
        vlm_output: str,
        instruction: str,
        context: Optional[Dict] = None
    ) -> List[Dict]:
        """
        从 VLM 输出生成动作序列
        
        Args:
            vlm_output: VLM 原始输出
            instruction: 用户指令
            context: 可选的上下文信息
        
        Returns:
            动作序列列表
        """
        # 解析 VLM 输出中的动作
        action = self._parse_action(vlm_output)
        
        # 生成动作参数
        action_params = self._build_action_params(action)
        
        # 添加到序列
        self.current_sequence = [action_params]
        self.execution_index = 0
        
        return self.current_sequence
    
    def _parse_action(self, vlm_output: str) -> str:
        """从 VLM 输解析动作"""
        valid_actions = list(self.ACTION_SPACE.keys())
        output_lower = vlm_output.lower().strip()
        
        for action in valid_actions:
            if action in output_lower:
                return action
        
        return 'forward'  # 默认前进
    
    def _build_action_params(self, action: str) -> Dict:
        """构建动作参数"""
        base_params = self.ACTION_SPACE.get(action, self.ACTION_SPACE['forward'])
        
        return {
            'action_name': action,
            'linear_vel': base_params['linear_vel'],
            'angular_vel': base_params['angular_vel'],
            'duration': base_params['duration'],
            'description': self._get_action_description(action)
        }
    
    def _get_action_description(self, action: str) -> str:
        """获取动作描述"""
        descriptions = {
            'forward': '机器人向前移动约0.25米',
            'turn_left': '机器人原地左转约15度',
            'turn_right': '机器人原地右转约15度',
            'stop': '机器人停止当前动作'
        }
        return descriptions.get(action, '未知动作')
    
    def get_next_action(self) -> Optional[Dict]:
        """获取下一个待执行的动作"""
        if self.execution_index < len(self.current_sequence):
            action = self.current_sequence[self.execution_index]
            self.execution_index += 1
            return action
        return None
    
    def is_sequence_complete(self) -> bool:
        """判断动作序列是否执行完成"""
        return self.execution_index >= len(self.current_sequence)
    
    def reset(self):
        """重置序列生成器"""
        self.current_sequence.clear()
        self.execution_index = 0


# 动作执行器
class ActionExecutor:
    """将动作参数转换为 ROS2 速度命令并执行"""
    
    def __init__(self, cmd_vel_publisher):
        self.cmd_vel_publisher = cmd_vel_publisher
        self.is_executing = False
    
    def execute_action(self, action_params: Dict) -> bool:
        """
        执行单个动作
        
        Args:
            action_params: 动作参数字典
        
        Returns:
            执行是否成功
        """
        from geometry_msgs.msg import Twist
        import time
        
        twist = Twist()
        twist.linear.x = action_params['linear_vel']
        twist.angular.z = action_params['angular_vel']
        
        duration = action_params['duration']
        
        # 发布速度命令
        self.cmd_vel_publisher.publish(twist)
        self.is_executing = True
        
        # 等待动作完成
        if duration > 0:
            time.sleep(duration)
        
        # 停止
        twist.linear.x = 0.0
        twist.angular.z = 0.0
        self.cmd_vel_publisher.publish(twist)
        self.is_executing = False
        
        return True
```

```python
class ActionHead(nn.Module):
    """动作预测头"""
    
    def __init__(self, input_dim=4096, num_actions=4):
        super().__init__()
        self.fc = nn.Linear(input_dim, num_actions)
        self.num_actions = num_actions
    
    def forward(self, vlm_features, temperature=1.0):
        """预测动作概率分布"""
        action_logits = self.fc(vlm_features)
        
        # 温度调节：temperature > 1 使分布更平滑（更随机）
        # temperature < 1 使分布更尖锐（更确定）
        action_probs = F.softmax(action_logits / temperature, dim=-1)
        
        return action_logits, action_probs
    
    def greedy_select(self, action_probs):
        """贪心选择：选择概率最高的动作"""
        # 返回概率最高的动作索引
        return torch.argmax(action_probs, dim=-1)
    
    def stochastic_select(self, action_probs, temperature=1.0):
        """随机采样：根据概率分布随机选择动作"""
        # 使用调整后的温度重新计算概率
        adjusted_probs = F.softmax(
            torch.log(action_probs + 1e-8) / temperature, dim=-1
        )
        # 按概率分布采样
        return torch.multinomial(adjusted_probs, 1).squeeze(-1)
```

```python
        self.objects = ['水杯', '手机', '钥匙', '书本', '遥控器', '钱包']
        self.directions = {
            'forward': ['前', '直', '进'],
            'left': ['左', '左转', '左拐'],
            'right': ['右', '右转', '右拐'],
            'stop': ['停', '停下', '停止', '到了', '到达']
        }
    
    def parse(self, instruction: str) -> dict:
        """解析导航指令"""
        result = {
            'raw': instruction,
            'room': None,
            'object': None,
            'direction': None
        }
        
        # 识别目标房间
        for room in self.rooms:
            if room in instruction:
                result['room'] = room
                break
        
        # 识别目标物体
        for obj in self.objects:
            if obj in instruction:
                result['object'] = obj
                break
        
        # 识别方向
        for direction, keywords in self.directions.items():
            for kw in keywords:
                if kw in instruction:
                    result['direction'] = direction
                    return result  # 找到即返回
        
        return result
    
    def to_goal(self, parsed: dict) -> str:
        """转换为导航目标描述"""
        parts = []
        if parsed['room']:
            parts.append(f"去{parsed['room']}")
        if parsed['object']:
            parts.append(f"拿{parsed['object']}")
        if parsed['direction']:
            parts.append(parsed['direction'])
        return ' '.join(parts) if parts else parsed['raw']


# 使用示例
if __name__ == '__main__':
    parser = SimpleInstructionParser()
    
    test_cases = [
        "去厨房",
        "去客厅拿水杯",
        "前进左转停下",
        "去卧室找我的手机"
    ]
    
    for instr in test_cases:
        result = parser.parse(instr)
        goal = parser.to_goal(result)
        print(f'指令: {instr}')
        print(f'解析: {result}')
        print(f'目标: {goal}')
        print()
```
