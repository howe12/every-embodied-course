## 1. RH20T 概述

### 1.1 什么是 RH20T

RH20T（Robotic Human-Object Interaction with 20 Tasks）是一个大规模机器人遥操作数据集，由华盛顿大学（University of Washington）机器人研究实验室于 2021 年发布。该数据集通过遥操作方式采集人类控制机器人的演示数据，涵盖了 20 种日常操作任务，是目前最全面的多任务机器人操作数据集之一。

RH20T 的核心目标是**解决机器人模仿学习中的数据稀缺问题**。传统机器人编程需要工程师手动编写控制逻辑，而 RH20T 提供了大量人类演示数据，让机器人能够从示例中学习操作技能，显著降低了机器人编程的门槛。

### 1.2 数据规模

RH20T 的数据规模在发布时是同类数据集的数倍，具体包括：

| 指标 | 数值 |
|------|------|
| 任务数量 | 20+ 种操作任务 |
| 演示片段 | 11,000+ 条 |
| 总时长 | 40,000+ 秒（约 11 小时） |
| 机器人类型 | WidowX 机械臂 + KUKa IIWA |
| 视角数量 | 3 个 RGB-D 相机视角 |
| 传感器通道 | 7 通道（RGB、Depth、关节角、末端位姿、力矩等） |

### 1.3 任务覆盖

RH20T 涵盖的任务类型非常丰富，包括：

- **拾取放置（Pick & Place）**：抓取物体并放置到目标位置
- **推拉操作（Push & Pull）**：推动或拉动物体改变其位置
- **倾倒任务（Pouring）**：将液体从容器倒入另一容器
- **搅拌操作（Stirring）**：使用工具搅拌容器内物质
- **开门关门（Door Opening）**：操作门把手开关门
- **抽屉操作（Drawer Manipulation）**：拉开或推上抽屉
- **折叠布料（Cloth Folding）**：将布料折叠成指定形状
- **擦拭表面（Wiping）**：用布擦拭桌面等表面
- **插入任务（Insertion）**：将物体插入指定孔位
- **堆叠物体（Stacking）**：将物体堆叠在一起

这些任务覆盖了日常生活中的大部分操作场景，能够很好地评估机器人的泛化能力。

## 2. 数据采集系统

### 2.1 遥操作设备

RH20T 数据集的核心采集设备是 **Phantom Omni 力反馈遥操作设备**（由 Sensable 公司生产）。Phantom Omni 是一种 6-DOF（自由度）力反馈输入设备，研究人员通过它来控制机械臂的运动。

Phantom Omni 的特点：
- **6 自由度位置跟踪**：能够精确跟踪操作员手部的位置和姿态
- **力反馈功能**：操作员可以感受到机械臂末端执行器与环境交互的力
- **笔形手柄设计**：操作员通过手持笔形手柄进行操作
- **工作空间**：约 1.3m × 1.7m × 0.6m

遥操作的工作原理是将 Phantom Omni 的运动映射到机械臂的末端执行器。当操作员移动手柄时，机械臂会实时复制该运动，同时记录操作员的动作作为示范数据。

### 2.2 传感器配置

RH20T 的传感器配置非常丰富，提供了多模态数据：

| 传感器类型 | 数量 | 参数 | 用途 |
|-----------|------|------|------|
| RGB 相机 | 3 | 640×480@30Hz | 视觉输入 |
| Depth 相机 | 3 | 640×480@30Hz | 深度感知 |
| 关节角度传感器 | 6 | 6 轴关节角 | 机械臂状态 |
| 末端执行器位姿 | 1 | 6-DOF 位置+姿态 | 末端控制 |
| 力矩传感器 | 6 | 6 轴力/力矩 | 交互力 |
| 夹爪状态 | 1 | 开/闭状态 | 夹爪控制 |

多视角相机的设置使得数据具有丰富的空间信息，便于进行三维重建和视角合成研究。

### 2.3 同步机制

RH20T 采用了**硬件时间戳同步机制**，确保所有传感器数据在时间上对齐：

- **统一时钟源**：所有设备共享一个高精度时钟，确保时间同步
- **硬件触发**：相机采用硬件触发模式，同时采集同一时刻的图像
- **时间戳记录**：每帧数据都带有微秒级时间戳，便于后续对齐处理
- **延迟补偿**：记录系统各环节的处理延迟，进行补偿校正

这种同步机制对于模仿学习至关重要，因为动作和视觉信息必须在时间上严格对齐，机器人才能正确学习视觉运动策略。

## 3. 数据格式

### 3.1 轨迹数据

轨迹数据是 RH20T 的核心，包含机器人状态和动作序列。

**数据字段说明**：

```python
{
    "timestamp": 1609459200000,      # 时间戳（毫秒）
    "joint_positions": [0.1, -0.5, 1.2, 0.3, -0.8, 0.5],  # 6 个关节角度（弧度）
    "joint_velocities": [0.02, -0.01, 0.05, 0.01, -0.02, 0.03],  # 关节速度（弧度/秒）
    "joint_torques": [0.5, -0.3, 1.2, 0.8, -0.6, 0.4],  # 6 轴力矩（Nm）
    "end_effector_pose": {          # 末端执行器位姿（基坐标系下）
        "position": [0.3, 0.2, 0.5],  # x, y, z（米）
        "orientation": [0.1, 0.2, 0.3, 0.9]  # 四元数 (qx, qy, qz, qw)
    },
    "gripper_state": 1,             # 夹爪状态：0=开，1=闭
    "action": [0.01, -0.005, 0.02, 0.005, -0.01, 0.003, 1]  # 动作：6轴增量 + 夹爪
}
```

**动作表示**：RH20T 使用 7 维动作向量，其中前 6 维是末端执行器的增量位置（相对于当前位姿），第 7 维是夹爪控制信号。这种表示方式便于直接用于行为克隆算法。

### 3.2 视频数据

视频数据以 **ROS Bag** 格式存储，包含 RGB 和 Depth 视频流：

## 4. 任务类型详解

### 4.1 操作任务

操作任务是 RH20T 的核心，占数据集的 70% 以上，主要包括：

**精确抓取（Precision Grasping）**
- 抓取小尺寸物体（如方块、球体）
- 要求末端执行器精确定位
- 需要考虑物体的形状和方向

**抓取放置（Pick and Place）**
- 从起始位置抓取物体
- 移动到目标位置上方
- 释放物体到目标区域
- 是最基础的机器人操作技能

**倾倒（Pouring）**
- 抓取容器（杯子、瓶子）
- 倾斜容器使液体流出
- 控制倾倒角度和速度
- 涉及重力和液体物理特性

### 4.2 导航任务

导航任务占数据集的 15% 左右：

**末端轨迹导航**
- 末端执行器沿指定路径移动
- 路径可以是直线、曲线或复杂形状
- 用于训练平滑运动控制

**目标点导航**
- 将末端执行器移动到目标位置
- 目标位置由视觉或语言指定
- 训练空间推理能力

### 4.3 复合任务

复合任务占数据集的 15% 左右，是多个子任务的组合：

**两步操作**
- 例如：打开抽屉 → 取出物品 → 关闭抽屉
- 需要动作序列规划

**多物体操作**
- 例如：将方块 A 放入托盘 → 将方块 B 叠放在 A 上
- 需要空间关系理解

**工具使用**
- 例如：使用海绵擦拭桌面
- 需要理解工具与环境的交互

### 5.2 预处理

下载的数据需要预处理才能用于模型训练：

```python
import json
import numpy as np
import os

def preprocess_rh20t(data_root, output_root):
    """
    RH20T 数据预处理
    
    参数:
        data_root: 原始数据根目录
        output_root: 处理后数据输出目录
    """
    os.makedirs(output_root, exist_ok=True)
    
    episode_dirs = sorted([d for d in os.listdir(data_root) 
                          if d.startswith("episode_")])
    
    for ep_dir in episode_dirs:
        ep_path = os.path.join(data_root, ep_dir)
        
        # 读取轨迹数据
        traj_file = os.path.join(ep_path, "trajectory.json")
        with open(traj_file, 'r') as f:
            trajectory = json.load(f)
        
        # 提取状态和动作序列
        states = []
        actions = []
        
        for frame in trajectory:
            # 拼接关节位置和末端位姿作为状态
            state = frame["joint_positions"] + frame["end_effector_pose"]["position"]
            action = frame["action"]
            
            states.append(state)
            actions.append(action)
        
        states = np.array(states, dtype=np.float32)
        actions = np.array(actions, dtype=np.float32)
        
        # 归一化处理
        states_mean = states.mean(axis=0)
        states_std = states.std(axis=0) + 1e-7
        states_norm = (states - states_mean) / states_std
        
        # 保存处理后的数据
        output_file = os.path.join(output_root, f"{ep_dir}.npz")
        np.savez(output_file,
                 states=states_norm,
                 actions=actions,
                 task_name=trajectory[0].get("task_name", "unknown"),
                 success=trajectory[-1].get("success", False))
        
        print(f"处理完成: {ep_dir}, 形状: states={states.shape}, actions={actions.shape}")

if __name__ == "__main__":
    preprocess_rh20t(
        data_root="./rh20t_raw",
        output_root="./rh20t_processed"
    )
```

### 5.3 格式转换

RH20T 原生格式为 ROS Bag，可以转换为多种常用格式：

```python
import rosbag
import json
import numpy as np
from pathlib import Path

def convert_bag_to_numpy(bag_path, output_dir):
    """
    将 ROS Bag 格式转换为 NumPy 数组
    
    参数:
        bag_path: ROS Bag 文件路径
        output_dir: 输出目录
    """
    output_dir = Path(output_dir)
    output_dir.mkdir(parents=True, exist_ok=True)
    
    bag = rosbag.Bag(bag_path)
    
    # 存储各话题数据
    rgb_data = {0: [], 1: [], 2: []}
    depth_data = {0: [], 1: [], 2: []}
    joint_data = []
    
    # 读取各话题数据
    for topic, msg, t in bag:
        if "/rgb_" in topic:
            cam_id = int(topic.split("_")[-1])
            rgb_data[cam_id].append({
                "timestamp": msg.header.stamp.to_nsec(),
                "data": np.frombuffer(msg.data, dtype=np.uint8)
            })
        elif "/depth_" in topic:
            cam_id = int(topic.split("_")[-1])
            depth_data[cam_id].append({
                "timestamp": msg.header.stamp.to_nsec(),
                "data": np.frombuffer(msg.data, dtype=np.uint16)
            })
        elif "/joint_states" in topic:
            joint_data.append({
                "timestamp": msg.header.stamp.to_nsec(),
                "positions": list(msg.position),
                "velocities": list(msg.velocity) if msg.velocity else [0]*6,
                "effort": list(msg.effort) if msg.effort else [0]*6
            })
    
    bag.close()
    
    # 保存为 npz 格式
    np.savez(output_dir / "camera_data.npz",
             rgb_0=np.array(rgb_data[0]),
             rgb_1=np.array(rgb_data[1]),
             rgb_2=np.array(rgb_data[2]),
             depth_0=np.array(depth_data[0]),
             depth_1=np.array(depth_data[1]),
             depth_2=np.array(depth_data[2]))
    
    np.savez(output_dir / "joint_data.npz",
             joints=np.array(joint_data))
    
    print(f"转换完成: {bag_path}")
    print(f"  RGB 帧数: {len(rgb_data[0])}")
    print(f"  关节状态数: {len(joint_data)}")

def convert_to_rlds_format(data_root, output_path):
    """
    转换为 RLDS（Reinforcement Learning Dataset Standard）格式
    
    RLDS 格式是 TensorFlow Datasets 的标准格式，便于与主流模仿学习框架兼容
    """
    from dataclasses import dataclass, field
    from typing import Dict, List, Any
    
    episodes = []
    episode_dirs = sorted([d for d in os.listdir(data_root) 
                           if d.startswith("episode_")])
    
    for ep_dir in episode_dirs:
        ep_path = Path(data_root) / ep_dir
        
        # 加载轨迹数据
        traj_file = ep_path / "trajectory.json"
        with open(traj_file) as f:
            traj_data = json.load(f)
        
        # 构建 RLDS 格式的 episode
        episode = {
            "episode_id": ep_dir,
            "task_name": traj_data[0].get("task_name", "unknown"),
            "success": traj_data[-1].get("success", False),
            "steps": []
        }
        
        for frame in traj_data:
            step = {
                "observation": {
                    "image": frame.get("rgb_data"),      # 需要从视频中提取
                    "depth": frame.get("depth_data"),
                    "joint_positions": frame["joint_positions"],
                    "end_effector_pose": frame["end_effector_pose"]
                },
                "action": frame["action"],
                "reward": 1.0 if frame.get("success") else 0.0,
                "is_terminal": frame.get("is_terminal", False)
            }
            episode["steps"].append(step)
        
        episodes.append(episode)
    
    # 保存为 JSONL 格式
    with open(output_path, 'w') as f:
        for ep in episodes:
            f.write(json.dumps(ep) + '\n')
    
    print(f"RLDS 格式转换完成，共 {len(episodes)} 个 episode")
    return output_path
```

## 6. 代码实战

### 6.1 数据加载器实现

下面实现一个完整的 RH20T 数据加载器：

```python
import json
import numpy as np
import torch
from torch.utils.data import Dataset, DataLoader
from pathlib import Path
from typing import List, Dict, Optional, Tuple

class RH20TDataset(Dataset):
    """
    RH20T 数据集加载器
    
    支持：
    - 按任务过滤
    - 数据归一化
    - 数据增强
    - 多模态数据加载
    """
    
    def __init__(
        self,
        data_root: str,
        tasks: Optional[List[str]] = None,
        normalize: bool = True,
        seq_length: int = 16,
        img_size: Tuple[int, int] = (224, 224)
    ):
        """
        初始化数据集
        
        参数:
            data_root: 数据根目录
            tasks: 任务列表，None 表示加载所有任务
            normalize: 是否归一化
            seq_length: 序列长度
            img_size: 图像 resize 大小
        """
        self.data_root = Path(data_root)
        self.tasks = tasks
        self.normalize = normalize
        self.seq_length = seq_length
        self.img_size = img_size
        
        # 收集所有有效的 episode
        self.episodes: List[Dict] = []
        self._load_episode_list()
        
        # 计算归一化参数
        if normalize:
            self._compute_normalization_params()
    
    def _load_episode_list(self):
        """加载所有 episode 的元数据"""
        for ep_dir in sorted(self.data_root.iterdir()):
            if not ep_dir.name.startswith("episode_"):
                continue
            
            traj_file = ep_dir / "trajectory.json"
            if not traj_file.exists():
                continue
            
            with open(traj_file, 'r') as f:
                meta = json.load(f)
            
            # 按任务过滤
            if self.tasks and meta[0].get("task_name") not in self.tasks:
                continue
            
            self.episodes.append({
                "path": ep_dir,
                "meta": meta,
                "task_name": meta[0].get("task_name", "unknown"),
                "success": meta[-1].get("success", False)
            })
        
        print(f"加载了 {len(self.episodes)} 个有效 episode")
    
    def _compute_normalization_params(self):
        """计算归一化参数（均值和标准差）"""
        all_states = []
        
        for ep in self.episodes:
            for frame in ep["meta"]:
                state = frame["joint_positions"] + frame["end_effector_pose"]["position"]
                all_states.append(state)
        
        all_states = np.array(all_states, dtype=np.float32)
        self.state_mean = torch.FloatTensor(all_states.mean(axis=0))
        self.state_std = torch.FloatTensor(all_states.std(axis=0) + 1e-7)
        
        # 动作归一化参数（使用固定值）
        self.action_mean = torch.zeros(7)
        self.action_std = torch.ones(7) * 0.5
    
    def __len__(self) -> int:
        return len(self.episodes)
    
    def __getitem__(self, idx: int) -> Dict[str, torch.Tensor]:
        """
        获取一个 episode 的数据
        返回：
            dict: 包含以下键的字典
                - states: 状态序列 (seq_len, state_dim)
                - actions: 动作序列 (seq_len, action_dim)
                - task_name: 任务名称
                - success: 是否成功
        """
        ep = self.episodes[idx]
        meta = ep["meta"]
        
        # 提取状态和动作序列
        states = []
        actions = []
        
        for frame in meta:
            # 构建状态向量：关节位置 + 末端位置
            joint_pos = np.array(frame["joint_positions"], dtype=np.float32)
            ee_pos = np.array(frame["end_effector_pose"]["position"], dtype=np.float32)
            state = np.concatenate([joint_pos, ee_pos])
            
            # 构建动作向量
            action = np.array(frame["action"], dtype=np.float32)
            
            states.append(state)
            actions.append(action)
        
        states = np.array(states, dtype=np.float32)
        actions = np.array(actions, dtype=np.float32)
        
        # 归一化
        if self.normalize:
            states = (states - self.state_mean.numpy()) / self.state_std.numpy()
            actions = (actions - self.action_mean.numpy()) / self.action_std.numpy()
        
        # 转换为 tensor
        states = torch.FloatTensor(states)
        actions = torch.FloatTensor(actions)
        
        return {
            "states": states,
            "actions": actions,
            "task_name": ep["task_name"],
            "success": torch.tensor(ep["success"], dtype=torch.float32)
        }

def create_rh20t_dataloader(
    data_root: str,
    batch_size: int = 32,
    tasks: Optional[List[str]] = None,
    shuffle: bool = True,
    num_workers: int = 4
) -> DataLoader:
    """
    创建 RH20T 数据加载器
    
    参数:
        data_root: 数据根目录
        batch_size: 批次大小
        tasks: 任务列表
        shuffle: 是否打乱
        num_workers: 数据加载线程数
    
    返回:
        DataLoader 实例
    """
    dataset = RH20TDataset(
        data_root=data_root,
        tasks=tasks,
        normalize=True,
        seq_length=16
    )
    
    def collate_fn(batch):
        """自定义批次整理函数"""
        # 将不同长度的序列 padding 到相同长度
        max_len = max(item["states"].shape[0] for item in batch)
        
        padded_states = []
        padded_actions = []
        masks = []
        
        for item in batch:
            seq_len = item["states"].shape[0]
            pad_len = max_len - seq_len
            
            if pad_len > 0:
                # Padding：用最后一步填充
                state_pad = torch.cat([item["states"]] * (pad_len // seq_len + 1), dim=0)[:pad_len]
                action_pad = torch.cat([item["actions"]] * (pad_len // seq_len + 1), dim=0)[:pad_len]
                
                states = torch.cat([item["states"], state_pad], dim=0)
                actions = torch.cat([item["actions"], action_pad], dim=0)
                mask = torch.cat([torch.ones(seq_len), torch.zeros(pad_len)], dim=0)
            else:
                states = item["states"]
                actions = item["actions"]
                mask = torch.ones(seq_len)
            
            padded_states.append(states)
            padded_actions.append(actions)
            masks.append(mask)
        
        return {
            "states": torch.stack(padded_states),      # (batch, max_len, state_dim)
            "actions": torch.stack(padded_actions),   # (batch, max_len, action_dim)
            "mask": torch.stack(masks),               # (batch, max_len)
            "task_names": [item["task_name"] for item in batch],
            "success": torch.stack([item["success"] for item in batch])
        }
    
    return DataLoader(
        dataset,
        batch_size=batch_size,
        shuffle=shuffle,
        num_workers=num_workers,
        collate_fn=collate_fn,
        pin_memory=True
    )
```

### 6.3 数据集构建

```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader
import numpy as np

class BehavioralCloningDataset:
    """
    基于 RH20T 的行为克隆数据集
    
    将多个任务的数据合并，构建统一的训练数据集
    """
    
    def __init__(self, data_root: str, tasks: list = None):
        """
        初始化数据集
        
        参数:
            data_root: 数据根目录
            tasks: 任务列表，None 表示全部任务
        """
        from pathlib import Path
        self.data_root = Path(data_root)
        self.tasks = tasks or []
        
        # 加载所有数据
        self.data = []
        self._load_all_episodes()
    
    def _load_all_episodes(self):
        """加载所有 episode 的状态-动作对"""
        for ep_dir in sorted(self.data_root.iterdir()):
            if not ep_dir.name.startswith("episode_"):
                continue
            
            traj_file = ep_dir / "trajectory.json"
            if not traj_file.exists():
                continue
            
            import json
            with open(traj_file, 'r') as f:
                meta = json.load(f)
            
            # 过滤指定任务
            task_name = meta[0].get("task_name", "unknown")
            if self.tasks and task_name not in self.tasks:
                continue
            
            # 提取状态-动作对
            for i in range(len(meta) - 1):
                state = self._build_state(meta[i])
                action = np.array(meta[i]["action"], dtype=np.float32)
                
                self.data.append({
                    "state": state,
                    "action": action,
                    "task": task_name,
                    "success": meta[-1].get("success", False)
                })
        
        print(f"共加载 {len(self.data)} 个状态-动作对")
    
    def _build_state(self, frame: dict) -> np.ndarray:
        """构建状态向量"""
        joint_pos = np.array(frame["joint_positions"], dtype=np.float32)
        ee_pos = np.array(frame["end_effector_pose"]["position"], dtype=np.float32)
        return np.concatenate([joint_pos, ee_pos])
    
    def get_dataloader(self, batch_size: int = 64, shuffle: bool = True) -> DataLoader:
        """返回 PyTorch DataLoader"""
        states = torch.FloatTensor(np.array([d["state"] for d in self.data]))
        actions = torch.FloatTensor(np.array([d["action"] for d in self.data]))
        
        dataset = torch.utils.data.TensorDataset(states, actions)
        return DataLoader(dataset, batch_size=batch_size, shuffle=shuffle, num_workers=4)

# ============ 示例：简单的行为克隆策略网络 ============

class SimpleBCPolicy(nn.Module):
    """
    简单的 MLP 策略网络
    
    输入：状态向量 (13维 = 6关节 + 3末端位置 + 4语言嵌入占位)
    输出：动作向量 (7维)
    """
    
    def __init__(self, state_dim: int = 13, action_dim: int = 7, hidden_dim: int = 256):
        super().__init__()
        
        self.network = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim // 2),
            nn.ReLU(),
            nn.Linear(hidden_dim // 2, action_dim),
            nn.Tanh()  # 动作归一化到 [-1, 1]
        )
        
        # 动作尺度参数（可学习）
        self.action_scale = nn.Parameter(torch.ones(action_dim) * 0.5)
        self.action_bias = nn.Parameter(torch.zeros(action_dim))
    
    def forward(self, state: torch.Tensor) -> torch.Tensor:
        """
        前向传播
        
        参数:
            state: 状态张量 (batch, state_dim)
        
        返回:
            action: 动作张量 (batch, action_dim)
        """
        action = self.network(state)
        return action * self.action_scale + self.action_bias

def train_bc_policy(
    data_root: str,
    tasks: list = None,
    epochs: int = 50,
    lr: float = 1e-3,
    batch_size: int = 64
):
    """
    训练行为克隆策略
    
    参数:
        data_root: 数据目录
        tasks: 训练任务列表
        epochs: 训练轮数
        lr: 学习率
        batch_size: 批次大小
    """
    # 创建数据集
    dataset = BehavioralCloningDataset(data_root, tasks)
    dataloader = dataset.get_dataloader(batch_size=batch_size)
    
    # 创建模型
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    policy = SimpleBCPolicy().to(device)
    optimizer = torch.optim.Adam(policy.parameters(), lr=lr)
    loss_fn = nn.MSELoss()
    
    # 训练循环
    for epoch in range(epochs):
        total_loss = 0.0
        num_batches = 0
        
        for states, actions in dataloader:
            states = states.to(device)
            actions = actions.to(device)
            
            # 前向传播
            pred_actions = policy(states)
            
            # 计算损失
            loss = loss_fn(pred_actions, actions)
            
            # 反向传播
            optimizer.zero_grad()
            loss.backward()
            torch.nn.utils.clip_grad_norm_(policy.parameters(), max_norm=1.0)
            optimizer.step()
            
            total_loss += loss.item()
            num_batches += 1
        
        avg_loss = total_loss / num_batches
        
        if (epoch + 1) % 10 == 0:
            print(f"Epoch {epoch+1}/{epochs}, Loss: {avg_loss:.6f}")
    
    # 保存模型
    torch.save(policy.state_dict(), "bc_policy.pth")
    print("模型已保存: bc_policy.pth")
    
    return policy

# ============ 使用示例 ============

if __name__ == "__main__":
    # 注意：实际使用时需要指定正确的 RH20T 数据目录
    DATA_ROOT = "./rh20t_processed"
    
    # 训练指定任务的策略
    policy = train_bc_policy(
        data_root=DATA_ROOT,
        tasks=["pick_place_cube", "push_cube"],
        epochs=50,
        lr=1e-3,
        batch_size=64
    )
```

### 简答题

11. **RH20T 数据集采集流程**：
    ```
    1. 实验准备：布置场景，摆放物体，调整相机位置
    2. 设备连接：将 Phantom Omni 连接到机械臂控制系统
    3. 同步校准：启动所有传感器，进行时间同步校准
    4. 任务演示：操作员通过 Phantom Omni 控制机械臂完成任务
    5. 数据记录：系统同步记录所有传感器数据和时间戳
    6. 质量检查：人工检查演示质量，标记成功/失败
    7. 数据保存：将数据打包保存为 ROS Bag 格式
    ```

12. **多视角相机的优势**：
    - **三维感知**：多视角可以恢复场景的三维结构，便于进行三维重建
    - **遮挡处理**：某一视角被遮挡时，其他视角可以提供补充信息
    - **视角合成**：可以生成任意视角的图像，增强数据多样性
    - **提高鲁棒性**：训练时可以随机选择视角，增强策略的泛化能力

13. **数据预处理主要步骤**：
    - 读取原始轨迹 JSON 文件
    - 提取状态向量（关节角度 + 末端位置）
    - 提取动作向量（7 维动作）
    - 过滤无效帧（如关节角度超出范围）
    - 计算均值和标准差进行归一化
    - 保存为 .npz 格式便于快速加载

14. **ROS Bag vs NumPy 格式对比**：
    | 特性 | ROS Bag | NumPy |
    |------|---------|-------|
    | 原始/处理后 | 原始数据格式 | 通常是处理后格式 |
    | 兼容性 | 需 ROS 环境 | 通用格式 |
    | 读取速度 | 较慢（需解析消息） | 快（内存映射） |
    | 文件大小 | 较大 | 可压缩，较小 |
    | 多模态支持 | 原生支持多话题 | 需额外组织结构 |
    | 适用场景 | 数据采集、ROS 环境 | 深度学习训练 |

15. **行为克隆的局限**：
    - **分布偏移（Distribution Shift）**：训练时见过的状态分布与测试时可能不同
    - **误差累积**：单步小误差会随时间步累积，导致策略崩溃
    - **因果混淆**：可能学到动作与状态的虚假相关，而非真正的因果关系
    - **数据效率低**：需要大量演示数据，覆盖各种状态

16. **按任务过滤实现**：

```python
import json
from pathlib import Path
from typing import List

def filter_episodes_by_task(
    data_root: str,
    task_names: List[str],
    fuzzy_match: bool = True
) -> List[dict]:
    """
    按任务名称过滤 episode
    
    参数:
        data_root: 数据目录
        task_names: 任务名称列表
        fuzzy_match: 是否模糊匹配
    
    返回:
        过滤后的 episode 信息列表
    """
    data_root = Path(data_root)
    filtered = []
    
    # 支持模糊匹配：将任务名转为小写，去除下划线
    if fuzzy_match:
        target_patterns = [t.lower().replace("_", "") for t in task_names]
    else:
        target_patterns = task_names
    
    for ep_dir in sorted(data_root.iterdir()):
        if not ep_dir.name.startswith("episode_"):
            continue
        
        traj_file = ep_dir / "trajectory.json"
        if not traj_file.exists():
            continue
        
        with open(traj_file, 'r') as f:
            meta = json.load(f)
        
        task = meta[0].get("task_name", "unknown")
        
        # 精确匹配
        if not fuzzy_match:
            if task in task_names:
                filtered.append({"path": str(ep_dir), "task": task, "success": meta[-1].get("success")})
            continue
        
        # 模糊匹配
        task_normalized = task.lower().replace("_", "")
        for pattern in target_patterns:
            if pattern in task_normalized or task_normalized in pattern:
                filtered.append({"path": str(ep_dir), "task": task, "success": meta[-1].get("success")})
                break
    
    return filtered

# 测试
```

17. **数据增强函数**：

```python
import numpy as np
from typing import Tuple

def augment_trajectory(
    states: np.ndarray,
    actions: np.ndarray,
    state_noise_std: float = 0.01,
    action_noise_std: float = 0.02,
    action_bounds: Tuple[np.ndarray, np.ndarray] = None
) -> Tuple[np.ndarray, np.ndarray]:
    """
    对轨迹数据进行增强（添加噪声扰动）
    
    参数:
        states: 状态序列 (T, state_dim)
        actions: 动作序列 (T, action_dim)
        state_noise_std: 状态噪声标准差
        action_noise_std: 动作噪声标准差
        action_bounds: 动作的 (min, max) 边界，None 表示不限制
    
    返回:
        增强后的 (states, actions)
    """
    T, state_dim = states.shape
    _, action_dim = actions.shape
    
    # 对状态添加高斯噪声
    state_noise = np.random.normal(0, state_noise_std, size=(T, state_dim))
    states_aug = states + state_noise
    
    # 对动作添加高斯噪声
    action_noise = np.random.normal(0, action_noise_std, size=(T, action_dim))
    actions_aug = actions + action_noise
    
    # 限制动作在合法范围内
    if action_bounds is not None:
        action_min, action_max = action_bounds
        actions_aug = np.clip(actions_aug, action_min, action_max)
    
    return states_aug.astype(np.float32), actions_aug.astype(np.float32)

def batch_augment(
    dataset: list,
    num_augmentations: int = 5,
    state_noise_std: float = 0.01,
    action_noise_std: float = 0.02
) -> list:
    """
    批量数据增强
    
    参数:
        dataset: 原始数据集，每个元素为 (states, actions) 元组
        num_augmentations: 每个样本增强的次数
        其他参数: 噪声参数
    
    返回:
        增强后的数据集（包含原始数据 + 增强数据）
    """
    augmented = list(dataset)  # 保留原始数据
    
    for states, actions in dataset:
        for _ in range(num_augmentations):
            states_aug, actions_aug = augment_trajectory(
                states, actions,
                state_noise_std=state_noise_std,
                action_noise_std=action_noise_std
            )
            augmented.append((states_aug, actions_aug))
    
    return augmented

# 使用示例
if __name__ == "__main__":
    # 假设有轨迹数据
    states = np.random.randn(100, 13).astype(np.float32)  # 100帧，13维状态
    actions = np.random.randn(100, 7).astype(np.float32)   # 100帧，7维动作
    
    # 动作范围限制（基于 RH20T 的实际关节限制）
    action_min = np.array([-0.5, -0.5, -0.5, -0.5, -0.5, -0.5, 0.0])
    action_max = np.array([0.5, 0.5, 0.5, 0.5, 0.5, 0.5, 1.0])
    
    states_aug, actions_aug = augment_trajectory(
        states, actions,
        state_noise_std=0.01,
        action_noise_std=0.02,
        action_bounds=(action_min, action_max)
    )
    
    print(f"原始: states={states.shape}, actions={actions.shape}")
    print(f"增强: states_aug={states_aug.shape}, actions_aug={actions_aug.shape}")
    print(f"动作范围验证: min={actions_aug.min():.3f}, max={actions_aug.max():.3f}")
```

18. **BC 训练脚本（含断点续训）**：

```python
import torch
import torch.nn as nn
from pathlib import Path
import numpy as np
import json

class BCPolicy(nn.Module):
    """行为克隆策略网络"""
    
    def __init__(self, state_dim=13, action_dim=7, hidden_dim=256):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim),
            nn.Tanh()
        )
        self.action_scale = nn.Parameter(torch.ones(action_dim) * 0.5)
        self.action_bias = nn.Parameter(torch.zeros(action_dim))
    
    def forward(self, state):
        return self.net(state) * self.action_scale + self.action_bias

def load_rh20t_data(data_root):
    """加载 RH20T 数据"""
    data_root = Path(data_root)
    states, actions = [], []
    
    for ep_dir in sorted(data_root.iterdir()):
        if not ep_dir.name.startswith("episode_"):
            continue
        traj_file = ep_dir / "trajectory.json"
        if not traj_file.exists():
            continue
        
        with open(traj_file) as f:
            meta = json.load(f)
        
        for frame in meta:
            joint = np.array(frame["joint_positions"], dtype=np.float32)
            ee = np.array(frame["end_effector_pose"]["position"], dtype=np.float32)
            state = np.concatenate([joint, ee])
            action = np.array(frame["action"], dtype=np.float32)
            states.append(state)
            actions.append(action)
    
    return torch.FloatTensor(np.array(states)), torch.FloatTensor(np.array(actions))

def train_with_resume(
    data_root: str,
    save_path: str = "bc_policy.pth",
    epochs: int = 100,
    lr: float = 1e-3,
    batch_size: int = 256
):
    """
    带断点续训的 BC 训练
    
    功能：
    - 自动检测已有的检查点
    - 从上次中断处继续训练
    - 记录最佳模型
    """
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    
    # 加载数据
    print("加载数据...")
    states, actions = load_rh20t_data(data_root)
    print(f"数据规模: {states.shape}")
    
    # 创建模型
    policy = BCPolicy().to(device)
    optimizer = torch.optim.Adam(policy.parameters(), lr=lr)
    loss_fn = nn.MSELoss()
    
    # 断点续训逻辑
    start_epoch = 0
    best_loss = float('inf')
    save_path = Path(save_path)
    
    if save_path.exists():
        print(f"检测到已有模型: {save_path}")
        checkpoint = torch.load(save_path, map_location=device)
        policy.load_state_dict(checkpoint['model_state_dict'])
        optimizer.load_state_dict(checkpoint['optimizer_state_dict'])
        start_epoch = checkpoint['epoch'] + 1
        best_loss = checkpoint.get('best_loss', float('inf'))
        print(f"从 epoch {start_epoch} 继续训练，最佳损失: {best_loss:.6f}")
    
    # 创建 DataLoader
    dataset = torch.utils.data.TensorDataset(states, actions)
    dataloader = torch.utils.data.DataLoader(
        dataset, batch_size=batch_size, shuffle=True, num_workers=4
    )
    
    # 训练循环
    for epoch in range(start_epoch, epochs):
        policy.train()
        total_loss = 0.0
        num_batches = 0
        
        for batch_states, batch_actions in dataloader:
            batch_states = batch_states.to(device)
            batch_actions = batch_actions.to(device)
            
            pred_actions = policy(batch_states)
            loss = loss_fn(pred_actions, batch_actions)
            
            optimizer.zero_grad()
            loss.backward()
            torch.nn.utils.clip_grad_norm_(policy.parameters(), max_norm=1.0)
            optimizer.step()
            
            total_loss += loss.item()
            num_batches += 1
        
        avg_loss = total_loss / num_batches
        print(f"Epoch {epoch+1}/{epochs}, Loss: {avg_loss:.6f}", 
              " *" if avg_loss < best_loss else "")
        
        # 保存检查点
        if avg_loss < best_loss:
            best_loss = avg_loss
            torch.save({
                'epoch': epoch,
                'model_state_dict': policy.state_dict(),
                'optimizer_state_dict': optimizer.state_dict(),
                'best_loss': best_loss
            }, save_path)
            print(f"  -> 保存最佳模型 (loss={best_loss:.6f})")
    
    print(f"\n训练完成！最佳损失: {best_loss:.6f}")
    print(f"模型保存位置: {save_path}")

if __name__ == "__main__":
    train_with_resume(
        data_root="./rh20t_processed",
        save_path="bc_policy.pth",
        epochs=100,
        lr=1e-3,
        batch_size=256
    )
```
