## 1. 项目概述与背景

### 1.1 语音交互的重要性

语音是人类最自然、最高效的交互方式之一。在机器人领域，语音交互使机器人从"程序驱动"向"自然交互"转变，具有以下核心价值：

| 特性 | 键盘/触屏交互 | 语音交互 |
|------|--------------|---------|
| **输入效率** | 需要视觉注意和手动操作 | 边做其他事边操控 |
| **学习成本** | 需要学习软件界面 | 自然语言，无需学习 |
| **可及性** | 需要视觉和肢体协调 | 视障/运动障碍者可用 |
| **场景适用** | 固定场景 | 远距离/双手忙碌时 |
| **情感连接** | 弱 | 强 |

在具身智能场景中，语音交互的核心优势尤为突出：

```
用户（双手拿东西）──语音指令──→ 机器人执行任务
     "帮我把桌上的水杯拿过来"
     
用户（远距离）──语音指令──→ 机器人移动到指定位置
     "去厨房看看炉子有没有关"
```

### 1.2 端到端语音控制架构

传统的语音控制系统采用流水线架构（ASR → NLU → DM → Planning → Execution → TTS），各环节独立优化，但存在误差累积的问题。端到端语音控制则将语音直接映射为机器人动作，通过大语言模型（LLM）实现"语音 → 意图 → 动作"的直接转换。

**流水线架构 vs 端到端架构**：

```
┌─────────────────────────────────┐     ┌─────────────────────────────────┐
│        流水线架构                │     │        端到端架构                 │
├─────────────────────────────────┤     ├─────────────────────────────────┤
│  语音 → ASR → NLU → DM → 动作   │     │  语音 → Whisper → LLM → 动作     │
│                                 │     │                                 │
│  优点：各模块可独立优化           │     │  优点：减少误差累积，更自然        │
│  缺点：误差在各环节累积          │     │  缺点：对 LLM 能力依赖强          │
└─────────────────────────────────┘     └─────────────────────────────────┘
```

本项目采用**混合架构**：利用 Whisper 进行高精度语音识别，将识别文本送入 LLM 进行意图理解和任务规划，再通过 ROS2 接口执行具体动作，最后通过 TTS 反馈执行结果。

### 1.3 技术选型

#### 1.3.1 语音识别：Whisper

Whisper 是 OpenAI 开发的开源语音识别模型，具有以下优势：

- **多语言支持**：支持 100+ 语言，包括中文方言
- **高性能**：在英文语音识别上达到人类水平
- **开源免费**：MIT 许可证，可商用
- **多版本可选**：tiny/base/small/medium/large，权衡精度与速度

| 模型 | 参数量 | 英文 WER | 中文 CER | 推理速度（RTF） |
|------|--------|---------|---------|----------------|
| tiny | 39M | ~4% | ~5% | 0.08 |
| base | 74M | ~3% | ~4% | 0.15 |
| small | 244M | ~2% | ~3% | 0.35 |
| medium | 769M | ~1.5% | ~2% | 0.85 |
| large | 1550M | ~1% | ~1.5% | 1.8 |

> WER = Word Error Rate（词错误率），越低越好
> CER = Character Error Rate（字错误率），越低越好
> RTF = Real-Time Factor，0.1 表示 1 秒音频需 0.1 秒处理

**选型建议**：
- 机器人端（嵌入式）：whisper-tiny 或 whisper-base，RTF 低
- 开发机/服务器：whisper-small 或 whisper-medium，精度更高

#### 1.3.2 大语言模型：LLM

LLM 是语音控制系统的"大脑"，负责：
1. **意图识别**：理解用户想做什么
2. **参数提取**：提取动作目标、约束条件
3. **任务规划**：将复杂指令拆解为机器人可执行的动作序列
4. **对话管理**：处理多轮对话、上下文记忆

**主流开源模型对比**：

| 模型 | 参数量 | 中文能力 | 工具调用 | 本地部署难度 |
|------|--------|---------|---------|-------------|
| Qwen2.5-7B | 7B | ⭐⭐⭐⭐⭐ | 支持 | 中等 |
| ChatGLM3-6B | 6B | ⭐⭐⭐⭐ | 支持 | 较易 |
| Yi-1.5-6B | 6B | ⭐⭐⭐⭐ | 支持 | 较易 |
| DeepSeek-7B | 7B | ⭐⭐⭐⭐ | 支持 | 中等 |

**商业模型（API 调用）**：
- GPT-4o：性能最强，成本较高
- Claude 3.5：长上下文优秀
- 通义千问（阿里）：中文优化好

#### 1.3.3 语音合成：TTS

| 方案 | 特点 | 推荐场景 |
|------|------|---------|
| Coqui TTS | 开源，支持多语言，部署复杂 | 本地部署优先 |
| Fish Speech | 开源中文优化，中文自然 | 中文机器人 |
| Azure TTS | 云服务，音质好，成本低 | 有网络环境 |
| GPT-SoVITS | 开源，克隆声音 | 个性化音色 |

**本项目推荐**：Fish Speech（中文优化）或 Coqui TTS（多语言）

### 2.2 ASR 模块（语音识别）

ASR（Automatic Speech Recognition）模块负责将麦克风采集的音频流转换为文本。其核心处理流程如下：

```
麦克风采集（16kHz, 16bit, 单声道）
        ↓
音频预处理（降噪、回声消除、VAD）
        ↓
Whisper Encoder（声学特征提取）
        ↓
Whisper Decoder（自回归生成文本）
        ↓
文本后处理（标点恢复、大小写规范化）
        ↓
意图理解模块
```

**VAD（Voice Activity Detection）** 是 ASR 的关键前置处理：
- 作用：检测用户是否在说话，过滤静音和背景噪音
- 常用算法：WebRTC VAD、Silero VAD（推荐，精度高）

### 2.3 NLU 模块（自然语言理解）

NLU 模块将 ASR 输出的文本转换为结构化的意图和参数：

```python
# NLU 输出示例
{
    "intent": "go_and_fetch",      # 意图：去拿东西
    "entities": {
        "target_object": "水杯",    # 目标物体
        "location": "厨房"          # 位置
    },
    "confidence": 0.92,            # 置信度
    "raw_text": "去厨房拿水杯"       # 原始文本
}
```

### 2.4 DM 模块（对话管理）

对话管理（Dialogue Management）负责：
1. **上下文追踪**：维护多轮对话历史
2. **对话状态更新**：根据当前轮更新状态
3. **策略选择**：决定系统下一步行为（回复/执行/询问）

```python
# 对话状态示例
{
    "session_id": "user_001_session_20260326",
    "turn_count": 3,
    "current_intent": "go_and_fetch",
    "context": {
        "mentioned_locations": ["厨房", "客厅"],
        "mentioned_objects": ["水杯", "苹果"],
        "confirmed": False,
        "pending_info": ["目标精确位置"]
    },
    "last_robot_action": "move_to_kitchen",
    "waiting_for_confirmation": True
}
```

### 2.5 TTS 模块（语音合成）

TTS 模块将机器人回复文本转换为语音：

```
文本输入
   ↓
文本预处理（数字/日期/符号规范化）
   ↓
语言合成（声学模型生成梅尔频谱）
   ↓
声码器（Vocoder，将频谱转为波形）
   ↓
音频后处理（音量标准化）
   ↓
扬声器输出
```

## 3. 核心技术详解

### 3.1 Whisper 语音识别原理

Whisper 采用 Transformer Encoder-Decoder 架构，端到端完成语音到文本的转换：

```
┌─────────────────────────────────────────────────────────────┐
│                      Whisper 模型架构                        │
└─────────────────────────────────────────────────────────────┘

音频（30秒片段）
     ↓
MFCC / Log-Mel Spectrogram（80通道，80ms帧长）
     ↓
┌─────────────────┐
│  Encoder Block  │  × 4层（small/large 更多层）
│  (Transformer)   │
└────────┬────────┘
         │ 提取的声学表征
         ▼
┌─────────────────┐
│  Decoder Block  │  × 4层
│  (Transformer)  │  自回归生成 token
└────────┬────────┘
         │
         ▼
   Token → 文本
```

**Whisper 的关键特性**：
1. **多任务训练**：同时训练语音识别、语音翻译、语音活动检测等多任务
2. **无需语言识别**：模型自动判断语言，无需单独的语种识别模块
3. **时间戳预测**：可输出每个词的时间戳（Web 版本支持）

### 3.2 LLM 意图理解原理

LLM 通过 In-Context Learning 实现意图理解。核心思想是：在 Prompt 中提供少量示例（Few-Shot），让模型学会理解用户意图的模式：

```python
# LLM 意图理解 Prompt 示例
SYSTEM_PROMPT = """
你是一个机器人控制系统的对话管理器。
用户的语音指令已经被转换为文本。
你的任务：
1. 识别用户意图（intent）
2. 提取关键参数（entities）
3. 生成机器人可执行的动作序列（actions）

可识别的意图类型：
- navigation: 移动到指定位置
- fetch_object: 去某处取某物
- patrol: 巡逻
- report_status: 报告状态
- answer_question: 回答问题
- stop: 停止当前动作
- cancel: 取消任务

输出格式（JSON）：
{
  "intent": "意图名称",
  "entities": {"参数名": "参数值"},
  "actions": [{"action": "动作", "params": {...}}],
  "response_text": "对用户的回复"
}
"""

# 用户输入
user_input = "去厨房把水杯拿给我"

# LLM 输出
{
  "intent": "fetch_object",
  "entities": {
    "object": "水杯",
    "source": "厨房",
    "destination": "用户当前位置"
  },
  "actions": [
    {"action": "navigate", "params": {"target": "厨房"}},
    {"action": "detect_object", "params": {"object": "水杯"}},
    {"action": "grasp", "params": {"object": "水杯"}},
    {"action": "return", "params": {"destination": "用户"}},
    {"action": "deliver", "params": {"object": "水杯"}}
  ],
  "response_text": "好的，我去厨房帮你拿水杯"
}
```

### 4.2 音频采集模块

```python
# audio_capture.py
"""
音频采集模块：负责从麦克风采集音频流，支持 VAD（语音活动检测）
"""

import rclpy
from rclpy.node import Node
from audio_common_msgs.msg import AudioData
import numpy as np
from collections import deque
import torch
import pyaudio


class AudioCapture(Node):
    """音频采集节点"""
    
    def __init__(self, node_name='audio_capture'):
        super().__init__(node_name)
        
        # 音频参数配置
        self.SAMPLE_RATE = 16000          # 采样率（Whisper 要求 16kHz）
        self.CHUNK_SIZE = 1600            # 每次读取的样本数（100ms）
        self.AUDIO_FORMAT = pyaudio.paInt16  # 16bit PCM
        self.CHANNELS = 1                 # 单声道
        
        # VAD 模型（使用 Silero VAD）
        self.vad_model, self.vad_utils = self._load_vad_model()
        self.get_logger().info('VAD 模型加载完成')
        
        # 音频缓冲区
        self.audio_buffer = deque(maxlen=160000)  # 最大缓存 10 秒
        self.is_speaking = False
        self.speech_start_time = None
        
        # PyAudio 对象
        self.audio = pyaudio.PyAudio()
        self.stream = None
        
        # ROS2 发布者：发布音频数据
        self.audio_publisher = self.create_publisher(
            AudioData, '/voice/audio_raw', 10
        )
        
        # 启动音频采集
        self._start_capture()
        
        self.get_logger().info('音频采集节点初始化完成')
    
    def _load_vad_model(self):
        """加载 Silero VAD 模型"""
        # Silero VAD 是轻量级高性能的语音活动检测模型
        model, utils = torch.hub.load(
            repo_or_dir='snakers4/silero-vad',
            model='silero_vad',
            trust_repo=True
        )
        return model, utils
    
    def _start_capture(self):
        """启动音频采集"""
        self.stream = self.audio.open(
            format=self.AUDIO_FORMAT,
            channels=self.CHANNELS,
            rate=self.SAMPLE_RATE,
            input=True,
            frames_per_buffer=self.CHUNK_SIZE,
            stream_callback=self._audio_callback
        )
        self.stream.start_stream()
        self.get_logger().info('音频流开始采集')
    
    def _audio_callback(self, in_data, frame_count, time_info, status):
        """
        音频流回调函数
        
        Args:
            in_data: 原始音频数据 bytes
            frame_count: 帧数量
            time_info: 时间信息
            status: 状态标志
        
        Returns:
            (in_data, pyaudio.paContinue): 继续采集
        """
        # 将 bytes 转换为 numpy 数组（int16）
        audio_data = np.frombuffer(in_data, dtype=np.int16)
        
        # 转换为 float32（0-1 范围）用于 VAD
        audio_float = audio_data.astype(np.float32) / 32768.0
        
        # VAD 检测
        speech_prob = self._vad_detect(audio_float)
        
        # 状态机管理
        if speech_prob > 0.5 and not self.is_speaking:
            # 检测到语音开始
            self.is_speaking = True
            self.speech_start_time = self.get_clock().now()
            self.get_logger().debug('检测到语音开始')
        
        if self.is_speaking:
            # 将音频数据加入缓冲区
            self.audio_buffer.append(audio_data)
            
            # 检测语音结束（持续低概率超过 1 秒）
            if speech_prob < 0.1:
                if self._check_speech_end():
                    self._publish_audio()
                    self.is_speaking = False
        
        return (in_data, pyaudio.paContinue)
    
    def _vad_detect(self, audio_float):
        """使用 Silero VAD 检测语音活动
        
        Args:
            audio_float: float32 音频数据，范围 [-1, 1]
        
        Returns:
            speech_prob: 语音存在的概率 [0, 1]
        """
        # 将音频转换为 tensor
        tensor = torch.from_numpy(audio_float).float()
        
        # 如果音频长度不足，进行填充
        if len(tensor) < 400:  # 最小需要 400 样本（25ms）
            tensor = torch.nn.functional.pad(tensor, (0, 400 - len(tensor)))
        
        # VAD 推理
        with torch.no_grad():
            speech_prob = self.vad_model(tensor, self.SAMPLE_RATE).item()
        
        return speech_prob
    
    def _check_speech_end(self):
        """检查语音是否结束（静音持续超过 1.5 秒）"""
        if self.speech_start_time is None:
            return False
        
        elapsed = (self.get_clock().now() - self.speech_start_time).nanoseconds / 1e9
        
        # 如果语音片段超过 30 秒，强制结束
        if elapsed > 30.0:
            return True
        
        # 缓冲区已满（10秒），强制结束
        if len(self.audio_buffer) * self.CHUNK_SIZE >= 160000:
            return True
        
        return False
    
    def _publish_audio(self):
        """发布完整语音片段到 ROS2 topic"""
        if len(self.audio_buffer) == 0:
            return
        
        # 合并所有音频片段
        audio_segments = np.concatenate(list(self.audio_buffer))
        audio_bytes = audio_segments.tobytes()
        
        # 创建 AudioData 消息并发布
        msg = AudioData()
        msg.data = audio_bytes
        self.audio_publisher.publish(msg)
        
        self.get_logger().info(
            f'发布语音片段，时长: {len(audio_segments)/self.SAMPLE_RATE:.2f}秒'
        )
        
        # 清空缓冲区
        self.audio_buffer.clear()
    
    def destroy_node(self):
        """清理资源"""
        if self.stream is not None:
            self.stream.stop_stream()
            self.stream.close()
        self.audio.terminate()
        super().destroy_node()
```

```python
"""

        self.text_publisher = self.create_publisher(

            10
        )

        self.confidence_publisher = self.create_publisher(
            String,
            '/voice/confidence',
            10
        )
        
        # 统计信息
        self.recognition_count = 0
        self.total_latency = 0.0
        
        self.get_logger().info(
            f'Whisper 节点初始化完成，模型: {self.model_size}，设备: {self.device}'
        )
    
    def _load_whisper_model(self):
        """加载 Whisper 模型"""
        self.get_logger().info(f'正在加载 Whisper {self.model_size} 模型...')
        
        model = whisper.load_model(
            self.model_size,
            device=self.device,
            download_root=self.model_dir
        )
        
        self.get_logger().info('Whisper 模型加载完成')
        return model
    
    def _on_audio_received(self, msg: AudioData):
        """接收到音频数据时的回调函数
        
        Args:
            msg: AudioData 消息，包含原始音频字节
        """
        start_time = time.time()
        
        # 将 bytes 转换为 numpy 数组
        audio_np = np.frombuffer(bytes(msg.data), dtype=np.int16)
        
        # 转换为 float32（Whisper 要求）
        audio_float = audio_np.astype(np.float32) / 32768.0
        
        # 如果音频过短（<0.3秒），忽略
        if len(audio_float) < 4800:  # 0.3秒 × 16kHz
            return
        
        self.get_logger().debug(
            f'接收到音频，时长: {len(audio_float)/16000:.2f}秒'
        )
        
        # Whisper 推理
        try:
            result = self.model.transcribe(
                audio_float,
                language=self.language,
                fp16=(self.device == 'cuda'),  # GPU 加速
                # 可选：提示词提升准确率
                initial_prompt="这是一个机器人语音控制系统，请准确识别中文语音。"
            )
            
            # 提取识别结果
            text = result['text'].strip()
            
            # 计算置信度（基于 tokens 的平均概率）
            if 'segments' in result and len(result['segments']) > 0:
                avg_probs = []
                for seg in result['segments']:
                    if 'tokens' in seg:
                        probs = torch.softmax(torch.tensor(seg['tokens']), dim=-1)
                        avg_prob = probs.max(dim=-1)[0].mean().item()
                        avg_probs.append(avg_prob)
                confidence = np.mean(avg_probs) if avg_probs else 0.0
            else:
                confidence = 0.0
            
            # 发布识别结果
            if text:
                self._publish_result(text, confidence)
                
                # 统计延迟
                latency = time.time() - start_time
                self.total_latency += latency
                self.recognition_count += 1
                avg_latency = self.total_latency / self.recognition_count
                
                self.get_logger().info(
                    f'识别完成: "{text}" (置信度: {confidence:.2f}, '
                    f'延迟: {latency:.2f}秒, 平均: {avg_latency:.2f}秒)'
                )
        
        except Exception as e:
            self.get_logger().error(f'Whisper 识别失败: {str(e)}')
    
    def _publish_result(self, text: str, confidence: float):
        """发布识别结果到 ROS2 topic
        
        Args:
            text: 识别的文本
            confidence: 置信度
        """
        # 发布文本
        text_msg = String()
        text_msg.data = text
        self.text_publisher.publish(text_msg)
        
        # 发布置信度（JSON 格式）
        conf_msg = String()
        conf_msg.data = json.dumps({'confidence': confidence})
        self.confidence_publisher.publish(conf_msg)
```

```python
        """

            'Authorization': f'Bearer {self.api_key}'

        full_messages = [{'role': 'system', 'content': system_prompt}] + messages
        
        payload = {
            'model': self.model_name,
            'messages': full_messages,
            'temperature': self.temperature,
            'max_tokens': self.max_tokens

                    f'{self.api_base}/chat/completions',
                    headers=headers,
                    json=payload,
                    timeout=30

                response.raise_for_status()
                result = response.json()
                return result['choices'][0]['message']['content']
            
            elif self.llm_type == 'qwen':
                # 阿里通义千问 API
                response = requests.post(
                    f'{self.api_base}/services/aigc/text-generation/generation',
                    headers=headers,
                    json=payload,
                    timeout=30
                )
                response.raise_for_status()
                result = response.json()
                return result['output']['text']
            
            else:
                # 默认 OpenAI 格式
                response = requests.post(
                    f'{self.api_base}/chat/completions',
                    headers=headers,
                    json=payload,
                    timeout=30
                )
                response.raise_for_status()
                result = response.json()
                return result['choices'][0]['message']['content']
        
        except requests.exceptions.RequestException as e:
            self.get_logger().error(f'LLM API 调用失败: {str(e)}')
            return json.dumps({
                'intent': 'unknown',
                'entities': {},
                'actions': [],
                'response_text': '网络出了点问题，请稍后再试。'
            })

        """
        self.robot_status.update(status)
        self.get_logger().debug(f'机器人状态已更新: {status}')
    
    def reset_conversation(self, session_id: Optional[str] = None):

        if target_id in self.conversations:
            self.conversations[target_id] = []
            self.get_logger().info(f'对话历史已重置: {target_id}')
```

```python
# action_executor.py
"""
动作执行器：订阅 LLM 生成的动作序列，通过 ROS2 接口执行机器人动作
支持动作类型：导航、机械臂控制、状态查询、TTS 反馈

import rclpy
from rclpy.node import Node
from std_msgs.msg import String
from geometry_msgs.msg import Twist
from nav_msgs.msg import Odometry
import json
import time


class ActionExecutor(Node):
    """机器人动作执行器节点"""
    
    def __init__(self, node_name='action_executor'):
        super().__init__(node_name)
        
        # 动作执行状态
        self.current_action_index = 0
        self.action_sequence = []
        self.is_executing = False
        
        # 导航状态订阅
        self.odom_subscriber = self.create_subscription(
            Odometry,
            '/odom',
            self._on_odom_received,
            10
        )
        
        # 当前位置
        self.current_position = {'x': 0.0, 'y': 0.0, 'theta': 0.0}
        
        # 速度命令发布者
        self.cmd_vel_publisher = self.create_publisher(
            Twist,
            '/cmd_vel',
            10
        )
        
        # 机械臂动作发布者
        self.arm_action_publisher = self.create_publisher(
            String,
            '/arm/action',
            10
        )
        
        # TTS 反馈发布者
        self.tts_publisher = self.create_publisher(
            String,
            '/tts/text',
            10
        )
        
        # 动作完成事件发布者
        self.action_done_publisher = self.create_publisher(
            String,
            '/action/executed',
            10
        )
        
        # ROS2 订阅者：订阅动作序列
        self.action_subscriber = self.create_subscription(
            String,
            '/robot/actions',
            self._on_actions_received,
            10
        )
        
        # 定时器：执行下一个动作
        self.execute_timer = self.create_timer(0.5, self._execute_step)
        
        self.get_logger().info('动作执行器初始化完成')
    
    def _on_odom_received(self, msg: Odometry):
        """接收里程计数据"""
        self.current_position['x'] = msg.pose.pose.position.x
        self.current_position['y'] = msg.pose.pose.position.y
        self.current_position['theta'] = 2.0 * msg.pose.pose.orientation.z
    
    def _on_actions_received(self, msg: String):
        """接收到动作序列时的回调"""
        try:
            actions = json.loads(msg.data)
            self.get_logger().info(f'接收到动作序列，共 {len(actions)} 个动作')
            
            # 开始执行动作序列
            self.action_sequence = actions
            self.current_action_index = 0
            self.is_executing = True
            
        except json.JSONDecodeError as e:
            self.get_logger().error(f'动作序列解析失败: {str(e)}')
    
    def _execute_step(self):
        """定时执行动作步骤"""
        if not self.is_executing:
            return
        
        if self.current_action_index >= len(self.action_sequence):
            # 所有动作执行完成
            self._on_sequence_complete()
            return
        
        # 获取当前动作
        action = self.action_sequence[self.current_action_index]
        action_name = action.get('action', '')
        params = action.get('params', {})
        
        self.get_logger().info(
            f'执行动作 [{self.current_action_index + 1}/{len(self.action_sequence)}]: '
            f'{action_name} - {params}'
        )
        
        # 执行动作
        success = self._execute_action(action_name, params)
        
        if success:
            self.current_action_index += 1
        else:
            # 动作执行失败，停止序列
            self.get_logger().error(f'动作执行失败: {action_name}')
            self.is_executing = False
            self._publish_action_result(action_name, False)
    
    def _execute_action(self, action_name: str, params: Dict) -> bool:
        """执行单个动作
        
        Args:
            action_name: 动作名称
            params: 动作参数
        
        Returns:
            动作是否执行成功
        """
        try:
            if action_name == 'navigate':
                return self._execute_navigate(params)
            elif action_name == 'grasp':
                return self._execute_grasp(params)
            elif action_name == 'place':
                return self._execute_place(params)
            elif action_name == 'patrol_start':
                return self._execute_patrol_start(params)
            elif action_name == 'report_battery':
                return self._execute_report_battery(params)
            elif action_name == 'report_position':
                return self._execute_report_position(params)
            elif action_name == 'stop_motion':
                return self._execute_stop_motion(params)
            elif action_name == 'cancel_task':
                return self._execute_cancel_task(params)
            else:
                self.get_logger().warn(f'未知动作类型: {action_name}')
                return True  # 未知动作，跳过
        
        except Exception as e:
            self.get_logger().error(f'动作执行异常: {str(e)}')
            return False
    
    def _execute_navigate(self, params: Dict) -> bool:
        """执行导航动作
        
        Args:
            params: 包含 location（目标位置）的字典
        
        Returns:
            是否成功（模拟场景返回 True）
        """
        location = params.get('location', 'unknown')
        speed = params.get('speed', 0.5)
        
        self.get_logger().info(f'开始导航到: {location}，速度: {speed}m/s')
        
        # 模拟导航执行（实际应用中应调用导航堆栈）
        # 1. 发布 TTS 反馈
        tts_msg = String()
        tts_msg.data = f'好的，我正在前往{location}'
        self.tts_publisher.publish(tts_msg)
        
        # 2. 发送速度命令（模拟移动）
        cmd_vel = Twist()
        cmd_vel.linear.x = speed
        self.cmd_vel_publisher.publish(cmd_vel)
        
        # 3. 模拟到达（实际应用中应监听导航完成事件）
        time.sleep(2)  # 模拟移动时间
        
        # 4. 停止移动
        cmd_vel.linear.x = 0.0
        self.cmd_vel_publisher.publish(cmd_vel)
        
        self.get_logger().info(f'已到达: {location}')
        
        # 发布 TTS 反馈
        tts_msg = String()
        tts_msg.data = f'我已到达{location}'
        self.tts_publisher.publish(tts_msg)
        
        return True
    
    def _execute_grasp(self, params: Dict) -> bool:

    def _execute_place(self, params: Dict) -> bool:

        tts_msg = String()
        tts_msg.data = '任务已完成，还有什么需要帮忙的吗？'
        self.tts_publisher.publish(tts_msg)
    
    def _publish_action_result(self, action_name: str, success: bool):
        """发布单个动作执行结果"""
        result_msg = String()
        result_msg.data = json.dumps({
            'action': action_name,
            'success': success,
            'timestamp': time.time()
        })
        self.action_done_publisher.publish(result_msg)
```

```python
# tts_node.py
"""
TTS 语音合成节点：将文本转换为语音并播放
支持 Coqui TTS、Fish Speech、Azure TTS 等后端
"""

import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import numpy as np
import requests
import io
import sounddevice as sd
import threading


class TTSNode(Node):
    """语音合成节点"""
    
    def __init__(self, node_name='tts_node'):
        super().__init__(node_name)
        
        # TTS 配置
        self.declare_parameter('tts_type', 'coqui')  # coqui/fish/azure
        self.declare_parameter('api_key', '')
        self.declare_parameter('voice_name', 'default')
        self.declare_parameter('sample_rate', 22050)
        
        self.tts_type = self.get_parameter('tts_type').value
        self.api_key = self.get_parameter('api_key').value
        self.voice_name = self.get_parameter('voice_name').value
        self.sample_rate = self.get_parameter('sample_rate').value
        
        # Coqui TTS 模型（首次使用需下载）
        self.tts_model = None
        self.tts_speaker = None
        
        # 播放锁（防止多线程同时播放）
        self.play_lock = threading.Lock()
        
        # ROS2 订阅者：订阅待合成文本
        self.text_subscriber = self.create_subscription(
            String,
            '/tts/text',
            self._on_text_received,
            10
        )
        
        # 初始化 TTS 模型
        if self.tts_type == 'coqui':
            self._init_coqui_tts()
        
        self.get_logger().info(f'TTS 节点初始化完成，类型: {self.tts_type}')
    
    def _init_coqui_tts(self):
        """初始化 Coqui TTS 模型"""
        try:
            from TTS.api import TTS
            
            # 使用中文模型（Coqui 提供了中文支持的模型）
            self.tts_model = TTS(model_name="tts_models/zh-CN/baker/tacotron2-DDC-GST")
            self.get_logger().info('Coqui TTS 模型加载完成')
        
        except ImportError:
            self.get_logger().warn('Coqui TTS 未安装，将使用占位音频')
    
    def _on_text_received(self, msg: String):
        """接收到待合成文本时的回调"""
        text = msg.data.strip()
        if not text:
            return
        
        self.get_logger().info(f'收到 TTS 请求: "{text}"')
        
        # 在后台线程合成并播放（避免阻塞主线程）
        thread = threading.Thread(target=self._synthesize_and_play, args=(text,))
        thread.start()
    
    def _synthesize_and_play(self, text: str):
        """合成语音并播放
        
        Args:
            text: 待合成的文本
        """
        with self.play_lock:
            try:
                # 合成语音
                audio_data = self._synthesize(text)
                
                if audio_data is not None:
                    # 播放音频
                    self._play_audio(audio_data)
                else:
                    self.get_logger().warn('语音合成失败')
            
            except Exception as e:
                self.get_logger().error(f'TTS 处理异常: {str(e)}')
    
    def _synthesize(self, text: str) -> Optional[np.ndarray]:
        """合成语音
        
        Args:
            text: 待合成的文本
        
        Returns:
            音频数据（numpy 数组），失败返回 None
        """
        if self.tts_type == 'coqui' and self.tts_model is not None:
            try:
                # Coqui TTS 合成
                wav = self.tts_model.tts(text=text, speaker=self.tts_speaker)
                return wav
            
            except Exception as e:
                self.get_logger().error(f'Coqui TTS 合成失败: {str(e)}')
                return None
        
        elif self.tts_type == 'azure':
            return self._azure_tts_synthesize(text)
        
        else:
            # 默认返回静音
            return np.zeros(int(self.sample_rate * 0.5))
    
    def _azure_tts_synthesize(self, text: str) -> Optional[np.ndarray]:
        """Azure TTS API 合成（需要网络）"""
        try:
            headers = {
                'Ocp-Apim-Subscription-Key': self.api_key,
                'Content-Type': 'application/ssml+xml'
            }
            
            ssml = f"""
            <speak version='1.0' xmlns='http://www.w3.org/2001/10/synthesis' 
                   xml:lang='zh-CN'>
                <voice name='zh-CN-XiaoxiaoNeural'>
                    {text}
                </voice>
            </speak>
            """
            
            endpoint = 'https://southeastasia.tts.speech.microsoft.com/cognitiveservices/v1'
            response = requests.post(
                endpoint,
                headers=headers,
                data=ssml.encode('utf-8'),
                timeout=10
            )
            
            if response.status_code == 200:
                # 将返回的音频数据（MP3）转换为 numpy 数组
                audio_bytes = response.content
                # 简化处理：实际应用中应使用 pydub 等库解码 MP3
                return np.frombuffer(audio_bytes, dtype=np.int16)
            else:
                self.get_logger().error(f'Azure TTS API 错误: {response.status_code}')
                return None
        
        except Exception as e:
            self.get_logger().error(f'Azure TTS 调用失败: {str(e)}')
            return None
    
    def _play_audio(self, audio_data: np.ndarray):
        """播放音频
        
        Args:
            audio_data: 音频数据数组
        """
        try:
            # 如果是 float32（-1 到 1 范围），直接播放
            if audio_data.dtype == np.float32:
                sd.play(audio_data, self.sample_rate)
            else:
                # int16 转换为 float32
                audio_float = audio_data.astype(np.float32) / 32768.0
                sd.play(audio_float, self.sample_rate)
            
            # 等待播放完成
            sd.wait()
        
        except Exception as e:
            self.get_logger().error(f'音频播放失败: {str(e)}')
```

### 6.2 动作状态机

```python
class ActionStateMachine:
    """
    动作状态机：管理动作执行的生命周期
    """
    
    class State(Enum):
        IDLE = 0           # 空闲
        EXECUTING = 1      # 执行中
        PAUSED = 2         # 暂停
        COMPLETED = 3      # 完成
        FAILED = 4         # 失败
    
    def __init__(self):
        self.state = self.State.IDLE
        self.current_action = None
        self.action_history = []
    
    def start_action(self, action: Dict):
        """开始执行新动作"""
        if self.state == self.State.EXECUTING:
            # 当前有动作在执行，需要先停止
            self.stop_current()
        
        self.current_action = action
        self.state = self.State.EXECUTING
        self.action_history.append({
            'action': action,
            'start_time': time.time(),
            'status': 'started'
        })
    
    def pause_action(self):
        """暂停当前动作"""
        if self.state == self.State.EXECUTING:
            self.state = self.State.PAUSED
    
    def resume_action(self):
        """恢复暂停的动作"""
        if self.state == self.State.PAUSED:
            self.state = self.State.EXECUTING
    
    def complete_action(self):
        """标记动作为完成"""
        if self.current_action:
            self.action_history[-1]['status'] = 'completed'
            self.action_history[-1]['end_time'] = time.time()
        self.state = self.State.COMPLETED
        self.current_action = None
    
    def fail_action(self, error: str):
        """标记动作失败"""
        if self.current_action:
            self.action_history[-1]['status'] = 'failed'
            self.action_history[-1]['error'] = error
            self.action_history[-1]['end_time'] = time.time()
        self.state = self.State.FAILED
        self.current_action = None
    
    def stop_current(self):
        """停止当前动作"""
        if self.current_action:
            # 发送停止命令到底层控制器
            self._send_stop_command()
            self.action_history[-1]['status'] = 'stopped'
            self.action_history[-1]['end_time'] = time.time()
```

```python
# test_rtf.py
"""测试 Whisper 推理实时因子"""

import time
import numpy as np
import torch
import whisper

def test_rtf(model_size='base', audio_duration=10.0):
    """测试指定模型的 RTF
    
    Args:
        model_size: Whisper 模型大小
        audio_duration: 测试音频时长（秒）
    """
    # 生成测试音频（1kHz 正弦波）
    sample_rate = 16000
    t = np.linspace(0, audio_duration, int(sample_rate * audio_duration))
    audio = (np.sin(2 * np.pi * 1000 * t) * 32767).astype(np.int16)
    audio_float = audio.astype(np.float32) / 32768.0
    
    # 加载模型
    device = 'cuda' if torch.cuda.is_available() else 'cpu'
    model = whisper.load_model(model_size, device=device)
    
    # 预热
    _ = model.transcribe(audio_float[:16000], language='zh')
    
    # 正式测试（多次取平均）
    times = []
    for _ in range(3):
        start = time.time()
        _ = model.transcribe(audio_float, language='zh')
        elapsed = time.time() - start
        times.append(elapsed)
        print(f'  单次耗时: {elapsed:.3f}s')
    
    avg_time = np.mean(times)
    rtf = avg_time / audio_duration
    
    print(f'\\n模型: {model_size}, 设备: {device}')
    print(f'音频时长: {audio_duration}s')
    print(f'平均耗时: {avg_time:.3f}s')
    print(f'RTF: {rtf:.4f}')
    print(f'实时倍率: 1:{1/rtf:.1f}')

if __name__ == '__main__':
    for size in ['tiny', 'base', 'small']:
        test_rtf(size)
        print()
```

**题目 1 参考答案**：

```python
class SimpleDialogueManager:
    """简化版对话管理器"""
    
    def __init__(self):
        self.conversation_history = []  # 对话历史，最多5轮
        self.max_history = 5
    
    def process(self, text: str) -> dict:
        """处理用户输入，返回意图和回复"""
        # 添加到历史
        self.conversation_history.append({'role': 'user', 'content': text})
        
        # 意图识别（简单关键词匹配）
        intent, entities = self._recognize_intent(text)
        
        # 生成回复和动作
        response, actions = self._generate_response(intent, entities)
        
        # 添加助手回复到历史
        self.conversation_history.append({'role': 'assistant', 'content': response})
        
        # 限制历史长度
        if len(self.conversation_history) > self.max_history * 2:
            self.conversation_history = self.conversation_history[-self.max_history * 2:]
        
        return {
            'intent': intent,
            'response': response,
            'actions': actions
        }
    
    def _recognize_intent(self, text: str) -> tuple:
        """识别意图和实体"""
        text = text.strip()
        
        # 导航关键词
        nav_keywords = ['去', '到', '前往', '移动到']
        for kw in nav_keywords:
            if kw in text:
                location = text.split(kw)[-1].replace('。', '').replace('吗', '')
                return 'navigation', {'location': location}
        
        # 取物关键词
        fetch_keywords = ['拿', '取', '拿过来', '取来', '帮我拿']
        for kw in fetch_keywords:
            if kw in text:
                parts = text.split(kw)
                if len(parts) > 1:
                    obj = parts[-1].replace('。', '')
                    return 'fetch_object', {'object': obj}
        
        # 问候关键词
        greeting_keywords = ['你好', '早上好', '晚上好', '嗨', 'hello']
        for kw in greeting_keywords:
            if kw in text:
                return 'greeting', {}
        
        return 'unknown', {}
    
    def _generate_response(self, intent: str, entities: dict) -> tuple:
        """生成回复和动作"""
        if intent == 'navigation':
            location = entities.get('location', '未知地点')
            return (
                f'好的，我前往{location}',
                [{'action': 'navigate', 'params': {'location': location}}]
            )
        elif intent == 'fetch_object':
            obj = entities.get('object', '东西')
            return (
                f'好的，我帮你拿{obj}',
                [{'action': 'grasp', 'params': {'object_name': obj}}]
            )
        elif intent == 'greeting':
            return (
                '你好！有什么可以帮你的吗？',
                []
            )
        else:
            return (
                '抱歉，我没有理解你的意思',
                []
            )
```
