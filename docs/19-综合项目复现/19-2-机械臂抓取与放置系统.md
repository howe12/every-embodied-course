## 1. 项目概述与目标

### 1.1 项目背景

机械臂抓取与放置是具身智能领域的核心技术之一，广泛应用于工业自动化、物流分拣、家庭服务等场景。与纯视觉导航不同，抓取系统需要精确的**几何推理**和**运动控制**能力——机器人不仅要"看见"目标物体，还要判断"从哪里抓"、"怎么抓"、"放到哪里"。

```
┌──────────────────────────────────────────────────────────────┐
│              机械臂抓取与放置系统核心问题链                     │
└──────────────────────────────────────────────────────────────┘

用户指令："把桌上的红色方块放到左边的盒子里"

  ┌─────────────┐    ┌──────────────┐    ┌────────────────┐
  │  视觉感知    │ -> │  抓取规划    │ -> │  运动执行      │
  │  "在哪里"   │    │  "怎么抓"   │    │  "动起来"     │
  └─────────────┘    └──────────────┘    └────────────────┘
       ↓                   ↓                    ↓
  RGB-D获取          抓取点计算            关节轨迹跟踪
  目标检测           碰撞检测              力控反馈
  3D定位             姿态优化              放置确认
```

### 1.2 项目目标

本项目目标是构建一套**完整的机械臂抓取与放置系统**，实现以下能力：

| 能力 | 描述 | 技术支撑 |
|------|------|---------|
| **视觉感知** | 目标检测、分割、3D定位 | YOLOv8 + RealSense D455 |
| **手眼标定** | 相机与机械臂末端坐标变换 | OpenCV + ROS2 标定工具 |
| **抓取检测** | 计算最优抓取姿态 | Grasp Pose Detection (GPD) |
| **运动规划** | 避障路径规划与关节控制 | MoveIt2 |
| **闭环控制** | 抓取反馈与放置确认 | 力矩传感器 + 视觉反馈 |
| **任务调度** | 多目标协调与任务序列 | ROS2 Action Server |

## 3. 手眼标定与相机配置

### 3.1 手眼标定原理

手眼标定是建立相机坐标系与机械臂末端坐标系之间变换关系的关键步骤。根据相机安装位置不同，分为**眼在手上（Eye-in-Hand）** 和**眼在外（Eye-to-Hand）** 两种方式：

```
┌──────────────────────────────────────────────────────────────┐
│                    手眼标定类型对比                            │
└──────────────────────────────────────────────────────────────┘

  眼在手上 (Eye-in-Hand)          眼在外 (Eye-to-Hand)
  
       ┌─────────┐                    ┌─────────┐
       │ 相机     │                    │  相机   │
       │ ↗︎       │                    │         │
       │ ↗︎       │                    │ ↗︎ ↗︎ ↗︎ │
       │↗︎        │                    │         │
  ─────┤ 末端    │                    │────┴────│
       └─────────┘                    │   末端  │
                                      └─────────┘
  相机随机械臂移动                    相机固定安装
  标定更频繁                          标定一次即可
```

手眼标定的核心方程：

$$
AX = XB
$$

其中：
- $A$：机械臂两次运动之间的末端变换矩阵
- $B$：相机两次观察标定板的位姿变换矩阵
- $X$：相机到机械臂末端的变换矩阵（待求）

### 3.2 相机配置代码

```python
"""
RealSense D455 相机配置节点
眼在手上（Eye-in-Hand）安装方式
"""

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image, CameraInfo
from geometry_msgs.msg import TransformStamped, Pose
import numpy as np
import cv2
from cv_bridge import CvBridge
import pyrealsense2 as rs


class RealSenseEyeInHand(Node):
    """
    眼在手上相机配置节点
    
    功能：
    - 配置 RealSense D455 相机参数
    - 发布对齐后的 RGB-D 图像
    - 提供相机内参和深度缩放参数
    """
    
    def __init__(self):
        super().__init__('realsense_eye_in_hand')
        
        # 声明参数
        self.declare_parameter('camera_name', 'd455')
        self.declare_parameter('depth_width', 640)
        self.declare_parameter('depth_height', 480)
        self.declare_parameter('color_width', 640)
        self.declare_parameter('color_height', 480)
        self.declare_parameter('fps', 30)
        self.declare_parameter('depth_align_to_color', True)
        
        self.camera_name = self.get_parameter('camera_name').get_parameter_value().string_value
        
        # 初始化 RealSense 管道
        self.pipeline = rs.pipeline()
        self.config = rs.config()
        
        # 使能深度和彩色流
        self.config.enable_stream(
            rs.stream.depth,
            self.get_parameter('depth_width').get_parameter_value().integer_value,
            self.get_parameter('depth_height').get_parameter_value().integer_value,
            rs.format.z16,
            self.get_parameter('fps').get_parameter_value().integer_value
        )
        
        self.config.enable_stream(
            rs.stream.color,
            self.get_parameter('color_width').get_parameter_value().integer_value,
            self.get_parameter('color_height').get_parameter_value().integer_value,
            rs.format.bgr8,
            self.get_parameter('fps').get_parameter_value().integer_value
        )
        
        # 启动管道
        self.profile = self.pipeline.start(self.config)
        
        # 获取深度传感器参数
        depth_sensor = self.profile.get_device().first_depth_sensor()
        self.depth_scale = depth_sensor.get_depth_scale()  # 深度缩放因子（米/单位）
        
        self.get_logger().info(f'相机启动成功，深度缩放因子: {self.depth_scale}')
        
        # 图像桥接器
        self.bridge = CvBridge()
        
        # 校准器（用于深度与彩色图对齐）
        self.align = rs.align(rs.stream.color) if self.get_parameter('depth_align_to_color').get_parameter_value().bool_value else None
        
        # 发布者
        self.color_pub = self.create_publisher(Image, '/camera/color/image_raw', 10)
        self.depth_pub = self.create_publisher(Image, '/camera/depth/image_rect_raw', 10)
        self.camera_info_pub = self.create_publisher(CameraInfo, '/camera/color/camera_info', 10)
        
        # 定时器
        self.timer = self.create_timer(0.033, self.timer_callback)  # 30Hz
        
    def timer_callback(self):
        """定期获取并发布图像"""
        frames = self.pipeline.wait_for_frames()
        
        # 对齐深度和彩色图
        if self.align:
            aligned_frames = self.align.process(frames)
            color_frame = aligned_frames.get_color_frame()
            depth_frame = aligned_frames.get_depth_frame()
        else:
            color_frame = frames.get_color_frame()
            depth_frame = frames.get_depth_frame()
        
        if not color_frame or not depth_frame:
            return
        
        # 转换并发布彩色图
        color_image = np.asanyarray(color_frame.get_data())
        color_msg = self.bridge.cv2_to_imgmsg(color_image, encoding='bgr8')
        color_msg.header.stamp = self.get_clock().now().to_msg()
        color_msg.header.frame_id = 'camera_color_optical_frame'
        self.color_pub.publish(color_msg)
        
        # 转换并发布深度图（米）
        depth_image = np.asanyarray(depth_frame.get_data()) * self.depth_scale
        depth_msg = self.bridge.cv2_to_imgmsg(depth_image.astype(np.float32), encoding='32FC1')
        depth_msg.header.stamp = color_msg.header.stamp
        depth_msg.header.frame_id = 'camera_depth_optical_frame'
        self.depth_pub.publish(depth_msg)
        
        # 发布相机内参
        intrinsics = color_frame.profile.as_video_stream_profile().intrinsics
        camera_info_msg = CameraInfo()
        camera_info_msg.header = color_msg.header
        camera_info_msg.width = intrinsics.width
        camera_info_msg.height = intrinsics.height
        camera_info_msg.k = [
            intrinsics.fx, 0, intrinsics.ppx,
            0, intrinsics.fy, intrinsics.ppy,
            0, 0, 1
        ]
        camera_info_msg.distortion_model = 'plumb_bob'
        camera_info_msg.d = list(intrinsics.coeffs)
        self.camera_info_pub.publish(camera_info_msg)
    
    def get_camera_extrinsics(self):
        """获取相机外参（相对于机械臂末端）"""
        # 从 ROS2 参数服务器获取手眼标定结果
        # 实际使用时通过标定程序计算得到
        pass
    
    def destroy_node(self):
        self.pipeline.stop()
        super().destroy_node()


def main(args=None):
    rclpy.init(args=args)
    node = RealSenseEyeInHand()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        node.get_logger().info('节点关闭中...')
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

## 4. 目标检测与3D定位

### 4.1 YOLOv8 目标检测

```python
"""
YOLOv8 目标检测节点
检测工作台上的目标物体，提供 2D 边界框
"""

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
from vision_msgs.msg import Detection2DArray, Detection2D, BoundingBox2D
from geometry_msgs.msg import PoseStamped, Point
from cv_bridge import CvBridge
import numpy as np
import cv2

try:
    from ultralytics import YOLO
    YOLO_AVAILABLE = True
except ImportError:
    YOLO_AVAILABLE = False


class YOLODetectionNode(Node):
    """
    YOLOv8 目标检测节点
    
    功能：
    - 订阅 RGB 图像
    - 运行 YOLOv8 目标检测
    - 发布 2D 检测结果和 3D 位置
    """
    
    def __init__(self):
        super().__init__('yolo_detection_node')
        
        # 声明参数
        self.declare_parameter('model_path', 'yolov8n.pt')
        self.declare_parameter('confidence_threshold', 0.5)
        self.declare_parameter('nms_threshold', 0.45)
        self.declare_parameter('target_classes', ['cup', 'bottle', 'bowl', 'box', 'cube'])
        
        self.model_path = self.get_parameter('model_path').get_parameter_value().string_value
        self.conf_thres = self.get_parameter('confidence_threshold').get_parameter_value().double_value
        self.nms_thres = self.get_parameter('nms_threshold').get_parameter_value().double_value
        self.target_classes = self.get_parameter('target_classes').get_parameter_value().string_array_value
        
        # 加载 YOLOv8 模型
        if YOLO_AVAILABLE:
            self.model = YOLO(self.model_path)
            self.get_logger().info(f'YOLOv8 模型加载成功: {self.model_path}')
        else:
            self.model = None
            self.get_logger().warn('ultralytics 未安装，使用模拟检测')
        
        # 相机内参（从相机标定获取）
        self.camera_intrinsics = {
            'fx': 615.0, 'fy': 615.0,
            'cx': 320.0, 'cy': 240.0
        }
        
        # 相机外参（手眼标定结果，相机到机械臂末端）
        self.camera_to_ee = np.eye(4)  # 待标定
        
        # 机械臂末端在基坐标系下的位姿（订阅 TF 或 URScript）
        self.ee_pose_in_base = None
        
        # 图像桥接器
        self.bridge = CvBridge()
        
        # 订阅者
        self.image_sub = self.create_subscription(
            Image,
            '/camera/color/image_raw',
            self.image_callback, 10
        )
        
        self.depth_sub = self.create_subscription(
            Image,
            '/camera/depth/image_rect_raw',
            self.depth_callback, 10
        )
        
        # 发布者
        self.detection_pub = self.create_publisher(Detection2DArray, '/detections_2d', 10)
        self.objects_3d_pub = self.create_publisher(PoseStamped, '/detected_objects_3d', 10)
        
        # 最新深度图缓存
        self.latest_depth = None
        
        self.get_logger().info('YOLOv8 检测节点已启动')
    
    def depth_callback(self, msg: Image):
        """缓存最新深度图"""
        try:
            self.latest_depth = self.bridge.imgmsg_to_cv2(msg, desired_encoding='32FC1')
        except Exception as e:
            self.get_logger().error(f'深度图处理失败: {e}')
    
    def image_callback(self, msg: Image):
        """图像回调，处理目标检测"""
        try:
            cv_image = self.bridge.imgmsg_to_cv2(msg, desired_encoding='bgr8')
            
            # YOLOv8 推理
            if self.model is not None:
                results = self.model(cv_image, verbose=False)[0]
                detections = self.process_yolo_results(results)
            else:
                detections = self.mock_detections()
            
            # 发布 2D 检测结果
            detection_msg = Detection2DArray()
            detection_msg.header = msg.header
            detection_msg.detections = detections
            self.detection_pub.publish(detection_msg)
            
            # 计算并发布 3D 位置
            if self.latest_depth is not None:
                self.publish_3d_poses(detections, msg.header)
            
        except Exception as e:
            self.get_logger().error(f'检测失败: {e}')
    
    def process_yolo_results(self, results):
        """解析 YOLOv8 检测结果"""
        detections = []
        
        boxes = results.boxes
        for box in boxes:
            # 类别过滤
            class_id = int(box.cls[0])
            class_name = results.names[class_id]
            
            if class_name not in self.target_classes:
                continue
            
            # 置信度过滤
            conf = float(box.conf[0])
            if conf < self.conf_thres:
                continue
            
            # 边界框
            x1, y1, x2, y2 = box.xyxy[0].cpu().numpy()
            
            detection = Detection2D()
            detection.bbox.center.x = float((x1 + x2) / 2)
            detection.bbox.center.y = float((y1 + y2) / 2)
            detection.bbox.size_x = float(x2 - x1)
            detection.bbox.size_y = float(y2 - y1)
            detection.results.append(class_name)
            detection.confidence = conf
            
            detections.append(detection)
        
        return detections
    
    def mock_detections(self):
        """模拟检测（测试用）"""
        detection = Detection2D()
        detection.bbox.center.x = 320.0
        detection.bbox.center.y = 240.0
        detection.bbox.size_x = 100.0
        detection.bbox.size_y = 100.0
        detection.results.append('cube')
        detection.confidence = 0.9
        return [detection]
    
    def depth_to_3d(self, u, v, depth):
        """
        将像素坐标和深度转换为相机坐标系下的 3D 坐标
        
        Args:
            u, v: 像素坐标
            depth: 深度值（米）
        
        Returns:
            (x, y, z): 相机坐标系下的 3D 坐标
        """
        fx = self.camera_intrinsics['fx']
        fy = self.camera_intrinsics['fy']
        cx = self.camera_intrinsics['cx']
        cy = self.camera_intrinsics['cy']
        
        x = (u - cx) * depth / fx
        y = (v - cy) * depth / fy
        z = depth
        
        return (x, y, z)
    
    def camera_to_robot_pose(self, x_cam, y_cam, z_cam):
        """
        将相机坐标系下的坐标转换为机械臂基坐标系
        
        变换链：相机 -> 末端 -> 基座
        """
        # 点在相机坐标系下
        point_cam = np.array([x_cam, y_cam, z_cam, 1.0])
        
        # 相机到末端（从标定结果）
        T_cam_ee = self.camera_to_ee
        
        # 末端到基座（需要从 TF 或 URScript 获取）
        if self.ee_pose_in_base is None:
            return None
        
        T_ee_base = self.ee_pose_in_base
        
        # 组合变换
        T_cam_base = T_ee_base @ T_cam_ee
        point_base = T_cam_base @ point_cam
        
        return point_base[:3]
    
    def publish_3d_poses(self, detections, header):
        """发布检测物体的 3D 位置"""
        if self.latest_depth is None:
            return
        
        for det in detections:
            u = int(det.bbox.center.x)
            v = int(det.bbox.center.y)
            
            # 获取该位置的深度
            if 0 <= v < self.latest_depth.shape[0] and 0 <= u < self.latest_depth.shape[1]:
                depth = self.latest_depth[v, u]
                
                if depth > 0 and depth < 5.0:  # 有效深度范围
                    x_cam, y_cam, z_cam = self.depth_to_3d(u, v, depth)
                    
                    # 转换到机械臂基座标系
                    pose_base = self.camera_to_robot_pose(x_cam, y_cam, z_cam)
                    
                    if pose_base is not None:
                        pose_msg = PoseStamped()
                        pose_msg.header = header
                        pose_msg.header.frame_id = 'base_link'
                        pose_msg.pose.position.x = pose_base[0]
                        pose_msg.pose.position.y = pose_base[1]
                        pose_msg.pose.position.z = pose_base[2]
                        self.objects_3d_pub.publish(pose_msg)


def main(args=None):

        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
            'fx': msg.k[0], 'fy': msg.k[4],
            'cx': msg.k[2], 'cy': msg.k[5]
        }
    
    def depth_callback(self, msg: Image):
        """从深度图生成点云"""
        try:
            depth = self.bridge.imgmsg_to_cv2(msg, desired_encoding='32FC1')
            self.latest_depth = depth
            
            if self.camera_intrinsics is None:
                return
            
            # 生成点云
            height, width = depth.shape
            points = []
            
            for v in range(0, height, 2):  # 下采样
                for u in range(0, width, 2):
                    d = depth[v, u]
                    if 0 < d < 5.0:  # 有效深度范围
                        x = (u - self.camera_intrinsics['cx']) * d / self.camera_intrinsics['fx']
                        y = (v - self.camera_intrinsics['cy']) * d / self.camera_intrinsics['fy']
                        z = d
                        points.append([x, y, z, 1.0])
            
            if points:
                pc_msg = pc2.create_cloud_float32(
                    header=msg.header,
                    fields=['x', 'y', 'z', 'rgb'],
                    points=points
                )
                pc_msg.header.frame_id = 'camera_color_optical_frame'
                self.pointcloud_pub.publish(pc_msg)
            
        except Exception as e:
            self.get_logger().error(f'点云生成失败: {e}')
    
    def euclidean_clustering(self, points, tolerance=0.02, min_size=100, max_size=25000):
        """
        简单的欧几里得聚类
        
        实际推荐使用 pcl_ros 或 open3d 的欧几里得聚类
        这里展示简化版本
        """
        # 简化的基于体素的聚类
        voxel_size = 0.01
        voxels = {}
        
        for p in points:
            vx = int(p[0] / voxel_size)
            vy = int(p[1] / voxel_size)
            vz = int(p[2] / voxel_size)
            key = (vx, vy, vz)
            
            if key not in voxels:
                voxels[key] = []
            voxels[key].append(p)
        
        # 过滤太小或太大的聚类
        clusters = [v for v in voxels.values() if min_size <= len(v) <= max_size]
        
        return clusters
    
    def compute_cluster_center(self, cluster_points):
        """计算聚类中心"""
        points_array = np.array(cluster_points)
        center = np.mean(points_array[:, :3], axis=0)
        return center.tolist()
```

## 5. 抓取姿态检测

### 5.1 抓取检测原理

抓取检测的核心是计算**抓取姿态（Grasp Pose）**，即夹爪在空间中应该到达的位置和方向：

```
┌──────────────────────────────────────────────────────────────┐
│                    抓取姿态表示法                              │
└──────────────────────────────────────────────────────────────┘

                    抓取姿态 = (位置, 方向, 开口宽度)
                    
                         夹爪
                         ↕ 开口宽度
                    ┌─────┼─────┐
                    │     │     │
                    │     │     │
                    │     ▼     │  ↔ 旋转角（绕深度轴）
                    │   物体    │
                    │         │
                    └─────────┘
                    
    位置: (x, y, z) - 夹爪中心在 3D 空间中的位置
    方向: (roll, pitch, yaw) - 夹爪的空间朝向
    开口宽度: w - 夹爪张开宽度（通常根据物体大小确定）
```

### 5.2 基于几何的抓取检测

```python
"""
抓取姿态检测节点
基于 YOLO 检测框和深度图计算抓取姿态
"""

import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image
from vision_msgs.msg import Detection2DArray
from geometry_msgs.msg import PoseArray, Pose, Quaternion
from std_msgs.msg import Float32MultiArray
import numpy as np
from cv_bridge import CvBridge


class GraspPoseDetector(Node):
    """
    抓取姿态检测节点
    
    方法：基于几何推理的抓取检测
    1. 获取 YOLO 检测的 2D 边界框
    2. 从深度图获取物体 3D 点云
    3. 计算物体主轴和尺寸
    4. 生成候选抓取姿态
    5. 评估并选择最优抓取姿态
    """
    
    def __init__(self):
        super().__init__('grasp_pose_detector')
        
        # 声明参数
        self.declare_parameter('gripper_width', 0.08)  # 夹爪最大开口宽度（米）
        self.declare_parameter('approach_height', 0.10)  # 接近高度（米）
        self.declare_parameter('grasp_depth_offset', 0.02)  # 抓取深度偏移
        self.declare_parameter('num_candidates', 10)  # 候选姿态数量
        self.declare_parameter('camera_height', 0.35)  # 相机安装高度（眼在手上）
        
        self.gripper_width = self.get_parameter('gripper_width').get_parameter_value().double_value
        self.approach_height = self.get_parameter('approach_height').get_parameter_value().double_value
        self.grasp_depth_offset = self.get_parameter('grasp_depth_offset').get_parameter_value().double_value
        self.num_candidates = self.get_parameter('num_candidates').get_parameter_value().integer_value
        self.camera_height = self.get_parameter('camera_height').get_parameter_value().double_value
        
        self.bridge = CvBridge()
        
        # 订阅
        self.detection_sub = self.create_subscription(
            Detection2DArray, '/detections_2d', self.detection_callback, 10)
        self.depth_sub = self.create_subscription(
            Image, '/camera/depth/image_rect_raw', self.depth_callback, 10)
        
        # 发布
        self.grasp_poses_pub = self.create_publisher(PoseArray, '/candidate_grasps', 10)
        self.best_grasp_pub = self.create_publisher(Pose, '/best_grasp', 10)
        self.grasp_quality_pub = self.create_publisher(Float32MultiArray, '/grasp_quality', 10)
        
        self.latest_depth = None
        
        self.get_logger().info('抓取姿态检测节点已启动')
    
    def depth_callback(self, msg: Image):
        """更新深度图"""
        self.latest_depth = self.bridge.imgmsg_to_cv2(msg, desired_encoding='32FC1')
    
    def detection_callback(self, msg: Detection2DArray):
        """处理检测结果，计算抓取姿态"""
        if self.latest_depth is None:
            return
        
        all_grasps = []
        all_qualities = []
        
        for detection in msg.detections:
            # 提取边界框
            u_center = int(detection.bbox.center.x)
            v_center = int(detection.bbox.center.y)
            width = int(detection.bbox.size_x)
            height = int(detection.bbox.size_y)
            
            u_min = max(0, u_center - width            u_min = max(0, u_center - width // 2)
            u_max = min(self.latest_depth.shape[1], u_center + width // 2)
            v_min = max(0, v_center - height // 2)
            v_max = min(self.latest_depth.shape[0], v_center + height // 2)
            
            # 提取物体点云（在边界框内）
            object_depth = self.latest_depth[v_min:v_max, u_min:u_max]
            points_3d = self.extract_3d_points(u_min, v_min, object_depth)
            
            if len(points_3d) < 50:
                continue
            
            # 计算物体中心
            center = np.mean(points_3d, axis=0)
            
            # 计算物体主轴方向（PCA）
            principal_axis = self.compute_principal_axis(points_3d)
            
            # 生成候选抓取姿态
            candidates = self.generate_grasp_candidates(center, principal_axis)
            
            # 评估并排序
            for grasp, quality in candidates:
                all_grasps.append(grasp)
                all_qualities.append(quality)
        
        if not all_grasps:
            return
        
        # 发布所有候选姿态
        poses_msg = PoseArray()
        poses_msg.header.stamp = self.get_clock().now().to_msg()
        poses_msg.header.frame_id = 'base_link'
        poses_msg.poses = [g[0] for g in all_grasps]
        self.grasp_poses_pub.publish(poses_msg)
        
        # 发布最优抓取
        best_idx = np.argmax(all_qualities)
        self.best_grasp_pub.publish(all_grasps[best_idx][0])
        
        # 发布质量分数
        quality_msg = Float32MultiArray()
        quality_msg.data = [float(q) for q in all_qualities]
        self.grasp_quality_pub.publish(quality_msg)
    
    def extract_3d_points(self, u_offset, v_offset, depth_roi):
        """从深度 ROI 提取 3D 点"""
        fx, fy = 615.0, 615.0
        cx, cy = 320.0, 240.0
        
        points = []
        for v in range(0, depth_roi.shape[0], 2):
            for u in range(0, depth_roi.shape[1], 2):
                d = depth_roi[v, u]
                if 0.05 < d < 2.0:  # 有效深度范围（5cm - 2m）
                    x = (u + u_offset - cx) * d / fx
                    y = (v + v_offset - cy) * d / fy
                    z = d
                    points.append([x, y, z])
        
        return np.array(points)
    
    def compute_principal_axis(self, points):
        """使用 PCA 计算物体主轴"""
        # 中心化
        centered = points - np.mean(points, axis=0)
        
        # 协方差矩阵
        cov = np.cov(centered.T)
        
        # 特征值分解
        eigenvalues, eigenvectors = np.linalg.eig(cov)
        
        # 最大特征值对应的特征向量为主轴
        principal_axis = eigenvectors[:, np.argmax(eigenvalues)]
        
        return principal_axis
    
    def generate_grasp_candidates(self, center, principal_axis):
        """
        生成候选抓取姿态
        
        每个候选包含：(pose, quality_score)
        """
        candidates = []
        
        # 抓取方向：垂直于主轴（从两侧抓取）
        grasp_direction = np.array([-1, 0, 0])  # 默认从 -X 方向接近
        
        # 生成不同旋转角度的候选
        for angle in np.linspace(-np.pi/4, np.pi/4, self.num_candidates):
            # 旋转抓取方向
            rotation = self.rotation_matrix_from_axis_angle(principal_axis, angle)
            rotated_direction = rotation @ grasp_direction
            
            # 抓取点：在物体中心沿主轴偏移
            grasp_offset = self.grasp_depth_offset * rotated_direction
            grasp_point = center + grasp_offset
            
            # 构建姿态
            pose = Pose()
            pose.position.x = float(grasp_point[0])
            pose.position.y = float(grasp_point[1])
            pose.position.z = float(grasp_point[2])
            
            # 计算朝向（四元数）
            orientation = self.direction_to_quaternion(rotated_direction)
            pose.orientation = orientation
            
            # 评估质量（简化的几何评估）
            quality = self.evaluate_grasp_quality(grasp_point, rotated_direction)
            
            candidates.append((pose, quality))
        
        return candidates
    
    def rotation_matrix_from_axis_angle(self, axis, angle):
        """从轴角创建旋转矩阵"""
        axis = axis / np.linalg.norm(axis)
        c = np.cos(angle)
        s = np.sin(angle)
        t = 1 - c
        
        x, y, z = axis
        return np.array([
            [t*x*x + c,   t*x*y - z*s, t*x*z + y*s],
            [t*x*y + z*s, t*y*y + c,   t*y*z - x*s],
            [t*x*z - y*s, t*y*z + x*s, t*z*z + c]
        ])
    
    def direction_to_quaternion(self, direction):
        """将方向向量转换为四元数（夹爪朝向）"""
        from geometry_msgs.msg import Quaternion
        
        # 默认向上方向
        up = np.array([0, 0, 1])
        
        # 旋转轴
        rotation_axis = np.cross(up, direction)
        rotation_axis_norm = np.linalg.norm(rotation_axis)
        
        if rotation_axis_norm < 1e-6:
            # 方向已经朝上
            if np.dot(up, direction) > 0:
                return Quaternion(x=0, y=0, z=0, w=1)
            else:
                return Quaternion(x=1, y=0, z=0, w=0)
        
        rotation_axis = rotation_axis / rotation_axis_norm
        
        # 旋转角度
        dot = np.dot(up, direction)
        angle = np.arccos(np.clip(dot, -1, 1))
        
        # 四元数
        q = Quaternion()
        q.x = rotation_axis[0] * np.sin(angle / 2)
        q.y = rotation_axis[1] * np.sin(angle / 2)
        q.z = rotation_axis[2] * np.sin(angle / 2)
        q.w = np.cos(angle / 2)
        
        return q
    
    def evaluate_grasp_quality(self, grasp_point, grasp_direction):
        """
        评估抓取质量
        
        基于以下因素：
        - 抓取点是否在合理范围
        - 接近方向是否可行
        - 物体尺寸是否适合夹爪
        """
        quality = 0.8  # 基础分数
        
        # 距离惩罚（太远或太近都不好）
        distance_penalty = abs(grasp_point[2] - 0.3) * 0.5
        quality -= distance_penalty
        
        # 方向惩罚（优先水平接近）
        horizontal_component = np.sqrt(grasp_direction[0]**2 + grasp_direction[1]**2)
        direction_score = horizontal_component
        quality *= direction_score
        
        return max(0.0, min(1.0, quality))


def main(args=None):
    rclpy.init(args=args)
    node = GraspPoseDetector()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        node.get_logger().info('节点关闭中...')
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
        Args:
            target_pose: 目标抓取姿态（geometry_msgs/Pose）
            approach_height: 接近时的高度偏移
        """
        # 简化的笛卡尔路径规划
        # 1. 计算预抓取位置（在目标上方）
        pre_grasp_pose = self.compute_pre_grasp_pose(target_pose, approach_height)
        
        # 2. 运动序列
        waypoints = [
            self.get_home_position(),  # 初始位置
            pre_grasp_pose,             # 预抓取位置
            target_pose,               # 抓取位置
        ]
        
        # 简化的关节空间规划（实际应使用 MoveIt2 的 Plan / Execute）
        for i, waypoint in enumerate(waypoints):
            self.get_logger().info(f'执行路径点 {i+1}/{len(waypoints)}')
            
            # 这里应该调用 MoveIt2 的规划服务
            # 简化版本：直接发送关节目标
            joint_goal = self.pose_to_joint_target(waypoint)
            
            success = self.send_joint_trajectory(joint_goal)
            
            if not success:
                self.get_logger().error(f'路径点 {i+1} 执行失败')
                return False
            
            # 到达抓取位置后等待稳定
            if i == 1:  # 预抓取位置后
                self.get_logger().info('到达预抓取位置，等待稳定...')
        
        return True
    
    def compute_pre_grasp_pose(self, grasp_pose, height_offset):
        """
        计算预抓取位置（目标上方）

        """
        pre_pose = Pose()
        pre_pose.position.x = grasp_pose.position.x
        pre_pose.position.y = grasp_pose.position.y
        pre_pose.position.z = grasp_pose.position.z + height_offset

        这里使用简化的近似
```

```python
整合所有模块，协调完成抓取任务

    抓取系统主控节点

    - 监听用户指令
    - 协调检测、规划、执行模块

        self.get_logger().info('抓取系统主控节点已启动')
        self.get_logger().info(f'状态: {self.state}')
    
    def grasp_pose_callback(self, msg: Pose):
        """收到抓取姿态"""
        if self.state != 'DETECTING':
            return
        
        self.current_grasp_pose = msg
        self.get_logger().info(f'收到抓取姿态: ({msg.position.x:.3f}, {msg.position.y:.3f}, {msg.position.z:.3f})')
        
        # 开始抓取
        self.state = 'GRASPING'
        self.publish_state()
        
        # 执行抓取（实际应调用 arm controller）
        success = self.execute_grasp(msg)
        
        if success:
            self.state = 'PLACING'
            self.publish_state()
            self.execute_place()
            self.state = 'COMPLETED'
        else:
            self.state = 'ERROR'
            self.handle_grasp_failure()
        
        self.publish_state()

        """
        执行抓取动作
        
        实际应调用 MoveIt2 运动规划和执行
        """
        self.get_logger().info('执行抓取...')
        
        # 模拟抓取过程
        # 1. 移动到预抓取位置
        # 2. 打开夹爪
        # 3. 下降到抓取位置
        # 4. 闭合夹爪
        # 5. 抬起
        
        for step in ['预抓取', '下降', '抓取', '抬起']:
            self.get_logger().info(f'  - {step}...')
        
        return True

        self.get_logger().info('执行放置...')
        
        place_x = self.get_parameter('place_location_x').get_parameter_value().double_value
        place_y = self.get_parameter('place_location_y').get_parameter_value().double_value

        # 模拟放置过程
        for step in ['移动到放置位置', '下降', '打开夹爪', '抬起']:
            self.get_logger().info(f'  - {step}...')

        """抓取结果反馈"""
        if msg.data:
            self.get_logger().info('抓取成功！')
        else:
            self.get_logger().warn('抓取失败')
            self.handle_grasp_failure()

        self.grasp_attempts += 1

            self.state = 'DETECTING'
            self.publish_state()
        else:
            self.get_logger().error('达到最大重试次数，抓取失败')
            self.state = 'ERROR'
            self.publish_state()

        """发布当前状态"""
        state_msg = String()
        state_msg.data = self.state
        self.system_state_pub.publish(state_msg)

        node.get_logger().info('节点关闭中...')
```

### 答案 1：手眼标定矩阵计算

**问题分析**：
手眼标定方程 $AX = XB$ 的含义：
- $A$：机械臂末端从姿态 1 到姿态 2 的相对变换
- $B$：相机从观察位置 1 到位置 2 观察标定板的相对变换
- $X$：相机坐标系到机械臂末端坐标系的变换（待求）

**求解方法**：
使用 Tsai-Lenz 算法或迭代方法求解。简化情况下：

$$X = A^{-1} \cdot X \cdot B$$

**简化计算**：
在给定的简化矩阵下（假设只有平移）：

$$
A = \begin{bmatrix} I & t_A \\ 0 & 1 \end{bmatrix}, 
B = \begin{bmatrix} I & t_B \\ 0 & 1 \end{bmatrix}
$$

其中 $t_A = (0.1, 0, 0)$，$t_B = (0.05, 0, 0)$

对于纯平移情况：
$$X_t = (I - R_B)^{-1} \cdot t_B + t_A$$

但由于 $R_B = I$（无旋转），需要使用完整的 Tsai-Lenz 算法。

**实际求解代码**：
```python
import numpy as np
from scipy.spatial.transform import Rotation

def solve_hand_eye_calibration(A_poses, B_poses):
    """
    求解手眼标定 AX = XB
    
    Args:
        A_poses: 机械臂末端位姿列表（4x4 矩阵）
        B_poses: 相机观察标定板位姿列表（4x4 矩阵）
    
    Returns:
        X: 相机到末端的变换矩阵
    """
    # 提取旋转和平移
    R_A = [p[:3, :3] for p in A_poses]
    t_A = [p[:3, 3] for p in A_poses]
    R_B = [p[:3, :3] for p in B_poses]
    t_B = [p[:3, 3] for p in B_poses]
    
    # 构建方程组求解旋转矩阵
    # 使用均值法
    Z, X_rot = estimate_rotation(R_A, R_B)
    
    # 求解平移向量
    t_AB = estimate_translation(R_A, t_A, t_B, X_rot)
    
    # 组合结果
    X = np.eye(4)
    X[:3, :3] = X_rot
    X[:3, 3] = t_AB
    
    return X
```

### 答案 2：3D 抓取姿态规划

**1. 物体中心在末端坐标系下的位置**

相机到末端的偏移：$T_{cam}^{ee} = (0, 0, 0.1)$（相机在末端下方 0.1m）

$$P_{cam} = (0.1, 0.05, 0.5)$$

由于相机相对于末端有偏移，末端在相机坐标系下的位置为：
$$P_{ee}^{cam} = (0, 0, -0.1)$$

$$P_{ee} = (0.1, 0.05, 0.5 - 0.1) = (0.1, 0.05, 0.4)$$

**2. 夹爪目标位置和姿态**

夹爪从正上方 10cm 处垂直抓取：

目标位置：
$$P_{grasp} = (0.1, 0.05, 0.4 + 0.1) = (0.1, 0.05, 0.5)$$

目标姿态（夹爪朝下，即 -Z 方向）：
- 夹爪 Z 轴指向：$(0, 0, -1)$
- 夹爪 X 轴（默认）：由于绕 Z 旋转 0°，保持 $(1, 0, 0)$

四元数计算：
```python
import numpy as np
from scipy.spatial.transform import Rotation

def pose_to_quaternion(position, z_direction):
    """
    根据位置和 Z 轴方向计算四元数
    
    Args:
        position: 位置 (x, y, z)
        z_direction: Z 轴方向（归一化）
    
    Returns:
        geometry_msgs/Pose
    """
    from geometry_msgs.msg import Pose, Quaternion
    
    # Z 轴朝下
    target_z = np.array([0, 0, -1])
    z = np.array(z_direction) / np.linalg.norm(z_direction)
    
    # 旋转轴
    rotation_axis = np.cross(target_z, z)
    angle = np.arccos(np.clip(np.dot(target_z, z), -1, 1))
    
    if np.linalg.norm(rotation_axis) < 1e-6:
        rot = Rotation.from_rotvec(0)
    else:
        rotation_axis = rotation_axis / np.linalg.norm(rotation_axis)
        rot = Rotation.from_rotvec(angle * rotation_axis)
    
    quat = rot.as_quat()  # (x, y, z, w)
    
    pose = Pose()
    pose.position.x = position[0]
    pose.position.y = position[1]
    pose.position.z = position[2]
    pose.orientation.x = quat[0]
    pose.orientation.y = quat[1]
    pose.orientation.z = quat[2]
    pose.orientation.w = quat[3]
    
    return pose
```

### 答案 3：ROS2 节点设计

```python
"""
任务调度器节点
管理抓取-放置任务队列
"""

import rclpy
from rclpy.node import Node
from geometry_msgs.msg import PoseArray, Pose
from std_msgs.msg import Bool, String
from std_srvs.srv import Trigger
import numpy as np


class TaskScheduler(Node):
    """
    任务调度器
    
    功能：
    - 接收物体检测和放置位置列表
    - 维护任务队列
    - 按顺序调度抓取-放置任务
    - 处理失败重试
    """
    
    def __init__(self):
        super().__init__('task_scheduler')
        
        # 参数
        self.declare_parameter('max_retries', 3)
        self.max_retries = self.get_parameter('max_retries').get_parameter_value().integer_value
        
        # 状态
        self.detected_objects = []  # 检测到的物体
        self.place_locations = []  # 放置位置
        self.task_queue = []  # 任务队列：(grasp_pose, place_pose)
        self.current_task = None
        self.retry_count = 0
        
        # 订阅
        self.objects_sub = self.create_subscription(
            PoseArray, '/detected_objects_3d', self.objects_callback, 10)
        self.place_sub = self.create_subscription(
            PoseArray, '/place_locations', self.place_callback, 10)
        self.grasp_result_sub = self.create_subscription(
            Bool, '/grasp_result', self.grasp_result_callback, 10)
        
        # 发布
        self.target_grasp_pub = self.create_publisher(Pose, '/target_grasp', 10)
        self.target_place_pub = self.create_publisher(Pose, '/target_place', 10)
        self.task_status_pub = self.create_publisher(String, '/task_status', 10)
        
        # 定时检查任务队列
        self.timer = self.create_timer(0.5, self.check_task_queue)
        
        self.get_logger().info('任务调度器已启动')
    
    def objects_callback(self, msg: PoseArray):
        """接收检测到的物体列表"""
        self.detected_objects = list(msg.poses)
        self.get_logger().info(f'收到 {len(self.detected_objects)} 个检测物体')
        
        # 更新任务队列
        self.update_task_queue()
    
    def place_callback(self, msg: PoseArray):
        """接收有效的放置位置列表"""
        self.place_locations = list(msg.poses)
        self.get_logger().info(f'收到 {len(self.place_locations)} 个放置位置')
        
        # 更新任务队列
        self.update_task_queue()
    
    def update_task_queue(self):
        """根据检测物体和放置位置更新任务队列"""
        if not self.detected_objects or not self.place_locations:
            return
        
        # 简单的贪心匹配：每个物体对应一个放置位置
        self.task_queue = []
        
        num_tasks = min(len(self.detected_objects), len(self.place_locations))
        
        for i in range(num_tasks):
            grasp_pose = self.detected_objects[i]
            place_pose = self.place_locations[i % len(self.place_locations)]
            self.task_queue.append((grasp_pose, place_pose))
        
        self.get_logger().info(f'任务队列已更新，共 {len(self.task_queue)} 个任务')
    
    def check_task_queue(self):
        """检查并执行任务队列中的任务"""
        if self.current_task is not None:
            return  # 正在执行任务
        
        if not self.task_queue:
            return  # 队列为空
        
        # 取下一个任务
        self.current_task = self.task_queue.pop(0)
        grasp_pose, place_pose = self.current_task
        
        self.get_logger().info(f'开始执行任务: 抓取 ({grasp_pose.position.x:.3f}, {grasp_pose.position.y:.3f}, {grasp_pose.position.z:.3f})')
        
        # 发布目标
        self.target_grasp_pub.publish(grasp_pose)
        self.target_place_pub.publish(place_pose)
        
        # 发布状态
        status = String()
        status.data = f'GRASPING object at ({grasp_pose.position.x:.3f}, {grasp_pose.position.y:.3f})'
        self.task_status_pub.publish(status)
    
    def grasp_result_callback(self, msg: Bool):
        """处理抓取结果"""
        if self.current_task is None:
            return
        
        if msg.data:
            self.get_logger().info('抓取成功')
            self.retry_count = 0
            # 任务完成，等待下一个
            self.current_task = None
            
            status = String()
            status.data = 'IDLE'
            self.task_status_pub.publish(status)
        else:
            self.retry_count += 1
            if self.retry_count < self.max_retries:
                self.get_logger().warn(f'抓取失败，重试 ({self.retry_count}/{self.max_retries})')
                # 重新发布抓取目标
                grasp_pose, _ = self.current_task
                self.target_grasp_pub.publish(grasp_pose)
            else:
                self.get_logger().error('达到最大重试次数，跳过当前任务')
                self.retry_count = 0
                self.current_task = None
                
                status = String()
                status.data = 'TASK_FAILED'
                self.task_status_pub.publish(status)


def main(args=None):
    rclpy.init(args=args)
    node = TaskScheduler()
    
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        node.get_logger().info('节点关闭中...')
    finally:
        node.destroy_node()
        rclpy.shutdown()


if __name__ == '__main__':
    main()
```

```python
"""
抓取质量评估器
综合评估抓取姿态的优劣
"""

import numpy as np
from geometry_msgs.msg import Pose


class GraspQualityEvaluator:
    """
    抓取质量评估器
    
    评估指标：
    1. 几何可达性
    2. 接近角度
    3. 抓取稳定性
    4. 碰撞风险
    """
    
    def __init__(self):
        # 机械臂工作空间参数
        self.workspace_min_radius = 0.3   # 最小工作半径（米）
        self.workspace_max_radius = 0.9   # 最大工作半径（米）
        
        # 夹爪参数
        self.max_gripper_width = 0.08  # 最大开口宽度（米）
        self.gripper_length = 0.05     # 夹爪长度（米）
        
        # 权重配置
        self.weights = {
            'reachability': 0.25,
            'approach_angle': 0.25,
            'stability': 0.30,
            'collision_risk': 0.20
        }
    
    def evaluate(self, grasp_pose, object_size_estimate=0.05):
        """
        评估抓取质量
        
        Args:
            grasp_pose: 抓取姿态 (geometry_msgs/Pose)
            object_size_estimate: 物体尺寸估计（米）
        
        Returns:
            quality: 总体质量分数 (0-1)
            details: 各项评分详情
        """
        position = np.array([
            grasp_pose.position.x,
            grasp_pose.position.y,
            grasp_pose.position.z
        ])
        
        # 计算各项指标
        reachability_score = self.evaluate_reachability(position)
        approach_score = self.evaluate_approach_angle(grasp_pose)
        stability_score = self.evaluate_stability(object_size_estimate)
        collision_score = self.evaluate_collision_risk(grasp_pose)
        
        # 综合评分
        quality = (
            self.weights['reachability'] * reachability_score +
            self.weights['approach_angle'] * approach_score +
            self.weights['stability'] * stability_score +
            self.weights['collision_risk'] * collision_score
        )
        
        details = {
            'reachability': reachability_score,
            'approach_angle': approach_score,
            'stability': stability_score,
            'collision_risk': collision_score,
            'total': quality
        }
        
        return quality, details
    
    def evaluate_reachability(self, position):
        """
        评估几何可达性
        
        距离在 0.3m ~ 0.9m 范围内得高分
        """
        distance = np.linalg.norm(position)
        
        if distance < self.workspace_min_radius:
            # 太近：惩罚
            score = distance / self.workspace_min_radius
        elif distance > self.workspace_max_radius:
            # 太远：惩罚
            score = self.workspace_max_radius / distance
        else:
            # 在范围内：满分
            score = 1.0
        
        return max(0.0, min(1.0, score))
    
    def evaluate_approach_angle(self, grasp_pose):
        """
        评估接近角度
        
        夹爪应该从水平方向接近物体（绕 Y 轴旋转约 90°）
        垂直接近得分较低
        """
        # 从四元数提取 Z 轴方向
        q = grasp_pose.orientation
        # 夹爪 Z 轴（在基坐标系下）
        z_axis = self.quaternion_rotate(q, np.array([0, 0, 1]))
        
        # 定义理想接近方向（水平，-X 方向）
        ideal_approach = np.array([-1, 0, 0])
        
        # 计算夹角
        cos_angle = np.dot(z_axis, ideal_approach)
        angle = np.arccos(np.clip(cos_angle, -1, 1))
        
        # 角度越小越好（0° = 理想，90° = 最差）
        # 归一化到 0-1
        score = 1.0 - (angle / np.pi)
        
        return score
    
    def evaluate_stability(self, object_size):
        """
        评估抓取稳定性
        
        夹爪开口宽度应适合物体尺寸
        物体尺寸约为夹爪最大宽度的 0.3-0.7 倍时最优
        """
        optimal_ratio = 0.5  # 最优比例
        tolerance = 0.3      # 容差
        
        ratio = object_size / self.max_gripper_width
        
        # 高斯衰减
        diff = abs(ratio - optimal_ratio)
        score = np.exp(-(diff ** 2) / (2 * tolerance ** 2))
        
        return score
    
    def evaluate_collision_risk(self, grasp_pose):
        """
        评估碰撞风险
        
        检查抓取路径是否可能发生碰撞
        简化版本：检查目标位置是否在障碍物附近
        """
        # 实际应用中应该进行碰撞检测
        # 这里使用简化的启发式规则
        
        position = np.array([
            grasp_pose.position.x,
            grasp_pose.position.y,
            grasp_pose.position.z
        ])
        
        # 假设工作台高度为 0.4m，物体应该在此高度附近
        table_height = 0.4
        
        # Z 坐标过低可能有碰撞风险
        if position[2] < 0.1:
            score = 0.3
        elif position[2] > table_height + 0.1:
            score = 0.8
        else:
            score = 1.0
        
        return score
    
    @staticmethod
    def quaternion_rotate(q, vector):
        """四元数旋转向量"""
        # 四元数旋转
        q = np.array([q.x, q.y, q.z, q.w])
        
        # 旋转矩阵
        R = np.array([
            [1 - 2*(q[1]**2 + q[2]**2), 2*(q[0]*q[1] - q[2]*q[3]), 2*(q[0]*q[2] + q[1]*q[3])],
            [2*(q[0]*q[1] + q[2]*q[3]), 1 - 2*(q[0]**2 + q[2]**2), 2*(q[1]*q[2] - q[0]*q[3])],
            [2*(q[0]*q[2] - q[1]*q[3]), 2*(q[1]*q[2] + q[0]*q[3]), 1 - 2*(q[0]**2 + q[1]**2)]
        ])
        
        return R @ vector


def select_best_grasp(grasps, object_size_estimate=0.05):
    """
    从候选抓取姿态中选择最优的一个
    
    Args:
        grasps: 候选抓取姿态列表
        object_size_estimate: 物体尺寸估计
    
    Returns:
        best_grasp: 最优抓取姿态
        quality: 对应质量分数
    """
    evaluator = GraspQualityEvaluator()
    
    best_grasp = None
    best_quality = -1
    
    for grasp in grasps:
        quality, _ = evaluator.evaluate(grasp, object_size_estimate)
        
        if quality > best_quality:
            best_quality = quality
            best_grasp = grasp
    
    return best_grasp, best_quality

if __name__ == '__main__':

    grasp.orientation.w = 1.0

    quality, details = evaluator.evaluate(grasp, object_size_estimate=0.04)
```
