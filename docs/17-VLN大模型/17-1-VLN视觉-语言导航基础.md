## 一、课程概述

传统的移动机器人导航依赖预先构建的地图和精确的GPS定位——目标以坐标形式给出，机器人沿规划好的路径移动即可。然而在真实家庭场景中，人们会说"去厨房帮我拿水杯"，而不是"导航到坐标(3.2, 1.8)"。**视觉-语言导航（Vision-Language Navigation，VLN）** 正是要解决这个问题：让机器人"听懂"人类的自然语言指令，通过摄像头感知环境，自主规划路径到达目标位置。

VLN 是具身智能的核心研究领域之一，也是 VLA（Vision-Language-Action）大模型的重要前置任务。与传统导航不同，VLN 需要处理**多模态信息**——同时理解语言指令中的语义意图和视觉观测中的空间关系，并在未知环境中做出序列决策。

本课程系统讲解 VLN 的基础概念、核心问题、主流数据集、评价指标和经典方法，并从代码层面实现一个简化的 VLN 环境、指令编码器、视觉-语言特征融合模块以及基础导航策略。

## 三、理论内容

### 3.1 VLN概述

#### 3.1.1 什么是视觉-语言导航

**视觉-语言导航（Vision-Language Navigation，VLN）** 是指智能体根据自然语言指令，在未知的三维环境中自主导航到目标位置的任务。智能体通过摄像头获取第一人称视角的视觉观测，结合语言指令的语义信息，在每一步做出前进、左转、右转等动作决策。

形式化定义：

设智能体在时刻 $t$ 获得视觉观测 $o_t \in \mathbb{R}^{H \times W \times 3}$ 和语言指令 $L = (w_1, w_2, ..., w_T)$，VLN 的目标是学习一个策略 $\pi(a_t | o_t, L, h_t)$，使得智能体能够成功到达目标位置，其中 $h_t = (o_0, a_0, o_1, a_1, ..., o_{t-1}, a_{t-1})$ 是历史轨迹。

**核心特点**：

| 特点 | 说明 |
|------|------|
| **多模态输入** | 同时处理视觉（RGB/深度图像）和语言（自然语言指令） |
| **序列决策** | 每步决策影响后续状态，是一个长程规划问题 |
| **跨模态理解** | 需要将语言中的语义实体（"厨房"）与视觉中的外观（图像区域）对应 |
| **未知环境** | 智能体一开始不知道环境地图，需要边探索边规划 |

**生活类比**：这就像你第一次去朋友家做客，朋友在电话里告诉你："进了小区大门左转，走到有蓝色招牌的楼门口，电梯到3楼，右手边第二间就是。"你一边听一边观察周围环境，按指示行动，最终找到目的地。这个过程正是 VLN 要解决的问题。

#### 3.1.2 VLN研究意义

1. **自然交互**：人类天然用自然语言描述目标，VLN 让机器人理解"去厨房"、"经过餐桌左转"这类指令。
2. **多模态融合标杆任务**：VLN 需要同时理解视觉和语言，是检验视觉-语言大模型（VLM）能力的重要任务。
3. **VLA 的前置任务**：VLN 训练得到的视觉-语言对齐能力，是 VLA 大模型的核心基础。
4. **具身智能核心能力**：自主导航是机器人完成各种任务的前提，VLN 使机器人能在非结构化环境中灵活行动。

#### 3.1.3 与传统导航的本质区别

| 维度 | 传统导航 | VLN |
|------|----------|-----|
| **目标表示** | GPS坐标、地图标注 | 自然语言描述 |
| **环境先验** | 需要预先构建地图 | 无需地图，边探索边规划 |
| **感知方式** | GPS、激光雷达 | 视觉摄像头 |
| **环境理解** | 几何层面（位置、距离） | 语义层面（物体、功能区域） |
| **典型指令** | "导航到经度xx纬度xx" | "走进厨房，左转，停在餐桌旁" |
| **适用场景** | 室外、道路 | 室内、家庭、服务机器人 |

#### 3.1.4 应用场景

- **家庭服务机器人**：理解"去厨房帮我拿水杯"，在陌生家庭环境中自主导航
- **酒店/商场服务机器人**：接待机器人理解"带我去会议室B"
- **医院送药机器人**：理解"送到312病房"
- **博物馆导览机器人**：根据游客语言问题自动导航到相关展区
- **仓储物流机器人**：理解"去A区3号货架取包裹"

### 3.2 VLN核心问题

VLN 任务面临四大核心挑战。

#### 3.2.1 指令理解

语言指令的不完整性、参照物歧义性、空间关系描述、长程指令记忆，构成指令理解的四大难题：

- **不完整性**：指令如"Go to the bedroom"没有说明从哪出发、经过哪里
- **参照物歧义**："大植物"可能匹配多个视觉物体
- **空间关系**："左边"是相对智能体朝向而言的，需结合当前朝向理解
- **长程记忆**：一条指令包含5-10个步骤，智能体需记住整个指令

**核心方法**：LSTM/Transformer 指令编码、数据增强（同轨迹多条指令）、指令分解为子目标。

#### 3.2.2 视觉感知

- **视觉混淆**：室内场景很多房间看起来相似
- **物体遮挡**：目标物体可能被墙壁、家具遮挡
- **视角变化**：第一人称视角随位置和朝向不断变化
- **光照变化**：室内外光照差异、阴影、反光影响视觉识别

**核心方法**：ResNet/ViT 视觉特征提取、全景表示、外观记忆、深度感知。

#### 3.2.3 动作规划

- **离散 vs 连续动作空间**：离散环境选下一节点；连续环境执行低级动作原语
- **误差累积**：每步误差导致后续观测偏离预期，错误累积可能使智能体完全偏离
- **探索 vs 利用**：需要在遵循指令探索和利用已知信息之间平衡

**核心方法**：航点预测（分阶段规划）、强化学习、图神经网络（GNN）、分层策略（高层去哪、低层怎么走）。

#### 3.2.4 历史记忆

智能体需要记住走过的路径、访问过的位置、完成/未完成的指令部分、失败尝试。

**核心方法**：LSTM/GRU 隐状态编码历史、视觉记忆模块、拓扑记忆图、注意力从历史检索。

### 3.3 主流VLN数据集

#### 3.3.1 R2R数据集——VLN的"Hello World"

**R2R**（Room-to-Room，Anderson et al., 2018）基于 **Matterport3D** 真实室内场景 3D 扫描构建：

- 轨迹数量：6,000+ 条
- 指令数量：21,000+ 条（每条轨迹 3-5 条不同表述指令）
- 平均路径长度：约 10 米
- 环境数量：90 个真实室内场景

任务设定：智能体从起点出发，根据分步导航指令（如"Walk down the stairs, turn left and enter the bedroom."）导航到目标位置。环境为离散导航图。

#### 3.3.2 Matterport3D环境

Matterport3D 是 R2R 等 VLN 数据集的基础 3D 环境：

- 90 个真实室内场景（公寓、办公室、商店等）
- 10,800+ 全景视角（每场景约 120 个视角点）
- 194,400 张 RGB-D 图像
- 108 种语义类别标注

提供 Matterport Simulator（Unity）和 Habitat Simulator（PyTorch 高效版）两种仿真平台。

#### 3.3.3 RxR数据集——多语言过程导向

- **多语言支持**：英语、印地语、泰卢固语
- **过程导向指令**：指令非常详细，像导游描述沿途每个细节
- **时间对齐**：指令中的词与路径位姿有时间戳对应
- **规模**：约 42,000 条连续轨迹，126,000 条指令

#### 3.3.4 CVDN对话导航

**CVDN**（Cooperativ Vision-Dialog Navigation）的创新在于引入**对话交互**：导航过程中智能体可以向远程人类协助者提问，人类根据当前位置给出额外指导。

#### 3.3.5 VLN-CE连续环境

**VLN-CE**（VLN-Continuous Environment）将 R2R 扩展到连续环境：

- 使用 Habitat Simulator 加载 Matterport3D 场景
- 智能体在连续 3D 空间中自由移动
- 动作空间：前进 0.25m、左转 15 度、右转 15 度、停止
- 更接近真实机器人的运动控制

### 3.4 VLN评价指标

#### 3.4.1 核心指标详解

**成功率（Success Rate, SR）**：

定义：智能体最终停在距离目标 3 米以内的比例。

$$
SR = \frac{1}{N} \sum_{i=1}^{N} \mathbb{I}(d(p_{final}^{(i)}, p_{target}^{(i)}) < 3m)
$$

**按长度加权的成功率（SPL）**：

SPL 是 VLN 最重要的综合指标，同时考虑成功率和路径效率：

$$
SPL = \frac{1}{N} \sum_{i=1}^{N} S_i \cdot \frac{l_i}{\max(l_i, p_i)}
$$

其中 $S_i \in \{0, 1\}$ 表示是否成功，$l_i$ 是最短路径长度，$p_i$ 是智能体实际走的路径长度。

**核心思想**：同样成功时，路径越短 SPL 越高；失败时 SPL 为 0。

**导航误差（Navigation Error, NE）**：

$$
NE = \frac{1}{N} \sum_{i=1}^{N} d(p_{final}^{(i)}, p_{target}^{(i)})
$$

**轨迹长度（Trajectory Length, TL）**：

智能体走过的平均路径长度。

**Oracle成功率（OSR）**：

智能体轨迹在任意时刻经过目标位置 3 米范围内的比例。

**指标对比总结**：

| 指标 | 衡量什么 | 理想值 |
|------|----------|--------|
| **SR** | 最终是否到达目标 | 高 |
| **SPL** | 成功 + 路径效率 | 高 |
| **NE** | 距离目标多远 | 低 |
| **TL** | 走了多远 | 与 $l_i$ 相近 |
| **OSR** | 是否经过目标附近 | 高 |

### 3.5 典型VLN模型

#### 3.5.1 Seq2Seq VLN——开创性架构

Seq2Seq VLN（Anderson et al., 2018）是 VLN 领域的开创性工作：

- **指令编码器**：LSTM 将指令编码为隐藏向量序列
- **视觉编码器**：ResNet 提取图像特征
- **跨模态注意力**：在每步解码时让视觉查询语言，确定下一步方向
- **动作解码器**：基于注意力上下文输出动作概率

**问题**：仅使用当前帧视觉信息，没有历史记忆；使用教师强制训练但推理时没有，导致误差累积。

#### 3.5.2 Speaker-Follower——双向学习

**Speaker-Follower**（Fried et al., 2018）提出双向学习框架：

- **Speaker（说话者）**：根据轨迹生成指令（图像→语言），用于数据增强
- **Follower（跟随者）**：根据指令导航（语言→动作），这是主策略

通过 Speaker 生成合成指令扩充训练数据，解决 R2R 数据量不足的问题。

#### 3.5.3 视觉记忆机制

将历史帧的视觉特征存储在记忆模块中，通过注意力从记忆中检索相关信息：

- **LSTM 记忆**：用 LSTM 隐状态编码历史轨迹
- **视觉记忆模块**：额外的记忆向量，通过读写操作维护
- **图记忆**：在离散环境中用图结构存储已访问节点

**代表工作**：RCM（显式建模子目标）、EGP（环境拓扑图全局规划）。

#### 3.5.4 方法分类总结

| 方法类别 | 代表模型 | 核心特点 |
|----------|---------|---------|
| **Seq2Seq** | Seq2Seq VLN | LSTM 编码器-解码器 + 跨模态注意力 |
| **双向学习** | Speaker-Follower | 指令生成 + 导航策略联合训练 |
| **强化学习** | EnvDrop | 环境 Dropout 提高泛化，RL 微调 |
| **预训练** | PREVALENT、VLN-BERT | 大规模视觉-语言预训练 |
| **分层规划** | ETPNav | 高层航点预测 + 低层动作执行 |
| **连续环境** | VLN-CE | HabitatSimulator + 低级动作原语 |

## 四、代码实战

### 4.1 简化版VLN环境

```python
"""
简化版VLN环境实现
基于离散网格世界的VLN任务环境
作者: Wendy
"""
import numpy as np
import random
from typing import Tuple, List, Dict


class SimpleVLNEnvironment:
    """
    简化版VLN环境：
    - 5x5 网格世界，每个格子是房间或走廊
    - 智能体可以向上/下/左/右移动
    - 目标用语言描述（如"去厨房"）
    - 支持视觉观测（当前位置的邻居房间类型）
    """
    
    # 房间类型（语义标签）
    ROOM_TYPES = [
        '起点', '厨房', '客厅', '卧室', '浴室',
        '餐厅', '走廊', '书房', '阳台', '杂物间'
    ]
    
    def __init__(self, grid_size=5, seed=None):
        """
        初始化VLN环境
        Args:
            grid_size: 网格世界的大小（grid_size x grid_size）
            seed: 随机种子
        """
        self.grid_size = grid_size
        self.num_rooms = grid_size * grid_size
        
        if seed is not None:
            random.seed(seed)
            np.random.seed(seed)
        
        self.room_types = self._generate_room_layout()
        # 动作空间：0=上, 1=下, 2=左, 3=右
        self.action_space = 4
        self.action_names = ['上', '下', '左', '右']
        
        self.agent_pos = None
        self.target_pos = None
        self.start_pos = None
        self.current_step = 0
        self.max_steps = 50
        
        self.current_instruction = None
        self.all_landmarks = []
    
    def _generate_room_layout(self) -> np.ndarray:
        """生成房间布局"""
        layout = np.zeros((self.grid_size, self.grid_size), dtype=int)
        
        for r in range(self.grid_size):
            for c in range(self.grid_size):
                is_edge = (r == 0 or r == self.grid_size - 1 or 
                          c == 0 or c == self.grid_size - 1)
                if is_edge:
                    layout[r, c] = random.randint(1, 9)
                else:
                    if random.random() < 0.7:
                        layout[r, c] = 6  # 走廊
                    else:
                        layout[r, c] = random.randint(1, 5)
        return layout
    
    def _get_neighbors(self, pos: Tuple[int, int]) -> List[Tuple[int, int]]:
        """获取相邻格子（4连通）"""
        row, col = pos
        neighbors = []
        if row > 0:
            neighbors.append((row - 1, col))
        if row < self.grid_size - 1:
            neighbors.append((row + 1, col))
        if col > 0:
            neighbors.append((row, col - 1))
        if col < self.grid_size - 1:
            neighbors.append((row, col + 1))
        return neighbors
    
    def _get_visual_observation(self, pos: Tuple[int, int]) -> np.ndarray:
        """
        获取视觉观测：当前房间类型 + 4个方向邻居类型
        返回 one-hot 拼接向量 (50,)
        """
        obs = np.zeros((5, len(self.ROOM_TYPES)))
        
        room_type = self.room_types[pos]
        obs[0, room_type] = 1.0
        
        neighbors = self._get_neighbors(pos)
        for i, neighbor_pos in enumerate(neighbors):
            neighbor_type = self.room_types[neighbor_pos]
            obs[i + 1, neighbor_type] = 1.0
        
        return obs.flatten()
    
    def _generate_instruction(self, start: Tuple[int, int], 
                             target: Tuple[int, int]) -> str:
        """根据起点和目标生成自然语言指令"""
        start_type = self.ROOM_TYPES[self.room_types[start]]
        target_type = self.ROOM_TYPES[self.room_types[target]]
        
        path = self._find_path(start, target)
        middle_landmarks = []
        
        for pos in path[1:-1]:
            room_type = self.ROOM_TYPES[self.room_types[pos]]
            if room_type != '走廊' and room_type not in middle_landmarks:
                middle_landmarks.append(room_type)
        
        if middle_landmarks:
            middle_str = '、'.join(middle_landmarks[:2])
            instruction = f"从{start_type}出发，经过{middle_str}，到达{target_type}。"
        else:
            instruction = f"从{start_type}出发，直行到达{target_type}。"
        
        return instruction
    
    def _find_path(self, start: Tuple[int, int], 
                   target: Tuple[int, int]) -> List[Tuple[int, int]]:
        """BFS寻路"""
        if start == target:
            return [start]
        
        queue = [start]
        visited = {start: None}
        
        while queue:
            current = queue.pop(0)
            if current == target:
                path = []
                pos = target
                while pos is not None:
                    path.append(pos)
                    pos = visited[pos]
                return path[::-1]
            
            for neighbor in self._get_neighbors(current):
                if neighbor not in visited:
                    visited[neighbor] = current
                    queue.append(neighbor)
        
        return [start, target]
    
    def reset(self) -> Tuple[np.ndarray, str, Dict]:
        """重置环境"""
        self.current_step = 0
        
        while True:
            self.start_pos = (
                random.randint(0, self.grid_size - 1),
                random.randint(0, self.grid_size - 1)
            )
            self.target_pos = (
                random.randint(0, self.grid_size - 1),
                random.randint(0, self.grid_size - 1)
            )
            manhattan_dist = (abs(self.start_pos[0] - self.target_pos[0]) + 
                             abs(self.start_pos[1] - self.target_pos[1]))
            if manhattan_dist >= 3:
                break
        
        self.agent_pos = self.start_pos
        self.current_instruction = self._generate_instruction(
            self.start_pos, self.target_pos
        )
        
        path = self._find_path(self.start_pos, self.target_pos)
        self.all_landmarks = []
        for pos in path:
            room_type = self.ROOM_TYPES[self.room_types[pos]]
            if room_type not in self.all_landmarks:
                self.all_landmarks.append(room_type)
        
        observation = self._get_visual_observation(self.agent_pos)
        
        info = {
            'agent_pos': self.agent_pos,
            'target_pos': self.target_pos,
            'start_pos': self.start_pos,
            'instruction': self.current_instruction,
            'grid_size': self.grid_size,
        }
        
        return observation, self.current_instruction, info
    
    def step(self, action: int) -> Tuple[np.ndarray, float, bool, Dict]:
        """执行动作"""
        self.current_step += 1
        
        row, col = self.agent_pos
        
        if action == 0 and row > 0:
            self.agent_pos = (row - 1, col)
        elif action == 1 and row < self.grid_size - 1:
            self.agent_pos = (row + 1, col)
        elif action == 2 and col > 0:
            self.agent_pos = (row, col - 1)
        elif action == 3 and col < self.grid_size - 1:
            self.agent_pos = (row, col + 1)
        
        old_dist = abs(self.agent_pos[0] - self.target_pos[0]) + \
                   abs(self.agent_pos[1] - self.target_pos[1])
        # 先用旧位置算距离
        old_dist = (abs(self.agent_pos[0] - self.target_pos[0]) + 
                    abs(self.agent_pos[1] - self.target_pos[1]))
        
        # 重新算：移动前的旧位置距离
        old_row, old_col = row, col
        if action == 0 and row > 0:
            old_row = row - 1
        elif action == 1 and row < self.grid_size - 1:
            old_row = row + 1
        elif action == 2 and col > 0:
            old_col = col - 1
        elif action == 3 and col < self.grid_size - 1:
            old_col = col + 1
        
        old_dist = abs(old_row - self.target_pos[0]) + abs(old_col - self.target_pos[1])
        new_dist = abs(self.agent_pos[0] - self.target_pos[0]) + \
                   abs(self.agent_pos[1] - self.target_pos[1])
        
        reward = (old_dist - new_dist) * 0.1
        
        terminated = False
        success = False
        if new_dist == 0:
            reward = 10.0
            terminated = True
            success = True
        elif self.current_step >= self.max_steps:
            terminated = True
        
        observation = self._get_visual_observation(self.agent_pos)
        
        info = {
            'agent_pos': self.agent_pos,
            'target_pos': self.target_pos,
            'step': self.current_step,
            'success': success,
            'distance': new_dist,
        }
        
        return observation, reward, terminated, info
    
    def get_instruction_tokens(self, instruction: str) -> List[int]:

        vocab = {

"""
VLN指令编码器与视觉-语言特征融合模块
基于PyTorch实现
作者: Wendy
"""
import torch
import torch.nn as nn
import torch.nn.functional as F
import numpy as np


class InstructionEncoder(nn.Module):
    """
    指令编码器：将自然语言指令编码为向量表示
    
    使用两层LSTM逐词处理指令，最后一个隐状态作为指令表示
    对应原始Seq2Seq VLN（Anderson et al., 2018）的指令编码部分
    """
    def __init__(self, vocab_size=100, embed_dim=64, hidden_dim=128, num_layers=2):
        super().__init__()
        self.vocab_size = vocab_size
        self.embed_dim = embed_dim
        self.hidden_dim = hidden_dim
        
        # 词嵌入层
        self.embedding = nn.Embedding(vocab_size, embed_dim, padding_idx=0)
        
        # LSTM编码器：处理变长指令序列
        self.lstm = nn.LSTM(
            embed_dim, hidden_dim,
            num_layers=num_layers,
            batch_first=True,
            bidirectional=True  # 双向LSTM捕获前后文
        )
        
        # 双向LSTM输出是 hidden_dim*2，用线性层投影回 hidden_dim
        self.projection = nn.Linear(hidden_dim * 2, hidden_dim)
    
    def forward(self, tokens: torch.Tensor) -> torch.Tensor:
        """
        Args:
            tokens: (batch, seq_len) token索引序列
        Returns:
            指令表示: (batch, hidden_dim)
        """
        # 词嵌入
        embedded = self.embedding(tokens)  # (batch, seq_len, embed_dim)
        
        # LSTM编码
        # output: (batch, seq_len, hidden_dim*2)
        # hidden: (num_layers*2, batch, hidden_dim)
        output, (hidden, cell) = self.lstm(embedded)
        
        # 取最后一个时间步的双向隐藏状态拼接
        # hidden[-2:] 是最后一层双向的两个方向
        forward_hidden = hidden[-2]    # (batch, hidden_dim)
        backward_hidden = hidden[-1]   # (batch, hidden_dim)
        
        # 双向拼接
        combined = torch.cat([forward_hidden, backward_hidden], dim=-1)
        # 投影到 hidden_dim
        instruction_repr = torch.tanh(self.projection(combined))
        
        return instruction_repr


class VisualEncoder(nn.Module):
    """
    视觉编码器：将视觉观测编码为特征向量
    
    在简化版VLN环境中，视觉输入是5个房间类型的one-hot向量（50维）
    实际VLN中使用ResNet/ViT提取图像特征
    """
    def __init__(self, input_dim=50, hidden_dim=128):
        super().__init__()
        
        # 简化版：两层全连接网络
        self.encoder = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU()
        )
    
    def forward(self, visual_obs: torch.Tensor) -> torch.Tensor:
        """
        Args:
            visual_obs: (batch, 50) 视觉观测
        Returns:
            视觉特征: (batch, hidden_dim)
        """
        return self.encoder(visual_obs)


class CrossModalAttention(nn.Module):
    """
    跨模态注意力模块：让视觉信息和语言信息交互
    
    这是Seq2Seq VLN的核心组件
    视觉查询语言，语言提供键值对，实现跨模态对齐
    """
    def __init__(self, query_dim=128, key_dim=128, value_dim=128, num_heads=4):
        super().__init__()
        
        self.num_heads = num_heads
        self.head_dim = key_dim // num_heads
        
        # 线性投影层
        self.query_proj = nn.Linear(query_dim, key_dim)
        self.key_proj = nn.Linear(key_dim, key_dim)
        self.value_proj = nn.Linear(value_dim, key_dim)
        
        self.output_proj = nn.Linear(key_dim, query_dim)
        
        self.scale = self.head_dim ** -0.5
    
    def forward(self, query: torch.Tensor, 
                memory: torch.Tensor,
                mask: torch.Tensor = None) -> torch.Tensor:
        """
        Args:
            query: (batch, query_len, query_dim) 查询向量
            memory: (batch, mem_len, key_dim) 记忆/键值对
            mask: (batch, query_len, mem_len) 注意力掩码
        Returns:
            output: (batch, query_len, query_dim)
        """
        batch_size = query.size(0)
        query_len = query.size(1)
        mem_len = memory.size(1)
        
        # 投影
        Q = self.query_proj(query)   # (batch, query_len, key_dim)
        K = self.key_proj(memory)     # (batch, mem_len, key_dim)
        V = self.value_proj(memory)   # (batch, mem_len, key_dim)
        
        # 多头注意力reshape
        Q = Q.view(batch_size, query_len, self.num_heads, self.head_dim).transpose(1, 2)
        K = K.view(batch_size, mem_len, self.num_heads, self.head_dim).transpose(1, 2)
        V = V.view(batch_size, mem_len, self.num_heads, self.head_dim).transpose(1, 2)
        
        # 注意力分数
        attn_scores = torch.matmul(Q, K.transpose(-2, -1)) * self.scale
        
        # 应用mask（如果有）
        if mask is not None:
            attn_scores = attn_scores.masked_fill(mask == 0, -1e9)
        
        # 归一化注意力权重
        attn_weights = F.softmax(attn_scores, dim=-1)
        
        # 加权求和
        attn_output = torch.matmul(attn_weights, V)  # (batch, num_heads, query_len, head_dim)
        
        # 合并多头
        attn_output = attn_output.transpose(1, 2).contiguous()
        attn_output = attn_output.view(batch_size, query_len, -1)
        
        # 输出投影
        output = self.output_proj(attn_output)
        
        return output


class VLNFusionModule(nn.Module):
    """
    VLN视觉-语言特征融合模块
    
    整合指令编码、视觉编码、历史记忆，输出跨模态上下文向量
    """
    def __init__(self, hidden_dim=128, num_heads=4, dropout=0.1):
        super().__init__()
        
        self.hidden_dim = hidden_dim
        
        # 指令编码器
        self.instruction_encoder = InstructionEncoder(
            vocab_size=100, embed_dim=64, hidden_dim=hidden_dim
        )
        
        # 视觉编码器
        self.visual_encoder = VisualEncoder(input_dim=50, hidden_dim=hidden_dim)
        
        # 跨模态注意力：视觉查询语言
        self.v2l_attention = CrossModalAttention(
            query_dim=hidden_dim, key_dim=hidden_dim, value_dim=hidden_dim,
            num_heads=num_heads
        )
        
        # 另一个跨模态注意力：语言查询视觉
        self.l2v_attention = CrossModalAttention(
            query_dim=hidden_dim, key_dim=hidden_dim, value_dim=hidden_dim,
            num_heads=num_heads
        )
        
        # 门控融合层
        self.gate_fc = nn.Linear(hidden_dim * 2, hidden_dim)
        
        # -dropout
        self.dropout = nn.Dropout(dropout)
        
        # 层归一化
        self.layer_norm = nn.LayerNorm(hidden_dim)
    
    def forward(self, visual_obs: torch.Tensor, 
                instruction_tokens: torch.Tensor) -> torch.Tensor:
        """
        视觉-语言特征融合
        
        Args:
            visual_obs: (batch, 50) 视觉观测
            instruction_tokens: (batch, max_seq_len) 指令token序列
        Returns:
            fused_repr: (batch, hidden_dim) 融合后的表示
        """
        # 编码指令
        instruction_repr = self.instruction_encoder(instruction_tokens)
        
        # 编码视觉
        visual_repr = self.visual_encoder(visual_obs)
        
        # 跨模态注意力交互
        # 视觉查询语言：让视觉关注指令中的相关部分
        v2l_out = self.v2l_attention(
            visual_repr.unsqueeze(1),  # (batch, 1, hidden_dim)
            instruction_repr.unsqueeze(1)  # (batch, 1, hidden_dim)
        )
        v2l_out = v2l_out.squeeze(1)  # (batch, hidden_dim)
        
        # 语言查询视觉：让语言关注视觉中的相关区域
        l2v_out = self.l2v_attention(
            instruction_repr.unsqueeze(1),
            visual_repr.unsqueeze(1)
        )
        l2v_out = l2v_out.squeeze(1)
        
        # 门控融合
        concat = torch.cat([v2l_out, l2v_out], dim=-1)
        gate = torch.sigmoid(self.gate_fc(concat))
        fused_repr = gate * v2l_out + (1 - gate) * l2v_out
        
        # 残差连接和层归一化
        fused_repr = self.layer_norm(visual_repr + fused_repr)
        fused_repr = self.dropout(fused_repr)
        
        return fused_repr

```python
"""
VLN基础导航策略实现
基于PyTorch + 简化VLN环境的端到端策略网络
包含：Seq2Seq VLN、LSTM历史记忆、简单基线策略
作者: Wendy
"""
import torch
import torch.nn as nn
import torch.nn.functional as F
import numpy as np
import random
from typing import Tuple, List, Dict
from collections import deque


class Seq2SeqVLNPolicy(nn.Module):
    """
    Seq2Seq VLN
class Seq2SeqVLNPolicy(nn.Module):
    """
    Seq2Seq VLN 导航策略
    
    核心组件：
    1. 指令编码器（LSTM）：将指令编码为固定向量
    2. 视觉编码器：将当前视觉观测编码为特征
    3. 历史记忆（LSTM）：维护历史轨迹的隐状态
    4. 跨模态注意力：视觉与语言信息的交互
    5. 动作解码器：输出4个动作的概率分布
    
    训练时使用教师强制（teacher forcing），推理时使用自回归
    """
    
    def __init__(self, vocab_size=100, embed_dim=64, hidden_dim=128,
                 visual_dim=50, action_dim=4, num_layers=1):
        super().__init__()
        
        self.hidden_dim = hidden_dim
        self.action_dim = action_dim
        
        # 指令编码器
        self.instruction_encoder = nn.Sequential(
            nn.Embedding(vocab_size, embed_dim, padding_idx=0),
            nn.LSTM(embed_dim, hidden_dim, num_layers=num_layers,
                   batch_first=True, bidirectional=True)
        )
        
        # 视觉编码器
        self.visual_encoder = nn.Sequential(
            nn.Linear(visual_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )
        
        # 历史记忆LSTM（关键：编码历史轨迹）
        self.history_lstm = nn.LSTMCell(hidden_dim * 2, hidden_dim)
        
        # 跨模态注意力
        self.cross_attention = nn.MultiheadAttention(
            embed_dim=hidden_dim, num_heads=4, batch_first=True
        )
        
        # 动作解码器
        self.action_head = nn.Sequential(
            nn.Linear(hidden_dim * 2, hidden_dim),
            nn.ReLU(),
            nn.Dropout(0.1),
            nn.Linear(hidden_dim, action_dim)
        )
        
        # 初始化
        self._init_weights()
    
    def _init_weights(self):
        """权重初始化"""
        for m in self.modules():
            if isinstance(m, nn.Linear):
                nn.init.xavier_uniform_(m.weight)
                if m.bias is not None:
                    nn.init.zeros_(m.bias)
    
    def encode_instruction(self, tokens: torch.Tensor) -> torch.Tensor:
        """
        编码指令为固定向量表示
        
        Args:
            tokens: (batch, seq_len)
        Returns:
            instr_repr: (batch, hidden_dim)
        """
        embedded, (hidden, cell) = self.instruction_encoder(tokens)
        # 取双向最后一层隐状态拼接
        forward_h = hidden[-2]
        backward_h = hidden[-1]
        return torch.cat([forward_h, backward_h], dim=-1)
    
    def encode_visual(self, obs: torch.Tensor) -> torch.Tensor:
        """编码视觉观测"""
        return self.visual_encoder(obs)
    
    def forward(self, visual_obs: torch.Tensor,
               instruction_tokens: torch.Tensor,
               history_states: Tuple[torch.Tensor, torch.Tensor] = None,
               return_history: bool = False):
        """
        前向传播
        
        Args:
            visual_obs: (batch, visual_dim) 当前视觉观测
            instruction_tokens: (batch, max_seq_len) 指令token
            history_states: (h, c) 历史隐状态元组
            return_history: 是否返回更新后的历史状态
        Returns:
            action_logits: (batch, action_dim) 动作概率
            new_history: (h, c) 更新后的历史状态
        """
        batch_size = visual_obs.size(0)
        
        # 编码指令
        instr_repr = self.encode_instruction(instruction_tokens)  # (batch, hidden_dim*2)
        
        # 编码视觉
        visual_repr = self.encode_visual(visual_obs)  # (batch, hidden_dim)
        
        # 历史记忆更新
        if history_states is None:
            h = torch.zeros(batch_size, self.hidden_dim, device=visual_obs.device)
            c = torch.zeros(batch_size, self.hidden_dim, device=visual_obs.device)
        else:
            h, c = history_states
        
        # 将指令和视觉拼接作为LSTM输入
        lstm_input = torch.cat([visual_repr, instr_repr[:, :self.hidden_dim]], dim=-1)
        h_new, c_new = self.history_lstm(lstm_input, (h, c))
        
        # 跨模态注意力：历史查询指令
        h_expanded = h_new.unsqueeze(1)  # (batch, 1, hidden_dim)
        instr_expanded = instr_repr.unsqueeze(1)  # (batch, 1, hidden_dim*2)
        
        # 投影到相同维度
        h_proj = h_new.unsqueeze(1)  # 作为query
        # 将指令投影为key和value
        k = nn.functional.linear(instr_repr, 
                                 torch.eye(self.hidden_dim * 2, self.hidden_dim,
                                          device=instr_repr.device).repeat(1, 1))
        v = nn.functional.linear(instr_repr,
                                  torch.eye(self.hidden_dim * 2, self.hidden_dim,
                                           device=instr_repr.device).repeat(1, 1))
        
        # 简化注意力：直接拼接后MLP
        context = torch.cat([h_new, instr_repr[:, :self.hidden_dim]], dim=-1)
        
        # 动作解码
        action_logits = self.action_head(context)
        
        if return_history:
            return action_logits, (h_new, c_new)
        return action_logits
    
    def get_action(self, visual_obs: torch.Tensor,
                  instruction_tokens: torch.Tensor,
                  history_states: Tuple[torch.Tensor, torch.Tensor] = None):
        """获取动作（贪心策略）"""
        with torch.no_grad():
            logits, new_history = self.forward(
                visual_obs, instruction_tokens, history_states, return_history=True
            )
            action = logits.argmax(dim=-1).item()
        return action, new_history


class VLNPPOAgent:
    """
    基于PPO的VLN导航智能体（简化版）
    
    使用PPO算法训练Seq2SeqVLNPolicy
    PPO核心思想：限制策略更新幅度，保证训练稳定
    """
    
    def __init__(self, policy: Seq2SeqVLNPolicy, lr=1e-3, gamma=0.99,
                 epsilon=0.2, device='cpu'):
        self.policy = policy.to(device)
        self.device = device
        self.gamma = gamma
        self.epsilon = epsilon
        
        # 优化器
        self.optimizer = torch.optim.Adam(policy.parameters(), lr=lr)
        
        # 用于存储轨迹
        self.trajectories = []
    
    def collect_trajectory(self, env, max_steps=50):
        """
        用当前策略收集一条轨迹
        
        Args:
            env: VLN环境
            max_steps: 最大步数
        Returns:
            trajectory: 包含 (obs, action, reward, done) 的列表
        """
        obs, instruction, info = env.reset()
        obs = torch.FloatTensor(obs).unsqueeze(0).to(self.device)
        instr_tokens = env.get_instruction_tokens(instruction)
        instr_tokens = torch.LongTensor(instr_tokens).unsqueeze(0).to(self.device)
        
        # padding到固定长度
        max_len = 20
        if instr_tokens.size(1) < max_len:
            padding = torch.zeros(1, max_len - instr_tokens.size(1),
                                 dtype=torch.long, device=self.device)
            instr_tokens = torch.cat([instr_tokens, padding], dim=1)
        
        trajectory = []
        history_states = None
        total_reward = 0
        
        for step in range(max_steps):
            # 获取动作
            action_logits, history_states = self.policy(
                obs, instr_tokens, history_states, return_history=True
            )
            action_probs = F.softmax(action_logits, dim=-1)
            action = action_probs.argmax(dim=-1).item()
            
            # 执行动作
            next_obs, reward, terminated, info = env.step(action)
            done = terminated
            
            # 存储
            trajectory.append({
                'obs': obs.clone(),
                'instr': instr_tokens.clone(),
                'action': action,
                'reward': reward,
                'done': done,
                'history': history_states,
                'probs': action_probs.clone()
            })
            
            total_reward += reward
            obs = torch.FloatTensor(next_obs).unsqueeze(0).to(self.device)
            
            if done:
                break
        
        return trajectory, total_reward
    
    def compute_returns(self, trajectory, gamma=0.99):
        """
        计算折扣回报和GAE
        
        Args:
            trajectory: 轨迹列表
            gamma: 折扣因子
        Returns:
            returns: 折扣回报
            advantages: GAE优势估计
        """
        rewards = [t['reward'] for t in trajectory]
        values = []  # 简化为reward-to-go
        
        # 计算折扣回报
        returns = []
        running_return = 0
        for r in reversed(rewards):
            running_return = r + gamma * running_return
            returns.insert(0, running_return)
        
        returns = torch.FloatTensor(returns)
        advantages = returns  # 简化版：用returns代替advantages
        
        return returns, advantages
    
    def update(self, trajectories):
        """
        PPO更新
        
        Args:
            trajectories: 多个轨迹的列表
        """
        all_loss = 0
        
        for traj in trajectories:
            returns, advantages = self.compute_returns(traj, self.gamma)
            
            # 重新计算当前策略的动作概率
            total_loss = 0.0
            for i, step_data in enumerate(traj):
                obs = step_data['obs']
                instr = step_data['instr']
                old_probs = step_data['probs']
                action = step_data['action']
                
                # 前向传播
                logits, _ = self.policy(obs, instr, None, return_history=True)
                new_probs = F.softmax(logits, dim=-1)
                
                # PPO裁剪损失
                prob_ratio = new_probs[0, action] / (old_probs[0, action] + 1e-8)
                clipped_ratio = torch.clamp(prob_ratio, 1 - self.epsilon,
                                            1 + self.epsilon)
                
                # 回报
                ret = advantages[i]
                
                # 策略损失（PPO裁剪）
                policy_loss = -torch.min(
                    prob_ratio * ret,
                    clipped_ratio * ret
                ).mean()
                
                total_loss += policy_loss
            
            # 更新
            self.optimizer.zero_grad()
            total_loss.backward()
            torch.nn.utils.clip_grad_norm_(self.policy.parameters(), 0.5)
            self.optimizer.step()
            
            all_loss += total_loss.item()
        
        return all_loss / len(trajectories)
    
    def train(self, env, num_iterations=20, trajectories_per_iter=4, max_steps=50):
        """
        训练循环
        
        Args:
            env: VLN环境
            num_iterations: 训练迭代数
            trajectories_per_iter: 每轮迭代收集的轨迹数
            max_steps: 每条轨迹最大步数
        """
        print("=" * 60)
        print("开始训练VLN PPO智能体")
        print("=" * 60)
        
        for iteration in range(num_iterations):
            # 收集轨迹
            trajectories = []
            total_rewards = []
            
            for _ in range(trajectories_per_iter):
                traj, total_reward = self.collect_trajectory(env, max_steps)
                trajectories.append(traj)
                total_rewards.append(total_reward)
            
            # 更新策略
            avg_loss = self.update(trajectories)
            
            # 统计
            success_count = sum(1 for t in total_rewards if t > 5)  # 简化的成功判断
            success_rate = success_count / len(total_rewards)
            avg_reward = np.mean(total_rewards)
            
            if (iteration + 1) % 5 == 0 or iteration == 0:
                print(f"Iter {iteration+1:3d}/{num_iterations} | "
                      f"Loss={avg_loss:.4f} | "
                      f"AvgReward={avg_reward:.2f} | "
                      f"SuccessRate={success_rate:.2f}")
        
        print("训练完成！")
        
        # 最终评估
        print("\n=== 最终评估（20 episodes）===")
        eval_rewards = []
        for _ in range(20):
            _, total_reward = self.collect_trajectory(env, max_steps)
            eval_rewards.append(total_reward)
        
        print(f"平均奖励: {np.mean(eval_rewards):.2f}")
        print(f"成功率: {sum(1 for r in eval_rewards if r > 5) / 20:.2f}")

### 简答题答案

**6. VLN与传统GPS导航的本质区别**

| 维度 | 传统GPS导航 | VLN |
|------|-------------|-----|
| **目标表示** | GPS坐标、路名 | 自然语言描述（"厨房"、"大植物旁"） |
| **环境先验** | 预建地图 | 无需地图，边探索边规划 |
| **感知方式** | GPS卫星信号 | 视觉摄像头 |
| **定位精度** | 米级精度 | 取决于视觉定位 |
| **适用场景** | 室外开阔道路 | 室内复杂环境 |

**VLN的优势**：
1. 自然交互：人类用自然语言描述目标是更自然的交互方式
2. 无需地图：避免了预先构建和维护地图的麻烦
3. 语义理解：能理解"去那个有大幅海报的咖啡厅"这类无法用坐标描述的目标
4. 室内适用：解决了GPS信号在室内不可用的问题

---

**7. VLN-CE相比R2R的难点分析**

**难点一：动作空间不同**
- R2R：智能体在预定义的导航图节点上选择下一跳，动作是离散的"跳转"
- VLN-CE：智能体执行低级动作原语（前进0.25m、左转15度），动作空间连续

**难点二：物理约束**
- R2R：可以"瞬移"到目标节点
- VLN-CE：必须真实移动，可能遇到障碍物、死胡同

**难点三：定位误差累积**
- R2R：节点位置精确已知
- VLN-CE：连续移动导致定位误差不断累积

**难点四：碰撞检测**
- R2R：无需考虑碰撞（预定义连通性）
- VLN-CE：需要实时检测与障碍物的碰撞

**连续动作空间的处理方式**：
- 航点预测：将连续空间分解为高层航点，低层控制器执行到达
- 强化学习：通过PPO等算法在连续动作空间中学习最优策略
- 模仿学习+修正：用专家轨迹训练，辅以在线修正（DAgger风格）

**8. 家庭服务机器人VLN导航系统设计**

**感知模块**：
- 传感器：RGB-D相机（Intel RealSense D455）+ 轮式里程计
- 视觉感知：使用CLIP/ViT提取图像特征，支持开放词汇识别
- 语义建图：维护环境的2D语义地图，标注房间类型和物体位置
- 技术选型：RealSense + ROS2 + OpenCV

**规划模块**：
- 全局规划：基于语言指令分解子目标，在语义地图上规划路径
- 局部规划：基于深度图像的实时避障（TEB局部规划器）
- 指令理解：用预训练VLM（如LLaVA）理解用户指令，提取地标和目标
- 技术选型：LLaVA指令理解 + RT-2风格的动作预测

**执行模块**：
- 运动控制：差速驱动底盘，ROS2 nav2导航栈
- 反馈机制：实时返回执行状态，支持追问
- 安全护栏：基于深度传感器的碰撞检测，紧急停止
- 技术选型：ROS2 nav2 + 嵌入式安全控制器

**完整pipeline**：
用户指令 → VLM理解 → 子目标分解 → 语义地图规划 → 局部避障执行 → 到达确认

## 七、本章小结

| 概念 | 要点 |
|------|------|
| **VLN** | Vision-Language-Navigation，视觉语言导航，让智能体根据语言指令在未知环境中导航 |
| **R2R** | Room-to-Room，经典VLN数据集，基于Matterport3D，包含21,000+条指令 |
| **VLN-CE** | 连续环境VLN，使用Habitat模拟器，更接近真实机器人 |
| **SR** | Success Rate，成功率，智能体在目标3米内停止的比例 |
| **SPL** | Success weighted by Path Length，加权成功率，综合成功和路径效率 |
| **NE** | Navigation Error，导航误差，最终位置与目标的平均距离 |
| **Seq2Seq VLN** | 开创性VLN模型，LSTM指令编码 + CNN视觉编码 + 跨模态注意力 |
| **Speaker-Follower** | 双向学习框架，指令生成扩充数据，解决数据稀缺问题 |
| **视觉记忆** | 用LSTM/GRU维护历史轨迹隐状态，解决历史依赖问题 |
| **跨模态对齐** | 注意力机制让视觉查询语言，实现语义实体和视觉区域的对应 |
