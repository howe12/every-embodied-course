## 1. 项目概述

### 1.1 项目目标

本项目旨在实现一个完整的 VLN 智能体，能够：

1. **理解自然语言指令**：智能体接收到 "Go straight and turn left at the brown sofa" 这样的指令后，能理解"直走"、"在棕色沙发处左转"等语义
2. **感知视觉环境**：通过第一人称视角的 RGB-D 图像理解当前所处位置
3. **自主决策导航**：结合语言和视觉信息，决定下一步的动作（前进、左转、右转、停止等）
4. **在仿真环境中执行**：在 Habitat 仿真平台中完成实际导航任务
5. **达到可观的性能指标**：在 R2R 数据集上达到 60%+ 的成功率

**项目边界**：

| 模块 | 内容 |
|------|------|
| 数据集 | R2R（Matterport3D） |
| 仿真平台 | Habitat-Sim |
| 模型 | 经典 Seq2Seq VLN 模型（带注意力机制） |
| 评估指标 | SR（成功率）、SPL（路径效率）、nDTW（动态时间规整） |
| 训练硬件 | GPU（建议 ≥ 16GB显存） |

## 2. 模型选择与训练

### 2.1 模型架构

本项目选用 **Seq2Seq + 视觉注意力** 的经典 VLN 模型，架构如下：

```
┌─────────────────────────────────────────────────────────┐
│                     输入层                               │
│  语言指令: "Walk past the kitchen..."  (T 个词元)       │
│  视觉图像: RGB 图像 (H×W×3)                             │
└──────────────┬──────────────────────────────────────────┘
               │
    ┌──────────▼──────────┐
    │   语言编码器 (LSTM)   │  输出: 语言上下文向量
    │   word2vec + BiLSTM  │
    └──────────┬──────────┘
               │
    ┌──────────▼──────────┐
    │   跨模态注意力      │  融合语言和视觉特征
    │  (Cross-Attention)  │
    └──────────┬──────────┘
               │
    ┌──────────▼──────────┐
    │   动作决策 (LSTM)    │  输出: 动作概率分布
    └──────────┬──────────┘
               │
         ┌─────▼─────┐
         │ Softmax   │
         │ 动作选择  │
         └───────────┘
```

**核心公式**：

视觉注意力的计算：

$$e_t = \mathbf{v}^\top \tanh(\mathbf{W}_1 \mathbf{h}_t + \mathbf{W}_2 \mathbf{f}_i)$$

其中 $\mathbf{h}_t$ 为当前时刻的语言隐状态，$\mathbf{f}_i$ 为第 $i$ 个视角的视觉特征，$\mathbf{W}_1, \mathbf{W}_2$ 为可学习权重。最终的上下文向量为所有视角的加权和：

$$\mathbf{c}_t = \sum_i \alpha_i \mathbf{f}_i, \quad \alpha_i = \frac{\exp(e_i)}{\sum_j \exp(e_j)}$$

动作概率分布：

$$P(a_t | a_{<t}, s_{<t}) = \text{Softmax}(\mathbf{W}_a [\mathbf{h}_t; \mathbf{c}_t])$$

### 2.3 训练流程

以下是完整的 VLN 模型训练流程：

```python
import torch
import torch.nn as nn
from torch.utils.data import DataLoader
from tqdm import tqdm

class VLNTrainer:
    """VLN 模型训练器"""
    
    def __init__(self, model, optimizer, train_loader, val_loader, config):
        self.model = model
        self.optimizer = optimizer
        self.train_loader = train_loader
        self.val_loader = val_loader
        self.config = config
        self.device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
        
        # 损失函数：动作预测交叉熵损失
        self.criterion = nn.CrossEntropyLoss(
            ignore_index=-1,  # 忽略填充位置
            reduction="mean"
        )
        
        # 学习率调度器：每 N 轮衰减一半
        self.scheduler = torch.optim.lr_scheduler.StepLR(
            optimizer=self.optimizer,
            step_size=config["lr_decay_every"],
            gamma=config["lr_decay"]
        )
    
    def train_epoch(self, epoch):
        """训练一个 epoch"""
        self.model.train()
        total_loss = 0.0
        total_correct = 0
        total_actions = 0
        
        pbar = tqdm(self.train_loader, desc=f"Epoch {epoch}")
        for batch_idx, batch in enumerate(pbar):
            # 将数据移动到设备上
            # batch 包含: instruction_ids, attention_mask,
            #            image_features, target_actions
            instruction_ids = batch["instruction_ids"].to(self.device)
            attention_mask = batch["attention_mask"].to(self.device)
            image_features = batch["image_features"].to(self.device)
            target_actions = batch["target_actions"].to(self.device)
            
            # 前向传播
            # 使用 teacher forcing：一定概率用真实标签作为解码器输入
            teacher_forcing_ratio = self.config["teacher_forcing_ratio"]
            use_teacher_forcing = torch.rand(1).item() < teacher_forcing_ratio
            
            # predicted_actions: [batch_size, seq_len, num_actions]
            predicted_actions = self.model(
                instruction_ids=instruction_ids,
                attention_mask=attention_mask,
                image_features=image_features,
                target_actions=target_actions if use_teacher_forcing else None
            )
            
            # 计算损失（展平以计算交叉熵）
            # 目标动作: [batch_size, seq_len]
            # 预测 logits: [batch_size, seq_len, num_actions]
            batch_size, seq_len, num_actions = predicted_actions.shape
            loss = self.criterion(
                predicted_actions.view(-1, num_actions),  # [batch_size*seq_len, num_actions]
                target_actions.view(-1)                      # [batch_size*seq_len]
            )
            
            # 反向传播
            self.optimizer.zero_grad()
            loss.backward()
            
            # 梯度裁剪，防止梯度爆炸
            torch.nn.utils.clip_grad_norm_(
                self.model.parameters(),
                max_norm=self.config["gradient_clip"]
            )
            
            self.optimizer.step()
            
            # 统计准确率
            with torch.no_grad():
                predicted_labels = predicted_actions.argmax(dim=-1)
                mask = (target_actions != -1)  # 忽略填充
                correct = ((predicted_labels == target_actions) & mask).sum().item()
                total = mask.sum().item()
                accuracy = correct / total if total > 0 else 0
            
            total_loss += loss.item()
            total_correct += correct
            total_actions += total
            
            pbar.set_postfix({
                "loss": f"{loss.item():.4f}",
                "acc": f"{accuracy:.4f}"
            })
        
        avg_loss = total_loss / len(self.train_loader)
        avg_acc = total_correct / total_actions if total_actions > 0 else 0
        return avg_loss, avg_acc
    
    def validate(self):
        """在验证集上评估模型"""
        self.model.eval()
        total_loss = 0.0
        
        with torch.no_grad():
            for batch in tqdm(self.val_loader, desc="Validating"):
                instruction_ids = batch["instruction_ids"].to(self.device)
                attention_mask = batch["attention_mask"].to(self.device)
                image_features = batch["image_features"].to(self.device)
                target_actions = batch["target_actions"].to(self.device)
                
                predicted_actions = self.model(
                    instruction_ids=instruction_ids,
                    attention_mask=attention_mask,
                    image_features=image_features,
                    target_actions=None  # 推理时不用 teacher forcing
                )
                
                batch_size, seq_len, num_actions = predicted_actions.shape
                loss = self.criterion(
                    predicted_actions.view(-1, num_actions),
                    target_actions.view(-1)
                )
                total_loss += loss.item()
        
        return total_loss / len(self.val_loader)
    
    def train(self):
        """完整训练流程"""
        best_val_loss = float("inf")
        
        for epoch in range(1, self.config["num_epochs"] + 1):
            # 训练
            train_loss, train_acc = self.train_epoch(epoch)
            
            # 验证
            val_loss = self.validate()
            
            # 学习率调度
            self.scheduler.step()
            
            print(f"\nEpoch {epoch}/{self.config['num_epochs']}:")
            print(f"  Train Loss: {train_loss:.4f}, Train Acc: {train_acc:.4f}")
            print(f"  Val Loss:   {val_loss:.4f}")
            print(f"  LR: {self.optimizer.param_groups[0]['lr']:.6f}")
            
            # 保存最佳模型
            if val_loss < best_val_loss:
                best_val_loss = val_loss
                self.save_checkpoint(epoch, is_best=True)
                print(f"  ✅ 新的最佳模型已保存！")
    
    def save_checkpoint(self, epoch, is_best=False):
        """保存模型检查点"""
        checkpoint = {
            "epoch": epoch,
            "model_state_dict": self.model.state_dict(),
            "optimizer_state_dict": self.optimizer.state_dict(),
            "scheduler_state_dict": self.scheduler.state_dict(),
        }
        
        # 保存最新检查点
        torch.save(checkpoint, "checkpoints/latest.pth")
        
        # 保存最佳检查点
        if is_best:
            torch.save(checkpoint, "checkpoints/best.pth")
```

### 3.2 环境配置

```python
import habitat
import numpy as np

# Habitat 配置
habitat_config = """
# 仿真器基础配置
SIMULATOR:
  AGENT_0:
    HEIGHT: 1.5              # 智能体高度（米）
    RADIUS: 0.17             # 智能体碰撞半径
  SCENE: ./data/scene.glb   # 3D 场景文件路径
  TURN AngLE: 30             # 每次旋转角度（度）
  
# 任务数据集配置
DATASET:
  TYPE: VLN-V1              # 任务类型
  SPLIT: train              # 数据集划分
  DATA_PATH: ./datasets/R2R/{split}.json
  
# 传感器配置（RGB-D 相机）
SNAPSHOT_SENSORS:
  RGB_SENSOR:
    SENSOR_TYPE: 1           # 彩色相机
    WIDTH: 640
    HEIGHT: 480
    HFOV: 90                 # 水平视场角
    POSITION: [0, 1.5, 0]   # 安装高度
  DEPTH_SENSOR:
    SENSOR_TYPE: 4           # 深度相机
    WIDTH: 640
    HEIGHT: 480
    MAX_DEPTH: 10.0          # 最大感知深度（米）
"""

class HabitatEnv:
    """Habitat 仿真环境封装"""
    
    def __init__(self, config_str):
        # 加载配置
        self.cfg = habitat.get_config(config_str=config_str)
        # 创建仿真环境
        self.env = habitat.Env(config=self.cfg)
        self.reset()
    
    def reset(self):
        """重置环境，返回初始观察"""
        observation = self.env.reset()
        return self._preprocess_obs(observation)
    
    def step(self, action):
        """
        执行动作并返回下一个观察
        action: 整数，0=停止, 1=前进, 2=左转, 3=右转
        """
        # 将整数动作映射为 Habitat 动作
        action_map = {
            0: habitat.SimulatorActions.STOP,
            1: habitat.SimulatorActions.MOVE_FORWARD,
            2: habitat.SimulatorActions.TURN_LEFT,
            3: habitat.SimulatorActions.TURN_RIGHT,
        }
        
        habitat_action = action_map.get(action, habitat.SimulatorActions.STOP)
        observation, reward, done, info = self.env.step(habitat_action)
        
        return self._preprocess_obs(observation), reward, done, info
    
    def _preprocess_obs(self, observation):
        """预处理观察数据"""
        return {
            "rgb": observation["rgb"],           # [H, W, 3] uint8
            "depth": observation["depth"],       # [H, W, 1] float32
            "position": self.env.get_agent(0).get_state().position,
            "rotation": self.env.get_agent(0).get_state().rotation,
        }
    
    def get_visual_features(self):
        """提取 ResNet 视觉特征（用于模型输入）"""
        obs = self.env.get_observation_at()  # 获取当前观察
        rgb_tensor = torch.from_numpy(obs["rgb"]).permute(2, 0, 1).float() / 255.0
        # 使用预训练的 ResNet50 提取特征
        with torch.no_grad():
            features = self.resnet_model(rgb_tensor.unsqueeze(0))
        return features.squeeze(0).numpy()  # [2048]
    
    def close(self):
        """关闭环境"""
        self.env.close()
```

### 3.3 场景加载

```python
import os
import json
import numpy as np
from habitat.core.dataset import Dataset
from habitat.core.simulator import Simulator

class R2RSceneLoader:
    """R2R 数据集场景加载器"""
    
    def __init__(self, data_root="./datasets/R2R"):
        self.data_root = data_root
        self.scenes = {}  # scan_id -> 场景连接图信息
        self.annotations = {}
    
    def load_annotations(self, split="train"):
        """加载数据划分标注"""
        annotation_path = os.path.join(self.data_root, f"{split}.json")
        
        with open(annotation_path, "r") as f:
            data = json.load(f)
        
        # 转换为方便查询的格式
        self.annotations[split] = {}
        for item in data:
            scan_id = item["scan"]
            if scan_id not in self.annotations[split]:
                self.annotations[split][scan_id] = []
            self.annotations[split][scan_id].append({
                "instruction": item["instructions"][0],  # 使用第一条指令
                "path": item["path"],
                "start_position": item["start_position"],
                "start_orientation": item["start_heading"],
                "goal_viewpoint": item["path"][-1],  # 路径最后一个点为目标
            })
        
        return self.annotations[split]
    
    def load_connectivity(self, scan_id):
        """加载指定扫描的连接图"""
        connectivity_path = os.path.join(
            self.data_root, "connectivity", f"{scan_id}_connectivity.json"
        )
        
        if not os.path.exists(connectivity_path):
            raise FileNotFoundError(f"连接图文件不存在: {connectivity_path}")
        
        with open(connectivity_path, "r") as f:
            connectivity = json.load(f)
        
        return connectivity
    
    def get_panorama_views(self, scan_id, viewpoint_id):
        """
        获取某个视角的 36 个子视图（每 30 度一个方位角，12 个高度）
        这是 VLN 中常用的多视角表示方法
        """
        connectivity = self.load_connectivity(scan_id)
        
        # 找到指定 viewpoint 的信息
        viewpoint_info = None
        for view in connectivity:
            if view["image_id"] == viewpoint_id:
                viewpoint_info = view
                break
        
        if viewpoint_info is None:
            raise ValueError(f"未找到视角: {viewpoint_id} in {scan_id}")
        
        # 返回该视角的所有连接信息
        return viewpoint_info.get("un_obss", [])
    
    def create_episode(self, scan_id, start_position, start_heading, goal_viewpoint, instruction):
        """创建一个 VLN 导航回合"""
        episode = {
            "episode_id": f"{scan_id}_{hash(instruction) % 100000}",
            "scene_id": f="./data/mp3d/{scan_id}/{scan_id}.glb",
            "start_position": start_position,
            "start_heading": start_heading,
            "goals": [{"position": goal_viewpoint, "radius": 1.0}],
            "instruction": instruction,
        }
        return episode
```

### 4.1 动作空间

VLN 任务中常用的动作空间定义为离散的 4 个动作：

| 动作 ID | 名称 | 说明 |Habitat 对应 |
|---------|------|------|-------------|
| 0 | **STOP** | 停止导航，目标到达 | `STOP` |
| 1 | **FORWARD** | 前进 0.25m（或场景定义的最小步长） | `MOVE_FORWARD` |
| 2 | **TURN_LEFT** | 左转 30° | `TURN_LEFT` |
| 3 | **TURN_RIGHT** | 右转 30° | `TURN_RIGHT` |

```python
# 动作空间定义
ACTION_SPACE = {
    0: {"name": "STOP", "description": "停止并判定为到达目标"},

### 4.2 决策模块

决策模块是 VLN 智能体的"大脑"，负责根据当前的语言指令和视觉观察选择最优动作。

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class VLNDecisionModule(nn.Module):
    """
    VLN 决策模块：融合语言指令和视觉观察，输出动作决策
    
    核心思想：维护一个隐状态 h_t，通过注意力机制动态关注
    视觉场景的不同部分和语言指令的不同词元
    """
    
    def __init__(self, config):
        super().__init__()
        self.hidden_size = config["hidden_size"]
        self.word_embedding_dim = config["word_embedding_dim"]
        self.vision_feature_dim = config["vision_feature_dim"]
        
        # 视觉特征投影层：将 ResNet 特征投影到隐藏维度
        self.vision_projection = nn.Sequential(
            nn.Linear(config["vision_feature_dim"], config["hidden_size"]),
            nn.ReLU(),
            nn.Dropout(config["dropout"])
        )
        
        # 语言编码器：BiLSTM 对指令进行编码
        self.language_encoder = nn.LSTM(
            input_size=config["word_embedding_dim"],
            hidden_size=config["hidden_size"] // 2,  # 因为是双向，所以除以 2
            num_layers=config["encoder_layers"],
            batch_first=True,
            bidirectional=True,
            dropout=config["dropout"] if config["encoder_layers"] > 1 else 0
        )
        
        # 跨模态注意力：语言查询视觉
        # 计算语言状态 h_t 对每个视觉特征 f_i 的注意力权重
        self.cross_attention = nn.MultiheadAttention(
            embed_dim=self.hidden_size,
            num_heads=8,
            dropout=config["dropout"],
            batch_first=True
        )
        
        # 动作解码器：LSTM 根据上下文解码出动作
        self.action_decoder = nn.LSTM(
            input_size=self.hidden_size + self.hidden_size,  # [视觉上下文; 上一步动作]
            hidden_size=config["hidden_size"],
            num_layers=config["decoder_layers"],
            batch_first=True,
            dropout=config["dropout"] if config["decoder_layers"] > 1 else 0
        )
        
        # 动作输出层
        self.action_head = nn.Linear(config["hidden_size"], config["num_actions"])
        
        # 停止动作输出层（判断是否应该停止）
        self.stop_head = nn.Linear(self.hidden_size, 1)
    
    def forward(self, instruction_tokens, instruction_mask, vision_features, 
                prev_action=None, hidden_state=None):
        """
        前向传播
        
        参数:
            instruction_tokens: [batch_size, max_instruction_length] 词元 ID
            instruction_mask:   [batch_size, max_instruction_length] 有效位置掩码
            vision_features:     [batch_size, num_views, vision_feature_dim] 多视角视觉特征
            prev_action:        [batch_size, hidden_size] 上一步动作的嵌入
            hidden_state:        tuple (h_0, c_0) 解码器隐状态
        
        返回:
            action_logits:  [batch_size, num_actions] 动作的 logit
            stop_logits:    [batch_size, 1] 停止信号的 logit
            attention_weights: [batch_size, num_views] 视觉注意力权重
            new_hidden_state:  tuple 更新后的隐状态
        """
        batch_size = instruction_tokens.shape[0]
        
        # 1. 语言编码
        # 将词 ID 转换为嵌入向量（这里假设已经完成了嵌入）
        language_output, (h_lang, c_lang) = self.language_encoder(
            instruction_tokens
        )
        # 取最后一个隐状态作为语言表示 [batch_size, hidden_size]
        language_state = h_lang[-1] if not self.language_encoder.bidirectional else torch.cat([h_lang[-1], h_lang[0]], dim=-1)
        
        # 2. 视觉特征投影
        # vision_features: [batch_size, num_views, vision_feature_dim]
        # -> vision_features: [batch_size, num_views, hidden_size]
        vision_features = self.vision_projection(vision_features)
        
        # 3. 跨模态注意力：语言查询视觉
        # language_state 作为 query，vision_features 作为 key 和 value
        # 使得智能体能够根据语言指令"关注"场景的不同部分
        attended_vision, attention_weights = self.cross_attention(
            query=language_state.unsqueeze(1),   # [batch_size, 1, hidden_size]
            key=vision_features,                    # [batch_size, num_views, hidden_size]
            value=vision_features                   # [batch_size, num_views, hidden_size]
        )
        attended_vision = attended_vision.squeeze(1)  # [batch_size, hidden_size]
        
        # 4. 解码器输入：如果有上一步动作则拼接，否则只用视觉上下文
        if prev_action is not None:
            decoder_input = torch.cat([attended_vision, prev_action], dim=-1)
        else:
            decoder_input = torch.cat([attended_vision, torch.zeros_like(attended_vision)], dim=-1)
        
        decoder_input = decoder_input.unsqueeze(1)  # [batch_size, 1, hidden_size*2]
        
        # 5. 动作解码
        if hidden_state is None:
            decoder_output, new_hidden_state = self.action_decoder(decoder_input)
        else:
            decoder_output, new_hidden_state = self.action_decoder(
                decoder_input, hidden_state
            )
        
        # 取解码器的输出作为决策特征
        decision_features = decoder_output.squeeze(1)  # [batch_size, hidden_size]
        
        # 6. 动作预测
        action_logits = self.action_head(decision_features)  # [batch_size, num_actions]
        
        # 7. 停止预测
        stop_logits = self.stop_head(decision_features)      # [batch_size, 1]
        
        return {
            "action_logits": action_logits,
            "stop_logits": stop_logits,
            "attention_weights": attention_weights,
            "hidden_state": new_hidden_state,
            "language_state": language_state,
            "attended_vision": attended_vision,
        }
    
    def sample_action(self, action_logits, temperature=1.0, greedy=False):
        """
        从动作分布中采样
        
        参数:
            action_logits: [batch_size, num_actions] 动作的原始分数
            temperature: 温度参数，控制随机性（越高越随机）
            greedy: 是否使用贪心策略（只选最高概率的动作）
        
        返回:
            action_ids: [batch_size] 采样的动作 ID
            action_probs: [batch_size, num_actions] 动作概率分布
        """
        action_probs = F.softmax(action_logits / temperature, dim=-1)
        
        if greedy:
            # 贪心：选概率最高的动作
            action_ids = action_probs.argmax(dim=-1)
        else:
            # 随机采样
            action_ids = torch.multinomial(action_probs, num_samples=1).squeeze(-1)
        
        return action_ids, action_probs
```

### 4.3 异常处理

在实际的 VLN 导航过程中，可能会遇到各种异常情况，需要特殊处理：

```python
class VLNNavigator:
    """VLN 导航器，含异常处理机制"""
    
    def __init__(self, model, env, config):
        self.model = model
        self.env = env
        self.config = config
        self.device = next(model.parameters()).device
        
        # 步数限制，防止无限循环
        self.max_steps = config.get("max_steps", 200)
        
        # 重复检测：记录最近 N 步的位置，检测循环
        self.loop_detection_window = config.get("loop_detection_window", 10)
        self.position_history = []
        
        # 障碍物检测：连续无法前进的次数
        self.stuck_counter = 0
        self.max_stuck = config.get("max_stuck", 5)
    
    def navigate(self, instruction, scan_id, start_position, start_heading):
        """
        执行一次完整的导航任务
        
        返回:
            trajectory: 导航轨迹（位置列表）
            success: 是否成功到达目标
            stop_reason: 停止原因
        """
        # 1. 初始化
        self.env.reset_scene(scan_id)
        self.env.set_agent_state(start_position, start_heading)
        
        self.position_history = []
        self.stuck_counter = 0
        trajectory = [start_position]
        hidden_state = None
        prev_action = None
        
        # 2. 对指令进行编码（预处理）
        instruction_tokens, attention_mask = self.tokenize_instruction(instruction)
        instruction_tokens = instruction_tokens.to(self.device)
        attention_mask = attention_mask.to(self.device)
        
        # 3. 逐步导航
        for step in range(self.max_steps):
            # 获取当前视觉观察
            vision_features = self.env.get_visual_features()  # [num_views, vision_dim]
            vision_features = torch.from_numpy(vision_features).unsqueeze(0).to(self.device)
            
            # 4. 模型决策
            outputs = self.model(
                instruction_tokens=instruction_tokens,
                instruction_mask=attention_mask,
                vision_features=vision_features,
                prev_action=prev_action,
                hidden_state=hidden_state
            )
            
            action_logits = outputs["action_logits"]
            stop_logits = outputs["stop_logits"]
            hidden_state = outputs["hidden_state"]
            
            # 5. 异常检测
            current_position = self.env.get_agent_position()
            
            # 检测循环（回到之前的位置）
            loop_detected = self._detect_loop(current_position)
            if loop_detected:
                print(f"⚠️ 步骤 {step}: 检测到循环，执行脱困策略")
                action_id = self._escape_loop_strategy()
            else:
                # 正常决策
                action_id, _ = self.model.sample_action(action_logits, temperature=0.7)
                action_id = action_id.item()
            
            # 6. 停止判断
            stop_prob = torch.sigmoid(stop_logits).item()
            if stop_prob > 0.7 and self._near_goal(current_position):
                action_id = 0  # 强制停止
            
            # 7. 执行动作
            if action_id != 0:  # 非停止动作
                success = self.env.step(action_id)
                if not success:
                    self.stuck_counter += 1
                    print(f"⚠️ 步骤 {step}: 动作 {action_id} 被阻塞 ({self.stuck_counter}/{self.max_stuck})")
                    
                    if self.stuck_counter >= self.max_stuck:
                        print("❌ 连续障碍超过限制，终止导航")
                        return trajectory, False, "STUCK"
                else:
                    self.stuck_counter = 0
                    new_position = self.env.get_agent_position()
                    trajectory.append(new_position)
            
            # 8. 更新 prev_action 嵌入（简化处理：用动作 ID 的 one-hot）
            prev_action = F.one_hot(
                torch.tensor([action_id]), 
                num_classes=4
            ).float().to(self.device)
            
            # 更新位置历史
            self.position_history.append(current_position)
            if len(self.position_history) > self.loop_detection_window:
                self.position_history.pop(0)
            
            # 9. 到达目标
            if action_id == 0:
                success = self._near_goal(current_position)
                return trajectory, success, "COMPLETED" if success else "WRONG_STOP"
        
        return trajectory, False, "MAX_STEPS_EXCEEDED"
    
    def _detect_loop(self, current_position, threshold=0.3):
        """检测是否陷入循环（当前位置与历史位置过于接近）"""
        for past_pos in self.position_history:
            distance = np.linalg.norm(
                np.array(current_position) - np.array(past_pos)
            )
            if distance < threshold:
                return True
        return False
    
    def _near_goal(self, current_position):
        """判断是否接近目标（距离 < 3m）"""
        goal_position = self.env.get_goal_position()
        distance = np.linalg.norm(
            np.array(current_position) - np.array(goal_position)
        )
        return distance < 3.0
    
    def _escape_loop_strategy(self):
        """脱困策略：当检测到循环时，随机旋转后继续前进"""
        import random
        # 随机选择左转或右转 90 度
        escape_action = random.choice([2, 3])  # 2=左转, 3=右转
        self.env.step(escape_action)
        self.env.step(escape_action)  # 转两次相当于 60 度（每次 30 度）
        return 1  # 脱困后执行前进

## 5. 性能评估

### 5.1 指标计算

VLN 任务的评估指标主要包括三大类：**成功率类**、**效率类**和**轨迹类**。

```python
import numpy as np
import json
from scipy.spatial.distance import directed_hausdorff

class VLNEvaluator:
    """VLN 模型评估器"""
    
    def __init__(self, ground_truth_data):
        """
        参数:
            ground_truth_data: list，每项包含 {
                "instruction": str,
                "path": [[x,y,z], ...],  # 真实轨迹
                "goal_position": [x,y,z],  # 目标位置
            }
        """
        self.ground_truth = ground_truth_data
    
    def compute_trajectory_length(self, trajectory):
        """计算轨迹总长度（米）"""
        if len(trajectory) < 2:
            return 0.0
        
        total_length = 0.0
        for i in range(1, len(trajectory)):
            dx = trajectory[i][0] - trajectory[i-1][0]
            dy = trajectory[i][1] - trajectory[i-1][1]
            dz = trajectory[i][2] - trajectory[i-1][2]
            total_length += np.sqrt(dx**2 + dy**2 + dz**2)
        
        return total_length
    
    def compute_success(self, predicted_trajectory, goal_position, success_threshold=3.0):
        """
        判断是否成功到达目标
        
        参数:
            predicted_trajectory: 预测轨迹 [[x,y,z], ...]
            goal_position: 目标位置 [x,y,z]
            success_threshold: 成功判定距离阈值（米）
        
        返回:
            bool: 是否成功
        """
        if len(predicted_trajectory) == 0:
            return False
        
        final_position = predicted_trajectory[-1]
        distance = np.linalg.norm(
            np.array(final_position) - np.array(goal_position)
        )
        return distance < success_threshold
    
    def compute_spl(self, predicted_trajectory, ground_truth_trajectory, 
                    goal_position, success_threshold=3.0):
        """
        计算 SPL（Success weighted by Path Length）指标
        
        SPL 是 VLN 最重要的综合指标，同时考虑成功率和路径效率：
        
        $$\text{SPL} = \frac{1}{N} \sum_{i=1}^{N} S_i \cdot \frac{l_i^*}{\max(l_i, l_i^*)}$$
        
        其中 $S_i$ 是第 $i$ 个任务是否成功，$l_i^*$ 是最短路径长度，
        $l_i$ 是实际轨迹长度。
        
        参数:
            predicted_trajectory: 预测轨迹
            ground_truth_trajectory: ground truth 轨迹
            goal_position: 目标位置
            success_threshold: 成功判定距离阈值
        
        返回:
            float: SPL 值 [0, 1]
        """
        is_success = self.compute_success(
            predicted_trajectory, goal_position, success_threshold
        )
        
        # 计算实际轨迹长度
        predicted_length = self.compute_trajectory_length(predicted_trajectory)
        
        # 计算 ground truth 轨迹长度作为最短路径估计
        optimal_length = self.compute_trajectory_length(ground_truth_trajectory)
        
        # 避免除零
        if optimal_length < 0.01:
            optimal_length = 0.01
        
        # SPL 公式
        if predicted_length < 0.01:
            return 0.0
        
        spl = is_success * (optimal_length / max(predicted_length, optimal_length))
        return spl
    
    def compute_ndtw(self, predicted_trajectory, ground_truth_trajectory, 
                     success_threshold=3.0):
        """
        计算 nDTW（normalized Dynamic Time Warping）指标
        
        nDTW 衡量预测轨迹与真实轨迹的相似度，使用动态时间规整（DTW）
        算法处理轨迹长度不一致的情况。它衡量智能体是否沿着类似的路径行进。
        
        参数:
            predicted_trajectory: 预测轨迹
            ground_truth_trajectory: 真实轨迹
            success_threshold: 成功判定距离阈值
        
        返回:
            float: nDTW 值 [0, 1]，越高越好
        """
        n = len(predicted_trajectory)
        m = len(ground_truth_trajectory)
        
        if n == 0 or m == 0:
            return 0.0
        
        # 构建 DTW 矩阵
        dtw_matrix = np.full((n + 1, m + 1), np.inf)
        dtw_matrix[0, 0] = 0
        
        for i in range(1, n + 1):
            for j in range(1, m + 1):
                # 计算当前点之间的距离
                dist = np.linalg.norm(
                    np.array(predicted_trajectory[i-1]) - 
                    np.array(ground_truth_trajectory[j-1])
                )
                
                # DTW 递推关系
                cost = dist
                dtw_matrix[i, j] = cost + min(
                    dtw_matrix[i-1, j],    # 插入
                    dtw_matrix[i, j-1],    # 删除
                    dtw_matrix[i-1, j-1]   # 匹配
                )
        
        # 计算原始 DTW 距离
        dtw_distance = dtw_matrix[n, m]
        
        # 归一化因子：假设最坏情况是两条轨迹完全分离
        max_distance = 100.0 * max(n, m)  # 假设最大距离 100m
        
        # nDTW = exp(-dtw_distance / max_distance)，归一化到 [0, 1]
        ndtw = np.exp(-dtw_distance / (max_distance * success_threshold))
        return ndtw
    
    def evaluate_trajectory(self, predicted_trajectory, ground_truth_trajectory,
                          goal_position, success_threshold=3.0):
        """评估单条轨迹，返回所有指标"""
        metrics = {
            "success": self.compute_success(predicted_trajectory, goal_position, success_threshold),
            "trajectory_length": self.compute_trajectory_length(predicted_trajectory),
            "optimal_length": self.compute_trajectory_length(ground_truth_trajectory),
            "spl": self.compute_spl(predicted_trajectory, ground_truth_trajectory, 
                                   goal_position, success_threshold),
            "ndtw": self.compute_ndtw(predicted_trajectory, ground_truth_trajectory, 
                                      success_threshold),
        }
        return metrics
    
    def evaluate_all(self, predicted_trajectories):
        """
        评估整个验证集
        
        参数:
            predicted_trajectories: list，每项为预测轨迹
        
        返回:
            dict: 所有指标的统计信息
        """
        total_success = 0
        total_spl = 0.0
        total_ndtw = 0.0
        n = len(predicted_trajectories)
        
        for i, pred_traj in enumerate(predicted_trajectories):
            gt = self.ground_truth[i]
            
            metrics = self.evaluate_trajectory(
                pred_traj["trajectory"],
                gt["path"],
                gt["goal_position"],
                success_threshold=3.0
            )
            
            total_success += int(metrics["success"])
            total_spl += metrics["spl"]
            total_ndtw += metrics["ndtw"]
        
        return {
            "success_rate": total_success / n,
            "average_spl": total_spl / n,
            "average_ndtw": total_ndtw / n,
            "num_episodes": n,
        }
```

### 5.2 结果分析

评估完成后，我们需要对结果进行深入分析，识别模型的强项和弱项。

```python
def analyze_results(results, ground_truth, predicted_trajectories):
    """
    分析 VLN 模型评估结果，识别问题模式
    
    分析维度：
    1. 轨迹长度分布：短轨迹 vs 长轨迹的成功率
    2. 指令复杂度：简单指令 vs 复杂指令的成功率
    3. 失败模式分类：过早停止、循环、偏离目标
    """
    print("=" * 60)
    print("VLN 模型评估结果分析")
    print("=" * 60)
    
    # 1. 基础指标
    print(f"\n📊 基础指标:")
    print(f"  成功率 (SR): {results['success_rate']:.2%}")
    print(f"  平均 SPL: {results['average_spl']:.4f}")
    print(f"  平均 nDTW: {results['average_ndtw']:.4f}")
    print(f"  评估样本数: {results['num_episodes']}")
    
    # 2. 按轨迹长度分析
    print(f"\n📏 按轨迹长度分析:")
    length_bins = [(0, 5), (5, 10), (10, 20), (20, 50)]
    for min_len, max_len in length_bins:
        short_traj_indices = []
        for i, gt in enumerate(ground_truth):
            opt_len = np.sum([
                np.linalg.norm(np.array(gt["path"][j]) - np.array(gt["path"][j-1]))
                for j in range(1, len(gt["path"]))
            ])
            if min_len <= opt_len < max_len:
                short_traj_indices.append(i)
        
        if short_traj_indices:
            successes = sum(1 for i in short_traj_indices 
                          if predicted_trajectories[i]["success"])
            print(f"  {min_len}-{max_len}m: {successes}/{len(short_traj_indices)} "
                  f"({successes/len(short_traj_indices):.2%})")
    
    # 3. 失败模式分析
    print(f"\n❌ 失败模式分类:")
    failure_reasons = {"TOO_EARLY": 0, "LOOP": 0, "OFF_TARGET": 0, 
                       "MAX_STEPS": 0, "COLLISION": 0}
    
    for pred in predicted_trajectories:
        reason = pred.get("stop_reason", "UNKNOWN")
        if reason in failure_reasons:
            failure_reasons[reason] += 1
    
    total_failures = sum(failure_reasons.values())
    for reason, count in failure_reasons.items():

## 6. 优化方向

### 6.1 数据增强

数据增强是提升 VLN 模型泛化能力的重要手段。

**方法一：指令增强（Instruction Augmentation）**

```python
import random
from typing import List

class InstructionAugmenter:
    """指令数据增强器"""
    
    def __init__(self, synonym_dict=None):
        # 方向词的同义词词典
        self.synonym_dict = synonym_dict or {
            "left": ["to the left", "on the left side", "leftward"],
            "right": ["to the right", "on the right side", "rightward"],
            "forward": ["straight ahead", "straight forward", "continue forward"],
            "straight": ["directly", "continue ahead", "go straight"],
            "turn": ["rotate", "spin", "go around"],
            "bedroom": ["the sleeping room", "the room with the bed"],
            "kitchen": ["the cooking area", "the food preparation room"],
            "living room": ["the lounge", "the sitting area", "the main room"],
        }
    
    def paraphrase(self, instruction: str) -> str:
        """
        对指令进行同义词替换，产生新的表达方式
        
        示例:
        "Turn left into the bedroom" -> "Rotate to the left into the sleeping room"
        """
        words = instruction.lower().split()
        paraphrased = []
        
        for word in words:
            # 去除标点
            clean_word = word.strip(".,!?")
            
            if clean_word in self.synonym_dict:
                # 随机选择同义词替换（30% 概率，保留部分原词）
                if random.random() < 0.3:
                    synonym = random.choice(self.synonym_dict[clean_word])
                    paraphrased.append(synonym)
                else:
                    paraphrased.append(word)
            else:
                paraphrased.append(word)
        
        result = " ".join(paraphrased)
        # 首字母大写，保持句子格式
        return result[0].upper() + result[1:] if result else result
    
    def augment_dataset(self, dataset, augmentation_ratio=3):
        """
        对整个数据集进行增强
        
        参数:
            dataset: 原始数据集
            augmentation_ratio: 每条数据增强的次数
        
        返回:
            augmented_dataset: 增强后的数据集
        """
        augmented = []
        
        for item in dataset:
            # 添加原始样本
            augmented.append(item)
            
            # 添加增强样本
            for _ in range(augmentation_ratio):
                new_item = item.copy()
                new_item["instruction"] = self.paraphrase(item["instruction"])
                augmented.append(new_item)
        
        return augmented
```

**方法二：轨迹增强（Trajectory Augmentation）**

```python
class TrajectoryAugmenter:
    """轨迹数据增强器"""
    
    def add_noise_to_trajectory(self, trajectory, noise_scale=0.1):
        """
        对轨迹添加微小噪声，模拟传感器误差
        """
        noisy_trajectory = []
        for pos in trajectory:
            noise = np.random.randn(3) * noise_scale
            noisy_pos = [pos[i] + noise[i] for i in range(3)]
            noisy_trajectory.append(noisy_pos)
        return noisy_trajectory
    
    def subsample_trajectory(self, trajectory, subsample_ratio=0.8):
        """
        随机降采样轨迹，减少轨迹点的数量
        使模型能处理不同密度的轨迹标注
        """
        n_points = len(trajectory)
        n_keep = max(2, int(n_points * subsample_ratio))
        indices = sorted(random.sample(range(n_points), n_keep))
        return [trajectory[i] for i in indices]
    
    def reverse_trajectory(self, trajectory, trajectory_id):
        """
        将轨迹反向，用反向终点作为新起点
        数据集容量翻倍，但方向相反
        """
        reversed_traj = trajectory[::-1]
        return reversed_traj
```

**方向二：引入对比学习损失**

除了标准的动作预测损失，引入对比学习损失可以提升跨模态对齐：

```python
class VLNContrastiveLoss(nn.Module):
    """
    对比学习损失：拉近相似（语言，视觉）对，推开不相似对
    
    损失函数：
    $$\mathcal{L}_{\text{contrastive}} = -\log \frac{\exp(\text{sim}(L, V^+)/\tau)}
    {\sum_{V^-} \exp(\text{sim}(L, V^-)/\tau)}$$
    """
    
    def __init__(self, temperature=0.1):
        super().__init__()
        self.temperature = temperature
    
    def forward(self, language_features, vision_features, positive_pairs=None):
        """
        参数:
            language_features: [batch_size, hidden_dim]
            vision_features: [batch_size, num_views, hidden_dim]
            positive_pairs: list of (lang_idx, view_idx) 正样本对
        
        返回:
            loss: 对比学习损失
        """
        # 将视觉特征池化
        vision_pooled = vision_features.mean(dim=1)  # [batch_size, hidden_dim]
        
        # 归一化
        language_features = F.normalize(language_features, dim=-1)
        vision_pooled = F.normalize(vision_pooled, dim=-1)
        
        # 计算余弦相似度矩阵
        similarity = torch.matmul(language_features, vision_pooled.T) / self.temperature
        # [batch_size, batch_size]
        
        # 对角线为正样本对
        labels = torch.arange(len(language_features)).to(language_features.device)
        
        # 交叉熵损失
        loss = F.cross_entropy(similarity, labels)
        return loss
```

### 6.3 域适应

域适应（Domain Adaptation）是解决仿真到真实（Sim-to-Real）迁移的关键技术。

```python
class DomainAdaptation:
    """
    域适应策略合集
    
    问题背景：
    仿真环境（Matterport3D）与真实环境存在视觉差异，
    训练好的模型在真实机器人上可能表现不佳。
    """
    
    @staticmethod
    def adversarial_domain_adaptation(model, feature_extractor, 
                                      domain_classifier, batch_source, batch_target):
        """
        对抗域适应：通过让域分类器无法区分源域和目标域的特征，
        使特征提取器学习到域不变的特征表示
        
        损失函数：
        $$\mathcal{L}_{\text{DA}} = \mathcal{L}_{\text{task}} - \lambda 
        \mathcal{L}_{\text{domain}}$$
        """
        # 从两个域提取特征
        source_features = feature_extractor(batch_source["image"])
        target_features = feature_extractor(batch_target["image"])
        
        # 合并特征，域标签：0=源域，1=目标域
        all_features = torch.cat([source_features, target_features], dim=0)
        domain_labels = torch.cat([
            torch.zeros(len(source_features)),
            torch.ones(len(target_features))
        ]).long().to(all_features.device)
        
        # 训练域分类器（梯度反转层实现对抗）
        domain_pred = domain_classifier(all_features)
        domain_loss = F.cross_entropy(domain_pred, domain_labels)
        
        return domain_loss
    
    @staticmethod
    def texture_randomization(env, randomization_prob=0.5):
        """
        纹理随机化：在仿真环境中随机改变物体纹理，
        迫使模型不依赖特定纹理，而学习几何和语义特征
        
        这是一种常见的 Sim-to-Real 策略
        """
        if random.random() < randomization_prob:
            # 随机改变场景中所有物体的颜色
            all_objects = env.get_all_objects()
            for obj in all_objects:
                new_color = np.random.rand(3)  # RGB
                obj.set_color(new_color)
```

```python
#!/usr/bin/env python3
"""
VLN 项目实战 - 完整训练脚本
功能：加载 R2R 数据集，训练 Seq2Seq VLN 模型，保存检查点
"""

import os
import sys
import json
import argparse
import numpy as np
import torch
import torch.nn as nn
from torch.utils.data import Dataset, DataLoader
from torchvision import transforms

# ========== 数据集定义 ==========

class R2RDataset(Dataset):
    """R2R 数据集加载器"""
    
    # 动作映射
    ACTION_MAP = {"stop": 0, "forward": 1, "left": 2, "right": 3}
    
    def __init__(self, split="train", data_root="./datasets/R2R", 
                 max_instruction_len=50, transform=None):
        self.split = split
        self.max_instruction_len = max_instruction_len
        self.transform = transform or self._default_transform()
        
        # 加载标注数据
        with open(os.path.join(data_root, f"{split}.json"), "r") as f:
            self.data = json.load(f)
    
    def _default_transform(self):
        return transforms.Compose([
            transforms.Resize((224, 224)),
            transforms.ToTensor(),
            transforms.Normalize(mean=[0.485, 0.456, 0.406],
                               std=[0.229, 0.224, 0.225])
        ])
    
    def __len__(self):
        return len(self.data)
    
    def __getitem__(self, idx):
        item = self.data[idx]
        
        # 解析指令
        instruction = item["instructions"][0]
        
        # 解析轨迹
        trajectory = item["path"]  # 路径点列表
        start_pos = item["start_position"]
        heading = item["start_heading"]
        
        # 加载视觉特征（实际应从预计算的特征文件加载）
        # 这里用随机初始化的特征做演示
        num_views = 36
        vision_features = np.random.randn(num_views, 2048).astype(np.float32)
        
        # 动作标签：每一步应该执行的动作（需要从轨迹计算）
        # 简化的动作标签计算：从当前位置指向下一个位置应该怎么走
        action_labels = self._compute_action_labels(trajectory)
        
        return {
            "instruction": instruction,
            "vision_features": vision_features,  # [36, 2048]
            "action_labels": action_labels,        # list of int
            "trajectory": trajectory,
            "scan_id": item["scan"],
        }
    
    def _compute_action_labels(self, trajectory):
        """根据轨迹计算每一步的动作标签"""
        # 简化处理：假设每一步都是前进
        labels = [1] * (len(trajectory) - 1) + [0]  # 最后一步停止
        return labels
    
    def collate_fn(self, batch):
        """DataLoader 的批量处理函数"""
        # 找最大指令长度和最大轨迹长度
        max_inst_len = max(len(item["instruction"].split()) 
                         for item in batch)
        max_inst_len = min(max_inst_len, self.max_instruction_len)
        max_traj_len = max(len(item["action_labels"]) for item in batch)
        
        batch_size = len(batch)
        
        # 初始化输出张量
        instruction_ids = torch.zeros(batch_size, max_inst_len, dtype=torch.long)
        attention_mask = torch.zeros(batch_size, max_inst_len, dtype=torch.long)
        vision_features = torch.zeros(batch_size, 36, 2048)
        action_labels = torch.zeros(batch_size, max_traj_len, dtype=torch.long)
        
        for i, item in enumerate(batch):
            # 处理指令
            words = item["instruction"].split()[:max_inst_len]
            instruction_ids[i, :len(words)] = torch.tensor(
                [hash(w) % 50000 for w in words]  # 简化的词 ID（实际用 tokenizer）
            )
            attention_mask[i, :len(words)] = 1
            
            # 处理视觉特征
            vision_features[i] = torch.from_numpy(item["vision_features"])
            
            # 处理动作标签
            labels = item["action_labels"][:max_traj_len]
            action_labels[i, :len(labels)] = torch.tensor(labels)
        
        return {
            "instruction_ids": instruction_ids,
            "attention_mask": attention_mask,
            "vision_features": vision_features,
            "target_actions": action_labels,
        }

# ========== 模型定义 ==========

class SimpleVLNModel(nn.Module):
    """简化版 VLN 模型（用于演示）"""
    
    def __init__(self, config):
        super().__init__()
        vocab_size = config.get("vocab_size", 50000)
        hidden_size = config.get("hidden_size", 512)
        vision_dim = config.get("vision_feature_dim", 2048)
        num_actions = config.get("num_actions", 4)
        
        # 词嵌入
        self.word_embedding = nn.Embedding(vocab_size, hidden_size)
        
        # 视觉投影
        self.vision_projection = nn.Sequential(
            nn.Linear(vision_dim, hidden_size),
            nn.ReLU(),
            nn.Dropout(0.3)
        )
        
        # 语言编码器
        self.language_encoder = nn.LSTM(
            hidden_size, hidden_size, batch_first=True, bidirectional=True
        )
        
        # 跨模态注意力
        self.cross_attention = nn.MultiheadAttention(
            hidden_size, num_heads=8, batch_first=True, dropout=0.1
        )
        
        # 动作解码器
        self.decoder = nn.LSTM(
            hidden_size * 2 + 4, hidden_size, batch_first=True
        )
        
        # 动作预测头
        self.action_head = nn.Linear(hidden_size, num_actions)
    
    def forward(self, instruction_ids, vision_features, target_actions=None):
        """
        前向传播
        
        参数:
            instruction_ids: [batch_size, seq_len]
            vision_features: [batch_size, 36, 2048]
            target_actions: [batch_size, max_len] 用于 teacher forcing
        
        返回:
            action_logits: [batch_size, max_len, num_actions]
        """
        batch_size, seq_len = instruction_ids.shape
        
        # 语言编码
        embedded = self.word_embedding(instruction_ids)  # [B, L, H]
        lang_out, (h_n, _) = self.language_encoder(embedded)
        lang_state = torch.cat([h_n[-1], h_n[-2]], dim=-1)  # 双向最后隐状态拼接
        
        # 视觉投影
        vision_proj = self.vision_projection(vision_features)  # [B, 36, H]
        
        # 跨模态注意力
        lang_query = lang_state.unsqueeze(1)  # [B, 1, H]
        attended_vision, _ = self.cross_attention(lang_query, vision_proj, vision_proj)
        attended_vision = attended_vision.squeeze(1)  # [B, H]
        
        # 动作解码（简化版：直接用注意力输出预测动作）
        # 实际模型应该逐时间步解码，这里简化处理
        max_len = seq_len
        
        # 重复视觉上下文以匹配序列长度
        attended_vision_expanded = attended_vision.unsqueeze(1).expand(-1, max_len, -1)
        lang_out_expanded = lang_out
        
        # 拼接语言和视觉特征
        combined = torch.cat([lang_out_expanded, attended_vision_expanded], dim=-1)
        
        # 解码
        decoder_out, _ = self.decoder(combined)
        
        # 动作预测
        action_logits = self.action_head(decoder_out)  # [B, L, num_actions]
        
        return action_logits

# ========== 训练主函数 ==========

```python
class SimpleAttention(nn.Module):
    def __init__(self, hidden_size, num_heads=4):
        super().__init__()
        self.hidden_size = hidden_size
        self.num_heads = num_heads
        self.head_dim = hidden_size // num_heads
        
        # 初始化 query, key, value 的投影层
        assert hidden_size % num_heads == 0, "hidden_size 必须能被 num_heads 整除"
        self.W_q = nn.Linear(hidden_size, hidden_size)  # Query 投影
        self.W_k = nn.Linear(hidden_size, hidden_size)  # Key 投影
        self.W_v = nn.Linear(hidden_size, hidden_size)  # Value 投影
        self.out_proj = nn.Linear(hidden_size, hidden_size)  # 输出投影
    
    def forward(self, query, key, value, mask=None):
        batch_size = query.size(0)
        
        # 1. 投影 query, key, value
        Q = self.W_q(query)  # [B, ..., H]
        K = self.W_k(key)
        V = self.W_v(value)
        
        # 重塑为多头形式 [B, num_heads, ..., head_dim]
        Q = Q.view(batch_size, -1, self.num_heads, self.head_dim).transpose(1, 2)
        K = K.view(batch_size, -1, self.num_heads, self.head_dim).transpose(1, 2)
        V = V.view(batch_size, -1, self.num_heads, self.head_dim).transpose(1, 2)
        
        # 2. 计算注意力分数：Q · K^T / sqrt(d_k)
        scores = torch.matmul(Q, K.transpose(-2, -1)) / (self.head_dim ** 0.5)
        # [B, num_heads, seq_len_q, seq_len_k]
        
        # 3. 应用 mask（如果有）
        if mask is not None:
            scores = scores.masked_fill(mask == 0, float("-inf"))
        
        # 4. 计算注意力权重并加权求和
        attention_weights = F.softmax(scores, dim=-1)
        # [B, num_heads, seq_len_q, seq_len_k]
        
        output = torch.matmul(attention_weights, V)
        # [B, num_heads, seq_len_q, head_dim]
        
        # 合并多头
        output = output.transpose(1, 2).contiguous().view(batch_size, -1, self.hidden_size)
        output = self.out_proj(output)
        
        return output, attention_weights
```

> **说明**：注意力机制的核心思想是"动态加权求和"——让模型学会在看到某个元素时，应该给上下文中的哪些部分分配更多的"注意力"权重，从而实现动态的信息选择。

**5. 模型分析（论述）**

- **外观偏置**：模型可能学习了环境中特定物体的外观（如某种颜色的沙发），而非语义概念
- **语言泛化不足**：模型可能只能理解训练集中出现过的指令表达方式，无法泛化到新指令

- **正则化**：增加 Dropout、使用 Label Smoothing

- **域适应技术**：使用对抗训练、纹理随机化等方法缩小 sim-to-real 差距

> - 在 VLN 研究中，**Val-Unseen（域外泛化）是更关键的指标**
> - 应该使用 Early Stopping 策略，监控 Val-Unseen 而非 Val-Seen
> - 需要更多关注模型架构和训练策略的泛化性设计

    words = inst_lower.split()
    
    # 遍历每个可能的同义词组，尝试替换
    for key, syns in synonyms.items():
        if key in inst_lower:
            for syn in syns:
                # 替换同义词（保持原文大小写风格的近似）
                new_inst = inst_lower.replace(key, syn)
                # 首字母大写
                new_inst = new_inst[0].upper() + new_inst[1:]
                augmented.append(new_inst)
    
    # ========== 策略二：指令简化 ==========
    # 提取关键动作词和目标词
    key_actions = ["turn left", "turn right", "go", "walk", "stop", "straight", "forward"]
    key_objects = ["bedroom", "kitchen", "living room", "bathroom", "sofa", "table", "door"]
    
    words = inst_lower.replace(",", " ").replace(".", " ").split()
    extracted_actions = [w for w in words if any(act in w for act in key_actions)]
    extracted_objects = [w for w in words if any(obj in w for obj in key_objects)]
    
    if extracted_objects:
        # 简化指令：只保留目标
        simple_inst = f"Go to the {extracted_objects[-1]}."
        simple_inst = simple_inst[0].upper() + simple_inst[1:]
        augmented.append(simple_inst)
    
    # 去重
    augmented = list(dict.fromkeys(augmented))
    return augmented if augmented else [instruction]

**7. 域适应分析（综合）**

> **答案**：

**① 数据层面**：

| 策略 | 说明 |
|------|------|
| **纹理随机化** | 在仿真环境中随机改变物体颜色/纹理，让模型不过度依赖特定外观 |
| **外观降级** | 将高分辨率纹理替换为低分辨率，迫使模型学习更鲁棒的表征 |
| **数据混合** | 混合多个仿真数据集（Matterport3D + Gibson），增加环境多样性 |
| **真实数据微调** | 在少量真实机器人数据上微调（Domain Adaptation） |

**② 模型架构层面**：

| 策略 | 说明 |
|------|------|
| **使用语义编码器** | 替换基于外观的 ResNet，用基于语义的编码器（如 SceneNet） |
| **图结构表示** | 使用场景拓扑图而非像素级图像，提升跨环境迁移能力 |
| **模块化设计** | 将导航策略与视觉感知解耦，感知模块更容易迁移 |
| **注意力可视化** | 分析模型关注的是语义目标还是外观特征，针对性改进 |

**③ 训练策略层面**：

| 策略 | 说明 |
|------|------|
| **对抗域适应（ADA）** | 训练域分类器，最小化源域和目标域特征的分布差异 |
| **梯度反转层（GRL）** | 在反向传播时反转域分类器梯度，实现端到端的域不变表示 |
| **一致性正则化** | 强制模型对同一场景的不同视图（如不同光照）输出一致的结果 |
| **元学习（MAML）** | 学习一个能从少量新环境样本快速适应的初始化 |
